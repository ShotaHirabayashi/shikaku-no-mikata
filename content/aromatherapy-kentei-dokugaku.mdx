## アロマテラピー検定とは？試験概要と合格率【2026年最新】

アロマテラピー検定は、**公益社団法人 日本アロマ環境協会（AEAJ）**が主催する民間資格です。受験資格は一切なく、年齢・学歴を問わず誰でも受験できます。

### 試験の基本情報（受験料・日程・形式）

| 項目 | 内容 |
|---|---|
| 主催 | 公益社団法人 日本アロマ環境協会（AEAJ） |
| 受験料 | 1級：6,600円／2級：6,600円／併願：13,200円（税込） |
| 試験形式 | オンライン（自宅受験） |
| 出題数 | 1級：70問／2級：55問 |
| 合格基準 | 正答率80% |
| 実施時期 | 年2回（5月・11月） |
| 次回試験 | 第54回：2026年5月実施、**申込開始は4月24日** |

**2025年5月（第52回）より香りテスト（精油をかいで答える問題）が廃止されました。** これにより、独学でも知識問題の対策だけで合格を目指せるようになっています。

### 1級と2級の違い｜迷ったら1級がおすすめ

2級で扱う精油は11種、1級は30種と範囲が広がりますが、**1級から直接受験が可能**です。併願もできるため、迷ったら1級を受験するのがおすすめ。1級に合格すればAEAJの上位資格への道も開けます。

### 合格率約90%のリアル｜落ちる10%の共通点

AEAJ公式サイトによると、1級・2級ともに合格率は約90%。ただし油断は禁物です。

不合格の10%に共通するのは、主に以下の2つです。

- **精油プロフィール（科名・抽出部位・製造法）の暗記不足**
- **歴史分野の軽視**（配点は少ないが確実に出題される）

逆に言えば、この2点を押さえれば独学でも合格は十分可能です。

## アロマテラピー検定を独学で合格する勉強法3ステップ

香りテスト廃止後の最新戦略は、**知識問題対策に全集中すること**。以下の3ステップで効率的に合格を目指しましょう。

### ステップ1：公式テキストを通読して全体像をつかむ

まずは公式テキストを最初から最後まで一通り読みましょう。この段階では暗記しなくてOK。「どんな分野が出るのか」「どこにボリュームがあるのか」を把握することが目的です。

出題分野のおおまかな配分は以下の通りです。

| 分野 | 配分目安 |
|---|---|
| 精油プロフィール | 約30% |
| 利用法・安全性 | 約25% |
| アロマテラピーのメカニズム | 約20% |
| 環境・法律 | 約15% |
| 歴史 | 約10% |

### ステップ2：精油プロフィール30種を繰り返し暗記する

最大の山場は1級で出題される**精油30種のプロフィール暗記**です。

効率的な覚え方は**グルーピング暗記法**。たとえば「ミカン科の精油」「花から抽出する精油」「水蒸気蒸留法の精油」のように分類してまとめると、バラバラに覚えるより格段に定着します。

### ステップ3：公式問題集を3周して出題パターンに慣れる

公式問題集は実際の試験と出題形式が近く、最も信頼できる対策教材です。

- **1周目**：全問解いて弱点を把握する
- **2周目**：間違えた問題を重点的に復習して定着させる
- **3周目**：通しで解いて仕上げる

歴史・法律分野は配点が少ないものの確実に出題されます。**試験直前の1週間で集中暗記**するのが効率的です。

## 独学の勉強期間はどれくらい？タイプ別スケジュール

受験者の約90%が3ヶ月以内に合格しているというデータがあります。勉強時間の目安は**合計30〜50時間**です。

### 短期集中型：2週間で合格するスケジュール

1日2〜3時間確保できる方向けのプランです。実際に2週間の独学で1級に合格した体験談もあります。

| 期間 | やること |

| 4〜7日目 | 精油プロフィール30種をグルーピング暗記 |

| 13〜14日目 | 問題集3周目＋歴史・法律の暗記仕上げ |

### 標準型：1〜2ヶ月でじっくり進めるスケジュール

1日30分〜1時間のペースで無理なく進めたい方におすすめです。

| 期間 | やること |
|---|---|
| 1〜2週目 | 公式テキスト通読（1日1〜2章ペース） |

- **精油30種暗記の中盤（15種あたり）**：似た名前の精油が増えて混乱しがち → グルーピング暗記法で分類整理する
- **歴史分野の退屈さ**：ストーリーとして流れをつかむと記憶に残りやすい → 年表を自作してみる

挫折しかけたら「合格率90%」を思い出してください。ほとんどの人が受かる試験です。
