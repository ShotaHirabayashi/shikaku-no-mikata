## FP3級とは

FP（ファイナンシャルプランナー）3級は、**お金に関する幅広い知識**を証明する国家資格です。税金、保険、年金、資産運用、不動産、相続など、生活に密着した6分野を学べます。

2024年からCBT（コンピューター）方式に移行し、**通年いつでも受験可能**になりました。

## 試験概要

| 項目 | 内容 |
|------|------|
| 試験方式 | CBT方式（通年受験可） |
| 受験料 | 学科4,000円 + 実技4,000円 |
| 試験時間 | 学科120分 + 実技60分 |
| 合格ライン | 各60%以上 |
| 受験資格 | 特になし |
| 合格率 | 学科約80%、実技約85% |

## 6つの学習分野

### 1. ライフプランニングと資金計画
社会保険、公的年金、住宅ローンなど、人生設計に関するお金の知識。

### 2. リスクマネジメント
生命保険、損害保険の種類と仕組み、適切な保険選び。

### 3. 金融資産運用
株式、債券、投資信託などの金融商品の基礎知識。

### 4. タックスプランニング
所得税、住民税の仕組み、確定申告、各種控除。

### 5. 不動産
不動産取引の基礎、法令上の制限、税金。

### 6. 相続・事業承継
相続税、贈与税の計算、遺言、相続対策。

## 最短合格のための学習スケジュール

### 2ヶ月プラン（1日1時間の場合）

| 期間 | 内容 | ポイント |
|------|------|---------|
| 1〜2週目 | テキスト通読（全6分野） | 全体像の把握に集中 |
| 3〜4週目 | テキスト2周目 + 問題演習 | 苦手分野を重点的に |
| 5〜6週目 | 過去問演習 | 出題傾向を把握 |
| 7〜8週目 | 模擬試験 + 弱点補強 | 合格ラインを安定して超える |

## 合格のコツ

1. **過去問を繰り返す** - FP3級は過去問からの類題が多い
2. **計算問題を確実に** - 税金や年金の計算は配点が高い
3. **法改正に注意** - 最新の税制改正を把握する
4. **実技は問題パターンを把握** - 出題パターンが限られている
5. **6割で合格** - 満点を目指さず、確実に合格点を取る

> 💡 **ポイント**: FP3級は独学で十分合格できる資格です。テキスト1冊と過去問サイトがあれば、追加費用はほぼかかりません。コスパ最強の資格と言えます。

## FP3級を取るメリット

### 実生活での活用

- **家計管理** - 保険の見直し、節税対策
- **資産運用** - NISAやiDeCoの理解
- **住宅購入** - ローンや税金の知識

### キャリアでの活用

- **金融業界への転職** - 基礎知識の証明
- **FP2級へのステップ** - 2級の受験資格を取得
- **副業** - お金のアドバイス、ライター

## まとめ

FP3級は、**人生で一番役立つ資格**と言っても過言ではありません。合格率が高く取得しやすいうえに、学んだ知識は毎日の生活に直結します。お金の知識を体系的に学びたい方は、ぜひチャレンジしてみてください。
