## ペン字検定（硬筆書写技能検定）とは

### 文部科学省後援の公的資格

硬筆書写技能検定は、**一般財団法人 日本書写技能検定協会**が実施する公的検定試験です。1964年の創設以来、延べ受験者数は1,000万人を超える歴史ある検定です。

| 項目 | 内容 |
|------|------|
| 正式名称 | 硬筆書写技能検定 |
| 主催 | 一般財団法人 日本書写技能検定協会 |
| 後援 | 文部科学省 |
| 級 | 5級・4級・3級・準2級・2級・準1級・1級 |
| 試験内容 | 実技＋理論 |
| 試験頻度 | 年3回（1月・6月・11月） |

| 級 | レベル | 対象者 |

## ペン字検定の取得方法

### 独学で取る場合

3級までは**市販のテキスト＋練習帳で独学可能**です。

1. 公式テキスト・問題集を購入
2. ペン字練習帳で毎日15〜30分練習
3. 理論科目の暗記（筆順、部首など）
4. 過去問で実力チェック
