## 問1

日本に仏教が公式に伝来（公伝）したとされる時期について、現在有力とされる説は何年か。

1. 478年
2. 538年
3. 607年
4. 645年

<QuizAnswer>

**答え: 2**

仏教の公伝は、百済の聖明王から欽明天皇に仏像や経典が伝えられた出来事を指します。「上宮聖徳法王帝説」や「元興寺縁起」に基づく538年説と、「日本書紀」に基づく552年説がありますが、現在の学界では538年説がより有力とされています。

</QuizAnswer>

## 問2

現存する世界最古の木造建築群として知られ、1993年に日本初の世界文化遺産に登録された寺院はどれか。

1. 東大寺
2. 法隆寺
3. 中尊寺
4. 薬師寺

<QuizAnswer>

**答え: 2**

法隆寺は飛鳥時代に建立された寺院で、金堂や五重塔などは現存する世界最古の木造建築群です。1993年に「法隆寺地域の仏教建造物」として姫路城とともに日本で最初の世界文化遺産に登録されました。エンタシスの柱など飛鳥様式の建築が特徴です。

</QuizAnswer>

## 問3

現存最古の和歌集である『万葉集』の表記に用いられた、漢字の音や訓を借りて日本語を書き表す方法を何というか。

1. 片仮名
2. 万葉仮名
3. 草仮名
4. 宣命書き

<QuizAnswer>

**答え: 2**

万葉仮名とは、漢字の音（おん）や訓（くん）を借りて日本語の音を表記する方法です。『万葉集』に多用されたことからこの名で呼ばれます。天皇から防人・農民まで幅広い階層の歌約4,500首が収められており、日本語表記の発展において重要な役割を果たしました。

</QuizAnswer>

## 問4

894年に遣唐使の廃止を建議した人物は誰か。

1. 空海
2. 最澄
3. 菅原道真
4. 藤原時平

<QuizAnswer>

**答え: 3**

菅原道真は894年（寛平6年）、唐の国内情勢の混乱や渡航の危険性を理由に遣唐使の廃止を建議しました。これにより遣唐使は以後派遣されなくなり、日本独自の国風文化が発展する契機の一つとなりました。菅原道真は後に学問の神として太宰府天満宮に祀られています。

</QuizAnswer>

## 問5

江戸時代、オランダ語の解剖書『ターヘル・アナトミア』を翻訳し、蘭学発展のきっかけとなった書物はどれか。

1. 蘭学事始
2. 解体新書
3. 西洋事情
4. 自然真営道

<QuizAnswer>

**答え: 2**

『解体新書』は1774年に杉田玄白・前野良沢らがオランダ語の解剖書『ターヘル・アナトミア』を翻訳して刊行したものです。これにより西洋医学への関心が高まり、蘭学が大きく発展しました。なお『蘭学事始』は杉田玄白が翻訳の苦労を回想して著した書物です。

</QuizAnswer>

## 問6

国学を大成し、『古事記』の注釈書である『古事記伝』全44巻を約35年かけて完成させた学者は誰か。

1. 賀茂真淵
2. 契沖
3. 本居宣長
4. 平田篤胤

<QuizAnswer>

**答え: 3**

本居宣長は江戸時代中期の国学者で、1764年に着手した『古事記伝』を1798年に完成させました。「もののあはれ」を日本文学の本質とする文学論でも知られています。賀茂真淵は宣長の師にあたり、契沖は国学の先駆者、平田篤胤は宣長の没後の門人とされます。

</QuizAnswer>

## 問7

葛飾北斎が描いた、「神奈川沖浪裏」などの作品を含む浮世絵の風景版画シリーズの名称はどれか。

1. 東海道五十三次
2. 名所江戸百景
3. 富嶽三十六景
4. 北斎漫画

<QuizAnswer>

**答え: 3**

『富嶽三十六景』は葛飾北斎が1831年頃から刊行した風景版画のシリーズで、富士山をさまざまな場所・角度から描いた作品群です。中でも「神奈川沖浪裏」は世界的に著名で、ヨーロッパの印象派の画家たちにも大きな影響を与えました。『東海道五十三次』は歌川広重の作品です。

</QuizAnswer>

## 問8

江戸時代に庶民の子弟に「読み・書き・そろばん」を教え、幕末には全国に1万数千か所以上存在した初等教育機関を何というか。

1. 藩校
2. 寺子屋
3. 私塾
4. 郷学

<QuizAnswer>

**答え: 2**

寺子屋は江戸時代に庶民の子弟を対象として「読み・書き・そろばん」などの実用的な知識を教えた初等教育機関です。幕末には全国に1万5千〜2万か所以上あったとされ、日本の高い識字率の基盤となりました。藩校は武士の子弟を対象とした藩の教育機関、私塾は特定の学者が開いた教育施設です。

</QuizAnswer>

## 問9

俳諧を芸術の域に高め、「わび・さび」の美意識を確立した「蕉風」の祖であり、『奥の細道』を著した人物は誰か。

1. 与謝蕪村
2. 小林一茶
3. 松尾芭蕉
4. 井原西鶴

<QuizAnswer>

**答え: 3**

松尾芭蕉は江戸時代前期の俳諧師で、1689年に江戸を出発して東北・北陸を巡った紀行文『奥の細道』で知られます。「夏草や兵どもが夢の跡」「閑さや岩にしみ入る蝉の声」など数多くの名句を残し、俳諧を「蕉風」として芸術的に高めました。与謝蕪村は蕉風の復興者、小林一茶は生活に根ざした句で知られます。

</QuizAnswer>

## 問10

日本の美術史において、日本の風物を題材に描かれた日本独自の絵画様式を何というか。

1. 南画
2. 大和絵
3. 水墨画
4. 文人画

<QuizAnswer>

**答え: 2**

大和絵は、中国風の唐絵（からえ）に対して、日本の風景や風俗、物語などを題材として描かれた日本独自の絵画様式です。平安時代の国風文化の発展とともに成立し、「源氏物語絵巻」などの絵巻物に代表されます。南画・文人画は中国の文人画の影響を受けた画風で、水墨画は墨の濃淡で描く中国起源の技法です。

</QuizAnswer>
