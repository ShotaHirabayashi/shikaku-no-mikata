## 問1

1868年、新政府が発表した新しい国家方針の基本を示した文書で、「広く会議を興し万機公論に決すべし」という一条で始まるものは何か。

1. 王政復古の大号令
2. 五箇条の御誓文
3. 五榜の掲示
4. 政体書


<QuizAnswer>

**答え: 2**

五箇条の御誓文は1868年（慶応4年）3月に明治天皇が神々に誓う形式で発布された、新政府の基本方針です。「広く会議を興し万機公論に決すべし」に始まる5か条からなり、公議世論の尊重や開国和親などが掲げられました。五榜の掲示は同時期に民衆向けに出された高札で、内容は異なります。

</QuizAnswer>

## 問2

1871年に実施された、全国の藩を廃止して府と県を置く制度改革を何というか。

1. 版籍奉還
2. 地租改正
3. 廃藩置県
4. 秩禄処分


<QuizAnswer>

**答え: 3**

廃藩置県は1871年（明治4年）に断行された改革で、全国の藩を廃止して中央政府が直接任命する府知事・県令を派遣する制度に切り替えました。これにより中央集権体制が確立されました。なお版籍奉還（1869年）は藩主が土地と人民を朝廷に返還したもので、廃藩置県の前段階にあたります。

</QuizAnswer>

## 問3

1877年に西郷隆盛を中心として鹿児島の士族が起こした、国内最後の大規模な士族反乱は何か。

1. 佐賀の乱
2. 萩の乱
3. 神風連の乱
4. 西南戦争


<QuizAnswer>

**答え: 4**

西南戦争（1877年）は、征韓論をめぐる政変で下野した西郷隆盛を擁して鹿児島の士族が蜂起した国内最後にして最大の士族反乱です。政府軍は徴兵制による軍隊でこれを鎮圧し、以後武力による政府への抵抗は終わりを告げました。これ以降、反政府運動は言論による自由民権運動へと転換していきます。

</QuizAnswer>

## 問4

自由民権運動の高まりを受けて、1881年に明治政府が出した、国会を開設する時期を約束した詔勅は何か。

1. 大日本帝国憲法発布の詔
2. 国会開設の勅諭
3. 立憲政体の詔書
4. 教育勅語


<QuizAnswer>

**答え: 2**

国会開設の勅諭は1881年（明治14年）に出され、1890年を期して国会を開設することを約束しました。この背景には自由民権運動の活発化や、北海道開拓使官有物払下げ事件をめぐる世論の批判がありました。この勅諭を受けて、板垣退助の自由党や大隈重信の立憲改進党など政党の結成が進みました。

</QuizAnswer>

## 問5

1889年に発布された大日本帝国憲法の起草の中心となった人物は誰か。

1. 大隈重信
2. 板垣退助
3. 伊藤博文
4. 山県有朋


<QuizAnswer>

**答え: 3**

伊藤博文はヨーロッパに渡り、ドイツ（プロイセン）の憲法学者グナイストやシュタインから立憲君主制の憲法理論を学びました。帰国後、井上毅らとともに大日本帝国憲法の起草にあたり、1889年（明治22年）2月11日に発布されました。この憲法は天皇主権を基本とし、アジアで最初の近代的憲法として知られます。

</QuizAnswer>

## 問6

日清戦争後の1895年に締結された講和条約で、日本が清から遼東半島・台湾・澎湖諸島の割譲と賠償金を獲得した条約は何か。

1. 南京条約
2. 下関条約
3. 北京条約
4. ポーツマス条約


<QuizAnswer>

**答え: 2**

下関条約（馬関条約）は1895年に日本の伊藤博文・陸奥宗光と清の李鴻章の間で締結されました。清は朝鮮の独立を認め、遼東半島・台湾・澎湖諸島を日本に割譲し、賠償金2億両（テール）を支払うことになりました。しかし直後にロシア・フランス・ドイツによる三国干渉を受け、日本は遼東半島を返還しました。

</QuizAnswer>

## 問7

1911年に外務大臣・小村寿太郎がアメリカとの間で達成した、幕末以来の不平等条約の完全な解消とは何か。

1. 治外法権の撤廃
2. 関税自主権の完全回復
3. 最恵国待遇の廃止
4. 居留地制度の廃止


<QuizAnswer>

**答え: 2**

1911年に小村寿太郎が日米通商航海条約の改正に成功し、関税自主権の完全回復を達成しました。これにより、1858年の安政の不平等条約以来の課題がすべて解消されました。なお治外法権（領事裁判権）の撤廃は、1894年に陸奥宗光がイギリスとの間で日英通商航海条約を締結して実現しています。

</QuizAnswer>

## 問8

1872年に政府が公布した、「学問は身を立るの財本」として国民皆学を目指した教育に関する法令は何か。

1. 教育勅語
2. 学制
3. 教育令
4. 学校教育法


<QuizAnswer>

**答え: 2**

学制は1872年（明治5年）に公布された日本最初の近代的学校制度に関する法令です。フランスの学校制度を参考にし、全国を学区に分けて小学校の設置を進め、身分・性別を問わず教育を受けることを目指しました。しかし当初は学費が自己負担であったため就学率は低く、のちに改正が重ねられました。

</QuizAnswer>

## 問9

大正時代に「憲政の本義を説いてその有終の美を済すの途を論ず」を発表し、藩閥政治を批判して政党内閣制を主張した政治学者は誰か。

1. 美濃部達吉
2. 吉野作造
3. 尾崎行雄
4. 犬養毅


<QuizAnswer>

**答え: 2**

吉野作造は東京帝国大学教授で、1916年に雑誌『中央公論』に論文を発表し、「民本主義」を唱えました。主権の所在の議論を避けつつ、政治の目的は民衆の利福にあり、政策決定は民衆の意向に基づくべきだと主張しました。この民本主義は大正デモクラシーの理論的支柱となり、普通選挙運動や政党政治の発展に大きな影響を与えました。

</QuizAnswer>

## 問10

第一次世界大戦後の1920年に設立された、国際平和と協力を目的とする初の国際機関で、日本が常任理事国となったものは何か。

1. 国際連合
2. 国際連盟
3. ワシントン会議
4. 国際司法裁判所


<QuizAnswer>

**答え: 2**

国際連盟は1920年にアメリカ大統領ウィルソンの提唱により設立された、世界初の国際平和維持機構です。日本はイギリス・フランス・イタリアとともに常任理事国となりました。ただし提唱国のアメリカは議会の反対により不参加でした。日本は1933年に満州事変をめぐる対立から脱退しています。

</QuizAnswer>
