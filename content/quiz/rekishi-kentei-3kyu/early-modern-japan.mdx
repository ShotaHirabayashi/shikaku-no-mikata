## 問1

織田信長が1573年に追放した室町幕府最後の将軍は誰か。

1. 足利義満
2. 足利義政
3. 足利義昭
4. 足利義輝


<QuizAnswer>

**答え: 3**

足利義昭は室町幕府第15代将軍で、1573年に織田信長によって京都から追放され、室町幕府は事実上滅亡しました。義満は第3代、義政は第8代、義輝は第13代の将軍です。

</QuizAnswer>

## 問2

豊臣秀吉が全国の田畑の面積や収穫量を調査し、石高制の基礎を築いた政策を何というか。

1. 地租改正
2. 太閤検地
3. 班田収授法
4. 荘園整理令


<QuizAnswer>

**答え: 2**

太閤検地は豊臣秀吉が1582年頃から全国的に実施した土地調査で、田畑の面積・等級・石高を統一基準で測量しました。これにより荘園制が完全に解体され、一地一作人の原則が確立されて、近世の石高制の基礎となりました。

</QuizAnswer>

## 問3

江戸幕府の将軍と大名の主従関係を基盤とする政治体制を何というか。

1. 律令体制
2. 封建体制
3. 幕藩体制
4. 摂関体制


<QuizAnswer>

**答え: 3**

幕藩体制とは、江戸幕府（幕）と各地の藩（大名領）が全国を支配する政治体制のことです。将軍が大名に領地を与え、大名は領地内で一定の自治権を持ちつつも、参勤交代などの義務を負いました。この体制は約260年間維持されました。

</QuizAnswer>

## 問4

江戸幕府が1635年に制度化し、大名を江戸と領地に交互に住まわせた制度は何か。

1. 参勤交代
2. 武家諸法度
3. 御成敗式目
4. 禁中並公家諸法度


<QuizAnswer>

**答え: 1**

参勤交代は3代将軍徳川家光が1635年の武家諸法度改定で制度化したもので、大名に1年おきに江戸と領国を往復させる制度です。大名の妻子は江戸に住むことが義務づけられ、往復の費用が大名の財力を消耗させることで、幕府への反乱を防ぐ効果がありました。

</QuizAnswer>

## 問5

江戸幕府の鎖国体制下で、ヨーロッパの国として唯一貿易を許されたのはどの国か。

1. スペイン
2. ポルトガル
3. イギリス
4. オランダ


<QuizAnswer>

**答え: 4**

鎖国体制下では、ヨーロッパ諸国のうちオランダだけが長崎の出島で貿易を許されました。オランダはキリスト教の布教を行わず、貿易のみを目的としていたため幕府に認められました。このほか、中国（清）とも長崎で貿易が続けられていました。

</QuizAnswer>

## 問6

8代将軍徳川吉宗が行った幕政改革は次のうちどれか。

1. 寛政の改革
2. 天保の改革
3. 享保の改革
4. 正徳の治


<QuizAnswer>

**答え: 3**

享保の改革（1716〜1745年）は8代将軍徳川吉宗が行った幕政改革で、江戸三大改革の一つです。目安箱の設置、公事方御定書の制定、新田開発の奨励、上米の制などを実施しました。寛政の改革は松平定信、天保の改革は水野忠邦が主導しました。

</QuizAnswer>

## 問7

元禄文化の時代に「奥の細道」を著した俳人は誰か。

1. 小林一茶
2. 与謝蕪村
3. 松尾芭蕉
4. 正岡子規


<QuizAnswer>

**答え: 3**

松尾芭蕉は江戸時代前期の俳諧師で、1689年に東北・北陸を旅して紀行文「奥の細道」を著しました。「古池や蛙飛びこむ水の音」などの名句を残し、俳諧を芸術の域に高めた人物です。元禄文化を代表する文学者の一人です。

</QuizAnswer>

## 問8

豊臣秀吉が農民から武器を没収し、武士と農民の身分を明確に分離した政策を何というか。

1. 刀狩
2. 人返し令
3. 棄捐令
4. 異国船打払令


<QuizAnswer>

**答え: 1**

刀狩は1588年に豊臣秀吉が発令した法令で、農民から刀・脇差・槍・鉄砲などの武器を没収しました。一揆の防止を直接の目的としつつ、兵農分離を推し進める重要な政策でした。没収した武器は大仏建立の釘などに使うと名目が付けられました。

</QuizAnswer>

## 問9

1853年に浦賀に来航し、日本に開国を迫ったアメリカの提督は誰か。

1. ハリス
2. ペリー
3. プチャーチン
4. モリソン


<QuizAnswer>

**答え: 2**

マシュー・ペリーは1853年にアメリカ東インド艦隊を率いて浦賀に来航し、フィルモア大統領の親書を持って日本に開国を要求しました。翌1854年に再来航し、日米和親条約が締結されて下田・箱館の2港が開港されました。これにより約200年続いた鎖国体制は終わりを迎えました。

</QuizAnswer>

## 問10

江戸時代に大坂が「天下の台所」と呼ばれた理由として最も適切なものはどれか。

1. 幕府の政治の中心地であったため
2. 全国の年貢米や特産物が集まる商業の中心地であったため
3. 最大の人口を有する都市であったため
4. 朝廷が置かれた文化の中心地であったため


<QuizAnswer>

**答え: 2**

大坂は江戸時代に全国の藩が蔵屋敷を置き、年貢米や各地の特産物が集積・取引される商業・流通の一大中心地でした。そのため「天下の台所」と呼ばれました。政治の中心は江戸、朝廷が置かれたのは京都であり、大坂は経済・商業の中心として繁栄しました。

</QuizAnswer>
