## 問1

1185年、源頼朝が全国に守護・地頭を設置する権限を朝廷から認められたが、このきっかけとなった人物の逃亡事件に関わるのは誰か。

1. 源義経
2. 源範頼
3. 源義仲
4. 平宗盛


<QuizAnswer>

**答え: 1**

源頼朝は、弟の源義経の追捕を名目として、1185年に朝廷から全国への守護・地頭の設置権を獲得した。これにより頼朝の支配権は全国に及ぶこととなり、鎌倉幕府の基盤が確立された。義経はその後奥州藤原氏のもとに逃れたが、1189年に衣川で自害した。

</QuizAnswer>

## 問2

鎌倉幕府の第3代執権・北条泰時が1232年に制定した武家法の基本法典は何か。

1. 建武式目
2. 武家諸法度
3. 御成敗式目（貞永式目）
4. 禁中並公家諸法度


<QuizAnswer>

**答え: 3**

御成敗式目（貞永式目）は、1232年に北条泰時が制定した日本初の本格的な武家法典である。全51か条からなり、武家社会の慣習や道理に基づいて御家人間の紛争処理や所領に関する規定を定めた。以後、武家法の基本として長く影響を与えた。

</QuizAnswer>

## 問3

1274年と1281年の二度にわたる元軍の襲来を何と呼ぶか。

1. 刀伊の入寇
2. 元寇（蒙古襲来）
3. 応永の外寇
4. 倭寇の乱


<QuizAnswer>

**答え: 2**

元寇（蒙古襲来）とは、元のフビライ・ハンが日本に派遣した遠征軍による二度の襲来を指す。1274年の文永の役と1281年の弘安の役の総称である。幕府は御家人を動員して防戦し撃退に成功したが、十分な恩賞を与えられなかったことが御家人の不満を招き、幕府衰退の一因となった。

</QuizAnswer>

## 問4

1333年に鎌倉幕府を滅ぼした後、天皇親政を復活させた「建武の新政」を行った天皇は誰か。

1. 後白河天皇
2. 後鳥羽天皇
3. 後醍醐天皇
4. 後村上天皇


<QuizAnswer>

**答え: 3**

後醍醐天皇は、幕府打倒を計画し、足利尊氏や新田義貞らの協力を得て1333年に鎌倉幕府を滅ぼした。その後、天皇自ら政治を行う建武の新政を開始したが、公家重視の政策が武士の不満を招き、わずか2年余りで崩壊した。

</QuizAnswer>

## 問5

室町幕府の第3代将軍で、南北朝の合一を実現し、日明貿易（勘合貿易）を開始した人物は誰か。

1. 足利尊氏
2. 足利義満
3. 足利義政
4. 足利義昭


<QuizAnswer>

**答え: 2**

足利義満は室町幕府の第3代将軍として、1392年に南北朝の合一を実現し、約60年にわたる分裂を終結させた。また、明との間で勘合貿易を開始し、「日本国王」として冊封を受けた。京都の北山に金閣を建てたことでも知られ、北山文化の中心人物である。

</QuizAnswer>

## 問6

1221年、後鳥羽上皇が幕府打倒を企てて挙兵したが敗北した事件を何というか。

1. 承久の乱
2. 治承・寿永の乱
3. 正中の変
4. 元弘の変


<QuizAnswer>

**答え: 1**

承久の乱は、1221年に後鳥羽上皇が北条義時追討の院宣を出して挙兵した事件である。しかし、北条政子の演説で結束した幕府軍に敗れ、後鳥羽上皇は隠岐に流された。この乱の後、幕府は京都に六波羅探題を設置して朝廷を監視し、幕府の優位が確立した。

</QuizAnswer>

## 問7

鎌倉時代に「南無妙法蓮華経」の題目を唱えることを説き、法華経を信仰の中心とした僧は誰か。

1. 法然
2. 親鸞
3. 日蓮
4. 一遍


<QuizAnswer>

**答え: 3**

日蓮は、法華経こそが唯一の正しい教えであると主張し、「南無妙法蓮華経」の題目を唱えることで救われると説いた。他宗派や幕府を激しく批判したため、伊豆や佐渡に流罪となった。日蓮の教えは日蓮宗（法華宗）として広まり、特に商工業者に信者が多かった。

</QuizAnswer>

## 問8

1467年に始まり、約11年間にわたって京都を中心に続いた大乱で、戦国時代の幕開けとされる争乱は何か。

1. 観応の擾乱
2. 応仁の乱
3. 明徳の乱
4. 嘉吉の乱


<QuizAnswer>

**答え: 2**

応仁の乱は、1467年に将軍後継問題や管領家の内紛をきっかけに、細川勝元と山名宗全を中心とする東西両軍が京都で衝突した大乱である。戦いは約11年間続き、京都は荒廃した。この乱を契機に幕府の権威は失墜し、各地で下剋上の風潮が広まり、戦国時代へと突入していった。

</QuizAnswer>

## 問9

室町時代の文化のうち、第8代将軍足利義政の時代を中心とする文化を何というか。

1. 北山文化
2. 東山文化
3. 桃山文化
4. 国風文化


<QuizAnswer>

**答え: 2**

東山文化は、第8代将軍足利義政が京都東山に建てた銀閣（慈照寺）に代表される文化である。書院造や枯山水の庭園、水墨画、茶の湯、生け花など、わび・さびの美意識を特徴とする。北山文化の華やかさに対し、簡素で奥深い美を追求した点が特徴で、後の日本文化に大きな影響を与えた。

</QuizAnswer>

## 問10

鎌倉幕府において、将軍と御家人の間に結ばれた主従関係の基本となる、土地を媒介とした恩恵と奉仕の関係を何というか。

1. 荘園制度
2. 封建制度
3. 御恩と奉公
4. 知行国制度


<QuizAnswer>

**答え: 3**

御恩と奉公は、鎌倉幕府における将軍と御家人の主従関係の根幹をなす仕組みである。将軍は御家人に対して本領安堵（先祖伝来の所領の保障）や新恩給与（新たな所領の付与）といった御恩を与え、御家人は戦時の軍役や平時の京都大番役・鎌倉番役などの奉公で応えた。この関係が崩れたことが幕府滅亡の一因となった。

</QuizAnswer>
