## 問1

1931年9月、関東軍が南満州鉄道の線路を爆破し、これを中国軍の仕業として軍事行動を開始した事件を何というか。

1. 盧溝橋事件
2. 柳条湖事件
3. 西安事件
4. 張作霖爆殺事件


<QuizAnswer>

**答え: 2**

1931年9月18日、関東軍は奉天（現・瀋陽）郊外の柳条湖で南満州鉄道の線路を自ら爆破し、これを中国軍の仕業として軍事行動を開始しました。この柳条湖事件が満州事変の発端となり、翌1932年には満州国が建国されました。盧溝橋事件（1937年）は日中戦争の発端となった別の事件です。

</QuizAnswer>

## 問2

1933年、満州国の建国をめぐり国際連盟がリットン調査団の報告に基づいて日本に不利な決議を採択した結果、日本がとった行動はどれか。

1. 日独伊三国同盟を締結した
2. 国際連盟を脱退した
3. 満州国を解散させた
4. 国際連合に加盟した


<QuizAnswer>

**答え: 2**

1933年2月、国際連盟総会でリットン調査団の報告書に基づき、満州国の不承認と日本軍の撤退を求める決議が採択されました。これに反発した日本は同年3月に国際連盟を脱退し、国際的に孤立を深めていきました。日独伊三国同盟の締結は1940年のことです。

</QuizAnswer>

## 問3

1937年7月、北京郊外で日中両軍が衝突し、日中戦争の発端となった事件はどれか。

1. 満州事変
2. 上海事変
3. 盧溝橋事件
4. 南京事件


<QuizAnswer>

**答え: 3**

1937年7月7日、北京郊外の盧溝橋付近で日中両軍が衝突する盧溝橋事件が発生しました。この事件をきっかけに日中戦争が本格化し、戦線は中国全土に拡大していきました。満州事変（1931年）から数えて「十五年戦争」とも呼ばれます。

</QuizAnswer>

## 問4

1941年12月、日本がアメリカ・イギリスに宣戦布告して太平洋戦争が始まったが、開戦時に日本軍が奇襲攻撃を行った場所はどこか。

1. ミッドウェー島
2. フィリピン・レイテ島
3. ハワイ・真珠湾
4. 沖縄本島


<QuizAnswer>

**答え: 3**

1941年12月8日（日本時間）、日本海軍はハワイの真珠湾にあるアメリカ海軍基地を奇襲攻撃し、太平洋戦争が開始されました。同時にマレー半島への上陸作戦も行われました。ミッドウェー海戦（1942年）は日本が大敗した転換点となった戦いです。

</QuizAnswer>

## 問5

1945年8月、日本がポツダム宣言を受諾して降伏した直接的な要因として、正しくないものはどれか。

1. 広島への原子爆弾投下
2. 長崎への原子爆弾投下
3. ソ連の対日参戦
4. 東京大空襲の実施


<QuizAnswer>

**答え: 4**

東京大空襲は1945年3月10日に行われた出来事であり、8月のポツダム宣言受諾の直接的な要因ではありません。8月6日の広島への原爆投下、8月8日のソ連の対日参戦、8月9日の長崎への原爆投下が相次ぎ、8月14日に日本政府はポツダム宣言の受諾を決定しました。

</QuizAnswer>

## 問6

戦後、GHQ（連合国軍最高司令官総司令部）が実施した民主化政策に含まれないものはどれか。

1. 財閥解体
2. 農地改革
3. 婦人参政権の付与
4. 殖産興業の推進


<QuizAnswer>

**答え: 4**

殖産興業は明治時代に政府が推進した近代産業育成政策であり、GHQの戦後改革には含まれません。GHQは財閥解体により経済の民主化を、農地改革により地主制の解体を、婦人参政権の付与により政治の民主化を進めました。これらは日本の非軍事化・民主化を目的とした改革でした。

</QuizAnswer>

## 問7

日本国憲法の三大原則として正しい組み合わせはどれか。

1. 国民主権・基本的人権の尊重・平和主義
2. 国民主権・三権分立・平和主義
3. 基本的人権の尊重・三権分立・地方自治
4. 国民主権・基本的人権の尊重・議院内閣制


<QuizAnswer>

**答え: 1**

日本国憲法は1946年11月3日に公布、1947年5月3日に施行されました。その三大原則は「国民主権」「基本的人権の尊重」「平和主義」です。三権分立や議院内閣制、地方自治は憲法に定められた重要な制度ですが、三大原則には含まれません。

</QuizAnswer>

## 問8

1951年にサンフランシスコ平和条約に調印し、日本の独立回復に尽力した当時の内閣総理大臣は誰か。

1. 鳩山一郎
2. 吉田茂
3. 岸信介
4. 片山哲


<QuizAnswer>

**答え: 2**

1951年9月8日、吉田茂首相はサンフランシスコ平和条約に調印しました。この条約は翌1952年4月28日に発効し、約7年間に及んだ連合国による占領が終結して日本は主権を回復しました。同日、日米安全保障条約も締結されています。

</QuizAnswer>

## 問9

1964年に東京オリンピックが開催されたが、この時期の日本経済の状況を表す言葉として最も適切なものはどれか。

1. バブル経済
2. 高度経済成長
3. デフレ不況
4. 特需景気


<QuizAnswer>

**答え: 2**

1950年代後半から1970年代前半にかけて、日本は年平均約10%の実質GDP成長率を記録する高度経済成長を遂げました。1964年の東京オリンピック開催はその象徴的な出来事であり、東海道新幹線の開通や高速道路の整備も同時期に進みました。特需景気は朝鮮戦争（1950-53年）に伴うもので、バブル経済は1980年代後半の現象です。

</QuizAnswer>

## 問10

1972年、日本と中華人民共和国が国交を正常化した際の日本の内閣総理大臣は誰か。

1. 佐藤栄作
2. 田中角栄
3. 池田勇人
4. 三木武夫


<QuizAnswer>

**答え: 2**

1972年9月、田中角栄首相が中国を訪問し、日中共同声明に署名して日中国交正常化が実現しました。これにより日本は中華人民共和国を中国の唯一の合法政府として承認し、中華民国（台湾）との外交関係は断絶されました。同年5月には沖縄返還も実現しており、1972年は日本外交の大きな転換点となった年です。

</QuizAnswer>
