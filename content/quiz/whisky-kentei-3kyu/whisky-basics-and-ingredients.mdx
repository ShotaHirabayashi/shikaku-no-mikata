## 問1

ウイスキーは、酒類の大分類において次のどれに該当するか。

1. 醸造酒
2. 蒸留酒
3. 混成酒
4. 発泡酒


<QuizAnswer>

**答え: 2**

ウイスキーは穀物を原料とした醸造酒をさらに蒸留して造る「蒸留酒」に分類されます。醸造酒はビールやワインのように発酵させたものをそのまま飲むお酒、混成酒はリキュールのように蒸留酒や醸造酒に果実・香料等を加えたお酒です。

</QuizAnswer>

## 問2

ウイスキーの定義として正しくないものはどれか。

1. 穀物を原料としている
2. 蒸留を行っている
3. 木製の樽で熟成している
4. 果実を原料に含んでいる


<QuizAnswer>

**答え: 4**

ウイスキーの基本的な定義は「穀物を原料とし、糖化・発酵・蒸留を経て、木製の樽で熟成させた蒸留酒」です。果実を原料とする蒸留酒はブランデーなどであり、ウイスキーには該当しません。

</QuizAnswer>

## 問3

モルトウイスキーの原料として正しいものはどれか。

1. トウモロコシ
2. 大麦麦芽（モルト）
3. ライ麦
4. 小麦


<QuizAnswer>

**答え: 2**

モルトウイスキーは大麦麦芽（モルト）のみを原料として造られるウイスキーです。トウモロコシ、ライ麦、小麦などはグレーンウイスキーの原料として使用されます。「モルト」とは大麦を発芽させた麦芽のことを指します。

</QuizAnswer>

## 問4

グレーンウイスキーの特徴として正しいものはどれか。

1. 大麦麦芽のみを原料とする
2. 単式蒸留器で蒸留される
3. クリーンで軽快な味わいが特徴である
4. 複雑で重厚な風味が特徴である


<QuizAnswer>

**答え: 3**

グレーンウイスキーはトウモロコシや小麦などの穀物を主原料とし、連続式蒸留器で蒸留されます。そのため、クリーンで軽快な味わいが特徴です。大麦麦芽のみを原料とし単式蒸留器で蒸留されるのはモルトウイスキーの特徴です。

</QuizAnswer>

## 問5

シングルモルトウイスキーの定義として正しいものはどれか。

1. 単一の樽から瓶詰めされたウイスキー
2. 一つの蒸留所で造られたモルトウイスキー
3. 一種類の大麦品種のみで造られたウイスキー
4. 一回だけ蒸留されたモルトウイスキー


<QuizAnswer>

**答え: 2**

シングルモルトウイスキーとは、一つの蒸留所で造られたモルトウイスキーのことです。「シングル」は単一の蒸留所を意味します。単一の樽から瓶詰めされたものは「シングルカスク」と呼ばれ、シングルモルトとは異なる概念です。

</QuizAnswer>

## 問6

ブレンデッドウイスキーの説明として正しいものはどれか。

1. 複数の蒸留所のモルトウイスキーだけを混ぜ合わせたもの
2. モルトウイスキーとグレーンウイスキーをブレンドしたもの
3. 異なる国のウイスキーを混ぜ合わせたもの
4. 同じ蒸留所の異なる熟成年数のウイスキーを混ぜ合わせたもの


<QuizAnswer>

**答え: 2**

ブレンデッドウイスキーは、モルトウイスキーとグレーンウイスキーをブレンドして造られます。世界で流通しているウイスキーの大部分がブレンデッドウイスキーです。なお、複数の蒸留所のモルトウイスキーだけを混ぜ合わせたものは「ブレンデッドモルト（ヴァッテッドモルト）」と呼ばれます。

</QuizAnswer>

## 問7

「カスクストレングス」のウイスキーとはどのようなものか。

1. 樽の木材の風味が特に強いウイスキー
2. 加水せずに樽出しそのままの度数で瓶詰めしたウイスキー
3. 特別に大きな樽で熟成されたウイスキー
4. 樽から直接グラスに注いで飲むウイスキー


<QuizAnswer>

**答え: 2**

カスクストレングスとは、樽（カスク）から出したウイスキーに加水によるアルコール度数の調整を行わず、そのままの度数（通常50〜60%程度）で瓶詰めしたものです。通常のウイスキーは加水して40〜46%程度に調整されますが、カスクストレングスは原酒本来の濃厚な味わいを楽しめます。

</QuizAnswer>

## 問8

ウイスキーのラベルに「12年」と表示されている場合、その意味として正しいものはどれか。

1. ブレンドに使用した原酒の平均熟成年数が12年である
2. ブレンドに使用した原酒の中で最も長く熟成した原酒が12年である
3. ブレンドに使用した原酒の中で最も若い原酒が12年以上熟成している
4. そのボトルが蒸留所で12年間保管されていた


<QuizAnswer>

**答え: 3**

ウイスキーのラベルに記載される熟成年数は、ブレンドに使用された原酒の中で最も若い（熟成期間が短い）原酒の年数を表しています。つまり「12年」と表示されていれば、使用されたすべての原酒が12年以上熟成されていることを意味します。これは消費者保護の観点から定められたルールです。

</QuizAnswer>

## 問9

次のうち、醸造酒に分類されるものはどれか。

1. ウイスキー
2. ブランデー
3. ビール
4. ジン


<QuizAnswer>

**答え: 3**

ビールは穀物（大麦麦芽）を発酵させて造る醸造酒です。ウイスキー、ブランデー、ジンはいずれも蒸留酒に分類されます。醸造酒は原料を発酵させたものをそのまま飲むお酒で、ビールのほかワインや日本酒なども醸造酒に該当します。

</QuizAnswer>

## 問10

「シングルカスク」のウイスキーの説明として正しいものはどれか。

1. 一つの蒸留所のモルトウイスキーだけを使ったもの
2. 一種類の穀物だけを原料にしたもの
3. 一つの樽の原酒だけを瓶詰めしたもの
4. 一度だけ蒸留して造られたもの


<QuizAnswer>

**答え: 3**

シングルカスクとは、単一（シングル）の樽（カスク）から取り出した原酒だけを瓶詰めしたウイスキーのことです。複数の樽の原酒を混ぜ合わせる（ヴァッティングする）ことをせず、一つの樽の個性がそのまま反映されるため、同じ銘柄でも樽ごとに味わいが異なるのが特徴です。

</QuizAnswer>
