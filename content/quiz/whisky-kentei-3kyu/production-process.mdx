## 問1

ウイスキーの製麦（モルティング）工程において、大麦を発芽させるために冷水に浸す工程を何と呼ぶか。

1. キルニング
2. スティーピング
3. マッシング
4. ターニング


<QuizAnswer>

**答え: 2**

製麦工程は「浸麦（スティーピング）」→「発芽（ジャーミネーション）」→「乾燥（キルニング）」の3段階で構成される。スティーピングでは大麦を約2日間冷水に浸し、水分量を約42〜43%まで引き上げて発芽を促す。ターニングは発芽中に麦芽をかき混ぜる作業のことである。

</QuizAnswer>

## 問2

製麦工程の乾燥（キルニング）段階でピートを焚いてスモーキーな風味をつけることをピーティングと呼ぶが、このピートの香りの強さを示す指標として用いられるのはどれか。

1. IBU値
2. フェノール値（ppm）
3. ボーメ度
4. プルーフ値


<QuizAnswer>

**答え: 2**

ピートの香りの強さはフェノール値（ppm）で表される。ノンピートの蒸留所では2〜3ppm以下、ヘビーピートのラフロイグでは約45ppm、さらにオクトモアでは80ppmを超えるスーパーヘビリーピーテッドの麦芽が使用される。ピートの煙に含まれるフェノール化合物が麦芽に吸着されることでスモーキーな風味が生まれる。

</QuizAnswer>

## 問3

糖化（マッシング）工程では、粉砕した麦芽にお湯を加えてデンプンを糖に変換する。この工程で得られる甘い液体を何と呼ぶか。

1. ウォッシュ
2. ローワイン
3. ウォート
4. フェインツ


<QuizAnswer>

**答え: 3**

糖化工程で麦芽中の酵素がデンプンを麦芽糖（マルトース）に分解し、得られた甘い麦汁を「ウォート」と呼ぶ。ウォートは冷却後にウォッシュバック（発酵槽）に送られる。ウォッシュは発酵後の液体、ローワインは初留で得られる蒸留液のことである。

</QuizAnswer>

## 問4

発酵工程で使用される発酵槽のことを何と呼ぶか。

1. マッシュタン
2. スピリッツセイフ
3. ウォッシュバック
4. ポットスチル


<QuizAnswer>

**答え: 3**

発酵槽は「ウォッシュバック」と呼ばれ、木製（オレゴンパインやカラマツ）またはステンレス製のものが使われる。木製のウォッシュバックは乳酸菌などの微生物が木目に棲みつき独特の風味を生むとされ、ステンレス製は洗浄や微生物管理が容易でクリーンな仕上がりになる。マッシュタンは糖化に使う槽である。

</QuizAnswer>

## 問5

スコッチモルトウイスキーの蒸留において、初留釜（ウォッシュスチル）で蒸留して得られる液体を何と呼ぶか。

1. ニューポット
2. ローワイン
3. フォアショッツ
4. ハーツ


<QuizAnswer>

**答え: 2**

初留釜（ウォッシュスチル）にアルコール度数約6〜8%のウォッシュを投入して蒸留すると、アルコール度数約20〜25%の「ローワイン（Low Wines）」が得られる。このローワインは再留釜（スピリッツスチル）でさらに蒸留され、最終的にアルコール度数約65〜75%のニューメイクスピリッツとなる。

</QuizAnswer>

## 問6

再留（スピリッツスチル）の蒸留で流出する液体は3つの部分に分けられる。樽熟成に回される中間部分の蒸留液を何と呼ぶか。

1. フォアショッツ
2. フェインツ
3. ローワイン
4. ミドルカット（ハーツ）


<QuizAnswer>

**答え: 4**

再留では蒸留液を「フォアショッツ（前留）」「ミドルカット／ハーツ（本留）」「フェインツ（後留）」の3つに分ける。樽熟成に回されるのはミドルカット（ハーツ）のみで、フォアショッツとフェインツは次回の蒸留に戻して再蒸留される。カットのタイミングは蒸留所ごとに異なり、ウイスキーの個性を決定づける重要な判断となる。

</QuizAnswer>

## 問7

連続式蒸留機を改良し1831年に特許を取得した人物の名前に由来する蒸留機の呼び名はどれか。

1. ローモンドスチル
2. コフィースチル
3. ランビックスチル
4. アランビックスチル


<QuizAnswer>

**答え: 2**

連続式蒸留機は1826年にロバート・スタインが発明し、1831年にイーニアス・カフェ（Aeneas Coffey）が改良して特許を取得したことから「コフィースチル（カフェスチル）」とも呼ばれる。連続的に大量のグレーンウイスキーを効率よく蒸留でき、ブレンデッドウイスキー誕生の基盤となった。

</QuizAnswer>

## 問8

ポットスチルの頭部（ヘッド）に球状の膨らみがある形状は、蒸気の還流を促し軽やかな原酒を生み出す。この形状を何と呼ぶか。

1. ストレート型
2. オニオン型
3. バルジ型（ボール型）
4. ランプグラス型


<QuizAnswer>

**答え: 3**

ポットスチルの頭部に球状の膨らみを持つ形状を「バルジ型（ボール型）」と呼ぶ。この膨らみの中で蒸気が対流・還流を起こすため、重い成分が釜に戻りやすくなり、華やかで軽やかな原酒が得られる。一方、ストレート型は蒸気がそのまま上昇するため、力強く重厚な原酒になる傾向がある。

</QuizAnswer>

## 問9

樽熟成中にウイスキーが樽の木目を通じて自然に蒸発する現象を何と呼ぶか。

1. マリッジ
2. エンジェルズシェア（天使の分け前）
3. カスクフィニッシュ
4. チャーリング


<QuizAnswer>

**答え: 2**

樽熟成中にウイスキーが自然蒸発する現象を「エンジェルズシェア（天使の分け前）」と呼ぶ。スコットランドや日本では年間約2〜3%が蒸発し、12年熟成のスコッチでは元の量の約65〜75%程度まで減少する。気温が高い地域ほど蒸発率は高く、台湾やインドでは年間10%前後に達することもある。

</QuizAnswer>

## 問10

モルトウイスキー同士を混ぜ合わせたウイスキーは、2009年のスコッチウイスキー規則改正以降、正式に何と呼ばれるようになったか。

1. ブレンデッドウイスキー
2. シングルモルト
3. ブレンデッドモルト
4. カスクストレングス


<QuizAnswer>

**答え: 3**

モルトウイスキー同士を混ぜ合わせる「ヴァッティング」で造られたウイスキーは、かつて「ヴァッテッドモルト」と呼ばれていたが、2009年のスコッチウイスキー規則改正により「ブレンデッドモルト」に名称変更された。一方、モルト原酒とグレーン原酒を混ぜ合わせる「ブレンディング」で造られるのが「ブレンデッドウイスキー」である。

</QuizAnswer>
