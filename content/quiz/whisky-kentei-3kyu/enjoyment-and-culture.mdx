## 問1

ウイスキーをテイスティングする際に使用される「トワイスアップ」とは、どのような飲み方か。

1. ウイスキーと炭酸水を1:1で混ぜる飲み方
2. ウイスキーと常温の水を1:1で混ぜ、氷を入れない飲み方
3. ウイスキーを2杯連続で飲む飲み方
4. ウイスキーと温かいお湯を1:1で混ぜる飲み方


<QuizAnswer>

**答え: 2**

トワイスアップはウイスキーと常温の水を1:1（等量）で混ぜ、氷を入れずに飲む方法です。加水によって香りの分子が解放され、ウイスキー本来の香りや味わいが引き出されるため、スコッチのマスターブレンダーがテイスティング時に用いる標準的な方法として知られています。水割りとは異なり、氷を使わず常温の水を用いる点が特徴です。

</QuizAnswer>

## 問2

ウイスキーのテイスティングにおいて、グラスを傾けて戻した際に内壁を伝う液体の筋のことを何と呼ぶか。

1. ノージング
2. フィニッシュ
3. レッグス（ティアーズ）
4. ボディ


<QuizAnswer>

**答え: 3**

グラスの内壁をゆっくり伝い落ちる液体の筋は「レッグス」または「ティアーズ（涙）」と呼ばれます。これはウイスキーのアルコール度数や粘度を推測する指標となります。筋がゆっくり太く流れるほど、アルコール度数が高く粘性のあるウイスキーであることを示しています。

</QuizAnswer>

## 問3

ウイスキーのテイスティンググラスとして世界的に広く使われている「グレンケアングラス」の特徴として正しいものはどれか。

1. 長いステム（脚）があり、手の体温がウイスキーに伝わらない
2. ステム（脚）がなく背が低いチューリップ型で安定感がある
3. 球形で大きなボウルを持つブランデーグラスと同型である
4. 口径が広く平らな形状で香りを拡散させる設計である


<QuizAnswer>

**答え: 2**

グレンケアングラスはスコットランドの蒸溜所のマスターブレンダーと共同開発されたテイスティング専用グラスです。ステム（脚）のない背の低いチューリップ型が特徴で、持ちやすく安定感があります。飲み口がすぼまっているため香りがグラス内に集中し、効率的にノージング（香りを嗅ぐこと）ができる設計になっています。

</QuizAnswer>

## 問4

飲み方の一つである「ミスト」の名前の由来として正しいものはどれか。

1. ウイスキーに霧状の水を噴霧して加水することから
2. クラッシュアイスを使うことでグラス表面に霧のような水滴がつくことから
3. スコットランドの霧深い産地にちなんで名付けられたことから
4. 氷が溶ける際に白い靄が立ち上ることから


<QuizAnswer>

**答え: 2**

ミストはクラッシュアイス（細かく砕いた氷）をグラスいっぱいに詰め、そこにウイスキーを注ぐ飲み方です。グラスの表面に霧（mist）のような細かい水滴が付着することからこの名前が付けられました。氷が急速に溶けるため、冷たくすっきりとした飲み口が特徴で、夏場に人気のある飲み方です。

</QuizAnswer>

## 問5

カクテル「ラスティネイル」に使われるウイスキーとリキュールの組み合わせとして正しいものはどれか。

1. バーボンとアマレット
2. スコッチとドランブイ
3. アイリッシュウイスキーとベイリーズ
4. ライウイスキーとスイートベルモット


<QuizAnswer>

**答え: 2**

ラスティネイルはスコッチウイスキーとドランブイ（スコッチベースのハーブ系リキュール）を混ぜて作るカクテルです。なお、選択肢1のバーボンとアマレットの組み合わせは「ゴッドファーザー」、選択肢4のライウイスキーとスイートベルモットの組み合わせは「マンハッタン」というカクテルになります。

</QuizAnswer>

## 問6

ウイスキーの保存方法として正しいものはどれか。

1. ワインと同様に瓶を横に寝かせて保存する
2. 冷蔵庫で低温保存し、香りを閉じ込めるのがよい
3. 直射日光を避け、瓶を立てて冷暗所に保管する
4. 開封後は空気に触れさせた方が味が良くなるため、栓を開けたままにする


<QuizAnswer>

**答え: 3**

ウイスキーは直射日光や紫外線による成分の変質を避けるため、冷暗所（15〜20度程度）で瓶を立てて保管するのが正しい保存方法です。横に寝かせるとアルコール度数の高い液体がコルクを侵食・劣化させるため、ワインとは逆に必ず立てて保管します。開封後は酸化が進むため、しっかり栓をして早めに消費することが推奨されます。

</QuizAnswer>

## 問7

「ウイスキー（whisky）」という言葉の語源として正しいものはどれか。

1. 古代ギリシャ語で「琥珀色の液体」を意味する言葉
2. ゲール語の「ウシュク・ベーハー（生命の水）」が変化した言葉
3. ラテン語で「穀物の精霊」を意味する言葉
4. 古英語で「蒸留する」を意味する動詞が名詞化した言葉


<QuizAnswer>

**答え: 2**

ウイスキーの語源はゲール語の「ウシュク・ベーハー（uisge beatha）」で、「生命の水」を意味します。これはラテン語の「アクア・ヴィテ（aqua vitae＝生命の水）」をゲール語に翻訳したものです。uisge beatha → usquebaugh → usky → whisky と変化して現在の名称になりました。

</QuizAnswer>

## 問8

日本でハイボールブームが起きた際の中心的な取り組みとして正しいものはどれか。

1. 2000年にニッカウヰスキーがハイボール専門バーを全国展開した
2. 2008年頃からサントリーが「角ハイボール」戦略で飲食店への営業を推進した
3. 2015年にアサヒビールがハイボール缶を日本で初めて発売した
4. 1990年代にキリンビールがハイボールのテレビCMを大量に放映した


<QuizAnswer>

**答え: 2**

2008年頃からサントリーが「角ハイボール」を中心としたハイボール戦略を展開しました。飲食店への専用サーバー（ハイボールタワー）の設置、作り方マニュアルの整備、ビール並みの価格設定を推進した結果、取扱店が急増し、2009年には「ハイボール」が流行語にノミネートされるほどのブームとなりました。これにより長年低迷していたウイスキー市場が復活しました。

</QuizAnswer>

## 問9

ウイスキーのテイスティングにおいて「フィニッシュ」とは何を指すか。

1. テイスティングの最初に行う外観の観察のこと
2. グラスに鼻を近づけて香りを嗅ぐ工程のこと
3. ウイスキーを飲み込んだ後に残る余韻のこと
4. テイスティングノートに評価を記録する作業のこと


<QuizAnswer>

**答え: 3**

フィニッシュとは、ウイスキーを飲み込んだ後に口の中に残る余韻のことを指します。余韻の長さ（短い・中程度・長い）、余韻の質（心地よさや変化するフレーバー）が評価のポイントとなります。一般的に、長く心地よいフィニッシュは高品質なウイスキーの指標とされています。

</QuizAnswer>

## 問10

ウイスキーと食事のペアリングの基本的な考え方として、最も適切なものはどれか。

1. ウイスキーはどんな料理にも合うため、ペアリングを考える必要はない
2. ウイスキーのボディの重さと料理の味わいの強さを合わせるのが基本である
3. ウイスキーは必ずストレートで料理と合わせなければならない
4. ペアリングではウイスキーと正反対の味の料理だけを選ぶのがルールである


<QuizAnswer>

**答え: 2**

ウイスキーと食事のペアリングでは、ウイスキーのボディ（ライト・ミディアム・フル）の重さと料理の味わいの強さを合わせるのが基本です。ペアリングの手法には、似た風味同士を合わせる「共鳴」（例：スモーキーなウイスキーと燻製料理）と、異なる風味でバランスをとる「対比」（例：甘いバーボンとスパイシー料理）の2つのアプローチがあります。また、ハイボールは炭酸が口をリセットするため揚げ物と好相性など、飲み方によっても合う料理が変わります。

</QuizAnswer>
