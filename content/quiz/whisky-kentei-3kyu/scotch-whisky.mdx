## 問1

スコッチウイスキーの産地として一般的に認められている地域の数はいくつか。

1. 4つ
2. 5つ
3. 6つ
4. 8つ


<QuizAnswer>

**答え: 3**

スコッチウイスキーの産地は、スペイサイド、ハイランド、アイラ、ローランド、キャンベルタウン、アイランズの6つに分類されるのが一般的です。なお、スコッチ・ウイスキー協会（SWA）の公式保護地域としては5つ（アイランズをハイランドに含める）とされることもありますが、ウイスキー検定では6大産地として扱われます。

</QuizAnswer>

## 問2

スコッチウイスキーの6大産地のうち、蒸留所の数が最も多く、「ウイスキーの聖地」とも呼ばれる産地はどこか。

1. ハイランド
2. アイラ
3. スペイサイド
4. ローランド


<QuizAnswer>

**答え: 3**

スペイサイドにはスコットランド全体のモルトウイスキー蒸留所の約半数にあたる50以上の蒸留所が集中しており、世界で最もウイスキー蒸留所が密集した地域です。グレンフィディック、ザ・マッカラン、グレンリベットなど世界的に有名な蒸留所が多数あります。

</QuizAnswer>

## 問3

アイラ島のモルトウイスキーに共通する最大の特徴として、最も適切なものはどれか。

1. 3回蒸留による軽やかな味わい
2. ピートによるスモーキーで潮の香りを帯びた風味
3. シェリー樽由来のフルーティーな甘み
4. トウモロコシを主原料とした軽い飲み口


<QuizAnswer>

**答え: 2**

アイラモルトは、ピート（泥炭）を使って麦芽を乾燥させることで生まれる強いスモーキーフレーバーが最大の特徴です。さらに海に囲まれた島の環境から、潮っぽさやヨード香といった独特の海洋性の風味も加わります。アードベッグやラフロイグは特にピートが強いことで知られています。

</QuizAnswer>

## 問4

かつて30以上の蒸留所があり「ウイスキーの首都」と呼ばれたが、現在は3つの蒸留所のみが残っている産地はどこか。

1. ローランド
2. アイランズ
3. キャンベルタウン
4. ハイランド


<QuizAnswer>

**答え: 3**

キャンベルタウンは19世紀後半に30以上の蒸留所を擁し「ウイスキーの首都（Whisky Capital of the World）」と呼ばれていましたが、衰退によりほとんどが閉鎖されました。現在はスプリングバンク、グレンスコシア、グレンガイル（キルケラン）の3蒸留所のみが操業しています。

</QuizAnswer>

## 問5

スコッチウイスキーの法的定義において、スコットランド国内での最低熟成期間は何年と定められているか。

1. 2年
2. 3年
3. 5年
4. 8年


<QuizAnswer>

**答え: 2**

スコッチウイスキー規則（Scotch Whisky Regulations 2009）により、スコッチウイスキーはスコットランド国内で最低3年間、容量700リットル以下のオーク樽で熟成しなければならないと定められています。またアルコール度数は最低40%で瓶詰めすることも規定されています。

</QuizAnswer>

## 問6

1823年に制定され、密造時代に終止符を打つきっかけとなった法律はどれか。

1. モルト税法
2. 酒税法
3. 物品税法（Excise Act）
4. ウイスキー規制法


<QuizAnswer>

**答え: 3**

1823年の物品税法（Excise Act）は、蒸留免許の取得手続きを簡素化し税率を引き下げたことで、合法的なウイスキー製造を経済的に成り立つものにしました。これにより、17世紀末から続いていたスコットランド各地での密造・密輸の時代が終わりへ向かい、多くの蒸留所が正式に免許を取得するようになりました。

</QuizAnswer>

## 問7

スカイ島にある唯一の伝統的蒸留所として知られ、力強いスパイシーな味わいが特徴のブランドはどれか。

1. ハイランドパーク
2. タリスカー
3. ジュラ
4. トバモリー


<QuizAnswer>

**答え: 2**

タリスカー（Talisker）はスカイ島で1830年に設立された蒸留所で、長らく島唯一の蒸留所でした（2017年にトラベイグが新設）。黒胡椒のようなスパイシーさと海の潮気を併せ持つ力強い味わいが特徴で、アイランズを代表するシングルモルトとして高い人気を誇ります。

</QuizAnswer>

## 問8

ブレンデッドスコッチウイスキーを初めて商業的に成功させたとされる人物は誰か。

1. ジョン・ウォーカー
2. アンドリュー・アッシャー
3. トーマス・デュワー
4. ジョージ・バランタイン


<QuizAnswer>

**答え: 2**

アンドリュー・アッシャー（Andrew Usher）は、1853年頃に「アッシャーズ・オールド・ヴァッテッド・グレンリベット」を発売し、モルトとグレーンをブレンドしたウイスキーを商業的に成功させた最初の人物とされています。この成功がブレンデッドスコッチの時代を切り開き、スコッチウイスキーの世界的な普及につながりました。

</QuizAnswer>

## 問9

ローランド地方のオーヘントッシャン蒸留所が、スコットランドでは珍しい製法として知られる特徴はどれか。

1. ピートを全く使わない無煙麦芽の使用
2. 3回蒸留（トリプルディスティレーション）
3. 小容量の樽（50リットル以下）での熟成
4. 100%ライ麦を原料とする製法


<QuizAnswer>

**答え: 2**

オーヘントッシャン（Auchentoshan）は、スコットランドでは極めて珍しい3回蒸留を行う蒸留所です。通常スコッチのモルトウイスキーは2回蒸留ですが、3回蒸留することでより軽やかでクリーンな味わいが生まれます。この製法はアイリッシュウイスキーに多く見られるもので、ローランドらしい穏やかなスタイルを形づくっています。

</QuizAnswer>

## 問10

オークニー諸島にある蒸留所で、世界最北の蒸留所の一つとして知られ、ヘザーピートを使用した独特のフレーバーが特徴のブランドはどれか。

1. スキャパ
2. ボウモア
3. ハイランドパーク
4. スプリングバンク


<QuizAnswer>

**答え: 3**

ハイランドパーク（Highland Park）はオークニー諸島カークウォールにある蒸留所で、世界最北の蒸留所の一つです。地元産のヘザー（ヒース）を含むピートを使用して麦芽を乾燥させるため、一般的なピートとは異なる華やかでフローラルなスモーキーさが特徴です。アイランズ地域を代表する銘柄として世界的に高い評価を受けています。

</QuizAnswer>
