## 問1

アイリッシュウイスキーの伝統的な蒸留方法として、多くの蒸留所で採用されている蒸留回数はどれか。

1. 1回蒸留
2. 2回蒸留
3. 3回蒸留
4. 4回蒸留

<QuizAnswer>

**答え: 3**

アイリッシュウイスキーは伝統的に3回蒸留（トリプルディスティレーション）を行うのが特徴です。スコッチウイスキーが通常2回蒸留であるのに対し、3回蒸留することでより軽やかで滑らかな味わいになります。ただし、クーリー蒸留所のように2回蒸留を行う例外もあります。

</QuizAnswer>

## 問2

アメリカの法律において、バーボンウイスキーのマッシュビル（穀物配合）に含まれるトウモロコシの最低比率として正しいものはどれか。

1. 41%以上
2. 51%以上
3. 61%以上
4. 80%以上

<QuizAnswer>

**答え: 2**

バーボンウイスキーは米国連邦規則により、マッシュビルの51%以上をトウモロコシとすることが定められています。なお、トウモロコシ80%以上の場合は「コーンウイスキー」という別のカテゴリに分類されます。バーボンはさらに新品の内面を焦がしたオーク樽での熟成が義務付けられています。

</QuizAnswer>

## 問3

テネシーウイスキーの製法上の最大の特徴である「リンカーン・カウンティ・プロセス」で使用される木炭の原料は何か。

1. オーク（ナラ）
2. サトウカエデ（シュガーメープル）
3. ヒッコリー
4. チェリー（サクラ）

<QuizAnswer>

**答え: 2**

リンカーン・カウンティ・プロセスとは、蒸留後のスピリッツをサトウカエデ（シュガーメープル）の木炭層で濾過する工程のことです。通常3〜4メートルの厚さの木炭層を一滴一滴通過させることで雑味が除去され、滑らかな味わいが生まれます。この工程がテネシーウイスキーをバーボンと区別する最大の特徴です。

</QuizAnswer>

## 問4

カナディアンウイスキーの製法で、軽くニュートラルな味わいの「ベースウイスキー」と、風味豊かな「フレーバリングウイスキー」を別々に蒸留・熟成した後に合わせる手法が特徴的であるが、フレーバリングウイスキーの主原料として最も多く使われる穀物はどれか。

1. トウモロコシ
2. 小麦
3. ライ麦
4. 大麦

<QuizAnswer>

**答え: 3**

カナディアンウイスキーは、トウモロコシ主体の軽いベースウイスキーと、ライ麦主体の風味豊かなフレーバリングウイスキーをブレンドする手法が特徴です。このためカナディアンウイスキーは歴史的に「ライ（Rye）」という通称で呼ばれることもあります。ブレンダーの裁量によって多様な味わいが生み出されます。

</QuizAnswer>

## 問5

世界で最も売れているアイリッシュウイスキーのブランドはどれか。

1. ブッシュミルズ
2. タラモアデュー
3. ジェムソン
4. パワーズ

<QuizAnswer>

**答え: 3**

ジェムソン（Jameson）は世界で最も売れているアイリッシュウイスキーで、年間販売量は1,000万ケースを超えています。コーク州のミドルトン蒸留所で生産されており、ペルノ・リカール傘下のアイリッシュ・ディスティラーズが所有しています。ブレンデッドタイプで、滑らかで飲みやすい味わいが世界中で人気です。

</QuizAnswer>

## 問6

ジャックダニエルの蒸留所がある、テネシー州の町はどこか。

1. ナッシュビル
2. メンフィス
3. リンチバーグ
4. タラホーマ

<QuizAnswer>

**答え: 3**

ジャックダニエル蒸留所はテネシー州リンチバーグにあります。1866年に正式登録された米国最古級の蒸留所の一つで、創業者はジャスパー・ニュートン「ジャック」ダニエルです。現在はブラウン＝フォーマン社が所有しており、世界で最も売れているアメリカンウイスキーブランドとなっています。

</QuizAnswer>

## 問7

アメリカンウイスキーの分類において、「ライウイスキー」のマッシュビルに含まれるライ麦の最低比率として正しいものはどれか。

1. 41%以上
2. 51%以上
3. 61%以上
4. 71%以上

<QuizAnswer>

**答え: 2**

アメリカのライウイスキーは、マッシュビルの51%以上をライ麦とすることが法律で定められています。この規定はバーボンのトウモロコシ51%以上と同じ基準です。ライウイスキーはスパイシーでドライな味わいが特徴で、バーボンと同様に新品の内面を焦がしたオーク樽で熟成されます。

</QuizAnswer>

## 問8

ブッシュミルズ蒸留所が蒸留のライセンスを取得したとされる年として、ラベルにも記載されている年号はどれか。

1. 1494年
2. 1608年
3. 1784年
4. 1823年

<QuizAnswer>

**答え: 2**

1608年にサー・トーマス・フィリップスがイングランド王ジェームズ1世から蒸留酒（アクアヴィテ）を蒸留するライセンスを取得しました。ブッシュミルズ蒸留所はこの年を起源として掲げ、ボトルにも「1608」と記載しています。北アイルランドのアントリム州に位置し、「世界最古のライセンスを持つ蒸留所」として知られています。

</QuizAnswer>

## 問9

1939年に英国王ジョージ6世のカナダ訪問を記念して作られた、紫色の袋が特徴的なカナディアンウイスキーのブランドはどれか。

1. カナディアンクラブ
2. クラウンローヤル
3. アルバータプレミアム
4. フォーティクリーク

<QuizAnswer>

**答え: 2**

クラウンローヤル（Crown Royal）は、1939年の英国王ジョージ6世とエリザベス王妃のカナダ訪問を記念して特別に作られたカナディアンウイスキーです。紫色のベルベット袋に入ったパッケージが特徴的で、現在はディアジオ傘下にあります。マニトバ州ギムリの蒸留所で生産され、カナディアンウイスキーの売上トップブランドです。

</QuizAnswer>

## 問10

アイリッシュウイスキー独自のカテゴリである「シングルポットスチルウイスキー」の原料として正しい組み合わせはどれか。

1. 麦芽大麦と未麦芽大麦
2. 麦芽大麦とトウモロコシ
3. 麦芽大麦とライ麦
4. 麦芽大麦と小麦

<QuizAnswer>

**答え: 1**

シングルポットスチルウイスキーはアイルランド独自のカテゴリで、麦芽大麦（モルテッドバーレー）と未麦芽大麦（アンモルテッドバーレー）を混合してポットスチルで蒸留したものです。代表的な銘柄にはレッドブレストやグリーンスポットがあり、いずれもミドルトン蒸留所で生産されています。クリーミーでスパイシーな独特の味わいが特徴です。

</QuizAnswer>
