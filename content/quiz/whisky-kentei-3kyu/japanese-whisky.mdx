## 問1

日本初のモルトウイスキー蒸溜所である山崎蒸溜所を建設した人物は誰か。

1. 竹鶴政孝
2. 鳥井信治郎
3. 肥土伊知郎
4. 佐治敬三


<QuizAnswer>

**答え: 2**

山崎蒸溜所は1923年に鳥井信治郎によって京都府と大阪府の境に建設された、日本初のモルトウイスキー蒸溜所です。鳥井は寿屋（のちのサントリー）の創業者であり、日本でのウイスキー製造を志して山崎の地を選びました。竹鶴政孝は初代工場長として招かれ、技術面を担当しました。

</QuizAnswer>

## 問2

竹鶴政孝がウイスキー造りを学ぶためにスコットランドに渡った年はいつか。

1. 1910年
2. 1918年
3. 1923年
4. 1934年


<QuizAnswer>

**答え: 2**

竹鶴政孝は1918年にスコットランドへ渡り、グラスゴー大学に学びながらロングモーン蒸留所などで実習を行いました。現地でスコットランド人のジェシー・ロベルタ・カウアン（リタ）と出会い結婚し、1920年に帰国しています。この留学で得た知識が、のちの日本ウイスキー産業の礎となりました。

</QuizAnswer>

## 問3

竹鶴政孝が独立して設立した会社（のちのニッカウヰスキー）が最初に建設した蒸溜所はどこか。

1. 宮城峡蒸溜所
2. 山崎蒸溜所
3. 白州蒸溜所
4. 余市蒸溜所


<QuizAnswer>

**答え: 4**

竹鶴政孝は1934年に大日本果汁株式会社（のちのニッカウヰスキー）を設立し、北海道余市に蒸溜所を建設しました。スコットランドに似た冷涼で湿潤な気候を求めて余市の地を選んだとされています。石炭直火蒸留という伝統的な製法を採用していることでも知られます。

</QuizAnswer>

## 問4

ニッカウヰスキーの第二の蒸溜所である宮城峡蒸溜所が設立されたのは何年か。

1. 1934年
2. 1958年
3. 1969年
4. 1973年


<QuizAnswer>

**答え: 3**

宮城峡蒸溜所は1969年に竹鶴政孝によって宮城県仙台市に設立されました。余市とは異なる穏やかな気候のもとで、より軽やかで華やかな原酒を造ることを目的としています。スチーム加熱による蒸留を採用しており、余市の力強いスタイルとは対照的な個性を持ちます。

</QuizAnswer>

## 問5

サントリーの白州蒸溜所が建設された年と、その記念すべき理由の組み合わせとして正しいものはどれか。

1. 1963年 − サントリー創業50周年記念
2. 1973年 − サントリーウイスキー製造50周年記念
3. 1983年 − サントリー創業100周年記念
4. 1969年 − サントリーウイスキー製造40周年記念


<QuizAnswer>

**答え: 2**

白州蒸溜所は1973年、山崎蒸溜所でのウイスキー製造開始から50周年を記念して、山梨県北杜市（南アルプスの森林地帯）に建設されました。標高約700mの高地に位置し、豊かな自然環境のなかで爽やかで軽快な原酒が造られています。

</QuizAnswer>

## 問6

日本で最初に発売された国産ウイスキーの名称と発売年の組み合わせとして正しいものはどれか。

1. サントリー角瓶 − 1937年
2. サントリー白札（しろふだ） − 1929年
3. ブラックニッカ − 1934年
4. サントリーオールド − 1940年


<QuizAnswer>

**答え: 2**

日本初の国産ウイスキーは、1929年4月1日に寿屋（現サントリー）から発売された「サントリーウヰスキー白札」です。しかし当時の日本の消費者にはスモーキーな風味が受け入れられず、商業的には苦戦しました。その後改良を重ね、1937年発売の角瓶が大ヒットとなりました。

</QuizAnswer>

## 問7

2021年に日本洋酒酒造組合が定めた「ジャパニーズウイスキー」の表示基準において、原酒の最低熟成期間として定められているのは何年以上か。

1. 2年以上
2. 3年以上
3. 5年以上
4. 規定なし


<QuizAnswer>

**答え: 2**

2021年4月1日に施行された日本洋酒酒造組合の自主基準では、日本国内で700リットル以下の木樽に詰めて3年以上熟成させることが「ジャパニーズウイスキー」の表示条件の一つとなっています。この基準はスコッチウイスキーの規定に倣ったもので、原料の糖化・発酵・蒸留・熟成・瓶詰めのすべてが日本国内で行われることも要件です。

</QuizAnswer>

## 問8

埼玉県秩父市にあるベンチャーウイスキーの秩父蒸溜所を創業した人物は誰か。

1. 竹鶴威
2. 佐治敬三
3. 肥土伊知郎
4. 樋口雄一


<QuizAnswer>

**答え: 3**

秩父蒸溜所は肥土伊知郎（あくと いちろう）が設立したベンチャーウイスキー株式会社により、2008年に蒸留を開始しました。肥土は廃業した羽生蒸溜所の原酒を引き取り「イチローズモルト」として世界的に評価を受けたことでも知られています。日本のクラフトウイスキーブームの先駆けとなった蒸溜所です。

</QuizAnswer>

## 問9

2015年の「ジム・マレーウイスキーバイブル」でワールド・ウイスキー・オブ・ザ・イヤーに選ばれ、世界的なジャパニーズウイスキーブームの契機となった銘柄はどれか。

1. 響 21年
2. 余市 20年
3. 山崎シェリーカスク 2013
4. 竹鶴 17年


<QuizAnswer>

**答え: 3**

山崎シェリーカスク2013は、2015年版の「ジム・マレーウイスキーバイブル」で97.5点という高得点を獲得し、ワールド・ウイスキー・オブ・ザ・イヤーに選出されました。この受賞は世界中で大きな話題となり、ジャパニーズウイスキーの国際的な知名度と需要を飛躍的に高めるきっかけとなりました。

</QuizAnswer>

## 問10

北海道厚岸町に蒸溜所を構える厚岸蒸溜所が、その立地を選んだ主な理由として最も適切なものはどれか。

1. 東京からのアクセスの良さ
2. スコットランドのアイラ島に似た冷涼で海霧の多い気候
3. 国内最大の麦芽生産地への近さ
4. 温暖な気候による熟成の促進


<QuizAnswer>

**答え: 2**

厚岸蒸溜所は2016年に北海道厚岸町に設立されました。冷涼で海霧が多く、潮風が吹く環境がスコットランドのアイラ島に類似しているとして選ばれました。泥炭（ピート）も地元で採取可能であり、スコットランドの伝統的なウイスキー造りに近い環境を日本国内で実現することを目指しています。

</QuizAnswer>
