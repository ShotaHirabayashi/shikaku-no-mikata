## 問1

香りの成分をキャッチする嗅細胞が多数存在する「嗅上皮」は、鼻腔のどの位置にありますか。

1. 鼻腔の入口付近
2. 鼻腔の中央部
3. 鼻腔の最上部
4. 鼻腔の下部


<QuizAnswer>

**答え: 3**

嗅上皮は鼻腔の最上部（上鼻道の天井部分）に位置しています。嗅上皮には約500万個の嗅細胞があり、そこから嗅毛（感覚毛）が粘液中に突き出して、香り分子をキャッチする役割を果たしています。

</QuizAnswer>

## 問2

香りの分子が嗅細胞の嗅毛にある受容体に結合すると、どのような変化が起こりますか。

1. 電気信号（インパルス）に変換される
2. 化学物質として直接脳に届けられる
3. 熱エネルギーに変換される
4. 香り分子がそのまま神経線維を移動する


<QuizAnswer>

**答え: 1**

香り分子が嗅毛の受容体に結合すると、嗅細胞のイオンチャネルが開いて脱分極が起こり、電気信号（インパルス）に変換されます。この電気信号が嗅神経を伝わって脳へ送られる仕組みになっています。香り分子そのものが脳に届くわけではありません。

</QuizAnswer>

## 問3

嗅細胞から発生した電気信号が最初に到達する脳の部位はどこですか。

1. 視床下部
2. 大脳皮質
3. 扁桃体
4. 嗅球


<QuizAnswer>

**答え: 4**

嗅細胞で変換された電気信号は、嗅神経を通じてまず「嗅球」に伝わります。嗅球は一次嗅覚中枢とも呼ばれ、香りの情報を中継する最初の脳の部位です。その後、嗅球から大脳辺縁系へと情報が伝達されます。

</QuizAnswer>

## 問4

香りの情報が嗅球を経由して最初に伝わる脳の領域はどこですか。

1. 小脳
2. 大脳辺縁系
3. 脳幹
4. 大脳基底核


<QuizAnswer>

**答え: 2**

嗅球に達した香りの情報は、次に大脳辺縁系へ伝わります。大脳辺縁系は感情や本能、記憶を司る領域で、扁桃体や海馬などを含んでいます。嗅覚は他の感覚と異なり、大脳辺縁系にダイレクトに伝わるのが大きな特徴です。

</QuizAnswer>

## 問5

嗅覚が視覚や聴覚など他の感覚と大きく異なる特徴として、正しいものはどれですか。

1. 電気信号に変換されずに脳に届く
2. 大脳皮質でのみ処理される
3. 視床を経由せず大脳辺縁系に直接伝わる
4. 意識的にしか感じ取ることができない


<QuizAnswer>

**答え: 3**

視覚や聴覚などの感覚情報は視床を経由してから大脳皮質に伝わりますが、嗅覚は視床を経由せず大脳辺縁系に直接伝達されます。このため、香りは感情や記憶と結びつきやすく、本能的な反応を引き起こしやすいという特徴があります。

</QuizAnswer>

## 問6

大脳辺縁系から香りの情報がさらに伝わる「視床下部」は、主にどのような機能を司っていますか。

1. 自律神経系や内分泌系の調節
2. 言語の理解と発話
3. 運動機能の制御
4. 視覚情報の処理


<QuizAnswer>

**答え: 1**

視床下部は自律神経系や内分泌系を支配する重要な中枢です。香りの情報が視床下部に伝わることで、体温調節、睡眠、ホルモン分泌、免疫機能などに影響を与えます。これがアロマテラピーによって心身のバランスが整えられる仕組みの基盤となっています。

</QuizAnswer>

## 問7

大脳辺縁系に含まれる部位のうち、香りをきっかけに過去の記憶が呼び起こされる現象に深く関わっているのはどこですか。

1. 視床下部
2. 下垂体
3. 扁桃体
4. 海馬


<QuizAnswer>

**答え: 4**

海馬は記憶の形成や想起に深く関わる部位です。嗅皮質から海馬に香りの情報が届くことで、過去の記憶が喚起されます。一方、扁桃体は好き嫌いや感情の反応に関わっており、両者は異なる役割を担っています。

</QuizAnswer>

## 問8

精油の成分が皮膚を透過して体内に吸収される理由として、最も適切なものはどれですか。

1. 精油成分は水溶性が高く、汗腺から容易に浸透するため
2. 精油成分は分子量が小さく親油性（脂溶性）であるため
3. 精油成分は皮膚表面で化学変化を起こして浸透するため
4. 精油成分は体温により気化して皮膚の隙間から入るため


<QuizAnswer>

**答え: 2**

精油成分は分子量が小さく、皮膚になじみやすい親油性（脂溶性）の性質を持っているため、皮膚を通り抜けることができます。皮膚を透過した精油成分は毛細血管に入り、血液循環によって全身に運ばれて作用します。

</QuizAnswer>

## 問9

体内環境を一定に保とうとする仕組みを「ホメオスタシス（恒常性）」といいますが、この維持に関わる3つの系の正しい組み合わせはどれですか。

1. 自律神経系・内分泌系・免疫系
2. 自律神経系・循環器系・消化器系
3. 中枢神経系・内分泌系・呼吸器系
4. 運動神経系・免疫系・循環器系


<QuizAnswer>

**答え: 1**

ホメオスタシス（恒常性）の維持には、自律神経系・内分泌系・免疫系の3つの系が互いに連携して働いています。アロマテラピーは嗅覚や皮膚吸収を通じてこれら3つの系に作用し、心身のバランスを整えることでホメオスタシスの維持に寄与すると考えられています。

</QuizAnswer>

## 問10

皮膚から吸収された精油成分が全身に届けられる経路として、正しいものはどれですか。

1. リンパ管のみを通って全身に届く
2. 神経線維を伝って脳に直接届く
3. 毛細血管に入り血液循環によって全身に運ばれる
4. 皮下脂肪に蓄積された後、少しずつ全身に届く


<QuizAnswer>

**答え: 3**

皮膚を透過した精油成分は真皮層の毛細血管に入り、血液循環によって全身の細胞へ運ばれます。経皮吸収された精油成分は約20分で全身に行きわたるとされています。これがアロマテラピートリートメントによって精油の作用が全身に及ぶ仕組みです。

</QuizAnswer>
