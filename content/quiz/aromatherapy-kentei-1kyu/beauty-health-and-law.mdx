## 問1

健康を維持・増進するために重要とされる基本的な3つの要素の組み合わせとして、正しいものはどれか。

1. 栄養・睡眠・入浴
2. 栄養・運動・休養
3. 食事・運動・瞑想
4. 運動・睡眠・リラクゼーション


<QuizAnswer>

**答え: 2**

健康づくりの三要素は「栄養・運動・休養」です。バランスのよい食事（栄養）、適度な身体活動（運動）、十分な休息（休養）の3つを日常生活の中で整えることが、心身の健康維持・増進の基本とされています。アロマテラピーはこれらの要素を補完する形で活用されます。

</QuizAnswer>

## 問2

皮膚の構造について、外側から内側への正しい順序はどれか。

1. 表皮・真皮・皮下組織
2. 真皮・表皮・皮下組織
3. 皮下組織・真皮・表皮
4. 表皮・皮下組織・真皮


<QuizAnswer>

**答え: 1**

皮膚は外側から「表皮」「真皮」「皮下組織」の3層構造になっています。表皮は外部の刺激や細菌から体を守るバリアの役割を果たし、真皮にはコラーゲンやエラスチンなどの線維組織があり肌の弾力を保っています。皮下組織は主に皮下脂肪で構成され、クッションや断熱の役割を担っています。

</QuizAnswer>

## 問3

皮膚の真皮層に多く含まれ、肌の弾力やハリを保つ役割を果たす線維状のタンパク質はどれか。

1. ケラチン
2. メラニン
3. コラーゲン
4. セラミド


<QuizAnswer>

**答え: 3**

真皮の大部分を構成するのはコラーゲン（膠原線維）で、肌の弾力やハリを保つ重要な役割を果たしています。ケラチンは表皮の角質層を構成するタンパク質、メラニンは紫外線から肌を守る色素、セラミドは角質層の細胞間脂質の主成分であり、それぞれ役割が異なります。

</QuizAnswer>

## 問4

薬機法（医薬品医療機器等法）において、精油（エッセンシャルオイル）はどのように位置づけられているか。

1. 医薬品
2. 医薬部外品
3. 化粧品
4. 医薬品・医薬部外品・化粧品のいずれにも該当しない雑貨


<QuizAnswer>

**答え: 4**

精油（エッセンシャルオイル）は、薬機法上「医薬品」「医薬部外品」「化粧品」のいずれにも該当せず、法律上は「雑貨」として扱われます。そのため、精油を販売する際に「○○に効く」「○○が治る」といった医薬品的な効能効果をうたうことは薬機法違反となります。精油の正しい法的位置づけを理解しておくことが大切です。

</QuizAnswer>

## 問5

ストレスの原因となる外部からの刺激のことを何というか。

1. ストレスレスポンス
2. ストレッサー
3. ホメオスタシス
4. アダプテーション


<QuizAnswer>

**答え: 2**

ストレスの原因となる外部からの刺激を「ストレッサー」といいます。ストレッサーには、暑さ・寒さなどの物理的なもの、人間関係や仕事のプレッシャーなどの心理・社会的なものがあります。一方、ストレッサーに対する心身の反応を「ストレス反応（ストレスレスポンス）」と呼びます。

</QuizAnswer>

## 問6

製造物責任法（PL法）に関する記述として、正しいものはどれか。

1. 製造物の欠陥により人の生命・身体・財産に被害が生じた場合、製造業者等が損害賠償責任を負う
2. PL法は日本国内で製造された製品にのみ適用され、輸入品は対象外である
3. 精油はPL法の対象外であるため、製造者に賠償責任は発生しない
4. PL法では、製造者に過失がなければ賠償責任は発生しない


<QuizAnswer>

**答え: 1**

製造物責任法（PL法）は、製造物の欠陥によって人の生命・身体・財産に被害が生じた場合、製造業者等に損害賠償責任を課す法律です。輸入品も対象に含まれ、輸入業者も責任を負います。また、PL法は「無過失責任」の原則に基づいており、製造者の過失の有無にかかわらず、製品の欠陥が証明されれば賠償責任が生じます。精油も製造物として対象となります。

</QuizAnswer>

## 問7

消防法における精油の取り扱いに関する記述として、正しいものはどれか。

1. 精油はすべて消防法上の第1石油類に分類される
2. 精油は水溶性であるため、消防法の規制対象にはならない
3. 精油は引火性があるため、火気の近くでの取り扱いには注意が必要である
4. 精油は引火点が200℃以上であり、常温で引火する危険はない


<QuizAnswer>

**答え: 3**

精油は揮発性の成分を含み、引火性があります。多くの精油の引火点は35℃〜110℃程度であり、火気の近くでの使用や保管には十分な注意が必要です。消防法では精油は第4類引火性液体に該当し、引火点の範囲によって第2石油類や第3石油類に分類されるものが多くあります。アロマポットなどで精油を使う際は、火の取り扱いに注意しましょう。

</QuizAnswer>

## 問8

あん摩マッサージ指圧師、はり師、きゅう師等に関する法律（あはき法）に照らして、正しい記述はどれか。

1. アロマテラピーの民間資格を取得すれば「マッサージ」と称して施術を行える
2. アロマテラピーのトリートメントは医療行為として法的に認められている
3. あはき法はアロマテラピーの活動とは一切関係がない
4. 「マッサージ」を業として行うには、あん摩マッサージ指圧師の国家資格が必要である


<QuizAnswer>

**答え: 4**

あはき法により、「マッサージ」を業として行うにはあん摩マッサージ指圧師の国家資格が必要と定められています。アロマテラピーの民間資格ではマッサージを行うことはできず、そのためアロマテラピーの施術では「マッサージ」ではなく「トリートメント」という用語が使われます。アロマテラピーを実践する際には、あはき法に抵触しないよう注意が必要です。

</QuizAnswer>

## 問9

睡眠に深く関わるホルモンで、夜間に分泌量が増加し、体内時計の調節に関与するものはどれか。

1. アドレナリン
2. メラトニン
3. インスリン
4. コルチゾール


<QuizAnswer>

**答え: 2**

メラトニンは脳の松果体から分泌されるホルモンで、夜間に分泌量が増加し、睡眠を促す作用があります。体内時計（サーカディアンリズム）の調節にも関与しており、規則正しい生活リズムの維持に重要な役割を果たしています。朝の光を浴びることでメラトニンの分泌リズムが整い、夜の自然な眠気につながります。

</QuizAnswer>

## 問10

医師法に照らして、アロマテラピーを実践する際に注意すべきこととして、正しい記述はどれか。

1. アロマテラピーの専門家であれば、精油の効能に基づいて病名を診断できる
2. 精油を用いた治療は代替医療として法律上認められている
3. 医師でない者が病名を診断したり、医学的な治療行為を行うことは医師法に抵触する
4. アロマテラピーは自然療法であるため、医師法の規制を受けない


<QuizAnswer>

**答え: 3**

医師法第17条により、医師でない者が医業（診断・治療など）を行うことは禁止されています。アロマテラピーの実践者が「この精油で○○が治ります」と言ったり、病名の診断を行ったりすることは医師法に抵触するおそれがあります。アロマテラピーはあくまでリラクゼーションや心身の健康維持を目的とし、医療行為とは明確に区別して行う必要があります。

</QuizAnswer>
