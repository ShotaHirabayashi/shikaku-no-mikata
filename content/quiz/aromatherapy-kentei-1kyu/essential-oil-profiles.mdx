## 問1

次のうち、シソ科に属する精油はどれか。

1. ゼラニウム
2. ジャスミン（アブソリュート）
3. クラリセージ
4. イランイラン


<QuizAnswer>

**答え: 3**

クラリセージはシソ科に属します。シソ科にはラベンダー、クラリセージ、ペパーミント、スイートマージョラム、パチュリ、メリッサ、ローズマリーの7種が含まれ、検定で最も多い科名です。ゼラニウムはフウロソウ科、ジャスミンはモクセイ科、イランイランはバンレイシ科です。

</QuizAnswer>

## 問2

ネロリの精油は、ビターオレンジのどの部位から抽出されるか。

1. 果皮
2. 花
3. 葉
4. 果実


<QuizAnswer>

**答え: 2**

ネロリはミカン科ですが、ビターオレンジ（ダイダイ）の「花」から水蒸気蒸留法で抽出されます。同じミカン科のスイートオレンジ・グレープフルーツ・レモン・ベルガモットが「果皮」から圧搾法で抽出されるのとは異なる点が、検定でよく問われるポイントです。

</QuizAnswer>

## 問3

次のうち、揮発性有機溶剤抽出法で得られる精油はどれか。

1. ローズオットー
2. フランキンセンス
3. ミルラ
4. ベンゾイン（レジノイド）


<QuizAnswer>

**答え: 4**

ベンゾイン（レジノイド）はエゴノキ科の樹脂から揮発性有機溶剤抽出法で得られます。検定対象30種のうち揮発性有機溶剤抽出法で抽出されるのは、ジャスミン（アブソリュート）、ローズ（アブソリュート）、ベンゾイン（レジノイド）の3種のみです。ローズオットーは水蒸気蒸留法で抽出される点に注意しましょう。

</QuizAnswer>

## 問4

ベチバーの精油は、植物のどの部位から抽出されるか。

1. 根
2. 葉
3. 心材
4. 樹脂


<QuizAnswer>

**答え: 1**

ベチバーはイネ科の植物で、乾燥させた「根」から水蒸気蒸留法で精油を抽出します。検定対象30種の中で抽出部位が「根」なのはベチバーだけであり、非常に出題されやすいポイントです。深く落ち着いた土のような香りが特徴で、ベースノートに分類されます。

</QuizAnswer>

## 問5

フランキンセンスと同じカンラン科に属する精油はどれか。

1. ベンゾイン
2. サンダルウッド
3. ミルラ
4. パチュリ


<QuizAnswer>

**答え: 3**

ミルラ（没薬）はフランキンセンス（乳香）と同じカンラン科に属し、どちらも樹脂から水蒸気蒸留法で抽出されます。古代から宗教儀式や薫香に使われてきた歴史を持つ点も共通しています。ベンゾインはエゴノキ科、サンダルウッドはビャクダン科、パチュリはシソ科です。

</QuizAnswer>

## 問6

サンダルウッド（白檀）の精油の抽出部位として正しいものはどれか。

1. 樹皮
2. 心材
3. 樹脂
4. 葉


<QuizAnswer>

**答え: 2**

サンダルウッドはビャクダン科に属し、木の幹の中心部分である「心材」から水蒸気蒸留法で精油を抽出します。検定対象30種の中で抽出部位が「心材」なのはサンダルウッドだけです。甘く深いウッディな香りが特徴で、ベースノートに分類されます。

</QuizAnswer>

## 問7

バンレイシ科に属する精油として正しいものはどれか。

1. イランイラン
2. ネロリ
3. ジャスミン（アブソリュート）
4. ローマンカモミール


<QuizAnswer>

**答え: 1**

イランイランはバンレイシ科に属し、花から水蒸気蒸留法で抽出されます。検定対象30種の中でバンレイシ科はイランイランのみです。「花の中の花」を意味する名前を持ち、濃厚で甘いエキゾチックな花の香りが特徴です。ネロリはミカン科、ジャスミンはモクセイ科、ローマンカモミールはキク科に属します。

</QuizAnswer>

## 問8

次のうち、圧搾法で精油が抽出されないものはどれか。

1. スイートオレンジ
2. グレープフルーツ
3. レモン
4. ネロリ


<QuizAnswer>

**答え: 4**

ネロリはミカン科ですが、ビターオレンジの花から水蒸気蒸留法で抽出されるため圧搾法ではありません。圧搾法で抽出される精油はスイートオレンジ、グレープフルーツ、レモン、ベルガモットの4種で、いずれもミカン科の果皮から抽出されます。ミカン科でも抽出部位と抽出法が異なるネロリは頻出問題です。

</QuizAnswer>

## 問9

メリッサの精油の別名として正しいものはどれか。

1. マジョラム
2. レモンバーム
3. オリバナム
4. 白檀


<QuizAnswer>

**答え: 2**

メリッサはシソ科に属し、別名「レモンバーム」とも呼ばれます。和名はセイヨウヤマハッカで、葉にレモンに似た爽やかな香りを持つことからこの別名がつきました。オリバナム（乳香）はフランキンセンスの別名、白檀はサンダルウッドの和名です。

</QuizAnswer>

## 問10

ジャーマンカモミールの精油に含まれる、精油を青色にする成分は何か。

1. リナロール
2. リモネン
3. カマズレン
4. ゲラニオール


<QuizAnswer>

**答え: 3**

ジャーマンカモミールの精油には「カマズレン」という成分が含まれており、精油が濃い青色（アズレンブルー）を呈するのが大きな特徴です。カマズレンは原料の花には含まれず、水蒸気蒸留の過程で生成される成分です。同じキク科のローマンカモミールにはこの青色は見られません。

</QuizAnswer>
