## 問1

基材30mlに対して1％濃度のトリートメントオイルを作る場合、精油は何滴必要か。

1. 3滴
2. 6滴
3. 10滴
4. 15滴


<QuizAnswer>

**答え: 2**

精油1滴は0.05mlです。1％濃度にするには、基材5mlあたり精油1滴が目安となります。したがって、30ml÷5＝6滴が正解です。計算式で確認すると、0.05ml×6滴＝0.3ml、0.3ml÷30ml×100＝1％となります。

</QuizAnswer>

## 問2

精油のドロッパー（精油瓶の中栓）から落ちる1滴の量として正しいものはどれか。

1. 0.01ml
2. 0.1ml
3. 0.05ml
4. 0.5ml


<QuizAnswer>

**答え: 3**

AEAJの基準では、精油1滴の量は約0.05mlと定められています。この数値は希釈濃度の計算の基礎となるため、必ず覚えておく必要があります。たとえば、基材5mlに精油1滴（0.05ml）を加えると、0.05÷5×100＝1％濃度となります。

</QuizAnswer>

## 問3

沐浴法における全身浴で使用する精油の滴数として適切なものはどれか。

1. 1〜5滴
2. 6〜10滴
3. 10〜15滴
4. 15〜20滴


<QuizAnswer>

**答え: 1**

全身浴で使用する精油の滴数は1〜5滴が適切です。なお、半身浴・手浴・足浴では1〜3滴が目安です。精油は水に溶けにくいため、無水エタノールや天然塩などに混ぜてから湯に入れ、よくかき混ぜてから入浴することが大切です。

</QuizAnswer>

## 問4

トリートメント法でフェイス（顔）に使用する場合の精油の希釈濃度の上限として適切なものはどれか。

1. 1％以下
2. 2％以下
3. 3％以下
4. 0.5％以下


<QuizAnswer>

**答え: 4**

フェイス（顔）へのトリートメントでは、精油の希釈濃度は0.5％以下とされています。顔の皮膚はボディに比べて薄く敏感なため、ボディ用の1％以下よりもさらに低い濃度が推奨されます。初めて使用する場合は、さらに低い濃度から始めるとよいでしょう。

</QuizAnswer>

## 問5

光毒性に注意が必要な精油の組み合わせとして正しいものはどれか。

1. ラベンダー、ティートリー、ペパーミント
2. イランイラン、ローズ、ジャスミン
3. ベルガモット、グレープフルーツ、レモン
4. ユーカリ、ローズマリー、ゼラニウム


<QuizAnswer>

**答え: 3**

光毒性とは、精油の成分が紫外線に反応して皮膚に炎症やシミなどを引き起こす性質のことです。主に柑橘系の精油に含まれるフロクマリン類が原因で、ベルガモット、グレープフルーツ、レモンなどが代表的です。これらの精油を肌に使用した後は、直射日光（紫外線）を避ける必要があります。

</QuizAnswer>

## 問6

3歳未満の乳幼児に対して行ってもよいとされるアロマテラピーの方法はどれか。

1. 芳香浴法
2. トリートメント法
3. 沐浴法
4. 吸入法


<QuizAnswer>

**答え: 1**

AEAJのガイドラインでは、3歳未満の乳児・幼児には芳香浴法以外のアロマテラピーは行わないこととされています。乳幼児は大人に比べて抵抗力が弱く、精油の影響を受けやすいためです。芳香浴法であっても、精油の量は少なめにし、長時間の使用は避けましょう。

</QuizAnswer>

## 問7

アロマテラピーで精油を希釈するために使用する植物油の別名として正しいものはどれか。

1. エッセンシャルオイル
2. キャリアオイル
3. ミネラルオイル
4. フレグランスオイル


<QuizAnswer>

**答え: 2**

精油を希釈するために使う植物油は「キャリアオイル」または「ベースオイル」と呼ばれます。「キャリア（carrier）」は「運ぶもの」という意味で、精油の成分を皮膚に届ける役割を果たすことに由来します。代表的なものにスイートアーモンドオイル、ホホバオイル、マカデミアナッツオイルなどがあります。

</QuizAnswer>

## 問8

精油を肌に使用する前に、アレルギー反応の有無を確認するために行うテストは何か。

1. リトマステスト
2. スクラッチテスト
3. プリックテスト
4. パッチテスト


<QuizAnswer>

**答え: 4**

パッチテストとは、精油を使用する前に少量を皮膚（前腕部の内側など）に塗布し、24〜48時間かけてかゆみ・赤み・腫れなどの異常反応が出ないか確認するテストのことです。皮膚の弱い方や初めて使う精油がある場合には、トラブルを防ぐために必ず行うことが推奨されています。

</QuizAnswer>

## 問9

吸入法を行う際の注意点として正しいものはどれか。

1. 精油を10滴以上使用する
2. 目を開けたまま蒸気を吸い込む
3. 目を閉じて蒸気を吸い込む
4. せき込んでいるときに積極的に行う


<QuizAnswer>

**答え: 3**

吸入法では、お湯に精油を1〜3滴落とし、立ち上がる蒸気を吸い込みますが、精油成分が目の粘膜を刺激する恐れがあるため、必ず目を閉じて行います。また、長時間行わないよう注意が必要です。せきが出ているときやぜんそくの方は、吸入法によって症状が悪化する場合があるため避けましょう。

</QuizAnswer>

## 問10

妊娠中のアロマテラピーの楽しみ方について、最も適切なものはどれか。

1. 芳香浴法以外のアロマテラピーを楽しむ場合は十分注意し、トリートメントを受ける際は医師や専門家に相談する
2. 妊娠中はすべてのアロマテラピーを自由に楽しんでよい
3. 妊娠中はいかなるアロマテラピーも一切行ってはならない
4. 沐浴法のみ通常通り行ってよい


<QuizAnswer>

**答え: 1**

AEAJのガイドラインでは、妊娠中は体調の変化に十分配慮し、芳香浴法以外のアロマテラピーを楽しむ場合は注意が必要とされています。特にトリートメントを受ける場合は、医師や経験を積んだ専門家に相談することが推奨されます。すべてを禁止するわけではなく、安全に配慮しながら楽しむことが大切です。

</QuizAnswer>
