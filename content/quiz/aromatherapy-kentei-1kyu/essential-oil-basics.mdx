## 問1

精油（エッセンシャルオイル）の説明として最も適切なものはどれか。

1. 植物から抽出された脂肪油のことである
2. 化学的に合成された芳香物質の総称である
3. 植物から抽出された天然の揮発性芳香物質である
4. 植物の種子から絞った植物油のことである


<QuizAnswer>

**答え: 3**

精油（エッセンシャルオイル）は、植物の花・葉・果皮・樹脂などから抽出された天然の揮発性芳香物質です。脂肪油や植物油（オリーブ油など）とは異なり、揮発性があるため空気中に蒸発します。また、化学合成された香料は精油には含まれません。

</QuizAnswer>

## 問2

精油が持つ特徴的な性質として正しいものはどれか。

1. 空気中で蒸発する揮発性がある
2. 水に溶けやすい水溶性がある
3. 常温では固体の状態である
4. 引火しにくく安全性が高い


<QuizAnswer>

**答え: 1**

精油は常温で液体から気体へと蒸発する「揮発性」を持っています。また、精油は水には溶けにくい「脂溶性（親油性）」の性質があり、水溶性ではありません。さらに精油は「可燃性」が高く引火しやすいため、火気の近くでの取り扱いには注意が必要です。

</QuizAnswer>

## 問3

精油を水に入れた場合の説明として正しいものはどれか。

1. 水に完全に溶ける
2. 水と均一に混ざり合う
3. ほとんどの精油は水より重いため沈む
4. ほとんどの精油は水に溶けず水面に浮く


<QuizAnswer>

**答え: 4**

精油は脂溶性（親油性）の物質であり、水には溶けません。また、ほとんどの精油は水よりも比重が軽いため、水に入れると表面に浮きます。このため、精油を水に混ぜて使用する際には、無水エタノールなどを用いて分散させる工夫が必要です。

</QuizAnswer>

## 問4

精油の保管容器として最も適切なものはどれか。

1. 透明なガラス瓶
2. 遮光性のガラス瓶（褐色・青色など）
3. ポリエチレン製のプラスチック容器
4. 蓋のない陶器の容器


<QuizAnswer>

**答え: 2**

精油は光（紫外線）によって成分が変化し劣化するため、遮光性のあるガラス瓶（褐色や青色）で保管する必要があります。透明なガラス瓶では光を遮ることができません。またプラスチック容器は精油の成分によって溶解・変質する恐れがあるため不適切です。

</QuizAnswer>

## 問5

開封後の精油の使用期限の目安として正しいものはどれか。

1. 柑橘系は約半年、その他の精油は約1年
2. すべての精油は開封後6か月以内
3. 柑橘系は約1年、その他の精油は約2年
4. すべての精油は開封後2年以内


<QuizAnswer>

**答え: 1**

開封後の精油は空気に触れることで酸化が進むため、使用期限の目安があります。柑橘系の精油（オレンジ・レモンなど）は酸化しやすいため開封後約半年、その他の精油は約1年が目安です。高温多湿を避け、冷暗所で密閉して保管することで品質を保つことができます。

</QuizAnswer>

## 問6

AEAJ（日本アロマ環境協会）の説明として正しいものはどれか。

1. 厚生労働省が管轄する国の行政機関である
2. 精油の製造・販売を主な業務とする企業である
3. 医療従事者のみが加入できる専門団体である
4. 公益社団法人としてアロマテラピーの普及・調査・研究を行う団体である


<QuizAnswer>

**答え: 4**

AEAJは「公益社団法人 日本アロマ環境協会」の略称で、アロマテラピーの普及・調査・研究を行う日本最大のアロマテラピー関連団体です。アロマテラピー検定をはじめとする各種資格認定や、安全なアロマテラピーを実践できる人材の育成、自然環境の保全・創造に向けた活動を行っています。国の行政機関や企業ではなく、公益社団法人です。

</QuizAnswer>

## 問7

水蒸気蒸留法で精油を抽出する際に、精油とともに得られる副産物は何か。

1. アブソリュート
2. フローラルウォーター（芳香蒸留水）
3. コンクリート
4. レジノイド


<QuizAnswer>

**答え: 2**

水蒸気蒸留法では、植物に水蒸気を当てて芳香成分を気化させ、冷却して液体に戻します。この液体は精油と水分に分離し、水分の部分が「フローラルウォーター（芳香蒸留水）」として得られます。ローズウォーターやラベンダーウォーターなどが代表的です。なお、コンクリートやアブソリュートは揮発性有機溶剤抽出法に関連する用語です。

</QuizAnswer>

## 問8

植物が精油（芳香物質）を体内で作り出す目的として適切でないものはどれか。

1. 昆虫を引き寄せて受粉を助ける誘因効果
2. 害虫やカビなどの有害な菌から身を守る忌避効果
3. 人間のアロマテラピーに利用されるため
4. 蒸散作用により自らを冷却して太陽熱から守る効果


<QuizAnswer>

**答え: 3**

植物が精油を作り出すのは、あくまで植物自身の生存のためです。昆虫を引き寄せる誘因効果、害虫やカビを遠ざける忌避効果、蒸散による冷却効果などが植物にとっての精油の役割です。「人間のアロマテラピーに利用されるため」というのは植物の目的ではなく、人間が植物の芳香物質を活用しているにすぎません。

</QuizAnswer>

## 問9

精油の化学的な特徴として正しいものはどれか。

1. 単一の化学成分で構成されている
2. 数十〜数百種類の有機化合物の混合物である
3. 主に無機化合物から構成されている
4. すべての精油が同一の化学組成を持っている


<QuizAnswer>

**答え: 2**

精油は単一の物質ではなく、数十〜数百種類もの有機化合物が混ざり合った複雑な混合物です。これらの有機化合物の組み合わせや比率が精油ごとに異なるため、それぞれの精油が固有の香りや特性を持ちます。精油は炭素・水素・酸素などからなる有機化合物であり、無機化合物ではありません。

</QuizAnswer>

## 問10

AEAJが定義する「アロマテラピー」の説明として最も適切なものはどれか。

1. 化学的に合成した香料を使ったリラクゼーション法のことである
2. 精油を内服して病気を治療する医療行為のことである
3. 花や植物の観賞により心身を癒す園芸療法のことである
4. 植物から抽出した精油を用いて美と健康に役立てていく自然療法のことである


<QuizAnswer>

**答え: 4**

AEAJはアロマテラピーを「植物から抽出した香り成分である精油（エッセンシャルオイル）を用いて、美と健康に役立てていく自然療法」と定義しています。化学合成された香料を使うものではなく、また日本では医療行為としては位置づけられていません。園芸療法とも異なり、精油の香りを活用する点が特徴です。

</QuizAnswer>
