## 問1

薬膳において、食材が身体に与える温冷の性質を5段階に分類したものを何というか。

1. 五味
2. 五行
3. 五性
4. 五色


<QuizAnswer>

**答え: 3**

食材の温冷の性質を「寒・涼・平・温・熱」の5段階に分類したものを「五性（ごせい）」といいます。五味は味の分類、五行は木火土金水の自然哲学、五色は食材の色による分類であり、それぞれ異なる概念です。五性は薬膳で食材を選ぶ際の最も基本的な指標の一つです。

</QuizAnswer>

## 問2

五性の分類において、トウガラシやコショウなど身体を強く温める性質をもつ食材はどれに分類されるか。

1. 温性
2. 平性
3. 涼性
4. 熱性


<QuizAnswer>

**答え: 4**

身体を強く温める性質をもつ食材は「熱性」に分類されます。トウガラシ、コショウ、山椒などが代表的な熱性食材です。「温性」も身体を温めますが、熱性ほど強くはありません。熱性と温性をまとめて「温熱性」と呼ぶこともあります。

</QuizAnswer>

## 問3

五性のうち「平性」の食材の特徴として正しいものはどれか。

1. 身体を強く冷やす作用がある
2. 温めも冷やしもせず、どの体質にも合わせやすい
3. 夏にのみ食べることが推奨される
4. 食材全体の約1割程度しか存在しない


<QuizAnswer>

**答え: 2**

平性の食材は身体を温めも冷やしもしない穏やかな性質をもち、どの体質・体調にも合わせやすいのが特徴です。米、キャベツ、大豆、豚肉などが代表例で、食材全体の5〜7割は平性に分類されるとされています。季節を問わず日常的に取り入れやすい食材です。

</QuizAnswer>

## 問4

薬膳における六味のうち、「収斂（しゅうれん）作用」があり、多汗や慢性的な下痢・咳を収める働きをもつ味はどれか。

1. 辛味
2. 甘味
3. 酸味
4. 鹹味


<QuizAnswer>

**答え: 3**

酸味には「収斂作用」があり、体内から過剰に漏れ出るものを引き締めて収める働きがあります。多汗、慢性の咳、下痢、頻尿などの症状に用いられます。代表的な酸味の食材には梅、レモン、酢、ザクロなどがあります。

</QuizAnswer>

## 問5

六味において「淡味（たんみ）」がもつ主な作用として正しいものはどれか。

1. 堅いものをやわらげ散らす
2. 体内の水分代謝を促進する
3. 気を巡らせ血行を促進する
4. 虚弱を補い痛みを緩和する


<QuizAnswer>

**答え: 2**

淡味は体内の水の代謝を良くする「滲湿（しんしつ）・利尿」の働きがあります。むくみ、下痢、めまいなどの水分代謝の異常に用いられます。選択肢1は鹹味、選択肢3は辛味、選択肢4は甘味の作用です。淡味は五味（酸・苦・甘・辛・鹹）に加えて六味として数えられます。

</QuizAnswer>

## 問6

薬膳における「帰経（きけい）」の説明として最も適切なものはどれか。

1. 食材の味を6つに分類する体系である
2. 食材が体内のどの臓腑・経絡に作用するかを示すものである
3. 食材の温冷の度合いを5段階で示すものである
4. 食材を収穫する季節を表す分類である


<QuizAnswer>

**答え: 2**

帰経とは、食薬がどの臓腑や経絡に対して作用するかを示す概念です。例えば「肺経に帰経する」食材は肺に対して効果が期待できることを意味します。食材の色や性味によって入る臓腑が異なるとされ、薬膳の食材選びにおいて五性・六味と並ぶ重要な分類基準です。

</QuizAnswer>

## 問7

薬膳における「昇降浮沈」の考え方で、「昇・浮」の傾向をもつ食材の特徴として正しいものはどれか。

1. 酸味・苦味で寒涼性のもの
2. 根や実など重い部位のもの
3. 辛味・甘味で温熱性のもの
4. 鹹味で沈降作用のあるもの


<QuizAnswer>

**答え: 3**

「昇・浮」は上昇・発散の性質を表し、辛味や甘味をもち、温熱性の食材が該当します。また花や葉のような軽い部位も昇・浮の傾向があります。一方「降・沈」は下降・泄利の性質で、酸味・苦味・鹹味をもち、寒涼性の食材、茎・根・実・貝類など重いものが該当します。

</QuizAnswer>

## 問8

「薬食同源」の考え方に基づく「食薬」と「生薬（中薬）」の違いについて、正しい説明はどれか。

1. 食薬は化学合成品であり、生薬は天然素材である
2. 食薬は味が良く日常的に食べられるもの、生薬は薬効が強いが味が良くないもの
3. 食薬は西洋医学の概念であり、生薬は中医学の概念である
4. 食薬と生薬に区別はなく、まったく同じものを指す


<QuizAnswer>

**答え: 2**

薬食同源の考え方では、もともと薬と食べ物に区別はありませんでした。長い歴史の中で、美味しく日常的に食べられるものが「食材（食薬）」として、味はよくないが薬効の強いものが「生薬（中薬）」として治療に使われるようになりました。クコの実や生姜など、食薬と生薬の両方の側面をもつものもあります。

</QuizAnswer>

## 問9

六味のうち「苦味」がもつ作用の組み合わせとして正しいものはどれか。

1. 発散・行気・活血
2. 清熱・燥湿・瀉下
3. 補益・緩急・調和
4. 軟堅・散結・潤下


<QuizAnswer>

**答え: 2**

苦味には「清熱（体内の余分な熱を冷ます）」「燥湿（体内の余分な湿を乾かす）」「瀉下（不要なものを排泄する）」の作用があります。選択肢1は辛味、選択肢3は甘味、選択肢4は鹹味の主な作用です。苦味の代表的な食材にはゴーヤ、セロリ、緑茶などがあります。

</QuizAnswer>

## 問10

薬膳において食材の食べ合わせを考える際、寒性の食材を使う料理に温性の薬味を加える目的として最も適切なものはどれか。

1. 料理の見た目を美しくするため
2. 食材の寒性を和らげ、胃腸への負担を軽減するため
3. 食材の栄養素を化学的に変化させるため
4. 調理時間を短縮するため


<QuizAnswer>

**答え: 2**

薬膳では食材の性質（五性）のバランスを考えた食べ合わせが重要です。例えば、寒性の刺身に温性の生姜やわさびを添えるのは、寒性を和らげて胃腸を冷やしすぎないようにする知恵です。このように五性のバランスを取ることで、食材の良い効能を活かしつつ身体への負担を軽減することができます。

</QuizAnswer>
