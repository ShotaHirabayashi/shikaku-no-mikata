## 問1

春の薬膳では、五臓のうちどの臓器を特に養うことが重要とされているか。

1. 脾
2. 肺
3. 肝
4. 腎


<QuizAnswer>

**答え: 3**

中医学では春は「木」の季節であり、五臓の中で肝と深く関係しています。春は肝の働きが活発になる時期で、肝の気が高ぶりやすく精神状態が不安定になりやすいため、肝を養い気の巡りを整える薬膳が重要です。セロリや菊花、三つ葉など香りのある食材が春の薬膳によく使われます。

</QuizAnswer>

## 問2

夏の薬膳で、体にこもった熱を冷まし暑さ対策として適切な食材の組み合わせはどれか。

1. くるみ・羊肉・シナモン
2. 苦瓜（ゴーヤ）・緑豆・スイカ
3. 黒豆・黒ごま・山芋
4. 白きくらげ・梨・百合根


<QuizAnswer>

**答え: 2**

夏は心を養う季節であり、体に熱がこもりやすいため、清熱作用のある食材が適しています。苦瓜（ゴーヤ）は苦味で心の熱を冷まし、緑豆は清熱解毒、スイカは体の余分な熱を取り水分を補う働きがあります。選択肢1は冬向け（温める食材）、3は腎を補う冬の食材、4は秋の肺を潤す食材です。

</QuizAnswer>

## 問3

秋の薬膳において、乾燥から肺を守るために積極的に取り入れるべき食材はどれか。

1. セロリ
2. 白きくらげ
3. 苦瓜
4. 生姜


<QuizAnswer>

**答え: 2**

秋は空気が乾燥し、五臓の中で肺がダメージを受けやすい季節です。白きくらげ（銀耳）は潤肺の代表的な食材で、肺を潤し乾燥による咳や肌荒れを防ぐ働きがあります。同様に梨や百合根、はちみつなども秋の薬膳でよく使われる潤いを補う食材です。

</QuizAnswer>

## 問4

冬の薬膳では腎を補うことが重要とされるが、腎を養う食材として最も適切なものはどれか。

1. 緑豆
2. きゅうり
3. くるみ
4. セロリ


<QuizAnswer>

**答え: 3**

冬は「水」の季節で腎と深く関係しており、腎を補い体を温める薬膳が大切です。くるみは補腎の代表的な食材で、腎を温め腰や膝のだるさを改善する働きがあります。黒豆・黒ごま・エビなどの黒い食材や温性の食材も冬の薬膳に適しています。緑豆やきゅうりは体を冷やす食材のため冬には不向きです。

</QuizAnswer>

## 問5

冷え性改善の薬膳メニューを組み立てる際、避けるべき食材はどれか。

1. 生姜
2. 長ネギ
3. きゅうり
4. シナモン


<QuizAnswer>

**答え: 3**

きゅうりは寒涼性の食材で、体の熱を冷ます作用があるため、冷え性の人が多く摂ると症状を悪化させる可能性があります。一方、生姜やシナモンは温熱性で体を芯から温め、長ネギも体を温めて血行を促進する働きがあり、いずれも冷え性改善の薬膳に適した食材です。

</QuizAnswer>

## 問6

むくみ対策の薬膳として、体内の余分な水分を排出する働きが期待できる食材はどれか。

1. 羊肉
2. 冬瓜
3. 黒ごま
4. なつめ


<QuizAnswer>

**答え: 2**

冬瓜（とうがん）は利水作用に優れた食材で、体内の余分な水分を排出しむくみを改善する代表的な薬膳食材です。カリウムも豊富で利尿を促します。同じくはと麦やとうもろこしのひげなども利水の働きがあり、むくみ対策の薬膳によく用いられます。

</QuizAnswer>

## 問7

不眠やストレスに悩む人の薬膳において、心を落ち着かせる「安神」の働きがある食材の組み合わせとして正しいものはどれか。

1. 蓮の実・百合根・なつめ
2. 唐辛子・山椒・にんにく
3. 苦瓜・セロリ・緑豆
4. 冬瓜・はと麦・とうもろこし


<QuizAnswer>

**答え: 1**

蓮の実（蓮子）は養心安神の代表的な食材で、心を鎮めて不眠を改善します。百合根は潤肺安神の作用があり、不安やイライラを和らげます。なつめは補血安神の働きがあり、精神を安定させ不眠の改善に役立ちます。この3つの組み合わせは不眠・ストレス対策の薬膳の定番です。

</QuizAnswer>

## 問8

美肌・アンチエイジングの薬膳として、肌にうるおいを与え老化防止が期待される「白きくらげとなつめのスープ」に加える食材として最もふさわしいものはどれか。

1. 唐辛子
2. クコの実
3. 大根
4. こんにゃく


<QuizAnswer>

**答え: 2**

クコの実（枸杞子）は養肝明目・補腎潤肺の効能があり、アンチエイジング食材として古くから珍重されてきました。白きくらげの潤い効果、なつめの補血効果にクコの実を加えることで、美肌と抗老化の相乗効果が期待できます。中国の伝統的な美容薬膳スープの代表的な組み合わせです。

</QuizAnswer>

## 問9

便秘対策の薬膳で、腸を潤して通じを良くする働きがある食材はどれか。

1. 生姜
2. 松の実
3. 山椒
4. 唐辛子


<QuizAnswer>

**答え: 2**

松の実は潤腸通便の効能があり、腸を油分で潤して便通を改善する代表的な薬膳食材です。特に高齢者や体力が落ちた人の乾燥性便秘に適しています。同様に黒ごまやくるみ、はちみつなども腸を潤す働きがあります。生姜や山椒、唐辛子は温める作用が主で、便秘改善を目的とした潤腸食材とは異なります。

</QuizAnswer>

## 問10

夏の薬膳において、暑さで食欲が落ちたときに取り入れるとよいとされる味覚はどれか。

1. 苦味
2. 辛味
3. 鹹味（しおからい味）
4. 酸味


<QuizAnswer>

**答え: 1**

中医学では苦味は心に入り、体の余分な熱を冷ます「清熱」の作用があるとされています。夏は暑さで心に負担がかかりやすく、苦瓜（ゴーヤ）や緑茶などの苦味のある食材を取り入れることで、体の熱を冷まし食欲を回復させる効果が期待できます。「夏には苦味」は薬膳の季節養生の基本的な考え方です。

</QuizAnswer>
