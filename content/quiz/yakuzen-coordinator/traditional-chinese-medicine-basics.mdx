## 問1

中医学における「陰陽学説」の基本的な考え方として、最も適切なものはどれか。

1. 陰と陽は完全に独立しており、互いに影響を与えない
2. すべての事物は陰と陽という相反する性質を持ち、相互に依存・対立している
3. 陰は良いもの、陽は悪いものを表す
4. 陰陽は人体にのみ適用される概念である

<QuizAnswer>

**答え: 2**

陰陽学説は古代中国の自然哲学に基づき、宇宙のすべての事物が陰と陽という相反する性質を持ち、互いに対立しながらも依存し合っていると考えます。陰と陽は善悪の区別ではなく、どちらも必要不可欠な存在です。また、人体だけでなく自然界のあらゆる現象に適用される普遍的な概念です。

</QuizAnswer>

## 問2

五行学説において、五行の正しい組み合わせはどれか。

1. 木・火・土・金・水
2. 木・火・風・金・水
3. 木・火・土・石・水
4. 木・光・土・金・水

<QuizAnswer>

**答え: 1**

五行学説では、万物は「木（もく）・火（か）・土（ど）・金（きん）・水（すい）」の5つの要素の運動と変化によって成り立つと考えます。これは古代中国の自然哲学であり、自然界や人体のさまざまな現象を5つに分類・整理し、それぞれの関係性を説明する理論です。

</QuizAnswer>

## 問3

五行の「相生（そうせい）」関係の説明として正しいものはどれか。

1. ある一行が別の一行を抑制・制約する関係
2. ある一行が別の一行を資生・促進・助長する関係
3. 五行が互いに打ち消し合う関係
4. 五行がすべて均等に作用する関係

<QuizAnswer>

**答え: 2**

相生とは、五行のある一行が別のある一行に対して、生み出し（資生）、促進し、助長する働きをする関係のことです。具体的には「木は火を生じ、火は土を生じ、土は金を生じ、金は水を生じ、水は木を生ず」という循環的な関係があります。一方、抑制・制約する関係は「相克」と呼ばれます。

</QuizAnswer>

## 問4

五行の「相克（そうこく）」関係において、「木」が克する（抑制する）対象はどれか。

1. 火
2. 金
3. 水
4. 土

<QuizAnswer>

**答え: 4**

五行の相克関係は「木は土を克し、土は水を克し、水は火を克し、火は金を克し、金は木を克す」という順序です。木が土を克する（木剋土）とは、木が根を張って土を押さえるというイメージで理解できます。相克は一方が他方を抑制・制約する関係であり、五行のバランスを保つために必要な仕組みです。

</QuizAnswer>

## 問5

中医学における「整体観念（せいたいかんねん）」の説明として最も適切なものはどれか。

1. 病気の原因を一つの臓器に限定して考える
2. 西洋医学の解剖学に基づいて人体を理解する考え方
3. 人体の各部分や人と自然環境は密接に関連し合い、切り離しては考えられないとする考え方
4. 体の外側だけを観察して健康状態を判断する方法

<QuizAnswer>

**答え: 3**

整体観念とは、人体を含む宇宙の万物はそれぞれが独立して存在するのではなく、密接に関連し協調し合う関係で成り立っているという中医学の基本的な考え方です。人体の内部の各臓器同士の関連だけでなく、人体と自然環境（季節・気候など）との関係も重視します。これは西洋医学の局所的な見方とは大きく異なる特徴です。

</QuizAnswer>

## 問6

中医学の診断・治療の基本原則である「弁証論治（べんしょうろんち）」の説明として正しいものはどれか。

1. 同じ病名であれば全員に同じ治療を行う方法
2. 症状のみに着目して即座に投薬する方法
3. 疾病の原因・性質・部位を総合的に分析し、それに基づいて治療方針を決定する方法
4. 西洋医学の検査データのみに基づいて治療を行う方法

<QuizAnswer>

**答え: 3**

弁証論治は中医学の診断と治療の全過程を指し、「弁証」（証を見極める診断）と「論治」（治療方針の決定）の二つの部分から構成されます。疾病の原因、性質、部位を総合的に分類・分析し、病気の本質を見極めた上で治療方針を決めます。そのため、同じ病名でも患者の体質や状態によって異なる治療が行われることがあります。

</QuizAnswer>

## 問7

薬膳の定義として最も適切なものはどれか。

1. 漢方薬を食事に混ぜて食べる料理法
2. 中医学の基本理論に基づき、食材の持つ効能を活かして健康の維持・増進を図る食事法
3. カロリー計算に基づいた西洋栄養学の食事療法
4. 特定の食材だけを食べ続ける健康法

<QuizAnswer>

**答え: 2**

薬膳とは、中国伝統医学（中医学）の基本理論に基づいて、食材が持つ本来の性質や効能を活かし、体のバランスを整えて健康の維持・増進を図る食事法です。単に漢方薬を食事に入れるということではなく、「食薬同源（しょくやくどうげん）」の考え方に立ち、日常の食材も含めて体質や季節に合わせた食事を組み立てます。

</QuizAnswer>

## 問8

中医学と西洋医学の違いとして最も適切なものはどれか。

1. 中医学は患部だけを診断し、西洋医学は体全体を診る
2. 中医学は体全体のバランスや体質を重視し、西洋医学は主に患部の病変に着目する
3. 中医学には予防の考え方がなく、病気になってから治療する
4. 西洋医学は自然治癒力を重視し、中医学は手術による治療を重視する

<QuizAnswer>

**答え: 2**

中医学は人体を一つの有機的な全体として捉え、体全体のバランスや患者の体質を重視して治療方針を立てます。自然治癒力を高めることを大切にし、「未病（みびょう）」の段階での予防も重視します。一方、西洋医学は主に病気になった患部を診断し、薬や手術によって病変を直接的に治療するアプローチを取ります。

</QuizAnswer>

## 問9

陰陽学説における「陰」に分類されるものの組み合わせとして正しいものはどれか。

1. 太陽・昼・熱・上
2. 月・夜・寒・下
3. 月・昼・熱・下
4. 太陽・夜・寒・上

<QuizAnswer>

**答え: 2**

陰陽学説では、自然界の事物を陰と陽に分類します。「陽」は太陽・昼・熱・上・動・明などの性質を持ち、「陰」は月・夜・寒・下・静・暗などの性質を持ちます。陰陽はあくまで相対的な概念であり、どちらが良い・悪いというものではなく、両者のバランスが保たれることが重要とされています。

</QuizAnswer>

## 問10

薬膳の歴史に関する記述として正しいものはどれか。

1. 薬膳は20世紀に日本で生まれた食事法である
2. 薬膳の考え方は古代中国に起源を持ち、2000年以上の歴史がある
3. 薬膳は西洋医学の発展とともにヨーロッパで確立された
4. 薬膳の理論は近代になってから初めて体系化された

<QuizAnswer>

**答え: 2**

薬膳の考え方は「医食同源」の概念とともに、紀元前2000年以上前の古代中国に起源を持ちます。春秋戦国時代にはすでに食物の五味と五臓の相関関係や食の禁忌に関する書物が存在し、薬膳の理論体系はこの時代から形成されていました。唐代から宋代にかけて、薬膳はさらに発展し、王室や貴族の間で健康長寿のための洗練された食事法として広まりました。

</QuizAnswer>
