## 問1

五臓のうち「血を蔵す」働きを持ち、全身の気の流れをスムーズにする疏泄（そせつ）機能を担う臓はどれか。

1. 心
2. 脾
3. 肝
4. 肺


<QuizAnswer>

**答え: 3**

肝は「血を蔵す（蔵血）」働きと、気の流れを調節する「疏泄」機能を持つ臓です。疏泄が正常に働くことで、情緒の安定や消化の促進、気血の円滑な運行が保たれます。肝の機能が低下すると、イライラや抑うつなどの情緒不安定が生じやすくなります。

</QuizAnswer>

## 問2

六腑のうち「三焦（さんしょう）」の働きとして最も適切なものはどれか。

1. 食物を受け入れ消化する
2. 水分を貯留し排尿を管理する
3. 気と水液の通路となり全身の水液代謝を統括する
4. 胆汁を貯蔵し排泄する


<QuizAnswer>

**答え: 3**

三焦は上焦・中焦・下焦に分かれ、気と水液（津液）が全身を巡るための通路として機能します。上焦は心・肺を含み気の散布を、中焦は脾・胃を含み消化吸収を、下焦は腎・膀胱を含み水液の排泄を主に管理しています。三焦は他の五腑と異なり、特定の形態を持たない「有名無形」の腑とされています。

</QuizAnswer>

## 問3

中医学における「気」のうち、体表を覆い外邪の侵入を防ぐ働きを持つものはどれか。

1. 元気（げんき）
2. 宗気（そうき）
3. 営気（えいき）
4. 衛気（えき）


<QuizAnswer>

**答え: 4**

衛気は体表を巡り、外邪（ウイルスや寒さなど）の侵入から身体を守る防衛機能を担います。また、皮膚の開閉を調節し、体温調節にも関与しています。衛気に対して営気は脈中を巡り、血とともに全身に栄養を運ぶ働きを持ちます。

</QuizAnswer>

## 問4

五臓の「脾」の主な機能として正しいものはどれか。

1. 血脈を主り、神志を司る
2. 運化を主り、血を統摂する
3. 水を主り、納気を司る
4. 宣発・粛降を主り、水道を通調する


<QuizAnswer>

**答え: 2**

脾は飲食物の消化吸収と栄養の運搬を行う「運化」機能を持ちます。また、血が脈外に漏れないように管理する「統血」の働きもあり、脾の機能が低下すると出血しやすくなります。脾は「後天の本」と呼ばれ、生まれた後の生命活動を支えるエネルギーの源とされています。

</QuizAnswer>

## 問5

臓腑の「表裏関係」において、肺と表裏をなす腑はどれか。

1. 胃
2. 大腸
3. 小腸
4. 膀胱


<QuizAnswer>

**答え: 2**

五臓と六腑は経絡を通じて表裏関係を持ちます。肺と大腸は表裏関係にあり、肺の粛降機能は大腸の伝導（排便）機能と密接に関連しています。そのため、肺の機能低下が便秘を引き起こしたり、逆に大腸の不調が肺に影響して咳が出ることがあります。

</QuizAnswer>

## 問6

「津液（しんえき）」に関する記述として正しいものはどれか。

1. 津は粘稠で関節や脳髄を潤し、液はさらさらで皮膚や筋肉を潤す
2. 津はさらさらで皮膚や筋肉を潤し、液は粘稠で関節や脳髄を潤す
3. 津液は肝で生成され、心の働きで全身に運ばれる
4. 津液の代謝には腎のみが関与する


<QuizAnswer>

**答え: 2**

津液は「津」と「液」に分けられます。津は性質がさらさらとしており、体表の皮膚や筋肉を潤します。液は粘稠な性質で、関節・脳髄・骨などの深部組織を潤し滋養します。津液の代謝には脾の運化、肺の宣発・粛降、腎の気化など複数の臓が関与しています。

</QuizAnswer>

## 問7

五臓のうち「先天の本」と呼ばれ、精を蔵し、成長・発育・生殖を主る臓はどれか。

1. 肝
2. 心
3. 腎
4. 脾


<QuizAnswer>

**答え: 3**

腎は両親から受け継いだ「先天の精」を蔵し、人体の成長・発育・生殖・老化を主る臓であり、「先天の本」と呼ばれます。また、腎は水液代謝の調節（主水）や、吸気を深く納める「納気」の機能も持っています。腎精の盛衰は、髪・骨・歯・聴力などに反映されるとされています。

</QuizAnswer>

## 問8

「血」の生成に関する記述として最も適切なものはどれか。

1. 血は腎のみで生成される
2. 血は主に脾胃で消化吸収された水穀の精微から生成される
3. 血は肺が外気から直接生成する
4. 血は肝が独自に生成し全身に送り出す


<QuizAnswer>

**答え: 2**

血は主に脾胃が飲食物を消化吸収して得た「水穀の精微」を原料として生成されます。水穀の精微は心肺の働きによって赤く変化し血となります。また、腎に蔵される精も血の生成に関与しており、「精血同源」という言葉があるように、腎精と血は互いに転化・補充し合う関係にあります。

</QuizAnswer>

## 問9

心と小腸の表裏関係に基づく現象として正しいものはどれか。

1. 心に熱がこもると小腸に波及し、尿が赤く濃くなることがある
2. 心の不調は大腸の便秘として現れやすい
3. 小腸の冷えは肺の咳を引き起こす
4. 心と小腸は表裏関係にないため互いに影響しない


<QuizAnswer>

**答え: 1**

心と小腸は表裏関係にあり、経絡を通じて互いに影響し合います。心に熱（心火）がこもると、その熱が表裏関係を通じて小腸に移り、小腸が水分の泌別（清濁の分離）を行う際に影響を受けて、尿が赤く濃くなったり、排尿時に痛みが生じることがあります。これを「心火下移小腸」といいます。

</QuizAnswer>

## 問10

気の働きのうち「固摂（こせつ）作用」の説明として正しいものはどれか。

1. 臓腑の活動を活発にし、体温を維持する
2. 血や津液が脈外に漏れ出ないようにコントロールする
3. 飲食物を気血津液に変換する
4. 外邪が体内に侵入するのを防ぐ


<QuizAnswer>

**答え: 2**

気の固摂作用とは、血が脈外に漏れないように保持したり、汗・尿・唾液などの津液がむやみに体外に流出しないように制御する働きです。また、内臓が正しい位置に保たれるよう支える作用も含まれます。固摂作用が低下すると、出血しやすくなったり、多汗、頻尿、内臓下垂などの症状が現れます。

</QuizAnswer>
