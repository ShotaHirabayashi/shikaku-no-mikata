## 問1

中医学における「気虚体質」の人に最も典型的な症状はどれか。

1. 顔色が赤く、のぼせやすい
2. 疲れやすく、息切れしやすい
3. 目が乾燥し、爪が割れやすい
4. 体がむくみ、体が重だるい


<QuizAnswer>

**答え: 2**

気虚体質は体内の「気」が不足した状態で、エネルギー不足により疲労感・倦怠感・息切れ・声が小さいなどの症状が現れます。免疫力の低下から風邪をひきやすいのも特徴です。選択肢1は陽熱体質、3は血虚体質、4は水毒（痰湿）体質の典型的な症状です。

</QuizAnswer>

## 問2

「血虚体質」の特徴として正しいものはどれか。

1. 顔色が青白く、めまいや立ちくらみが起きやすい
2. イライラしやすく、脇腹が張る感じがする
3. 手足がほてり、寝汗をかきやすい
4. 痰が多く、体が重だるい


<QuizAnswer>

**答え: 1**

血虚体質は血が不足している状態で、顔色が青白い・蒼白、めまい、立ちくらみ、動悸、不眠、目のかすみ、爪が割れやすいなどの症状が特徴的です。女性に多く見られ、月経量が少ない・月経周期が遅れるなどの婦人科症状を伴うこともあります。選択肢2は気滞、3は陰虚の特徴です。

</QuizAnswer>

## 問3

「陰虚体質」と「陽虚体質」の違いについて、正しい記述はどれか。

1. 陰虚は冷えが強く、陽虚はほてりが強い
2. 陰虚は体に熱感があり、陽虚は冷えの症状が出やすい
3. 陰虚は痰が多く、陽虚はのどが渇きやすい
4. 陰虚も陽虚も、どちらも同じ症状が現れる


<QuizAnswer>

**答え: 2**

陰虚体質は体を潤し冷やす「陰」が不足するため、手足のほてり・のどの渇き・寝汗・便秘などの熱っぽい症状が現れます。一方、陽虚体質は体を温める「陽」が不足するため、冷え性・寒がり・顔色が白い・下痢しやすいなどの冷えの症状が中心です。両者は正反対の性質を持ちます。

</QuizAnswer>

## 問4

「気滞（きたい）」体質の人に見られやすい症状として、最も適切なものはどれか。

1. 顔色が蒼白で、疲れやすい
2. 精神的に不安定で、脇腹や胸が張りやすい
3. 手足が冷え、腰がだるい
4. 皮膚が乾燥し、髪にツヤがない


<QuizAnswer>

**答え: 2**

気滞体質は気の流れが滞った状態で、ストレスや感情の抑圧が主な原因です。精神的にイライラしやすい、憂うつになりやすい、ため息が多い、胸や脇腹の張り・痛みなどが特徴的な症状です。症状が精神状態やストレスに左右されやすく、ゲップやおならが多くなることもあります。

</QuizAnswer>

## 問5

「瘀血（おけつ）」体質の特徴として正しいものはどれか。

1. むくみやすく、体が重い
2. 唇や舌の色が暗紫色で、シミができやすい
3. 食欲不振で軟便になりやすい
4. のどが渇きやすく、寝汗をかく


<QuizAnswer>

**答え: 2**

瘀血体質は血の巡りが悪く、血が滞っている状態です。唇や舌の色が暗紫色になる、顔色がくすむ、シミ・そばかすができやすい、肩こり、頭痛、月経痛が強いなどの症状が特徴的です。刺すような固定した痛みが出やすく、女性では月経血に血塊が混じることもあります。

</QuizAnswer>

## 問6

「水毒（痰湿）」体質に最も当てはまる特徴はどれか。

1. のぼせやすく、冷たい飲み物を好む
2. 目がかすみ、爪が薄くもろい
3. 体がむくみやすく、胃がもたれやすい
4. 寝つきが悪く、夢を多く見る


<QuizAnswer>

**答え: 3**

水毒（痰湿）体質は体内の水分代謝が悪く、余分な水分や老廃物が溜まった状態です。むくみ、体の重だるさ、胃もたれ、吐き気、軟便・下痢、おりものが多いなどが代表的な症状です。雨天や湿気の多い時期に症状が悪化しやすいのも特徴で、ぽっちゃり体型の人に多く見られます。

</QuizAnswer>

## 問7

舌診において、舌の色が淡白（薄い色）である場合、最も疑われる体質はどれか。

1. 陽熱体質
2. 瘀血体質
3. 気虚または陽虚体質
4. 気滞体質


<QuizAnswer>

**答え: 3**

舌診では舌の色が体質を判断する重要な指標となります。舌が淡白色（薄いピンクや白っぽい色）の場合は、気や陽が不足している気虚・陽虚体質が疑われます。反対に舌が紅色なら陰虚や熱証、暗紫色なら瘀血が考えられます。舌診は中医学の体質判断において最も基本的な望診の一つです。

</QuizAnswer>

## 問8

「陽熱体質」の人に見られる特徴として、最も適切なものはどれか。

1. 寒がりで厚着を好む
2. 顔が赤く、暑がりで冷たいものを好む
3. 気分が落ち込みやすく、ため息が多い
4. むくみやすく、雨の日に体調が悪化する


<QuizAnswer>

**答え: 2**

陽熱体質は体内に余分な熱がこもりやすい体質で、顔が赤い、暑がり、冷たい飲食物を好む、口が渇く、便秘がちなどの熱の症状が特徴です。体格がしっかりしていて声が大きい傾向があり、ニキビや吹き出物ができやすいのも特徴の一つです。

</QuizAnswer>

## 問9

舌診で舌の苔（舌苔）が厚く白い場合に考えられる体質・状態として、最も適切なものはどれか。

1. 陰虚で体内の潤いが不足している
2. 体内に冷えや湿が溜まっている
3. 瘀血で血行が悪い
4. 気滞でストレスが溜まっている


<QuizAnswer>

**答え: 2**

舌苔は体内の状態を反映する重要な指標です。舌苔が厚く白い場合は、体内に寒（冷え）や湿（余分な水分）が溜まっている状態を示します。一方、舌苔が黄色い場合は体内に熱がある状態、舌苔がほとんどない場合は陰虚（潤い不足）を示唆します。舌苔の色と厚さを観察することで体質の傾向を読み取ることができます。

</QuizAnswer>

## 問10

体質診断チェックの方法として、正しい記述はどれか。

1. 体質は生まれつき決まっており、生涯変化することはない
2. 体質診断では、問診・望診・聞診・切診の四診を総合的に用いる
3. 一人の人間は必ず一つの体質タイプのみに分類される
4. 体質診断は血液検査の数値のみで判定する


<QuizAnswer>

**答え: 2**

中医学の体質診断では「四診」と呼ばれる4つの診察法を総合的に活用します。望診（顔色・舌・体型の観察）、聞診（声や呼吸音・体臭の確認）、問診（症状・生活習慣の聞き取り）、切診（脈診など）を組み合わせて判断します。体質は加齢・季節・生活習慣・ストレスなどで変化し得るものであり、また複数の体質タイプが重なる「複合体質」も珍しくありません。

</QuizAnswer>
