## 問1

ストレスチェック制度は、主にどの段階の予防を目的として導入されたか。

1. 二次予防（メンタルヘルス不調の早期発見・早期治療）
2. 一次予防（メンタルヘルス不調の未然防止）
3. 三次予防（メンタルヘルス不調からの職場復帰支援）
4. 一次予防と三次予防の両方を等しく目的としている

<QuizAnswer>

**答え: 2**

ストレスチェック制度の主な目的は、労働者自身にストレスへの気づきを促し、メンタルヘルス不調を未然に防止する「一次予防」です。二次予防（早期発見・早期治療）や三次予防（職場復帰支援）も重要ですが、ストレスチェック制度はあくまで一次予防を主目的としています。あわせて、検査結果を集団的に分析し、職場環境の改善につなげることも目的とされています。

</QuizAnswer>

## 問2

ストレスチェック制度において、事業者は労働者に対してストレスチェックをどのくらいの頻度で実施することが義務づけられているか。

1. 6か月以内ごとに1回
2. 必要と認めたときに随時
3. 2年以内ごとに1回
4. 1年以内ごとに1回

<QuizAnswer>

**答え: 4**

労働安全衛生法に基づくストレスチェック制度では、事業者は常時使用する労働者に対し、「1年以内ごとに1回」ストレスチェックを実施することが義務づけられています。定期的に実施することで、労働者が自身のストレスの状態を継続的に把握し、変化に気づくことができるようになっています。

</QuizAnswer>

## 問3

ストレスによって現れるストレス反応は、大きく3つの側面に分類される。その正しい組合せはどれか。

1. 心理面・身体面・行動面
2. 心理面・身体面・社会面
3. 感情面・認知面・行動面
4. 精神面・肉体面・対人面

<QuizAnswer>

**答え: 1**

ストレス反応は「心理面」「身体面」「行動面」の3つの側面に分類されます。心理面では不安やイライラ、身体面では頭痛や不眠、行動面では飲酒量の増加や遅刻などが代表的な反応です。これら3つの側面は互いに影響し合い、1つの側面の変化が他の側面にも波及することがあります。

</QuizAnswer>

## 問4

次のうち、ストレスによる「身体面」の反応として最も適切なものはどれか。

1. 仕事のミスが増える
2. イライラして怒りっぽくなる
3. 頭痛や肩こりが続く
4. 遅刻や欠勤が増える

<QuizAnswer>

**答え: 3**

頭痛や肩こりは、ストレスによる代表的な「身体面」の反応です。その他にも、動悸、胃痛、下痢、便秘、食欲低下、不眠なども身体面のストレス反応に含まれます。選択肢2のイライラは「心理面」、選択肢1のミスの増加や選択肢4の遅刻・欠勤は「行動面」の反応に分類されます。

</QuizAnswer>

## 問5

次のうち、ストレスによる「行動面」の反応として最も適切なものはどれか。

1. 気分が落ち込み、憂うつになる
2. 飲酒量や喫煙量が増える
3. 動悸や息切れがする
4. 不安感が強くなる

<QuizAnswer>

**答え: 2**

飲酒量や喫煙量の増加は、ストレスによる代表的な「行動面」の反応です。その他にも、遅刻・欠勤の増加、仕事上のミスの増加、過食や拒食、引きこもりなどが行動面の反応に含まれます。選択肢1や4は「心理面」、選択肢3は「身体面」のストレス反応に該当します。

</QuizAnswer>

## 問6

職業性ストレス簡易調査票（57項目版）は、大きく3つの領域で構成されている。その正しい組合せはどれか。

1. 仕事の量、仕事の質、対人関係
2. 生活習慣、職場環境、家庭環境
3. ストレッサー、ストレス耐性、ストレス発散度
4. 仕事のストレス要因、心身のストレス反応、周囲のサポート

<QuizAnswer>

**答え: 4**

職業性ストレス簡易調査票（57項目版）は、「仕事のストレス要因」（17項目）、「心身のストレス反応」（29項目）、「周囲のサポート」（11項目）の3つの領域で構成されています。ストレスチェック制度では、この3領域を含む調査票を使用することが義務づけられており、厚生労働省はこの57項目版を標準的な調査票として推奨しています。

</QuizAnswer>

## 問7

職業性ストレス簡易調査票の標準的な項目数として正しいものはどれか。

1. 57項目
2. 36項目
3. 45項目
4. 80項目

<QuizAnswer>

**答え: 1**

厚生労働省がストレスチェック制度の標準的な調査票として推奨しているのは「57項目版」の職業性ストレス簡易調査票です。なお、より詳細な分析を行うための80項目版や120項目版もありますが、法令上の要件を満たす標準的なものは57項目版です。事業場の実情に合わせて項目を追加することも認められています。

</QuizAnswer>

## 問8

厚生労働省の「労働者の疲労蓄積度自己診断チェックリスト」で評価する2つの項目の正しい組合せはどれか。

1. 仕事の負担度と人間関係
2. 身体的疲労度と精神的疲労度
3. 自覚症状と勤務の状況
4. 睡眠時間と残業時間

<QuizAnswer>

**答え: 3**

「労働者の疲労蓄積度自己診断チェックリスト」は、「自覚症状」と「勤務の状況」の2つの項目から疲労蓄積度を総合的に判定するものです。自覚症状では体調の変化を、勤務の状況では労働時間や休日の取得状況などを確認します。2023年の改正では、食欲・睡眠・勤務間インターバルに関する項目が追加されました。

</QuizAnswer>

## 問9

ストレスへの気づきにおいて、「いつもと違う自分」に気づくために最も重要なこととして適切なものはどれか。

1. 身体面の変化だけを注意深く観察する
2. 日頃から自分の「普段の状態」を把握しておく
3. ストレスチェックの結果のみを判断基準にする
4. 他の人と比較して自分のストレスレベルを判断する

<QuizAnswer>

**答え: 2**

「いつもと違う自分」に気づくためには、まず日頃から自分の「普段の状態」を把握しておくことが重要です。普段の自分を知っていなければ、変化に気づくことはできません。身体面だけでなく心理面や行動面も含めた3つの側面で自分の状態を観察し、睡眠、食欲、気分、行動パターンなどの変化に注意を払うことがセルフケアの基本となります。

</QuizAnswer>

## 問10

同じ出来事を経験しても、ストレス反応の現れ方には個人差がある。その主な理由として最も適切なものはどれか。

1. ストレス反応は遺伝的要因のみによって決まるため
2. 年齢が高いほどストレスを感じにくくなるため
3. 出来事を経験した季節や時間帯によって反応が異なるため
4. 出来事に対する受け止め方（認知）が人によって異なるため

<QuizAnswer>

**答え: 4**

同じ出来事でも、それをどのように受け止めるか（認知）は人によって異なり、その認知の違いがストレス反応の個人差を生みます。たとえば、上司からの指摘を「自分への攻撃」と受け止める人と「成長のためのアドバイス」と受け止める人では、ストレス反応の現れ方が大きく異なります。このような認知の偏り（認知の歪み）に気づくことも、ストレスへの気づきにおいて重要なポイントです。

</QuizAnswer>
