## 問1

令和5年「労働安全衛生調査（実態調査）」によると、現在の仕事や職業生活に関することで強い不安、悩み、ストレスを感じている労働者の割合として、最も近いものはどれか。

1. 約50％
2. 約60％
3. 約80％
4. 約90％


<QuizAnswer>

**答え: 3**

令和5年労働安全衛生調査（実態調査）によると、強い不安、悩み、ストレスを感じている労働者の割合は82.7％でした。この数値は令和4年調査の82.2％とほぼ同水準であり、労働者の約8割以上が何らかのストレスを抱えている現状が続いています。メンタルヘルスケアの重要性を裏付ける基本的な統計データです。

</QuizAnswer>

## 問2

「労働者の心の健康の保持増進のための指針」で示されている4つのケアに含まれないものはどれか。

1. ファミリーケア
2. セルフケア
3. ラインによるケア
4. 事業場外資源によるケア


<QuizAnswer>

**答え: 1**

4つのケアとは、①セルフケア、②ラインによるケア、③事業場内産業保健スタッフ等によるケア、④事業場外資源によるケアです。「ファミリーケア」は4つのケアには含まれていません。これら4つのケアが継続的かつ計画的に行われることが、効果的なメンタルヘルスケアの推進に重要とされています。

</QuizAnswer>

## 問3

メンタルヘルスケアの活動領域において、メンタルヘルス不調となった労働者の職場復帰の支援等を行うことは何と呼ばれるか。

1. 一次予防
2. 二次予防
3. 事後対応
4. 三次予防


<QuizAnswer>

**答え: 4**

メンタルヘルスケアの活動領域は三段階に分かれます。一次予防はメンタルヘルス不調の未然防止、二次予防は早期発見と適切な対応、三次予防はメンタルヘルス不調となった労働者の職場復帰支援等です。三次予防では、休職した労働者が円滑に職場復帰できるよう計画的に支援することが求められます。

</QuizAnswer>

## 問4

「心の健康づくり計画」に盛り込むべき事項として、適切でないものはどれか。

1. 事業者がメンタルヘルスケアを積極的に推進する旨の表明に関すること
2. 労働者個人のストレス原因と病歴の全社的な共有に関すること
3. 事業場における心の健康づくりの体制の整備に関すること
4. 心の健康づくり計画の実施状況の評価及び計画の見直しに関すること


<QuizAnswer>

**答え: 2**

心の健康づくり計画には、事業者の方針表明、体制整備、問題点の把握とケアの実施、人材確保と事業場外資源の活用、労働者の健康情報の保護、計画の評価と見直しなどを盛り込みます。労働者個人のストレス原因や病歴を全社的に共有することは、プライバシー保護の観点から不適切であり、計画に盛り込むべき事項ではありません。

</QuizAnswer>

## 問5

4つのケアのうち「ラインによるケア」の説明として、最も適切なものはどれか。

1. 労働者自身がストレスに気づき、自ら予防・対処すること
2. 事業場外の専門機関や専門家を活用し支援を受けること
3. 管理監督者が職場環境等の改善や労働者への相談対応を行うこと
4. 産業医が事業場全体のメンタルヘルス方針を単独で決定すること


<QuizAnswer>

**答え: 3**

ラインによるケアとは、労働者と日常的に接する管理監督者が、心の健康に関して職場環境等の改善や労働者に対する相談対応を行うケアのことです。選択肢1はセルフケア、選択肢2は事業場外資源によるケアの説明にあたります。「ライン」とは指揮命令系統のことであり、直属の上司にあたる管理監督者が中心的な役割を担います。

</QuizAnswer>

## 問6

「労働者の心の健康の保持増進のための指針」を策定・公表した機関はどれか。

1. 厚生労働省
2. 経済産業省
3. 内閣府
4. 文部科学省


<QuizAnswer>

**答え: 1**

「労働者の心の健康の保持増進のための指針」は、労働安全衛生法第70条の2第1項に基づき厚生労働省が策定したものです。平成18年（2006年）に策定され、平成27年（2015年）にストレスチェック制度の創設等を踏まえて改正されました。事業場におけるメンタルヘルスケアの原則的な実施方法を定めた重要な指針です。

</QuizAnswer>

## 問7

令和5年「労働安全衛生調査（実態調査）」において、労働者がストレスと感じている事柄の内容として最も割合が高かったものはどれか。

1. 対人関係（セクハラ・パワハラを含む）
2. 仕事の量
3. 会社の将来性
4. 仕事の失敗、責任の発生等


<QuizAnswer>

**答え: 4**

令和5年労働安全衛生調査では、ストレスの内容として「仕事の失敗、責任の発生等」が39.7％で最も多く、次いで「仕事の量」が39.4％、「対人関係（セクハラ・パワハラを含む）」が29.6％という結果でした。なお、過去の調査では「仕事の量」や「対人関係」が上位だった年もあり、調査年によって順位が変動する傾向があります。

</QuizAnswer>

## 問8

メンタルヘルスケアを推進するにあたり、事業者が最初に行うべきこととして最も適切なものはどれか。

1. 事業場外の専門機関との契約締結
2. メンタルヘルスケアを積極的に推進する旨の方針表明
3. 全労働者への個別カウンセリングの実施
4. 管理監督者の人事異動


<QuizAnswer>

**答え: 2**

メンタルヘルス対策推進の第一歩は、事業者がメンタルヘルスケアを積極的に推進する旨の方針を表明することです。事業者方針の表明により、組織全体でメンタルヘルスケアに取り組む姿勢が明確になり、労働者や管理監督者の理解と協力が得られやすくなります。その後、衛生委員会での調査審議、心の健康づくり計画の策定へと進みます。

</QuizAnswer>

## 問9

メンタルヘルスケアの活動領域における「一次予防」の目的として、最も適切なものはどれか。

1. メンタルヘルス不調を未然に防止すること
2. メンタルヘルス不調を早期に発見し対処すること
3. 休職者の職場復帰を支援すること
4. 退職者へのアフターケアを行うこと


<QuizAnswer>

**答え: 1**

一次予防とは、メンタルヘルス不調を未然に防止することを目的としています。具体的には、職場環境の改善やストレスマネジメント研修の実施などが含まれます。選択肢2の早期発見・対処は二次予防、選択肢3の職場復帰支援は三次予防にあたります。一次予防は最も根本的な取り組みであり、メンタルヘルスケアの基盤となるものです。

</QuizAnswer>

## 問10

心の健康づくり計画の策定にあたり、調査審議を行う場として「労働者の心の健康の保持増進のための指針」で定められているのはどれか。

1. 取締役会
2. 労働組合大会
3. 全社朝礼
4. 衛生委員会（安全衛生委員会）


<QuizAnswer>

**答え: 4**

「労働者の心の健康の保持増進のための指針」では、心の健康づくり計画の策定にあたって衛生委員会（安全衛生委員会）において調査審議を行うことが定められています。衛生委員会は労働安全衛生法に基づき設置される機関であり、労使の代表が参加して職場の安全衛生に関する重要事項を審議します。メンタルヘルスケアの推進においても中心的な役割を果たします。

</QuizAnswer>
