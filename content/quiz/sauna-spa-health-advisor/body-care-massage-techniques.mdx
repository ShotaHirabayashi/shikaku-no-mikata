## 問1

ボディケアの基本手技のひとつである「軽擦法（けいさつほう）」の説明として、最も適切なものはどれか。

1. 筋肉をつかみ上げて弧を描くようにこねる手技
2. 手のひらや指で皮膚表面をやさしくなでるようにさする手技
3. 親指や手根で一点を持続的に圧迫する手技
4. 手のこぶしで筋肉をリズミカルにたたく手技


<QuizAnswer>

**答え: 2**

軽擦法は、手のひらや指を皮膚に密着させ、やさしくなでるようにさする手技です。マッサージの開始時と終了時に行われることが多く、皮膚表面の血液循環やリンパの流れを促進する効果があります。選択肢1は揉捏法、選択肢3は指圧法の説明です。

</QuizAnswer>

## 問2

軽擦法を行う際の基本的な方向として、正しいものはどれか。

1. 心臓から遠ざかる方向に向かって行う
2. 筋肉の走行に対して直角方向に行う
3. 心臓に向かう方向（末梢から中枢）に向かって行う
4. 方向は特に決まりがなく、自由に行ってよい


<QuizAnswer>

**答え: 3**

軽擦法は、心臓に向かう方向、つまり体の末端（末梢）から中心部（中枢）に向かって行うのが基本です。これにより静脈血やリンパ液の還流が促進され、老廃物の排出や血行促進の効果が高まります。

</QuizAnswer>

## 問3

揉捏法（じゅうねつほう）の主な作用として、最も適切なものはどれか。

1. 皮膚表面の角質を除去する
2. 筋肉に作用し、組織の新陳代謝を盛んにする
3. 骨格のゆがみを矯正する
4. 皮下脂肪を直接分解する


<QuizAnswer>

**答え: 2**

揉捏法は、筋肉をつかみ上げてこねるようにもむ手技で、主に筋肉に作用します。筋肉の機能を亢進させて萎縮を予防するとともに、組織の新陳代謝を盛んにし、血行を改善する効果があります。

</QuizAnswer>

## 問4

指圧法における「通常圧法」の一点圧の持続時間として、一般的に適切とされるものはどれか。

1. 10〜15秒
2. 3〜5秒
3. 30秒〜1分
4. 1秒以内


<QuizAnswer>

**答え: 2**

通常圧法では、一点に対して3〜5秒の持続圧をかけるのが基本とされています。呼吸に合わせてゆっくりと圧を加え、循環器系や自律神経系の働きを改善する効果が期待できます。短すぎても長すぎても効果が十分に得られません。

</QuizAnswer>

## 問5

入浴後にセルフマッサージを行う場合、マッサージを始める前に行うべきこととして、最も適切なものはどれか。

1. 激しい運動をして体をさらに温める
2. コップ1杯程度の水分を補給する
3. 冷水シャワーで体を完全に冷やす
4. 30分以上安静にしてから始める


<QuizAnswer>

**答え: 2**

入浴後のセルフマッサージの前には、コップ1杯程度の水分を補給することが推奨されます。これはマッサージによって老廃物が流れやすくなるため、水分補給によってその排出を助ける効果があります。入浴で失われた水分の補給にもなります。

</QuizAnswer>

## 問6

セルフマッサージにおける下半身のマッサージの基本的な方向として、正しいものはどれか。

1. 上から下に向かって行う
2. 下から上に向かって行う
3. 内側から外側に向かって行う
4. 円を描くように回転させて行う


<QuizAnswer>

**答え: 2**

セルフマッサージでは、下半身は下から上に向かって行うのが基本です。これはリンパ液や静脈血の流れに沿った方向であり、老廃物を効率的にリンパ節へ送り出す効果があります。上半身は外側から内側に向かって行うのが基本とされています。

</QuizAnswer>

## 問7

ボディケアにおけるマッサージの力加減について、最も適切なものはどれか。

1. 最初から強い力で押し、痛みを感じる程度が効果的である
2. 最初はソフトに行い、徐々に力加減を調整する
3. 終始一定の強い圧力を維持する
4. 痛みを我慢するほど強く行うほうが効果が高い


<QuizAnswer>

**答え: 2**

ボディケアのマッサージでは、最初はソフトに行い、徐々に力加減を調整していくのが正しい方法です。強すぎる力でのマッサージは、筋肉や血管を傷めたり、揉み返しの原因になる可能性があります。心地よいと感じる程度の圧が適切です。

</QuizAnswer>

## 問8

軽擦法の効果を高めるために最も重要なポイントはどれか。

1. できるだけ速いスピードで手を動かすこと
2. 指先だけを使って点で刺激すること
3. 手のひらと皮膚の密着性を高めること
4. 皮膚から手を離しながらリズミカルに行うこと


<QuizAnswer>

**答え: 3**

軽擦法の効果を高めるポイントは、手のひらと皮膚の密着性です。適度な圧をかけて手のひらをしっかり皮膚に密着させることで、血行促進やリンパの流れを促す効果が大きく変わります。手を浮かせたり指先だけで行うと効果が減少します。

</QuizAnswer>

## 問9

揉捏法を腹部に行う場合の効果として、最も適切なものはどれか。

1. 腹筋を鍛えて引き締める効果がある
2. 胃腸の蠕動運動を高め、便通を改善する効果がある
3. 皮下脂肪を直接燃焼させる効果がある
4. 腹部の体温を下げてクールダウンする効果がある


<QuizAnswer>

**答え: 2**

揉捏法を腹部に行う場合は、圧力をかけたままゆっくりとマッサージを行うことで、胃腸の蠕動運動（ぜんどううんどう）の機能を高め、便通を改善する効果があります。腹部への揉捏法は他の部位と比べて特にゆっくり丁寧に行うことが大切です。

</QuizAnswer>

## 問10

入浴後にボディケア（保湿ケア）を行う際、浴室から出てからの推奨される目安時間として、最も適切なものはどれか。

1. 5分以内
2. 15分以内
3. 30分以内
4. 時間は特に気にしなくてよい


<QuizAnswer>

**答え: 1**

入浴後のボディケアは、浴室から出てから5分以内に行うことが推奨されています。入浴後は肌が柔らかく保湿成分が浸透しやすい状態ですが、時間が経つと肌の水分が急速に蒸発し、入浴前より乾燥してしまうことがあります。タオルで軽く押さえるように水分を拭き取ったら、速やかに保湿ケアを行いましょう。

</QuizAnswer>
