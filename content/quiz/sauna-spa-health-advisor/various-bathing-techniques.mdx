## 問1

半身浴を行う際、お湯に浸かる深さとして正しいのはどれか。

1. 足首まで浸かる
2. へそのあたりまで浸かる
3. みぞおちのあたりまで浸かる
4. 肩まで浸かる


<QuizAnswer>

**答え: 3**

半身浴では、お湯の深さをみぞおち（胸の下）あたりまでにするのが正しいやり方です。肩まで浸かると全身浴になり、心臓への水圧負担が大きくなります。みぞおちまでの深さにすることで、心臓や肺への負担を軽減しながら、下半身を中心にゆっくりと温めることができます。

</QuizAnswer>

## 問2

温冷交代浴において、温かいお湯と冷たい水を交互に浴びることで期待される主な効果はどれか。

1. 筋肉量の増加
2. 血管の拡張と収縮の繰り返しによる血行促進
3. 体脂肪の直接的な燃焼
4. 骨密度の向上


<QuizAnswer>

**答え: 2**

温冷交代浴では、温かいお湯で血管が拡張し、冷たい水で血管が収縮します。この拡張と収縮を繰り返すことで血管のポンプ作用が高まり、血流が促進されます。これにより疲労回復やリフレッシュ効果が得られるのが、温冷交代浴の最大の特徴です。

</QuizAnswer>

## 問3

高温浴（42〜44℃）の入浴で主に刺激されるのはどちらの神経か。

1. 副交感神経が刺激され、眠くなる
2. 交感神経が刺激され、活動的になる
3. 運動神経が刺激され、反射が速くなる
4. 感覚神経が刺激され、痛みに敏感になる


<QuizAnswer>

**答え: 2**

42℃以上の高温浴では交感神経が刺激され、心拍数や血圧が上昇し、体が活動的な状態になります。一方、38℃以下の微温浴では副交感神経が優位になり、リラックス効果が得られます。目的に応じて温度を使い分けることが大切です。

</QuizAnswer>

## 問4

微温浴（35〜38℃）の特徴として正しいものはどれか。

1. 短時間（5分以内）で入るのが効果的である
2. 入浴直後に血圧が急上昇する
3. 副交感神経が働き、リラックス効果が得られる
4. 交感神経を刺激して目覚めを良くする


<QuizAnswer>

**答え: 3**

微温浴（35〜38℃）はぬるめのお湯にゆっくり浸かる入浴法で、副交感神経が優位になり、精神的にも安らぎを感じリラックスできます。高温浴と異なり入浴直後から血圧は下降傾向となるため、20分程度の長めの入浴が適しています。

</QuizAnswer>

## 問5

水風呂に入る際の正しい入り方はどれか。

1. 頭から一気に飛び込む
2. 心臓から遠い足先からゆっくり浸かる
3. 上半身から先に浸かる
4. 息を止めて一気に全身を沈める


<QuizAnswer>

**答え: 2**

水風呂に入る際は、心臓から遠い足先からゆっくりと浸かり、徐々に水温に慣れていくのが正しい方法です。一気に全身を沈めると心臓への負担が大きくなり、血圧の急激な変動（ヒートショック）により失神や心臓発作を引き起こす危険があります。

</QuizAnswer>

## 問6

足浴を行う際の適切なお湯の温度はどれか。

1. 30〜34℃
2. 35〜38℃
3. 39〜42℃
4. 43〜46℃


<QuizAnswer>

**答え: 3**

足浴の適温は39〜42℃とされています。心地よいと感じられる温度で、足先からふくらはぎあたりまでを浸けます。これより低いと十分な温熱効果が得られず、高すぎるとやけどや血圧上昇のリスクがあります。時間は10〜15分程度が目安です。

</QuizAnswer>

## 問7

手浴（手の部分浴）が全身を早く温められる理由として正しいものはどれか。

1. 手の皮膚は体の中で最も薄いため
2. 手は心臓に近いため、温まった血液がすぐに心臓に届くから
3. 手には体温調節の中枢があるため
4. 手の面積が体全体の半分を占めるため


<QuizAnswer>

**答え: 2**

手は心臓に比較的近い位置にあるため、手浴で温められた血液がすぐに心臓に戻り、そこから全身へ送り出されます。そのため、足浴に比べて全身を早く温めることができます。また、手は「第二の脳」とも呼ばれ、手を温めることでリラックス効果や快眠効果も期待できます。

</QuizAnswer>

## 問8

サウナや温冷交代浴の後に行う外気浴（休憩）の主な目的として最も適切なものはどれか。

1. 体をさらに温めて発汗を促す
2. 水分を蒸発させて体重を減らす
3. サウナや水風呂で刺激を受けた体を通常の状態に戻す
4. 紫外線を浴びてビタミンDを生成する


<QuizAnswer>

**答え: 3**

外気浴（休憩）の最も重要な目的は、サウナの高温や水風呂の冷水で強い刺激を受けた体を、通常の正常な状態にリセットすることです。外気浴により交感神経と副交感神経のバランスが整い、深いリラックス状態に入ることで「ととのう」感覚が生まれます。

</QuizAnswer>

## 問9

半身浴に適したお湯の温度と入浴時間の組み合わせとして正しいものはどれか。

1. 42〜44℃で5〜10分
2. 38〜40℃で20〜30分
3. 35〜37℃で60分以上
4. 44〜46℃で3〜5分


<QuizAnswer>

**答え: 2**

半身浴の適温は38〜40℃のぬるめのお湯で、入浴時間は20〜30分が理想的です。ぬるめのお湯にゆっくり浸かることで体温の上昇がゆるやかになり、体への負担が少なく、じんわりと汗をかくことができます。高温で短時間だと全身浴と変わらず、半身浴の効果が十分に得られません。

</QuizAnswer>

## 問10

水風呂に入る時間の目安として適切なものはどれか。

1. 1〜2分程度
2. 5〜10分程度
3. 15〜20分程度
4. 30分以上


<QuizAnswer>

**答え: 1**

水風呂に全身で浸かる時間は1〜2分程度が適切です。長時間入りすぎると体が冷えすぎてしまい、低体温症や体調不良のリスクがあります。また、水風呂に入る前にはシャワーや掛け湯で汗を流すのがマナーであり、体への急激な刺激を和らげる効果もあります。

</QuizAnswer>
