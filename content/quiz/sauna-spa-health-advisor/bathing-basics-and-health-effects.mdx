## 問1

入浴の「3大効果」として正しい組み合わせはどれか。

1. 温熱効果・水圧効果・浮力効果
2. 温熱効果・発汗効果・浮力効果
3. 血行効果・水圧効果・リラックス効果
4. 温熱効果・水圧効果・洗浄効果


<QuizAnswer>

**答え: 1**

入浴の3大効果は「温熱効果」「水圧効果（静水圧作用）」「浮力効果」の3つです。これらの物理的作用が身体に同時に働くことで、血行促進・新陳代謝の向上・疲労回復など、さまざまな健康効果がもたらされます。発汗や洗浄も入浴の恩恵ですが、3大効果には含まれません。

</QuizAnswer>

## 問2

入浴による温熱効果の説明として最も適切なものはどれか。

1. 体温が上昇することで発汗が促され、体内の水分が急速に失われる
2. 皮膚の毛細血管や皮下の血管が広がり、全身の血流が促進される
3. 体内の脂肪が直接燃焼され、ダイエット効果が得られる
4. 血管が収縮することで血圧が上昇し、心臓が活発に動く


<QuizAnswer>

**答え: 2**

温熱効果とは、湯の温かさにより皮膚の毛細血管や皮下の血管が拡張し、全身の血流が促進される作用のことです。血流が良くなることで、酸素や栄養素が体の隅々まで行き渡りやすくなり、老廃物の排出も促されます。血管は「収縮」ではなく「拡張」するのがポイントです。

</QuizAnswer>

## 問3

入浴時の水圧効果（静水圧作用）について、正しい記述はどれか。

1. 水圧は水深に関係なく全身に均一にかかる
2. 水圧により腹部の内臓が刺激を受け、血管やリンパ管が圧迫されることで血液やリンパ液の循環が促進される
3. 水圧は皮膚表面にのみ作用し、内臓には影響を及ぼさない
4. 水圧の効果は半身浴でも全身浴とまったく同じである


<QuizAnswer>

**答え: 2**

水圧効果（静水圧作用）では、水の圧力が手足の血管や腹部の内臓を圧迫し、血液やリンパ液が心臓に戻りやすくなることで全身の循環が促進されます。水圧は水深1mにつき約76mmHg（0.1気圧）ずつ増加するため、水深が深いほど大きな圧力がかかります。したがって、全身浴と半身浴では水圧の影響が異なります。

</QuizAnswer>

## 問4

首まで湯船に浸かったとき、浮力の作用で水中での体重はおよそどのくらいになるか。

1. 通常の約1/2
2. 通常の約1/3
3. 通常の約1/5
4. 通常の約1/10


<QuizAnswer>

**答え: 4**

アルキメデスの原理により、水中では浮力が働きます。首まで湯船に浸かった場合、水中での体重は通常の約1/10にまで軽減されます。たとえば体重60kgの人であれば、水中ではわずか約6kgになります。この大幅な荷重軽減により、普段体を支えている筋肉や関節の負担が減り、リラックス効果が得られます。

</QuizAnswer>

## 問5

入浴による浮力効果がもたらす作用として最も適切なものはどれか。

1. 筋肉や関節への負荷が軽減され、身体の緊張がほぐれる
2. 水中で体重が重くなり、筋力トレーニング効果が得られる
3. 浮力によって血圧が大幅に上昇する
4. 浮力の作用で呼吸数が増加し、心肺機能が高まる


<QuizAnswer>

**答え: 1**

浮力効果の最大の特徴は、水中で体重が大幅に軽減されることで、常に体を支えている筋肉や関節への負荷が減少し、身体の緊張がほぐれることです。この筋肉の弛緩作用に加え、ふわっとした浮遊感によるリラックス効果も得られます。なお、呼吸数の増加や心肺機能の向上は水圧効果によるものです。

</QuizAnswer>

## 問6

入浴時の水圧によって横隔膜が押し上げられることで起こる変化として正しいものはどれか。

1. 肺の容量が増大し、呼吸がゆっくりになる
2. 肺の容量が減少し、呼吸の回数が増えることで心肺機能が高まる
3. 横隔膜が押し上げられることで消化機能が促進される
4. 腹筋が強化され、体幹トレーニングの効果がある


<QuizAnswer>

**答え: 2**

入浴時、水圧によって腹部が圧迫されると横隔膜が押し上げられ、肺の容量が減少します。すると体は不足する空気を補おうとして呼吸の回数が自然に増加し、結果として心肺機能が高まります。これは水圧効果（静水圧作用）の重要な作用のひとつです。

</QuizAnswer>

## 問7

38～40℃程度のぬるめの湯に入浴したとき、自律神経にはどのような影響があるか。

1. 交感神経が優位になり、心身が興奮状態になる
2. 交感神経と副交感神経の両方が同時に抑制される
3. 副交感神経が優位になり、心身がリラックス状態になる
4. 自律神経にはほとんど影響を及ぼさない


<QuizAnswer>

**答え: 3**

38～40℃程度のぬるめの湯に浸かると、自律神経のうち副交感神経が優位になります。副交感神経は心身をリラックスさせ、休息させる働きがあるため、血圧や脈拍の変化が少なく、筋肉の弛緩やリラックス効果が得られます。一方、42℃以上の熱い湯では交感神経が優位になり、心身が興奮状態になります。

</QuizAnswer>

## 問8

入浴による血行促進と新陳代謝の関係について、正しい記述はどれか。

1. 血行が促進されても新陳代謝には直接的な影響はない
2. 新陳代謝が高まるのは入浴中だけで、入浴後はすぐに元に戻る
3. 血行促進により体内の老廃物や疲労物質が排出されやすくなる
4. 血行が促進されると新陳代謝は逆に低下する


<QuizAnswer>

**答え: 3**

入浴によって温熱効果・水圧効果・浮力効果が同時に働くことで全身の血行が促進されると、新陳代謝が高まります。その結果、体内に蓄積された老廃物や疲労物質（乳酸など）が血流に乗って運ばれ、体外に排出されやすくなります。これが入浴による疲労回復メカニズムの基本的な仕組みです。

</QuizAnswer>

## 問9

42℃以上の熱い湯に入浴した場合の身体への影響として正しいものはどれか。

1. 副交感神経が優位になり、深いリラックス状態になる
2. 交感神経が優位になり、血圧の上昇や脈拍の増加がみられる
3. 自律神経への影響はなく、温熱効果だけが強まる
4. 血管が拡張して血圧が大きく低下する


<QuizAnswer>

**答え: 2**

42℃以上の熱い湯に入ると、交感神経が刺激されて優位になり、心身は興奮状態になります。その結果、血圧の上昇や脈拍の増加がみられます。リラックス効果を得たい場合は38～40℃程度のぬるめの湯が推奨されます。高温浴は身体への負担が大きいため、特に高齢者や心臓疾患のある方は注意が必要です。

</QuizAnswer>

## 問10

入浴による疲労回復のメカニズムとして最も適切な説明はどれか。

1. 入浴中に大量の汗をかくことで、疲労物質が汗とともに直接排出される
2. 浮力により筋肉に強い刺激が加わり、筋力が回復する
3. 温熱・水圧・浮力の相乗効果により血行が促進され、老廃物や疲労物質の排出が促される
4. 水圧によって筋肉が圧迫され、疲労物質が直接分解される


<QuizAnswer>

**答え: 3**

入浴による疲労回復は、温熱効果（血管拡張・血流促進）、水圧効果（血液・リンパ循環の促進）、浮力効果（筋肉・関節の緊張緩和）の3つが相乗的に働くことで実現します。血行が促進されることにより、筋肉に蓄積された老廃物や疲労物質が効率よく運び出され、同時に酸素や栄養素が全身に行き渡ることで、疲労回復が促されます。

</QuizAnswer>
