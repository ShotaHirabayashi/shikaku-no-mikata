## 問1

サウナの発祥の地として知られる国はどこか。

1. ロシア
2. ドイツ
3. フィンランド
4. スウェーデン


<QuizAnswer>

**答え: 3**

サウナはフィンランドが発祥の地として世界的に知られています。その歴史は約2000年以上前にさかのぼり、最初は土を掘った穴に石を置き、火を焚いて蒸気を発生させる簡素なものでした。フィンランドでは現在もサウナ文化が深く根付いており、人口約550万人に対してサウナが約300万個あるといわれています。

</QuizAnswer>

## 問2

日本でサウナが広く知られるきっかけとなった出来事として、最も適切なものはどれか。

1. 1956年メルボルンオリンピック
2. 1964年東京オリンピック
3. 1970年大阪万博
4. 1972年札幌冬季オリンピック


<QuizAnswer>

**答え: 2**

1964年の東京オリンピックの際、選手村にフィンランドのサウナが設置され、それがマスコミに大きく取り上げられたことで日本でサウナが広く知られるようになりました。これが「第1次サウナブーム」と呼ばれています。当時の高度経済成長も相まって、サウナ施設が急速に普及しました。

</QuizAnswer>

## 問3

ドライサウナの一般的な温度と湿度の組み合わせとして、最も適切なものはどれか。

1. 温度40〜50℃、湿度80〜100%
2. 温度60〜70℃、湿度40〜60%
3. 温度80〜100℃、湿度5〜10%
4. 温度100〜120℃、湿度30〜50%


<QuizAnswer>

**答え: 3**

日本で主流のドライサウナは、温度80〜100℃、湿度5〜10%という高温低湿が特徴です。からっとした熱さで短時間で大量の発汗を促します。一方、ウェットサウナ（湿式サウナ）は温度が低く湿度が高いのが特徴で、蒸気によって体感温度を高める方式です。

</QuizAnswer>

## 問4

フィンランド式サウナで、サウナストーンに水をかけて蒸気を発生させる行為を何というか。

1. アウフグース
2. ロウリュ
3. ヴィヒタ
4. メトス


<QuizAnswer>

**答え: 2**

ロウリュ（フィンランド語: löyly）は、熱したサウナストーンに水をかけて水蒸気を発生させる行為のことです。フィンランド語で「蒸気」を意味し、語源には「魂、呼吸」という意味もあります。なお、アウフグースはドイツ発祥の、タオルで蒸気を仰ぐサービスを指します。

</QuizAnswer>

## 問5

サウナの正しい入り方の基本的な流れとして、最も適切なものはどれか。

1. 水風呂 → サウナ → 休憩
2. サウナ → 休憩 → 水風呂
3. サウナ → 水風呂 → 休憩
4. 休憩 → サウナ → 水風呂


<QuizAnswer>

**答え: 3**

サウナの基本的な入り方は「サウナ → 水風呂 → 休憩（外気浴）」の3ステップを1セットとし、これを2〜3セット繰り返すのが一般的です。サウナで体を十分に温めてから水風呂で冷やし、その後休憩で体を落ち着かせることで、自律神経が整い心身のリフレッシュにつながります。

</QuizAnswer>

## 問6

サウナに入る前の準備として、最も重要なものはどれか。

1. 軽い飲酒で体を温めておく
2. コップ1〜2杯の水分を摂る
3. 満腹になるまで食事をする
4. 激しい運動をして体温を上げておく


<QuizAnswer>

**答え: 2**

サウナに入る前には、コップ1〜2杯程度の水分補給が非常に重要です。サウナでは大量の汗をかくため、水分が不足した状態で入ると脱水症状を起こす危険があります。なお、アルコールやカフェインを含む飲料は利尿作用があるため、サウナ前の水分補給としては適していません。

</QuizAnswer>

## 問7

サウナの利用を避けるべき状況として、誤っているものはどれか。

1. 飲酒後
2. 食後すぐ
3. 十分な睡眠をとった翌朝
4. 体調不良のとき


<QuizAnswer>

**答え: 3**

十分な睡眠をとった翌朝は体調が良好な状態であり、サウナの利用に問題はありません。一方、飲酒後は脱水症状のリスクが高まり、食後すぐは消化に必要な血液がサウナの熱により分散されるため避けるべきです。体調不良時も立ちくらみや脱水のリスクがあるため利用を控えましょう。

</QuizAnswer>

## 問8

サウナ1セットあたりのサウナ室の滞在時間の目安として、最も適切なものはどれか。

1. 1〜3分
2. 6〜12分
3. 20〜30分
4. 40〜60分


<QuizAnswer>

**答え: 2**

サウナ室の1回あたりの滞在時間は、6〜12分程度が目安とされています。無理に長時間入ると、脱水症状やのぼせの原因になるため危険です。体調や体質には個人差があるため、自分の体の状態に合わせて、無理のない時間で退室することが大切です。

</QuizAnswer>

## 問9

ウェットサウナの特徴として、最も適切なものはどれか。

1. 温度が高く湿度が低い
2. 温度が低く湿度が高い
3. 温度も湿度も非常に高い
4. 温度も湿度も非常に低い


<QuizAnswer>

**答え: 2**

ウェットサウナ（湿式サウナ）は、ドライサウナと比較して温度が低く湿度が高いのが特徴です。蒸気を利用して体感温度を上げるため、肌や喉への負担が少なく、サウナ初心者や乾燥が気になる方にも利用しやすいタイプです。スチームサウナやミストサウナもウェットサウナに分類されます。

</QuizAnswer>

## 問10

サウナに入る前に体を洗う（かけ湯やシャワーをする）理由として、最も適切なものはどれか。

1. 体を冷やしてサウナの効果を高めるため
2. 体の汚れや皮脂を落とし、衛生を保つとともに発汗しやすくするため
3. 水風呂に入る練習をするため
4. サウナ室の温度を下げないようにするため


<QuizAnswer>

**答え: 2**

サウナに入る前に体を洗うのは、衛生面への配慮と発汗効果を高めるためです。体の汚れや皮脂が毛穴をふさいでいると汗が出にくくなるため、事前に洗い流すことでスムーズな発汗が促されます。また、多くの人が利用する共用空間であるため、マナーとしても入室前の洗体は基本とされています。

</QuizAnswer>
