## 問1

「万能のツボ」とも呼ばれ、頭痛・肩こり・歯痛・ストレスなど幅広い症状に効果があるとされるツボ「合谷（ごうこく）」の正しい位置はどこか。

1. 足の裏の中央よりやや上、足指を曲げたときにへこむ部分
2. 手の甲側で、親指と人差し指の骨が交わるくぼみの部分
3. 内くるぶしの一番高いところから指4本分上
4. 手のひら側の手首のシワから指3本分ひじ寄りの中央


<QuizAnswer>

**答え: 2**

合谷（ごうこく）は手の甲側にあり、親指と人差し指の骨が合流するV字のくぼみに位置します。頭痛、肩こり、歯痛、目の疲れ、便秘、ストレスなど非常に幅広い症状に効果があるとされ、「万能のツボ」として知られています。東洋医学では「四総穴」の一つに数えられる代表的なツボです。

</QuizAnswer>

## 問2

頭頂部に位置し、自律神経の調整や頭痛の緩和、ストレス解消に効果があるとされるツボはどれか。

1. 天柱（てんちゅう）
2. 風池（ふうち）
3. 百会（ひゃくえ）
4. 労宮（ろうきゅう）


<QuizAnswer>

**答え: 3**

百会（ひゃくえ）は頭頂部にあり、両耳の先端を結んだ線と、鼻から頭頂部へ向かう正中線が交わる点に位置します。「百（多くの）経絡が会（交わる）場所」という名前の通り、頭痛・めまい・不眠・ストレスなど多くの症状に効果があるとされています。頭頂部を指の腹で垂直にやさしく押すのが基本です。

</QuizAnswer>

## 問3

足の裏にあり、体力や気力の回復に効果があるとされるツボ「湧泉（ゆうせん）」の正しい位置はどれか。

1. 足の親指と人差し指の骨が交わるくぼみ
2. 足の裏の中央よりやや指先寄りで、足指を曲げたときにへこむ部分
3. 内くるぶしの真下のくぼみ
4. かかとの中央部分


<QuizAnswer>

**答え: 2**

湧泉（ゆうせん）は足の裏の中央よりやや指先寄りにあり、足の指を内側に曲げたときにできるくぼみの部分に位置します。「泉が湧く」という名前の通り、生命エネルギーが湧き出るツボとされ、疲労回復・体力増進・冷え性の改善などに効果があります。腎臓の機能を高めるツボとしても知られています。

</QuizAnswer>

## 問4

ツボ押しの力加減と時間について、最も適切なものはどれか。

1. 強く押すほど効果が高いので、痛みを我慢して押し続けるとよい
2. 1か所につき1分以上、できるだけ長く押し続けるのが効果的である
3. 「痛気持ちいい」程度の力で5秒ほど押し、ゆっくり離すのを数回繰り返す
4. ツボは軽く触れる程度の力で10秒以上押し続けるのが正しい


<QuizAnswer>

**答え: 3**

ツボ押しの基本は「痛気持ちいい」と感じる程度の力加減で、5秒ほど押してからゆっくり力を抜いて離す動作を3〜5回繰り返すことです。強く押しすぎると「もみ返し」と呼ばれる炎症を起こす場合があり、逆効果になります。長時間押し続けることも効果の向上にはつながりません。

</QuizAnswer>

## 問5

別名「女性のツボ」とも呼ばれ、冷え性の改善や女性ホルモンの調節に効果があるとされる「三陰交（さんいんこう）」の位置として正しいものはどれか。

1. 膝のお皿の外側下端から指4本分下のすねの外側
2. 手のひらの中央、中指と薬指の間
3. 足首の外側のくるぶしの後ろのくぼみ
4. 内くるぶしの一番高いところから指4本分上、すねの骨のきわ


<QuizAnswer>

**答え: 4**

三陰交（さんいんこう）は内くるぶしの最も高い部分から指4本分上方、すねの骨（脛骨）の内側きわに位置します。肝・腎・脾の三つの陰経が交わることからこの名がつきました。冷え性、生理不順、更年期障害など女性特有の症状に効果があるとされ、「女性のツボ」とも呼ばれています。

</QuizAnswer>

## 問6

肩こりに対して即効性が高く、首の付け根と肩先の中間にあるツボはどれか。

1. 合谷（ごうこく）
2. 肩井（けんせい）
3. 湧泉（ゆうせん）
4. 足三里（あしさんり）


<QuizAnswer>

**答え: 2**

肩井（けんせい）は、首の付け根（第7頸椎）と肩先を結んだ線の中間点に位置するツボです。肩こりで最も代表的なツボであり、押すと心地よさを感じやすく即効性が期待できます。デスクワークなどで肩がこったときに、反対の手の中指で垂直に押すと効果的です。

</QuizAnswer>

## 問7

入浴時のツボ押しについて、最も適切な説明はどれか。

1. 入浴中は血圧が下がるため、ツボ押しは一切行わないほうがよい
2. 入浴で体が温まり血行がよくなった状態では、ツボ押しの効果が高まりやすい
3. 入浴中は熱いお湯の刺激があるため、ツボは強めに押す必要がある
4. 入浴前の冷えた状態のほうがツボの効果は高い


<QuizAnswer>

**答え: 2**

入浴によって体が温まると血液やリンパの流れがよくなり、筋肉も緩むため、ツボ押しの効果が高まりやすい状態になります。ただし、入浴中に強い力でツボを刺激するのは逆効果になることがあるため、ゆるやかな力加減で行うことが大切です。リラックスした状態でのツボ押しが最も効果的とされています。

</QuizAnswer>

## 問8

手のひらの中央付近、手を握ったときに中指と薬指の先端が当たる位置にあり、ストレスや緊張の緩和に効果があるツボはどれか。

1. 内関（ないかん）
2. 合谷（ごうこく）
3. 労宮（ろうきゅう）
4. 曲池（きょくち）


<QuizAnswer>

**答え: 3**

労宮（ろうきゅう）は手のひらの中央付近にあり、手を軽く握ったときに中指と薬指の先端が当たる位置に位置します。「労」は疲労や心労を意味し、ストレス・緊張・不安感・イライラ・不眠などの精神的な症状の緩和に効果があるとされています。手軽に押せるため、仕事中のストレス緩和にも活用しやすいツボです。

</QuizAnswer>

## 問9

膝のお皿の外側下端から指4本分ほど下がった位置にあり、「灸をすえれば長生きする」とも言われる胃腸の調子を整え疲労回復に効くツボはどれか。

1. 三陰交（さんいんこう）
2. 湧泉（ゆうせん）
3. 百会（ひゃくえ）
4. 足三里（あしさんり）


<QuizAnswer>

**答え: 4**

足三里（あしさんり）は膝のお皿（膝蓋骨）の外側下端から指4本分ほど下、すねの骨のすぐ外側に位置します。古くから「足三里に灸すれば長生きする」と言われるほど健康維持に重要なツボで、胃腸の不調、食欲不振、疲労回復、足のだるさ、免疫力向上などに幅広い効果があるとされています。松尾芭蕉も旅の前に足三里に灸をすえたと伝えられています。

</QuizAnswer>

## 問10

手首の内側にあり、自律神経を整える効果やストレスからくる吐き気・胃の不快感の緩和に用いられるツボ「内関（ないかん）」の位置として正しいものはどれか。

1. 手首の外側（手の甲側）のシワの中央
2. 手首の内側のシワから指3本分ひじ寄り、2本の腱の間
3. 手首の内側のシワの小指側のくぼみ
4. ひじの内側の曲がりジワの外端


<QuizAnswer>

**答え: 2**

内関（ないかん）は手のひら側の手首のシワから、ひじに向かって指3本分（約5cm）上がった位置にある、2本の腱の間にあります。心包経に属するツボで、自律神経の調整、ストレスによるイライラや不安感の緩和、吐き気・乗り物酔い・胃の不快感の軽減に効果があるとされています。精神を落ち着ける作用があり、緊張する場面の前に押すのも効果的です。

</QuizAnswer>
