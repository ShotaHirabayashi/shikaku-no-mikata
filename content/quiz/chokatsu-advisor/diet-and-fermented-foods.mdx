## 問1

納豆の発酵に関わる主な微生物はどれか。

1. 乳酸菌
2. 麹菌
3. 納豆菌（枯草菌）
4. 酢酸菌


<QuizAnswer>

**答え: 3**

納豆は大豆を納豆菌（枯草菌の一種、Bacillus subtilis var. natto）で発酵させた食品です。納豆菌は稲わらなどに自然に生息しており、大豆のタンパク質を分解してナットウキナーゼなどの酵素や、ビタミンK2を豊富に生成します。乳酸菌はヨーグルトや漬物、麹菌は味噌や醤油、酢酸菌はお酢の発酵に関わる微生物です。

</QuizAnswer>

## 問2

水溶性食物繊維の特徴として正しいものはどれか。

1. 水に溶けず、便のカサを増やして腸を刺激する
2. 水に溶けてゲル状になり、糖や脂質の吸収速度を遅らせる
3. 胃酸で完全に分解され、腸には届かない
4. 動物性食品に多く含まれている


<QuizAnswer>

**答え: 2**

水溶性食物繊維は水に溶けるとゲル状になる性質があり、糖質や脂質の吸収速度を緩やかにする働きがあります。また、腸内で善玉菌のエサとなり、短鎖脂肪酸の産生を促します。便のカサを増やして腸を刺激するのは不溶性食物繊維の特徴です。水溶性食物繊維は海藻類、果物、大麦などの植物性食品に多く含まれています。

</QuizAnswer>

## 問3

味噌の発酵に欠かせない微生物はどれか。

1. 納豆菌
2. 酵母菌
3. 麹菌（こうじきん）
4. 酢酸菌


<QuizAnswer>

**答え: 3**

味噌は大豆・塩・麹（米麹・麦麹・豆麹）を原料とし、麹菌（Aspergillus oryzae）の働きで発酵・熟成させて作られます。麹菌が大豆のタンパク質や炭水化物を分解することで、うまみ成分であるアミノ酸や糖が生まれます。発酵の過程では酵母菌や乳酸菌も関与しますが、味噌づくりの出発点として不可欠なのは麹菌です。

</QuizAnswer>

## 問4

オリゴ糖が腸内環境の改善に役立つ主な理由はどれか。

1. 胃で完全に消化・吸収されてエネルギーになるから
2. 悪玉菌を直接殺菌する作用があるから
3. 消化されにくく大腸まで届き、善玉菌のエサになるから
4. 腸壁を物理的にコーティングして保護するから


<QuizAnswer>

**答え: 3**

オリゴ糖は消化酵素で分解されにくい性質を持ち、小腸で吸収されずに大腸まで到達します。大腸に届いたオリゴ糖はビフィズス菌などの善玉菌のエサとなり、善玉菌の増殖を助けます。代表的なものにフラクトオリゴ糖、ガラクトオリゴ糖、大豆オリゴ糖などがあり、玉ねぎ、バナナ、大豆、はちみつなどに含まれています。

</QuizAnswer>

## 問5

不溶性食物繊維を多く含む食品の組み合わせとして最も適切なものはどれか。

1. わかめ・昆布・もずく
2. ごぼう・さつまいも・きのこ類
3. りんご・バナナ・アボカド
4. 大麦・オートミール・こんにゃく


<QuizAnswer>

**答え: 2**

不溶性食物繊維はごぼう、さつまいも、きのこ類、豆類、穀類の外皮などに多く含まれています。これらは水に溶けず、腸内で水分を吸収して膨らみ、便のカサを増やすことで腸のぜんどう運動を促進します。わかめ・昆布などの海藻類や、大麦・オートミール・こんにゃくは水溶性食物繊維が豊富な食品です。

</QuizAnswer>

## 問6

ぬか漬けが腸に良いとされる理由として最も適切なものはどれか。

1. ぬか床の塩分が腸壁を強化するから
2. 野菜のビタミンCがすべて失われ、消化が楽になるから
3. ぬか床に含まれる乳酸菌が発酵を行い、生きた菌や有機酸が摂取できるから
4. ぬかの油分が腸内をコーティングするから


<QuizAnswer>

**答え: 3**

ぬか漬けはぬか床に含まれる植物性乳酸菌の働きで野菜を発酵させた食品です。植物性乳酸菌は胃酸や胆汁酸に強く、生きたまま腸に届きやすい特徴があります。また発酵過程で生成される有機酸（乳酸など）が腸内環境を弱酸性に保ち、善玉菌が活動しやすい環境を作ります。さらに、ぬか由来のビタミンB群が野菜に移行し栄養価も高まります。

</QuizAnswer>

## 問7

腸活における正しい水分補給の方法として、最も適切なものはどれか。

1. 一度に大量の冷水を飲むと腸が活性化して効果的である
2. 食事中は水分を一切摂らないほうが消化に良い
3. 起床時にコップ1杯の常温の水や白湯を飲むと腸が刺激される
4. カフェインの多い飲料で水分補給するのが最も効率的である


<QuizAnswer>

**答え: 3**

朝起きてすぐにコップ1杯（約200ml）の常温の水や白湯を飲むことで、胃腸が穏やかに刺激され、腸のぜんどう運動が促されます。冷たすぎる水は腸への刺激が強すぎるため、常温か白湯が推奨されます。また、1日を通してこまめに水分を摂ることが大切で、一度に大量に飲むのではなく、少量ずつ複数回に分けて摂取するのが効果的です。

</QuizAnswer>

## 問8

発酵食品と食物繊維を組み合わせて食べることが腸活に効果的とされる理由はどれか。

1. 発酵食品の酸味が食物繊維の苦味を消してくれるから
2. 発酵食品が善玉菌を補い、食物繊維がその善玉菌のエサとなるから
3. 両方を一緒に摂ると体温が上がり代謝が倍増するから
4. 食物繊維が発酵食品の塩分を中和してくれるから


<QuizAnswer>

**答え: 2**

発酵食品にはヨーグルトの乳酸菌や納豆の納豆菌など生きた善玉菌が含まれており、これを摂取することで腸内に善玉菌を補うことができます（プロバイオティクス）。一方、食物繊維やオリゴ糖はその善玉菌のエサとなり増殖を助けます（プレバイオティクス）。この両方を組み合わせる「シンバイオティクス」の考え方が、効率的な腸活の基本とされています。

</QuizAnswer>

## 問9

ヨーグルトに含まれる乳酸菌の効果を高める食べ方として、最も適切なものはどれか。

1. 空腹時に大量に食べると吸収が最大になる
2. 加熱してから食べると菌が活性化する
3. バナナやはちみつなどオリゴ糖を含む食品と一緒に食べる
4. 食後すぐに冷水を飲んで胃酸を薄めてから食べる


<QuizAnswer>

**答え: 3**

ヨーグルトの乳酸菌やビフィズス菌の効果を高めるには、善玉菌のエサとなるオリゴ糖や食物繊維を一緒に摂ることが有効です。バナナにはフラクトオリゴ糖、はちみつにもオリゴ糖が含まれており、腸内で善玉菌の増殖を助けます。なお、空腹時は胃酸が強いため菌が死滅しやすく、食後など胃酸が薄まったタイミングで食べるほうが菌が腸に届きやすくなります。

</QuizAnswer>

## 問10

腸に良い食材を日常的に取り入れる方法として、最も適切なものはどれか。

1. 特定の発酵食品を1種類だけ大量に食べ続ける
2. 味噌汁にわかめや根菜を加え、毎日の食事で発酵食品と食物繊維を無理なく摂る
3. サプリメントだけで必要な菌と食物繊維をすべて補う
4. 週に1回まとめて発酵食品を大量に食べればよい


<QuizAnswer>

**答え: 2**

腸活は毎日の食事で継続的に取り組むことが重要です。味噌汁は発酵食品である味噌を使い、具材にわかめ（水溶性食物繊維）や根菜類（不溶性食物繊維）を加えることで、発酵食品と食物繊維を同時に摂取できる理想的な腸活メニューです。1種類の食品に偏らず、さまざまな発酵食品や食物繊維を組み合わせて、毎日少しずつ継続することが腸内環境の改善に効果的です。

</QuizAnswer>
