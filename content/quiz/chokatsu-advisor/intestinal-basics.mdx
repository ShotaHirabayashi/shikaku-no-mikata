## 問1

小腸は3つの部位に分かれています。胃に最も近い部分から順に正しく並べたものはどれですか。

1. 空腸→十二指腸→回腸
2. 十二指腸→回腸→空腸
3. 十二指腸→空腸→回腸
4. 回腸→空腸→十二指腸


<QuizAnswer>

**答え: 3**

小腸は胃に近い方から「十二指腸→空腸→回腸」の順に並んでいます。十二指腸は胃から送られた食物を最初に受け取る部分で、膵液や胆汁が合流します。空腸と回腸では主に栄養素の消化・吸収が行われます。

</QuizAnswer>

## 問2

成人の小腸の長さとして最も適切なものはどれですか。

1. 約1.5メートル
2. 約3メートル
3. 約6メートル
4. 約12メートル


<QuizAnswer>

**答え: 3**

成人の小腸の長さは約6〜7メートルで、消化管全体の約80％を占めています。この長さにより食物と消化液が十分に混ざり合い、効率的に栄養素を吸収することができます。一方、大腸は約1.5〜2メートルです。

</QuizAnswer>

## 問3

小腸の内壁にある「絨毛（じゅうもう）」の主な役割として正しいものはどれですか。

1. 食物を物理的にすりつぶして細かくする
2. 粘膜の表面積を広げて栄養素の吸収効率を高める
3. 有害な細菌を直接殺菌する
4. 便を固形にするために水分を吸収する


<QuizAnswer>

**答え: 2**

絨毛は小腸の内壁に無数に存在する小さな突起で、さらにその表面には微絨毛が発達しています。これらの構造により小腸の粘膜表面積はテニスコート約1面分にまで広がり、栄養素を非常に効率よく吸収できるようになっています。

</QuizAnswer>

## 問4

大腸の主な役割として最も適切なものはどれですか。

1. タンパク質をアミノ酸に分解する
2. 胆汁を分泌して脂肪を乳化する
3. 栄養素の大部分を吸収する
4. 水分やミネラルを吸収し、便を固形にする


<QuizAnswer>

**答え: 4**

大腸の主な役割は、小腸から送られてきた液状の内容物から水分やミネラルを吸収し、固形の便にして肛門へ運ぶことです。栄養素の大部分の消化・吸収は小腸で行われており、大腸では主に水分の吸収と便の形成が行われます。

</QuizAnswer>

## 問5

腸の「ぜんどう運動（蠕動運動）」の説明として正しいものはどれですか。

1. 腸の筋肉が収縮と弛緩を繰り返し、内容物を肛門方向へ送る運動
2. 腸の粘膜が振動して栄養素を吸収する運動
3. 腸内の細菌が活発に動くことで起こる運動
4. 横隔膜の上下運動によって腸が刺激される運動


<QuizAnswer>

**答え: 1**

ぜんどう運動とは、腸の壁にある筋肉（輪走筋と縦走筋）が波のように収縮と弛緩を繰り返すことで、食物や消化物を口側から肛門側へ送り出す運動です。この運動は腸管神経系によって自律的にコントロールされており、意識しなくても自動的に行われます。

</QuizAnswer>

## 問6

腸が「第二の脳」と呼ばれる最も大きな理由はどれですか。

1. 腸の形状が脳のシワに似ているため
2. 腸に約1億個以上の神経細胞があり、脳からの指令がなくても自律的に機能できるため
3. 腸で記憶や思考が行われることが証明されたため
4. 腸の重さが脳とほぼ同じであるため


<QuizAnswer>

**答え: 2**

腸には約1億個以上の神経細胞が存在し、これは脳に次いで体内で最も多い数です。腸は独自の腸管神経系を持ち、脳からの指令がなくても消化活動やぜんどう運動を自律的に調整できることから「第二の脳」と呼ばれています。

</QuizAnswer>

## 問7

「脳腸相関」の説明として正しいものはどれですか。

1. 脳の病気が必ず腸の病気を引き起こすという一方向の関係
2. 腸の状態が脳に影響を与えるが、脳から腸への影響はない
3. 脳と腸が自律神経やホルモンなどを介して双方向に情報を送り合う関係
4. 脳と腸が血管で直接つながっている構造のこと


<QuizAnswer>

**答え: 3**

脳腸相関とは、脳と腸が自律神経やホルモン、サイトカインなどを介して双方向に信号を送り合い、互いの機能に影響を及ぼし合うことを指します。例えばストレスを感じるとお腹が痛くなるのは脳から腸への影響であり、腸の状態が気分に影響するのは腸から脳への影響です。

</QuizAnswer>

## 問8

体内で作られるセロトニンのうち、腸で産生される割合として最も近いものはどれですか。

1. 約10％
2. 約30％
3. 約50％
4. 約90％


<QuizAnswer>

**答え: 4**

体内のセロトニンの約90％以上は腸（主に小腸粘膜）で産生されています。セロトニンは「幸せホルモン」とも呼ばれ、精神の安定や睡眠に関わる重要な神経伝達物質です。このことも腸が「第二の脳」と呼ばれる理由の一つです。

</QuizAnswer>

## 問9

小腸の免疫器官である「パイエル板」の説明として正しいものはどれですか。

1. 小腸の絨毛の先端にある消化酵素を分泌する組織
2. 主に回腸に存在し、M細胞が抗原を取り込んで免疫細胞に情報を伝える器官
3. 大腸にのみ存在する水分吸収に特化した組織
4. 十二指腸で胆汁を受け取るための開口部


<QuizAnswer>

**答え: 2**

パイエル板は主に空腸から回腸にかけて点在する免疫器官で、絨毛が存在しない平坦な領域です。パイエル板にあるM細胞が腸管内の細菌などの抗原を取り込み、T細胞やB細胞などの免疫細胞に情報を伝えることで、病原体の体内への侵入を防ぐ役割を果たしています。

</QuizAnswer>

## 問10

体全体の免疫細胞のうち、腸に集中している割合として最も近いものはどれですか。

1. 約20％
2. 約40％
3. 約70％
4. 約95％


<QuizAnswer>

**答え: 3**

体全体の免疫細胞の約70％が腸に集中しているとされています。腸は食べ物とともに外部から侵入する細菌やウイルスと常に接触する器官であるため、パイエル板をはじめとする強力な免疫システムが備わっています。腸は体内最大の免疫器官と呼ばれています。

</QuizAnswer>
