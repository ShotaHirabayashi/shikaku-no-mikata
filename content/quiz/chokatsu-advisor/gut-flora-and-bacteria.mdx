## 問1

腸内に生息する細菌が、種類ごとにまとまって腸壁に張り付いている様子がお花畑のように見えることから名付けられた名称はどれか。

1. 腸内バイオーム
2. 腸内フローラ
3. 腸内マイクロバイオティクス
4. 腸内コロニー


<QuizAnswer>

**答え: 2**

腸内に棲む細菌が菌種ごとの塊となって腸壁にびっしりと張り付いている様子が、品種ごとに並んで咲くお花畑（flora）に見えることから「腸内フローラ」と呼ばれるようになりました。正式名称は「腸内細菌叢（ちょうないさいきんそう）」です。ヒトの腸内には約1,000種類、100兆〜1,000兆個もの細菌が生息しています。

</QuizAnswer>

## 問2

腸内細菌における善玉菌・悪玉菌・日和見菌の理想的なバランスとして正しいものはどれか。

1. 3:1:6
2. 1:1:8
3. 2:1:7
4. 4:1:5


<QuizAnswer>

**答え: 3**

腸内細菌の理想的なバランスは「善玉菌2：悪玉菌1：日和見菌7」とされています。善玉菌が2割存在する腸内環境では悪玉菌の働きが抑制され、腸内が酸性に保たれます。日和見菌は全体の約7割を占め、善玉菌が優勢であれば善玉菌に味方し、悪玉菌が優勢になると悪玉菌に加勢する特徴があります。

</QuizAnswer>

## 問3

腸内細菌のうち、全体の約7割を占め、優勢な菌の側に味方する性質を持つのはどれか。

1. 善玉菌
2. 悪玉菌
3. 乳酸菌
4. 日和見菌


<QuizAnswer>

**答え: 4**

日和見菌は腸内細菌の約7割を占める最も多い菌群です。日和見菌は善玉菌が優勢なときは善玉菌に加勢して発酵活動を行い、悪玉菌が優勢になると悪玉菌に加勢して腐敗活動を行います。そのため、腸内環境を良好に保つには、善玉菌を増やして日和見菌を味方につけることが重要です。

</QuizAnswer>

## 問4

ビフィズス菌と乳酸菌の生息場所に関する記述として正しいものはどれか。

1. ビフィズス菌は主に大腸に、乳酸菌は主に小腸に生息する
2. ビフィズス菌は主に小腸に、乳酸菌は主に大腸に生息する
3. どちらも主に小腸に生息する
4. どちらも主に大腸に生息する


<QuizAnswer>

**答え: 1**

ビフィズス菌は酸素のあるところで生育できない「偏性嫌気性菌」であり、酸素がほとんど届かない大腸に主に生息しています。一方、乳酸菌は「通性嫌気性菌」で、ある程度の酸素があっても生育でき、主に小腸に生息しています。大腸内の善玉菌の約99.9％はビフィズス菌が占めています。

</QuizAnswer>

## 問5

ビフィズス菌が乳酸菌と異なる特徴として、乳酸に加えて多く産生する物質はどれか。

1. アルコール
2. アンモニア
3. 酢酸
4. メタンガス


<QuizAnswer>

**答え: 3**

乳酸菌は代謝物として主に「乳酸」をつくり出しますが、ビフィズス菌は乳酸に加えて「酢酸」をより多く産生する点が大きな違いです。酢酸には強い殺菌力があり、悪玉菌の増殖を抑える働きがあります。この酢酸の産生がビフィズス菌の重要な特徴の一つです。

</QuizAnswer>

## 問6

プロバイオティクスの説明として正しいものはどれか。

1. 腸内の有害物質を吸着して排出する薬剤のこと
2. 腸内細菌のエサとなる食品成分のこと
3. 腸内フローラのバランスを改善することで人に有益な作用をもたらす生きた微生物のこと
4. 腸内の悪玉菌を直接殺菌する抗菌物質のこと


<QuizAnswer>

**答え: 3**

プロバイオティクスは「十分量を摂取したときに宿主（ヒト）に有益な作用をもたらす生きた微生物」と定義されています。代表的なものにビフィズス菌や乳酸菌があります。なお、選択肢2の「腸内細菌のエサとなる食品成分」はプレバイオティクスの説明であり、混同しないよう注意が必要です。

</QuizAnswer>

## 問7

プレバイオティクスに該当するものとして最も適切なのはどれか。

1. ビフィズス菌
2. 乳酸菌
3. オリゴ糖
4. 酪酸菌


<QuizAnswer>

**答え: 3**

プレバイオティクスは、善玉菌のエサとなることで腸内の有用菌を増やし育てる食品成分のことです。代表的なものにオリゴ糖や食物繊維があります。選択肢1・2・4はいずれも菌（生きた微生物）そのものであり、プロバイオティクスに該当します。

</QuizAnswer>

## 問8

プロバイオティクスとプレバイオティクスを組み合わせて摂取する方法を何というか。

1. マルチバイオティクス
2. シンバイオティクス
3. ポストバイオティクス
4. メタバイオティクス


<QuizAnswer>

**答え: 2**

シンバイオティクスとは、プロバイオティクス（生きた有用菌）とプレバイオティクス（有用菌のエサとなる成分）を同時に摂取する方法です。例えば、ヨーグルト（プロバイオティクス）とオリゴ糖（プレバイオティクス）を一緒に摂ることで、腸内環境をより効果的に整えることができると考えられています。

</QuizAnswer>

## 問9

悪玉菌の代表的な菌として正しい組み合わせはどれか。

1. ウェルシュ菌・大腸菌（有毒株）・黄色ブドウ球菌
2. ビフィズス菌・アシドフィルス菌・フェカリス菌
3. バクテロイデス・連鎖球菌・無毒株大腸菌
4. 乳酸桿菌・ガセリ菌・ラクトコッカス菌


<QuizAnswer>

**答え: 1**

悪玉菌の代表格にはウェルシュ菌、大腸菌（有毒株）、黄色ブドウ球菌などがあります。これらは腸内でタンパク質を腐敗させ、アンモニアや硫化水素などの有害物質を産生します。選択肢2・4は善玉菌の例、選択肢3は日和見菌の例です。

</QuizAnswer>

## 問10

腸内細菌を育てる方法に関する記述として、誤っているものはどれか。

1. プロバイオティクスで善玉菌そのものを外から補給する
2. プレバイオティクスで善玉菌のエサとなる成分を摂取する
3. 抗生物質を日常的に服用して悪玉菌を減らす
4. 善玉菌を含む食品と善玉菌のエサを組み合わせて摂取する


<QuizAnswer>

**答え: 3**

抗生物質は悪玉菌だけでなく善玉菌も含め腸内細菌を広く殺菌してしまうため、日常的に服用すると腸内フローラのバランスが大きく崩れます。腸内細菌を育てるには、プロバイオティクス（善玉菌そのもの）の摂取やプレバイオティクス（善玉菌のエサ）の摂取、さらに両者を組み合わせたシンバイオティクスが効果的です。

</QuizAnswer>
