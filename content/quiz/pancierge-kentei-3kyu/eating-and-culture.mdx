## 問1

パンを保存する際、最も避けるべき保存方法はどれか。

1. 常温で密閉袋に入れて保存する
2. 冷凍庫で保存する
3. 冷蔵庫で保存する
4. パンケースに入れて常温で保存する


<QuizAnswer>

**答え: 3**

冷蔵庫の温度帯（0〜5℃）はデンプンの老化（ベータ化／β化）が最も進みやすい温度帯であり、パンが硬くなりやすいため、冷蔵保存は最も避けるべき方法です。すぐに食べない場合は、1個ずつラップで包んで冷凍保存するのが推奨されます。

</QuizAnswer>

## 問2

硬くなったフランスパンを温め直す際、外はカリッと中はふんわりに仕上げるために有効な方法はどれか。

1. 電子レンジで2分加熱する
2. 霧吹きで表面に水を吹きかけてからトースターで焼く
3. 油を塗ってからフライパンで焼く
4. 自然解凍して常温に戻す


<QuizAnswer>

**答え: 2**

硬くなったパンや冷凍パンを温め直す際は、霧吹きで表面に水分を与えてからトースターで焼くと、外はカリッと中はふんわりとした食感に仕上がります。電子レンジだけでは表面がべたつきやすく、カリッとした食感は得られません。

</QuizAnswer>

## 問3

フランス料理のテーブルマナーとして、パンの正しい食べ方はどれか。

1. ナイフで一口大に切って食べる
2. 手で一口大にちぎって食べる
3. フォークに刺して食べる
4. そのままかじって食べる


<QuizAnswer>

**答え: 2**

フランス料理のマナーでは、パンはナイフで切るのではなく、手で一口大にちぎって食べるのが正しい作法です。バターを塗る場合も、まとめて塗るのではなく、ちぎった一片ごとにバターを塗って食べるのがマナーとされています。

</QuizAnswer>

## 問4

サンドイッチの名称の由来とされる人物は誰か。

1. フランスの料理人サンドウィッチ氏
2. ドイツのパン職人サンドウィッチ氏
3. イギリスのサンドウィッチ伯爵
4. イタリアの貴族サンドウィッチ公爵


<QuizAnswer>

**答え: 3**

サンドイッチは、18世紀イギリスの第4代サンドウィッチ伯爵ジョン・モンタギューに由来するとされています。カードゲームに熱中するあまり食事の手を止めたくなかった伯爵が、パンに具材を挟んで片手で食べられるようにしたのが始まりと伝えられています。

</QuizAnswer>

## 問5

「クロックムッシュ」の上に目玉焼きを乗せたフランス発祥の料理を何というか。

1. クロックマダム
2. ガレット
3. タルティーヌ
4. パニーニ


<QuizAnswer>

**答え: 1**

クロックムッシュはハムとチーズを挟みベシャメルソースをかけて焼いたフランスの温製サンドイッチです。その上に目玉焼きを乗せたものがクロックマダムと呼ばれます。「ムッシュ（紳士）」「マダム（婦人）」という名前の対比もフランスらしい特徴です。

</QuizAnswer>

## 問6

薄くスライスしたパンをカリカリに焼き上げた、スープやパテの添え物として使われるトーストの名称はどれか。

1. シナモントースト
2. メルバトースト
3. フレンチトースト
4. ラスク


<QuizAnswer>

**答え: 2**

メルバトーストは、薄くスライスしたパンをカリカリに焼き上げたもので、スープやパテなどの添え物として使われます。19世紀のオーストラリア出身のオペラ歌手ネリー・メルバに由来する名前で、ホテルの料理人が彼女のために考案したとされています。

</QuizAnswer>

## 問7

フランスで「パン・ペルデュ（pain perdu）」と呼ばれる料理は、日本では一般的に何と呼ばれているか。

1. クルトン
2. ラスク
3. フレンチトースト
4. ブルスケッタ


<QuizAnswer>

**答え: 3**

フレンチトーストはフランスでは「パン・ペルデュ（pain perdu＝失われたパン）」と呼ばれます。硬くなったパンを卵・牛乳・砂糖を混ぜた液に浸してバターで焼く料理で、もともと古くなったパンを無駄にしない知恵から生まれた食べ方です。

</QuizAnswer>

## 問8

イタリア発祥で、焼いたパンにニンニクを擦り付けオリーブオイルをかけ、トマトなどの具材を乗せた料理はどれか。

1. カナッペ
2. タルティーヌ
3. ブルスケッタ
4. クロスティーニ


<QuizAnswer>

**答え: 3**

ブルスケッタはイタリア発祥の料理で、焼いたパンにニンニクを擦り付け、オリーブオイルをかけた上にトマトやバジルなどの具材を乗せて食べます。カナッペはフランス発祥の一口サイズの前菜、タルティーヌはフランスのオープンサンドで、それぞれ異なる特徴を持っています。

</QuizAnswer>

## 問9

JAS規格において「ジャム」と「マーマレード」を区別する最大の特徴はどれか。

1. 砂糖の使用量の違い
2. 柑橘類の果皮が含まれているかどうか
3. 加熱時間の長さ
4. 果実の原産国の違い


<QuizAnswer>

**答え: 2**

JAS規格では、マーマレードは柑橘類の果皮が含まれているジャム類と定義されています。一般的なジャムは果実に砂糖を加えて加熱濃縮したものですが、マーマレードは柑橘類の果皮の存在が最大の特徴であり、独特のほろ苦さと香りを生み出しています。

</QuizAnswer>

## 問10

バター系のクロワッサンやブリオッシュに最も相性が良いとされる飲み物はどれか。

1. 緑茶
2. 炭酸水
3. カフェオレ
4. ブラックコーヒー（深煎り）


<QuizAnswer>

**答え: 3**

バターを豊富に使用したクロワッサンやブリオッシュには、ミルクのまろやかさが加わったカフェオレが相性抜群とされています。フランスの朝食でもクロワッサンとカフェオレは定番の組み合わせです。ブラックコーヒーの深煎りはハード系のパンとの相性が良いとされています。

</QuizAnswer>
