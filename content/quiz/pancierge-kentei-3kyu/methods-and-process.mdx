## 問1

パンの製法のうち、すべての材料を一度にミキシングして作る最もシンプルな方法を何というか。

1. 中種法
2. ポーリッシュ法
3. ストレート法（直捏法）
4. サワー種法


<QuizAnswer>

**答え: 3**

ストレート法（直捏法）は、すべての材料を最初から一度に混ぜ合わせて生地を作る、最も基本的でシンプルな製パン法です。工程が短く全体で4〜5時間程度で完成しますが、発酵耐性が低く老化が早いというデメリットもあります。

</QuizAnswer>

## 問2

中種法において、中種に使用する小麦粉の割合として最も一般的なものはどれか。

1. 30〜40%
2. 50〜60%
3. 60〜70%
4. 90〜100%


<QuizAnswer>

**答え: 3**

中種法では、全体の小麦粉のうち60〜70%を酵母・水と合わせて先に発酵させます。特に70%中種法が最も一般的です。この方法により発酵耐性が高まり、安定した品質のパンが作れるようになります。

</QuizAnswer>

## 問3

ポーリッシュ法（液種法）の液種における小麦粉と水の配合比率として正しいものはどれか。

1. 小麦粉2：水1
2. 小麦粉1：水1（同量）
3. 小麦粉1：水2
4. 小麦粉3：水1


<QuizAnswer>

**答え: 2**

ポーリッシュ法の液種は、小麦粉と水を同量（1:1）で混ぜ合わせて作ります。水分比率が100%のため液状になるのが特徴で、この液種を発酵させてから本捏ねに加えることで、風味豊かでクラストの薄いパリッとしたパンに仕上がります。

</QuizAnswer>

## 問4

製パンの基本工程として正しい順序はどれか。

1. 計量→ミキシング→分割→一次発酵→成形→最終発酵→焼成
2. 計量→ミキシング→一次発酵→分割→丸め→ベンチタイム→成形→最終発酵→焼成
3. 計量→ミキシング→一次発酵→成形→分割→ベンチタイム→最終発酵→焼成
4. 計量→一次発酵→ミキシング→分割→丸め→成形→最終発酵→焼成


<QuizAnswer>

**答え: 2**

製パンの基本工程は「計量→ミキシング→一次発酵→（パンチ）→分割→丸め→ベンチタイム→成形→最終発酵（ホイロ）→焼成」の順です。一次発酵の後に分割・丸めを行い、ベンチタイムで生地を休ませてから成形するという流れが正しい順序です。

</QuizAnswer>

## 問5

ベンチタイムの主な目的として最も適切なものはどれか。

1. 酵母を活性化させて発酵を促進する
2. 分割・丸めで緊張したグルテンを弛緩させ、成形しやすくする
3. 生地の水分を飛ばして扱いやすくする
4. 生地に焼き色がつきやすいよう表面を乾燥させる


<QuizAnswer>

**答え: 2**

ベンチタイムは、分割・丸めの作業で締まったグルテンを緩め（弛緩させ）、生地を成形しやすい状態に戻すために行います。一般的に15〜20分程度置き、その間は乾燥を防ぐために布やビニールをかぶせておきます。

</QuizAnswer>

## 問6

パンチ（ガス抜き）の目的として正しくないものはどれか。

1. 大きな気泡を抜いてきめ細かいクラムにする
2. グルテンに刺激を与えてガス保持力を高める
3. 酵母に酸素を供給して活性を促す
4. 生地の水分量を減らして固くする


<QuizAnswer>

**答え: 4**

パンチの目的は、ガスの均一化、グルテンの強化、酵母への酸素供給、生地温度の均一化などです。生地の水分量を減らすことはパンチの目的ではありません。パンチは一次発酵の途中で行い、発酵をさらに促進させる効果があります。

</QuizAnswer>

## 問7

最終発酵（ホイロ）の温度と湿度の目安として最も適切なものはどれか。

1. 温度25〜28℃、湿度50〜60%
2. 温度30〜33℃、湿度60〜70%
3. 温度35〜38℃、湿度75〜85%
4. 温度40〜45℃、湿度90〜95%


<QuizAnswer>

**答え: 3**

最終発酵（ホイロ）は、温度35〜38℃、湿度75〜85%の環境で行うのが一般的です。この条件で40〜60分程度発酵させ、生地が型の8〜9分目まで膨らんだら発酵完了の目安となります。温度が高すぎると過発酵になるため注意が必要です。

</QuizAnswer>

## 問8

ミキシングにおいて、生地がまとまりボウルから離れ始め、粉っぽさが消える段階を何というか。

1. つかみ取り段階（ピックアップステージ）
2. 水切れ段階（クリーンアップステージ）
3. 結合段階（デベロップメントステージ）
4. 破壊段階（レットダウンステージ）


<QuizAnswer>

**答え: 2**

水切れ段階（クリーンアップステージ）は、材料が十分に混ざり合い、生地がまとまってボウルの壁から離れ始める段階です。その前のつかみ取り段階ではまだベタベタした状態で、水切れ段階を経て結合段階へ進むとグルテンが発達し弾力が出てきます。

</QuizAnswer>

## 問9

一般的なパン生地のこね上げ温度の目安として最も適切なものはどれか。

1. 20〜22℃
2. 27〜28℃
3. 33〜35℃
4. 40〜42℃


<QuizAnswer>

**答え: 2**

一般的なパン生地のこね上げ温度は27〜28℃が目安です。この温度が適正でないと発酵に影響が出るため、仕込み水の温度で調整します。なお、フランスパンなどは24〜25℃とやや低めに設定し、ゆっくり発酵させることで風味を引き出します。

</QuizAnswer>

## 問10

中種法の特徴として正しいものはどれか。

1. 全工程がストレート法より短時間で完了する
2. 発酵耐性が高く、パンの老化が遅い
3. 小麦粉と同量の水で液状の種を作る
4. 乳酸菌の力で生地に酸味を加える


<QuizAnswer>

**答え: 2**

中種法は、あらかじめ材料の一部で中種を作って発酵させるため、発酵耐性が高く安定した品質のパンが作れます。また保水性が高まるため老化が遅くなるのも大きな利点です。選択肢1はストレート法の特徴、3はポーリッシュ法、4はサワー種法の説明です。

</QuizAnswer>
