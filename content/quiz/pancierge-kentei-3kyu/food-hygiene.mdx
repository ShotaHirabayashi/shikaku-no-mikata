## 問1

食中毒予防の三原則として正しいものはどれか。

1. つけない・増やさない・やっつける
2. 洗う・冷やす・加熱する
3. 消毒する・密封する・冷凍する
4. 分ける・包む・温める


<QuizAnswer>

**答え: 1**

食中毒予防の三原則は「つけない（清潔）」「増やさない（迅速・冷却）」「やっつける（加熱）」です。食品に菌をつけないよう手洗いや器具の洗浄を行い、低温保存で菌の増殖を抑え、十分な加熱で菌を死滅させることが基本となります。パン製造の現場でも、この三原則に基づいた衛生管理が求められます。

</QuizAnswer>

## 問2

黄色ブドウ球菌による食中毒の特徴として正しいものはどれか。

1. 潜伏期間が2〜5日と長い
2. 産生する毒素（エンテロトキシン）は加熱しても分解されない
3. 主に魚介類の生食が原因で発生する
4. アルコール消毒では死滅しない


<QuizAnswer>

**答え: 2**

黄色ブドウ球菌が産生するエンテロトキシンは耐熱性が非常に高く、100℃で加熱しても分解されません。そのため、菌が増殖して毒素を産生してしまうと、その後の加熱調理では食中毒を防げません。手指の傷や化膿した部分に多く存在するため、パン製造時には手袋の着用が重要です。

</QuizAnswer>

## 問3

食品の変質のうち、「腐敗」の説明として正しいものはどれか。

1. 空気中の酸素により油脂が酸化すること
2. 微生物の作用により炭水化物が分解されること
3. 微生物の作用によりタンパク質が分解されること
4. 紫外線により食品中のビタミンが破壊されること


<QuizAnswer>

**答え: 3**

腐敗とは、微生物の作用によってタンパク質が分解される現象で、アンモニアや硫化水素などの悪臭物質が生じます。一方、微生物が炭水化物を分解する現象は「変敗」、油脂が酸素によって酸化する現象は「酸敗（酸化）」と呼ばれます。それぞれ原因と対象成分が異なる点を区別しておくことが重要です。

</QuizAnswer>

## 問4

ノロウイルスによる食中毒の対策として正しいものはどれか。

1. 70%エタノールによる消毒が最も効果的である
2. 食品の中心温度を60℃で10分間加熱すれば十分である
3. 85〜90℃で90秒以上加熱する必要がある
4. 酢に漬けることで速やかに不活化できる


<QuizAnswer>

**答え: 3**

ノロウイルスは一般的な細菌と異なり、アルコール消毒の効果が低いことが特徴です。加熱による不活化には85〜90℃で90秒以上が必要とされており、通常の細菌に対する75℃1分間という基準よりも高い温度と時間が求められます。汚染箇所の消毒には次亜塩素酸ナトリウム（1000ppm）が有効です。

</QuizAnswer>

## 問5

低温殺菌（パスチャライゼーション）の条件として正しいものはどれか。

1. 30℃で60分間
2. 63℃で30分間
3. 100℃で15分間
4. 121℃で20分間


<QuizAnswer>

**答え: 2**

低温殺菌（パスチャライゼーション）は63℃で30分間加熱する方法（LTLT法）で、主に牛乳の殺菌に用いられます。高温で短時間殺菌するよりも栄養素や風味が保たれやすいのが特徴です。なお、121℃で15〜20分間は高圧蒸気滅菌（オートクレーブ）の条件で、芽胞を形成する菌にも有効です。

</QuizAnswer>

## 問6

食品の保存において、冷蔵保存の温度として適切なものはどれか。

1. −18℃以下
2. 0〜10℃（5℃以下推奨）
3. 15〜20℃
4. 25〜30℃


<QuizAnswer>

**答え: 2**

冷蔵保存は0〜10℃（一般的には5℃以下が推奨）で保存することにより、微生物の増殖速度を抑制する方法です。ただし、低温でも微生物は完全には死滅せず、ゆっくりと増殖するため、保存期間には限界があります。−18℃以下は冷凍保存の温度帯です。

</QuizAnswer>

## 問7

HACCPに関する記述として正しいものはどれか。

1. 2021年6月より全食品事業者にHACCPに沿った衛生管理が義務化された
2. HACCPは大規模事業者のみに適用される制度である
3. HACCPとは食品の最終製品のみを検査する方式である
4. HACCPは日本独自の衛生管理制度である


<QuizAnswer>

**答え: 1**

HACCP（Hazard Analysis and Critical Control Point＝危害要因分析重要管理点）は、2021年6月1日から全食品事業者に完全義務化されました。小規模事業者には「HACCPの考え方を取り入れた衛生管理」という簡略版が認められています。HACCPは最終製品の検査だけでなく、製造工程全体を通じて危害要因を管理する国際的な衛生管理手法です。

</QuizAnswer>

## 問8

腸炎ビブリオによる食中毒の特徴として正しいものはどれか。

1. 嫌気性で、真空包装された食品で増殖しやすい
2. 海水中に生息し、魚介類が主な原因食品である
3. 乳製品が主な感染源で、冬季に多発する
4. 少量の菌（数十個程度）でも発症する


<QuizAnswer>

**答え: 2**

腸炎ビブリオは海水中に生息する細菌で、魚介類の生食による食中毒の原因として知られています。増殖速度が非常に速いことが特徴ですが、真水や酸に弱い性質があります。夏場に多く発生するため、魚介類は低温で保存し、真水でよく洗うことが予防に効果的です。

</QuizAnswer>

## 問9

食品表示法で定められたアレルゲンの特定原材料（表示義務のある品目）に含まれないものはどれか。

1. 小麦
2. 卵
3. 大豆
4. えび


<QuizAnswer>

**答え: 3**

特定原材料として表示が義務づけられているのは、卵・乳・小麦・そば・落花生・えび・かに・くるみの8品目です。大豆は「特定原材料に準ずるもの」として表示が推奨されていますが、義務ではありません。パンには小麦・卵・乳が使われることが多く、アレルゲン表示は特に重要です。

</QuizAnswer>

## 問10

パン製造現場で、作業者の手指に傷がある場合の対応として最も適切なものはどれか。

1. 消毒液に手を浸してからそのまま作業する
2. 傷が小さければそのまま作業してよい
3. 耐水性絆創膏を貼り、さらにゴム手袋を着用する
4. 傷のある手は使わず片手で作業する


<QuizAnswer>

**答え: 3**

手指の傷口には黄色ブドウ球菌が多く存在するため、パン製造時には耐水性の絆創膏で傷を覆った上からゴム手袋を着用することが求められます。黄色ブドウ球菌が産生する毒素は加熱しても分解されないため、菌を食品につけないことが最も重要な予防策です。傷が小さくても素手での作業は避けなければなりません。

</QuizAnswer>
