## 問1

世界で最初に「発酵パン」が作られたとされる古代文明はどれか。

1. 古代メソポタミア
2. 古代エジプト
3. 古代ギリシャ
4. 古代ローマ


<QuizAnswer>

**答え: 2**

古代メソポタミアでは小麦粉と水を混ぜて焼いた無発酵のパン（平焼きパン）が作られていましたが、偶然の自然発酵により「発酵パン」が誕生したのは古代エジプトとされています。古代エジプトではパンが主食であるだけでなく、神への供物や給料の代わりとしても使われていました。

</QuizAnswer>

## 問2

日本語の「パン」という言葉は、どの国の言語に由来するか。

1. オランダ語
2. スペイン語
3. ポルトガル語
4. フランス語


<QuizAnswer>

**答え: 3**

「パン」はポルトガル語の「pão（パォン）」に由来します。1543年にポルトガル人が種子島に鉄砲を伝えた際、パンも一緒に日本へもたらされました。当時の南蛮貿易を通じて、ポルトガル語由来の言葉が多く日本に伝わり、「パン」もその一つです。

</QuizAnswer>

## 問3

日本にパンが初めて伝来したのは、何年のことか。

1. 1543年
2. 1600年
3. 1639年
4. 1720年


<QuizAnswer>

**答え: 1**

1543年（天文12年）にポルトガル人が種子島に漂着し、鉄砲とともにパンも日本に伝えられました。これが日本におけるパンの歴史の始まりとされています。その後、南蛮貿易が盛んになる安土桃山時代には、キリスト教の宣教師を通じてパン作りが広まりました。

</QuizAnswer>

## 問4

1842年（天保13年）に軍用の「兵糧パン」を製造し、「パンの祖」と呼ばれる人物は誰か。

1. 木村安兵衛
2. 山岡鉄舟
3. 田辺玄平
4. 江川太郎左衛門


<QuizAnswer>

**答え: 4**

伊豆韮山の代官であった江川太郎左衛門（江川英龍）は、外国からの脅威に備えて保存性に優れた兵糧としてパンに着目し、1842年に日本人として初めて本格的にパンを焼きました。この功績から「パンの祖」と呼ばれています。毎年4月12日は「パンの記念日」として制定されています。

</QuizAnswer>

## 問5

1869年（明治2年）に銀座で「木村屋」を創業し、日本のパン文化の礎を築いた人物は誰か。

1. 江川太郎左衛門
2. 木村安兵衛
3. 山岡鉄舟
4. 木村英三郎


<QuizAnswer>

**答え: 2**

木村安兵衛は1869年（明治2年）に東京・銀座に「文英堂」（のちの木村屋）を開業しました。日本人の嗜好に合うパンの開発に取り組み、息子の英三郎とともに酒種を用いたあんパンを考案するなど、日本独自のパン文化を切り開いた人物です。

</QuizAnswer>

## 問6

木村屋の酒種あんパンが明治天皇に献上されたのは何年か。

1. 1869年（明治2年）
2. 1872年（明治5年）
3. 1875年（明治8年）
4. 1880年（明治13年）


<QuizAnswer>

**答え: 3**

1875年（明治8年）4月4日、山岡鉄舟の仲介により、木村屋の桜あんパンが明治天皇に献上されました。これがきっかけで木村屋は宮中御用達となり、あんパンは日本中に広まりました。この献上日にちなんで、4月4日は「あんパンの日」とされています。

</QuizAnswer>

## 問7

明治天皇への酒種あんパン献上を仲介した人物は誰か。

1. 山岡鉄舟
2. 勝海舟
3. 福沢諭吉
4. 渋沢栄一


<QuizAnswer>

**答え: 1**

山岡鉄舟は幕末から明治にかけて活躍した政治家・剣術家で、明治天皇の侍従を務めていました。木村安兵衛と親交があり、その縁で木村屋の酒種あんパンを明治天皇に献上する機会を取り持ちました。この献上が成功し、あんパンの名声が全国に広まるきっかけとなりました。

</QuizAnswer>

## 問8

江戸時代、鎖国政策のもとでパンの製造が細々と続けられていたのは主にどこか。

1. 京都
2. 江戸
3. 長崎の出島
4. 横浜


<QuizAnswer>

**答え: 3**

江戸時代の鎖国政策により、パンの文化は日本国内でほぼ途絶えましたが、長崎の出島ではオランダ人向けにパンが細々と製造され続けていました。出島は鎖国下でも唯一西洋との交流が許された場所であり、西洋の食文化がわずかに残る拠点でした。

</QuizAnswer>

## 問9

戦後の日本でパン食が急速に普及した要因として、最も関係が深いのはどれか。

1. フランスからの技術指導
2. アメリカの余剰小麦と学校給食
3. ドイツ人パン職人の招聘
4. 国産小麦の大増産


<QuizAnswer>

**答え: 2**

第二次世界大戦後、深刻な食糧難に対してGHQ（連合国軍最高司令官総司令部）やLARA（アジア救済公認団体）による食糧援助が行われ、アメリカの余剰小麦が大量に日本に送られました。この小麦を使って学校給食でパンが提供されるようになり、コッペパンと脱脂粉乳の組み合わせが定番となって、日本人のパン食習慣が定着していきました。

</QuizAnswer>

## 問10

日本において「学校給食法」が制定されたのは何年か。

1. 1947年（昭和22年）
2. 1950年（昭和25年）
3. 1952年（昭和27年）
4. 1954年（昭和29年）


<QuizAnswer>

**答え: 4**

学校給食法は1954年（昭和29年）6月3日に公布・施行されました。戦後まもなくから始まっていた学校給食を法律で制度化したもので、これによりパンを中心とした給食が全国の学校に広がりました。学校給食は日本の子どもたちにパン食を定着させる大きな役割を果たしました。

</QuizAnswer>
