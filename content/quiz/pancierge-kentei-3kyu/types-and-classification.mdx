## 問1

フランスパンの一種で、名前が「棒」や「杖」を意味し、長さ約60〜80cmの細長い形状が特徴的なパンはどれか。

1. バタール
2. フィセル
3. バゲット
4. エピ


<QuizAnswer>

**答え: 3**

バゲット（baguette）はフランス語で「棒・杖」を意味し、長さ約60〜80cmの細長い形状が特徴です。小麦粉・水・塩・イーストの基本4材料だけで作られるリーンなパンの代表格で、表面には7本程度のクープ（切れ目）が入れられます。

</QuizAnswer>

## 問2

パンの分類において、小麦粉・水・塩・イーストの基本材料のみで作られ、油脂や糖類をほとんど使わないパンの総称として正しいものはどれか。

1. ソフトなパン
2. リッチなパン
3. ハードなパン
4. リーンなパン


<QuizAnswer>

**答え: 4**

リーン（lean）とは「脂肪のない、やせた」という意味で、小麦粉・水・塩・イーストの基本4材料を中心に作られるパンを指します。バゲットやカンパーニュが代表例です。反対に、バター・卵・砂糖・乳製品を豊富に使うパンは「リッチなパン」と呼ばれ、クロワッサンやブリオッシュがその代表です。

</QuizAnswer>

## 問3

ドイツのパンに関する記述として正しいものはどれか。

1. ドイツのパンの種類は約300種類といわれる
2. プンパーニッケルは白い小麦粉のみで作られる軽いパンである
3. ドイツには約1500種類以上のパンがあるといわれる
4. ドイツではライ麦よりも小麦の使用量が圧倒的に多い


<QuizAnswer>

**答え: 3**

ドイツは世界で最もパンの種類が多い国として知られ、約1500種類以上のパンがあるといわれています。北部では冷涼な気候に適したライ麦を使ったパンが多く、サワー種（サワードウ）を使った酸味のあるパンが特徴的です。プンパーニッケルは粗挽きライ麦を90%以上使い、16時間以上の長時間焼成で作る黒くずっしりとしたパンです。

</QuizAnswer>

## 問4

イタリアのパン「フォカッチャ」の特徴として正しいものはどれか。

1. 表面にオリーブオイルを塗り、指でくぼみをつけて焼く平焼きパン
2. 細長いスティック状で、前菜として食べるパン
3. パネトーネ種を使ったドーム型のクリスマスパン
4. スリッパの形に似た大きな気泡が特徴のパン


<QuizAnswer>

**答え: 1**

フォカッチャはイタリアの代表的な平焼きパンで、表面にオリーブオイルをたっぷり塗り、指で押してくぼみをつけてから焼くのが特徴です。語源はラテン語の「フォーカス（囲炉裏）」に由来し、古代ローマ時代から食べられていたとされます。なお、選択肢2はグリッシーニ、3はパネトーネ、4はチャバタの説明です。

</QuizAnswer>

## 問5

インドの代表的なパンである「ナン」の焼き方として正しいものはどれか。

1. 鉄板（タワー）の上で焼く
2. タンドール（壺型窯）の内壁に貼り付けて焼く
3. オーブンの天板に並べて焼く
4. 油で揚げて作る


<QuizAnswer>

**答え: 2**

ナンはインドの発酵平焼きパンで、タンドールと呼ばれる壺型の窯の内壁に生地を貼り付けて高温で焼き上げます。一方、同じインドの平焼きパンであるチャパティは酵母を使わない無発酵パンで、鉄板（タワー）で焼きます。インドの家庭では日常的にチャパティが食べられ、ナンは外食やレストランで食べることが多いとされています。

</QuizAnswer>

## 問6

日本で初めてあんパンを開発し、明治天皇にも献上したことで知られるパン店はどれか。

1. 新宿中村屋
2. 木村屋総本店
3. 名花堂
4. 敷島製パン


<QuizAnswer>

**答え: 2**

あんパンは1874年（明治7年）に木村屋総本店が開発した日本独自のパンです。酒種（酒粕由来の酵母）を使って発酵させた生地であんこを包むという画期的な製品で、翌1875年に明治天皇に献上されたことで爆発的な人気を得ました。なお、新宿中村屋はクリームパン、名花堂はカレーパンの元祖として知られています。

</QuizAnswer>

## 問7

クロワッサン（croissant）の名前の意味として正しいものはどれか。

1. 黄金色
2. 三日月
3. 巻き貝
4. 王冠


<QuizAnswer>

**答え: 2**

クロワッサン（croissant）はフランス語で「三日月」を意味します。その起源はオーストリアにあるとされ、1683年のウィーン包囲戦でオスマン帝国軍を撃退した記念に、オスマン帝国の象徴である三日月の形をしたパンが作られたという逸話があります。バターを折り込んだ発酵生地から作られ、サクサクとした層状の食感が特徴のリッチなパンです。

</QuizAnswer>

## 問8

イタリアのパン「グリッシーニ」の特徴として正しいものはどれか。

1. ドーム型でドライフルーツがたっぷり入った菓子パン
2. 大きな気泡があり、もちもちとした食感の四角い平たいパン
3. 細長いスティック状のパンで、ピエモンテ州トリノ発祥である
4. オリーブオイルをたっぷり使った丸い平焼きパン


<QuizAnswer>

**答え: 3**

グリッシーニはイタリア・ピエモンテ州トリノ発祥の細長いスティック状のパンです。17世紀に病弱な公爵のために宮廷医が消化の良いパンとして考案したのが起源とされています。カリカリとした食感で、イタリアンレストランでは前菜やワインのおつまみとして提供されることが多いパンです。

</QuizAnswer>

## 問9

中東の平焼きパン「ピタ」の最大の特徴はどれか。

1. 表面に粗い岩塩がまぶしてある
2. 高温で焼くことで内部がポケット状の空洞になる
3. 表面にバターを塗って何層にも折り重ねる
4. トウモロコシの粗挽き粉をまぶして焼く


<QuizAnswer>

**答え: 2**

ピタは中東発祥の平焼きパンで、約300度の高温で一気に焼くことにより、内部の水分が蒸発して生地が膨らみ、中がポケット状の空洞になるのが最大の特徴です。この空洞に肉や野菜などの具材を詰めて食べるのが一般的で、ケバブサンドなどに広く使われています。

</QuizAnswer>

## 問10

パンの分類で、バター・卵・砂糖・乳製品を豊富に使い、コクのある濃厚な味わいが特徴のパンに該当しないものはどれか。

1. クロワッサン
2. ブリオッシュ
3. パン・ド・カンパーニュ
4. デニッシュ


<QuizAnswer>

**答え: 3**

パン・ド・カンパーニュはフランス語で「田舎パン」を意味し、小麦粉にライ麦粉を混ぜて作る大型のリーンなパンです。バターや卵をほとんど使わず、素朴な味わいと日持ちの良さが特徴です。一方、クロワッサン・ブリオッシュ・デニッシュはいずれもバター・卵・砂糖などを豊富に使った「リッチなパン」の代表格です。

</QuizAnswer>
