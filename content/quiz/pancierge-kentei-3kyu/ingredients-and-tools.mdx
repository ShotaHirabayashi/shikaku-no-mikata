## 問1

パン作りに最も一般的に使用される小麦粉はどれか。

1. 薄力粉
2. 中力粉
3. 強力粉
4. 準強力粉


<QuizAnswer>

**答え: 3**

強力粉はタンパク質含有量が11.5〜13.0%と最も多く、水を加えてこねると強いグルテンが形成されるため、パン作りに最も適しています。グルテンの網目構造が発酵で生じた炭酸ガスをしっかり包み込み、パンをふっくら膨らませます。

</QuizAnswer>

## 問2

小麦粉に含まれるグリアジンとグルテニンが水と結合してできるものは何か。

1. デンプン
2. グルテン
3. カゼイン
4. レシチン


<QuizAnswer>

**答え: 2**

小麦粉中のグリアジン（粘着性を持つタンパク質）とグルテニン（弾力性を持つタンパク質）が水を加えてこねることで結合し、網目状のグルテンが形成されます。グリアジンは生地の伸びやすさ、グルテニンは弾力を担い、この2つが合わさることでパン生地特有の粘弾性が生まれます。

</QuizAnswer>

## 問3

ドライイーストとインスタントドライイーストの最も大きな違いはどれか。

1. インスタントドライイーストは冷凍保存が必要である
2. ドライイーストは使用前に予備発酵が必要である
3. インスタントドライイーストは生イーストと同じ水分量である
4. ドライイーストは天然酵母の一種である


<QuizAnswer>

**答え: 2**

ドライイーストは粒が大きく、使用前にぬるま湯に溶かして予備発酵させる必要があります。一方、インスタントドライイーストは顆粒が細かく、予備発酵不要で直接粉に混ぜて使うことができます。この手軽さから、家庭での製パンではインスタントドライイーストが最も広く使われています。

</QuizAnswer>

## 問4

製パンにおける塩の役割として、正しくないものはどれか。

1. グルテンを引き締め、生地に弾力を与える
2. イーストの発酵を促進し、発酵時間を短縮する
3. パンの風味を向上させ、味を引き締める
4. 過発酵を防ぐ効果がある


<QuizAnswer>

**答え: 2**

塩はイーストの発酵を促進するのではなく、逆に発酵を抑制する働きがあります。これにより過発酵を防ぎ、発酵をコントロールする役割を果たします。また、グルテンを引き締めて生地にコシを与え、パンの風味を向上させる効果もあります。一般的な配合量はベーカーズパーセントで約2%です。

</QuizAnswer>

## 問5

砂糖を多く配合したパン生地（対粉12%以上）を作る際に使用するイーストはどれか。

1. 生イースト
2. 天然酵母
3. 耐糖性イースト
4. ドライイースト


<QuizAnswer>

**答え: 3**

砂糖の配合量が対粉12%以上になると、浸透圧の影響で通常のイーストは活動が阻害されてしまいます。そのため、高糖度の生地には耐糖性イースト（糖分の多い環境でも発酵力を維持できるイースト）を使用します。菓子パンやブリオッシュなど砂糖を多く使うパンに適しています。

</QuizAnswer>

## 問6

製パンにおけるバターの水分含有量として正しいものはどれか。

1. 約1〜2%
2. 約15〜16%
3. 約30〜35%
4. 約50%


<QuizAnswer>

**答え: 2**

バターの水分含有量は約15〜16%です。残りは乳脂肪（約80%以上）と乳固形分で構成されています。一方、ショートニングは水分をほとんど含まず無味無臭であるため、サクサクとした食感（ショートネス）を出す目的で使われます。バターは風味が豊かで、クロワッサンやブリオッシュに欠かせない材料です。

</QuizAnswer>

## 問7

油脂を生地に加えるタイミングとして最も適切なのはどれか。

1. 最初にすべての材料と一緒に加える
2. 生地をある程度こねてグルテンが形成されてから加える
3. 一次発酵が終わった後に加える
4. 焼成の直前に生地の表面に塗る


<QuizAnswer>

**答え: 2**

油脂にはグルテンの形成を阻害する性質があるため、最初から加えるとグルテンがうまく発達しません。生地をある程度こねてグルテンがしっかり形成されてから油脂を加えることで、グルテンの網目構造を壊すことなく、生地に伸展性やソフトさを付与することができます。これは製パンの基本的なテクニックです。

</QuizAnswer>

## 問8

砂糖を使わないフランスパンなどで、イーストの栄養源を補い焼き色を良くするために使用する材料はどれか。

1. ビタミンC
2. ベーキングパウダー
3. モルト（麦芽エキス）
4. ゼラチン


<QuizAnswer>

**答え: 3**

モルト（麦芽エキス）は大麦の麦芽から抽出されたもので、含まれるアミラーゼという酵素がデンプンを麦芽糖に分解し、イーストの栄養源となります。砂糖を配合しないフランスパンなどのリーンなパンでは特に重要な材料で、クラスト（外皮）の美しい焼き色やツヤを出す効果もあります。

</QuizAnswer>

## 問9

製パンにおいて、生地の分割やカットに使用する金属製の道具は何か。

1. ドレッジ
2. スケッパー
3. めん棒
4. 番重


<QuizAnswer>

**答え: 2**

スケッパーは金属製で持ち手が付いた道具で、生地の分割やカットに使用します。一方、ドレッジ（カード）はプラスチック製で柔軟性があり、ボウルに残った生地をこそぎ取ったり、生地を扱ったりするのに使います。両者は似た用途ですが、素材と主な使い方が異なります。

</QuizAnswer>

## 問10

脱脂粉乳（スキムミルク）が製パンで多く使われる理由として、最も適切なものはどれか。

1. 牛乳より脂肪分が多く、パンがリッチに仕上がるから
2. 保存性が高く計量しやすいため、扱いやすいから
3. イーストの発酵を促進する効果が牛乳より強いから
4. グルテンの形成を助ける特殊なタンパク質を含むから


<QuizAnswer>

**答え: 2**

脱脂粉乳は牛乳から脂肪分と水分を除いて粉末にしたもので、常温で長期保存ができ、粉末なので正確な計量がしやすいという利点があります。また、牛乳のように仕込み水の量を調整する必要がなく、直接粉に混ぜて使えるため、製パンの現場で広く利用されています。乳糖によるメイラード反応で焼き色も良くなります。

</QuizAnswer>
