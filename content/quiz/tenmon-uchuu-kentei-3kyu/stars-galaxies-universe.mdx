## 問1

恒星が一生の大部分を過ごす、水素をヘリウムに変える核融合反応が安定して行われている段階を何というか。

1. 赤色巨星段階
2. 主系列星段階
3. 白色矮星段階
4. 原始星段階


<QuizAnswer>

**答え: 2**

恒星は誕生後、中心部で水素がヘリウムに変わる核融合反応によりエネルギーを生み出し、この安定した段階を主系列星段階と呼ぶ。太陽もこの段階にあり、恒星の一生の約90%をこの段階で過ごす。HR図上では左上から右下に伸びる帯状の領域（主系列）に位置する。

</QuizAnswer>

## 問2

HR図（ヘルツシュプルング・ラッセル図）の縦軸と横軸は、それぞれ何を表しているか。

1. 縦軸：質量、横軸：年齢
2. 縦軸：明るさ（光度）、横軸：表面温度（スペクトル型）
3. 縦軸：距離、横軸：大きさ
4. 縦軸：密度、横軸：自転速度


<QuizAnswer>

**答え: 2**

HR図は縦軸に恒星の明るさ（光度や絶対等級）、横軸に表面温度（またはスペクトル型）をとった図である。横軸は高温が左、低温が右に配置される点が特徴的で、恒星の進化段階を視覚的に理解するための最も重要な図の一つである。

</QuizAnswer>

## 問3

太陽程度の質量の恒星が最終的にたどり着く天体はどれか。

1. 中性子星
2. ブラックホール
3. 白色矮星
4. 超新星残骸


<QuizAnswer>

**答え: 3**

太陽程度の質量の恒星は、赤色巨星を経て外層のガスを放出し惑星状星雲となり、中心部は白色矮星として残る。白色矮星は核融合反応を行わず、余熱でゆっくりと冷えていく天体である。中性子星やブラックホールになるには、太陽の約8倍以上の大質量星である必要がある。

</QuizAnswer>

## 問4

天の川銀河（銀河系）の形状として最も適切なものはどれか。

1. 楕円銀河
2. 不規則銀河
3. 棒渦巻銀河
4. レンズ状銀河


<QuizAnswer>

**答え: 3**

天の川銀河は棒渦巻銀河に分類される。中心部に棒状の構造があり、そこから渦巻き状の腕が伸びている。直径は約10万光年で、太陽系は中心から約2万6千光年離れた位置にある。かつては通常の渦巻銀河と考えられていたが、赤外線観測により中心の棒状構造が確認された。

</QuizAnswer>

## 問5

1光年とはどのような単位か。

1. 光が1年間に進む距離
2. 光が1年間に進む時間
3. 地球から太陽までの距離
4. 地球が1年間に公転する距離


<QuizAnswer>

**答え: 1**

1光年は光が真空中を1年間かけて進む距離であり、約9.46兆km（約9.46×10¹²km）に相当する。天文学では恒星間や銀河間の距離を表す際に広く使われる単位である。なお、選択肢3の地球から太陽までの平均距離は「1天文単位（約1.5億km）」と呼ばれ、光年とは異なる単位である。

</QuizAnswer>

## 問6

宇宙が膨張していることを示す観測事実として、遠くの銀河ほどスペクトルが赤い方にずれる現象を発見した天文学者は誰か。

1. アインシュタイン
2. ハッブル
3. ガモフ
4. ホーキング


<QuizAnswer>

**答え: 2**

エドウィン・ハッブルは1929年、遠方の銀河ほど速い速度で遠ざかっていることを観測的に示した。銀河のスペクトルが赤い方にずれる「赤方偏移」の大きさが距離に比例するという関係は「ハッブルの法則」と呼ばれ、宇宙膨張の直接的な証拠となった。この発見はビッグバン理論の重要な観測的根拠となっている。

</QuizAnswer>

## 問7

ビッグバン理論を支持する重要な観測証拠の一つで、宇宙のあらゆる方向からほぼ均一に観測される電磁波を何というか。

1. 宇宙線
2. ガンマ線バースト
3. 宇宙マイクロ波背景放射
4. X線放射


<QuizAnswer>

**答え: 3**

宇宙マイクロ波背景放射（CMB）は、ビッグバンから約38万年後に宇宙が十分冷えて光が直進できるようになった時代の光の名残である。1965年にペンジアスとウィルソンによって偶然発見され、約2.7K（-270℃）の黒体放射としてあらゆる方向からほぼ均一に観測される。この発見はビッグバン理論を強く支持する証拠となった。

</QuizAnswer>

## 問8

太陽の約30倍以上の質量をもつ大質量星が超新星爆発を起こした後、最終的に形成される可能性がある天体はどれか。

1. 白色矮星
2. 褐色矮星
3. 赤色矮星
4. ブラックホール


<QuizAnswer>

**答え: 4**

非常に大きな質量をもつ恒星が超新星爆発を起こした後、残された中心核の質量が太陽の約3倍を超えると、自らの重力に耐えきれず際限なく収縮してブラックホールが形成される。ブラックホールでは重力が極めて強く、光さえも脱出できない領域（事象の地平面）が存在する。質量がそれより小さい場合は中性子星となる。

</QuizAnswer>

## 問9

銀河の形態による分類で、ハッブルが提唱した分類法に含まれない銀河の種類はどれか。

1. 渦巻銀河
2. 楕円銀河
3. 矮小銀河
4. 不規則銀河


<QuizAnswer>

**答え: 3**

ハッブルの銀河分類（ハッブル音叉図）では、銀河を大きく楕円銀河・渦巻銀河（通常の渦巻銀河と棒渦巻銀河）・不規則銀河に分類した。矮小銀河は銀河の大きさに基づく呼び方であり、ハッブルの形態分類の基本カテゴリーには含まれない。この分類体系は現在でも銀河の形態を記述する基本として広く使われている。

</QuizAnswer>

## 問10

中性子星の中でも、非常に強い磁場をもち周期的な電磁波パルスを発する天体を何というか。

1. クエーサー
2. パルサー
3. セファイド変光星
4. 超新星


<QuizAnswer>

**答え: 2**

パルサーは高速で自転する中性子星で、強い磁場に沿って電磁波のビームを放射している。このビームが灯台のように回転するため、地球からは規則正しいパルス（脈動）として観測される。1967年にジョスリン・ベルによって初めて発見され、その極めて正確な周期性から当初は人工的な信号かと考えられたほどである。

</QuizAnswer>
