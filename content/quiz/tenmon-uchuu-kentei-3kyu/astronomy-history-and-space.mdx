## 問1

1543年に著書『天球の回転について』で地動説を提唱し、太陽を中心とする宇宙モデルを発表した天文学者は誰か。

1. ガリレオ・ガリレイ
2. ニコラウス・コペルニクス
3. ヨハネス・ケプラー
4. ティコ・ブラーエ


<QuizAnswer>

**答え: 2**

コペルニクスは1543年に『天球の回転について（De revolutionibus）』を出版し、当時主流だったプトレマイオスの天動説に対して、太陽を中心に惑星が公転するという地動説を提唱しました。ただし当時はすぐには受け入れられず、後のガリレオやケプラーの研究によって支持されるようになりました。

</QuizAnswer>

## 問2

1608年に望遠鏡の特許を申請した、オランダの眼鏡職人は誰か。

1. ハンス・リッペルハイ
2. ガリレオ・ガリレイ
3. アイザック・ニュートン
4. クリスティアーン・ホイヘンス


<QuizAnswer>

**答え: 1**

1608年、オランダの眼鏡職人ハンス・リッペルハイが2枚のレンズを組み合わせた望遠鏡の特許を申請しました。同時期に複数人が申請したため特許は認められませんでしたが、リッペルハイは望遠鏡の発明者の一人として広く知られています。翌1609年にガリレオがこの望遠鏡を改良し、天体観測に初めて使用しました。

</QuizAnswer>

## 問3

ガリレオ・ガリレイが1610年に自作の望遠鏡で発見し、地動説の有力な証拠となった天体は何か。

1. 土星の環
2. 火星の極冠
3. 木星の衛星
4. 金星の満ち欠け


<QuizAnswer>

**答え: 3**

ガリレオは1610年に望遠鏡で木星を観測し、木星の周りを回る4つの衛星（イオ・エウロパ・ガニメデ・カリスト）を発見しました。これらは「ガリレオ衛星」と呼ばれています。すべての天体が地球の周りを回るとする天動説に反し、木星の周りを回る天体の存在は地動説を支持する重要な証拠となりました。

</QuizAnswer>

## 問4

ケプラーが発見した惑星運動の法則のうち、「惑星は太陽を一つの焦点とする楕円軌道を描く」というのは第何法則か。

1. 第3法則
2. 第2法則
3. 第4法則
4. 第1法則


<QuizAnswer>

**答え: 4**

「惑星は太陽を一つの焦点とする楕円軌道上を運動する」はケプラーの第1法則（楕円軌道の法則）です。ケプラーはティコ・ブラーエの精密な観測データをもとに、1609年に第1・第2法則を、1619年に第3法則を発表しました。これらの法則は後にニュートンの万有引力の法則の基礎となりました。

</QuizAnswer>

## 問5

1957年10月4日、世界初の人工衛星を打ち上げた国はどこか。

1. アメリカ
2. ソビエト連邦（ソ連）
3. イギリス
4. フランス


<QuizAnswer>

**答え: 2**

1957年10月4日、ソビエト連邦が世界初の人工衛星「スプートニク1号」の打ち上げに成功しました。この出来事はアメリカに大きな衝撃を与え「スプートニク・ショック」と呼ばれました。これをきっかけにアメリカは1958年にNASA（アメリカ航空宇宙局）を設立し、米ソの宇宙開発競争が本格化しました。

</QuizAnswer>

## 問6

1969年、アポロ11号で人類初の月面着陸を果たした宇宙飛行士は誰か。

1. バズ・オルドリン
2. ジョン・グレン
3. ニール・アームストロング
4. アラン・シェパード


<QuizAnswer>

**答え: 3**

1969年7月20日、アポロ11号の船長ニール・アームストロングが人類で初めて月面に降り立ちました。彼の「これは一人の人間にとっては小さな一歩だが、人類にとっては偉大な飛躍である」という言葉は有名です。同乗のバズ・オルドリンも月面に降りましたが、最初の一歩を踏み出したのはアームストロングでした。

</QuizAnswer>

## 問7

屈折望遠鏡の色収差（色のにじみ）を解決するために、凹面鏡を使った反射望遠鏡を発明した科学者は誰か。

1. アイザック・ニュートン
2. ガリレオ・ガリレイ
3. エドウィン・ハッブル
4. ウィリアム・ハーシェル


<QuizAnswer>

**答え: 1**

アイザック・ニュートンは、レンズを使う屈折望遠鏡では光の色がにじむ（色収差）という問題を解決するために、凹面鏡を使った反射望遠鏡を1668年に製作しました。鏡の表面反射を利用するため色収差が生じず、シャープな像が得られます。現代の大型天体望遠鏡の多くも反射望遠鏡の原理を採用しています。

</QuizAnswer>

## 問8

国際宇宙ステーション（ISS）の建設が始まったのは何年か。

1. 1990年
2. 1995年
3. 2001年
4. 1998年


<QuizAnswer>

**答え: 4**

国際宇宙ステーション（ISS）は1998年に最初のモジュールが打ち上げられ、建設が始まりました。アメリカ、ロシア、日本、ヨーロッパ、カナダなど15カ国が参加する国際共同プロジェクトです。日本は「きぼう」実験棟を提供し、多くの日本人宇宙飛行士がISSに長期滞在しています。

</QuizAnswer>

## 問9

2003年に打ち上げられ、小惑星イトカワからサンプルを持ち帰ることに成功した日本の探査機の名前は何か。

1. かぐや
2. はやぶさ
3. あかつき
4. のぞみ


<QuizAnswer>

**答え: 2**

「はやぶさ」は2003年にJAXA（宇宙航空研究開発機構）が打ち上げた小惑星探査機で、小惑星イトカワに着陸してサンプルを採取し、2010年に地球に帰還しました。数々のトラブルを乗り越えて帰還した姿は大きな感動を呼びました。後継機「はやぶさ2」は小惑星リュウグウのサンプルを2020年に持ち帰り、アミノ酸や液体の水が発見されるなど重要な成果を上げています。

</QuizAnswer>

## 問10

古代ギリシャで、地球を中心に天体が回っているとする「天動説」の体系を完成させた天文学者は誰か。

1. アリストテレス
2. エラトステネス
3. ヒッパルコス
4. プトレマイオス（トレミー）


<QuizAnswer>

**答え: 4**

プトレマイオス（クラウディオス・プトレマイオス）は2世紀にエジプトのアレクサンドリアで活動した天文学者で、著書『アルマゲスト』において天動説の体系を完成させました。地球を宇宙の中心に置き、周転円などの仕組みを用いて惑星の複雑な動きを説明したこのモデルは、コペルニクスの地動説が登場する16世紀まで約1400年間にわたり西洋天文学の標準的な宇宙観であり続けました。

</QuizAnswer>
