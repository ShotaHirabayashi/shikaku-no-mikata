## 問1

太陽系の惑星のうち、最も大きい惑星はどれか。

1. 土星
2. 天王星
3. 木星
4. 海王星


<QuizAnswer>

**答え: 3**

木星は太陽系最大の惑星で、赤道直径は約14万3,000kmあり、地球の約11倍に相当します。質量も地球の約318倍と圧倒的に大きく、太陽系の惑星の中で群を抜いています。木星は主に水素とヘリウムから構成されるガス惑星（巨大ガス惑星）です。

</QuizAnswer>

## 問2

太陽系の惑星を太陽に近い順に並べたとき、4番目にくる惑星はどれか。

1. 地球
2. 木星
3. 火星
4. 金星


<QuizAnswer>

**答え: 3**

太陽に近い順に、水星・金星・地球・火星・木星・土星・天王星・海王星と並ぶため、4番目は火星です。火星は地球型惑星の中で最も外側に位置し、太陽からの平均距離は約2億2,800万km（約1.5天文単位）です。

</QuizAnswer>

## 問3

地球型惑星（岩石惑星）に分類される惑星の組み合わせとして正しいものはどれか。

1. 水星・金星・地球・木星
2. 水星・金星・地球・火星
3. 金星・地球・火星・土星
4. 地球・火星・天王星・海王星


<QuizAnswer>

**答え: 2**

地球型惑星は水星・金星・地球・火星の4つで、いずれも岩石や金属を主成分とし、サイズが比較的小さく密度が大きいのが特徴です。これに対して木星・土星はガス惑星（巨大ガス惑星）、天王星・海王星は氷惑星（巨大氷惑星）に分類されます。

</QuizAnswer>

## 問4

2006年に国際天文学連合（IAU）の決議によって「準惑星」に再分類された天体はどれか。

1. セレス（ケレス）
2. 冥王星
3. エリス
4. 上記すべて


<QuizAnswer>

**答え: 4**

2006年のIAU総会で惑星の定義が明確化され、「軌道近くから他の天体を排除している」という条件を満たさない冥王星は惑星から準惑星に再分類されました。同時に、小惑星帯のセレス（ケレス）や冥王星より大きい天体エリスも準惑星に分類されました。現在IAUが認定する準惑星は、冥王星・エリス・ケレス・マケマケ・ハウメアの5つです。

</QuizAnswer>

## 問5

太陽系の小惑星帯（メインベルト）が存在する場所として正しいものはどれか。

1. 水星と金星の軌道の間
2. 地球と火星の軌道の間
3. 火星と木星の軌道の間
4. 木星と土星の軌道の間


<QuizAnswer>

**答え: 3**

小惑星帯（メインベルト）は火星と木星の軌道の間、太陽から約2.1〜3.3天文単位の領域に広がっています。ここには数十万個以上の小惑星が存在し、最大の天体は準惑星にも分類されているセレス（ケレス）で、直径は約940kmです。

</QuizAnswer>

## 問6

彗星（すいせい）の「尾」が伸びる方向について正しい説明はどれか。

1. 常に彗星の進行方向の後ろ側に伸びる
2. 常に太陽と反対の方向に伸びる
3. 常に地球の方向に向かって伸びる
4. 常に彗星の進行方向の前方に伸びる


<QuizAnswer>

**答え: 2**

彗星の尾は太陽風や太陽の放射圧によって吹き流されるため、常に太陽と反対の方向に伸びます。そのため、太陽から遠ざかるときは進行方向の前方に尾が伸びることもあります。彗星の尾にはガスでできたイオンテイル（青白い）と、ダストでできたダストテイル（黄白い）の2種類があります。

</QuizAnswer>

## 問7

土星の特徴として正しいものはどれか。

1. 太陽系の惑星の中で最も平均密度が小さく、水に浮くほどである
2. 太陽系の惑星の中で自転周期が最も長い
3. 環（リング）を持つ唯一の惑星である
4. 衛星を1つも持たない


<QuizAnswer>

**答え: 1**

土星の平均密度は約0.69g/cm³で、水の密度（1g/cm³）よりも小さく、太陽系の惑星の中で最も密度が低い天体です。なお、環（リング）は木星・天王星・海王星にも存在しますが、土星の環が最も大きく壮麗です。土星にはタイタンやエンケラドゥスなど140以上の衛星が確認されています。

</QuizAnswer>

## 問8

海王星の軌道の外側、約30〜50天文単位の領域に広がる天体群の密集地帯を何と呼ぶか。

1. オールトの雲
2. カイパーベルト（エッジワース・カイパーベルト）
3. メインベルト（小惑星帯）
4. ラグランジュ点


<QuizAnswer>

**答え: 2**

カイパーベルト（エッジワース・カイパーベルト）は、海王星軌道の外側約30〜50天文単位の黄道面付近に広がる天体密集領域です。冥王星やマケマケ、ハウメアなどの準惑星もここに存在します。一方、オールトの雲はさらに遠方（数千〜数万天文単位）に球殻状に広がると考えられている長周期彗星の供給源です。

</QuizAnswer>

## 問9

JAXAの探査機「はやぶさ2」がサンプルを採取して地球に持ち帰った小惑星の名前はどれか。

1. イトカワ
2. リュウグウ
3. ベンヌ
4. トリフネ


<QuizAnswer>

**答え: 2**

はやぶさ2は小惑星リュウグウを探査し、2020年12月にサンプルを地球へ持ち帰りました。リュウグウの試料からはアミノ酸を含む多種類の有機物や液体の水が発見され、太陽系の起源に迫る重要な成果を上げています。なお、イトカワは初代はやぶさが探査した小惑星、ベンヌはNASAのオサイリス・レックスが探査した小惑星です。

</QuizAnswer>

## 問10

太陽系が誕生したのはおよそ何年前とされているか。

1. 約10億年前
2. 約46億年前
3. 約100億年前
4. 約138億年前


<QuizAnswer>

**答え: 2**

太陽系は約46億年前に、巨大な分子雲（星間ガスとちりの集まり）が重力で収縮することで形成されたと考えられています。この年齢は隕石に含まれる放射性同位体の分析から求められました。なお、約138億年前は宇宙そのものの誕生（ビッグバン）の時期にあたります。

</QuizAnswer>
