## 問1

地球の自転によって、天体が1日に約1回、東から西へ動いて見える現象を何というか。

1. 年周運動
2. 歳差運動
3. 日周運動
4. 固有運動


<QuizAnswer>

**答え: 3**

地球が西から東へ約24時間で1回自転しているため、天体は見かけ上、東から西へ1日に約1回転する動きを見せます。これを日周運動といいます。年周運動は地球の公転による1年周期の見かけの動き、歳差運動は地軸の向きがゆっくり変化する現象です。

</QuizAnswer>

## 問2

北極星付近を中心に星が反時計回りに回って見えるのは、どの方角の空を観察したときか。

1. 南の空
2. 東の空
3. 西の空
4. 北の空


<QuizAnswer>

**答え: 4**

北の空では、天の北極（北極星付近）を中心に星が反時計回りに日周運動して見えます。これは地球が西から東（反時計回り）に自転しているためです。一方、南の空では星は東から西へ（時計回りに）弧を描いて動いて見えます。

</QuizAnswer>

## 問3

地球の公転により、同じ時刻に見える星座が少しずつ西へずれていく。1か月でおよそ何度ずれるか。

1. 約15度
2. 約30度
3. 約45度
4. 約60度


<QuizAnswer>

**答え: 2**

地球は太陽のまわりを1年（約365日）で360度公転するため、1か月（約30日）では約30度分、星座の見える位置が西へずれていきます。これが年周運動の効果であり、季節ごとに見える星座が変わる理由です。

</QuizAnswer>

## 問4

日本（北緯35度）において、春分の日の太陽の南中高度はおよそ何度か。

1. 約35度
2. 約55度
3. 約65度
4. 約78度


<QuizAnswer>

**答え: 2**

春分の日の太陽の南中高度は「90度 − その地点の緯度」で計算できます。北緯35度の場合、90° − 35° = 55°となります。春分・秋分の日は太陽が天の赤道上にあるため、地軸の傾きによる補正は不要です。

</QuizAnswer>

## 問5

季節の変化が生じる主な原因はどれか。

1. 地球と太陽の距離が季節ごとに大きく変わるため
2. 太陽の放出するエネルギーが季節ごとに変化するため
3. 地球の自転速度が季節ごとに変化するため
4. 地球の地軸が公転面に対して約23.4度傾いているため


<QuizAnswer>

**答え: 4**

季節の変化は、地球の地軸が公転面の垂線に対して約23.4度傾いていることが主な原因です。この傾きにより、夏は太陽の南中高度が高く昼が長くなり、冬はその逆になります。地球と太陽の距離の変化は季節にほとんど影響しません。

</QuizAnswer>

## 問6

太陽が天球上を1年かけて通る見かけの道筋を何というか。

1. 天の赤道
2. 黄道
3. 子午線
4. 白道


<QuizAnswer>

**答え: 2**

黄道とは、地球の公転によって太陽が天球上を1年かけて一周する見かけの通り道のことです。天の赤道は地球の赤道を天球に投影したもの、白道は月が天球上を通る道筋です。黄道は天の赤道に対して約23.4度傾いています。

</QuizAnswer>

## 問8

ある星が午後10時に南中した。同じ星が2か月後に南中するのはおよそ何時か。

1. 午後6時
2. 午後8時
3. 午前0時（真夜中）
4. 午前2時


<QuizAnswer>

**答え: 1**

地球の公転により、星の南中時刻は1か月で約2時間ずつ早くなります。2か月後には約4時間早くなるため、午後10時 − 4時間 = 午後6時ごろに南中することになります。これは、恒星の日周運動が太陽より約4分早い（恒星日が太陽日より短い）ことの積み重ねです。

</QuizAnswer>

## 問9

二十四節気のうち、1年で最も昼の時間が長い日を何というか。

1. 春分
2. 夏至
3. 秋分
4. 立夏


<QuizAnswer>

**答え: 2**

夏至は北半球において1年で最も昼の時間が長く、太陽の南中高度が最も高くなる日です。二十四節気は太陽の黄道上の位置（黄経）に基づいて1年を24等分したもので、夏至は太陽黄経が90度のときにあたります。春分・秋分は昼と夜の長さがほぼ等しくなります。

</QuizAnswer>

## 問10

天球上で、地球の自転軸を北に延長した点を何というか。

1. 天頂
2. 天底
3. 天の北極
4. 北極点


<QuizAnswer>

**答え: 3**

天の北極は、地球の自転軸を北側に延長して天球と交わる点です。現在、この点の近くに北極星（ポラリス）があるため、北極星はほとんど動かずに見えます。天頂は観測者の真上の点、北極点は地球表面上の地理的な点であり、天球上の点ではありません。

</QuizAnswer>
