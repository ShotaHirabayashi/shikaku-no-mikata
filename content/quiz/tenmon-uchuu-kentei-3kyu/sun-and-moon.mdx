## 問1

太陽のエネルギー源として正しいものはどれか。

1. 石炭や石油のような化学反応による燃焼
2. 太陽内部のウランによる核分裂反応
3. 水素の原子核が融合してヘリウムになる核融合反応
4. 太陽が収縮することで発生する重力エネルギー


<QuizAnswer>

**答え: 3**

太陽のエネルギー源は、中心部（核）で起こる核融合反応です。4つの水素原子核が融合して1つのヘリウム原子核になる際、わずかに質量が減少し、その分がアインシュタインの式 E=mc² に従ってエネルギーに変換されます。太陽中心部の温度は約1600万度にも達し、この高温・高圧の環境で核融合が持続しています。

</QuizAnswer>

## 問2

太陽の表面にあたる層を何と呼ぶか。

1. 光球
2. 彩層
3. コロナ
4. 対流層


<QuizAnswer>

**答え: 1**

太陽の表面にあたる層は「光球」と呼ばれ、私たちが普段目にしている太陽の姿はこの光球です。光球の温度は約6000℃です。その外側に彩層、さらに外側にコロナがあります。コロナの温度は約100万〜300万℃と光球より高く、これは「コロナ加熱問題」として現在も研究が続いています。

</QuizAnswer>

## 問3

太陽の黒点について正しい説明はどれか。

1. 黒点は周囲より温度が高いため黒く見える
2. 黒点は周囲より温度が低いため黒く見える
3. 黒点は太陽表面に開いた穴である
4. 黒点は太陽表面で常に同じ位置にとどまっている


<QuizAnswer>

**答え: 2**

黒点は太陽の光球上に現れる暗い領域で、周囲の光球（約6000℃）に比べて温度が約4000℃と低いため、相対的に暗く（黒く）見えます。黒点は強い磁場を持つ領域で、磁場がエネルギーの対流を妨げるため温度が下がります。黒点は太陽表面を移動しており、その動きから太陽が自転していることもわかります。

</QuizAnswer>

## 問4

太陽の黒点数が増減を繰り返す周期として最も近いものはどれか。

1. 約1年
2. 約5年
3. 約8年
4. 約11年


<QuizAnswer>

**答え: 4**

太陽活動は約11年の周期で活発になったり静穏になったりを繰り返しており、黒点数もこの周期に合わせて増減します。黒点が多い時期を「太陽活動極大期」、少ない時期を「太陽活動極小期」と呼びます。太陽活動が活発な時期には太陽フレアも発生しやすくなります。

</QuizAnswer>

## 問5

月が地球から見て太陽と反対側にあるとき、月はどのように見えるか。

1. 新月
2. 満月
3. 上弦の月
4. 下弦の月


<QuizAnswer>

**答え: 2**

月が地球から見て太陽と反対側にあるとき、太陽の光が月の地球側の面全体を照らすため「満月」に見えます。逆に、月が太陽と同じ方向にあるときは太陽の光が月の裏側を照らすため、地球からは月が見えない「新月」になります。月は約29.5日で満ち欠けの1周期（朔望月）を繰り返します。

</QuizAnswer>

## 問6

日食が起こるときの天体の並び方として正しいものはどれか。

1. 月―太陽―地球の順に一直線に並ぶ
2. 太陽―地球―月の順に一直線に並ぶ
3. 太陽―月―地球の順に一直線に並ぶ
4. 地球―太陽―月の順に一直線に並ぶ


<QuizAnswer>

**答え: 3**

日食は、太陽と地球の間に月が入り、太陽―月―地球の順に一直線に並んだときに起こります。月が太陽の一部を隠すのが部分日食、太陽全体を隠すのが皆既日食です。なお、新月のたびに日食が起こるわけではなく、月の公転軌道面が地球の公転軌道面に対して約5度傾いているため、条件が揃ったときだけ発生します。

</QuizAnswer>

## 問7

月食が起こるのはどのような月の形のときか。

1. 満月のとき
2. 新月のとき
3. 上弦の月のとき
4. 三日月のとき


<QuizAnswer>

**答え: 1**

月食は、太陽―地球―月が一直線に並び、地球の影の中を月が通過することで起こります。この配置になるのは満月のときだけです。地球の影に月が完全に入ると皆既月食、一部だけ入ると部分月食になります。皆既月食のとき、月は地球の大気で屈折した赤い光に照らされて赤銅色に見えることがあります。

</QuizAnswer>

## 問8

潮の満ち引き（潮汐）が最も大きくなる「大潮」はどのようなときに起こるか。

1. 月と太陽が地球に対して直角の方向にあるとき
2. 月が地球から最も遠い位置にあるとき
3. 地球が太陽に最も近い位置にあるとき
4. 月と太陽が地球に対して一直線上に並ぶとき


<QuizAnswer>

**答え: 4**

大潮は、月と太陽が地球に対して一直線上に並ぶとき（新月と満月の頃）に起こります。このとき月と太陽の起潮力が重なり合い、潮の満ち引きの差が最も大きくなります。逆に、月と太陽が直角の方向にずれるとき（上弦・下弦の月の頃）は起潮力が打ち消し合い、潮位差が最も小さい「小潮」になります。

</QuizAnswer>

## 問9

月の表面に見える暗い平坦な部分を何と呼ぶか。

1. 月の海
2. 月の湖
3. 月の谷
4. 月の砂漠


<QuizAnswer>

**答え: 1**

月の表面に見える暗い平坦な領域は「月の海」と呼ばれます。実際に水があるわけではなく、過去に巨大な隕石が衝突してできた盆地に、月内部から玄武岩質の溶岩が噴出して固まった地形です。月の海は比較的クレーターが少なく平坦であるのに対し、明るい高地（陸）の部分には多くのクレーターや山脈が見られます。

</QuizAnswer>

## 問10

太陽フレアが地球に影響を与えた結果として起こりうる現象はどれか。

1. 地球全体の気温が急激に上昇する
2. 月の表面に新たなクレーターができる
3. 高緯度地域でオーロラが活発になる
4. 地球の自転速度が変化する


<QuizAnswer>

**答え: 3**

太陽フレアが発生すると、大量の荷電粒子（プラズマ）が太陽風として宇宙空間に放出されます。これが地球の磁気圏に到達すると、磁力線に沿って極地方の上空に導かれ、大気中の酸素や窒素と衝突して発光する現象がオーロラです。大規模な太陽フレアの際には、通常より低い緯度でもオーロラが観測されることがあります。

</QuizAnswer>
