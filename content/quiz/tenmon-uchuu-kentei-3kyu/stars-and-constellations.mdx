## 問1

全天で最も明るい恒星はどれか。

1. カノープス
2. ベガ
3. シリウス
4. アルクトゥルス


<QuizAnswer>

**答え: 3**

シリウスはおおいぬ座α星で、見かけの等級は約−1.5等と全天で最も明るい恒星です。冬の南の空に青白く輝き、「冬の大三角」の一角を成す代表的な1等星です。古代エジプトではナイル川の氾濫を知らせる星として重視されていました。

</QuizAnswer>

## 問2

恒星にギリシャ文字を割り当てる「バイエル符号」は、原則として何の順番に基づいているか。

1. 赤経の順番
2. 発見された順番
3. 明るさの順番
4. 星座内での位置の順番


<QuizAnswer>

**答え: 3**

バイエル符号は1603年にドイツの天文学者ヨハン・バイエルが星図『ウラノメトリア』で導入した命名法です。星座ごとに原則として明るい順にα（アルファ）、β（ベータ）、γ（ガンマ）…とギリシャ文字を割り当てます。なお、赤経の順番に番号を付けるのはフラムスティード番号です。

</QuizAnswer>

## 問3

現在、国際天文学連合（IAU）が定めている星座の数はいくつか。

1. 48
2. 72
3. 88
4. 108


<QuizAnswer>

**答え: 3**

1922年の国際天文学連合（IAU）総会で88の星座が正式に定められました。古代ギリシャのプトレマイオスが『アルマゲスト』にまとめた48星座を基礎に、大航海時代以降に追加された南天の星座などを加えて88星座となっています。星座の境界線は1930年に最終確定しました。

</QuizAnswer>

## 問4

夏の大三角を構成する星の組み合わせとして正しいものはどれか。

1. シリウス・プロキオン・ベテルギウス
2. ベガ・アルタイル・デネブ
3. アルクトゥルス・スピカ・デネボラ
4. カペラ・アルデバラン・リゲル


<QuizAnswer>

**答え: 2**

夏の大三角は、こと座のベガ、わし座のアルタイル、はくちょう座のデネブの3つの1等星を結んだものです。選択肢1は冬の大三角、選択肢3は春の大三角の組み合わせです。七夕伝説ではベガが織姫、アルタイルが彦星に対応しています。

</QuizAnswer>

## 問5

星の色と表面温度の関係について、正しいものはどれか。

1. 赤い星ほど表面温度が高い
2. 黄色い星ほど表面温度が高い
3. 青白い星ほど表面温度が高い
4. 星の色と表面温度に関係はない


<QuizAnswer>

**答え: 3**

恒星の色は表面温度によって決まります。表面温度が約3,000度と低い星は赤く見え、約6,000度の太陽は黄色、約10,000度以上の高温の星は青白く見えます。オリオン座のリゲルは表面温度が約12,000度で青白く、同じオリオン座のベテルギウスは約3,500度で赤く輝いています。

</QuizAnswer>

## 問6

2世紀にそれまで知られていた星座を48個にまとめた天文学者は誰か。

1. アリスタルコス
2. プトレマイオス
3. ヒッパルコス
4. エラトステネス


<QuizAnswer>

**答え: 2**

古代ローマ帝国時代の2世紀、天文学者プトレマイオス（トレミー）が著書『アルマゲスト』の中で48の星座をまとめました。これがのちの88星座の基礎となっています。ヒッパルコスは恒星の明るさの等級を考案した天文学者として知られています。

</QuizAnswer>

## 問7

多くの恒星の固有名（シリウス、アルタイル、アルデバランなど）は、主にどの言語に由来するものが多いか。

1. ギリシャ語
2. ラテン語
3. アラビア語
4. サンスクリット語


<QuizAnswer>

**答え: 3**

恒星の固有名にはアラビア語に由来するものが非常に多いです。これは中世イスラム世界で天文学が大きく発展し、多くの星にアラビア語の名前が付けられたためです。例えばアルデバラン（追う者）、アルタイル（飛ぶ鷲）、デネブ（尾）などはいずれもアラビア語が語源です。

</QuizAnswer>

## 問8

さそり座の心臓にあたる位置に赤く輝く1等星の名前はどれか。

1. アルデバラン
2. ベテルギウス
3. アンタレス
4. ポルックス


<QuizAnswer>

**答え: 3**

アンタレスはさそり座α星で、赤く輝く1等星です。その名前はギリシャ語で「火星（アレス）に対抗するもの」を意味し、火星に匹敵する赤さを持つことに由来します。表面温度が約3,500度と低いため赤く見える赤色超巨星で、夏の南の空の目印となります。

</QuizAnswer>

## 問9

全天の1等星（1.5等級より明るい恒星）は全部でいくつあるか。

1. 12個
2. 21個
3. 36個
4. 48個


<QuizAnswer>

**答え: 2**

全天で1.5等級より明るい恒星は全部で21個あり、これらを1等星と呼びます。最も明るいシリウス（−1.5等）から最も暗い1等星のレグルス（1.3等）まで幅があります。日本からはこのうち15個の1等星を見ることができます。

</QuizAnswer>

## 問10

オリオン座で青白く輝く1等星リゲルは、ギリシャ文字を用いたバイエル符号では何と表されるか。

1. オリオン座α星
2. オリオン座β星
3. オリオン座γ星
4. オリオン座δ星


<QuizAnswer>

**答え: 2**

リゲルはオリオン座β（ベータ）星です。オリオン座で最も明るい星はリゲルですが、α星にはベテルギウスが割り当てられています。バイエル符号は原則として明るい順ですが、例外も多く、オリオン座はその代表例です。リゲルの名前はアラビア語で「巨人の左足」を意味する言葉に由来しています。

</QuizAnswer>
