## 問1

一般用医薬品の点眼薬に配合される成分のうち、目の調節機能を改善する目的で用いられるものはどれか。

1. スルファメトキサゾール
2. コンドロイチン硫酸エステルナトリウム
3. ネオスチグミンメチル硫酸塩
4. クロモグリク酸ナトリウム


<QuizAnswer>

**答え: 3**

ネオスチグミンメチル硫酸塩は、コリンエステラーゼの働きを抑えることでアセチルコリンの働きを助け、毛様体筋に作用して目の調節機能を改善する成分です。スルファメトキサゾールは抗菌成分、コンドロイチン硫酸エステルナトリウムは角膜保護成分、クロモグリク酸ナトリウムはアレルギー用成分であり、いずれも調節機能改善の目的では用いられません。

</QuizAnswer>

## 問2

外用の殺菌消毒薬に関する次の記述のうち、正しいものはどれか。

1. エタノールは、粘膜面や創傷面の消毒に広く用いられる。
2. ポビドンヨードは、ヨウ素による酸化作用により殺菌消毒作用を示す。
3. オキシドールは、真菌に対して特に強い殺菌力を持つ。
4. ベンザルコニウム塩化物は、結核菌やウイルスに対して強い効果がある。


<QuizAnswer>

**答え: 2**

ポビドンヨードは、ヨウ素をポリビニルピロリドン（PVP）に結合させた成分で、徐々にヨウ素が遊離して酸化作用により殺菌消毒効果を発揮します。エタノールは粘膜面や創傷面への使用は適しません（刺激が強い）。オキシドールは活性酸素による殺菌作用を持ちますが、真菌に特に強いわけではありません。ベンザルコニウム塩化物は結核菌やウイルスには効果が不十分です。

</QuizAnswer>

## 問3

漢方処方製剤「葛根湯」に関する次の記述のうち、正しいものはどれか。

1. 体力虚弱で、汗が出やすい人のかぜの初期に適する。
2. 構成生薬にマオウを含まない。
3. 体力中等度以上の人で、感冒の初期（汗をかいていないもの）に適する。
4. 水様性の鼻水が多く出る場合に最も適する。


<QuizAnswer>

**答え: 3**

葛根湯は、体力中等度以上の人で、かぜの初期で汗をかいていない状態（表実証）に用いられる漢方処方製剤です。構成生薬にマオウ（麻黄）を含むため、胃腸が弱い人や発汗傾向の著しい人には不向きです。水様性の鼻水が多い場合には小青竜湯がより適しています。

</QuizAnswer>

## 問4

皮膚に用いる外用消炎鎮痛薬の配合成分とその作用の組み合わせについて、誤っているものはどれか。

1. インドメタシン ── 非ステロイド性抗炎症成分
2. ケトプロフェン ── 非ステロイド性抗炎症成分
3. フェルビナク ── 非ステロイド性抗炎症成分
4. ジフェンヒドラミン ── 局所麻酔成分


<QuizAnswer>

**答え: 4**

ジフェンヒドラミンは局所麻酔成分ではなく、抗ヒスタミン成分です。皮膚の痒みを抑える目的で外用薬に配合されます。インドメタシン、ケトプロフェン、フェルビナクはいずれも非ステロイド性抗炎症成分（NSAIDs）であり、筋肉痛・関節痛・打撲・捻挫等の鎮痛消炎に用いられます。

</QuizAnswer>

## 問5

生薬に関する次の記述のうち、正しいものはどれか。

1. カンゾウ（甘草）の主な有効成分はエフェドリンである。
2. マオウ（麻黄）には、交感神経を刺激する作用がある。
3. ダイオウ（大黄）は、鎮痛を目的として配合される。
4. ブシ（附子）は、瀉下（しゃげ）作用を期待して配合される。


<QuizAnswer>

**答え: 2**

マオウ（麻黄）にはエフェドリンが含まれ、交感神経を刺激する作用（気管支拡張、発汗促進等）があります。カンゾウの主な有効成分はグリチルリチン酸であり、抗炎症作用を示します。ダイオウは瀉下（便通促進）作用を主に期待して配合されます。ブシは心筋の収縮力を高めて血液循環を改善する作用や鎮痛作用があり、瀉下作用はありません。

</QuizAnswer>

## 問6

痔疾用薬に配合される成分に関する次の記述のうち、誤っているものはどれか。

1. リドカインは、局所麻酔成分として痛み・痒みを鎮める目的で配合される。
2. プレドニゾロン酢酸エステルは、ステロイド性抗炎症成分として配合される。
3. 酸化亜鉛は、粘膜表面に不溶性の膜を形成して保護する目的で配合される。
4. タンニン酸は、血管収縮成分として止血の目的で配合される。


<QuizAnswer>

**答え: 4**

タンニン酸は血管収縮成分ではなく、収れん保護止血成分です。粘膜表面のタンパク質と結合して被膜を形成し、粘膜の保護・止血に働きます。痔疾用薬で血管収縮成分として用いられるのはテトラヒドロゾリン塩酸塩やメチルエフェドリン塩酸塩などのアドレナリン作動成分です。リドカインは局所麻酔成分、プレドニゾロン酢酸エステルはステロイド性抗炎症成分、酸化亜鉛は保護成分として正しい記述です。

</QuizAnswer>

## 問7

漢方処方製剤「小青竜湯」に関する次の記述のうち、正しいものはどれか。

1. 体力虚弱で胃腸が弱い人に最も適する。
2. うすい水様の痰を伴う咳や鼻水が出るものに適する。
3. 構成生薬にカンゾウを含まない。
4. インターフェロン製剤と併用しても問題ない。


<QuizAnswer>

**答え: 2**

小青竜湯は、体力中等度又はやや虚弱で、うすい水様の痰を伴う咳や鼻水が出る人のかぜ、気管支炎、アレルギー性鼻炎等に用いられます。構成生薬にマオウとカンゾウの両方を含むため、胃腸が弱い人には注意が必要です。また、マオウを含む漢方処方製剤全般に共通して、他の医薬品との相互作用に留意が必要です。

</QuizAnswer>

## 問8

口内炎用薬に関する次の記述のうち、正しいものはどれか。

1. シコンエキスは、組織修復促進作用を期待して口内炎用薬に配合される。
2. アズレンスルホン酸ナトリウムは、殺菌消毒を目的として配合される。
3. 口内炎が長期間（2週間以上）治らない場合でも、一般用医薬品で対処を続けてよい。
4. デキサメタゾンは、抗ヒスタミン成分として配合される。


<QuizAnswer>

**答え: 1**

シコン（紫根）は、ムラサキ科の植物の根を用いた生薬で、組織修復促進や抗菌作用を期待して口内炎用薬に配合されます。アズレンスルホン酸ナトリウムは殺菌消毒成分ではなく、抗炎症成分です。口内炎が2週間以上治らない場合は口腔がん等の可能性もあるため、医療機関への受診を勧めるべきです。デキサメタゾンはステロイド性抗炎症成分です。

</QuizAnswer>

## 問9

一般用検査薬に関する次の記述のうち、正しいものはどれか。

1. 一般用検査薬の検査結果は、確定診断として用いることができる。
2. 尿糖検査薬は、食後1〜2時間の尿を検体とすることが望ましい。
3. 妊娠検査薬は、月経予定日当日から検査が可能である。
4. 尿タンパク検査薬の場合、原則として早朝尿（起床直後の尿）を用いる。


<QuizAnswer>

**答え: 4**

尿タンパク検査薬では、運動直後や入浴後は一過性のタンパク尿が出ることがあるため、原則として早朝尿（起床直後の尿）を検体とします。一般用検査薬の結果はあくまで目安であり、確定診断はできません。尿糖検査は食後1〜2時間が望ましいという記述は一見正しそうですが、正確には食後2〜3時間の尿とされることが多いです。妊娠検査薬は月経予定日が過ぎて概ね1週間以降に検査するものが一般的です。

</QuizAnswer>

## 問10

禁煙補助剤に関する次の記述のうち、誤っているものはどれか。

1. 咀嚼剤（ニコチンガム）は、ゆっくりと断続的に噛むことが望ましい。
2. ニコチンを有効成分とする禁煙補助剤は、非喫煙者が使用してはならない。
3. 禁煙補助剤を使用中は、喫煙を完全にやめる必要がある。
4. 貼付剤（ニコチンパッチ）は、入浴時にも貼付したまま使用する。


<QuizAnswer>

**答え: 4**

ニコチンパッチ（一般用医薬品のもの）は、入浴時には剥がす必要があります（医療用のニコチネルTTSは貼付したまま入浴可能ですが、一般用医薬品では異なります）。ニコチンガムはゆっくり断続的に噛むことで口腔粘膜からニコチンを吸収させます。ニコチン製剤は非喫煙者（特に妊婦・小児等）が使用すると副作用のおそれがあるため使用禁止です。使用中の喫煙はニコチンの過剰摂取となるため、完全に禁煙する必要があります。

</QuizAnswer>
