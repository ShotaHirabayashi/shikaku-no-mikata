## 問1

胃から分泌される消化酵素ペプシノーゲンは、胃酸によって活性化されペプシンとなる。ペプシンが分解する栄養素として正しいものはどれか。

1. 炭水化物
2. タンパク質
3. 脂質
4. ビタミン


<QuizAnswer>

**答え: 2**

ペプシンは胃で働くタンパク質分解酵素です。ペプシノーゲンとして分泌された後、胃酸（塩酸）によって活性型のペプシンに変換され、タンパク質を消化します。炭水化物の消化は主に唾液や膵液中のアミラーゼ、脂質の消化は膵液中のリパーゼが担っています。

</QuizAnswer>

## 問2

肝臓の働きに関する記述のうち、正しいものはどれか。

1. 肝臓は胃の左下方に位置する臓器である
2. 肝臓では、古くなった赤血球の処理は行われない
3. 肝臓は、アルコールやアンモニアなどの有害物質を代謝・解毒する
4. 肝臓で産生された胆汁は、タンパク質の消化酵素として働く


<QuizAnswer>

**答え: 3**

肝臓は体内最大の臓器で、横隔膜の直下、右上腹部に位置します。アルコールやアンモニアなどの有害物質を無毒化（解毒）する重要な機能を持っています。また、古くなった赤血球を分解してビリルビンを生成する働きもあります。胆汁は消化酵素ではなく、脂質の乳化を助ける役割を果たします。

</QuizAnswer>

## 問3

心臓の構造と血液循環に関する記述のうち、正しいものはどれか。

1. 右心室から全身に血液が送り出される
2. 左心房には肺から戻ってきた血液が流入する
3. 心臓の拍動は随意的にコントロールできる
4. 動脈血とは二酸化炭素を多く含む血液のことである


<QuizAnswer>

**答え: 2**

肺で酸素を取り込んだ血液（動脈血）は、肺静脈を通って左心房に流入します。左心室から全身へ送り出されるのが体循環、右心室から肺へ送り出されるのが肺循環です。心臓の拍動は自律神経によって不随意的に調節されており、意識的にコントロールすることはできません。

</QuizAnswer>

## 問4

血液中の成分に関する記述のうち、誤っているものはどれか。

1. 赤血球に含まれるヘモグロビンは、酸素を全身の組織に運搬する
2. 白血球は、体内に侵入した細菌やウイルスなどの異物を排除する
3. 血小板は、損傷した血管からの出血を抑える働きがある
4. アルブミンは赤血球に含まれるタンパク質である


<QuizAnswer>

**答え: 4**

アルブミンは赤血球ではなく、血漿（けっしょう）中に含まれるタンパク質です。血漿中のアルブミンは、血液の浸透圧を保持するほか、体内のさまざまな物質と結合して運搬する役割を担っています。赤血球に含まれるタンパク質はヘモグロビンであり、酸素の運搬を行います。

</QuizAnswer>

## 問5

呼吸器系に関する記述のうち、正しいものはどれか。

1. 咽頭は食道と気道の両方に通じており、飲食物を飲み込む際に喉頭蓋が気道を塞ぐ
2. 肺自体に筋肉があり、自力で膨らんだり縮んだりして呼吸を行う
3. 気管支の内壁には線毛がなく、異物の排出は咳反射のみで行われる
4. 肺胞でのガス交換では、二酸化炭素が血液中に取り込まれる


<QuizAnswer>

**答え: 1**

咽頭は食道と気道の両方に通じる部分であり、飲食物を飲み込むときには喉頭蓋（こうとうがい）が反射的に閉じて気道への侵入を防ぎます。肺自体には筋肉がなく、横隔膜や肋間筋の動きによって受動的に膨らんだり縮んだりします。気管支の内壁には線毛があり、異物を排出する機能を持っています。

</QuizAnswer>

## 問6

腎臓と泌尿器系に関する記述のうち、正しいものはどれか。

1. 腎臓では、糸球体からボウマン嚢に血液がそのまま流れ込む
2. 腎小体（糸球体とボウマン嚢）と尿細管を合わせてネフロンという
3. 膀胱の排尿筋は交感神経の刺激により収縮し、排尿が起こる
4. 腎臓は左右1つずつあり、腹腔の前面に位置する


<QuizAnswer>

**答え: 2**

ネフロン（腎単位）は、糸球体とそれを包むボウマン嚢からなる腎小体と、それに続く尿細管で構成される腎臓の基本的な機能単位です。糸球体では血液がろ過されて原尿が作られますが、血液そのものが流れ込むわけではありません。排尿は副交感神経の刺激で排尿筋が収縮して起こります。腎臓は腹腔の背面（後腹膜）に位置しています。

</QuizAnswer>

## 問7

目の構造と機能に関する記述のうち、正しいものはどれか。

1. 水晶体は、虹彩の後方にあり、厚みを変えることでピント調節を行う
2. 角膜には血管が豊富に分布しており、栄養が血液から供給される
3. 網膜には視細胞がなく、光の情報は直接視神経に伝わる
4. 結膜は眼球の内部を満たす透明なゼリー状の組織である


<QuizAnswer>

**答え: 1**

水晶体は虹彩の後方に位置する透明な凸レンズ状の組織で、毛様体の筋肉（毛様体筋）によって厚みが変化し、遠近のピント調節を行います。角膜には血管がなく、房水や涙液から栄養分や酸素が供給されます。網膜には視細胞（錐状体・杆状体）が存在し、光を感知します。眼球内部を満たすゼリー状の組織は硝子体です。

</QuizAnswer>

## 問8

皮膚の構造と機能に関する記述のうち、誤っているものはどれか。

1. 皮膚の最も外側の層は表皮であり、角質層が外界からの異物侵入を防いでいる
2. 汗腺にはエクリン腺とアポクリン腺の2種類がある
3. 皮脂腺から分泌される皮脂は、皮膚表面を保護し水分の蒸発を防ぐ
4. メラニン色素は真皮で産生され、紫外線から皮膚を守る


<QuizAnswer>

**答え: 4**

メラニン色素は真皮ではなく、表皮の最下層にあるメラノサイト（メラニン産生細胞）で産生されます。メラニンは紫外線を吸収して皮膚の深部への損傷を防ぐ役割を果たしています。汗腺のうちエクリン腺は全身に分布して体温調節に関与し、アポクリン腺は腋窩（わきの下）など限られた部位に存在します。

</QuizAnswer>

## 問9

自律神経系に関する記述のうち、正しいものはどれか。

1. 交感神経が優位になると、消化管の運動が活発になる
2. 副交感神経が優位になると、瞳孔が散大する
3. 交感神経が優位になると、気管支が拡張する
4. 副交感神経が優位になると、心拍数が増加する


<QuizAnswer>

**答え: 3**

交感神経は「闘争か逃走か（fight or flight）」の反応に関わり、気管支を拡張させて多くの酸素を取り込めるようにします。交感神経が優位のときは消化管の運動は抑制され、瞳孔は散大します。逆に副交感神経が優位のときは、消化管の運動が活発になり、瞳孔は縮小し、心拍数は減少します。

</QuizAnswer>

## 問10

医薬品の吸収・代謝・排泄に関する記述のうち、正しいものはどれか。

1. 内服薬の有効成分は、主に胃で吸収されて全身に分布する
2. 肝初回通過効果とは、吸収された有効成分が全身循環に入る前に肝臓で代謝を受けることをいう
3. 腎臓で排泄される医薬品の代謝物は、すべて脂溶性の物質に変換されている
4. 坐剤（坐薬）は内服薬と同様に、肝初回通過効果を大きく受ける


<QuizAnswer>

**答え: 2**

肝初回通過効果（first-pass effect）とは、消化管から吸収された有効成分が門脈を通って肝臓に運ばれ、全身循環に入る前に代謝を受ける現象です。内服薬の有効成分は主に小腸で吸収されます（胃ではありません）。代謝物は一般に水溶性に変換されて排泄されやすくなります。坐剤は直腸下部の静脈から吸収されるため、肝初回通過効果を受けにくいのが特徴です。

</QuizAnswer>
