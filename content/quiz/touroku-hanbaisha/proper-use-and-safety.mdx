## 問1

一般用医薬品の添付文書に関する次の記述のうち、正しいものはどれか。

1. 添付文書の内容は一度承認されると変更されることはない
2. 添付文書の内容は必要に応じて随時改訂され、改訂年月が記載される
3. 添付文書は医療用医薬品にのみ添付が義務づけられている
4. 添付文書には有効成分の名称は記載されない

<QuizAnswer>

**答え: 2**

一般用医薬品の添付文書の内容は、医薬品の有効性・安全性等に係る新たな知見や使用に係る情報に基づき、必要に応じて随時改訂されます。添付文書には改訂年月が記載されており、購入者等はその記載によって最新の情報であるかを確認できます。添付文書は一般用医薬品にも添付が義務づけられています。

</QuizAnswer>

## 問2

一般用医薬品の添付文書における「使用上の注意」の構成として、正しいものはどれか。

1. 「効能・効果」「用法・用量」「保管上の注意」
2. 「してはいけないこと」「相談すること」「その他の注意」
3. 「禁忌」「慎重投与」「重大な副作用」
4. 「警告」「使用制限」「保管方法」

<QuizAnswer>

**答え: 2**

一般用医薬品の添付文書における「使用上の注意」は、「してはいけないこと」「相談すること」「その他の注意」の3項目から構成されています。「してはいけないこと」と「相談すること」には、それぞれ定められた標識的マークが付されており、購入者が一目で認識できるようになっています。

</QuizAnswer>

## 問3

医薬品の副作用等の報告制度に関する次の記述のうち、誤っているものはどれか。

1. 医師、歯科医師、薬剤師のほか、登録販売者も副作用等の報告を行う義務がある
2. 報告先は厚生労働大臣である
3. 報告は医薬品との因果関係が明確な場合のみ行えばよい
4. 医薬品によるものと疑われる健康被害についても報告の対象となる

<QuizAnswer>

**答え: 3**

副作用等の報告は、医薬品との因果関係が明確でなくても、医薬品によるものと疑われる場合には報告の対象となります。報告者には医師、歯科医師、薬剤師のほか、登録販売者その他の医薬関係者も含まれます。報告先は厚生労働大臣であり、実務上はPMDA（医薬品医療機器総合機構）を通じて行われます。

</QuizAnswer>

## 問4

医薬品副作用被害救済制度に関する次の記述のうち、正しいものはどれか。

1. 救済給付の請求先は厚生労働省である
2. 救済給付の請求先はPMDA（医薬品医療機器総合機構）である
3. 救済給付の請求先は都道府県知事である
4. 救済給付の請求先は製薬企業である

<QuizAnswer>

**答え: 2**

医薬品副作用被害救済制度における救済給付の請求は、健康被害を受けた本人またはその遺族が、PMDA（独立行政法人 医薬品医療機器総合機構）に対して行います。PMDAが請求を受け付けた後、厚生労働大臣の諮問機関である薬事・食品衛生審議会の答申を受けて、給付の可否が判定されます。

</QuizAnswer>

## 問5

医薬品副作用被害救済制度の給付の種類に含まれないものはどれか。

1. 医療費
2. 障害年金
3. 休業補償給付
4. 遺族年金

<QuizAnswer>

**答え: 3**

医薬品副作用被害救済制度の給付の種類は、医療費、医療手当、障害年金、障害児養育年金、遺族年金、遺族一時金、葬祭料の7種類です。「休業補償給付」は労災保険の給付であり、本制度の給付には含まれません。

</QuizAnswer>

## 問6

一般用医薬品の安全対策に関する次の記述のうち、正しいものはどれか。

1. アンプル入りかぜ薬は、成分の吸収が緩やかなため副作用が少ないとされた
2. 小柴胡湯とインターフェロン製剤の併用により間質性肺炎が報告され、併用が禁忌とされた
3. 塩酸フェニルプロパノールアミン（PPA）は、副作用の報告がなかったため販売が継続された
4. アンプル入りかぜ薬による死亡事例は報告されていない

<QuizAnswer>

**答え: 2**

小柴胡湯については、1991年に間質性肺炎の副作用が使用上の注意に記載され、その後インターフェロン製剤との併用例で間質性肺炎が報告されたことから、1994年にインターフェロン製剤との併用が禁忌とされました。アンプル入りかぜ薬は吸収が速く血中濃度が急速に上昇するため副作用を生じやすく、複数の死亡例が報告されて回収措置がとられています。

</QuizAnswer>

## 問7

一般用医薬品の添付文書の「してはいけないこと」に記載される内容として、最も適切なものはどれか。

1. 副作用と思われる症状が出た場合に相談すべき専門家の連絡先
2. 守らないと症状が悪化したり、副作用・事故等が起こりやすくなる事項
3. 効能・効果の詳細な説明
4. 医薬品の保管方法と使用期限

<QuizAnswer>

**答え: 2**

「してはいけないこと」の項目には、守らないと症状が悪化したり、副作用や事故等が起こりやすくなる事項が記載されます。具体的には、特定の人（アレルギー歴のある人等）が使用してはいけない場合や、併用してはいけない医薬品、服用後に車の運転をしてはいけない場合などが含まれます。

</QuizAnswer>

## 問8

医薬品副作用被害救済制度の対象とならないものはどれか。

1. 一般用医薬品の使用による入院を要する程度の健康被害
2. 医療用医薬品の使用による入院を要する程度の健康被害
3. 製品不良など製造販売業者の責任が明確な場合の健康被害
4. 適正な使用目的に従い適正に使用された医薬品による健康被害

<QuizAnswer>

**答え: 3**

医薬品副作用被害救済制度は、医薬品を適正に使用したにもかかわらず発生した副作用による健康被害を救済するものです。製品不良など製造販売業者の損害賠償責任が明確な場合は、本制度の対象とはなりません。また、救済給付の対象となるには、入院を要する程度の医療（入院治療相当）が必要です。

</QuizAnswer>

## 問9

一般用医薬品の製品表示に関する次の記述のうち、正しいものはどれか。

1. 使用期限の表示は医療用医薬品のみに義務づけられている
2. 一般用医薬品の外箱には、添付文書の「使用上の注意」の記載は不要である
3. 一般用医薬品には、外箱等に「使用期限」の記載が義務づけられている
4. 使用期限は開封後の品質保持期限を示している

<QuizAnswer>

**答え: 3**

一般用医薬品の外箱等には使用期限の記載が義務づけられており、適切な保存条件のもとで製造後3年を超えて性状及び品質が安定であることが確認されていない医薬品には、使用期限を表示しなければなりません。なお、使用期限は未開封状態での品質保持期限であり、開封後の期限ではない点に注意が必要です。

</QuizAnswer>

## 問10

医薬品の適正使用のための啓発活動に関する次の記述のうち、正しいものはどれか。

1. 「薬と健康の週間」は毎年3月に実施されている
2. 「6・26国際麻薬乱用撲滅デー」を踏まえ、毎年6月20日〜7月19日が「ダメ。ゼッタイ。」普及運動の期間とされている
3. 登録販売者は医薬品の適正使用の啓発活動に関与する必要はない
4. 薬物乱用防止の啓発は医療従事者のみを対象としている

<QuizAnswer>

**答え: 2**

「6・26国際麻薬乱用撲滅デー」を踏まえ、毎年6月20日〜7月19日が「ダメ。ゼッタイ。」普及運動の期間とされています。また、「薬と健康の週間」は毎年10月17日〜23日に実施されます。登録販売者を含む医薬関係者は、適正使用の啓発活動に積極的に参加・協力することが求められており、一般消費者への普及啓発も重要な役割です。

</QuizAnswer>
