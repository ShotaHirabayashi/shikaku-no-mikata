## 問1

医薬品に関する以下の記述のうち、最も適切なものはどれか。

1. 医薬品は、製造販売後も有効性や安全性に関する情報が収集され続ける。
2. 一般用医薬品は副作用のリスクがないため、誰でも安心して使用できる。
3. 医薬品は食品と同様に、安全性の確認は不要である。
4. 医薬品の効果は使用する人の状態に関係なく、常に一定である。


<QuizAnswer>

**答え: 1**

医薬品は市販後も副作用や有効性に関する情報が継続的に収集・評価されており、必要に応じて添付文書の改訂や販売中止などの措置が講じられます。一般用医薬品であっても副作用のリスクは存在し、使用者の体質や体調によって効果や副作用の現れ方は異なります。

</QuizAnswer>

## 問2

医薬品のリスク評価に関する以下の記述のうち、正しいものはどれか。

1. 医薬品の非臨床試験では、ヒトを対象とした有効性の確認が行われる。
2. 医薬品の投与量と効果の関係では、投与量を増やすほど効果が高まり続ける。
3. 医薬品の投与量が治療量上限を超えると、効果よりも有害反応が強く発現するおそれがある。
4. 一般用医薬品は医療用医薬品と異なり、リスク評価の対象外である。


<QuizAnswer>

**答え: 3**

医薬品は少量であれば薬効が期待できますが、治療量上限を超えると有害反応（中毒）が強く発現するおそれがあります。非臨床試験は動物実験等であり、ヒトを対象とするのは臨床試験です。また、一般用医薬品もリスク評価の対象です。

</QuizAnswer>

## 問3

健康食品と医薬品に関する以下の記述のうち、誤っているものはどれか。

1. 健康食品は法律上の定義がなく、一般的に用いられる用語である。
2. 医薬品的な効能効果を標榜する食品は、無承認無許可医薬品とみなされることがある。
3. 栄養機能食品は、国が定めた規格基準に適合すれば届出なしに栄養成分の機能を表示できる。
4. 健康食品であれば、医薬品と併用しても相互作用は一切起こらない。


<QuizAnswer>

**答え: 4**

健康食品であっても、医薬品と併用した場合に相互作用が生じる可能性があります。例えば、セント・ジョーンズ・ワート（セイヨウオトギリソウ）などの健康食品成分は、医薬品の代謝に影響を与えることが知られています。健康食品だから安全とは限らない点を理解しておくことが重要です。

</QuizAnswer>

## 問4

医薬品の副作用に関する以下の記述のうち、最も適切なものはどれか。

1. 副作用とは、医薬品を適正に使用した場合にのみ発生するものをいう。
2. 副作用は、アレルギー（過敏反応）によるものと、それ以外の薬理作用によるものに大別できる。
3. アレルギーは、医薬品の薬理作用の強さに比例して発生する。
4. 一般用医薬品では、重篤な副作用は起こりえない。


<QuizAnswer>

**答え: 2**

医薬品の副作用は、免疫機構の関与するアレルギー（過敏反応）と、免疫機構が関与しない薬理作用等によるものに大別されます。アレルギーは薬理作用の強さとは関係なく、微量でも発生しうるものです。また、副作用は不適正な使用の場合にも生じ、一般用医薬品でも重篤な副作用が起こる可能性があります。

</QuizAnswer>

## 問5

小児への医薬品使用に関する以下の記述のうち、正しいものはどれか。

1. 医薬品の使用上の注意において、「小児」とは一般的に7歳未満をいう。
2. 小児は大人と比べて身体の大きさに対する水分の割合が低い。
3. 小児は肝臓や腎臓の機能が未発達であるため、医薬品の成分の代謝・排泄に時間がかかることがある。
4. 5歳以上の小児であれば、大人用の医薬品を半量にして服用させてよい。


<QuizAnswer>

**答え: 3**

小児は肝臓や腎臓の機能が未発達であるため、医薬品の成分の代謝・排泄に時間がかかり、作用が強く出過ぎたり、副作用がより強く発現するおそれがあります。なお、使用上の注意における「小児」とは一般に15歳未満を指し、小児は大人と比べて体内の水分割合が高いのが特徴です。

</QuizAnswer>

## 問6

高齢者への医薬品使用に関する以下の記述のうち、誤っているものはどれか。

1. 高齢者は生理機能が衰えているため、若年者と比べて副作用が現れやすい傾向がある。
2. 医薬品の使用上の注意において、「高齢者」とは一般的におおよそ65歳以上を指す。
3. 高齢者は持病や複数の医薬品を使用していることが多く、相互作用のリスクが高い。
4. 高齢者であっても、基礎体力や生理機能の衰えには個人差がないため、一律に対応できる。


<QuizAnswer>

**答え: 4**

高齢者の基礎体力や生理機能の衰えの程度には個人差が大きいため、一般的な傾向として副作用のリスクが高いことを認識しつつも、個々の状態に応じた対応が必要です。年齢のみから一概にどの程度リスクが増大しているかを判断することは難しいとされています。

</QuizAnswer>

## 問7

妊婦・授乳婦への医薬品使用に関する以下の記述のうち、正しいものはどれか。

1. 一般用医薬品は安全性が高いため、妊婦が自己判断で使用しても問題ない。
2. ビタミンA含有製剤は、妊娠前後の大量摂取により胎児に先天異常を起こす危険性が高まるとされている。
3. 授乳中の女性が医薬品を服用しても、その成分が乳汁中に移行することはない。
4. 妊娠の有無が不明な場合は、通常通り医薬品を使用しても差し支えない。


<QuizAnswer>

**答え: 2**

ビタミンA含有製剤は、妊娠前後の期間に過剰摂取すると胎児に先天異常を起こす危険性が高まることが知られています。妊婦は一般用医薬品であっても安易に使用せず、医師等に相談することが望ましいとされています。また、授乳中の服用では成分が乳汁中に移行する可能性があるため、注意が必要です。

</QuizAnswer>

## 問8

セルフメディケーションと受診勧奨に関する以下の記述のうち、最も適切なものはどれか。

1. セルフメディケーションとは、専門家の助言を一切受けずに自分だけで病気を治すことである。
2. 一般用医薬品で対処可能な範囲を超えていると判断される場合は、医療機関の受診を勧めることが重要である。
3. 症状が重篤であっても、まずは一般用医薬品を試してから受診するよう勧めるべきである。
4. 登録販売者は、医薬品の販売のみが業務であり、受診勧奨を行う必要はない。


<QuizAnswer>

**答え: 2**

セルフメディケーションはWHOにより「自分自身の健康に責任を持ち、軽度な身体の不調は自分で手当てすること」と定義されています。登録販売者は情報提供を行う専門家として、一般用医薬品で対処できる範囲を超える症状と判断した場合には、医療機関の受診を勧めることが重要な役割の一つです。

</QuizAnswer>

## 問9

サリドマイド訴訟に関する以下の記述のうち、正しいものはどれか。

1. サリドマイドは解熱鎮痛薬として販売されていた。
2. サリドマイドにより、妊娠している女性が使用した場合に出生児に四肢欠損等の先天異常が発生した。
3. サリドマイドの光学異性体のうち、R体が催奇形性の原因とされている。
4. サリドマイドによる薬害は日本だけで発生し、海外では被害が確認されていない。


<QuizAnswer>

**答え: 2**

サリドマイドは催眠鎮静剤として販売され、胃腸薬にも配合されていました。妊娠している女性が服用したことにより、出生児に四肢欠損や耳の障害等の先天異常（サリドマイド胎芽症）が多数発生しました。催奇形性はS体によるものとされていますが、R体を分離して摂取しても体内でR体とS体は相互に転換するため、安全とはいえません。この薬害は西ドイツをはじめ世界的に発生しました。

</QuizAnswer>

## 問10

スモン訴訟に関する以下の記述のうち、誤っているものはどれか。

1. スモンの原因物質は、整腸剤として使用されていたキノホルムである。
2. スモンの症状として、初期には腹部の膨満感から始まり、激しい腹痛を伴う下痢が生じる。
3. スモンの症状は消化器症状にとどまり、神経症状や視覚障害は報告されていない。
4. スモン訴訟は、サリドマイド訴訟とともに医薬品副作用被害救済制度の創設のきっかけとなった。


<QuizAnswer>

**答え: 3**

スモン（亜急性脊髄視神経症）の症状は消化器症状にとどまらず、次第に下半身の痺れや脱力感が現れ、麻痺が上半身にも広がり、重篤な場合には失明に至ることもありました。スモン訴訟およびサリドマイド訴訟を契機として、1979年に医薬品副作用被害救済制度が創設されました。

</QuizAnswer>
