## 問1

医薬品医療機器等法（薬機法）の目的として、正しいものはどれか。

1. 医薬品の製造コストを削減し、国民が安価に医薬品を購入できるようにすること
2. 医薬品等の品質、有効性及び安全性の確保等により、保健衛生の向上を図ること
3. すべての医薬品を薬剤師の管理のもとで販売することを義務付けること
4. 医薬品の輸出入を促進し、国際的な医薬品流通を活性化すること


<QuizAnswer>

**答え: 2**

医薬品医療機器等法第1条では、医薬品、医薬部外品、化粧品、医療機器及び再生医療等製品の品質、有効性及び安全性の確保並びに保健衛生上の危害の発生・拡大の防止のために必要な規制を行い、保健衛生の向上を図ることを目的としています。製造コストの削減や輸出入の促進は法の目的に含まれません。

</QuizAnswer>

## 問2

一般用医薬品のリスク区分のうち、「その副作用等により日常生活に支障を来す程度の健康被害が生ずるおそれがある医薬品のうち、特にリスクが高いもの」に該当するのはどれか。

1. 要指導医薬品
2. 第1類医薬品
3. 第2類医薬品
4. 第3類医薬品


<QuizAnswer>

**答え: 2**

第1類医薬品は、一般用医薬品の中で特にリスクが高いものとして分類されています。販売時には薬剤師による書面を用いた情報提供が義務付けられており、登録販売者が販売することはできません。第2類医薬品は「リスクが比較的高いもの」、第3類医薬品は「リスクが比較的低いもの」に分類されます。

</QuizAnswer>

## 問3

登録販売者が販売できる医薬品の範囲として、正しいものはどれか。

1. 要指導医薬品、第1類医薬品、第2類医薬品、第3類医薬品のすべて
2. 第1類医薬品、第2類医薬品、第3類医薬品
3. 第2類医薬品及び第3類医薬品
4. 第3類医薬品のみ


<QuizAnswer>

**答え: 3**

登録販売者が販売できるのは、一般用医薬品のうち第2類医薬品及び第3類医薬品に限られます。第1類医薬品及び要指導医薬品の販売は薬剤師のみが行うことができます。これは各医薬品のリスクに応じた適切な情報提供を確保するためです。

</QuizAnswer>

## 問4

要指導医薬品に関する記述のうち、正しいものはどれか。

1. 登録販売者が対面で情報提供を行えば販売できる
2. インターネットでの販売（特定販売）が認められている
3. 薬剤師が対面で情報提供及び指導を行った上で販売しなければならない
4. 購入者の求めがあった場合にのみ情報提供を行えばよい


<QuizAnswer>

**答え: 3**

要指導医薬品は、スイッチ直後品目や劇薬など、一般用医薬品としてのリスクが確定していない医薬品が該当します。薬剤師が対面で書面を用いた情報提供及び指導を行うことが義務付けられており、インターネット販売（特定販売）は認められていません。登録販売者では販売できません。

</QuizAnswer>

## 問5

店舗販売業に関する記述のうち、誤っているものはどれか。

1. 店舗販売業の許可は、店舗ごとに、その店舗の所在地の都道府県知事が与える
2. 店舗販売業では、要指導医薬品及び一般用医薬品を販売することができる
3. 店舗販売業では、医療用医薬品の販売も行うことができる
4. 店舗販売業者は、店舗管理者を置かなければならない


<QuizAnswer>

**答え: 3**

店舗販売業では、要指導医薬品及び一般用医薬品のみを販売することができ、医療用医薬品を販売することはできません。医療用医薬品を取り扱うには薬局の開設許可が必要です。店舗販売業の許可は店舗ごとに都道府県知事（保健所設置市・特別区の場合はその長）が与え、店舗管理者の設置も必要です。

</QuizAnswer>

## 問6

医薬品の販売業の許可の種類として、医薬品医療機器等法に規定されていないものはどれか。

1. 店舗販売業
2. 配置販売業
3. 卸売販売業
4. 通信販売業


<QuizAnswer>

**答え: 4**

医薬品医療機器等法に規定されている医薬品販売業の許可は、「店舗販売業」「配置販売業」「卸売販売業」の3種類です。「通信販売業」という許可区分は存在しません。なお、一般用医薬品のインターネット販売は、店舗販売業の許可を受けた上で「特定販売」として行うことができます。

</QuizAnswer>

## 問7

医薬部外品に関する記述のうち、正しいものはどれか。

1. 医薬部外品は、医薬品と同様に効能効果の承認を受ける必要はない
2. 医薬部外品は、人体に対する作用が緩和なものであり、販売に許可は不要である
3. 医薬部外品には、吐きけその他の不快感の防止を目的とするものが含まれる
4. 医薬部外品は、薬剤師又は登録販売者でなければ販売できない


<QuizAnswer>

**答え: 3**

医薬部外品の定義には、吐きけその他の不快感又は口臭・体臭の防止、あせも・ただれ等の防止、脱毛の防止・育毛・除毛などの目的で使用され、人体に対する作用が緩和なものが含まれます。医薬部外品は製造販売にあたり承認が必要であり、販売には一般小売店でも可能ですが、製造販売には許可が必要です。

</QuizAnswer>

## 問8

一般用医薬品の第2類医薬品の販売時における情報提供について、正しいものはどれか。

1. 薬剤師又は登録販売者が、書面を用いて情報提供を行うことが義務付けられている
2. 薬剤師又は登録販売者が、情報提供に努めなければならない（努力義務）
3. 購入者から質問がない限り、情報提供を行う必要はない
4. 薬剤師のみが情報提供を行うことができる


<QuizAnswer>

**答え: 2**

第2類医薬品の販売時の情報提供は「努力義務」とされています。第1類医薬品は薬剤師による書面を用いた情報提供が「義務」であるのに対し、第2類医薬品は薬剤師又は登録販売者による情報提供の「努力義務」、第3類医薬品は法律上の情報提供の規定はありません（求めがあれば対応）。

</QuizAnswer>

## 問9

医薬品の容器・外箱等への法定表示事項として、正しいものはどれか。

1. 製造販売業者の氏名又は名称及び住所
2. 医薬品の市場での販売価格
3. 医薬品を推奨する医師の氏名
4. 製造に使用した原材料の産地名


<QuizAnswer>

**答え: 1**

医薬品医療機器等法では、医薬品の直接の容器又は直接の被包に、製造販売業者の氏名又は名称及び住所を記載することが義務付けられています。そのほか、名称、成分・分量、用法・用量、製造番号又は製造記号、使用期限なども法定表示事項に含まれます。販売価格、推奨医師名、原材料の産地名は法定表示事項ではありません。

</QuizAnswer>

## 問10

医薬品の広告に関する規制について、誤っているものはどれか。

1. 医薬品の虚偽又は誇大な広告は禁止されている
2. 承認前の医薬品について、その名称や効能効果に関する広告をしてはならない
3. 一般用医薬品であれば、医薬関係者が推薦している旨の広告を自由に行える
4. 医薬品等の広告規制は、医薬品医療機器等法第66条から第68条に規定されている


<QuizAnswer>

**答え: 3**

医薬品医療機器等法第66条では、医薬品等について虚偽又は誇大な広告を禁止しており、医薬関係者が保証したと誤解されるおそれのある記事を広告することも禁じられています。一般用医薬品であっても、医師や薬剤師などが推薦している旨の広告を自由に行うことはできません。第68条では承認前の医薬品の広告も禁止されています。

</QuizAnswer>
