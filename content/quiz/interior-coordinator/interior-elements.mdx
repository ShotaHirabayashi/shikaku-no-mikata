## 問1

ウィンドウトリートメントのうち、羽根（スラット）の角度を調整することで光量や視線をコントロールでき、横型のものを特に何と呼ぶか。

1. ロールスクリーン
2. ローマンシェード
3. ベネシャンブラインド
4. プリーツスクリーン


<QuizAnswer>

**答え: 3**

ベネシャンブラインドは横型のブラインドで、スラット（羽根）を回転させて光量や視線を調整できます。一方、縦型のブラインドはバーチカルブラインドと呼ばれます。ロールスクリーンはロール状の布を巻き上げて開閉するもの、ローマンシェードは布をたたみ上げるタイプのウィンドウトリートメントです。

</QuizAnswer>

## 問2

カーペットの製法のうち、基布にミシン針でパイルを刺し込み、裏面にラテックスなどの接着剤を塗布して固定する方法はどれか。

1. ウィルトン
2. アキスミンスター
3. タフテッド
4. ニードルパンチ


<QuizAnswer>

**答え: 3**

タフテッドカーペットは、基布にミシン針でパイルを刺し込み、裏面にラテックスなどの接着剤をコーティングしてパイルを固定する製法です。アメリカで20世紀に開発され、織りカーペットの約30倍の生産速度で大量生産が可能なため、現在最も普及している製法です。

</QuizAnswer>

## 問3

照明器具のうち、天井に埋め込んで設置するため器具の存在感が少なく、空間をすっきりと見せることができるものはどれか。

1. シーリングライト
2. ペンダントライト
3. ブラケットライト
4. ダウンライト


<QuizAnswer>

**答え: 4**

ダウンライトは天井に埋め込んで設置する照明器具で、器具自体が目立たないため空間をすっきりと広く見せる効果があります。シーリングライトは天井面に直付けする照明、ペンダントライトは天井から吊り下げる照明、ブラケットライトは壁面に取り付ける照明です。

</QuizAnswer>

## 問4

キッチンの作業効率を高めるために、シンク・コンロ・冷蔵庫の3点を結んでできる三角形のことを何と呼ぶか。

1. アクティビティゾーン
2. ワークトライアングル
3. サービスライン
4. ユーティリティトライアングル


<QuizAnswer>

**答え: 2**

ワークトライアングルとは、キッチンにおけるシンク・コンロ・冷蔵庫の3つの中心点を結んでできる三角形のことです。この三角形が正三角形に近く、3辺の合計が360cm〜660cm程度であると作業効率がよいとされています。キッチンレイアウトの基本的な考え方として重要です。

</QuizAnswer>

## 問5

カーペットのパイル形状のうち、毛先がループ状になっており、耐久性に優れゴミが入りにくい特徴を持つものはどれか。

1. カットパイル
2. ループパイル
3. シャギーパイル
4. フリーズパイル


<QuizAnswer>

**答え: 2**

ループパイルは毛先がループ（輪奈）状になっているパイルで、耐久性に優れ、毛足にゴミが入りにくく掃除が容易であるなど実用性が高いのが特徴です。カットパイルはループの先端をカットしたもので柔らかな肌触り、シャギーパイルはカットパイルの中でも毛足が25mm以上と長いものを指します。

</QuizAnswer>

## 問6

壁装材のうち、ビニル樹脂を主原料とし、施工性がよく色柄が豊富で、住宅の内装仕上げに最も多く使用されているものはどれか。

1. 織物壁紙
2. 紙壁紙
3. ビニルクロス
4. 珪藻土壁紙


<QuizAnswer>

**答え: 3**

ビニルクロス（ビニル壁紙）は、塩化ビニル樹脂を主原料とした壁装材で、日本の住宅内装に最も多く使用されています。色柄のバリエーションが非常に豊富で、耐水性・耐久性に優れ、施工性がよく価格も比較的安価であることが普及の理由です。

</QuizAnswer>

## 問7

家具の構造で、板状の部材を組み合わせて箱形に構成する方式を何と呼ぶか。

1. 脚物家具
2. 箱物家具
3. 張り物家具
4. 吊り物家具


<QuizAnswer>

**答え: 2**

家具は構造により大きく「脚物家具」と「箱物家具」に分類されます。箱物家具は板状の部材を組み合わせて箱形に構成するもので、タンス・食器棚・本棚などが該当します。脚物家具は脚部を持つ椅子やテーブルなどを指し、張り物家具はソファなど詰め物と張り地で構成される家具の分類です。

</QuizAnswer>

## 問8

浴室の壁・天井などに多く用いられ、目地が少なく防水性に優れた成形パネルを組み合わせて構成する浴室の方式はどれか。

1. 在来工法浴室
2. ユニットバス（システムバス）
3. ハーフユニットバス
4. 露天風呂式浴室


<QuizAnswer>

**答え: 2**

ユニットバス（システムバス）は、工場で成形された防水性の高いパネル（壁・床・天井）と浴槽を現場で組み立てる方式です。目地が少なく防水性に優れ、施工期間が短いのが特徴です。在来工法浴室は現場でモルタルやタイルを施工する従来の方法、ハーフユニットバスは浴槽と洗い場のみがユニット化されたものです。

</QuizAnswer>

## 問9

照明器具のうち、天井から吊り下げて使用し、ダイニングテーブルの上などに設置されることが多い器具はどれか。

1. フットライト
2. スポットライト
3. ペンダントライト
4. シーリングライト


<QuizAnswer>

**答え: 3**

ペンダントライトは天井からコードやチェーンで吊り下げるタイプの照明器具で、ダイニングテーブルの上に設置して食卓を照らす用途に多く使われます。光源の位置が低くなるため、テーブル面を効果的に照らし、食事の雰囲気を演出するのに適しています。シーリングライトは天井直付けで部屋全体を均一に照らす器具です。

</QuizAnswer>

## 問10

ウィンドウトリートメントのうち、布をロール状に巻き上げて開閉し、窓まわりだけでなく間仕切りや収納の目隠しにも使用されるものはどれか。

1. カフェカーテン
2. ロールスクリーン
3. バーチカルブラインド
4. スワッグバランス


<QuizAnswer>

**答え: 2**

ロールスクリーンは、ロール状に巻かれた布を引き下げたり巻き上げたりして開閉するウィンドウトリートメントです。窓まわりの遮光・調光だけでなく、部屋の間仕切りや収納スペースの目隠しとしても活用されます。フラットな見た目でシンプルな印象を与え、省スペースであることも特徴です。

</QuizAnswer>
