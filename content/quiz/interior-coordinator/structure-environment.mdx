## 問1

木造軸組工法（在来工法）において、地震や風などの水平力に抵抗するために、柱と梁で構成される軸組の中に斜めに入れる補強材を何というか。

1. 火打ち梁
2. 筋かい（筋交い）
3. 間柱
4. 大引

<QuizAnswer>

**答え: 2**

筋かい（筋交い）は、柱と梁で構成された四角形の枠内に斜めに取り付ける補強材で、地震や風などの水平力に対する耐力を確保する役割があります。火打ち梁は床面の水平剛性を高める斜材、間柱は壁下地を取り付けるための補助的な柱、大引は1階床を支える水平部材です。

</QuizAnswer>

## 問2

鉄筋コンクリート造（RC造）の構造形式のうち、柱や梁を用いず耐力壁と床スラブで建物を支える構造はどれか。

1. トラス構造
2. ラーメン構造
3. 壁式構造
4. ブレース構造

<QuizAnswer>

**答え: 3**

壁式構造は、柱・梁を設けず鉄筋コンクリート製の耐力壁と床スラブで建物を支える構造形式です。室内に柱や梁の出っ張りがなくすっきりした空間が得られますが、耐力壁を撤去できないため間取り変更の自由度は低くなります。一般に5階以下の低中層建物に適用されます。

</QuizAnswer>

## 問3

断熱工法に関する次の記述のうち、最も適当なものはどれか。

1. 充填断熱工法は、構造躯体の外側全面を断熱材で覆う工法である
2. 外張り断熱工法は、柱と柱の間に断熱材を充填する工法である
3. 充填断熱工法は、柱部分で断熱材が途切れるためヒートブリッジが生じやすい
4. 外張り断熱工法は、充填断熱工法に比べてコストが低い

<QuizAnswer>

**答え: 3**

充填断熱工法は柱や間柱の間に断熱材を充填する工法であり、柱などの構造部材の部分で断熱材が途切れるため、そこが熱橋（ヒートブリッジ）となり熱が伝わりやすくなります。外張り断熱工法は躯体の外側全面を断熱材で覆うため熱橋が生じにくいですが、コストは充填断熱より高くなります。

</QuizAnswer>

## 問4

昼光率に関する次の記述のうち、正しいものはどれか。

1. 昼光率は、室内のある点の照度を直射日光による屋外照度で割った値である
2. 昼光率は、天候や時刻によって大きく変動する
3. 昼光率は、室内のある点の照度を全天空照度で割った百分率である
4. 昼光率は、数値が小さいほど採光条件がよいことを示す

<QuizAnswer>

**答え: 3**

昼光率は、室内のある点における照度を、屋外の全天空照度（直射日光を除いた天空光のみによる水平面照度）で割り、百分率で表したものです。昼光率は窓の大きさ・位置・室内の反射率など建築的条件で決まるため、天候や時刻が変わっても値はほぼ一定という特性があります。数値が大きいほど採光条件が良好です。

</QuizAnswer>

## 問5

床衝撃音の遮音性能を示すL値について、正しい記述はどれか。

1. L値は数値が大きいほど遮音性能が高い
2. L値は壁の空気伝搬音に対する遮音性能を示す
3. L値は数値が小さいほど遮音性能が高い
4. L値はドアやサッシの遮音性能を示す

<QuizAnswer>

**答え: 3**

L値は上階から下階への床衝撃音の伝わりやすさを示す指標で、数値が小さいほど遮音性能が高いことを意味します（例：L-40はL-60より静か）。壁の空気伝搬音に対する遮音性能はD値、ドアやサッシの遮音性能はT値で表されます。なお、LH（重量床衝撃音）とLL（軽量床衝撃音）の区分があります。

</QuizAnswer>

## 問6

24時間換気システムにおいて、住宅で最も一般的に採用されている換気方式はどれか。

1. 第1種換気（機械給気＋機械排気）
2. 第2種換気（機械給気＋自然排気）
3. 第3種換気（自然給気＋機械排気）
4. 第4種換気（自然給気＋自然排気）

<QuizAnswer>

**答え: 3**

第3種換気は、排気を機械ファンで行い給気は各居室の給気口から自然に取り込む方式で、住宅で最も一般的に採用されています。室内が負圧になるため壁内結露が起きにくく、設備費・電気代ともに低コストです。第2種換気は室内が正圧となり病院のクリーンルーム等に使われますが、住宅では壁内結露のリスクがあるためほとんど使われません。

</QuizAnswer>

## 問7

熱貫流率（U値）に関する記述として、最も適当なものはどれか。

1. U値の単位はW/(m・K)で、数値が大きいほど断熱性能が高い
2. U値の単位はW/(m²・K)で、数値が小さいほど断熱性能が高い
3. U値の単位はW/(m²・K)で、数値が大きいほど断熱性能が高い
4. U値の単位はJ/(m²・K)で、数値が小さいほど断熱性能が高い

<QuizAnswer>

**答え: 2**

熱貫流率（U値）は、室内外の温度差が1K（1℃）のとき、1時間に壁体1m²を通過する熱量を表す値で、単位はW/(m²・K)です。数値が小さいほど熱を伝えにくく、断熱性能が高いことを意味します。かつては「K値」と呼ばれていましたが、2009年の改正省エネ法以降「U値」に統一されました。

</QuizAnswer>

## 問8

鉄骨造（S造）の特徴に関する記述のうち、誤っているものはどれか。

1. 鉄骨は熱に弱く、高温になると強度が低下するため耐火被覆が必要である
2. 工場で加工された鋼材を使用するため、品質が安定している
3. 鉄骨造は木造に比べて遮音性能が高い
4. 重量鉄骨造は大スパンの空間を実現しやすい

<QuizAnswer>

**答え: 3**

鉄骨造は一般に遮音性能が低く、木造と比べても生活音が伝わりやすいという特徴があります。鉄骨は熱伝導率が高いため音が伝わりやすく、遮音対策が必要です。一方、工場加工による品質安定性、耐火被覆の必要性、重量鉄骨による大空間の実現はいずれも鉄骨造の正しい特徴です。

</QuizAnswer>

## 問9

住宅の給水方式のうち、受水槽を設けず水道本管の水圧のみで直接給水する方式はどれか。

1. 高置水槽方式
2. ポンプ直送方式
3. 直結直圧方式
4. 増圧直結方式

<QuizAnswer>

**答え: 3**

直結直圧方式は、水道本管の水圧だけで蛇口まで直接給水する方式で、受水槽やポンプが不要です。3階建て程度までの低層建物や戸建住宅に適しています。増圧直結方式はポンプで加圧するため受水槽は不要ですがポンプは必要です。高置水槽方式とポンプ直送方式はいずれも受水槽を使用します。

</QuizAnswer>

## 問10

住宅の分電盤に収められているブレーカーのうち、配線や電気機器の漏電を検知して電路を遮断し、感電事故や漏電火災を防止する役割をもつものはどれか。

1. アンペアブレーカー（サービスブレーカー）
2. 安全ブレーカー（配線用遮断器）
3. 漏電遮断器（漏電ブレーカー）
4. ノーヒューズブレーカー

<QuizAnswer>

**答え: 3**

漏電遮断器（漏電ブレーカー）は、配線や電気機器の漏電（絶縁不良による電流漏れ）を零相変流器（ZCT）で検出し、自動的に電路を遮断して感電事故や漏電火災を防止します。一般住宅用では定格感度電流30mA、動作時間0.1秒以内のものが使用されます。アンペアブレーカーは契約容量超過時、安全ブレーカーは各回路の過電流・短絡時にそれぞれ作動します。

</QuizAnswer>
