## 問1

インテリアコーディネーター資格試験制度が創設された年として正しいものはどれか。

1. 1973年（昭和48年）
2. 1978年（昭和53年）
3. 1983年（昭和58年）
4. 1988年（昭和63年）


<QuizAnswer>

**答え: 3**

インテリアコーディネーター資格試験は、1983年（昭和58年）に通商産業省（現・経済産業省）の許可を得て設立された社団法人インテリア産業協会によって創設されました。インテリア産業界で唯一の横断的組織として、資格認定事業を中心にインテリアの普及促進を行っています。

</QuizAnswer>

## 問2

1973年に通商産業省（現・経済産業省）に設置され、「インテリア」という言葉が広く使われるきっかけとなった組織はどれか。

1. インテリアデザイン推進協議会
2. インテリア産業振興対策委員会
3. 住宅インテリア研究会
4. 住環境整備審議会


<QuizAnswer>

**答え: 2**

1973年（昭和48年）に通商産業省にインテリア産業振興対策委員会が設置されたことで、「インテリア」という言葉が一般的に使われるようになりました。これがインテリア産業の振興と、後のインテリアコーディネーター誕生への道筋をつけた重要な出来事です。

</QuizAnswer>

## 問3

インテリアコーディネーター誕生の社会的背景として、最も適切なものはどれか。

1. バブル経済による高級輸入家具の需要増大
2. 核家族化の進行やLDK間取りの普及による住空間への関心の高まり
3. 海外からのインテリアデザイナーの大量流入
4. 政府による住宅建設の全面禁止と再開


<QuizAnswer>

**答え: 2**

1970年代に団地やマンションの建設が進み、LDKという間取りや洋室が一般化しました。同時に核家族化が進行し、家電・家具の需要が高まる中で、住空間のインテリアに対する関心が高まりました。こうした消費者の要望に応える専門家として、インテリアコーディネーターが誕生しました。

</QuizAnswer>

## 問4

インテリアコーディネーターの業務の流れとして、最初に行うべきものはどれか。

1. 見積書の作成
2. 施工管理と現場監理
3. 顧客へのヒアリングと要望の把握
4. インテリアエレメントの発注


<QuizAnswer>

**答え: 3**

インテリアコーディネーターの業務は、まず顧客へのヒアリングから始まります。顧客の好み、ライフスタイル、予算、どのような空間で過ごしたいかなどを丁寧に聞き取ることが、適切なインテリア計画の第一歩です。その後、プランニング、プレゼンテーション、見積もり、発注、施工管理という流れで業務を進めます。

</QuizAnswer>

## 問5

インテリアコーディネーターの資格試験を実施している団体はどれか。

1. 一般社団法人日本建築士会連合会
2. 公益社団法人インテリア産業協会
3. 一般財団法人住宅金融普及協会
4. 公益社団法人日本インテリアデザイナー協会


<QuizAnswer>

**答え: 2**

インテリアコーディネーター資格試験は、公益社団法人インテリア産業協会が実施しています。同協会は1983年に設立され、2012年に公益社団法人に移行しました。インテリアコーディネーター（IC）とキッチンスペシャリスト（KS）の資格認定を主な事業としています。

</QuizAnswer>

## 問6

インテリアコーディネーターの職能に関する説明として、最も適切なものはどれか。

1. 建築の構造設計を行い、建築確認申請を提出する
2. 幅広い商品知識を持ち、住まい手にとって快適な生活空間を総合的にコーディネートする
3. 不動産の売買契約に関する重要事項説明を行う
4. 建築工事の施工についての技術的な総括管理を行う


<QuizAnswer>

**答え: 2**

インテリアコーディネーターは、インテリアに関する幅広い商品知識を持ち、住まい手が求める生活空間を総合的にコーディネートする専門家です。建築確認申請は建築士、重要事項説明は宅地建物取引士、施工の技術的総括は施工管理技士の業務であり、ICの職能とは異なります。

</QuizAnswer>

## 問7

インテリアコーディネーターが活躍する場として、適切でないものはどれか。

1. 住宅メーカーやリフォーム会社
2. インテリアショップやショールーム
3. 不動産登記を行う法務局
4. 設計事務所やデザイン事務所


<QuizAnswer>

**答え: 3**

インテリアコーディネーターは、住宅メーカー、リフォーム会社、インテリアショップ、ショールーム、設計事務所、デザイン事務所など、住空間や商業空間に関わる幅広い場で活躍しています。不動産登記を行う法務局はインテリアコーディネーターの業務とは関係がありません。

</QuizAnswer>

## 問8

インテリアコーディネーターが顧客にプレゼンテーションを行う際に用いる手法として、最も適切でないものはどれか。

1. イメージボードやサンプルボードの作成
2. 平面図やパースを用いた空間提案
3. 素材サンプルやカタログを用いた説明
4. 建築基準法に基づく構造計算書の提示


<QuizAnswer>

**答え: 4**

インテリアコーディネーターのプレゼンテーションでは、イメージボードやサンプルボード、平面図・パース、素材サンプル・カタログなどを活用して、顧客にわかりやすく空間提案を行います。構造計算書の作成・提示は建築士の業務であり、ICのプレゼンテーション手法には含まれません。

</QuizAnswer>

## 問9

インテリア産業の流通において、メーカーから消費者に商品が届くまでの一般的な流通経路として、最も適切なものはどれか。

1. メーカー → 消費者（すべて直販）
2. メーカー → 卸売業者 → 小売業者 → 消費者
3. 消費者 → 小売業者 → メーカー
4. 卸売業者 → メーカー → 小売業者 → 消費者


<QuizAnswer>

**答え: 2**

インテリア産業における一般的な流通経路は、メーカーが製造した商品が卸売業者（問屋）を経由し、小売業者（インテリアショップ・百貨店等）を通じて消費者に届く形です。近年は直販やネット販売も増えていますが、基本的な流通構造としてはメーカー→卸→小売→消費者の経路が基本です。

</QuizAnswer>

## 問10

インテリアコーディネーターの業務におけるコンサルティングの説明として、最も適切なものはどれか。

1. 顧客の要望を聞かず、コーディネーター自身の好みだけでプランを決定する
2. 顧客のライフスタイルや好み、予算などを把握し、専門的な立場からインテリアの助言・提案を行う
3. 商品の販売のみを行い、空間全体の提案は行わない
4. 建築工事の請負契約の締結を主な目的とする


<QuizAnswer>

**答え: 2**

インテリアコーディネーターのコンサルティングとは、顧客のライフスタイル、好み、家族構成、予算などを丁寧にヒアリングした上で、専門的な知識と経験に基づいてインテリア空間全体の助言・提案を行うことです。一方的に決めるのではなく、顧客との対話を通じて最適な住空間を実現することがICの重要な役割です。

</QuizAnswer>
