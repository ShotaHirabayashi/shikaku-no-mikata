## 問1

19世紀後半のイギリスで、産業革命による大量生産品の質の低下に反発し、中世の手仕事の美しさを見直すことを提唱した運動はどれか。

1. アール・ヌーヴォー
2. バウハウス
3. アーツ・アンド・クラフツ運動
4. デ・スティル


<QuizAnswer>

**答え: 3**

アーツ・アンド・クラフツ運動は、ウィリアム・モリスを中心に19世紀後半のイギリスで起こった運動です。産業革命による機械生産で粗悪な製品が大量に出回ったことに対し、中世の手工芸の美しさと生活と芸術の調和を目指しました。この運動は後のアール・ヌーヴォーやドイツ工作連盟など、各国のデザイン運動に大きな影響を与えました。

</QuizAnswer>

## 問2

1929年のバルセロナ万国博覧会のドイツ館のために、スペイン国王夫妻を迎える椅子としてデザインされた「バルセロナチェア」の作者は誰か。

1. ル・コルビュジエ
2. ミース・ファン・デル・ローエ
3. チャールズ・イームズ
4. アルヴァ・アアルト


<QuizAnswer>

**答え: 2**

バルセロナチェアは、ミース・ファン・デル・ローエが1929年のバルセロナ万国博覧会ドイツ館のためにデザインした椅子です。スペイン国王アルフォンソ13世夫妻のために制作され、フラットバーのスチールフレームと革張りの座面が特徴的なモダニズムの名作家具として知られています。

</QuizAnswer>

## 問3

平安時代の貴族の住宅様式で、南面する主屋（寝殿）を中心に東西の対屋を渡殿でつなぎ、南に庭園と池を配した様式はどれか。

1. 書院造
2. 数寄屋造
3. 寝殿造
4. 町家造


<QuizAnswer>

**答え: 3**

寝殿造は平安時代に成立した上流貴族の住宅様式です。中央に主人の居所である寝殿を配し、左右対称に対屋（たいのや）を設け、それらを渡殿（わたどの）でつなぐ構成が特徴です。自然との調和を重視し、南庭には池を設けた優雅な住まいでした。

</QuizAnswer>

## 問4

アール・ヌーヴォーの装飾的特徴として、最も適切なものはどれか。

1. 直線と幾何学模様を多用した合理的なデザイン
2. 古代ギリシャ・ローマの古典様式を忠実に再現したデザイン
3. 植物や昆虫など自然をモチーフとした優美な曲線によるデザイン
4. 装飾を一切排除した機能主義的なデザイン


<QuizAnswer>

**答え: 3**

アール・ヌーヴォー（「新しい芸術」の意味）は、19世紀末から20世紀初頭にかけてヨーロッパで流行した美術様式です。草花や昆虫など自然界のモチーフから着想を得た、流れるような優美な曲線が最大の特徴です。一方、その後に登場するアール・デコは直線的・幾何学的なデザインが特徴であり、対照的な様式として区別されます。

</QuizAnswer>

## 問5

1919年にドイツのワイマールで「芸術と技術の新しい統一」を理念として設立された造形学校はどれか。

1. ウルム造形大学
2. バウハウス
3. エコール・デ・ボザール
4. ロイヤル・カレッジ・オブ・アート


<QuizAnswer>

**答え: 2**

バウハウスは、1919年にヴァルター・グロピウスがドイツのワイマールに設立した国立造形学校です。美術学校と工芸学校を統合し、「芸術と技術の新しい統一」を基本理念として掲げました。マルセル・ブロイヤーのワシリーチェアなど、多くの名作家具や建築を生み出し、近代デザインに多大な影響を与えました。

</QuizAnswer>

## 問6

ル・コルビュジエがピエール・ジャンヌレ、シャルロット・ペリアンと共同でデザインし、「グランコンフォール（大いなる快適）」の通称で知られるソファはどれか。

1. LC2
2. エッグチェア
3. ワシリーチェア
4. スワンチェア


<QuizAnswer>

**答え: 1**

LC2は1928年にル・コルビュジエ、ピエール・ジャンヌレ、シャルロット・ペリアンの共同デザインにより制作されたソファです。「最小の構成で最大の快適性」を目指してデザインされ、通称「グランコンフォール（大いなる快適）」と呼ばれています。ニューヨーク近代美術館（MoMA）に永久保存されている20世紀を代表する名作家具です。

</QuizAnswer>

## 問7

室町時代から近世初頭にかけて成立した日本の住宅様式で、床の間・違棚・付書院などの座敷飾りを備え、武家の格式を表現した様式はどれか。

1. 寝殿造
2. 書院造
3. 数寄屋造
4. 主殿造


<QuizAnswer>

**答え: 2**

書院造は室町時代から近世初頭にかけて成立した武家住宅の様式です。床の間、違棚（ちがいだな）、付書院（つけしょいん）などの座敷飾りを備え、主人の権威と格式を表現しました。寝殿造の流れを汲みながらも、武家社会の秩序を反映した様式であり、後の日本住宅の基本形に大きな影響を与えました。

</QuizAnswer>

## 問8

チャールズ＆レイ・イームズ夫妻が活躍した、1950年代を中心とするアメリカのデザイン潮流を何と呼ぶか。

1. ポストモダン
2. アール・デコ
3. ミッドセンチュリーモダン
4. ハイテクスタイル


<QuizAnswer>

**答え: 3**

ミッドセンチュリーモダンは、1940年代後半から1960年代にかけてのアメリカを中心としたデザイン潮流を指します。チャールズ＆レイ・イームズ夫妻はその代表的デザイナーであり、ハーマンミラー社から数多くの家具を発表しました。成形合板やFRP（繊維強化プラスチック）など新素材を活用した、有機的で親しみやすいデザインが特徴です。

</QuizAnswer>

## 問9

アーツ・アンド・クラフツ運動の影響を受け、ドイツで建築家や工芸家が中心となり、芸術と産業の融合を目指して1907年に結成された団体はどれか。

1. ウィーン分離派
2. ドイツ工作連盟
3. デ・スティル
4. シカゴ派


<QuizAnswer>

**答え: 2**

ドイツ工作連盟（ドイツャー・ヴェルクブント）は、1907年にヘルマン・ムテジウスらを中心に結成された団体です。イギリスのアーツ・アンド・クラフツ運動の影響を受けつつ、手工芸の復興ではなく機械生産と芸術の融合を目指しました。この運動は後にヴァルター・グロピウスによるバウハウスの設立へとつながっていきます。

</QuizAnswer>

## 問10

書院造の格式を排し、茶の湯の「わび」の精神を反映して、丸太や竹などの自然素材を用い、自由で洗練された意匠を特徴とする日本の建築様式はどれか。

1. 寝殿造
2. 町家造
3. 書院造
4. 数寄屋造


<QuizAnswer>

**答え: 4**

数寄屋造は、安土桃山時代に成立した建築様式で、「書院造に草庵風茶室の手法を取り入れた様式」と定義されます。千利休が大成した「わび茶」の精神を反映し、書院造が重んじた格式や様式を極力排した質素で洗練されたデザインが特徴です。丸太や竹、漆喰壁など自然素材を活かし、自由度の高い意匠が用いられました。

</QuizAnswer>
