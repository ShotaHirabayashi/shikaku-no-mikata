## 問1

製図において、平面図は建物をある高さで水平に切断して上から見下ろした図であるが、一般的にその切断する高さはどれか。

1. 床面から約0.5mの高さ
2. 床面から約1.2mの高さ
3. 床面から約2.0mの高さ
4. 床面から約2.5mの高さ


<QuizAnswer>

**答え: 2**

平面図は、床面から約1.2m（FL+1,200mm）の高さで水平に切断し、上から見下ろした図です。この高さにすることで、窓や出入口などの開口部が切断面に含まれ、間取り・壁・建具の配置が適切に表現されます。インテリアコーディネーターの製図では基本中の基本となる知識です。

</QuizAnswer>

## 問2

製図で使用される線の種類について、正しい説明はどれか。

1. 一点鎖線は、隠れて見えない部分の輪郭を示す
2. 破線は、中心線や基準線を示す
3. 太い実線は、切断面の輪郭線を示す
4. 二点鎖線は、寸法線や引出線を示す


<QuizAnswer>

**答え: 3**

太い実線は切断面の輪郭線や壁の断面など、主要な形状を表す線として使用されます。破線は隠れて見えない部分を示し、一点鎖線は中心線・基準線・切断位置を示します。二点鎖線は仮想線や将来の計画線を表します。線の種類と用途の対応は製図の基本であり、頻出項目です。

</QuizAnswer>

## 問3

透視図法に関する記述として、正しいものはどれか。

1. 一点透視図法は、消失点が2つあり、建物の外観を描くのに適している
2. 二点透視図法は、垂直線のみが平行で、左右2つの消失点に水平方向の線が収束する
3. アイソメトリック図（等角投影図）は、消失点が1つあり、遠近感が表現される
4. 三点透視図法は、消失点が2つで、見上げや見下ろしの構図に用いる


<QuizAnswer>

**答え: 2**

二点透視図法は左右に2つの消失点（VP）を持ち、垂直線のみが平行に描かれます。建物外観や部屋のコーナーを斜めから見た構図に適しています。一点透視図法は消失点が1つで室内正面の構図に適し、三点透視図法は消失点が3つで見上げ・見下ろしの構図に用います。アイソメトリック図は消失点を持たない平行投影法です。

</QuizAnswer>

## 問4

建築基準法における内装制限に関する記述として、正しいものはどれか。

1. 内装制限は壁・天井・床のすべてに適用される
2. 内装制限の対象となる壁は、床面からの高さにかかわらず全面が対象である
3. 内装制限は壁と天井が対象であり、床は対象外である
4. IHクッキングヒーターを使用するキッチンは火気使用室に該当する


<QuizAnswer>

**答え: 3**

建築基準法第35条の2に基づく内装制限は、壁（床面から1.2mを超える部分）と天井の仕上げが対象であり、床は対象外です。これは試験でも頻出のひっかけポイントです。また、壁についても床面から1.2m以下の腰壁部分は制限の対象外となります。IHクッキングヒーターは電気調理器であるため、火気使用室には該当しません。

</QuizAnswer>

## 問5

建築基準法のシックハウス対策に関する記述として、**誤っている**ものはどれか。

1. ホルムアルデヒド発散建材の等級で、F☆☆☆☆は使用面積の制限なく使用できる
2. クロルピリホスを含む建材は、居室への使用が全面禁止されている
3. 住宅の居室には、換気回数0.5回/h以上の24時間機械換気設備の設置が義務付けられている
4. シックハウス対策の規制は新築住宅のみが対象であり、リフォームには適用されない


<QuizAnswer>

**答え: 4**

建築基準法第28条の2に基づくシックハウス対策は、新築・リフォームを問わず全ての建築物が対象です。F☆☆☆☆（フォースター）は放散速度が最も低く使用面積の制限がありません。クロルピリホス（有機リン系防蟻剤）は全面使用禁止です。住宅の居室には換気回数0.5回/h以上の24時間機械換気設備の設置が義務付けられています。

</QuizAnswer>

## 問6

消防法の防炎規制に関する記述として、正しいものはどれか。

1. 防炎規制は一般住宅を含むすべての建物に適用される
2. カーテンは防炎対象物品であるが、レースカーテンは対象外である
3. 壁紙（クロス）は防炎対象物品に含まれる
4. 高さ31mを超える高層建築物は防炎防火対象物に該当する


<QuizAnswer>

**答え: 4**

消防法の防炎規制では、高さ31mを超える高層建築物のほか、地下街・劇場・病院・ホテル・百貨店などが防炎防火対象物に該当します。レースカーテンを含むカーテン全般、布製ブラインド、じゅうたん（2㎡超）などが防炎対象物品です。壁紙・クロスは防炎対象物品には含まれません（壁・天井の仕上げは建築基準法の内装制限の対象）。

</QuizAnswer>

## 問7

製品安全や品質に関するマークの説明として、正しいものはどれか。

1. SGマークは、経済産業省が交付する鉱工業品の品質規格マークである
2. BLマークは、農林水産物の品質を保証する農林水産省管轄のマークである
3. SGマークが付いた製品には、製品の欠陥による人身事故に対する賠償措置が講じられている
4. JISマークは「日本工業規格」の略称で、農林水産品を対象としている


<QuizAnswer>

**答え: 3**

SGマーク（Safe Goods）は製品安全協会が認証するマークで、最大の特徴は対人賠償保険が付帯していることです。製品の欠陥により人身被害が発生した場合、賠償措置が講じられます。JISは2019年に「日本工業規格」から「日本産業規格」に名称変更され、鉱工業品全般が対象です。BLマーク（Better Living）は住宅部品の品質・性能を評価するマークです。

</QuizAnswer>

## 問8

住宅品確法（住宅の品質確保の促進等に関する法律）に関する記述として、正しいものはどれか。

1. 新築住宅の瑕疵担保責任の期間は、引渡しから5年間である
2. 住宅性能表示制度は、すべての新築住宅に義務付けられている
3. 瑕疵担保責任の対象は、構造耐力上主要な部分と雨水の浸入を防止する部分である
4. 「新築住宅」とは、建設工事完了から3年以内で未入居の住宅をいう


<QuizAnswer>

**答え: 3**

住宅品確法では、新築住宅の売主・請負人に対し、「構造耐力上主要な部分」と「雨水の浸入を防止する部分」について、引渡しから10年間の瑕疵担保責任を義務付けています（強行規定で短縮不可）。住宅性能表示制度は任意制度であり義務ではありません。「新築住宅」の定義は、建設工事完了から1年以内かつ未入居の住宅です。

</QuizAnswer>

## 問9

建築基準法における居室の採光と換気に関する記述として、正しいものはどれか。

1. 居室の有効採光面積は、床面積の1/10以上必要である
2. 居室の換気に有効な開口部面積は、床面積の1/20以上必要である
3. 居室の有効採光面積は、床面積の1/20以上必要である
4. 居室の換気に有効な開口部面積は、床面積の1/7以上必要である


<QuizAnswer>

**答え: 2**

建築基準法第28条では、居室の採光について有効採光面積は床面積の1/7以上、換気について換気に有効な開口部面積は床面積の1/20以上と定められています。「採光は1/7、換気は1/20」という組み合わせは試験頻出のポイントです。採光と換気の数値を取り違えるひっかけ問題が出やすいので、しっかり区別して覚えましょう。

</QuizAnswer>

## 問10

アイソメトリック図（等角投影図）に関する記述として、正しいものはどれか。

1. 3つの軸がそれぞれ90度の角度で交わる
2. 奥行き方向に遠近感があり、遠くのものほど小さく描かれる
3. 3つの軸がそれぞれ120度の角度で交わり、各軸方向の寸法が同じ縮尺で測れる
4. 消失点を1つ持ち、一点透視図法の簡略版である


<QuizAnswer>

**答え: 3**

アイソメトリック図（等角投影図）は、X・Y・Z の3軸がそれぞれ120度の角度で交わる平行投影法です。消失点を持たないため遠近感は表現されませんが、3軸すべて同じ縮尺で寸法が測れるという実務上の大きな利点があります。透視図法とは異なり、家具の寸法確認やインテリアの配置検討など、正確な寸法把握が必要な場面で活用されます。

</QuizAnswer>
