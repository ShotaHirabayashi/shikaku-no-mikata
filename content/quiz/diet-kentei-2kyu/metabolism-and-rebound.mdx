## 問1

基礎代謝とは、どのような状態で消費されるエネルギーのことか。

1. 軽い運動をしているときに消費されるエネルギー
2. 食事を消化・吸収するときに消費されるエネルギー
3. 生命維持のために安静時でも消費される最小限のエネルギー
4. 日常生活の活動全般で消費されるエネルギー


<QuizAnswer>

**答え: 3**

基礎代謝とは、呼吸・心拍・体温維持など、生命活動を維持するために安静にしていても無意識のうちに消費される最小限のエネルギーのことです。寝ているだけでも消費されるエネルギーであり、1日の総エネルギー消費量の約60％を占めています。

</QuizAnswer>

## 問2

1日の総エネルギー消費量に占める基礎代謝量の割合として、最も適切なものはどれか。

1. 約40％
2. 約50％
3. 約70％
4. 約60％


<QuizAnswer>

**答え: 4**

人の1日の総エネルギー消費量は、基礎代謝量（約60％）、身体活動量（約30％）、食事誘発性熱産生・DIT（約10％）の3つに大別されます。基礎代謝が最も大きな割合を占めており、ダイエットにおいて基礎代謝を高めることが重要とされる理由の一つです。

</QuizAnswer>

## 問3

基礎代謝が最も高くなる季節はいつか。

1. 春
2. 夏
3. 秋
4. 冬


<QuizAnswer>

**答え: 4**

基礎代謝は冬に最も高くなります。これは、外気温が低い冬に体温を一定に保つため、体内でより多くの熱を産生する必要があるからです。研究によると、基礎代謝が最も低い時期と比べて、冬場は1日あたり100kcal前後も基礎代謝量が増加するとされています。

</QuizAnswer>

## 問4

食事誘発性熱産生（DIT）において、摂取エネルギーに対する熱産生の割合が最も高い栄養素はどれか。

1. タンパク質
2. 脂質
3. 炭水化物（糖質）
4. 食物繊維


<QuizAnswer>

**答え: 1**

食事誘発性熱産生（DIT）は栄養素によって割合が異なり、タンパク質は摂取エネルギーの約30％、糖質は約6％、脂質は約4％が熱として消費されます。タンパク質が最も高いDITを示すため、タンパク質を多く含む食事は代謝を高める効果があるとされています。

</QuizAnswer>

## 問5

ダイエット中の停滞期（プラトー）が起こる主な原因として正しいものはどれか。

1. 筋肉量が急激に増加するため
2. 水分の摂りすぎにより体重が増えるため
3. ホメオスタシス（恒常性維持機能）が働くため
4. 消化酵素の分泌が過剰になるため


<QuizAnswer>

**答え: 3**

停滞期（プラトー）は、体重が減少することで身体のホメオスタシス（恒常性維持機能）が働き、少ないエネルギーでも生命を維持できるように基礎代謝を低下させることで起こります。一般的に、体重の約5％が減少したタイミングで発動しやすく、1ヶ月前後続くことが多いとされています。

</QuizAnswer>

## 問6

ホメオスタシス（恒常性維持機能）が最大限に働くとされるのは、1ヶ月間に体重の何％以上が減少した場合か。

1. 約1％以上
2. 約3％以上
3. 約5％以上
4. 約10％以上


<QuizAnswer>

**答え: 3**

ホメオスタシスの機能は、1ヶ月に体重の5％以上が減少すると最大限に発揮されるとされています。このため、急激な食事制限によるダイエットは停滞期やリバウンドを引き起こしやすくなります。月に体重の5％未満のペースで減量することが、リバウンドを防ぐポイントの一つです。

</QuizAnswer>

## 問7

リバウンドのメカニズムに関わる要因として、ホメオスタシスのほかに重要なものはどれか。

1. 成長ホルモンの過剰分泌
2. レプチンの分泌量の減少
3. アドレナリンの急激な増加
4. メラトニンの分泌リズムの乱れ


<QuizAnswer>

**答え: 2**

リバウンドには主に2つのメカニズムが関与しています。1つはホメオスタシスによる基礎代謝の低下、もう1つは満腹感を伝えるホルモンであるレプチンの分泌量の減少です。ダイエットで食事量が減るとレプチンの分泌も減少し、満腹感を感じにくくなるため、食べ過ぎてしまいやすくなります。

</QuizAnswer>

## 問8

基礎代謝量が加齢とともに低下する主な理由として、最も適切なものはどれか。

1. 骨密度が低下するため
2. 除脂肪量（筋肉量など）が減少するため
3. 体内の水分量が増加するため
4. 消化器官の活動が活発になるため


<QuizAnswer>

**答え: 2**

加齢に伴い基礎代謝量が低下する主な理由は、除脂肪量（筋肉量など）の減少です。筋肉は全身の基礎代謝の約25％を担う代謝活性の高い組織であるため、加齢による筋肉量の減少がそのまま基礎代謝の低下につながります。このため、年齢を重ねても筋肉量を維持することがダイエットにおいて重要です。

</QuizAnswer>

## 問9

基礎代謝量のピークを迎える年齢として正しい組み合わせはどれか。

1. 男性：18〜20歳、女性：15〜17歳
2. 男性：15〜17歳、女性：12〜14歳
3. 男性：20〜25歳、女性：18〜20歳
4. 男性：12〜14歳、女性：10〜12歳


<QuizAnswer>

**答え: 2**

基礎代謝量のピークは、男性では15〜17歳、女性では12〜14歳とされています。この年代は成長期にあたり、身体の発育に伴い筋肉量や臓器の活動が最も盛んなため、基礎代謝量も最大になります。それ以降は加齢とともに徐々に低下していきます。

</QuizAnswer>

## 問10

ダイエットの停滞期を乗り越えるための対処法として、最も適切なものはどれか。

1. 食事量をさらに大幅に減らして体重減少を促す
2. 焦らず現在の体重を維持し、停滞期が過ぎるのを待つ
3. ダイエットを完全に中止して元の食事量に戻す
4. 水分摂取を極力控えて体重を減らす


<QuizAnswer>

**答え: 2**

停滞期はホメオスタシスによる正常な身体反応であり、通常1ヶ月前後で終わります。焦って極端な食事制限をするとホメオスタシスがさらに強く働き、逆効果になる可能性があります。停滞期には現在の体重と食事・運動習慣を維持しながら、身体が新しい体重に慣れるのを待つことが、リバウンドを防ぎつつダイエットを成功させるための適切な対処法です。

</QuizAnswer>
