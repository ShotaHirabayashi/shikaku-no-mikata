## 問1

白色脂肪細胞の主な役割として、正しいものはどれか。

1. 脂肪を燃焼して熱を産生する
2. 余分なエネルギーを中性脂肪として蓄積する
3. 筋肉の収縮を助ける
4. 骨密度を維持する


<QuizAnswer>

**答え: 2**

白色脂肪細胞は、体内の余分なエネルギーを中性脂肪として蓄積する役割を持つ細胞です。皮下や内臓の周囲に分布しており、エネルギーの貯蔵庫として機能します。脂肪を燃焼して熱を産生するのは褐色脂肪細胞の役割です。

</QuizAnswer>

## 問2

褐色脂肪細胞に多く含まれ、熱産生に関与するタンパク質はどれか。

1. コラーゲン
2. アルブミン
3. UCP-1（脱共役タンパク質1）
4. ヘモグロビン


<QuizAnswer>

**答え: 3**

褐色脂肪細胞にはミトコンドリアが豊富に存在し、UCP-1（uncoupling protein-1：脱共役タンパク質1）というタンパク質の働きにより、ATPを合成する代わりにエネルギーを熱として放出します。これが褐色脂肪細胞の熱産生メカニズムの中心的な仕組みです。

</QuizAnswer>

## 問3

白色脂肪細胞と褐色脂肪細胞の形態的な違いとして、正しいものはどれか。

1. 白色脂肪細胞は多数の小さな脂肪滴を持ち、褐色脂肪細胞は一つの大きな脂肪滴を持つ
2. 白色脂肪細胞は一つの大きな脂肪滴を持ち、褐色脂肪細胞は多数の小さな脂肪滴を持つ
3. 両者ともに一つの大きな脂肪滴を持つが、大きさが異なる
4. 両者ともに多数の小さな脂肪滴を持つが、数が異なる


<QuizAnswer>

**答え: 2**

白色脂肪細胞は細胞内に一つの大きな脂肪滴（単房性脂肪滴）を含むのが特徴です。一方、褐色脂肪細胞は多数の小さな脂肪滴（多房性脂肪滴）を含み、ミトコンドリアも豊富に存在します。この構造の違いが、それぞれの機能の違いに直結しています。

</QuizAnswer>

## 問4

体脂肪率について、男性が「肥満」と判定される基準値として一般的なものはどれか。

1. 15%以上
2. 20%以上
3. 25%以上
4. 30%以上


<QuizAnswer>

**答え: 3**

厚生労働省の基準では、成人男性の体脂肪率が25%以上になると「体脂肪量過剰（肥満）」と判定されます。一方、成人女性の場合は30%以上が肥満の基準とされています。男女で基準値が異なるのは、女性の方がホルモンの影響などで体脂肪率が高い傾向にあるためです。

</QuizAnswer>

## 問5

内臓脂肪と皮下脂肪の違いについて、正しい記述はどれか。

1. 皮下脂肪は指でつまむことができないが、内臓脂肪はつまむことができる
2. 内臓脂肪は生活習慣病との関連が低く、皮下脂肪の方がリスクが高い
3. 内臓脂肪は腹部の臓器のまわりにつき、指でつまむことができない
4. 皮下脂肪は男性に多く、内臓脂肪は女性に多い


<QuizAnswer>

**答え: 3**

内臓脂肪は腹部の臓器（腸や肝臓など）のまわりにつく脂肪で、体の深部に存在するため外から指でつまむことができません。一方、皮下脂肪は皮膚のすぐ下にたまるため、指でつまむとやわらかく感じられます。一般的に内臓脂肪は男性に、皮下脂肪は女性につきやすい傾向があります。

</QuizAnswer>

## 問6

脂肪細胞の数が増加しやすい時期として、正しい組み合わせはどれか。

1. 胎児期・乳児期・思春期
2. 乳児期・学童期・成人期
3. 胎児期・学童期・中年期
4. 新生児期・思春期・更年期


<QuizAnswer>

**答え: 1**

脂肪細胞の数が増加しやすい時期は、胎児期（特に出生前3か月間）、乳児期（生後1年）、思春期の3つの時期とされています。これらの時期に過剰な栄養摂取があると脂肪細胞の数が増え、一度増えた脂肪細胞の数はダイエットで痩せても減らないため、将来的に肥満になりやすい体質となります。

</QuizAnswer>

## 問7

メタボリックシンドロームの診断において、日本の基準でのウエスト周囲径（腹囲）の基準値として正しいものはどれか。

1. 男性80cm以上、女性85cm以上
2. 男性85cm以上、女性90cm以上
3. 男性90cm以上、女性80cm以上
4. 男性90cm以上、女性85cm以上


<QuizAnswer>

**答え: 2**

日本のメタボリックシンドロームの診断基準では、ウエスト周囲径（おへその高さの腹囲）が男性85cm以上、女性90cm以上が必須条件とされています。これに加えて、血圧・血糖・脂質の3項目のうち2つ以上が基準値を超えた場合にメタボリックシンドロームと診断されます。女性の基準が男性より大きいのは、皮下脂肪の厚さが考慮されているためです。

</QuizAnswer>

## 問8

家庭用の体脂肪計で広く使われている測定方法はどれか。

1. 水中体重秤量法
2. 二重エネルギーX線吸収法（DEXA法）
3. 生体インピーダンス法（BI法）
4. X線CT法


<QuizAnswer>

**答え: 3**

生体インピーダンス法（BI法）は、体に微弱な電流を流し、その電気抵抗（インピーダンス）を計測して体脂肪率を推定する方法です。脂肪は電気を通しにくい性質を利用しており、家庭用体脂肪計として最も広く普及しています。ただし、体内の水分量の変化により測定値が変動しやすいため、飲食・運動・入浴後を避け、毎日同じ時間・同じ条件で測定することが推奨されます。

</QuizAnswer>

## 問9

褐色脂肪細胞が主に分布している部位として、正しいものはどれか。

1. 腹部や臀部
2. 鎖骨付近や肩甲骨周辺
3. 太ももやふくらはぎ
4. 手のひらや足の裏


<QuizAnswer>

**答え: 2**

褐色脂肪細胞は主に鎖骨付近、肩甲骨周辺、首の周りなどに分布しています。成人では新生児に比べてその量は大幅に減少しますが、寒冷刺激などによって活性化され、脂肪を燃焼して熱を産生する役割を担います。白色脂肪細胞が全身に広く分布するのに対し、褐色脂肪細胞は限られた部位にのみ存在します。

</QuizAnswer>

## 問10

肥満のメカニズムにおける脂肪細胞の変化について、正しいものはどれか。

1. 成人の肥満は主に脂肪細胞の数の増加によって起こる
2. 成人の肥満は主に脂肪細胞が中性脂肪を蓄えて肥大することで起こる
3. 脂肪細胞の数は成人期以降も際限なく増え続ける
4. 一度肥大した脂肪細胞はダイエットをしても小さくならない


<QuizAnswer>

**答え: 2**

成人の肥満は、主に既存の脂肪細胞が中性脂肪を蓄えて肥大化することによって起こります。脂肪細胞の数が増加しやすいのは胎児期・乳児期・思春期であり、成人期では主に細胞の肥大が肥満の原因となります。なお、ダイエットで体重が減少すると脂肪細胞は小さくなりますが、細胞の数自体は減少しないため、リバウンドしやすい要因の一つとなっています。

</QuizAnswer>
