## 問1

有酸素運動の主なエネルギー源として、最も適切なものはどれか。

1. クレアチンリン酸
2. グリコーゲン（糖質）
3. 脂肪酸
4. アミノ酸


<QuizAnswer>

**答え: 3**

有酸素運動では、酸素を使って体内の脂肪酸を分解しATP（アデノシン三リン酸）を産生します。低〜中強度の運動を長時間継続することで、体内に大量に蓄えられた体脂肪がエネルギー源として利用されます。一方、クレアチンリン酸やグリコーゲンは主に無酸素運動で優先的に使われるエネルギー源です。

</QuizAnswer>

## 問2

EPOC（Excess Post-Exercise Oxygen Consumption）の日本語訳として正しいものはどれか。

1. 運動中最大酸素摂取量
2. 運動後過剰酸素消費量
3. 安静時酸素消費基準量
4. 運動前酸素蓄積量


<QuizAnswer>

**答え: 2**

EPOCは「Excess Post-Exercise Oxygen Consumption」の略で、日本語では「運動後過剰酸素消費量」と訳されます。これは「アフターバーン効果」とも呼ばれ、激しい運動の後に身体が通常以上に酸素を消費し、エネルギーを使い続ける現象のことです。筋肉の修復・回復、体温の調整、老廃物の処理などのために運動後も数時間〜最大48時間程度カロリー消費が続くとされています。

</QuizAnswer>

## 問3

脂肪燃焼に最も効率的とされる運動強度は、最大心拍数のおよそ何％の範囲か。

1. 40〜50％
2. 60〜70％
3. 80〜90％
4. 90〜100％


<QuizAnswer>

**答え: 2**

脂肪燃焼に最も効率的な運動強度は、最大心拍数の約60〜70％とされています。この強度では、エネルギー消費に占める脂肪の燃焼割合が約50％と最も高くなります。運動強度が高すぎると糖質がエネルギー源の中心となり、脂肪の燃焼割合はかえって低下します。この適度な強度帯は「ファットバーンゾーン」とも呼ばれています。

</QuizAnswer>

## 問4

ダイエット目的で有酸素運動と無酸素運動を組み合わせる場合、一般的に推奨される順番はどれか。

1. 有酸素運動 → 無酸素運動
2. 無酸素運動 → 有酸素運動
3. 有酸素運動のみで十分である
4. どちらを先にしても効果に違いはない


<QuizAnswer>

**答え: 2**

ダイエット目的では「無酸素運動（筋トレ）→ 有酸素運動」の順番が推奨されます。先に筋トレを行うことで成長ホルモンやアドレナリンが分泌され、脂肪の分解が促進されます。その後に有酸素運動を行うことで、分解された脂肪酸を効率的にエネルギーとして燃焼させることができます。

</QuizAnswer>

## 問5

NEAT（Non-Exercise Activity Thermogenesis）の説明として正しいものはどれか。

1. 運動によって消費されるエネルギーのこと
2. 食事を摂取した際に発生する熱エネルギーのこと
3. 運動以外の日常生活活動で消費されるエネルギーのこと
4. 睡眠中に消費される基礎的なエネルギーのこと


<QuizAnswer>

**答え: 3**

NEATは「Non-Exercise Activity Thermogenesis（非運動性活動熱産生）」の略で、意図的な運動以外の日常生活活動で消費されるエネルギーを指します。家事、通勤時の歩行、立ち仕事、姿勢の維持などが含まれます。座っている時間を減らして立つ・歩く時間を増やすことで、1日あたり数百kcalの消費量の差が生じることもあり、ダイエットにおいて重要な要素とされています。

</QuizAnswer>

## 問6

アフターバーン効果（EPOC）を最も効果的に引き出すトレーニング方法はどれか。

1. ゆっくりとしたウォーキング
2. 長時間の静的ストレッチ
3. HIIT（高強度インターバルトレーニング）
4. 低負荷・高回数のリズム体操


<QuizAnswer>

**答え: 3**

HIIT（High-Intensity Interval Training）は短時間で全力に近い高強度の運動と休息を交互に繰り返すトレーニング法で、アフターバーン効果を最も効果的に引き出すとされています。高強度の運動により体内で一時的に酸素が不足する状態が生じ、運動後に身体がその状態を回復させるために長時間にわたってエネルギーを消費し続けます。低強度の運動ではこの効果はほとんど得られません。

</QuizAnswer>

## 問7

運動前のウォーミングアップとして最も適切なストレッチの種類はどれか。

1. 静的ストレッチ（スタティックストレッチ）
2. 動的ストレッチ（ダイナミックストレッチ）
3. PNFストレッチ
4. バリスティックストレッチ


<QuizAnswer>

**答え: 2**

運動前のウォーミングアップには、動的ストレッチ（ダイナミックストレッチ）が最も適切です。動的ストレッチは筋肉を動かしながら伸ばす方法で、心拍数を上げ血流を促進し、筋肉や関節を運動に適した状態に準備します。一方、静的ストレッチは運動後のクールダウンに適しており、運動前に行うと筋力やパフォーマンスが一時的に低下する可能性があるとされています。

</QuizAnswer>

## 問8

筋肉1kgあたりの基礎代謝量（安静時のエネルギー消費量）として、一般的に示されている値に最も近いものはどれか。

1. 約5kcal/日
2. 約13kcal/日
3. 約50kcal/日
4. 約100kcal/日


<QuizAnswer>

**答え: 2**

筋肉（骨格筋）1kgあたりの基礎代謝量は約13kcal/日とされています。この数値だけを見ると小さく感じますが、筋肉量が増えると骨などの除脂肪組織も増加するため、除脂肪量1kgの増加で基礎代謝が約50kcal上がるという研究結果もあります。さらに、筋肉量が多い人はNEAT（非運動性活動熱産生）も自然と高まるため、総合的なエネルギー消費量の増加につながります。

</QuizAnswer>

## 問9

無酸素運動の特徴として正しいものはどれか。

1. 長時間継続して行う低強度の運動である
2. 主に脂肪酸をエネルギー源として使う
3. 短時間で強い負荷をかける高強度の運動である
4. 酸素をまったく使わずに行う運動である


<QuizAnswer>

**答え: 3**

無酸素運動とは、短距離走やウェイトトレーニングのように短時間で強い負荷をかける高強度の運動です。主にグルコース（糖質）をエネルギー源として利用します。「無酸素」という名前ですが、まったく酸素を使わないわけではなく、酸素を使わないエネルギー産生系（解糖系やクレアチンリン酸系）が主に働くことからこの名称がつけられています。

</QuizAnswer>

## 問10

筋力トレーニングによる基礎代謝向上の効果を最大化するために、特に鍛えることが推奨される部位はどれか。

1. 前腕の筋群
2. 下半身の筋群（大腿四頭筋・大臀筋など）
3. 手指の筋群
4. 首まわりの筋群


<QuizAnswer>

**答え: 2**

基礎代謝を効率よく向上させるには、大きな筋肉が集中する下半身のトレーニングが有効とされています。大腿四頭筋、ハムストリングス、大臀筋などは人体の中でも特に大きな筋群であり、これらを鍛えることで筋肉量の増加効率が高く、基礎代謝の向上に大きく寄与します。スクワットやランジなどの下半身トレーニングがダイエットにおいて重視される理由もここにあります。

</QuizAnswer>
