## 問1

人体の神経系は大きく中枢神経系と末梢神経系に分けられる。中枢神経系を構成する2つの器官の正しい組み合わせはどれか。

1. 脳と脊髄
2. 脳と自律神経
3. 脊髄と末梢神経
4. 脳と感覚神経


<QuizAnswer>

**答え: 1**

中枢神経系は脳と脊髄から構成されます。脳は思考・感覚・運動の統合的な制御を行い、脊髄は脳と末梢神経をつなぐ情報伝達の経路であると同時に、反射などの機能も担っています。自律神経や感覚神経は末梢神経系に分類されます。

</QuizAnswer>

## 問2

成人の人体を構成する骨の数として最も適切なものはどれか。

1. 約106個
2. 約206個
3. 約305個
4. 約350個


<QuizAnswer>

**答え: 2**

成人の人体には約206個の骨があります。赤ちゃんの骨は約305個ありますが、成長とともに複数の骨が癒合して1つになるため、大人では約206個になります。なお、両手だけで54個の骨があり、全身の骨の約4分の1を占めています。

</QuizAnswer>

## 問3

自律神経系のうち、交感神経が優位になったときの身体の反応として正しいものはどれか。

1. 心拍数が低下する
2. 消化器官の活動が活発になる
3. 心拍数と血圧が上昇する
4. 瞳孔が縮小する


<QuizAnswer>

**答え: 3**

交感神経は身体を活発化させる「闘争か逃走か」の反応を担っており、優位になると心拍数・血圧の上昇、気管支の拡張、瞳孔の散大などが起こります。一方、心拍数の低下や消化活動の促進、瞳孔の縮小は副交感神経が優位なときの反応です。

</QuizAnswer>

## 問4

人体の筋肉は3種類に分類される。心臓の壁を構成する筋肉の種類はどれか。

1. 骨格筋
2. 平滑筋
3. 横紋筋
4. 心筋


<QuizAnswer>

**答え: 4**

人体の筋肉は骨格筋・平滑筋・心筋の3種類に分類されます。心筋は心臓の壁を構成する特殊な筋肉で、自分の意志では動かせない不随意筋です。骨格筋は随意筋として関節運動に関わり、平滑筋は消化管や血管の壁に存在する不随意筋です。

</QuizAnswer>

## 問5

五感（視覚・聴覚・嗅覚・味覚・触覚）のうち、「特殊感覚」に分類されないものはどれか。

1. 視覚
2. 聴覚
3. 触覚
4. 嗅覚


<QuizAnswer>

**答え: 3**

視覚・聴覚・嗅覚・味覚は、それぞれ眼・耳・鼻・舌という特定の器官に感覚受容器が局在しているため「特殊感覚」に分類されます。一方、触覚は皮膚全体に受容器が分布しており「体性感覚（一般感覚）」に分類されます。

</QuizAnswer>

## 問6

消化器官において、栄養素の吸収が最も盛んに行われる臓器はどれか。

1. 胃
2. 小腸
3. 大腸
4. 食道


<QuizAnswer>

**答え: 2**

小腸は長さ約6メートル以上の管状の臓器で、消化管全体の約80%を占めます。内壁には絨毛という細かい突起が無数にあり、表面積を大きくすることで効率よく栄養素を吸収しています。大腸は主に水分やミネラルの吸収を行い、胃や食道は栄養吸収の主要な場ではありません。

</QuizAnswer>

## 問7

副交感神経の神経伝達物質として正しいものはどれか。

1. アドレナリン
2. ノルアドレナリン
3. アセチルコリン
4. ドーパミン


<QuizAnswer>

**答え: 3**

副交感神経の末端ではアセチルコリンが神経伝達物質として放出されます。一方、交感神経の末端ではノルアドレナリンが主な神経伝達物質です。アセチルコリンは副交感神経だけでなく、自律神経の節前線維や運動神経でも使われる重要な神経伝達物質です。

</QuizAnswer>

## 問8

膵臓の役割として正しいものはどれか。

1. 胆汁を生成して脂肪の消化を助ける
2. 消化酵素を含む膵液の分泌とホルモンの分泌を行う
3. 栄養素の貯蔵と解毒作用を行う
4. 水分の再吸収を行い便を固形にする


<QuizAnswer>

**答え: 2**

膵臓は胃の裏側にある長さ15〜20cmの臓器で、2つの重要な役割があります。1つは消化酵素（トリプシン、膵アミラーゼ、膵リパーゼなど）を含む膵液を分泌して食物の消化を助けること、もう1つはインスリンなどのホルモンを分泌して血糖値を調節することです。胆汁の生成は肝臓、解毒作用も肝臓の役割です。

</QuizAnswer>

## 問9

人体を構成する最小の単位として正しいものはどれか。

1. 器官
2. 組織
3. 細胞
4. 器官系


<QuizAnswer>

**答え: 3**

人体の構造は、細胞→組織→器官→器官系→個体という階層で成り立っています。細胞は生命活動を営む最小の単位であり、同じ種類の細胞が集まって組織（上皮組織・結合組織・筋組織・神経組織）を形成し、さらに複数の組織が集まって器官を構成しています。

</QuizAnswer>

## 問10

自律神経のうち、副交感神経が優位になったときの身体の状態として正しいものはどれか。

1. 血圧が上昇し、筋肉が緊張する
2. 気管支が拡張し、呼吸が速くなる
3. 消化器官の活動が促進され、心拍数が低下する
4. 瞳孔が散大し、発汗が増加する


<QuizAnswer>

**答え: 3**

副交感神経が優位になると身体はリラックス状態となり、消化器官の活動が促進され、心拍数や血圧が低下します。唾液の分泌が増え、空腹感を感じやすくなるのも副交感神経の作用です。血圧上昇・気管支拡張・瞳孔散大・発汗増加はいずれも交感神経が優位なときの反応です。

</QuizAnswer>
