## 問1

長時間のデスクワーク後に足のむくみが生じる主な原因として、最も適切なものはどれか。

1. 水分の摂りすぎによりリンパ液が増加するため
2. 筋肉の動きが減少しリンパ液の流れが滞るため
3. 足の皮膚温度が上昇しリンパ管が拡張するため
4. 重力により足のリンパ節が閉塞するため


<QuizAnswer>

**答え: 2**

リンパ液は血液と異なり心臓のようなポンプを持たず、周囲の筋肉の収縮によって流れています。長時間同じ姿勢で座り続けると筋肉の動きが減少し、リンパ液の循環が滞ってむくみが生じます。適度に立ち上がったりストレッチをすることが予防に効果的です。

</QuizAnswer>

## 問2

冷え性の改善にリンパケアが有効とされる理由として、最も適切なものはどれか。

1. リンパケアにより体内の水分量が減少し体温が上がるため
2. リンパケアにより筋肉量が直接増加するため
3. リンパの流れが促進されると同時に血流も改善されるため
4. リンパケアにより自律神経が完全に遮断されるため


<QuizAnswer>

**答え: 3**

リンパケアを行うとリンパの流れが促進されるとともに、血液循環も改善されます。血流が良くなることで老廃物の排出が促され、全身のめぐりが良くなり、末端の冷えの緩和が期待できます。冷え性はリンパと血液の両方の循環が関わっている症状です。

</QuizAnswer>

## 問3

肩こり・首こりが慢性化している人に見られるリンパの滞りのサインとして、最も適切なものはどれか。

1. 肩周辺の皮膚が青白く変色する
2. 肩や首周辺を触るとゴリゴリとした硬い感触がある
3. 肩関節の可動域が急激に広がる
4. 首のリンパ節が完全に消失する


<QuizAnswer>

**答え: 2**

リンパの流れが滞っている部位を触ると、ゴリゴリとした硬い感触が見られることがあります。これは筋繊維が凝り固まり、周囲の組織と癒着している状態です。肩こりや首こりが長引く背景にはリンパの滞りが関係していることが多く、セルフチェックの手がかりになります。

</QuizAnswer>

## 問4

リンパの流れを良くするために積極的に摂取すべき食材の組み合わせとして、最も適切なものはどれか。

1. 加工食品・精製糖・乳製品
2. 緑色の葉物野菜・海藻類・柑橘類
3. インスタント食品・高塩分食品・清涼飲料水
4. アルコール飲料・カフェイン飲料・揚げ物


<QuizAnswer>

**答え: 2**

リンパの流れを改善するには、カリウムや食物繊維を豊富に含む食材が効果的です。緑色の葉物野菜、海藻類、柑橘類のほか、アボカドやナッツ類なども推奨されています。一方、加工食品・過剰な塩分・精製糖はリンパ系の機能を低下させるため避けるべきとされています。

</QuizAnswer>

## 問5

リンパの流れを改善するための生活習慣として、適切でないものはどれか。

1. 1日2リットル程度の水分をこまめに摂取する
2. ぬるめのお湯にゆっくり浸かる入浴を習慣にする
3. 睡眠時間を削って深夜まで活動する
4. ウォーキングなどの適度な運動を日常に取り入れる


<QuizAnswer>

**答え: 3**

睡眠不足はリンパ液の流れを滞らせる大きな原因の一つです。毎日7〜8時間程度の十分な睡眠を確保することが推奨されています。水分補給はリンパ液の粘度を下げて流れをスムーズにし、入浴は血行促進を通じてリンパの流れを改善し、適度な運動は筋肉を動かしてリンパ液の循環を促します。

</QuizAnswer>

## 問6

リンパマッサージが禁忌（行ってはいけない）とされる状態として、最も適切なものはどれか。

1. 軽い肩こりを感じている状態
2. 体内に血栓がある状態
3. 軽い運動をした直後の状態
4. 十分な睡眠をとった翌朝の状態


<QuizAnswer>

**答え: 2**

体内の静脈やリンパ管に血栓がある場合、リンパマッサージは禁忌とされています。マッサージの刺激によって血栓が移動し、肺塞栓症など重篤な合併症を引き起こす危険性があるためです。そのほか、悪性腫瘍がある場合、発熱時、皮膚に炎症や感染症がある場合なども禁忌とされています。

</QuizAnswer>

## 問7

リンパの流れが滞っているときに現れやすい症状の組み合わせとして、最も適切なものはどれか。

1. むくみ・冷え・肌荒れ・全身の倦怠感
2. 視力低下・聴力低下・味覚障害・嗅覚異常
3. 骨密度の低下・歯の変色・爪の変形・脱毛
4. 食欲亢進・体重急増・多汗・動悸


<QuizAnswer>

**答え: 1**

リンパの流れが滞ると、むくみ、冷え、肌荒れ・くすみ、全身の倦怠感や疲労感が現れやすくなります。これはリンパ液の循環不良により老廃物が体内に残留し、組織に余分な水分が溜まることが原因です。一見リンパと無関係に見える症状もリンパの滞りが関係していることがあります。

</QuizAnswer>

## 問8

リンパマッサージを避けるべきタイミングとして、適切でないものはどれか。

1. 飲酒後
2. 食後2時間以内
3. 発熱時
4. 入浴後の体が温まった状態


<QuizAnswer>

**答え: 4**

入浴後の体が温まった状態は、血行が良くなっているためリンパマッサージに適したタイミングです。一方、飲酒後はアルコールの回りが早くなり危険であり、食後すぐは消化に影響を与える可能性があり、発熱時は体に余計な負担をかけるため、いずれもリンパマッサージは避けるべきとされています。

</QuizAnswer>

## 問9

ストレスとリンパの流れの関係について、最も適切な説明はどれか。

1. ストレスを感じるとリンパ液の生成量が10倍になる
2. ストレスはリンパの流れに一切影響を与えない
3. 過度なストレスはリンパ液の流れを滞らせる原因となる
4. ストレスが多いほどリンパの流れは速くなる


<QuizAnswer>

**答え: 3**

過度なストレスは自律神経のバランスを乱し、リンパ液の流れを滞らせる原因となります。ストレスにより交感神経が優位になると筋肉が緊張し、リンパ管の周囲の筋肉の動きが制限されます。適度な運動や趣味などでストレスを解消する方法を持つことが、リンパの流れを良好に保つうえで重要です。

</QuizAnswer>

## 問10

むくみ対策としてカリウムを多く含む食品の摂取が推奨される理由として、最も適切なものはどれか。

1. カリウムがリンパ管を物理的に広げるため
2. カリウムが体内の余分なナトリウムの排出を促し水分バランスを整えるため
3. カリウムがリンパ液の色を変化させて流れやすくするため
4. カリウムが筋肉を直接刺激してリンパ管を収縮させるため


<QuizAnswer>

**答え: 2**

カリウムには体内の余分なナトリウム（塩分）を尿とともに排出する作用があり、体内の水分バランスを整える働きがあります。塩分の過剰摂取は体に水分を溜め込みむくみの原因となるため、カリウムを豊富に含むバナナ、ほうれん草、海藻類などを積極的に摂取することがむくみ改善に効果的です。

</QuizAnswer>
