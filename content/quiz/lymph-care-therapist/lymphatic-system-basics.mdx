## 問1

リンパ液の主成分は、血液中のどの成分に由来するか。

1. 赤血球
2. 血小板
3. 血漿（けっしょう）
4. ヘモグロビン


<QuizAnswer>

**答え: 3**

リンパ液は、毛細血管から滲み出した血漿成分が組織間隙（組織液）となり、それが毛細リンパ管に取り込まれたものです。そのため、リンパ液の組成は血漿に近く、水分を主体に電解質やタンパク質（アルブミン、グロブリンなど）を含んでいます。赤血球や血小板は通常リンパ液には含まれません。

</QuizAnswer>

## 問2

人体に存在するリンパ節の数として、最も適切なものはどれか。

1. 約50個
2. 約100個
3. 約600個
4. 約3,000個


<QuizAnswer>

**答え: 3**

リンパ節は全身に約300〜600個（文献によっては400〜700個）存在するとされています。特に頸部・腋窩部・腹部・鼠径部に多く集中しており、その半数以上が腹部に位置しています。リンパ液中の異物や病原体をろ過する「関所」としての役割を担っています。

</QuizAnswer>

## 問3

全身のリンパ管が最終的に合流して静脈に注ぐ際、最も大きなリンパ管の名称はどれか。

1. 毛細リンパ管
2. 集合リンパ管
3. 胸管（きょうかん）
4. 門脈


<QuizAnswer>

**答え: 3**

全身のリンパ管は合流しながら太くなり、最終的に胸管と右リンパ本幹の2本にまとまります。胸管は体内最大のリンパ管で、下半身と左上半身のリンパ液を集めて左静脈角（左鎖骨下静脈と左内頸静脈の合流部）に注ぎます。門脈は肝臓に関わる血管であり、リンパ管ではありません。

</QuizAnswer>

## 問4

リンパ管の中を流れるリンパ液の方向について、正しいものはどれか。

1. 心臓から末梢に向かって流れる
2. 末梢から心臓に向かって一方向に流れる
3. 動脈と同様に拍動によって双方向に流れる
4. 方向性はなく自由に行き来する


<QuizAnswer>

**答え: 2**

リンパ系は一方向性の経路であり、リンパ液は末梢（組織）から心臓に向かって流れます。リンパ管の内部には静脈と同様の弁（逆流防止弁）が備わっており、リンパ液が逆流しないようになっています。血液循環のように心臓のポンプ作用で流れるのではなく、主に筋肉の収縮や呼吸運動によって流れが促されます。

</QuizAnswer>

## 問5

リンパ節の免疫における主な役割として、最も適切なものはどれか。

1. 赤血球を生成する
2. リンパ液中の異物や病原体をろ過・排除する
3. ホルモンを分泌して体温を調節する
4. 酸素と二酸化炭素のガス交換を行う


<QuizAnswer>

**答え: 2**

リンパ節は免疫システムにおける「関所」として機能し、リンパ液中に含まれるウイルス・細菌・がん細胞などの異物をろ過して排除する役割を持っています。リンパ節内にはリンパ球やマクロファージなどの免疫細胞が集まっており、異物を検知すると免疫応答を発動して体を守ります。

</QuizAnswer>

## 問6

リンパ液に多く含まれる免疫細胞はどれか。

1. 赤血球
2. 血小板
3. リンパ球
4. 骨芽細胞


<QuizAnswer>

**答え: 3**

リンパ液は細胞外液の組成とほぼ等しいですが、多数のリンパ球を含んでいるのが大きな特徴です。リンパ球はT細胞やB細胞などに分類される免疫細胞であり、リンパ節を中心に異物の認識や排除を担っています。赤血球や血小板は通常リンパ液中には存在しません。

</QuizAnswer>

## 問7

リンパ系と血液循環系の違いとして、正しいものはどれか。

1. リンパ系には心臓のようなポンプ器官がある
2. リンパ系は閉鎖循環系であり、血管と直接つながらない
3. リンパ系は開放系であり、心臓のようなポンプ器官を持たない
4. リンパ管には弁がなく、血管には弁がある


<QuizAnswer>

**答え: 3**

血液循環系は心臓をポンプとした閉鎖循環系ですが、リンパ系は毛細リンパ管から始まり最終的に静脈に合流する開放系（一方向性）であり、専用のポンプ器官を持ちません。リンパ液の流れは骨格筋の収縮や呼吸運動、リンパ管自体の収縮などによって促されます。また、リンパ管にも静脈と同様に逆流防止弁があります。

</QuizAnswer>

## 問8

右リンパ本幹が集めるリンパ液の範囲として、正しいものはどれか。

1. 全身のリンパ液
2. 下半身のリンパ液
3. 右上半身のリンパ液
4. 左上半身のリンパ液


<QuizAnswer>

**答え: 3**

右リンパ本幹は右上半身（右頭頸部・右上肢・右胸部）のリンパ液を集めて右静脈角に注ぎます。一方、下半身と左上半身のリンパ液は胸管が集めて左静脈角に注ぎます。つまり、体全体のリンパ液の大部分は胸管を経由しており、右リンパ本幹が担当するのは右上半身のみです。

</QuizAnswer>

## 問9

リンパ浮腫の分類において、がんの手術や放射線治療の後に発症するものは何と呼ばれるか。

1. 原発性（一次性）リンパ浮腫
2. 続発性（二次性）リンパ浮腫
3. 急性リンパ浮腫
4. 特発性リンパ浮腫


<QuizAnswer>

**答え: 2**

がんの手術でリンパ節を郭清（切除）したり、放射線治療を受けたりすることでリンパの流れが障害され発症するリンパ浮腫は「続発性（二次性）リンパ浮腫」と呼ばれます。日本をはじめとする先進国では、がん治療に伴う二次性リンパ浮腫が最も多くみられます。一方、原発性（一次性）は原因不明の先天的なリンパ管の異常などによるものです。

</QuizAnswer>

## 問10

リンパ系の役割として、誤っているものはどれか。

1. 組織液を回収して静脈に戻す
2. 小腸で吸収された脂肪分を運搬する
3. 酸素を全身の細胞に届ける
4. 異物や病原体のろ過・免疫応答を行う


<QuizAnswer>

**答え: 3**

酸素を全身の細胞に届けるのは血液循環系（赤血球のヘモグロビン）の役割であり、リンパ系の機能ではありません。リンパ系の主な役割は、①組織液（余分な水分）を回収して静脈に戻すこと、②小腸で吸収された脂肪分を運搬すること（乳び管）、③リンパ節で異物や病原体をろ過し免疫応答を行うこと、の3つです。

</QuizAnswer>
