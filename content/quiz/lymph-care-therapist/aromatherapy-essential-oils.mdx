## 問1

精油（エッセンシャルオイル）の抽出方法のうち、最も多くの植物に用いられている代表的な方法はどれか。

1. 圧搾法
2. 水蒸気蒸留法
3. 溶剤抽出法
4. 超臨界流体抽出法


<QuizAnswer>

**答え: 2**

水蒸気蒸留法は、原料植物に水蒸気を通して精油成分を気化させ、冷却して水と精油に分離する方法です。最も歴史が古く、多くの精油がこの方法で抽出されています。比較的安価な装置で行える一方、熱に弱いデリケートな成分は壊れてしまうことがあります。

</QuizAnswer>

## 問2

柑橘系の精油（オレンジ、レモン、グレープフルーツなど）の果皮から精油を抽出する際に主に用いられる方法はどれか。

1. 水蒸気蒸留法
2. 溶剤抽出法
3. 超臨界流体抽出法
4. 圧搾法


<QuizAnswer>

**答え: 4**

圧搾法は、柑橘系果実の果皮を物理的に圧搾して精油を取り出す方法です。柑橘系の油胞は他の植物に比べて大きいため、圧搾するだけで簡単に精油を抽出できます。熱を加えないため、フレッシュな香り成分をそのまま保つことができるのが利点です。

</QuizAnswer>

## 問3

キャリアオイルの役割として最も適切なものはどれか。

1. 精油の香りを強めるために使用する
2. 精油を希釈して肌に安全に使用できるようにする
3. 精油の保存期間を延ばすために使用する
4. 精油の色を変化させるために使用する


<QuizAnswer>

**答え: 2**

キャリアオイルは植物の種子や果実から抽出された植物油で、精油を希釈して肌に安全に塗布するために使用します。「キャリア（carrier＝運ぶもの）」の名の通り、精油の有効成分を肌に届ける役割を担っています。精油は原液のまま肌に塗布すると刺激が強すぎるため、キャリアオイルで適切な濃度に希釈することが重要です。

</QuizAnswer>

## 問4

ホホバオイルの特徴として正しいものはどれか。

1. 酸化しやすく保存期間が短い
2. 独特の強い香りがある
3. 肌質を問わず使用でき、安定性が高い
4. 粘度が高く重い使用感がある


<QuizAnswer>

**答え: 3**

ホホバオイルは肌質を問わず幅広く使用でき、酸化しにくく安定性が高いことが大きな特徴です。実際にはオイルではなく液体ワックスに分類され、ほぼ無臭で軽い質感のため、アロマトリートメントのベースとして非常に人気があります。初心者にも扱いやすいキャリアオイルの一つです。

</QuizAnswer>

## 問5

精油の「光毒性」について正しい説明はどれか。

1. 精油が光を浴びると成分が分解されて効果がなくなること
2. 精油を塗布した肌が紫外線に当たると炎症やシミを起こすこと
3. 精油を暗所に保管すると毒性が増すこと
4. 精油の色が光によって変色すること


<QuizAnswer>

**答え: 2**

光毒性とは、精油に含まれる一部の成分（フロクマリン類など）が紫外線と反応し、肌に炎症・色素沈着・火ぶくれなどを引き起こす性質のことです。ベルガモット、レモン、グレープフルーツなどの柑橘系精油に光毒性を持つものが多く、塗布後は直射日光を避ける必要があります。

</QuizAnswer>

## 問6

アロマトリートメント用のブレンドオイルを作る際、成人の全身トリートメントに推奨される精油の希釈濃度はどれか。

1. 5%〜10%
2. 3%〜5%
3. 1%以下
4. 0.01%以下


<QuizAnswer>

**答え: 3**

成人の全身トリートメントでは、精油の希釈濃度は1%以下が推奨されています。例えばキャリアオイル50mlに対して精油を10滴（約0.5ml）加えると約1%の濃度になります。精油1滴は約0.05mlとして計算します。濃度が高すぎると肌トラブルの原因となるため、適切な濃度を守ることが大切です。

</QuizAnswer>

## 問7

ラベンダー精油の特徴として誤っているものはどれか。

1. リラックス効果があり、安眠を促す作用がある
2. 主な芳香成分に酢酸リナリルやリナロールが含まれる
3. 覚醒作用が強く、集中力を高める効果が主である
4. 抗菌作用があり、切り傷や虫刺されにも活用される


<QuizAnswer>

**答え: 3**

ラベンダー精油は鎮静・リラックス効果が最大の特徴であり、覚醒作用が強いという説明は誤りです。主な芳香成分である酢酸リナリルやリナロールには鎮静作用があり、不眠や頭痛の緩和に役立ちます。また、抗菌作用も持ち合わせており、アロマテラピーで最も汎用性の高い精油の一つとされています。

</QuizAnswer>

## 問8

敏感肌の方のアロマトリートメントに最も適したキャリアオイルはどれか。

1. スイートアーモンドオイル
2. ココナッツオイル
3. オリーブオイル
4. ひまし油


<QuizAnswer>

**答え: 1**

スイートアーモンドオイルはマイルドな使い心地で、ビタミンEや脂肪酸が豊富に含まれています。アレルギー反応が少なく、保湿効果が高いため、敏感肌の方にも適したキャリアオイルとして広く推奨されています。初心者にも扱いやすく、アロマトリートメントの定番オイルの一つです。

</QuizAnswer>

## 問9

ペパーミント精油に最も多く含まれ、冷却作用をもたらす主要成分はどれか。

1. リモネン
2. リナロール
3. メントール
4. オイゲノール


<QuizAnswer>

**答え: 3**

ペパーミント精油の主要成分はℓ-メントール（メントール）で、スーッとした清涼感・冷却作用をもたらします。メントールには緊張型頭痛や筋肉の痛みを和らげる鎮痛効果もあります。また、抗菌・抗ウイルス効果も期待でき、リフレッシュしたいときに適した精油です。

</QuizAnswer>

## 問10

溶剤抽出法で得られる、精油成分とワックス成分が混ざった半固体状の中間生成物を何と呼ぶか。

1. ハイドロゾル
2. コンクリート
3. アブソリュート
4. チンキ


<QuizAnswer>

**答え: 2**

溶剤抽出法では、揮発性の有機溶剤に原料植物を浸し、精油成分を抽出します。この過程で最初に得られる半固体状の物質を「コンクリート」と呼びます。コンクリートからさらにエタノールで精油成分だけを抽出した最終的な精油は「アブソリュート」と呼ばれます。ジャスミンやローズなど、熱に弱いデリケートな花から精油を得る際にこの方法が用いられます。

</QuizAnswer>
