## 問1

リンパケアセラピストが施術メニューの名称を決める際、国家資格を持たない場合に使用を避けるべき表現はどれか。

1. リンパトリートメント
2. リンパケア
3. リンパマッサージ
4. リンパボディケア


<QuizAnswer>

**答え: 3**

「マッサージ」という名称は、あん摩マッサージ指圧師の国家資格を持つ者のみが業として使用できます。あはき法（あん摩マッサージ指圧師、はり師、きゅう師等に関する法律）により、無資格者がマッサージを名乗ることは法律に抵触する可能性があります。そのため、国家資格を持たないセラピストは「トリートメント」「ケア」「ボディケア」などの表現を使用します。

</QuizAnswer>

## 問2

セラピストがクライアントの個人情報や施術内容を第三者に漏らしてはならないという原則を何というか。

1. インフォームド・コンセント
2. 守秘義務
3. 説明責任
4. 善管注意義務


<QuizAnswer>

**答え: 2**

守秘義務とは、クライアントから知り得た個人情報や相談内容をみだりに第三者に漏らしてはならないという義務です。クライアントはプライバシーが守られるからこそ安心して身体の悩みを打ち明けることができます。セラピストにとって信頼関係の基盤となる非常に重要な倫理原則です。

</QuizAnswer>

## 問3

個人事業主としてリラクゼーションサロンを開業する際、事業の形態や規模に関わらず必ず提出が必要な届出はどれか。

1. 保健所への営業許可申請
2. 都道府県知事への届出
3. 税務署への開業届
4. 消防署への防火対象物使用開始届


<QuizAnswer>

**答え: 3**

個人事業主としてサロンを開業する場合、貸店舗・自宅サロン・出張サロンなど形態を問わず、税務署への開業届の提出が必要です。一般的なリラクゼーションサロンの場合、保健所への届出は原則不要ですが、提供するサービス内容（まつ毛エクステなど）によっては別途届出が必要になる場合があります。

</QuizAnswer>

## 問4

セラピストが施術前にクライアントに対して施術内容やリスクを説明し、同意を得ることを何というか。

1. カウンセリングシート
2. クーリングオフ
3. インフォームド・コンセント
4. コンプライアンス


<QuizAnswer>

**答え: 3**

インフォームド・コンセントとは「十分な説明に基づく同意」を意味し、施術者がクライアントに対して施術内容・目的・リスク・代替手段などを事前に説明し、理解と同意を得ることです。もともと医療の分野で確立された概念ですが、セラピストにおいてもクライアントの自己決定権を尊重するために非常に重要な手続きです。

</QuizAnswer>

## 問5

セラピストの守秘義務に関する記述として、最も適切なものはどれか。

1. いかなる場合でも例外なくクライアントの情報を第三者に伝えてはならない
2. クライアントの同意があれば、SNSに施術のビフォーアフター写真を掲載してよい
3. クライアントが自傷や他害の恐れがある場合は、関係機関に情報提供できる
4. 施術に関する情報は守るが、雑談で聞いた私的な情報は守秘の対象外である


<QuizAnswer>

**答え: 3**

守秘義務には例外があり、クライアント本人や第三者の生命・安全に危険が及ぶ恐れがある場合には、必要最低限の情報を関係機関に伝えることが認められています。選択肢1のように「例外なし」ではなく、選択肢4のように雑談で得た情報も守秘の対象です。また、SNS掲載は書面での明確な同意が必要であり、口頭だけの同意では不十分です。

</QuizAnswer>

## 問6

「あはき法」の正式名称として正しいものはどれか。

1. あん摩マッサージ指圧師、はり師、きゅう師等に関する法律
2. 医療類似行為の規制等に関する法律
3. 柔道整復師、あん摩マッサージ師等に関する法律
4. 東洋医学療法士の資格及び業務に関する法律


<QuizAnswer>

**答え: 1**

あはき法の正式名称は「あん摩マッサージ指圧師、はり師、きゅう師等に関する法律」です。「あはき」は「あん摩・はり・きゅう」の頭文字から取った略称です。この法律により、あん摩・マッサージ・指圧・はり・きゅうを業として行うにはそれぞれの国家資格が必要と定められています。

</QuizAnswer>

## 問7

セラピストがカウンセリングを行う際の基本姿勢として、最も適切なものはどれか。

1. 専門家として自分の意見を積極的に伝え、クライアントを指導する
2. クライアントの話を傾聴し、共感的な態度で受容する
3. 施術時間を確保するため、カウンセリングはできるだけ短時間で終える
4. クライアントの訴えに対して、医学的な診断名を伝える


<QuizAnswer>

**答え: 2**

カウンセリングの基本姿勢は、クライアントの話を「傾聴」し、「共感」と「受容」の態度で接することです。一方的に指導するのではなく、クライアント自身が安心して話せる環境をつくることが重要です。また、セラピストは医師ではないため、医学的な診断を行うことは法律で禁じられています。

</QuizAnswer>

## 問8

リラクゼーションサロンにおける衛生管理の基本として、誤っているものはどれか。

1. 使用したタオルは施術ごとに洗濯済みの清潔なものに交換する
2. セラピストは施術前後に手洗い・手指消毒を行う
3. 施術ベッドやリネン類は1日の営業終了時にまとめて1回消毒すればよい
4. 施術室は定期的に換気を行い、清潔な環境を保つ


<QuizAnswer>

**答え: 3**

施術ベッドやリネン類は、クライアントごとに消毒・交換するのが衛生管理の基本です。1日1回の消毒では不十分であり、クライアント間での感染リスクが生じます。タオルの交換、手指消毒、室内換気とあわせて、すべてクライアントごとに清潔な状態を確保することがサロン運営の基本です。

</QuizAnswer>

## 問9

セラピストが広告や宣伝を行う際に、法律上してはならない表現として最も適切なものはどれか。

1. 「リラクゼーション目的のリンパケアサロンです」
2. 「リンパの流れを整え、むくみがスッキリします」
3. 「リンパケアで○○病が治ります」
4. 「お客様の声：施術後、身体が軽くなりました」


<QuizAnswer>

**答え: 3**

医師免許を持たないセラピストが「○○病が治る」などの医療効果を謳う広告を出すことは、医師法や景品表示法、薬機法（旧薬事法）に抵触します。特定の病気の治療や診断を示唆する表現は禁止されています。リラクゼーション目的であることを明示し、医療行為と誤解されない範囲で広告を行う必要があります。

</QuizAnswer>

## 問10

セラピストとクライアントの関係において、職業倫理上避けるべき「多重関係」の説明として正しいものはどれか。

1. 同じクライアントに対して複数の施術メニューを提供すること
2. セラピストとクライアントという関係以外に、私的な関係を同時に持つこと
3. 複数のセラピストが1人のクライアントを担当すること
4. クライアントが複数のサロンに通うこと


<QuizAnswer>

**答え: 2**

多重関係（二重関係）とは、セラピストがクライアントとの専門的な関係に加えて、友人・恋愛・ビジネスパートナーなどの私的な関係を同時に持つことを指します。多重関係はクライアントへの客観的な対応を困難にし、利益相反や搾取のリスクが生じるため、職業倫理上避けるべきとされています。

</QuizAnswer>
