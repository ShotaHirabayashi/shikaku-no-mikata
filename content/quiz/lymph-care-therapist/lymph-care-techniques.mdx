## 問1

セルフリンパマッサージを全身に行う場合、最初に刺激すべき部位はどこか。

1. 足首
2. 鎖骨周辺
3. 鼠径部（そけいぶ）
4. 膝裏

<QuizAnswer>

**答え: 2**

鎖骨周辺は全身のリンパ液が最終的に静脈へ合流する「出口」にあたります。ここを最初に開いておくことで、その後のマッサージで流したリンパ液がスムーズに排出されるようになります。全身のセルフリンパマッサージでは、鎖骨→顔→首・デコルテ→腕→脚→お腹の順が基本です。

</QuizAnswer>

## 問2

リンパマッサージの基本手技のうち、手のひらで皮膚を軽くなでるようにさする手技を何というか。

1. 揉捏法（じゅうねつほう）
2. 叩打法（こうだほう）
3. 軽擦法（けいさつほう）
4. 振動法（しんどうほう）

<QuizAnswer>

**答え: 3**

軽擦法はリンパマッサージにおいて最も基本的な手技で、手のひらを皮膚に密着させて軽くさすることでリンパの流れを促進します。強い圧は不要で、手のひらと皮膚の密着性を保ちながら、優しくなでるように行うことがポイントです。

</QuizAnswer>

## 問3

セルフリンパマッサージを行う際の適切な圧の強さについて、正しいものはどれか。

1. 筋肉に届くようしっかりと強い圧をかける
2. 皮膚が赤くなる程度の中程度の圧をかける
3. 優しく撫でる程度の軽い圧で行う
4. 痛気持ちいいと感じる程度に押し込む

<QuizAnswer>

**答え: 3**

リンパマッサージは優しく撫でる程度の軽い圧で十分に効果を発揮します。圧が強すぎるとリンパ管を圧迫してしまい、かえってリンパ液の流れを妨げることになります。リンパ管は皮膚のすぐ下の浅い層を走っているため、皮膚1枚を動かすような優しいタッチが基本です。

</QuizAnswer>

## 問4

リンパドレナージュにおいて、鎖骨下のリンパの出口付近で「押して離す」動作を繰り返す手技を何というか。

1. エフルラージュ
2. ポンピング
3. ニーディング
4. タッピング

<QuizAnswer>

**答え: 2**

ポンピングとは、押して離す（プッシュ&アップ）動作を優しく繰り返し、ポンプのようにリンパ液の流れを促す手技です。静脈角（鎖骨付近）のリンパの出口で行うことが多く、リンパドレナージュの代表的な手技の一つです。エフルラージュは鎖骨下から胸・肩・首をさすって流す別の手技を指します。

</QuizAnswer>

## 問5

脚のセルフリンパマッサージを行う場合、正しい手順（流す方向）はどれか。

1. 太もも → 膝裏 → ふくらはぎ → 足首 → 足先
2. 足先 → 足首 → ふくらはぎ → 膝裏 → 太もも → 鼠径部
3. 膝裏 → ふくらはぎ → 足首 → 太もも → 鼠径部
4. 鼠径部 → 太もも → 膝裏 → ふくらはぎ → 足先

<QuizAnswer>

**答え: 2**

リンパマッサージの基本原則は「末端からリンパ節に向かって流す」ことです。脚の場合、足先→足首→ふくらはぎ→膝裏→太もも→鼠径部の順に、身体の末端から心臓に近い方向へリンパ液を導くように行います。この順序を守ることで、リンパ液が自然な循環に乗りやすくなります。

</QuizAnswer>

## 問6

腹式呼吸がリンパケアに効果的とされる理由として、最も適切なものはどれか。

1. 腹筋が鍛えられ、お腹周りのリンパ管が太くなるため
2. 横隔膜が動くことで胸腔が陰圧になり、リンパ液が心臓方向へ流れやすくなるため
3. 呼吸によって体温が上がり、リンパ液が蒸発して老廃物が排出されるため
4. 口から息を吐くことでリンパ液が口腔内から直接排出されるため

<QuizAnswer>

**答え: 2**

腹式呼吸では横隔膜が大きく上下に動くことで、胸腔内が陰圧（圧力が低い状態）になります。この陰圧によってリンパ液が心臓方向へ引き上げられやすくなり、全身のリンパ循環が促進されます。リンパケアの実践において、腹式呼吸は手技と組み合わせることで効果を高める重要な技法です。

</QuizAnswer>

## 問7

顔のセルフリンパマッサージにおいて、リンパを流す基本的な方向として正しいものはどれか。

1. 顔の外側から内側（中心）に向かって流す
2. 顔の内側から外側（耳の前方）に向かって流す
3. 額から顎に向かって下方向に流す
4. 顎から額に向かって上方向に流す

<QuizAnswer>

**答え: 2**

顔のセルフリンパマッサージでは、小鼻・頬・目の周り・額の順に、顔の内側（中心）から外側の耳介前リンパ節に向かって優しくなでるように流すのが基本です。耳の前方にあるリンパ節に老廃物を集め、そこからさらに首を通って鎖骨方向へと流していきます。

</QuizAnswer>

## 問8

セルフリンパマッサージを行うタイミングとして、最も効果的とされるのはどれか。

1. 朝起きてすぐの空腹時
2. 食事の直後
3. 入浴後の就寝前
4. 激しい運動の直後

<QuizAnswer>

**答え: 3**

セルフリンパマッサージは入浴後の就寝前が最も効果的とされています。入浴で体が温まることで血流が良くなり、筋肉も柔らかくなっているため、リンパの流れが促進されやすい状態です。また、就寝中は筋肉の動きが少なくなるため、寝る前にリンパを流しておくと翌朝のすっきり感が実感しやすくなります。

</QuizAnswer>

## 問9

リンパドレナージュの施術における特徴として、正しいものはどれか。

1. 指先で強くツボを押す指圧に近い手技である
2. 皮膚1枚だけを動かすような非常に優しいタッチで行う
3. オイルを使わず、素手で筋肉を強く揉みほぐす手技である
4. 振動を加える機器を使用して行うのが基本である

<QuizAnswer>

**答え: 2**

リンパドレナージュはヨーロッパで生まれたオールハンドの手技で、「ドレナージュ（排液）」という名の通り、リンパ液の排出を促す施術です。最大の特徴は皮膚1枚だけを動かすような驚くほど優しいタッチにあります。一般的なマッサージが「圧をかける」ことを基本とするのに対し、リンパドレナージュは「さする」ことを基本とする点が大きく異なります。

</QuizAnswer>

## 問10

四指圧迫法（ししあっぱくほう）でリンパ節を刺激する際の正しい手順はどれか。

1. 素早く強く押し、すぐに離すことを繰り返す
2. 5〜10秒かけてゆっくり押し、5〜10秒止め、5〜10秒かけてゆっくり戻す
3. 1秒ごとにリズミカルに押したり離したりを繰り返す
4. 30秒以上連続で押し続け、一気に離す

<QuizAnswer>

**答え: 2**

四指圧迫法は4本の指を使って優しく押す手技で、垂直ではなく少し角度をつけて表層から皮膚に指をあてます。5〜10秒かけてゆっくり押し、5〜10秒そのまま止めて、5〜10秒かけてゆっくり戻すのが基本です。急激な圧の変化はリンパ管に負担をかけるため、ゆっくりとした動作で行うことが重要です。

</QuizAnswer>
