## 問1

色が持つ機能的な役割のうち、注意を向けていない状態でも目に飛び込んでくる色の性質を何というか。

1. 可読性
2. 視認性
3. 誘目性
4. 識別性


<QuizAnswer>

**答え: 3**

誘目性とは、注意を向けていない状態でも対象物が自然に目に入ってくる性質のことです。一方、視認性は注意を向けた状態での発見しやすさを指します。一般に、暖色系で高彩度の色ほど誘目性が高いとされています。

</QuizAnswer>

## 問2

JIS安全色（JIS Z 9103）において、「禁止」や「防火」を意味する安全色として定められているのはどの色か。

1. 黄
2. 青
3. 緑
4. 赤


<QuizAnswer>

**答え: 4**

JIS安全色では、赤は「禁止」「防火」「停止」「高度の危険」を意味する色として定められています。黄は「注意・警告」、緑は「安全・避難・衛生」、青は「指示・誘導」をそれぞれ表します。

</QuizAnswer>

## 問3

色が生活の中で果たす役割として、「識別性」の活用例に最も該当するものはどれか。

1. 路線図で各路線を異なる色で塗り分ける
2. 工事現場の看板を蛍光黄色にする
3. 病院の壁を淡い緑色にして安らぎを与える
4. 非常口の表示を暗闇でも見えるようにする


<QuizAnswer>

**答え: 1**

識別性とは、複数の対象を色によって区別しやすくする性質のことです。路線図で各路線を異なる色で塗り分けることは、色による識別の代表的な活用例です。選択肢2は誘目性、3は色の心理的効果、4は視認性に関する例です。

</QuizAnswer>

## 問4

暖色系で彩度の高い色が持つ心理的効果として、正しいものはどれか。

1. 集中力を高め、冷静さを促す
2. 眠気を誘い、リラックスさせる
3. 心を高ぶらせ、興奮感を与える
4. 時間の経過を遅く感じさせる


<QuizAnswer>

**答え: 3**

暖色系で彩度の高い色は「興奮色」と呼ばれ、心を高ぶらせる心理的効果があります。赤やオレンジなどがこれに該当し、活気や食欲の増進にもつながります。反対に、寒色系で彩度の低い色は「沈静色」と呼ばれ、心を落ち着かせる効果があります。

</QuizAnswer>

## 問5

2018年に改正されたJIS安全色（JIS Z 9103）の改正理由として、最も適切なものはどれか。

1. 国際規格との色番号を統一するため
2. 色覚の多様性に配慮し、より多くの人が見分けやすい色にするため
3. 印刷コストを削減するために使用色数を減らすため
4. 蛍光色を新たに追加するため


<QuizAnswer>

**答え: 2**

2018年のJIS安全色改正は、色覚の多様性（色覚障害、弱視、白内障など）に配慮したカラーユニバーサルデザインの観点から行われました。従来の安全色では「赤が暗くて黒と紛らわしい」「青と緑の区別がつきにくい」といった指摘があり、8色すべてに調整が加えられました。

</QuizAnswer>

## 問6

色彩環境の考え方において、街並みや景観の基調となる色として最も適切なのはどれか。

1. 高彩度の暖色系の色
2. 低彩度で落ち着いた色
3. 原色に近い鮮やかな色
4. 蛍光色のような目立つ色


<QuizAnswer>

**答え: 2**

景観や街並みの基調色（ベースカラー）には、土や石、木材など自然素材に近い低彩度で落ち着いた色が適しているとされています。高彩度の色は周囲との調和を乱しやすいため、景観色彩計画では基調色を低彩度に抑え、全体の統一感を保つことが重要とされています。

</QuizAnswer>

## 問7

カラーユニバーサルデザイン（CUD）の基本的な考え方として、最も適切なものはどれか。

1. すべての表示を白黒のモノクロにする
2. できるだけ多くの人が見分けやすい色の使い方をする
3. 高齢者だけを対象に色の見やすさを改善する
4. 色の使用を最小限にして文字だけで情報を伝える


<QuizAnswer>

**答え: 2**

カラーユニバーサルデザイン（CUD）とは、色覚の多様性に配慮し、できるだけ多くの人にとって見分けやすい色の組み合わせや表現を用いるという考え方です。色の使用を排除するのではなく、色の選び方や形・文字との併用など、多角的な情報伝達を工夫することが重要です。

</QuizAnswer>

## 問8

生活の中で色が果たす役割のうち、「可読性」の説明として最も適切なものはどれか。

1. 色によって複数の対象を区別しやすくすること
2. 文字や数字が背景の上で読み取りやすいこと
3. 注意を向けていなくても目に飛び込んでくること
4. 対象物が遠くからでも発見しやすいこと


<QuizAnswer>

**答え: 2**

可読性とは、文字や数字などが背景色との関係で読みやすいかどうかを表す性質です。一般に、文字色と背景色の明度差が大きいほど可読性が高くなります。選択肢1は識別性、3は誘目性、4は視認性の説明にあたります。

</QuizAnswer>

## 問9

JIS安全色において、「安全」「避難」「衛生」を表す色はどれか。

1. 赤
2. 黄
3. 緑
4. 青


<QuizAnswer>

**答え: 3**

JIS安全色では、緑は「安全」「避難」「衛生・救護」を意味する色として定められています。非常口のサインに緑色が使われているのはこのためです。赤は「禁止・防火」、黄は「注意・警告」、青は「指示・誘導」を表します。

</QuizAnswer>

## 問10

色が見えることによって私たちの生活にもたらされる効用として、最も適切でないものはどれか。

1. 危険や安全の情報を素早く判断できる
2. 物の状態や変化を見分けることができる
3. 空間や環境に心理的な影響を与える
4. 音や香りの強さを正確に測定できる


<QuizAnswer>

**答え: 4**

色は視覚を通じて得られる情報であり、音や香りの強さを測定する機能はありません。色が見えることで、危険の察知（信号・標識）、食べ物の鮮度や状態の判断、空間の快適さや心理的効果など、生活の多くの場面で重要な役割を果たしています。

</QuizAnswer>
