## 問1

人間の目に見える光（可視光線）の波長範囲として、最も適切なものはどれか。

1. 約180～380nm
2. 約380～780nm
3. 約780～1200nm
4. 約1200～2000nm


<QuizAnswer>

**答え: 2**

可視光線とは、人間の目が光として感じることのできる電磁波の波長範囲で、およそ380nm（紫）から780nm（赤）までです。これより短い波長は紫外線、長い波長は赤外線と呼ばれ、いずれも肉眼では見ることができません。

</QuizAnswer>

## 問2

太陽光などの白色光をプリズムに通すと、光が波長ごとに分かれて虹色の帯（スペクトル）が現れる。この現象を何というか。

1. 干渉
2. 回折
3. 分光
4. 偏光


<QuizAnswer>

**答え: 3**

白色光をプリズムに通すと、波長によって屈折率が異なるため、光が波長ごとに分かれます。この現象を「分光（ぶんこう）」といいます。ニュートンが太陽光をプリズムで分光し、白色光が様々な色の光の混合であることを示した実験は有名です。

</QuizAnswer>

## 問3

可視光線の中で、最も波長が短い光の色として正しいものはどれか。

1. 赤
2. 黄
3. 緑
4. 紫


<QuizAnswer>

**答え: 4**

可視光線のスペクトルにおいて、紫の光は約380～430nmと最も波長が短く、赤の光は約640～780nmと最も波長が長くなります。虹の色の並びは外側から赤・橙・黄・緑・青・藍・紫の順で、内側の紫ほど波長が短くなっています。

</QuizAnswer>

## 問4

光の三原色（加法混色の三原色）の正しい組み合わせはどれか。

1. 赤（R）・緑（G）・青（B）
2. 赤（R）・黄（Y）・青（B）
3. シアン（C）・マゼンタ（M）・イエロー（Y）
4. 赤（R）・緑（G）・黄（Y）


<QuizAnswer>

**答え: 1**

光の三原色は赤（Red）・緑（Green）・青（Blue）の3色で、これらを混ぜ合わせると白色光になります。これを加法混色といい、色光を重ねるほど明るくなるのが特徴です。なお、選択肢3のシアン・マゼンタ・イエローは色料（絵の具やインク）の三原色で、減法混色に用いられます。

</QuizAnswer>

## 問5

減法混色に関する説明として、最も適切なものはどれか。

1. 色光を重ねるほど明るくなり、3色を混ぜると白になる
2. 色料（インクや絵の具）を混ぜるほど暗くなり、3色を混ぜると理論上黒に近づく
3. 赤・緑・青の3色の光を混ぜ合わせる方法である
4. テレビやスマートフォンの画面で使われている混色方式である


<QuizAnswer>

**答え: 2**

減法混色とは、色料（インク・絵の具・フィルターなど）を混ぜ合わせる混色方法です。色料を混ぜるごとに光が吸収されて暗くなり、シアン・マゼンタ・イエローの三原色をすべて混ぜると理論上は黒になります。選択肢1・3・4は加法混色の説明です。

</QuizAnswer>

## 問6

色温度に関する説明として、正しいものはどれか。

1. 色温度の単位はルクス（lx）で表される
2. 色温度が高いほど赤みを帯びた光色になる
3. 色温度が高いほど青みを帯びた光色になる
4. 色温度は光の明るさ（照度）を表す指標である


<QuizAnswer>

**答え: 3**

色温度は光の色合いを数値で表す指標で、単位はケルビン（K）を用います。色温度が高くなるほど青白い光色になり、低くなるほど赤みを帯びた暖かい光色になります。例えば、ろうそくの光は約2000Kで赤みが強く、晴天の昼光は約6500Kで青白い光です。

</QuizAnswer>

## 問7

演色性に関する説明として、最も適切なものはどれか。

1. 光源そのものの色の鮮やかさを示す指標である
2. 光源が物体の色の見え方に与える影響を示す性質で、Raの値が高いほど自然光に近い色再現ができる
3. 光源の明るさの度合いを表す数値である
4. 光源から出る光の波長の長さを示す指標である


<QuizAnswer>

**答え: 2**

演色性とは、光源が照らす物体の色の見え方に及ぼす性質のことです。平均演色評価数（Ra）は0～100の数値で表され、100に近いほど基準光（自然光）の下での色の見え方に近く、色を忠実に再現できることを意味します。美術館や食品売り場など、色の正確さが重要な場所ではRaの高い光源が選ばれます。

</QuizAnswer>

## 問8

メタメリズム（条件等色）の説明として、正しいものはどれか。

1. どのような光源の下でも常に同じ色に見える現象
2. 分光分布が異なる2つの色が、特定の条件下では同じ色に見える現象
3. 色を見続けると補色が見える現象
4. 周囲の色によって色の見え方が変わる現象


<QuizAnswer>

**答え: 2**

メタメリズム（条件等色）とは、分光分布（波長ごとの光の強さの組み合わせ）が異なる2つの色が、ある特定の照明光や観察者の条件の下では同じ色に見える現象です。照明が変わると色が異なって見える場合があるため、衣服やインテリアの色合わせでは複数の光源で確認することが重要です。

</QuizAnswer>

## 問9

物体に白色光が当たったとき、その物体が赤く見える理由として正しいものはどれか。

1. 物体が赤い光を吸収し、それ以外の光を反射しているから
2. 物体が赤い光を反射し、それ以外の波長の光を吸収しているから
3. 物体自身が赤い光を発光しているから
4. 物体の表面温度が高く、赤い光を放射しているから


<QuizAnswer>

**答え: 2**

物体の色は、光源から当たった光のうち、物体が吸収せずに反射した光の波長によって決まります。赤く見える物体は、白色光に含まれる赤の波長域の光を主に反射し、青や緑などそれ以外の波長の光を吸収しています。このように物体の色は、光の選択的な吸収と反射によって生まれます。

</QuizAnswer>

## 問10

一般的な照明の色温度について、低い順に正しく並べたものはどれか。

1. 昼光色 → 昼白色 → 電球色
2. 電球色 → 昼光色 → 昼白色
3. 電球色 → 昼白色 → 昼光色
4. 昼白色 → 電球色 → 昼光色


<QuizAnswer>

**答え: 3**

照明の色温度は、電球色（約2700K）→ 昼白色（約5000K）→ 昼光色（約6500K）の順に高くなります。電球色は赤みのある暖かい光で、色温度が最も低く、昼光色は青白いさわやかな光で色温度が最も高くなります。色温度が低いほど暖色系、高いほど寒色系の光色になることを覚えておきましょう。

</QuizAnswer>
