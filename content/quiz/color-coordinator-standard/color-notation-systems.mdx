## 問1

色の三属性のうち、有彩色・無彩色の両方に共通して存在する属性はどれか。

1. 色相
2. 彩度
3. 明度
4. 色温度


<QuizAnswer>

**答え: 3**

明度は色の明るさの度合いを表す属性で、有彩色にも無彩色にも存在する唯一の属性です。無彩色（白・灰・黒）は色相と彩度を持たず、明度のみで区別されます。色温度は色の三属性には含まれません。

</QuizAnswer>

## 問2

マンセル表色系における基本5色相の正しい組み合わせはどれか。

1. R（赤）、Y（黄）、G（緑）、B（青）、P（紫）
2. R（赤）、O（橙）、Y（黄）、G（緑）、B（青）
3. R（赤）、Y（黄）、G（緑）、B（青）、W（白）
4. R（赤）、YR（黄赤）、G（緑）、B（青）、P（紫）


<QuizAnswer>

**答え: 1**

マンセル表色系の基本5色相は R（赤）、Y（黄）、G（緑）、B（青）、P（紫）です。これに中間5色相の YR（黄赤）、GY（黄緑）、BG（青緑）、PB（青紫）、RP（赤紫）を加えた10色相が主要色相となります。各色相をさらに10分割し、合計100色相で表現します。

</QuizAnswer>

## 問3

マンセル表色系で有彩色を表記する場合の正しい順序はどれか。

1. 明度 色相/彩度
2. 彩度 色相/明度
3. 色相 彩度/明度
4. 色相 明度/彩度


<QuizAnswer>

**答え: 4**

マンセル表色系の有彩色の表記法は「H V/C」、すなわち「色相（Hue） 明度（Value）/彩度（Chroma）」の順です。例えば「5R 4/14」は、色相5R、明度4、彩度14を意味します。無彩色の場合は「N＋明度」（例：N5）で表記します。

</QuizAnswer>

## 問4

PCCS（日本色研配色体系）の色相環は何色相で構成されているか。

1. 10色相
2. 24色相
3. 20色相
4. 12色相


<QuizAnswer>

**答え: 2**

PCCSの色相環は24色相で構成されています。心理4原色である赤（2:R）、黄（8:Y）、緑（12:G）、青（18:B）を基準に、まず心理補色を加えて8色相、さらに補間して12色相、最終的に24色相に分割されます。色相番号1（紫みの赤：1:pR）から24（赤紫：24:RP）まで配置されています。

</QuizAnswer>

## 問5

PCCSのトーン分類において、有彩色のトーンは全部でいくつあるか。

1. 8種類
2. 10種類
3. 12種類
4. 14種類


<QuizAnswer>

**答え: 3**

PCCSでは有彩色のトーンを12種類に分類しています。v（ビビッド）、b（ブライト）、s（ストロング）、dp（ディープ）、lt（ライト）、sf（ソフト）、d（ダル）、dk（ダーク）、p（ペール）、ltg（ライトグレイッシュ）、g（グレイッシュ）、dkg（ダークグレイッシュ）の12種類です。

</QuizAnswer>

## 問6

CIE xy色度図の外周の曲線部分は何と呼ばれるか。

1. スペクトル軌跡
2. 純紫軌跡
3. 等色関数
4. 色度境界線


<QuizAnswer>

**答え: 1**

CIE xy色度図の外周の馬蹄形をした曲線部分は「スペクトル軌跡」と呼ばれ、単色光（スペクトル光）の色度を結んだ線です。一方、馬蹄形の底辺にあたる直線部分は「純紫軌跡」と呼ばれ、スペクトルには存在しない紫系の色を表しています。

</QuizAnswer>

## 問7

JIS Z 8102「物体色の色名」における系統色名の基本色名として、有彩色は何色が定められているか。

1. 8色
2. 12色
3. 13色
4. 10色


<QuizAnswer>

**答え: 4**

JIS Z 8102の系統色名では、有彩色の基本色名として赤、黄赤、黄、黄緑、緑、青緑、青、青紫、紫、赤紫の10色が定められています。これに無彩色の白、灰色、黒の3色を加えた計13色が基本色名の全体となります。系統色名はこれらの基本色名に修飾語を組み合わせて表現します。

</QuizAnswer>

## 問8

CIE表色系の三刺激値X、Y、Zのうち、輝度（明るさ）に対応するように設計されているのはどれか。

1. X
2. Y
3. Z
4. XとZの平均値


<QuizAnswer>

**答え: 2**

CIE XYZ表色系では、三刺激値のうちY値が輝度（明るさ）に対応するように設計されています。これはCIEが意図的に等色関数ȳ（ワイバー）を人間の明所視の比視感度曲線（V(λ)）と一致させたためです。X値とZ値には輝度の情報は含まれていません。

</QuizAnswer>

## 問9

JIS規格における「慣用色名」の説明として最も適切なものはどれか。

1. 伝統的・習慣的に使われてきた固有の色名である
2. 修飾語と基本色名を組み合わせて体系的に表現する色名である
3. 国際的に統一された色の番号で表す色名である
4. マンセル値で直接指定する色名である


<QuizAnswer>

**答え: 1**

慣用色名は「あかね色」「藍色」「朱色」「利休鼠」「萌黄」など、伝統的・習慣的に使われてきた固有の色名です。選択肢2の説明は「系統色名」にあたります。慣用色名は文化的背景を持つ固有名ですが、色名だけでは正確な色が特定しにくい場合があるため、JISでは各慣用色名に対応する系統色名も定められています。

</QuizAnswer>

## 問10

マンセル表色系で無彩色を「N5」と表記した場合、「5」が表しているのはどれか。

1. 色相
2. 彩度
3. 明度
4. 色相番号


<QuizAnswer>

**答え: 3**

マンセル表色系で無彩色は「N＋明度」の形式で表記します。Nは Neutral（無彩色）を意味し、続く数値は明度を表します。N5は明度5の中間的な灰色を意味します。無彩色は色相と彩度を持たないため、明度のみで表現されます。マンセルの明度は理想的な黒を0、理想的な白を10として設定されています。

</QuizAnswer>
