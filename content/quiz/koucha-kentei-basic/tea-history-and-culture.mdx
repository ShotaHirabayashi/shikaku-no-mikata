## 問1

ヨーロッパに初めて茶を持ち込んだ国はどこか。

1. イギリス
2. ポルトガル
3. オランダ
4. スペイン


<QuizAnswer>

**答え: 3**

1610年にオランダ東インド会社が中国から茶を買い付け、ヨーロッパに初めて茶を持ち込みました。当時オランダは海上貿易で世界に進出しており、東洋の産物をヨーロッパに紹介する役割を果たしていました。イギリスに紅茶が入ったのはそれより後の1657年頃とされています。

</QuizAnswer>

## 問2

1662年にイギリス国王チャールズ2世のもとに嫁ぎ、イギリス宮廷に茶を飲む習慣を広めた人物は誰か。

1. アンナ・マリア
2. キャサリン・オブ・ブラガンザ
3. クイーン・アン
4. ヴィクトリア女王


<QuizAnswer>

**答え: 2**

ポルトガルのブラガンザ家出身のキャサリンは、「万病に効く東洋の神秘薬」として茶と大量の砂糖を持参してイギリスに嫁ぎました。彼女のサロンでは宮廷の貴婦人たちに砂糖入りの茶がふるまわれ、上流階級に茶を飲む習慣が広まるきっかけとなりました。

</QuizAnswer>

## 問3

アフタヌーンティーの習慣を始めたとされる人物は誰か。

1. キャサリン・オブ・ブラガンザ
2. クイーン・アン
3. ヴィクトリア女王
4. アンナ・マリア（第7代ベッドフォード公爵夫人）


<QuizAnswer>

**答え: 4**

19世紀、第7代ベッドフォード公爵夫人アンナ・マリア（1783〜1857）が、夕食までの空腹を満たすために午後4時頃に自室で軽食とお茶を楽しむ習慣を始めました。やがてこれが友人を招いての社交の場となり、イギリス上流階級にアフタヌーンティーの文化が広まっていきました。

</QuizAnswer>

## 問4

ボストン茶会事件が起きた年はいつか。

1. 1773年
2. 1776年
3. 1789年
4. 1765年


<QuizAnswer>

**答え: 1**

1773年12月、イギリス本国が茶法を制定して東インド会社に茶の専売権を与えたことに反発した植民地の急進派が、モホーク族に変装してボストン港の船に乗り込み、342箱の茶箱を海中に投棄しました。この事件はアメリカ独立戦争の引き金の一つとなった歴史的事件です。

</QuizAnswer>

## 問5

1717年にロンドンで紅茶専門店「ゴールデン・ライオン」を開業し、現在も続く紅茶ブランドの礎を築いた人物は誰か。

1. トーマス・リプトン
2. ロバート・ブルース
3. トーマス・トワイニング
4. ロバート・フォーチュン


<QuizAnswer>

**答え: 3**

トーマス・トワイニングは1717年にロンドンのストランド街に紅茶専門店「ゴールデン・ライオン」を開業しました。当時、茶はコーヒーハウスで男性だけが飲めるものでしたが、トワイニングは女性も入れる専門店を開き、紅茶の普及に貢献しました。1837年にはヴィクトリア女王から王室御用達の認定を受けています。

</QuizAnswer>

## 問6

18世紀初頭のイギリスで、一日に何度も紅茶を飲む習慣を持ち、宮廷での紅茶文化を定着させた女王は誰か。

1. エリザベス1世
2. アン女王
3. メアリー2世
4. ヴィクトリア女王


<QuizAnswer>

**答え: 2**

アン女王（在位1702〜1714年）は紅茶を非常に好み、一日に何度も紅茶を飲む習慣を持っていました。前代のメアリー2世も茶を好みましたが、アン女王の時代にイギリス宮廷における紅茶文化がより一層定着したとされています。

</QuizAnswer>

## 問7

ボストン茶会事件の直接の原因となった、イギリス本国が制定した法律は何か。

1. 茶法（Tea Act）
2. 印紙法（Stamp Act）
3. 航海条例（Navigation Acts）
4. タウンゼンド諸法（Townshend Acts）


<QuizAnswer>

**答え: 1**

1773年にイギリス本国議会が制定した茶法（Tea Act）は、経営難に陥っていた東インド会社に植民地での茶の専売権を与えるものでした。これにより植民地の茶商人は打撃を受け、「代表なくして課税なし」の主張とともに強く反発し、ボストン茶会事件へとつながりました。

</QuizAnswer>

## 問8

明治時代に中国やインドに渡り、紅茶の栽培・製造法を調査して日本に持ち帰り、日本の紅茶産業の基礎を築いた人物は誰か。

1. 岡倉天心
2. 千利休
3. 大谷嘉兵衛
4. 多田元吉


<QuizAnswer>

**答え: 4**

多田元吉は1875年（明治8年）に中国の紅茶産地を、翌1876年にはインドのアッサムやダージリンを訪れ、茶の栽培・製造法を調査しました。帰国後に著した『紅茶製法纂要』は日本初の本格的な紅茶製造技術書となり、日本の国産紅茶生産の礎を築きました。

</QuizAnswer>

## 問9

スコットランド出身で、セイロン（現スリランカ）に茶園を所有し、「茶園から直接ティーポットへ」をスローガンに紅茶の大衆化に貢献した人物は誰か。

1. トーマス・トワイニング
2. トーマス・リプトン
3. ジェームズ・テイラー
4. ロバート・ブルース


<QuizAnswer>

**答え: 2**

トーマス・リプトンはスコットランドのグラスゴー出身の実業家で、セイロンに茶園を購入し、中間業者を排除して品質の良い紅茶を安価に提供するビジネスモデルを確立しました。1895年にはヴィクトリア女王から王室御用達に認定され、紅茶を上流階級だけでなく一般大衆にも広める大きな役割を果たしました。

</QuizAnswer>

## 問10

イギリスに最初に茶が伝わった17世紀頃、茶はどのような場所で主に飲まれていたか。

1. 教会
2. 一般家庭
3. コーヒーハウス
4. レストラン


<QuizAnswer>

**答え: 3**

17世紀のイギリスでは、茶はコーヒーハウスで提供されていました。コーヒーハウスは男性のみが出入りできる社交場で、コーヒーとともに茶も飲まれていました。女性は入場できなかったため、後にトワイニングが女性も入れる紅茶専門店を開業したことが画期的とされています。

</QuizAnswer>
