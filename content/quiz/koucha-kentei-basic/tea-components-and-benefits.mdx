## 問1

紅茶の渋みの主な原因となっている成分はどれか。

1. カフェイン
2. テアニン
3. タンニン（ポリフェノール）
4. ビタミンC


<QuizAnswer>

**答え: 3**

紅茶の渋みはタンニン（ポリフェノール）によるものです。タンニンは紅茶の乾燥茶葉中に約10〜18％含まれており、紅茶の味わいを特徴づける重要な成分です。カフェインは苦味、テアニンはうま味・甘味に関わる成分です。

</QuizAnswer>

## 問2

紅茶に含まれるテアニンは、どのような種類の成分か。

1. ポリフェノール
2. アミノ酸
3. ビタミン
4. 脂肪酸


<QuizAnswer>

**答え: 2**

テアニンはお茶に特有のアミノ酸の一種です。茶樹の根で合成され葉に蓄えられる成分で、お茶以外の食品にはほとんど含まれていません。リラックス効果やうま味に関わる成分として知られています。

</QuizAnswer>

## 問3

紅茶100mlあたりのカフェイン含有量として、日本食品標準成分表に記載されているおおよその数値はどれか。

1. 約10mg
2. 約30mg
3. 約60mg
4. 約100mg


<QuizAnswer>

**答え: 2**

紅茶100mlあたりのカフェイン含有量は約30mgです。これはコーヒー（100mlあたり約60mg）のおよそ半分にあたります。なお、緑茶（煎茶）は100mlあたり約20mgで、紅茶はコーヒーと緑茶の中間程度のカフェイン量です。

</QuizAnswer>

## 問4

紅茶の発酵（酸化）過程で、カテキン類が結合して生成される赤橙色の色素成分はどれか。

1. テアニン
2. テアフラビン
3. カロテン
4. アントシアニン


<QuizAnswer>

**答え: 2**

テアフラビンは、紅茶の製造過程で2つのカテキン分子が酸化酵素の働きにより結合して生成される成分です。紅茶の美しい橙赤色（水色）を生み出す重要な色素であり、紅茶の品質を左右する指標のひとつとされています。

</QuizAnswer>

## 問5

テアニンを摂取したときに脳内で増加するとされる脳波はどれか。

1. β波（ベータ波）
2. δ波（デルタ波）
3. α波（アルファ波）
4. θ波（シータ波）


<QuizAnswer>

**答え: 3**

テアニンを摂取すると、約40〜50分後にリラックス状態を示すα波（アルファ波）が増加することが研究で確認されています。α波が優勢になることで、ストレスの軽減や緊張状態の緩和、自律神経の安定などの効果が期待されます。

</QuizAnswer>

## 問6

紅茶に含まれるカフェインの作用として、正しくないものはどれか。

1. 覚醒作用（眠気を覚ます）
2. 利尿作用
3. 鉄分の吸収を促進する作用
4. 脂肪の分解を促す作用


<QuizAnswer>

**答え: 3**

カフェインには覚醒作用、利尿作用、脂肪分解促進作用などがあります。鉄分の吸収に影響を与えるのはカフェインではなくタンニン（ポリフェノール）であり、タンニンは鉄分の吸収を「促進」ではなく「阻害（緩やかにする）」する作用があります。

</QuizAnswer>

## 問7

紅茶に含まれるテアルビジンの特徴として、最も適切なものはどれか。

1. 紅茶の明るいオレンジ色を生み出す
2. 紅茶の褐色を強め、複雑な酸化重合体である
3. 紅茶のうま味を生み出すアミノ酸である
4. 紅茶の香り成分の中心である


<QuizAnswer>

**答え: 2**

テアルビジンは、テアフラビンがさらに複雑に酸化重合した成分の集合体で、紅茶の褐色を強める役割を持っています。テアフラビンが明るい橙赤色を、テアルビジンが深い褐色をもたらし、両者のバランスが紅茶の水色（すいしょく）を決定します。化学的には未解明の部分が多い成分です。

</QuizAnswer>

## 問8

紅茶のポリフェノールに期待される健康効果として、最も関連が深いものはどれか。

1. 骨密度の増加
2. 視力の回復
3. 抗酸化作用
4. 血糖値の急上昇


<QuizAnswer>

**答え: 3**

紅茶のポリフェノール（テアフラビン、テアルビジン、カテキン類など）には強い抗酸化作用があり、体内の活性酸素を除去する働きが期待されています。この抗酸化作用により、生活習慣病の予防やアンチエイジングへの効果があると考えられています。

</QuizAnswer>

## 問9

紅茶に含まれるテアニンが持つとされるカフェインとの関係について、正しいものはどれか。

1. カフェインの効果を増強する
2. カフェインの興奮作用を穏やかにする
3. カフェインを分解して無効にする
4. カフェインの吸収を完全に阻害する


<QuizAnswer>

**答え: 2**

テアニンにはカフェインの興奮作用を穏やかに抑制する働きがあります。そのため、紅茶にはカフェインが含まれているにもかかわらず、コーヒーに比べてマイルドな覚醒効果をもたらすとされています。これが紅茶を飲むとリラックスしながらも適度に集中できる理由のひとつです。

</QuizAnswer>

## 問10

緑茶に多く含まれるカテキン類が、紅茶の製造過程で酸化・変化することで生まれる紅茶特有のポリフェノールの組み合わせとして正しいものはどれか。

1. テアニンとカフェイン
2. テアフラビンとテアルビジン
3. タウリンとカテキン
4. ビタミンCとタンニン


<QuizAnswer>

**答え: 2**

紅茶の製造過程（酸化発酵）で、緑茶に多く含まれるカテキン類が酸化酵素の作用により結合・重合し、テアフラビンやテアルビジンといった紅茶特有のポリフェノールが生成されます。これらは紅茶の赤い水色や独特の渋み・コクをもたらす重要な成分です。

</QuizAnswer>
