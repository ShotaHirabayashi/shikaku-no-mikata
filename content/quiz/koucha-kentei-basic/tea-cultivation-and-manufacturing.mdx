## 問1

茶の木の学名（属名を含む）として正しいものはどれか。

1. カメリア・ジャポニカ
2. カメリア・シネンシス
3. テア・インディカ
4. テア・アッサミカ


<QuizAnswer>

**答え: 2**

緑茶・紅茶・烏龍茶などすべてのお茶は、同じ「カメリア・シネンシス（Camellia sinensis）」という茶の木から作られます。カメリア・ジャポニカは観賞用の椿（ツバキ）の学名であり、茶の木とは別の種です。

</QuizAnswer>

## 問2

茶樹の品種のうち「アッサム種」の特徴として正しいものはどれか。

1. 灌木型で最大3m程度にしか成長しない
2. 寒さに強く、日本や中国で広く栽培されている
3. 葉が大きく、葉面に深いシワがあるのが特徴である
4. カテキン含有量が少なく、主に緑茶の原料に使われる


<QuizAnswer>

**答え: 3**

アッサム種はインドのアッサム地方を原産とする品種で、葉が大きく葉面に深いシワが走っているのが特徴です。喬木型で最大10mほどに成長し、高温多湿を好みます。カテキン含有量が多く酸化酵素の活性が強いため、主に紅茶の原料として使われます。選択肢1・2・4はいずれも中国種の特徴です。

</QuizAnswer>

## 問3

茶樹の品種のうち「中国種」が主に緑茶の原料に適している理由として、最も正しいものはどれか。

1. 葉が大きく水分を多く含むため
2. 高温多湿の環境でよく育つため
3. カテキン含有量が少なく、酸化酵素の活性が弱いため
4. 茶葉の色素が濃く、鮮やかな緑色が出やすいため


<QuizAnswer>

**答え: 3**

中国種はカテキン（渋味成分）の含有量が少なく、酸化酵素の活性も弱いため、酸化発酵がしづらい特徴があります。そのため、発酵させずに作る緑茶の原料として適しています。一方、アッサム種はカテキンが多く酸化酵素の活性が強いため、完全発酵させる紅茶に向いています。

</QuizAnswer>

## 問4

紅茶のオーソドックス製法における工程の正しい順序はどれか。

1. 揉捻 → 萎凋 → 酸化発酵 → 乾燥
2. 萎凋 → 酸化発酵 → 揉捻 → 乾燥
3. 萎凋 → 揉捻 → 乾燥 → 酸化発酵
4. 萎凋 → 揉捻 → 酸化発酵 → 乾燥


<QuizAnswer>

**答え: 4**

オーソドックス製法の工程は「萎凋（いちょう）→ 揉捻（じゅうねん）→ 酸化発酵 → 乾燥」の順に進みます。まず萎凋で茶葉の水分を取り除き、揉捻で細胞組織を壊して酸化酵素を空気に触れさせ、酸化発酵で紅茶の色・香り・味を形成し、最後に乾燥で酵素の活性を止めて仕上げます。

</QuizAnswer>

## 問5

紅茶の製造工程における「萎凋（いちょう）」の主な目的として正しいものはどれか。

1. 茶葉の酸化酵素を完全に破壊すること
2. 茶葉の水分を取り除き、次の揉捻工程をやりやすくすること
3. 茶葉を丸い粒状に成形すること
4. 茶葉に含まれる微生物を活性化させること


<QuizAnswer>

**答え: 2**

萎凋は、摘み取った生葉に含まれている水分の約半分を取り除く工程です。これにより生葉の総重量は60〜65%に減少し、次の揉捻工程で茶葉を揉みやすくします。現在では萎凋槽を使い8〜10時間温風を送る「人工萎凋」が主流ですが、かつては15〜20時間かけて日陰干しにする「自然萎凋」が行われていました。

</QuizAnswer>

## 問6

紅茶の製造工程における「揉捻（じゅうねん）」で起こることとして、最も適切なものはどれか。

1. 高温の熱風で茶葉を乾燥させ、酵素の働きを止める
2. 茶葉の細胞組織を破壊し、酸化酵素を含む成分を外部に絞り出す
3. 茶葉を蒸して鮮度を保つ
4. 茶葉を低温の発酵室で数日間寝かせる


<QuizAnswer>

**答え: 2**

揉捻は茶葉に撚りを与えながら揉むことで、茶葉の細胞組織を破壊し、葉の中の酸化酵素を含んだ成分を外部に絞り出す工程です。酸化酵素が空気中の酸素に触れることで活性化し、カテキンなどの酸化発酵が始まります。揉捻の時間は通常45〜90分程度です。

</QuizAnswer>

## 問7

CTC製法の「CTC」が表す3つの工程の正しい組み合わせはどれか。

1. Cut（切断）、Turn（回転）、Compress（圧縮）
2. Crush（押しつぶす）、Tear（引きちぎる）、Curl（丸める）
3. Clean（洗浄）、Toast（焙煎）、Cool（冷却）
4. Collect（収集）、Trim（整形）、Classify（分類）


<QuizAnswer>

**答え: 2**

CTC製法のCTCは、Crush（押しつぶす）、Tear（引きちぎる）、Curl（丸める）の頭文字をとったものです。この3つの工程をひとつの機械で連続的に行うことで、オーソドックス製法に比べて大幅に生産効率を高めています。世界の紅茶総生産量の約3分の2がCTC製法で生産されています。

</QuizAnswer>

## 問8

紅茶の酸化発酵工程について正しいものはどれか。

1. 微生物の働きによって発酵が進む
2. 茶葉中の酸化酵素の働きによりカテキン類が酸化する
3. 100度以上の高温環境で発酵させる
4. 発酵室で24時間以上放置する必要がある


<QuizAnswer>

**答え: 2**

紅茶の「発酵」は、一般的な食品の発酵（微生物による）とは異なり、茶葉中の酸化酵素の働きによってカテキン類が酸化する化学反応です。室温25〜26度、湿度90%の発酵室で2〜3時間放置することで、緑色だった茶葉が鮮やかな赤銅色に変化し、紅茶独特の芳香が生まれます。

</QuizAnswer>

## 問9

緑茶・烏龍茶・紅茶の発酵度による分類として正しいものはどれか。

1. 緑茶＝完全発酵茶、烏龍茶＝半発酵茶、紅茶＝不発酵茶
2. 緑茶＝半発酵茶、烏龍茶＝不発酵茶、紅茶＝完全発酵茶
3. 緑茶＝不発酵茶、烏龍茶＝半発酵茶、紅茶＝完全発酵茶
4. 緑茶＝不発酵茶、烏龍茶＝完全発酵茶、紅茶＝半発酵茶


<QuizAnswer>

**答え: 3**

緑茶は茶葉を発酵させずに作る「不発酵茶」、烏龍茶は途中で発酵を止めて作る「半発酵茶」、紅茶は完全に発酵させて作る「完全発酵茶」に分類されます。すべて同じカメリア・シネンシスの茶葉から作られますが、発酵度合いの違いによって色・香り・味わいが大きく異なります。

</QuizAnswer>

## 問10

紅茶の製造工程における「乾燥」の主な目的として正しいものはどれか。

1. 茶葉に香ばしい焙煎香をつけるため
2. 茶葉の水分を増やして柔らかくするため
3. 酸化酵素の活性を止め、発酵を終了させるため
4. 茶葉の色を緑色に戻すため


<QuizAnswer>

**答え: 3**

乾燥工程では、100度前後の高温熱風を当てることで酸化酵素の活性を止め、それ以上の発酵が進まないようにします。発酵終了時に約60%あった茶葉の水分を3〜5%まで減少させることで、保存性も高めます。この工程のタイミングが紅茶の品質を大きく左右するため、熟練した判断が求められます。

</QuizAnswer>
