## 問1

紅茶をおいしくいれるための基本原則「ゴールデンルール」は、どこの国に古くから伝わるいれ方か。

1. フランス
2. インド
3. イギリス
4. 中国


<QuizAnswer>

**答え: 3**

ゴールデンルールはイギリスに古くから伝わる紅茶のいれ方の基本原則です。茶葉の持ち味を最大限に引き出すための5つのポイントがまとめられており、紅茶をおいしくいれるための最も正統な方法として知られています。

</QuizAnswer>

## 問2

ゴールデンルールの5つのポイントに含まれないものはどれか。

1. ティーポットをあらかじめ温めておく
2. 茶葉の分量を正しく量る
3. 茶葉を蒸らす時間をしっかり待つ
4. ミネラルウォーターを使用する


<QuizAnswer>

**答え: 4**

ゴールデンルールの5つのポイントは、①良質な茶葉を使う、②ティーポットを温める、③茶葉の分量を量る、④沸騰したての熱湯を使う、⑤蒸らし時間を守る、です。水は汲みたての新鮮な水道水が推奨されており、ミネラルウォーターの使用は必須条件ではありません。

</QuizAnswer>

## 問3

紅茶をいれるとき、ティーポットに熱湯を注ぐと茶葉が上下に浮き沈みする現象を何というか。

1. ローリング
2. ジャンピング
3. ブリューイング
4. スティーピング


<QuizAnswer>

**答え: 2**

この現象は「ジャンピング」と呼ばれます。沸騰したての熱湯をポットに勢いよく注ぐと、お湯の対流によって茶葉が上下に浮き沈みし、茶葉の成分が十分に抽出されます。ジャンピングが起こることで、紅茶の味や香りがしっかりと引き出されます。

</QuizAnswer>

## 問4

紅茶をいれる際に使用する水として最も適切なものはどれか。

1. 一度沸騰させて冷ました湯を再沸騰させたもの
2. 汲みたての水道水を沸騰させたもの
3. 長時間くみ置きした水を沸騰させたもの
4. 80℃程度に温めたミネラルウォーター


<QuizAnswer>

**答え: 2**

紅茶をいれるには、汲みたての新鮮な水道水を勢いよく沸騰させたものが最適です。汲みたての水には空気が多く含まれており、ジャンピングを起こしやすくなります。二度沸かしの湯やくみ置きの水は空気が抜けてしまうため、ジャンピングが起こりにくく不向きです。

</QuizAnswer>

## 問5

細かい茶葉（ブロークンタイプ）を使って紅茶をいれる場合、蒸らし時間の目安として適切なものはどれか。

1. 30秒〜1分
2. 2分半〜3分
3. 5分〜6分
4. 8分〜10分


<QuizAnswer>

**答え: 2**

細かい茶葉（ブロークンタイプ）の蒸らし時間は2分半〜3分が目安です。一方、大きい茶葉（フルリーフタイプ）は3〜4分が目安となります。蒸らし時間が短すぎると味が薄く、長すぎると渋みが強く出てしまうため、茶葉の大きさに合わせた適切な時間を守ることが大切です。

</QuizAnswer>

## 問6

ティーバッグで紅茶をいれる際の正しい手順はどれか。

1. カップにティーバッグを入れてから熱湯を注ぐ
2. カップに水を入れてから電子レンジで加熱する
3. ティーバッグを熱湯に浸して素早く上下に振る
4. カップにぬるめの湯を入れてからティーバッグを入れる


<QuizAnswer>

**答え: 1**

ティーバッグで紅茶をいれる際は、カップにティーバッグを入れてから沸騰した熱湯を注ぎ、小皿やソーサーなどで蓋をして蒸らすのが正しい方法です。ティーバッグを上下に振ると余分な渋みが出てしまいます。また、ぬるい湯では茶葉の成分が十分に抽出されません。

</QuizAnswer>

## 問7

おいしいアイスティーをいれる方法として、一般的に推奨されている手法はどれか。

1. 通常の濃さで紅茶をいれ、冷蔵庫で数時間かけてゆっくり冷やす
2. 通常の半分の湯量で2倍の濃さにいれ、氷の上に注いで急冷する
3. 常温の水に茶葉を長時間浸して自然抽出する
4. 通常どおりにいれた紅茶をそのまま自然に冷ます


<QuizAnswer>

**答え: 2**

おいしいアイスティーの基本は、通常の半分の湯量で2倍の濃さに抽出し、たっぷりの氷の上に一気に注いで急冷する方法（オンザロックス方式）です。ゆっくり冷やすと「クリームダウン」と呼ばれる白濁現象が起きやすくなるため、急冷することが透明で美しいアイスティーを作るポイントです。

</QuizAnswer>

## 問8

ミルクティーに使用するミルクとして最も適切なものはどれか。

1. コーヒー用クリーム（ポーション）
2. 生クリーム
3. 普通の牛乳
4. コンデンスミルク（練乳）


<QuizAnswer>

**答え: 3**

ミルクティーに最も適しているのは普通の牛乳です。コーヒー用のクリーム（ポーション）は植物性脂肪が多く紅茶の風味を損なうため適しません。牛乳は温めずに冷たいままか常温のものを使うことで、紅茶の香りをより楽しむことができます。

</QuizAnswer>

## 問9

チャイ（マサラチャイ）を作る際の手順として正しいものはどれか。

1. ポットに茶葉を入れ、熱湯を注いでから牛乳を加える
2. 牛乳を先に沸騰させてから茶葉とスパイスを加える
3. 鍋で水と茶葉・スパイスを煮出してから牛乳を加えて煮込む
4. 冷たい牛乳に茶葉とスパイスを入れてゆっくり温める


<QuizAnswer>

**答え: 3**

チャイの基本的な作り方は、鍋に水を入れて沸騰させ、茶葉とスパイスを加えて煮出した後、牛乳を加えてさらに煮込む方法です。先に水で茶葉の成分をしっかり抽出してから牛乳を加えることで、紅茶の風味とスパイスの香りが調和した濃厚なチャイが作れます。

</QuizAnswer>

## 問10

ジャンピングが起こるために重要な条件として、最も適切なものはどれか。

1. ティーポットをできるだけ小さいものにする
2. 茶葉を通常の2倍の量にする
3. 沸騰直後の熱湯を勢いよく注ぐ
4. お湯の温度を80℃程度に下げて注ぐ


<QuizAnswer>

**答え: 3**

ジャンピングを起こすためには、汲みたての水を沸かした沸騰直後の熱湯（100℃）を勢いよく注ぐことが重要です。沸騰直後のお湯には空気が含まれており、この空気の泡が茶葉に付着して浮力を生み、対流とともに茶葉が上下に動くジャンピングが起こります。温度が低いとこの現象が起こりにくくなります。

</QuizAnswer>
