## 問1

ダージリン紅茶は、その優れた香りと味わいから特別な別名で呼ばれることがあります。その別名として正しいものはどれですか。

1. 紅茶の女王
2. 紅茶のシャンパン
3. 紅茶のブルゴーニュ
4. 紅茶の宝石


<QuizAnswer>

**答え: 2**

ダージリンはインド西ベンガル州北部のヒマラヤ山麓で栽培される紅茶で、繊細でフルーティーな香りと上品な渋みから「紅茶のシャンパン」と称されます。特にセカンドフラッシュ（夏摘み）の上質な茶葉にこの名が用いられます。なお「紅茶のブルゴーニュ」はキーマン紅茶の別名です。

</QuizAnswer>

## 問2

スリランカの紅茶は茶園の標高によって3つに分類されます。標高約1,200m以上の高地で栽培されるものを何と呼びますか。

1. ミディアムグロウンティー
2. プレミアムグロウンティー
3. ハイグロウンティー
4. トップグロウンティー


<QuizAnswer>

**答え: 3**

スリランカ（セイロン）の紅茶は、標高約1,200m（4,000フィート）以上を「ハイグロウンティー」、約600〜1,200mを「ミディアムグロウンティー」、約600m以下を「ローグロウンティー」と分類します。ハイグロウンティーは爽やかな香りと適度な渋みが特徴で、ウヴァ、ヌワラエリヤ、ディンブラなどの産地が代表的です。

</QuizAnswer>

## 問3

ダージリン紅茶のセカンドフラッシュ（夏摘み）に見られる、マスカットに似た甘く芳醇な香りを何と呼びますか。

1. オレンジフレーバー
2. スモーキーフレーバー
3. マスカテルフレーバー
4. フローラルフレーバー


<QuizAnswer>

**答え: 3**

ダージリンのセカンドフラッシュに特有の、マスカットのような甘くフルーティーな香りは「マスカテルフレーバー」と呼ばれます。これは上質なダージリン茶園の夏摘み茶葉に特徴的に現れる香りで、紅茶愛好家の間で非常に高く評価されています。

</QuizAnswer>

## 問4

世界三大銘茶に数えられる組み合わせとして正しいものはどれですか。

1. ダージリン・アッサム・キーマン
2. ダージリン・ウヴァ・キーマン
3. ダージリン・ウヴァ・ニルギリ
4. アッサム・ウヴァ・キーマン


<QuizAnswer>

**答え: 2**

世界三大銘茶は、インドの「ダージリン」、スリランカの「ウヴァ」、中国の「キーマン（祁門）」の3つです。それぞれの産地で育まれた独特の香りと風味が世界的に高い評価を受けています。アッサムやニルギリも有名な産地ですが、三大銘茶には含まれません。

</QuizAnswer>

## 問5

紅茶の茶葉のグレード（等級）は何を基準に分類されていますか。

1. 茶葉の品質や味の良さ
2. 茶葉の大きさや形状
3. 茶葉の産地と収穫時期
4. 茶葉の発酵度合い


<QuizAnswer>

**答え: 2**

紅茶のグレード（等級）は、茶葉の大きさや形状によって分類されるもので、品質の良し悪しを示すものではありません。仕上げ工程でふるいにかけて大きさ別に選別され、OP（オレンジペコー）やBOP（ブロークンオレンジペコー）などに分けられます。それぞれのグレードに適したいれ方や味わいの特徴があります。

</QuizAnswer>

## 問6

アッサム紅茶の産地として正しい説明はどれですか。

1. インド南部の高原地帯で栽培される
2. インド北東部のブラマプトラ河流域の平原で栽培される
3. スリランカ中央部の山岳地帯で栽培される
4. 中国安徽省の山間部で栽培される


<QuizAnswer>

**答え: 2**

アッサムはインド北東部のブラマプトラ河（ブラマプトラ川）の両岸に広がる広大な平原地帯で栽培されます。世界最大の紅茶産地であり、力強いコクと甘みのある味わいが特徴です。濃い水色でミルクティーに適しています。

</QuizAnswer>

## 問7

スリランカの七大紅茶産地のうち、ミディアムグロウン（中地産）に分類される代表的な産地はどれですか。

1. ウヴァ
2. ルフナ
3. ヌワラエリヤ
4. キャンディ


<QuizAnswer>

**答え: 4**

キャンディはスリランカのミディアムグロウン（標高約600〜1,200m）に分類される代表的な産地です。クセが少なくバランスの良い飲みやすい紅茶として知られています。ウヴァとヌワラエリヤはハイグロウン、ルフナはローグロウンに分類されます。

</QuizAnswer>

## 問8

中国の紅茶「キーマン（祁門）」の香りの特徴として最も適切なものはどれですか。

1. マスカットのような爽やかな果実香
2. 蘭やバラに似た花香とスモーキーフレーバー
3. 柑橘系のシトラスフレーバー
4. ミントのような清涼感のある香り


<QuizAnswer>

**答え: 2**

キーマン紅茶は中国安徽省祁門県で生産される紅茶で、蘭やバラに似た華やかな香りに加え、独特のスモーキーフレーバーが特徴です。渋みが少なくマイルドで甘みのある味わいを持ち、イギリスでは伝統的にアフタヌーンティーで愛飲されてきました。

</QuizAnswer>

## 問9

ダージリンのシーズンティーのうち、春に最初に摘まれる新茶を何と呼びますか。

1. オータムナル
2. ファーストフラッシュ
3. セカンドフラッシュ
4. モンスーンフラッシュ


<QuizAnswer>

**答え: 2**

ダージリンでは収穫時期によって紅茶が区別されます。春（3〜4月頃）に摘まれる最初の新茶を「ファーストフラッシュ」と呼び、フレッシュで爽やかな香りと若々しい渋みが特徴です。夏摘みは「セカンドフラッシュ」、秋摘みは「オータムナル」と呼ばれ、それぞれ異なる風味を楽しめます。

</QuizAnswer>

## 問10

紅茶の茶葉グレードで「BOP」とは何の略称ですか。

1. Best Orange Pekoe
2. Broken Orange Pekoe
3. Blended Orange Pekoe
4. Bold Orange Pekoe


<QuizAnswer>

**答え: 2**

BOPは「Broken Orange Pekoe（ブロークンオレンジペコー）」の略で、OP（オレンジペコー）の茶葉を細かく砕いたものを指します。茶葉のサイズは2〜3mm程度で、OPよりも抽出が早く、濃い水色と力強い味わいが得られます。セイロンティーなどで広く用いられるグレードです。

</QuizAnswer>
