## 問1

温泉分析書において、「泉温」とは何の温度を指すか。

1. 浴槽に注がれた時点での湯の温度
2. 源泉から地上に湧出または汲み上げられた時点での温度
3. 配管を通って浴槽に届いた直後の温度
4. 1日の平均的な湯船の温度


<QuizAnswer>

**答え: 2**

温泉分析書に記載されている「泉温」は、私たちが入浴する湯船の温度ではなく、源泉から地上に湧出または汲み上げられた時点での温度を示しています。そのため、実際の入浴時の温度とは異なる場合があります。加温・加水などの処理が行われる前の、源泉そのものの温度です。

</QuizAnswer>

## 問2

温泉分析書の陽イオン・陰イオンの欄に記載されている「ミリバル（mval）」とは何を表す単位か。

1. イオンの重さを表す単位
2. イオンの電気量を表す単位
3. イオンの濃度を表す単位
4. イオンの溶解度を表す単位


<QuizAnswer>

**答え: 2**

ミリバル（mval）はイオンの電気量を表す単位で、温泉分析の分野で使用されるものです。ミリグラム（mg）だけではイオンとしての電気素量がわからないため、イオンの欄にはミリバルという欄が設けられています。この単位を使うことで、陽イオンと陰イオンの電気的なバランスを確認することができます。

</QuizAnswer>

## 問3

温泉分析書において、陽イオン全体のミリバル合計値と陰イオン全体のミリバル合計値の関係として正しいものはどれか。

1. 陽イオンの合計値は陰イオンの合計値の約2倍になる
2. 両者の合計値は常に1,000mvalになる
3. 両者の合計値はほぼ同一になる
4. 陰イオンの合計値は陽イオンの合計値より常に大きい


<QuizAnswer>

**答え: 3**

温泉水は電気的に中性であるため、陽イオン全体が帯びているプラスの電気量と、陰イオン全体が帯びているマイナスの電気量は等しくなります。したがって、陽イオンと陰イオンのミリバル合計値はほぼ同一になります。この性質は分析結果の正確性を確認する指標としても用いられます。

</QuizAnswer>

## 問4

温泉分析書の成分表で、泉質名を決定する際に基本となるのはどの数値か。

1. ミリグラム（mg）の値
2. ミリモル（mmol）の値
3. ミリバル%（mval%）の値
4. 成分総計の値


<QuizAnswer>

**答え: 3**

泉質名を決定する際に基本となるのは「ミリバル%（mval%）」の値です。ミリバル%は陽イオン・陰イオンそれぞれの合計に対する各成分の比率を表しており、ミリバル%が最大のものがその温泉の主成分となります。さらに20ミリバル%以上の成分は副成分として泉質名に反映されます。

</QuizAnswer>

## 問5

温泉分析書の有効期限（再分析を受けるべき期間）は何年以内と定められているか。

1. 3年以内
2. 5年以内
3. 10年以内
4. 15年以内


<QuizAnswer>

**答え: 3**

温泉成分の分析を受けるべき期間は10年以内ごとと定められています。温泉の成分は時間の経過とともに変化する可能性があるため、10年以内ごとに再分析を行い、必要に応じて掲示内容を変更しなければなりません。この規定により、利用者に正確な温泉情報が提供されるようになっています。

</QuizAnswer>

## 問6

温泉分析書の成分表において、陽イオン・陰イオンの各欄はどのような項目で構成されているか。

1. 成分名・ミリグラム（mg）・モル（mol）・モル%
2. 成分名・ミリグラム（mg）・ミリバル（mval）・ミリバル%（mval%）
3. 成分名・グラム（g）・ミリバル（mval）・重量%
4. 成分名・ppm・ミリグラム（mg）・ミリバル（mval）


<QuizAnswer>

**答え: 2**

温泉分析書の陽イオン・陰イオンの欄は、「成分名」「ミリグラム（mg）」「ミリバル（mval）」「ミリバル%（mval%）」の4項目で構成されています。ミリグラムは成分の重量、ミリバルはイオンの電気量、ミリバル%は各イオングループ内での比率を示します。これらの情報を総合して泉質が判定されます。

</QuizAnswer>

## 問7

温泉分析書において、「成分総計」はどの項目の下に記載されているか。

1. 陽イオンの表の下
2. 陰イオンの表の下
3. 溶存ガス成分の表の下
4. 非解離成分の表の下


<QuizAnswer>

**答え: 3**

成分総計は溶存ガス成分の表の下に記載されています。成分総計とは、陽イオン・陰イオン・非解離成分・溶存ガス成分をすべて合算した値で、温泉中に実際に溶解している物質の総量を表します。この成分総計（溶存物質総量）は、療養泉の判定や泉質名の決定において重要な基準となります。

</QuizAnswer>

## 問8

療養泉の特殊成分として、総硫黄の基準値は温泉水1kg中何mg以上か。

1. 1mg以上
2. 2mg以上
3. 10mg以上
4. 20mg以上


<QuizAnswer>

**答え: 2**

療養泉の特殊成分における総硫黄の基準値は、温泉水1kgあたり2mg以上と定められています。総硫黄が2mg以上含まれていれば「硫黄泉」に分類されます。なお、総硫黄とは硫化水素イオン（HS⁻）、チオ硫酸イオン（S₂O₃²⁻）、遊離硫化水素（H₂S）の合計を指します。

</QuizAnswer>

## 問9

温泉分析書で泉質名に副成分として記載されるのは、ミリバル%がいくつ以上の成分か。

1. 5ミリバル%以上
2. 10ミリバル%以上
3. 20ミリバル%以上
4. 30ミリバル%以上


<QuizAnswer>

**答え: 3**

泉質名において、ミリバル%が最大の成分が主成分となり、さらに20ミリバル%以上の成分が副成分として泉質名に含まれます。例えば、陰イオンで塩化物イオンが最大（主成分）で、炭酸水素イオンが20ミリバル%以上（副成分）であれば、「含炭酸水素塩−塩化物泉」のように副成分も泉質名に反映されます。

</QuizAnswer>

## 問10

療養泉の特殊成分として、遊離二酸化炭素（遊離CO₂）の基準値は温泉水1kg中何mg以上か。

1. 250mg以上
2. 500mg以上
3. 1,000mg以上
4. 2,000mg以上


<QuizAnswer>

**答え: 3**

療養泉の特殊成分における遊離二酸化炭素（遊離CO₂）の基準値は、温泉水1kgあたり1,000mg以上と定められています。この基準を満たすと「二酸化炭素泉（炭酸泉）」に分類されます。二酸化炭素泉は入浴すると肌に細かい気泡が付着するのが特徴で、血行促進効果が高いとされています。

</QuizAnswer>
