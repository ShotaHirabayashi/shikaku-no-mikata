## 問1

温泉に入る前の「かけ湯」は、体のどこから始めるのが正しいマナーとされているか。

1. 頭から順に下へかける
2. 心臓に近い胸からかける
3. 心臓から遠い足元からかける
4. 両手から同時にかける


<QuizAnswer>

**答え: 3**

かけ湯は心臓から遠い部分から始めるのが正しいマナーです。足元（つま先）から始めて、ひざ、腰、手首、腕、肩と徐々に心臓に近づけていきます。これは入浴時の急激な血圧上昇を防ぎ、体をお湯の温度に慣らすための大切な手順です。

</QuizAnswer>

## 問2

かけ湯は一般的に何杯程度かけるのが望ましいとされているか。

1. 3杯程度
2. 5杯程度
3. 10杯程度
4. 20杯程度


<QuizAnswer>

**答え: 3**

かけ湯は10杯程度が目安とされています。具体的には、左右の足元に2杯ずつ（計4杯）、両腕に2杯ずつ（計4杯）、背中とお腹に1杯ずつ（計2杯）で合計10杯という数え方が温泉ソムリエの教えでも紹介されています。腰回りは特に念入りに行うとよいでしょう。

</QuizAnswer>

## 問3

温泉の浴槽に入る際、タオルの扱い方として正しいマナーはどれか。

1. タオルを浴槽のお湯で洗ってから使う
2. タオルを浴槽の縁に置いておく
3. タオルを浴槽に入れず、頭の上に乗せるか浴槽外に置く
4. タオルを浴槽の注ぎ口付近に置く


<QuizAnswer>

**答え: 3**

タオルを浴槽内に入れたり、浴槽のお湯でゆすいだりするのはマナー違反です。また、浴槽の注ぎ口（湯口）付近にタオルを置くことも禁止されています。タオルは頭の上に乗せるか、浴槽の外に置くのが正しいマナーです。頭にタオルを乗せることにはのぼせ防止の効果もあります。

</QuizAnswer>

## 問4

浴室から脱衣所に戻る際の正しいマナーはどれか。

1. そのまま脱衣所に入ってからタオルで拭く
2. 浴室内で体をしっかり絞ったタオルで拭いてから脱衣所に入る
3. 脱衣所の入口でバスタオルを取って拭く
4. 自然乾燥させてから脱衣所に入る


<QuizAnswer>

**答え: 2**

脱衣所に戻る前に、浴室内で絞ったタオルを使って体の水滴をしっかり拭き取るのが基本マナーです。濡れたまま脱衣所に入ると床が水浸しになり、他の利用者に不快な思いをさせてしまいます。これは温泉だけでなく、銭湯やスーパー銭湯でも共通のマナーです。

</QuizAnswer>

## 問5

「源泉かけ流し」の説明として正しいものはどれか。

1. 源泉を一度加熱してから浴槽に注ぐ方式
2. 源泉を浴槽に常に新しく注ぎ、あふれた湯は再利用しない方式
3. 源泉を循環ろ過して清潔に保つ方式
4. 源泉を水で薄めて適温にして提供する方式


<QuizAnswer>

**答え: 2**

源泉かけ流しとは、常に新しい源泉を浴槽に注ぎ入れ、あふれ出たお湯は再利用しない方式のことです。一方、循環式は排出された温泉をろ過・消毒して再利用する方式です。源泉かけ流しは温泉本来の成分や鮮度が保たれるため、温泉愛好家から高く評価されています。

</QuizAnswer>

## 問6

江戸時代に作られた「温泉番付」で、当時の最高位である東西の大関に選ばれた温泉の組み合わせとして正しいものはどれか。

1. 道後温泉と別府温泉
2. 草津温泉と有馬温泉
3. 下呂温泉と城崎温泉
4. 箱根温泉と登別温泉


<QuizAnswer>

**答え: 2**

江戸時代の寛政年間に作られた温泉番付では、効能の高さをもとにランク付けされ、当時の最高位である大関には草津温泉（東）と有馬温泉（西）が選ばれました。温泉番付は相撲の番付表を模して作られたもので、日本の温泉文化を象徴する歴史的な資料です。

</QuizAnswer>

## 問7

日本の温泉文化において「湯治（とうじ）」の伝統的な最低滞在期間とされていたのはどれか。

1. 3日間
2. 7日間
3. 21日間（三廻り）
4. 30日間


<QuizAnswer>

**答え: 3**

湯治とは温泉に一定期間滞在して病気の治療や健康回復を図ることです。伝統的に7日間を「一廻り」と数え、少なくとも「三廻り」（21日間）は滞在すべきとされていました。この考え方は中世から受け継がれており、温泉の効能を十分に得るための知恵として日本の温泉文化に根付いています。

</QuizAnswer>

## 問8

混雑している温泉の洗い場でのマナーとして、最も適切なものはどれか。

1. 使っていない洗い場にタオルを置いて場所を確保する
2. シャワーのお湯が周囲の人にかからないよう配慮する
3. 洗い場が空くまで浴槽内で待つ
4. 立ったままシャワーを使って素早く済ませる


<QuizAnswer>

**答え: 2**

混雑時の洗い場では、シャワーのお湯や泡が周囲の人にかからないよう配慮することが重要なマナーです。タオルや道具を置いての場所取りはマナー違反にあたります。また、立ったままシャワーを使うとお湯が飛び散りやすいため、座って使うのが基本です。

</QuizAnswer>

## 問9

温泉ソムリエの資格で学ぶ内容として、その主な2つの柱はどれか。

1. 温泉の経営知識と接客技術
2. 温泉の知識と正しい入浴法の技術
3. 温泉の掘削技術と泉質分析
4. 温泉旅館の料理知識と宿泊サービス


<QuizAnswer>

**答え: 2**

温泉ソムリエは温泉ソムリエ協会が認定する民間資格で、温泉についての「知識」と正しい入浴法という「技術」の2つを柱として学びます。温泉の泉質や効能を理解し、安全で効果的な入浴法を実践・伝達できることが温泉ソムリエの役割です。入浴事故を減らすことも重要な目的の一つです。

</QuizAnswer>

## 問10

日本の温泉・入浴文化に大きな影響を与えたとされる、8世紀に大陸から伝わった仏教の経典は何か。

1. 般若心経（はんにゃしんぎょう）
2. 法華経（ほけきょう）
3. 温室経（うんじつきょう）
4. 阿弥陀経（あみだきょう）


<QuizAnswer>

**答え: 3**

8世紀の中頃、仏教経典とともに大陸から伝わった「温室経（うんじつきょう）」は、「風呂に入ると功徳が得られる」という教えを説いたお経です。この経典の影響で寺院に浴堂（温室）が設けられるようになり、日本の入浴文化・温泉文化の発展に大きく寄与しました。

</QuizAnswer>
