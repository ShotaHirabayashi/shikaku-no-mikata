## 問1

温泉ソムリエ入浴五か条の「その一」では、入浴時の水分補給についてどのように述べられているか。

1. 入浴後にコップ1杯の水を飲むべし
2. 入浴前後に1杯ずつの水を飲むべし
3. 入浴中にこまめに水を飲むべし
4. 入浴の1時間前に500mlの水を飲むべし


<QuizAnswer>

**答え: 2**

温泉ソムリエ入浴五か条の「その一」は「入浴前後に1杯ずつの水を飲むべし」です。入浴すると発汗により血液粘度が高まり「ドロドロ血」の状態になるため、入浴後だけでなく入浴前にも水分補給をすることがポイントです。入浴前の水分補給によって脱水を予防し、安全に入浴することができます。

</QuizAnswer>

## 問2

温泉ソムリエ入浴五か条で推奨される「かけ湯」の正しい順序はどれか。

1. 頭から順に下半身へかける
2. 胸から順に手足へかける
3. 足先など心臓から遠い部位から順にかける
4. 左半身から右半身の順にかける


<QuizAnswer>

**答え: 3**

入浴五か条の「その二」では「入浴前には足先など心臓の遠くから順に十分なかけ湯をすべし」と述べられています。かけ湯には体の汚れを落とすマナーとしての意味だけでなく、温泉の泉質や温度に体を慣らすという重要な役割があります。心臓に近い部位にいきなり熱いお湯をかけると血圧が急変動する恐れがあるため、足先から徐々にかけていきます。

</QuizAnswer>

## 問3

温泉ソムリエが推奨する入浴の順序として正しいものはどれか。

1. 全身浴→半身浴→足浴
2. 半身浴→足浴→全身浴
3. 足浴→全身浴→半身浴
4. 足浴→半身浴→全身浴


<QuizAnswer>

**答え: 4**

温泉ソムリエ入浴五か条の「その三」では、足浴→半身浴→全身浴の順にゆっくり入浴することが推奨されています。体を徐々に温泉の温度に慣らしていくことで、血圧の急激な変動を防ぎ、体への負担を軽減できます。いきなり肩まで浸かる全身浴をすると、水圧や温度による体への負担が大きくなります。

</QuizAnswer>

## 問4

「分割浴」の説明として正しいものはどれか。

1. 1日のうち朝と夜の2回に分けて入浴する方法
2. 短時間の入浴と休憩を繰り返す方法
3. 複数の異なる泉質の温泉に順番に入る方法
4. 体の部位ごとに分けて順番に温める方法


<QuizAnswer>

**答え: 2**

分割浴とは、短時間の入浴と浴槽から出て休憩することを繰り返す入浴法です。入浴五か条の「その四」で「一気に長湯せず、分割浴をすべし」と推奨されています。例えば3分入浴して休憩を3回繰り返すと合計9分の入浴になりますが、一気に10分間入浴するよりも湯冷めしにくいという効果があります。

</QuizAnswer>

## 問5

分割浴が一気に長湯するよりも優れている点として、温泉ソムリエが特に強調しているのはどれか。

1. 温泉成分の吸収量が増える
2. 湯冷めしにくい
3. カロリー消費が多い
4. 肌への刺激が少ない


<QuizAnswer>

**答え: 2**

温泉ソムリエ入浴五か条では、分割浴の最大のメリットとして「湯冷めしにくい」ことを挙げています。例えば3分×3回の計9分間の分割浴は、一気に10分間入浴するよりも湯冷めしにくいとされています。心拍数を急上昇させることなく血流量を増やすことができるため、体に負担をかけずに効率的に体を温めることができます。

</QuizAnswer>

## 問6

温泉ソムリエ入浴五か条で推奨される「温冷交互浴」について、正しい説明はどれか。

1. 全身を熱い湯と冷たい水に交互に浸ける
2. 顔に温タオルと冷タオルを交互にあてる
3. ひざ下にお湯と水を交互にかける
4. 温かい内湯と冷たい露天風呂を交互に入る


<QuizAnswer>

**答え: 3**

入浴五か条の「その五」では「疲労回復には、ひざ下の温冷交互浴が有効なり」と述べられています。ひざ下にお湯を3分、水を1分かけることを3〜5回繰り返す方法です。心臓から遠い末端部分で行うことで安全に末梢血管を広げ、足裏の反射区に溜まった乳酸などの疲労物質が排泄されやすくなり、疲労回復に高い効果があります。

</QuizAnswer>

## 問7

温冷交互浴の時間配分として、温泉ソムリエが推奨するのはどれか。

1. お湯1分・水1分を5〜10回
2. お湯3分・水1分を3〜5回
3. お湯5分・水3分を2〜3回
4. お湯10分・水5分を1〜2回


<QuizAnswer>

**答え: 2**

温泉ソムリエが推奨する温冷交互浴は、お湯を3分・水を1分かけることを3〜5回繰り返す方法です。この配分により末梢血管が効果的に広がり、疲労物質の排泄が促進されます。また、温冷交互浴に慣れていない方は「温（お湯）」で始まり「温（お湯）」で終わらせることがポイントとされています。

</QuizAnswer>

## 問8

温泉から上がる際の「上がり湯」（真水のシャワーで体を流すこと）について、温泉ソムリエの考え方として最も適切なものはどれか。

1. 衛生のため、必ず真水で全身を洗い流すべきである
2. 温泉成分を肌に残すため、基本的に上がり湯はしない方がよい
3. 温泉の種類に関係なく、必ず温泉のお湯で上がり湯をすべきである
4. 上がり湯は頭だけにかけるのが正しい


<QuizAnswer>

**答え: 2**

温泉には皮膚の表面に膜を作り保湿する効果があるため、基本的に上がり湯（真水で洗い流すこと）はしない方がよいとされています。せっかくの温泉成分を洗い流してしまうのはもったいないためです。ただし、酸性泉や硫黄泉など肌への刺激が強い泉質の場合は、肌荒れ防止のために真水で軽く流した方がよい場合もあります。

</QuizAnswer>

## 問9

半身浴の正しいやり方として、お湯に浸かる深さの目安はどれか。

1. 腰（へそ）の高さまで
2. みぞおちの高さまで
3. 胸の高さまで
4. 肩の高さまで


<QuizAnswer>

**答え: 2**

半身浴は「みぞおち」の高さまでお湯に浸かる入浴法です。みぞおちまでの深さにすることで心臓への水圧による負担を軽減しながら、時間をかけてゆっくり入浴でき、全身の血行が促進されます。上半身が寒い場合は、肩からお湯をかけたり、乾いたタオルを肩にかけたりするとよいとされています。

</QuizAnswer>

## 問10

温泉入浴時に「湯船から出るタイミング」の目安として、温泉ソムリエが推奨するのはどれか。

1. 入浴開始から正確に15分が経過したとき
2. 額にうっすら汗をかいたとき
3. 指先がふやけてしわしわになったとき
4. 心拍数が普段の2倍になったとき


<QuizAnswer>

**答え: 2**

温泉ソムリエが推奨する湯船から出るタイミングの目安は「額にうっすら汗をかいたとき」です。これは体が十分に温まったサインであり、それ以上長く浸かると「のぼせ」や体への過度な負担につながる恐れがあります。時間で測るよりも、自分の体の反応を見て判断することが安全で効果的な入浴のポイントです。

</QuizAnswer>
