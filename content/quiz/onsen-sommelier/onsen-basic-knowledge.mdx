## 問1

日本において「温泉」を法的に定義している法律はどれか。

1. 鉱泉法
2. 温泉法
3. 地下水利用法
4. 公衆浴場法


<QuizAnswer>

**答え: 2**

日本における温泉の定義は「温泉法」（昭和23年制定）によって定められています。温泉法では、地中からゆう出する温水、鉱水及び水蒸気その他のガスで、一定の温度または物質を有するものを「温泉」と定義しています。公衆浴場法は浴場の衛生基準等を定める別の法律です。

</QuizAnswer>

## 問2

温泉法において、泉源での温度が何℃以上であれば、成分に関わらず「温泉」に該当するか。

1. 20℃
2. 30℃
3. 25℃
4. 42℃


<QuizAnswer>

**答え: 3**

温泉法では、泉源における水温が摂氏25℃以上であれば、含有成分に関係なく「温泉」と定義されます。25℃未満であっても、温泉法の別表に定められた19種類の物質のうち、いずれか1つ以上が規定量含まれていれば温泉に該当します。

</QuizAnswer>

## 問3

温泉法の温度基準（25℃以上）を満たさない場合でも温泉に該当するためには、別表に定められた物質を規定量以上含む必要がある。この物質は全部で何種類か。

1. 10種類
2. 15種類
3. 19種類
4. 25種類


<QuizAnswer>

**答え: 3**

温泉法の別表には19種類の物質（リチウムイオン、水素イオン、遊離炭酸、ラドンなど）と、それぞれの規定量が定められています。泉源の温度が25℃未満であっても、これら19種類のうちいずれか1つ以上を規定量以上含んでいれば、温泉法上の「温泉」に該当します。

</QuizAnswer>

## 問4

「療養泉」の定義を定めているのはどれか。

1. 温泉法
2. 鉱泉分析法指針
3. 公衆浴場法
4. 薬機法


<QuizAnswer>

**答え: 2**

療養泉は、環境省が定めた「鉱泉分析法指針」の中で定義されています。温泉法が「温泉」を広く定義するのに対し、鉱泉分析法指針では温泉のうち「特に治療の目的に供し得るもの」を療養泉として区別しています。泉質名がつくのは療養泉のみであり、すべての温泉に泉質名がつくわけではありません。

</QuizAnswer>

## 問5

鉱泉分析法指針による泉温の分類で、「低温泉」に該当する温度範囲はどれか。

1. 25℃未満
2. 25℃以上34℃未満
3. 34℃以上42℃未満
4. 42℃以上


<QuizAnswer>

**答え: 2**

鉱泉分析法指針では、泉温により4段階に分類されます。冷鉱泉（25℃未満）、低温泉（25℃以上34℃未満）、温泉（34℃以上42℃未満）、高温泉（42℃以上）です。なお、ここでいう「温泉」は泉温分類上の名称であり、温泉法の「温泉」の定義とは異なります。

</QuizAnswer>

## 問6

鉱泉の液性（pH値）による分類で、「アルカリ性」に該当するのはpHがいくつ以上の場合か。

1. 7.0以上
2. 7.5以上
3. 8.0以上
4. 8.5以上


<QuizAnswer>

**答え: 4**

鉱泉分析法指針では、pH値により5段階に分類されます。酸性（pH3未満）、弱酸性（pH3以上6未満）、中性（pH6以上7.5未満）、弱アルカリ性（pH7.5以上8.5未満）、アルカリ性（pH8.5以上）です。pH7.0が中性というイメージがありますが、温泉の分類ではpH6〜7.5が「中性」とされている点に注意が必要です。

</QuizAnswer>

## 問7

鉱泉の浸透圧による分類で、溶存物質総量が1kg中に8g未満の温泉は何と呼ばれるか。

1. 高張性
2. 等張性
3. 低張性
4. 無張性


<QuizAnswer>

**答え: 3**

浸透圧による分類では、溶存物質総量（1kgあたり）が8g未満を「低張性」、8g以上10g未満を「等張性」、10g以上を「高張性」と分類します。日本の温泉の大多数は低張性で、体の浸透圧より低いため水分が体に吸収されやすく、指先がしわしわになりやすい特徴があります。

</QuizAnswer>

## 問8

火山の熱によらず、地下深くなるほど温度が上昇する現象を利用した温泉を何と呼ぶか。

1. マグマ性温泉
2. 非火山性温泉（深層地下水型）
3. 化石温泉
4. 間欠泉


<QuizAnswer>

**答え: 2**

地下では深度が深くなるほど地温が上昇し、一般的に100mごとに約2〜3℃上昇します（地温勾配）。この仕組みで温められた地下水が温泉となるものを「非火山性温泉（深層地下水型）」と呼びます。火山性温泉がマグマの熱を直接利用するのに対し、深層地下水型は火山がない地域でも温泉が湧く理由を説明するものです。

</QuizAnswer>

## 問9

温泉の湧出形態のうち、自然の圧力で地表に湧き出す温泉を何と呼ぶか。

1. 動力揚湯泉
2. 掘削泉
3. 自噴泉
4. 圧入泉


<QuizAnswer>

**答え: 3**

地下の圧力によって温泉水が自然に地表まで湧き出すものを「自噴泉」と呼びます。断層や破砕帯などの地下の割れ目が温泉水の通り道となり、湧出します。一方、ボーリングで掘削し、ポンプを使って汲み上げる方式は「動力揚湯」と呼ばれます。現在の日本の温泉は、掘削・動力揚湯によるものが多数を占めています。

</QuizAnswer>

## 問10

温泉法で定義する「温泉」から除外されているガスはどれか。

1. 硫化水素を主成分とする天然ガス
2. 炭酸ガスを主成分とする天然ガス
3. 炭化水素を主成分とする天然ガス
4. ラドンを主成分とする天然ガス


<QuizAnswer>

**答え: 3**

温泉法では、温泉を「地中からゆう出する温水、鉱水及び水蒸気その他のガス」と定義していますが、「炭化水素を主成分とする天然ガス」は明確に除外されています。これは天然ガス（メタンなど）の採掘と温泉利用を区別するためです。硫化水素や炭酸ガスなど他のガスは除外対象ではありません。

</QuizAnswer>
