## 問1

PCCSとは何の略称か。

1. Practical Color Coordinate System
2. Primary Color Classification System
3. Professional Color Control System
4. Pure Color Combination System

<details>
<summary>答えと解説を見る</summary>

**答え: 1**

PCCSはPractical Color Coordinate System（日本色研配色体系）の略称です。日本色彩研究所が開発した配色体系で、色相とトーン（色調）の2つの軸で色を体系的に分類しています。

</details>

## 問2

PCCSの色相環は何色相で構成されているか。

1. 10色相
2. 12色相
3. 20色相
4. 24色相

<details>
<summary>答えと解説を見る</summary>

**答え: 4**

PCCSの色相環は24色相で構成されています。赤を1番として時計回りに番号が振られ、1:R（赤）から24:RP（赤紫）まで24色が等間隔に配置されています。

</details>

## 問3

PCCSのトーン分類で「v（ビビッド）」はどのような色調を表すか。

1. 明るく薄い色
2. 最も鮮やかな色
3. 暗くくすんだ色
4. 灰色がかった色

<details>
<summary>答えと解説を見る</summary>

**答え: 2**

v（ビビッド）は各色相における最も彩度が高く鮮やかな色調（純色）を表すトーンです。PCCSではv, b, s, dp, lt, sf, d, dk, p, ltg, g, dgの12種のトーンに分類されています。

</details>

## 問4

PCCSのトーン「p（ペール）」はどのような色調か。

1. 暗く深い色
2. 鈍い灰色がかった色
3. 明るく淡い色
4. 鮮やかで強い色

<details>
<summary>答えと解説を見る</summary>

**答え: 3**

p（ペール）は明度が高く彩度が低い、明るく淡い色調です。パステルカラーに相当し、やさしく柔らかい印象を与えます。

</details>

## 問5

マンセル表色系で色を表記する順序として正しいものはどれか。

1. 明度 色相/彩度
2. 色相 明度/彩度
3. 彩度 色相/明度
4. 色相 彩度/明度

<details>
<summary>答えと解説を見る</summary>

**答え: 2**

マンセル表色系では「色相 明度/彩度」（HV/C）の順で表記します。例えば「5R 4/14」は、色相5R、明度4、彩度14を表します。

</details>

## 問6

マンセル表色系の色相は何色相で構成されているか。

1. 5色相
2. 10色相
3. 20色相
4. 100色相

<details>
<summary>答えと解説を見る</summary>

**答え: 2**

マンセル表色系の基本色相はR（赤）・Y（黄）・G（緑）・B（青）・P（紫）の5色で、その中間にYR・GY・BG・PB・RPの5色を加え、合計10色相が基本となります。各色相をさらに10分割して100色相とすることもあります。

</details>

## 問7

マンセル表色系で無彩色を表す記号はどれか。

1. W
2. G
3. N
4. A

<details>
<summary>答えと解説を見る</summary>

**答え: 3**

マンセル表色系では無彩色をN（Neutral）で表記します。N0が理想的な黒、N10が理想的な白で、その間の数値で灰色の明度を示します。例えばN5は中明度の灰色です。

</details>

## 問8

PCCSにおいて、暗清色に分類されるトーンはどれか。

1. lt（ライト）
2. dp（ディープ）
3. sf（ソフト）
4. p（ペール）

<details>
<summary>答えと解説を見る</summary>

**答え: 2**

dp（ディープ）は純色に黒を加えた暗清色のトーンです。深みのある色調で、落ち着いた印象を与えます。暗清色にはdp（ディープ）とdk（ダーク）が含まれます。lt（ライト）とp（ペール）は明清色、sf（ソフト）は中間色です。

</details>

## 問9

マンセル表色系において、彩度の最大値は色相によって異なる。この説明は正しいか。

<details>
<summary>答えと解説を見る</summary>

**答え: ○（正しい）**

マンセル表色系の彩度（クロマ）は、色相や明度によって最大値が異なります。例えば赤（5R）は彩度14程度まで存在しますが、青緑などはそれより低い値が上限です。これはマンセル表色系が知覚的な等歩度を重視した結果です。

</details>

## 問10

PCCSのトーン記号「ltg」が表す色調の説明として正しいものはどれか。

1. 明るい灰みの色
2. 暗い灰みの色
3. 鮮やかな色
4. 深い色

<details>
<summary>答えと解説を見る</summary>

**答え: 1**

ltg（ライトグレイッシュ）は明るい灰みの色調を表します。明度がやや高く、灰色を含んだ落ち着いた印象の色です。グレイッシュ系のトーンにはltg（明るい灰み）、g（灰み）、dg（暗い灰み）があります。

</details>
