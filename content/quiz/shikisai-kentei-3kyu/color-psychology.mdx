## 問1

赤・橙・黄など、温かみを感じさせる色を何というか。

1. 寒色
2. 暖色
3. 中性色
4. 進出色


<QuizAnswer>

**答え: 2**

赤・橙・黄系の色は暖かさや熱を連想させるため暖色と呼ばれます。反対に、青系の色は冷たさを感じさせるため寒色と呼ばれます。緑や紫はどちらにも属さない中性色です。

</QuizAnswer>

## 問2

手前に飛び出して見える色のことを何というか。

1. 後退色
2. 膨張色
3. 進出色
4. 収縮色


<QuizAnswer>

**答え: 3**

暖色系で明度・彩度が高い色は、実際の位置より手前に飛び出して見えるため進出色と呼ばれます。反対に、寒色系で明度が低い色は奥に引っ込んで見えるため後退色と呼ばれます。

</QuizAnswer>

## 問3

周囲の色の影響で、実際とは異なる色に見える現象を総称して何というか。

1. 色の恒常性
2. 色の対比
3. 色の残像
4. 色の面積効果


<QuizAnswer>

**答え: 2**

色の対比とは、隣り合う色同士の影響で、実際とは異なる色に知覚される現象の総称です。色相対比、明度対比、彩度対比、補色対比などの種類があります。

</QuizAnswer>

## 問4

灰色が明るい背景の上では暗く、暗い背景の上では明るく見える現象を何というか。

1. 色相対比
2. 彩度対比
3. 明度対比
4. 補色対比


<QuizAnswer>

**答え: 3**

明度対比は、背景色の明度の影響を受けて、同じ色でも明るさが異なって見える現象です。明るい背景上では色がより暗く、暗い背景上ではより明るく知覚されます。

</QuizAnswer>

## 問5

赤い色をしばらく見た後に白い面を見ると、青緑色が見える現象を何というか。

1. 色の同化
2. 色の対比
3. 継時対比（残像）
4. 面積効果


<QuizAnswer>

**答え: 3**

特定の色をしばらく注視した後に白い面などを見ると、元の色の補色が見える現象を継時対比（残像）といいます。赤の補色である青緑が見えるのはこの現象です。陰性残像とも呼ばれます。

</QuizAnswer>

## 問6

実際には同じ大きさでも、明るい色の方が暗い色より大きく見える現象を何というか。

1. 色の同化
2. 進出・後退
3. 膨張・収縮
4. 色の恒常性


<QuizAnswer>

**答え: 3**

膨張色・収縮色の効果により、明るい色（高明度色）は実際より大きく膨らんで見え（膨張色）、暗い色（低明度色）は小さく縮んで見えます（収縮色）。暖色は膨張して見え、寒色は収縮して見える傾向があります。

</QuizAnswer>

## 問7

色の持つ心理的効果の説明として誤っているものはどれか。

1. 青い部屋は赤い部屋より涼しく感じる
2. 暖色系は食欲を増進させる効果がある
3. 黒い物体は白い物体より軽く感じる
4. 赤は注意を引く効果があり信号や標識に使われる


<QuizAnswer>

**答え: 3**

黒い物体は白い物体より重く感じます（色の重量感）。明度が低い色ほど重く感じ、明度が高い色ほど軽く感じるのが正しい関係です。引っ越し業者の段ボールに明るい色が使われるのもこの効果を利用しています。

</QuizAnswer>

## 問8

色の同化現象の説明として正しいものはどれか。

1. 隣り合う色同士が互いに反対方向に強調されて見える
2. 細い線や小さな点のパターンで、地色が模様の色に近づいて見える
3. 大きな面積の色が小さな面積より鮮やかに見える
4. 時間の経過とともに色の見え方が変化する


<QuizAnswer>

**答え: 2**

色の同化現象は、細い線や小さい模様のパターンにおいて、地色が模様の色に引き寄せられて見える現象です。対比が色の差を強調するのに対し、同化は色の差を縮小させる効果があります。果物のネットなどに利用されています。

</QuizAnswer>

## 問9

次のうち、寒色に分類される色はどれか。

1. 黄
2. 橙
3. 緑
4. 青


<QuizAnswer>

**答え: 4**

寒色は冷たさや涼しさを感じさせる色で、青を中心とした色相が該当します。黄と橙は暖色、緑は中性色（暖色にも寒色にも属さない）に分類されます。

</QuizAnswer>

## 問10

誘目性（目を引きやすさ）が最も高い色として正しいものはどれか。

1. 低彩度の寒色
2. 高彩度の暖色
3. 無彩色の灰色
4. 低明度の中性色


<QuizAnswer>

**答え: 2**

誘目性は注意を向けていない状態でも目を引く性質で、高彩度の暖色（特に赤や橙）が最も誘目性が高くなります。信号機の赤やセールの看板に暖色系が多用されるのはこの特性を活用しています。

</QuizAnswer>
