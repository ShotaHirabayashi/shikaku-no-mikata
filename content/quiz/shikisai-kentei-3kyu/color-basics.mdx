## 問1

人間の目に見える光の波長範囲（可視光線）は、およそ何nmから何nmか。

1. 200nm〜500nm
2. 380nm〜780nm
3. 400nm〜900nm
4. 500nm〜1000nm

### 答え

**2**

### 解説

可視光線の波長範囲はおよそ380nm〜780nmです。これより短い波長は紫外線、長い波長は赤外線と呼ばれ、人間の目では知覚できません。

## 問2

光の三原色（加法混色の三原色）の正しい組み合わせはどれか。

1. 赤・黄・青
2. 赤・緑・青
3. シアン・マゼンタ・イエロー
4. 赤・緑・黄

### 答え

**2**

### 解説

光の三原色は赤（R）・緑（G）・青（B）です。この3色を混ぜ合わせると白色光になります（加法混色）。選択肢3のシアン・マゼンタ・イエローは色料の三原色（減法混色の三原色）です。

## 問3

色料の三原色（減法混色の三原色）をすべて混ぜ合わせると、理論上何色になるか。

1. 白
2. 灰色
3. 黒
4. 茶色

### 答え

**3**

### 解説

色料の三原色（シアン・マゼンタ・イエロー）をすべて混ぜ合わせると、理論上は黒になります。これを減法混色といい、混ぜるほど暗くなる特徴があります。光の三原色の加法混色（混ぜると白）とは逆の性質です。

## 問4

太陽光をプリズムに通すと、光が分かれてスペクトルが見える現象を何というか。

1. 屈折
2. 反射
3. 分光
4. 散乱

### 答え

**3**

### 解説

太陽光（白色光）をプリズムに通すと波長ごとに屈折率が異なるため、赤から紫までの連続したスペクトルに分かれます。この現象を分光といいます。

## 問5

物体が赤く見えるのは、その物体がどのような性質を持っているからか。

1. 赤い光を吸収し、他の光を反射している
2. 赤い光を反射し、他の光を吸収している
3. すべての光を均等に反射している
4. 赤い光を自ら発している

### 答え

**2**

### 解説

物体の色は、物体が反射した光の色によって決まります。赤い物体は赤い波長の光を反射し、それ以外の波長の光を吸収しているため赤く見えます。

## 問6

彩度を持たない色のことを何というか。

1. 有彩色
2. 無彩色
3. 純色
4. 中間色

### 答え

**2**

### 解説

白・灰色・黒のように彩度（色みの鮮やかさ）を持たない色を無彩色といいます。無彩色は明度の違いだけで表現されます。対して、少しでも色みを持つ色は有彩色と呼ばれます。

## 問7

加法混色の説明として正しいものはどれか。

1. 混色するほど明るくなる
2. 混色するほど暗くなる
3. 混色しても明るさは変わらない
4. 色相だけが変化する

### 答え

**1**

### 解説

加法混色は光の混色であり、混ぜ合わせるほど明るくなります。赤・緑・青の三原色をすべて混ぜると白色光になります。テレビやスマートフォンのディスプレイは加法混色の原理を利用しています。

## 問8

次のうち、有彩色はどれか。

1. 白
2. 灰色
3. 黒
4. ベージュ

### 答え

**4**

### 解説

ベージュは黄みを帯びた薄い色であり、色みを持つため有彩色です。白・灰色・黒は色みを持たない無彩色です。

## 問9

虹に見える色の順番として正しいものはどれか（外側から内側）。

1. 赤・橙・黄・緑・青・藍・紫
2. 紫・藍・青・緑・黄・橙・赤
3. 赤・黄・橙・緑・藍・青・紫
4. 紫・青・藍・緑・黄・赤・橙

### 答え

**1**

### 解説

虹は外側から赤・橙・黄・緑・青・藍・紫の順に並んでいます。これは波長の長い赤から短い紫へと並んだもので、可視光線のスペクトルと同じ順序です。

## 問10

テレビやパソコンのモニターが色を表示する仕組みとして正しいものはどれか。

1. 減法混色
2. 加法混色
3. 併置加法混色
4. 中間混色

### 答え

**2**

### 解説

テレビやパソコンのモニターはRGB（赤・緑・青）の光の三原色を使った加法混色で色を表現しています。各ピクセルがRGBの発光量を調整することで、さまざまな色を表示します。
