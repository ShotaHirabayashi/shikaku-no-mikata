## 問1

猫が「真性肉食動物（obligate carnivore）」と呼ばれる理由として、最も適切なものはどれか。

1. 猫は犬よりも多くの歯を持ち、肉を噛み切る能力が高いため
2. 猫は体内でタウリンやアラキドン酸を十分に合成できず、動物性食品からの摂取が不可欠であるため
3. 猫は犬よりも消化管が長く、肉の消化吸収効率が高いため
4. 猫は植物性食品を一切消化できないため


<QuizAnswer>

**答え: 2**

猫は真性肉食動物であり、タウリン、アラキドン酸、プレフォームドビタミンAなどを体内で十分に合成できないため、動物性食品から摂取する必要があります。犬はこれらの栄養素を体内で合成できるため、雑食寄りの食性に分類されます。なお、猫は植物性食品を「一切消化できない」わけではなく、消化能力が犬より低いという違いです。

</QuizAnswer>

## 問2

犬の永久歯の本数として正しいものはどれか。

1. 28本
2. 30本
3. 36本
4. 42本


<QuizAnswer>

**答え: 4**

犬の永久歯は42本（上顎20本・下顎22本）です。猫の永久歯は30本で、犬より少なくなっています。これは猫の方が後臼歯の数が少なく、食物をすりつぶす機能よりも引き裂く機能に特化しているためです。なお、犬の乳歯は28本、猫の乳歯は26本です。

</QuizAnswer>

## 問3

犬と猫の正常な体温の範囲として、最も適切なものはどれか。

1. 36.0〜37.0℃
2. 37.0〜38.0℃
3. 38.0〜39.0℃
4. 39.5〜40.5℃


<QuizAnswer>

**答え: 3**

犬も猫も正常体温は約38.0〜39.0℃で、人間（約36.5℃前後）よりも高めです。39.5℃以上になると発熱が疑われ、37.0℃以下では低体温として注意が必要です。日常的に体温を把握しておくことで、体調変化の早期発見につながります。

</QuizAnswer>

## 問4

猫の食事においてタウリンが欠乏した場合に起こりうる疾患として、正しいものはどれか。

1. 甲状腺機能亢進症と腎不全
2. 拡張型心筋症と網膜変性症
3. 糖尿病と膵炎
4. 変形性関節症と椎間板ヘルニア


<QuizAnswer>

**答え: 2**

タウリンが欠乏すると、猫では拡張型心筋症（心筋の収縮力低下）や網膜中心変性症（進行すると失明）が起こる可能性があります。猫は体内でタウリンを十分に合成できないため、食事からの摂取が必須です。これが犬用フードを猫に与えてはいけない大きな理由の一つでもあります。

</QuizAnswer>

## 問5

犬のコアワクチンに含まれる感染症の組み合わせとして、正しいものはどれか。

1. ジステンパー、パルボウイルス感染症、犬伝染性肝炎
2. ジステンパー、レプトスピラ症、犬コロナウイルス感染症
3. パルボウイルス感染症、ケンネルコフ、レプトスピラ症
4. 犬伝染性肝炎、犬コロナウイルス感染症、ケンネルコフ


<QuizAnswer>

**答え: 1**

犬のコアワクチンは、犬ジステンパー、犬パルボウイルス感染症、犬伝染性肝炎（犬アデノウイルス1型）の3種です。レプトスピラ症や犬コロナウイルス感染症、パラインフルエンザ（ケンネルコフ関連）はノンコアワクチンに分類され、生活環境やリスクに応じて接種を検討します。

</QuizAnswer>

## 問6

犬猫に寄生するノミに関する記述として、誤っているものはどれか。

1. 犬にも猫にも最も多く寄生するのはネコノミである
2. ノミは瓜実条虫の中間宿主となる
3. ノミの寄生はアレルギー性皮膚炎の原因となることがある
4. ノミは宿主特異性が高く、ネコノミは犬には寄生しない


<QuizAnswer>

**答え: 4**

ネコノミは宿主特異性が低く、猫だけでなく犬や人にも寄生・吸血します。実際に犬に寄生するノミの多くもネコノミです。ノミは瓜実条虫の中間宿主となり、またノミの唾液に対するアレルギー反応によるアレルギー性皮膚炎の原因にもなります。

</QuizAnswer>

## 問7

ズーノーシス（人獣共通感染症）であるトキソプラズマ症に関する記述として、正しいものはどれか。

1. 犬が終宿主であり、犬の糞便中にオーシストが排出される
2. 猫が終宿主であり、特に妊婦は胎児への影響に注意が必要である
3. 感染経路は主にダニの咬傷であり、ペットとの接触では感染しない
4. ワクチンが実用化されており、猫への定期接種で予防できる


<QuizAnswer>

**答え: 2**

トキソプラズマ症の終宿主は猫であり、感染した猫の糞便中にオーシスト（虫卵のような構造）が排出されます。妊婦が初感染すると胎児に先天性トキソプラズマ症を引き起こす恐れがあるため、特に注意が必要です。予防には猫のトイレの早めの清掃や手洗いの徹底が重要で、現在トキソプラズマに対する実用化されたワクチンはありません。

</QuizAnswer>

## 問8

犬と猫の感覚器官に関する記述として、正しいものはどれか。

1. 犬も猫も人間と同じ3色型の色覚を持っている
2. 犬も猫もタペタム（輝板）を持ち、暗所での視力に優れている
3. 犬の聴覚は猫よりも高周波の音を聞き取ることができる
4. 猫にはヤコブソン器官（鋤鼻器）がなく、フレーメン反応を示さない


<QuizAnswer>

**答え: 2**

犬も猫も眼の網膜の裏にタペタム（輝板）という反射層を持っており、わずかな光を効率よく利用できるため、暗所での視力に優れています。犬と猫の色覚は2色型（青・黄系）で、人間のような3色型ではありません。また、猫の方が犬より高周波の音を聞き取ることができ、猫にもヤコブソン器官がありフレーメン反応を示します。

</QuizAnswer>

## 問9

猫の栄養管理に関する記述として、正しいものはどれか。

1. 猫はβカロテンをビタミンAに効率よく変換できるため、野菜で十分にビタミンAを摂取できる
2. 猫は犬よりもタンパク質の要求量が低いため、低タンパク質フードが適している
3. 猫はリノール酸からアラキドン酸を体内で合成できないため、動物性脂肪からの摂取が必要である
4. 猫は犬と同様にトリプトファンからナイアシンを十分に合成できる


<QuizAnswer>

**答え: 3**

猫はリノール酸からアラキドン酸への変換酵素（Δ6-デサチュラーゼ）の活性が低く、アラキドン酸を食事から直接摂取する必要があります。また、猫はβカロテンからビタミンAへの変換もできず、ナイアシンの合成能力も低いため、これらも動物性食品から摂取が必要です。タンパク質の要求量は犬より高く、体重あたり約1.5〜2倍必要とされます。

</QuizAnswer>

## 問10

高齢猫に特に多くみられる疾患として、最も適切なものはどれか。

1. 股関節形成不全
2. 慢性腎臓病（CKD）
3. 犬糸状虫症（フィラリア症）
4. ジステンパー


<QuizAnswer>

**答え: 2**

慢性腎臓病（CKD）は高齢猫に非常に多くみられる疾患で、多飲多尿が初期兆候として現れます。定期的な血液検査（BUN・クレアチニン・SDMA）や尿検査による早期発見が重要です。股関節形成不全は主に大型犬で問題となり、ジステンパーは犬の感染症、フィラリア症も主に犬で問題となる疾患です。

</QuizAnswer>
