## 問1

動物愛護管理法（動物の愛護及び管理に関する法律）の目的として、最終的に実現を図るとされているのは次のうちどれか。

1. 動物の権利の確立
2. 人と動物の共生する社会の実現
3. 野生動物の保護と繁殖
4. ペット産業の健全な発展


<QuizAnswer>

**答え: 2**

動物愛護管理法第1条では、動物の愛護に関する事項と管理に関する事項を定めることで、「人と動物の共生する社会の実現を図ること」を目的としています。動物の権利の確立や産業振興が目的ではなく、人と動物が共に暮らせる社会づくりが法の究極的な目標です。

</QuizAnswer>

## 問2

第一種動物取扱業の登録の有効期間として正しいものはどれか。

1. 3年
2. 5年
3. 7年
4. 10年


<QuizAnswer>

**答え: 2**

第一種動物取扱業の登録の有効期間は5年で、引き続き営業を行う場合は更新の手続きが必要です。営利目的で動物の取扱いを業として行う者は、都道府県知事または政令指定都市の長に登録しなければなりません。

</QuizAnswer>

## 問3

2019年の動物愛護管理法改正により、動物をみだりに殺し、又は傷つけた者に対する罰則はどのように変更されたか。

1. 1年以下の懲役又は100万円以下の罰金
2. 3年以下の懲役又は300万円以下の罰金
3. 5年以下の懲役又は500万円以下の罰金
4. 7年以下の懲役又は700万円以下の罰金


<QuizAnswer>

**答え: 3**

2019年の法改正により、愛護動物の殺傷に対する罰則は「5年以下の懲役又は500万円以下の罰金」に厳罰化されました。改正前は「2年以下の懲役又は200万円以下の罰金」でした。なお、虐待・遺棄については「1年以下の懲役又は100万円以下の罰金」となっています。

</QuizAnswer>

## 問4

狂犬病予防法に基づく犬の登録について、正しいものはどれか。

1. 犬を取得した日から14日以内に登録し、毎年更新する必要がある
2. 犬を取得した日から30日以内に登録し、登録は生涯1回有効である
3. 犬を取得した日から60日以内に登録し、3年ごとに更新する必要がある
4. 犬を取得した日から90日以内に登録し、登録は生涯1回有効である


<QuizAnswer>

**答え: 2**

狂犬病予防法では、犬の所有者は犬を取得した日から30日以内（生後90日以内の犬は生後90日を経過した日から30日以内）に市区町村長に登録を申請しなければなりません。登録は犬の一生を通じて1回で有効であり、毎年の更新は不要です。ただし、狂犬病予防注射は毎年1回受けさせる義務があります。

</QuizAnswer>

## 問5

ペットフード安全法（愛がん動物用飼料の安全性の確保に関する法律）の対象となる動物はどれか。

1. 犬、猫、うさぎ
2. 犬及び猫のみ
3. すべての愛玩動物
4. 犬、猫、鳥類


<QuizAnswer>

**答え: 2**

ペットフード安全法の対象動物は犬及び猫のみです。この法律は2007年に米国で中国産メラミン混入ペットフードによる健康被害が発生したことを契機に制定され、2009年6月に施行されました。製造業者・輸入業者には届出義務があり、有害物質を含むペットフードの製造・輸入・販売が禁止されています。

</QuizAnswer>

## 問6

2022年6月に施行されたマイクロチップ装着の義務化について、正しいものはどれか。

1. すべての犬猫の飼い主にマイクロチップ装着が義務づけられた
2. 犬猫等販売業者には義務、一般の飼い主には努力義務とされた
3. 犬のみが対象で、猫は努力義務とされた
4. マイクロチップ装着は努力義務にとどまり、義務化はされていない


<QuizAnswer>

**答え: 2**

2022年6月1日施行の改正動物愛護管理法により、犬猫等販売業者（ブリーダー・ペットショップ等）にはマイクロチップの装着が義務づけられました。一方、すでに犬猫を飼っている一般の飼い主については努力義務とされています。ただし、マイクロチップが装着された犬猫を購入・譲り受けた場合は、30日以内に変更届出を行う義務があります。

</QuizAnswer>

## 問7

鳥獣保護管理法における野生鳥獣の取扱いについて、正しいものはどれか。

1. 野生鳥獣は自由に捕獲してよいが、販売には許可が必要である
2. 野生鳥獣の捕獲は原則禁止で、狩猟や許可捕獲などの例外がある
3. 野生鳥獣は都道府県への届出のみで捕獲できる
4. 野生鳥獣の捕獲禁止は哺乳類のみに適用され、鳥類は対象外である


<QuizAnswer>

**答え: 2**

鳥獣保護管理法では、野生鳥獣の捕獲は原則として禁止されています。例外として、狩猟免許を取得し狩猟者登録を行った者が狩猟期間中に狩猟鳥獣を捕獲する場合や、学術研究・有害鳥獣駆除等の目的で都道府県知事等の許可を得た場合に限り捕獲が認められます。鳥類・哺乳類ともに保護の対象です。

</QuizAnswer>

## 問8

第一種動物取扱業の業種に含まれないものはどれか。

1. 競りあっせん業
2. 譲受飼養業
3. 診療業
4. 貸出し業


<QuizAnswer>

**答え: 3**

第一種動物取扱業の業種は、販売、保管、貸出し、訓練、展示、競りあっせん、譲受飼養の7種類です。動物の診療は獣医師法に基づく獣医師の業務であり、動物取扱業の業種には含まれません。競りあっせん業と譲受飼養業は2013年の法改正で追加されました。

</QuizAnswer>

## 問9

2020年6月施行の改正動物愛護管理法により、特定動物（危険な動物）の飼養について変更された内容として正しいものはどれか。

1. 特定動物の飼養許可の申請先が市区町村長に変更された
2. 愛玩目的での特定動物の飼養・保管が禁止された
3. 特定動物のリストからすべての爬虫類が除外された
4. 特定動物の飼養許可制度が届出制に緩和された


<QuizAnswer>

**答え: 2**

2019年改正（2020年6月1日施行）により、愛玩目的での特定動物の飼養・保管が全面的に禁止されました。これにより、特定動物（ライオン、トラ、ワニなど約650種）をペットとして飼うことはできなくなりました。飼養が認められるのは学術研究や展示などの目的に限定され、さらに特定動物との交雑種も規制対象に追加されました。

</QuizAnswer>

## 問10

狂犬病予防法に基づく集合注射の実施期間として定められているのはいつか。

1. 毎年1月1日から3月31日まで
2. 毎年4月1日から6月30日まで
3. 毎年7月1日から9月30日まで
4. 毎年10月1日から12月31日まで


<QuizAnswer>

**答え: 2**

狂犬病予防法では、犬の所有者は毎年1回、狂犬病の予防注射を受けさせる義務があり、集合注射の実施期間は毎年4月1日から6月30日までと定められています。注射後に交付される注射済票は犬に装着する義務があり、鑑札とともに犬の身元を証明する重要な標識です。違反した場合は20万円以下の罰金が科されます。

</QuizAnswer>
