## 問1

動物福祉の基本原則として国際的に広く認められている「5つの自由（Five Freedoms）」は、どの国で最初に体系化されたか。

1. アメリカ
2. ドイツ
3. イギリス
4. フランス


<QuizAnswer>

**答え: 3**

「5つの自由」は、1965年にイギリスで発表されたブランベル報告書をもとに、1979年にイギリスの農用動物福祉委員会（FAWC）が体系化したものです。ルース・ハリソンの著書『アニマル・マシーン』（1964年）が契機となり、家畜福祉の調査委員会が設置されたことがきっかけでした。現在では家畜だけでなく、ペットを含むすべての飼育動物に適用される国際的な動物福祉の基準となっています。

</QuizAnswer>

## 問2

動物福祉の「5つの自由」に含まれないものはどれか。

1. 飢えと渇きからの自由
2. 恐怖と抑圧からの自由
3. 経済的負担からの自由
4. 本来の行動がとれる自由


<QuizAnswer>

**答え: 3**

動物福祉の「5つの自由」は、①飢えと渇きからの自由、②不快からの自由、③痛み・負傷・病気からの自由、④本来の行動がとれる自由、⑤恐怖と抑圧からの自由の5つです。「経済的負担からの自由」は動物福祉の概念には含まれません。これらは動物が身体的にも精神的にも良好な状態で生活できることを保障するための基準です。

</QuizAnswer>

## 問3

1822年にイギリスで成立した、世界初の動物虐待防止法として知られる法律は何か。

1. 動物権利法
2. マーチン法
3. ブランベル法
4. ベンサム法


<QuizAnswer>

**答え: 2**

1822年にイギリスで成立した「マーチン法（畜獣の虐待および不当な取り扱いを防止する法律）」は、世界で初めて動物虐待を法的に禁止した画期的な法律です。この法律の名称は、法案を提出したリチャード・マーチン議員に由来しています。マーチン法の成立後、1824年にはSPCA（動物虐待防止協会、のちのRSPCA）が設立され、動物保護運動が世界的に広がるきっかけとなりました。

</QuizAnswer>

## 問4

「終生飼養」の考え方として最も適切なものはどれか。

1. 動物が病気になったら速やかに保健所に引き渡すこと
2. 動物がその命を終えるまで適切に飼養する責任を果たすこと
3. 動物を一定期間飼った後、次の飼い主を探すこと
4. 動物を自然に帰してあげること


<QuizAnswer>

**答え: 2**

終生飼養とは、飼い主が動物をその命が終わるまで責任をもって適切に飼養することを意味します。安易な飼育放棄や遺棄は動物愛護の理念に反する行為です。飼い主には、動物の高齢化や病気など生涯を通じたケアに対応する覚悟と責任が求められます。終生飼養の原則は、殺処分数の削減や動物の福祉向上において非常に重要な考え方です。

</QuizAnswer>

## 問5

実験動物の福祉に関する「3Rの原則」に含まれないものはどれか。

1. Replacement（代替）
2. Reduction（削減）
3. Refinement（苦痛軽減）
4. Reproduction（繁殖）


<QuizAnswer>

**答え: 4**

実験動物の福祉における3Rの原則は、Replacement（代替：動物実験を他の方法に置き換える）、Reduction（削減：使用する動物の数を減らす）、Refinement（苦痛軽減：動物にできる限り苦痛を与えない方法を用いる）の3つです。Reproduction（繁殖）は3Rには含まれません。この原則は、動物実験の倫理的な実施において国際的に広く採用されている指針です。

</QuizAnswer>

## 問6

動物愛護の考え方における「動物福祉（アニマルウェルフェア）」の説明として最も適切なものはどれか。

1. 動物に人間と同等の法的権利を与えるべきとする考え方
2. 人間の管理下にある動物の身体的・精神的な苦痛を軽減し、良好な状態を保つ考え方
3. 野生動物をすべて保護区に収容して管理する考え方
4. 動物の利用を一切禁止するべきとする考え方


<QuizAnswer>

**答え: 2**

動物福祉（アニマルウェルフェア）とは、人間の管理下にある動物が身体的にも精神的にも良好な状態で生活できるよう配慮する考え方です。動物の利用そのものを否定するのではなく、利用する場合でもできる限り苦痛を軽減し、その動物本来の行動や生き方を尊重することを目指します。動物の権利（アニマルライツ）が動物利用自体を問題視するのに対し、動物福祉はより実践的なアプローチをとっている点が特徴です。

</QuizAnswer>

## 問7

日本における犬猫の殺処分数の推移について、正しい記述はどれか。

1. 近年は増加傾向にあり、年間50万頭を超えている
2. 近年は大幅に減少しており、自治体や保護団体の譲渡活動が貢献している
3. 過去30年間ほぼ横ばいで変化がない
4. 犬の殺処分数は減少しているが、猫の殺処分数は増加している


<QuizAnswer>

**答え: 2**

日本の犬猫の殺処分数は大幅に減少しています。2012年度には約16万頭だった殺処分数が、2023年度には約9,000頭にまで減少しました。これは、自治体や動物愛護団体による譲渡活動の推進、終生飼養の啓発、不妊去勢手術の普及など、さまざまな取り組みの成果です。ただし、ゼロにはなっておらず、引き続き保護活動や適正飼養の普及が求められています。

</QuizAnswer>

## 問8

ペット産業における倫理的問題として、いわゆる「パピーミル（子犬工場）」が批判される主な理由はどれか。

1. 子犬の販売価格が高すぎること
2. 繁殖犬の福祉を無視した過剰な繁殖が行われること
3. 外国産の犬種のみを扱うこと
4. インターネット販売を行っていること


<QuizAnswer>

**答え: 2**

パピーミル（子犬工場）とは、利益を最優先し、繁殖犬の健康や福祉を無視して劣悪な環境で過剰に繁殖を行う悪質な業者を指します。狭いケージでの飼育、十分な医療や栄養の不足、休みなく繁殖させ続けるなど、動物福祉の「5つの自由」に著しく反する行為が問題となっています。こうした施設の存在は、ペット産業全体の倫理的課題として社会的に注目されています。

</QuizAnswer>

## 問9

動物虐待の種類として、「ネグレクト」に該当するものはどれか。

1. 動物を殴るなどの暴力を加えること
2. 十分な食事や水を与えず、世話を怠ること
3. 動物を闘わせて楽しむこと
4. 動物を野山に放して遺棄すること


<QuizAnswer>

**答え: 2**

ネグレクトとは、動物に対して必要な食事・水・清潔な環境・医療などの世話を怠る行為を指し、「飼育放棄」とも呼ばれる虐待の一形態です。殴る・蹴るなどの積極的な暴力（アクティブな虐待）とは異なり、適切なケアを「しない」という消極的な形の虐待です。外見上わかりにくい場合もありますが、動物に深刻な苦痛を与える重大な虐待行為として認識されています。

</QuizAnswer>

## 問10

愛玩動物飼養管理士の社会的役割として最も適切なものはどれか。

1. 動物病院で獣医師の代わりに診療を行うこと
2. ペットショップの経営を行い、売上を最大化すること
3. 動物の適正飼養に関する知識を普及し、人と動物の共生社会の実現に貢献すること
4. 野生動物の捕獲と管理を専門的に行うこと


<QuizAnswer>

**答え: 3**

愛玩動物飼養管理士は、動物の愛護と適正な飼養管理に関する専門的な知識を有し、その知識を広く社会に普及・啓発する役割を担っています。具体的には、飼い主への適正飼養の助言、動物愛護活動への参加、地域における人と動物の共生社会づくりへの貢献などが期待されています。獣医療行為を行う資格ではなく、あくまで飼養管理と愛護精神の普及を担う専門家です。

</QuizAnswer>
