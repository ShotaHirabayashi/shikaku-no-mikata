## 問1

犬の「社会化期」として最も適切な時期はどれか。

1. 生後1～3週齢
2. 生後3～12週齢
3. 生後6～18か月齢
4. 生後1～2歳

<QuizAnswer>

**答え: 2**

犬の社会化期は生後3～12週齢（約3～13週齢）とされています。この時期に人や他の犬、さまざまな環境刺激に慣れさせることで、成犬になってからの恐怖反応や問題行動を予防できます。社会化期を過ぎると新しい刺激に対する警戒心が強まるため、この時期の経験が非常に重要です。

</QuizAnswer>

## 問2

パブロフの犬の実験で知られる「古典的条件づけ（レスポンデント条件づけ）」の説明として正しいものはどれか。

1. 動物が自発的に行った行動の結果によって、その行動の頻度が変化する学習
2. 本来は反応を引き起こさない刺激が、無条件刺激と繰り返し対提示されることで反応を引き起こすようになる学習
3. 他の個体の行動を観察することで新しい行動を獲得する学習
4. 試行錯誤を繰り返すことで問題解決能力が向上する学習

<QuizAnswer>

**答え: 2**

古典的条件づけ（レスポンデント条件づけ）は、パブロフがイヌの唾液分泌実験で発見した学習理論です。もともと唾液分泌を引き起こさないベルの音（中性刺激）を、餌（無条件刺激）と繰り返し対提示することで、ベルの音だけで唾液が出るようになります。選択肢1はオペラント条件づけ、選択肢3は観察学習（社会的学習）の説明です。

</QuizAnswer>

## 問3

オペラント条件づけにおいて、犬が「おすわり」をした直後におやつを与えたところ、「おすわり」の頻度が増えた。この手続きは何と呼ばれるか。

1. 正の罰
2. 負の強化
3. 正の強化
4. 負の罰

<QuizAnswer>

**答え: 3**

正の強化とは、行動の直後に好ましい刺激（おやつなど）を与えることで、その行動の頻度が増加する手続きです。「正」は刺激を加えること、「強化」は行動が増えることを意味します。犬のしつけでは、正の強化を用いた報酬ベースのトレーニングが最も推奨されています。

</QuizAnswer>

## 問4

猫の社会化期として最も適切な時期はどれか。

1. 生後2～9週齢頃
2. 生後8～16週齢頃
3. 生後3～6か月齢
4. 生後6か月～1歳

<QuizAnswer>

**答え: 1**

猫の社会化期は生後2～9週齢頃とされ、犬の社会化期（生後3～12週齢）に比べて早い時期に始まり、期間も短いのが特徴です。この時期に人に触れられる経験や他の猫との交流がないと、人や他の動物に対して恐怖心を持ちやすくなります。社会化期の短さから、猫はより早い段階での適切な対応が求められます。

</QuizAnswer>

## 問5

犬の「カーミングシグナル」の説明として正しいものはどれか。

1. 攻撃の前兆として相手を威嚇するための行動
2. 自分や相手を落ち着かせたり、敵意がないことを示すための行動
3. 飼い主に食事を要求するための行動パターン
4. 繁殖期に異性を引きつけるための行動

<QuizAnswer>

**答え: 2**

カーミングシグナルとは、犬が自分自身や相手を落ち着かせるために行うボディランゲージの一種です。視線をそらす、あくびをする、地面のにおいを嗅ぐ、体を横に向けるなどが代表的な例です。これらのシグナルを理解することで、犬のストレスや不安を早期に察知し、問題行動の予防に役立てることができます。

</QuizAnswer>

## 問6

犬が尾を低い位置にして後ろ足の間に巻き込んでいるとき、最も考えられる感情はどれか。

1. 喜び・興奮
2. 恐怖・不安
3. 攻撃性・威嚇
4. リラックス・安心

<QuizAnswer>

**答え: 2**

犬が尾を後ろ足の間に巻き込むのは、恐怖や不安、服従を示す典型的なボディランゲージです。反対に、尾を高く上げて振るのは興奮や喜びを、尾をピンと立てて毛を逆立てるのは威嚇や攻撃性を示します。犬のボディランゲージを正しく読み取ることは、適切な対応やしつけを行うために重要です。

</QuizAnswer>

## 問7

犬のしつけにおいて、望ましくない行動への対処法として最も適切なものはどれか。

1. 問題行動をした直後に大声で叱り、体罰を与える
2. 問題行動が起きた数時間後に叱って反省させる
3. 問題行動を無視し、望ましい行動をとったときに褒める
4. 問題行動をやめるまで狭い場所に長時間閉じ込める

<QuizAnswer>

**答え: 3**

望ましくない行動を無視し（消去）、望ましい行動をとったときに褒める（正の強化）方法が、現代の動物行動学に基づくしつけとして最も推奨されています。体罰は恐怖心や攻撃性を増加させる恐れがあり、時間が経ってからの叱責は犬がなぜ叱られているか理解できません。行動の直後に適切な対応をすることが学習の基本原則です。

</QuizAnswer>

## 問8

猫が柱や家具の角に頬や体の側面をこすりつける行動の主な目的はどれか。

1. 皮膚のかゆみを解消するため
2. 顔の分泌腺からフェロモンを付けてマーキングするため
3. 爪を研いで鋭くするため
4. 筋肉をストレッチするため

<QuizAnswer>

**答え: 2**

猫の頬や顎の周辺には臭腺（フェロモンを分泌する腺）があり、物や人に顔をこすりつけることでフェイシャルフェロモンを付着させるマーキング行動を行います。これは自分の縄張りを示すとともに、安心感を得る行動でもあります。猫が飼い主に顔をこすりつけるのも、信頼の表現とマーキングの両方の意味があります。

</QuizAnswer>

## 問9

オペラント条件づけにおける「負の強化」の例として正しいものはどれか。

1. 犬が吠えたときにおやつを取り上げたら、吠える頻度が減った
2. 犬がおすわりしたときに不快な音を止めたら、おすわりの頻度が増えた
3. 犬がおすわりしたときにおやつを与えたら、おすわりの頻度が増えた
4. 犬が飛びついたときに不快な音を鳴らしたら、飛びつきの頻度が減った

<QuizAnswer>

**答え: 2**

負の強化とは、行動の直後に不快な刺激を取り除く（負）ことで、その行動の頻度が増加する（強化）手続きです。選択肢1は負の罰（好ましい刺激の除去で行動が減少）、選択肢3は正の強化（好ましい刺激の付加で行動が増加）、選択肢4は正の罰（不快な刺激の付加で行動が減少）に該当します。

</QuizAnswer>

## 問10

犬の社会化期に十分な社会化が行われなかった場合、成犬になってから最も起こりやすい問題行動はどれか。

1. 食糞行動
2. 見知らぬ人や犬に対する過度な恐怖・攻撃反応
3. 特定の場所でしか排泄しない
4. 飼い主の食事中に物を欲しがる

<QuizAnswer>

**答え: 2**

社会化期に人、他の犬、さまざまな環境刺激に十分に触れる経験がないと、成犬になってから見知らぬ人や犬、新しい環境に対して強い恐怖心を抱き、恐怖性攻撃行動を示しやすくなります。これは社会化不足による最も代表的な問題行動であり、一度定着すると矯正が難しいため、社会化期の経験が非常に重要とされています。

</QuizAnswer>
