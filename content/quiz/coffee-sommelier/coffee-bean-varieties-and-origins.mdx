## 問1

コーヒーの三大原種のうち、世界の生産量の約60〜70%を占め、主に標高1,000m以上の高地で栽培される品種はどれか。

1. ロブスタ種
2. リベリカ種
3. アラビカ種
4. エクセルサ種


<QuizAnswer>

**答え: 3**

アラビカ種は世界のコーヒー生産量の約60〜70%を占める最も多く生産されている品種です。標高1,000m以上の高地で栽培され、酸味が豊かでフルーティーな風味が特徴ですが、病害虫や霜に弱いという弱点があります。一方、ロブスタ種は約30%、リベリカ種は1%未満のシェアです。

</QuizAnswer>

## 問2

ロブスタ種（カネフォラ種）の特徴として正しいものはどれか。

1. 酸味が豊かでフルーティーな味わいである
2. カフェイン含有量はアラビカ種の約2倍である
3. 栽培には標高1,500m以上の高地が必要である
4. 世界の生産量に占める割合は1%未満である


<QuizAnswer>

**答え: 2**

ロブスタ種のカフェイン含有量は1.7〜4%で、アラビカ種（0.8〜1.4%）の約2倍です。標高1,000m以下の低地で栽培でき、病害虫に強く収穫量が多いのが特徴です。苦味が強く渋みがあるため、インスタントコーヒーやブレンドの増量用に多く使われています。

</QuizAnswer>

## 問3

アラビカ種の品種のうち、原種に最も近いとされ、滑らかでさわやかな酸味と繊細な味わいが特徴の品種はどれか。

1. カトゥーラ
2. ブルボン
3. ゲイシャ
4. ティピカ


<QuizAnswer>

**答え: 4**

ティピカはアラビカ種の原種に最も近い品種とされ、滑らかでさわやかな酸味と繊細な味わいが特徴です。ただし病害虫に弱く収穫量が少ないという欠点があります。ブルボンはティピカの突然変異種で甘さが特徴、カトゥーラはブルボンの突然変異種で樹高が低いのが特徴です。

</QuizAnswer>

## 問4

コーヒー生産量が世界第1位の国はどこか。

1. コロンビア
2. ベトナム
3. エチオピア
4. ブラジル


<QuizAnswer>

**答え: 4**

ブラジルは世界最大のコーヒー生産国で、世界全体の約30〜35%のシェアを占めています。代表的な銘柄は「サントス」で、南米最大のコーヒー輸出港の名前に由来します。なお、ブラジルの格付けではNo.1は「混入物が完全にゼロはあり得ない」という考えから欠番で、No.2が最高等級となっています。

</QuizAnswer>

## 問5

コロンビアコーヒーの等級制度において、スクリーン17以上の大粒豆を指す最高等級の名称はどれか。

1. エキセルソ
2. スプレモ
3. SHB
4. AA


<QuizAnswer>

**答え: 2**

コロンビアではスクリーンサイズ（豆の大きさ）で等級を決定しており、スクリーン17以上（6.75mm以上）の豆が80%以上のものを「スプレモ（Supremo）」と呼び、これが最高等級です。「Supremo」はスペイン語で「最高級」を意味します。エキセルソはスクリーン14〜16の次の等級です。

</QuizAnswer>

## 問6

エチオピアのコーヒー産地のうち、ジャスミンやベリー系の華やかな香りで知られ、同国最高品質とされる産地はどれか。

1. ハラー
2. シダモ
3. イルガチェフェ
4. リム


<QuizAnswer>

**答え: 3**

イルガチェフェはエチオピア南部シダモ地方の中の特定地域で、ジャスミンやベリー、シトラス系の華やかな香りが特徴です。紅茶のように軽やかで甘みとコクがあり、エチオピアの最高品質のコーヒーとして世界的に評価されています。なお、エチオピアの格付けは欠点数方式で、G1が最高等級です。

</QuizAnswer>

## 問7

インドネシア・スマトラ島北部で生産され、重厚なコクと強い苦味が特徴のコーヒーはどれか。

1. トラジャ
2. マンデリン
3. コナ
4. サントス


<QuizAnswer>

**答え: 2**

マンデリンはインドネシア・スマトラ島北部で生産されるコーヒーで、重厚なコクと強い苦味、酸味がほとんどないのが特徴です。インドネシア独特の「スマトラ式（ウェットハル）」と呼ばれる精製方法で処理され、スパイシーでオリエンタルな味わいを持ちます。深煎りとの相性が良いコーヒーです。

</QuizAnswer>

## 問8

ゲイシャ種の説明として正しいものはどれか。

1. ブラジルで1937年に発見されたブルボンの突然変異種である
2. ケニアのスコット研究所で選抜された品種である
3. エチオピア南西部ゲシャ村が原産で、ジャスミンのような華やかな香りが特徴である
4. ティピカとブルボンの自然交配によって生まれた品種である


<QuizAnswer>

**答え: 3**

ゲイシャ種はエチオピア南西部カファ地方のゲシャ村が原産の品種で、ジャスミンのような上品な花の香りとフルーティーな酸味が特徴です。2004年にパナマのエスメラルダ農園が品評会で過去最高価格で落札され、「ゲイシャショック」と呼ばれる一大ブームを巻き起こしました。栽培が難しく収穫量が少ないため非常に高価です。

</QuizAnswer>

## 問9

グアテマラやコスタリカなど中米諸国で多く採用されている、コーヒー豆の格付け方式はどれか。

1. 欠点数による方式
2. スクリーンサイズ（豆の大きさ）による方式
3. 栽培標高による方式
4. カップテスト（味覚評価）による方式


<QuizAnswer>

**答え: 3**

グアテマラ、コスタリカ、エルサルバドル、ホンジュラスなど中米諸国では、栽培標高によるコーヒー豆の格付けが広く採用されています。高地栽培ほど寒暖差が大きく豆が硬く締まり高品質とされます。最高等級は「SHB（Strictly Hard Bean）」や「SHG（Strictly High Grown）」と呼ばれ、標高1,200m以上が基準です。

</QuizAnswer>

## 問10

「コーヒーの王様」とも呼ばれるジャマイカ・ブルーマウンテンの特徴として正しいものはどれか。

1. 強い酸味と軽やかなボディが最大の特徴である
2. 苦味・酸味・コクの絶妙なバランスの良さが最大の特徴である
3. 麻袋で輸出されるのが伝統的な出荷方法である
4. ロブスタ種を中心に栽培されている


<QuizAnswer>

**答え: 2**

ブルーマウンテンはジャマイカ東部ブルーマウンテン山脈の標高800〜1,200mの限定エリアで栽培され、苦味・酸味・コクが絶妙に調和したバランスの良さが最大の特徴です。「コーヒーの王様」とも称されます。輸出には一般的な麻袋ではなく樽（バレル）が使われており、これはブルーマウンテン独自の慣習です。

</QuizAnswer>
