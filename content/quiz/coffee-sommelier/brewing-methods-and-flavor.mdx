## 問1

ペーパードリップで使用するペーパーフィルターの特徴として、最も適切なものはどれか。

1. コーヒーオイルをそのまま通すため、コクのある味わいになる
2. 金属フィルターより目が粗いため、微粉が多く混じる
3. コーヒーオイルや微粉を吸着・除去するため、すっきりとした味わいになる
4. 布フィルターと同等の目の粗さで、まろやかな味わいになる


<QuizAnswer>

**答え: 3**

ペーパーフィルターは繊維が細かく、コーヒーオイルや微粉をしっかり吸着・除去します。そのため、フレンチプレスやネルドリップに比べて、クリアですっきりとした味わいのコーヒーに仕上がるのが特徴です。

</QuizAnswer>

## 問2

フレンチプレスの抽出方法として正しいものはどれか。

1. 極細挽きの豆を使い、高圧で短時間に抽出する
2. 粗挽きの豆を湯に浸し、約4分後に金属メッシュのプランジャーで押し下げて濾す
3. 細挽きの豆をフランネル布のフィルターで濾過する
4. 中細挽きの豆にお湯を少しずつ注ぎ、重力で抽出する


<QuizAnswer>

**答え: 2**

フレンチプレスは浸漬法（イマージョン）の代表的な器具で、粗挽きの豆を約90〜95℃のお湯に4分間浸した後、金属メッシュのプランジャーを押し下げて抽出します。金属フィルターのためコーヒーオイルが通り、コクのある味わいが特徴です。

</QuizAnswer>

## 問3

サイフォンによるコーヒー抽出の原理として正しいものはどれか。

1. ポンプによる高圧力で湯を押し上げ、コーヒーを抽出する
2. 下部フラスコの水が蒸気圧で上部に押し上げられ、火を消すと気圧差で下部に戻る
3. 電動モーターで湯を循環させ、繰り返しコーヒー粉を通過させる
4. 重力を利用して上部から下部へ湯をゆっくり落とす透過法である


<QuizAnswer>

**答え: 2**

サイフォンは蒸気圧を利用した抽出器具です。下部フラスコの水を加熱すると蒸気圧で湯が上部に押し上げられ、コーヒー粉と混ざります。火を消すと下部の気圧が下がり、その圧力差によって抽出液がフィルターを通って下部に引き戻される仕組みです。

</QuizAnswer>

## 問4

エスプレッソの抽出に必要な圧力として、一般的に正しいとされるものはどれか。

1. 約2気圧
2. 約5気圧
3. 約9気圧
4. 約15気圧


<QuizAnswer>

**答え: 3**

エスプレッソの抽出には約9気圧（9bar）の圧力が必要とされています。この高い圧力によって、極細挽きのコーヒー粉から20〜30秒という短時間で濃厚なコーヒーを抽出します。また、9気圧の圧力がクレマ（表面の金色の泡）を生み出す要因にもなっています。

</QuizAnswer>

## 問5

コーヒーの抽出に使う湯温と味の関係について、正しいものはどれか。

1. 湯温が高いほど酸味が強くなり、低いほど苦味が強くなる
2. 湯温が高いほど苦味が強くなり、低いほど酸味が際立ちやすい
3. 湯温は味にほとんど影響を与えず、挽き目だけが重要である
4. 沸騰直後の100℃の湯を使うのが最もバランスのよい味になる


<QuizAnswer>

**答え: 2**

湯温が高いと苦味成分がより多く抽出され、低いと苦味の抽出が抑えられるため酸味が相対的に際立ちます。一般的にコーヒー抽出の適温は90〜96℃とされ、沸騰直後の100℃の湯は雑味や過度な苦味の原因になるため避けるべきとされています。

</QuizAnswer>

## 問6

コーヒー豆の挽き目と適した抽出器具の組み合わせとして、正しいものはどれか。

1. 極細挽き — フレンチプレス
2. 粗挽き — エスプレッソマシン
3. 中細挽き — ペーパードリップ
4. 粗挽き — マキネッタ（モカポット）


<QuizAnswer>

**答え: 3**

ペーパードリップにはグラニュー糖程度の粒度の中細挽きが適しています。極細挽きはエスプレッソマシン、粗挽きはフレンチプレスに適しています。挽き目が細かいほど表面積が増えて抽出が速く進むため、抽出方法に合った挽き目を選ぶことが重要です。

</QuizAnswer>

## 問7

ペーパードリップにおける「蒸らし」の主な目的として、最も適切なものはどれか。

1. コーヒー粉の温度を均一にするため
2. コーヒー粉に含まれる炭酸ガスを放出させ、湯の浸透を均一にするため
3. コーヒー粉を完全に乾燥させてから抽出を始めるため
4. コーヒー粉の色を確認して焙煎度を判断するため


<QuizAnswer>

**答え: 2**

蒸らし（ブルーミング）は、少量のお湯を注いでコーヒー粉に含まれる炭酸ガス（CO₂）を放出させる工程です。炭酸ガスが残っていると湯をはじいて均一な抽出の妨げになります。新鮮な豆ほどガスを多く含んでおり、蒸らし時にコーヒー粉がふっくらと膨らみます。一般的に20〜30秒間蒸らすのが目安です。

</QuizAnswer>

## 問8

コーヒーの抽出に使う水について、軟水と硬水の影響として正しいものはどれか。

1. 硬水を使うとまろやかで甘みのある味わいになる
2. 軟水を使うとコーヒー本来の風味がバランスよく引き出されやすい
3. 水の硬度はコーヒーの味に影響しない
4. 硬度300mg/L以上の硬水が最もコーヒーに適している


<QuizAnswer>

**答え: 2**

軟水はミネラル分が少ないため、コーヒー本来の風味をバランスよく引き出し、まろやかな味わいになります。一方、硬水はカルシウムやマグネシウムなどのミネラルがコーヒーの成分と反応し、苦味や渋味が出やすくなる傾向があります。日本の水道水は軟水（約50〜60mg/L）で、コーヒー抽出に適しています。

</QuizAnswer>

## 問9

急冷式アイスコーヒーと水出しコーヒー（コールドブリュー）の違いとして、正しいものはどれか。

1. 急冷式は水で長時間かけて抽出し、水出し式は熱湯で短時間に抽出する
2. 急冷式は酸味が出やすく香りが華やか、水出し式はまろやかで酸味が少ない
3. どちらも同じ味わいになるため、抽出方法による違いはない
4. 水出し式は抽出時間が約10分と短く、急冷式は8時間以上かかる


<QuizAnswer>

**答え: 2**

急冷式は熱湯でドリップしたコーヒーを氷で急速に冷やす方法で、熱湯抽出ならではの華やかな香りや酸味が残るのが特徴です。水出し式（コールドブリュー）は冷水で8〜12時間かけてゆっくり抽出するため、苦味や酸味が穏やかでまろやかな甘みのある味わいになります。

</QuizAnswer>

## 問10

プロペラ式（ブレードグラインダー）のコーヒーミルの特徴として、正しいものはどれか。

1. 挽き目の均一性が高く、エスプレッソ用の極細挽きに最も適している
2. 回転する刃で豆を叩き割るため、粒度にムラが出やすい
3. 臼式（バーグラインダー）より高価だが、業務用に広く使われている
4. 豆を切り刻む構造のため、熱が発生しにくく風味が損なわれない


<QuizAnswer>

**答え: 2**

プロペラ式（ブレードグラインダー）は回転するプロペラ状の刃で豆を粉砕するため、挽き時間で粗さを調整しますが、粒度にムラが出やすいのが欠点です。安価で手軽に使える入門用として人気がありますが、均一な挽き目を求める場合は臼式（コニカル刃やフラット刃）のグラインダーが適しています。

</QuizAnswer>
