## 問1

ラテアートの技法のうち、ミルクピッチャーからスチームミルクを直接注ぎながらデザインを描く方法を何というか。

1. エッチング
2. フリーポア
3. スクラッチ
4. トレーシング


<QuizAnswer>

**答え: 2**

ラテアートの技法は大きく「フリーポア」と「エッチング」に分けられます。フリーポアはミルクピッチャーから直接注ぎながら、流量や位置を操作してデザインを描く方法です。一方、エッチングは注ぎ終えた後に楊枝やピンなどの道具を使ってフォーム表面に絵を描く技法です。

</QuizAnswer>

## 問2

エスプレッソ用として使われる「デミタスカップ」の容量として最も適切なものはどれか。

1. 30〜40ml
2. 60〜90ml
3. 150〜180ml
4. 200〜300ml


<QuizAnswer>

**答え: 2**

デミタスカップの容量は60〜90ml程度です。「デミタス」はフランス語で「半分のカップ」を意味し、エスプレッソやトルココーヒーなど少量で濃いコーヒーを提供する際に使用されます。マグカップは200〜300ml程度、カプチーノカップは約180mlが一般的です。

</QuizAnswer>

## 問3

SCA（スペシャルティコーヒー協会）のカッピングにおいて、スペシャルティコーヒーと認定されるために必要な最低スコアは何点か。

1. 70点
2. 75点
3. 80点
4. 85点


<QuizAnswer>

**答え: 3**

SCAのカッピング評価において、100点満点中80点以上のスコアを獲得したコーヒーがスペシャルティコーヒーとして認定されます。評価項目にはアロマ、フレーバー、アフターテイスト、酸味、ボディ、バランスなど10項目があり、それぞれ専門のカッパーによって採点されます。

</QuizAnswer>

## 問4

カプチーノとカフェラテの違いとして正しいものはどれか。

1. カプチーノはミルクを使わず、カフェラテはミルクを使う
2. カプチーノはフォームミルクが厚く、カフェラテはフォームミルクが薄い
3. カフェラテはエスプレッソを使わず、ドリップコーヒーにミルクを加える
4. カプチーノとカフェラテは全く同じ飲み物である


<QuizAnswer>

**答え: 2**

カプチーノはエスプレッソ・スチームミルク・フォームミルクをほぼ均等に配合し、厚いフォーム（泡）の層が特徴です。一方、カフェラテはエスプレッソに対してスチームミルクの割合が多く、フォームミルクは1cm未満の薄い層です。どちらもエスプレッソベースの飲み物ですが、ミルクとフォームの比率が異なります。

</QuizAnswer>

## 問5

カッピングの手順において、コーヒーの粉にお湯を注いでから表面の「クラスト」をスプーンで崩す（ブレイク）までの時間として適切なものはどれか。

1. 30秒
2. 1分
3. 4分
4. 10分


<QuizAnswer>

**答え: 3**

SCAのカッピングプロトコルでは、93℃のお湯を注いでから約4分後にクラスト（表面に浮いた粉の層）をスプーンで3回かき混ぜてブレイクします。この際に立ち上るアロマを評価します。ブレイク後は浮いた粉や泡を取り除き、やや冷めてから味の評価（スラーピング）を行います。

</QuizAnswer>

## 問6

イタリア語で「染みをつけた」という意味を持ち、エスプレッソに少量のフォームミルクを加えた飲み物はどれか。

1. アフォガート
2. マキアート
3. コンパンナ
4. アメリカーノ


<QuizAnswer>

**答え: 2**

マキアート（macchiato）はイタリア語で「染みをつけた・印をつけた」を意味します。エスプレッソに少量のフォームミルクを「染み」のように加えた飲み物で、エスプレッソの風味が主体となるのが特徴です。アフォガートは「溺れた」、コンパンナは「クリームを添えた」という意味のイタリア語です。

</QuizAnswer>

## 問7

コーヒーに含まれる主要なポリフェノールで、特に生豆に多く含まれ、焙煎が深くなるほど減少する成分はどれか。

1. タンニン
2. カテキン
3. クロロゲン酸
4. アントシアニン


<QuizAnswer>

**答え: 3**

クロロゲン酸はコーヒーに含まれる主要なポリフェノールで、生豆の乾燥重量の約12〜18%を占めます。抗酸化作用や血圧低下作用があるとされています。焙煎が深くなるほど分解されて含有量が減少するため、浅煎りのコーヒーのほうがクロロゲン酸を多く摂取できます。

</QuizAnswer>

## 問8

深煎りコーヒーとのフードペアリングとして最も適切な組み合わせはどれか。

1. フレッシュフルーツやフルーツタルト
2. ダークチョコレートやどら焼き
3. バニラクッキーやビスケット
4. 柑橘系のゼリー


<QuizAnswer>

**答え: 2**

深煎りコーヒーは苦味が強くコクがあるため、しっかりとした甘さを持つダークチョコレートやあんこを使った和菓子（どら焼き・大福など）との相性が良いとされています。甘さが苦味とバランスを取り、互いの風味を引き立てます。フレッシュフルーツや柑橘系は酸味のある浅煎りコーヒーとの組み合わせが向いています。

</QuizAnswer>

## 問9

ラテアートに適したスチームミルクの温度として最も適切な範囲はどれか。

1. 40〜45℃
2. 60〜65℃
3. 75〜80℃
4. 90〜95℃


<QuizAnswer>

**答え: 2**

ラテアートに最適なスチームミルクの温度は60〜65℃です。この温度帯では乳糖の甘みが引き出され、きめ細かいマイクロフォームが作れます。65℃を超えるとミルクのタンパク質が変性して泡が粗くなり、甘みも失われるため、ラテアートには不向きになります。

</QuizAnswer>

## 問10

エスプレッソにバニラアイスクリーム（またはジェラート）をのせた、イタリア発祥のデザートコーヒーはどれか。

1. カフェモカ
2. エスプレッソ・コンパンナ
3. アフォガート
4. カフェ・マキアート


<QuizAnswer>

**答え: 3**

アフォガート（affogato）はイタリア語で「溺れた」を意味し、バニラジェラートやアイスクリームに熱いエスプレッソを注いだデザートコーヒーです。冷たいアイスと熱いエスプレッソの温度差が楽しめます。コンパンナはエスプレッソにホイップクリームをのせた飲み物で、アイスクリームは使いません。

</QuizAnswer>
