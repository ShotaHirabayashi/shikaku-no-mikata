## 問1

焙煎の8段階において、最も浅い焙煎度合いはどれか。

1. シナモンロースト
2. ミディアムロースト
3. ライトロースト
4. ハイロースト


<QuizAnswer>

**答え: 3**

焙煎の8段階は浅い方から順に、ライトロースト→シナモンロースト→ミディアムロースト→ハイロースト→シティロースト→フルシティロースト→フレンチロースト→イタリアンローストです。ライトローストは最も浅煎りで、淡い黄褐色をしており、主にカッピング（品質検査）用に使われます。

</QuizAnswer>

## 問2

焙煎度合いが深くなるにつれて、味わいはどのように変化するか。

1. 酸味が増し、苦味が減る
2. 酸味が減り、苦味が増す
3. 酸味も苦味もともに増す
4. 酸味も苦味もともに減る


<QuizAnswer>

**答え: 2**

焙煎が進むほど、コーヒー豆に含まれるクロロゲン酸が分解されて酸味は減少し、代わりにキナ酸などの苦味成分が生成されるため苦味が増加します。浅煎りは明るい酸味が特徴的で、深煎りになるほど強い苦味と重厚なコクが際立ちます。

</QuizAnswer>

## 問3

焙煎中に起こる「1ハゼ（ファーストクラック）」の主な原因として正しいものはどれか。

1. 豆の表面の油脂が燃焼する
2. 豆の内部に蓄積された水蒸気とCO2の圧力が放出される
3. 豆のシルバースキンが剥がれ落ちる
4. 豆に含まれるカフェインが気化する


<QuizAnswer>

**答え: 2**

1ハゼは焙煎温度が約196〜205℃に達した際に起こる現象で、豆の内部に蓄積された水蒸気と二酸化炭素（CO2）の圧力が一気に放出されることで「パチパチ」という破裂音が鳴ります。この発熱反応により豆は50〜100%ほど膨張し、浅煎りから中煎りへと移行する目安となります。

</QuizAnswer>

## 問4

焙煎中に起こる「メイラード反応」の説明として正しいものはどれか。

1. 糖質がアミノ酸と反応し、褐色物質や香気成分を生成する
2. 水分が蒸発して豆が膨張する物理的変化
3. カフェインが熱で分解されて苦味が生まれる
4. 脂質が酸化して独特の風味が失われる


<QuizAnswer>

**答え: 1**

メイラード反応は、コーヒー豆に含まれるアミノ酸と還元糖が加熱によって反応し、メラノイジンと呼ばれる褐色物質や、フランやピラジンなどの数百種類にのぼる香気成分を生成する化学反応です。この反応はコーヒーの褐色、香り、コク（ボディ感）の形成に大きく寄与しています。

</QuizAnswer>

## 問5

日本で最も一般的に飲まれている焙煎度合いとされるのはどれか。

1. フレンチロースト
2. シナモンロースト
3. シティロースト
4. イタリアンロースト


<QuizAnswer>

**答え: 3**

シティローストは焙煎の8段階で5番目（中煎り〜やや深煎り）に位置し、日本で最も広く消費されている焙煎度合いです。明確なロースト香があり、苦味と甘みのバランスが良いのが特徴です。名前の由来はニューヨークシティで標準的な焙煎度合いだったことに由来します。

</QuizAnswer>

## 問6

2ハゼ（セカンドクラック）が起こる温度帯として最も適切なものはどれか。

1. 約150〜160℃
2. 約170〜180℃
3. 約196〜205℃
4. 約220〜225℃


<QuizAnswer>

**答え: 4**

2ハゼは約220〜225℃で発生し、1ハゼよりも小さく速い「ピチピチ」という音が特徴です。主にCO2ガスの圧力によって豆の細胞構造がさらに崩壊する現象で、豆の表面に油脂が浮き出てきます。2ハゼ以降はフレンチロースト〜イタリアンローストの深煎り領域に入ります。

</QuizAnswer>

## 問7

ブレンドコーヒーにおける「アフターミックス」の説明として正しいものはどれか。

1. 抽出後のコーヒー液を混ぜ合わせる方法
2. 生豆の状態でブレンドしてから一緒に焙煎する方法
3. 各豆を個別に最適な焙煎度で焙煎してから混ぜ合わせる方法
4. 焙煎後に香料を添加して風味を調整する方法


<QuizAnswer>

**答え: 3**

アフターミックスとは、各産地のコーヒー豆をそれぞれに最適な焙煎度合いで個別に焙煎した後にブレンドする方法です。各豆の個性を最大限に引き出せるため風味のコントロールに優れており、スペシャルティコーヒーショップで主流の手法です。一方、生豆を混ぜてから焙煎する方法は「プレミックス」と呼ばれます。

</QuizAnswer>

## 問8

ブレンドコーヒーを作る際、全体の30〜50%以上を占めて味の土台を作る豆の役割を何と呼ぶか。

1. アクセント
2. ベース
3. フレーバー
4. フィニッシュ


<QuizAnswer>

**答え: 2**

ブレンドコーヒーでは、味の土台（基盤）となる豆を「ベース」と呼び、全体の30〜50%以上を占めます。ベースにはブラジルやコロンビアなど、バランスが良くクセの少ない豆がよく使われます。これに個性を加える「アクセント」（20〜40%）や、隠し味となる豆（10〜20%）を組み合わせてブレンドを構成します。

</QuizAnswer>

## 問9

焙煎したコーヒー豆の鮮度を保つために避けるべき「4つの大敵」に含まれないものはどれか。

1. 酸素
2. 湿気
3. 光
4. 振動


<QuizAnswer>

**答え: 4**

焙煎豆の鮮度を損なう4つの大敵は「酸素・湿気・熱・光」です。酸素は酸化による風味劣化を、湿気は品質低下を、熱と光は化学変化の促進を引き起こします。振動は鮮度劣化の主要因には含まれません。保存には遮光性のある密閉容器を使い、常温で2週間以内に飲みきるのが理想的です。

</QuizAnswer>

## 問10

焙煎中に起こる「カラメル化」の説明として正しいものはどれか。

1. タンパク質が熱で凝固する反応
2. 糖質がアミノ酸なしに加熱分解され、甘い香りや褐色を生む反応
3. カフェインが別の化合物に変換される反応
4. 脂質が高温で重合する反応


<QuizAnswer>

**答え: 2**

カラメル化は約170〜200℃の温度帯で、コーヒー豆に含まれる糖質がアミノ酸を介さずに熱分解される反応です。キャラメルやトフィーのような甘い香りと褐色を生み出します。ただし焙煎がさらに進むと糖が炭化し、甘味は減少して苦味が増していきます。メイラード反応とは異なり、アミノ酸を必要としない点が特徴です。

</QuizAnswer>
