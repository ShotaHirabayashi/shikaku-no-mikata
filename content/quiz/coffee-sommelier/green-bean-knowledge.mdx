## 問1

コーヒーチェリーの構造において、種子（生豆）の表面を薄く覆っている銀色の皮は何と呼ばれるか。

1. パーチメント
2. ミューシレージ
3. シルバースキン
4. パルプ


<QuizAnswer>

**答え: 3**

シルバースキン（銀皮）は、コーヒーの種子の表面に薄く張り付いている銀色の皮のことです。パーチメントはその外側にある内果皮、ミューシレージはパーチメントの外側にある粘液質、パルプは果肉部分を指します。焙煎時にシルバースキンが剥がれ落ちたものは「チャフ」と呼ばれます。

</QuizAnswer>

## 問2

コーヒーチェリーの精製方法のうち、収穫した実をそのまま天日干しにして乾燥させた後、外皮や果肉を取り除いて生豆を取り出す方法はどれか。

1. ウォッシュド
2. ハニープロセス
3. セミウォッシュド
4. ナチュラル


<QuizAnswer>

**答え: 4**

ナチュラル（乾燥式）は、コーヒーチェリーを果肉がついたまま天日干しで乾燥させ、乾燥後に脱穀して生豆を取り出す最も伝統的な精製方法です。果肉の甘みが豆に移るため、フルーティーで甘みのある風味に仕上がるのが特徴です。水を大量に使わないため、水資源の乏しい地域でも行えます。

</QuizAnswer>

## 問3

欠点豆のうち、精製過程で発酵槽に長時間浸けすぎたり、水が汚染されたりすることが原因で生じる豆はどれか。

1. カビ豆
2. 発酵豆
3. 貝殻豆
4. 未成熟豆


<QuizAnswer>

**答え: 2**

発酵豆は、ウォッシュド精製の過程でコーヒー豆を発酵槽に浸ける際、長時間浸けすぎたり、発酵槽の衛生状態が悪かったりすることで発生します。刺激的な酸味や不快な味の原因となるため、ハンドピックで必ず取り除く必要があります。

</QuizAnswer>

## 問4

コーヒーノキ（アラビカ種）の栽培に適した年間平均気温として、最も適切なものはどれか。

1. 約10℃
2. 約20℃
3. 約30℃
4. 約40℃


<QuizAnswer>

**答え: 2**

アラビカ種のコーヒーノキは、年間平均気温が約20℃前後の環境が栽培に適しています。気温が高すぎると実の成熟が早まり品質が低下し、逆に霜が降りるほど低温になると木が枯れてしまいます。なお、ロブスタ種（カネフォラ種）はやや高温の約25℃前後が適温とされています。

</QuizAnswer>

## 問5

コーヒーの収穫方法で、枝の根元から先端に向かって手で一気にしごき、実をすべて落とす方法を何と呼ぶか。

1. ハンドピック
2. ストリッピング
3. セレクトピッキング
4. メカニカルハーベスト


<QuizAnswer>

**答え: 2**

ストリッピングは、枝についた実を根元から先端に向けて一気にしごき落とす収穫方法です。効率は良いものの、完熟した実だけでなく未熟な実や過熟した実、葉や小枝なども一緒に混ざってしまうため、品質は手摘み（セレクトピッキング）に劣ります。主にブラジルなどの大規模農園で採用されています。

</QuizAnswer>

## 問6

ハニープロセスの特徴として正しいものはどれか。

1. 果肉も外皮もすべてつけたまま乾燥させる
2. 外皮と果肉を除去し、ミューシレージ（粘液質）を残したまま乾燥させる
3. 水洗いで果肉もミューシレージもすべて除去してから乾燥させる
4. 機械で強制的に高温乾燥させる方法である


<QuizAnswer>

**答え: 2**

ハニープロセスは、ウォッシュドとナチュラルの中間的な精製方法です。外皮と果肉（パルプ）は除去しますが、ミューシレージ（粘液質）を残したまま乾燥させます。ミューシレージの糖分がハチミツのように粘り気を帯びることが名前の由来で、ウォッシュドよりも甘みやコクがありつつ、ナチュラルほどクセのないバランスの取れた味わいに仕上がります。

</QuizAnswer>

## 問7

コーヒーの生豆に青っぽいカビが見られる欠点豆が生じる主な原因はどれか。

1. 虫が豆の内部に侵入したため
2. 焙煎温度が高すぎたため
3. 乾燥が不十分だったり、保管時に湿気を帯びたため
4. 収穫が早すぎて未成熟だったため


<QuizAnswer>

**答え: 3**

カビ豆は、精製時の乾燥が不十分だったり、輸送・保管時に湿気を帯びてしまうことが主な原因で発生します。生豆の表面に青っぽいカビが見られるのが特徴で、カビ臭や不快な風味の原因となります。生豆の品質管理において、適切な水分値（11〜13%程度）の維持と通気性の良い保管環境が重要です。

</QuizAnswer>

## 問8

コーヒーチェリーの構造で、パーチメント（内果皮）のすぐ外側にある、ヌルヌルとした粘液質の層を何と呼ぶか。

1. シルバースキン
2. エンドスパーム
3. アウタースキン
4. ミューシレージ


<QuizAnswer>

**答え: 4**

ミューシレージは、パーチメント（内果皮）と果肉（パルプ）の間にあるヌルヌルとした粘液質の層です。コーヒーチェリーが熟すにつれて発達し、糖分を多く含んでいます。ウォッシュド精製では発酵・水洗いで除去されますが、ハニープロセスではこのミューシレージを残して乾燥させることで甘みのある風味を生み出します。

</QuizAnswer>

## 問9

コーヒー農園で、コーヒーノキに直射日光が当たりすぎないよう日陰を作る目的で植えられる背の高い樹木を何と呼ぶか。

1. シェードツリー
2. マザーツリー
3. カバーツリー
4. ガードツリー


<QuizAnswer>

**答え: 1**

シェードツリー（日陰樹）は、コーヒーノキを強い直射日光から守るために植えられる背の高い樹木のことです。日光の調整だけでなく、強風や霜害からコーヒーノキを保護する役割も果たします。さらに、シェードツリーの落ち葉が良質な肥料となり、土壌を豊かにする効果もあります。

</QuizAnswer>

## 問10

欠点豆の除去作業で、焙煎前に生豆を一粒ずつ目視で確認し、手作業で欠点豆や異物を取り除く工程を何と呼ぶか。

1. カッピング
2. スクリーニング
3. ハンドピック
4. グレーディング


<QuizAnswer>

**答え: 3**

ハンドピック（ハンドピッキング）は、生豆を一粒ずつ目で見て手で触りながら、欠点豆（黒豆・発酵豆・虫食い豆・カビ豆・未成熟豆など）や異物を取り除く手作業のことです。生豆には一般的に5〜10%程度の欠点豆が混入しているとされ、この工程を丁寧に行うことでコーヒーの味の均一性と品質が大きく向上します。

</QuizAnswer>
