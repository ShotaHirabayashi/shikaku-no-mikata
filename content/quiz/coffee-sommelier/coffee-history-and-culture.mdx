## 問1

コーヒーの発見にまつわる「カルディ伝説」の舞台となった国はどこか。

1. イエメン
2. トルコ
3. エチオピア
4. インド


<QuizAnswer>

**答え: 3**

カルディ伝説は、エチオピアの山羊飼いカルディが、ヤギたちが赤い実を食べた後に興奮して飛び跳ねる様子に気づいたことがコーヒー発見のきっかけとされる物語です。この伝説が文献に初めて登場したのは1671年のことですが、コーヒーの起源がエチオピアにあるという点は広く認められています。

</QuizAnswer>

## 問2

コーヒーのもう一つの発見伝説「オマール伝説」に登場するシェイク・オマールが追放されていた地域と関係が深い港町はどこか。

1. アデン
2. モカ
3. ジッダ
4. マサワ


<QuizAnswer>

**答え: 2**

オマール伝説では、イエメンのモカの聖職者・医師であったシェイク・オマールが町を追放され、飢えに苦しむ中でコーヒーの実を発見したとされています。この「奇跡の飲み物」の噂がモカに伝わり、オマールは帰還を許されました。モカ港はその後、コーヒー貿易の一大拠点として栄えました。

</QuizAnswer>

## 問3

16世紀にオスマン帝国の首都イスタンブールに登場し、階級を超えた社交の場として発展したコーヒーを提供する施設は何と呼ばれたか。

1. カフェテリア
2. カフヴェハネ
3. サロン・ド・テ
4. タベルナ


<QuizAnswer>

**答え: 2**

オスマン帝国では1554年にイスタンブールに最初のコーヒーハウス「カフヴェハネ（qahveh khaneh）」が開業しました。カフヴェハネは階級・職業・宗派を超えた社交の場となり、詩人や学者が文学・政治・哲学を議論する知的空間として発展しました。16世紀後半にはイスタンブールだけで約600軒に達したとされています。

</QuizAnswer>

## 問4

ヨーロッパで最初にコーヒーが伝わり、1645年頃にヨーロッパ初のコーヒーハウスが開業した都市はどこか。

1. ロンドン
2. パリ
3. ウィーン
4. ヴェネツィア


<QuizAnswer>

**答え: 4**

コーヒーはオスマン帝国から地中海交易路を経てイタリアのヴェネツィアに伝わり、1645年頃にヨーロッパ初のコーヒーハウスが開業しました。ヴェネツィアにはトルコ商人のコミュニティがあり、コーヒー文化を受け入れる素地がありました。ここからヨーロッパ各地にコーヒー文化が広がっていきました。

</QuizAnswer>

## 問5

17世紀のロンドンで、1ペニーでコーヒーを飲みながら知識人と議論できる場として親しまれたコーヒーハウスの別名は何か。

1. ペニー大学
2. ペニーサロン
3. コーヒーアカデミー
4. ジェントルマンズクラブ


<QuizAnswer>

**答え: 1**

17世紀のロンドンのコーヒーハウスは「ペニー大学（Penny University）」と呼ばれました。1ペニーのコーヒー代を払えば、身分に関係なく学者や商人と自由に議論できたことからこの名が付きました。最盛期にはロンドンだけで約550軒ものコーヒーハウスがあり、知識と情報の交換の場として機能していました。

</QuizAnswer>

## 問6

1888年（明治21年）に東京・上野で日本初の喫茶店「可否茶館（かひちゃかん）」を開業した人物は誰か。

1. 渋沢栄一
2. 鄭永慶
3. 福沢諭吉
4. 水野龍


<QuizAnswer>

**答え: 2**

1888年に鄭永慶（てい えいけい）がエール大学留学後、ロンドンのコーヒーハウスに感銘を受けて東京・上野に「可否茶館」を開業しました。西洋風の内装や革椅子、新聞を備えた本格的な喫茶店でしたが、経営は振るわず1893年に閉店しています。これが日本における喫茶店文化の始まりとされています。

</QuizAnswer>

## 問7

コーヒー栽培に適した地帯を指す「コーヒーベルト」は、おおむねどの範囲に位置するか。

1. 北緯10度〜南緯10度
2. 北緯25度〜南緯25度
3. 北緯35度〜南緯35度
4. 北緯45度〜南緯45度


<QuizAnswer>

**答え: 2**

コーヒーベルトとは、おおむね北緯25度から南緯25度の間の熱帯・亜熱帯地域を指します。この地帯は安定した温暖な気候、適度な降水量、高地の冷涼な気温など、コーヒー栽培に適した条件が揃っています。世界70か国以上がこのベルト上に位置し、コーヒーを生産しています。

</QuizAnswer>

## 問8

コーヒーの「セカンドウェーブ」を象徴する、1971年にアメリカで創業したコーヒーチェーンはどれか。

1. ドトールコーヒー
2. ブルーボトルコーヒー
3. スターバックス
4. ピーツコーヒー


<QuizAnswer>

**答え: 3**

セカンドウェーブ（1960年代〜2000年代初頭）は、エスプレッソベースのドリンクや産地による味の違いへの注目が特徴で、コーヒーが単なる飲み物から「体験」へと変化した時代です。1971年創業のスターバックスはその象徴的存在であり、カフェラテやカプチーノなどの文化を世界中に広めました。なお、ブルーボトルコーヒーはサードウェーブの代表格です。

</QuizAnswer>

## 問9

1683年のウィーン包囲戦の後、トルコ軍が残したコーヒー豆をきっかけにコーヒーハウス文化が発展したウィーンについて、正しい記述はどれか。

1. ウィーンのコーヒーハウス文化はUNESCO無形文化遺産に登録されている
2. ウィーンのコーヒーハウスはヨーロッパで最初に開業した
3. ウィーンではコーヒーの栽培にも成功した
4. ウィーンのコーヒーハウスは王族専用の施設だった


<QuizAnswer>

**答え: 1**

1683年のウィーン包囲戦後、トルコ軍が残したコーヒー豆を使ってコーヒーハウスが誕生し、独自のカフェ文化が発展しました。ウィーンのコーヒーハウス文化は2011年にUNESCO無形文化遺産に登録されています。ヨーロッパ初のコーヒーハウスはヴェネツィア（1645年頃）であり、ウィーンではありません。

</QuizAnswer>

## 問10

コーヒーの「サードウェーブ（第3の波）」の特徴として最も適切なものはどれか。

1. インスタントコーヒーの大量生産と家庭への普及
2. エスプレッソマシンの普及とチェーン店の世界展開
3. シングルオリジンやトレーサビリティを重視し、コーヒーをクラフトとして扱う
4. 缶コーヒーの開発と自動販売機での販売拡大


<QuizAnswer>

**答え: 3**

サードウェーブ（2000年代初頭〜）は、コーヒーをワインのようなクラフト・芸術品として扱う潮流です。シングルオリジン（単一産地）の豆を浅煎りで提供し、産地の個性を引き出すことを重視します。ダイレクトトレード、トレーサビリティ（生産履歴の追跡）、サステナビリティといったキーワードが特徴です。選択肢1はファーストウェーブ、選択肢2はセカンドウェーブの特徴です。

</QuizAnswer>
