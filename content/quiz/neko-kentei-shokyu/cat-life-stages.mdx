## 問1

猫の乳歯は全部で何本あるでしょうか？

1. 20本
2. 24本
3. 26本
4. 30本


<QuizAnswer>

**答え: 3**

猫の乳歯は全部で26本です。内訳は、切歯が上下左右3本ずつで12本、犬歯が上下左右1本ずつで4本、臼歯が上顎6本・下顎4本で10本です。なお、永久歯になると後臼歯が加わり、合計30本になります。

</QuizAnswer>

## 問2

猫の「社会化期」とは、一般的に生後どの時期を指すでしょうか？

1. 生後1日～2週齢
2. 生後2週齢～7週齢ごろ
3. 生後3か月～6か月
4. 生後6か月～1歳


<QuizAnswer>

**答え: 2**

猫の社会化期は一般的に生後2週齢～7週齢ごろとされています。この時期に人や他の動物と多く触れ合うことで、成猫になってからの社交性や性格に大きな影響を与えます。犬の社会化期と比べて早い時期に訪れ、期間も短いのが特徴です。

</QuizAnswer>

## 問3

猫の全身の骨の数は約何本でしょうか？

1. 約180本
2. 約206本
3. 約220本
4. 約244本


<QuizAnswer>

**答え: 4**

猫の骨は約244本あり、人間（約206本）より約40本ほど多くなっています。多い分のほとんどは背骨と尻尾を構成する骨です。尾椎だけでも一般的に18～19個あり、この骨の多さが猫のしなやかな動きを可能にしています。

</QuizAnswer>

## 問4

子猫の乳歯が永久歯に生え替わる時期として正しいのはどれでしょうか？

1. 生後3か月～7か月ごろ
2. 生後1か月～3か月ごろ
3. 生後8か月～12か月ごろ
4. 生後1歳～1歳半ごろ


<QuizAnswer>

**答え: 1**

猫の乳歯は生後3か月ごろから抜け始め、7か月ごろまでにすべて永久歯に生え替わります。生え替わりの順番は一般的に、下の切歯→上の切歯→下の犬歯→上の犬歯→臼歯の順です。この時期の子猫は歯茎がむずがゆく、ものをよく噛む傾向があります。

</QuizAnswer>

## 問5

完全室内飼いの猫の平均寿命として最も近いのはどれでしょうか？

1. 約10歳
2. 約16歳
3. 約20歳
4. 約25歳


<QuizAnswer>

**答え: 2**

ペットフード協会の調査によると、完全室内飼いの猫の平均寿命は約16歳（16.22歳）です。外出する猫の平均寿命が約14歳（13.75歳）であるのに比べ、約2年半ほど長くなっています。室内飼いは交通事故や感染症のリスクが低いことが長寿の要因と考えられます。

</QuizAnswer>

## 問6

猫の妊娠期間はおよそどのくらいでしょうか？

1. 約1か月（30日前後）
2. 約3か月（90日前後）
3. 約4か月（120日前後）
4. 約2か月（63～65日前後）


<QuizAnswer>

**答え: 4**

猫の妊娠期間は約63～65日で、およそ2か月です。1回の出産で1頭から最大8頭ほどの子猫を産むことができます。猫は日照時間が14時間を超えると発情する「長日繁殖動物」であり、主に春に出産のピークを迎えます。

</QuizAnswer>

## 問7

成猫の1日の平均的な睡眠時間として最も適切なのはどれでしょうか？

1. 約5～6時間
2. 約8～10時間
3. 約14～16時間
4. 約20～22時間


<QuizAnswer>

**答え: 3**

成猫は1日に約14～16時間眠るとされ、1日の約3分の2を睡眠に費やしています。「猫（ねこ）」の語源が「寝子（ねこ）」に由来するという説もあるほどです。ただし、その大半は浅い眠り（レム睡眠）であり、実際に熟睡している時間は合計3時間程度と言われています。

</QuizAnswer>

## 問8

猫の換毛期は年に何回、いつ頃訪れるでしょうか？

1. 年2回、春と秋
2. 年1回、春のみ
3. 年3回、春・夏・秋
4. 年4回、各季節ごと


<QuizAnswer>

**答え: 1**

猫の換毛期は年2回、春と秋に訪れます。春には防寒用の厚い冬毛が抜けて密度の少ない夏毛に替わり、秋には夏毛が抜けて保温性の高い冬毛が生えてきます。ただし、室内飼いの猫は空調の影響で換毛期が不明瞭になり、年間を通じて毛が抜けることもあります。

</QuizAnswer>

## 問9

猫の鎖骨に関する説明として正しいのはどれでしょうか？

1. 非常に小さく退化しており、肩の関節とつながっていない
2. 人間と同じ大きさで肩の関節としっかりつながっている
3. 鎖骨は完全になくなっており存在しない
4. 人間より大きく頑丈にできている


<QuizAnswer>

**答え: 1**

猫の鎖骨は非常に小さく退化しており、人間のように肩の関節にはつながっていません。このため、猫は狭い隙間を通り抜けたり、着地の衝撃を柔軟に吸収したりすることができます。鎖骨が自由であることが、猫特有のしなやかな動きを生み出す大きな要因の一つです。

</QuizAnswer>

## 問10

猫の被毛のうち、保温の役割を果たす柔らかい下毛のことを何と呼ぶでしょうか？

1. トップコート
2. ガードヘア
3. オーバーコート
4. アンダーコート


<QuizAnswer>

**答え: 4**

猫の被毛は外側の硬い上毛（オーバーコート）と、内側の柔らかい下毛（アンダーコート）の2層で構成されています。アンダーコートは密度が高く、主に保温の役割を果たします。オーバーコートとアンダーコートの両方を持つ猫を「ダブルコート」、アンダーコートがほとんどない猫を「シングルコート」と呼びます。

</QuizAnswer>
