## 問1

猫がしっぽをまっすぐ上に立てて近づいてくるとき、その猫はどのような気持ちを表していますか？

1. 威嚇している
2. 怯えている
3. 機嫌がよく、友好的な挨拶をしている
4. 狩りに集中している


<QuizAnswer>

**答え: 3**

猫がしっぽをピンとまっすぐ立てるのは、機嫌がよく友好的な気持ちの表れです。飼い主に近づくときや、仲の良い猫同士の挨拶としてもよく見られるポーズです。逆にしっぽを下げたり足の間に挟んだりするのは、恐怖や不安のサインになります。

</QuizAnswer>

## 問2

成猫同士のコミュニケーションについて、正しい説明はどれですか？

1. 成猫同士はニャーと鳴き合って会話するのが主なコミュニケーション手段である
2. 成猫同士ではにおい（フェロモン）やボディランゲージが主なコミュニケーション手段である
3. 成猫同士はゴロゴロ音だけでコミュニケーションをとる
4. 猫はもともと単独行動をするため、猫同士のコミュニケーション手段は存在しない


<QuizAnswer>

**答え: 2**

成猫同士は、人間に対するように「ニャー」と鳴き合うことはほとんどありません。猫同士ではにおい（フェロモン）によるマーキングや、しっぽ・耳・体の姿勢などのボディランゲージが主要なコミュニケーション手段です。ニャーという鳴き声は、主に人間に対して発達したコミュニケーション方法だと考えられています。

</QuizAnswer>

## 問3

猫が窓の外の鳥を見ているとき、歯をカチカチ鳴らすような「カカカッ」という音を出すことがあります。この行動は何と呼ばれ、どのような気持ちを表していますか？

1. クラッキングといい、歯が痛いサインである
2. チャタリングといい、獲物への興奮やもどかしさの表れである
3. スプレーの前兆であり、縄張りを主張しようとしている
4. グルーミングの一種であり、リラックスしている


<QuizAnswer>

**答え: 2**

この行動は「チャタリング（クラッキング）」と呼ばれ、獲物を目にしたときの興奮やもどかしさの表れだと考えられています。主に窓越しに鳥や虫を見ているときに見られます。狩猟本能が刺激されているものの獲物に手が届かないフラストレーションが、この独特な音として表れるとされています。

</QuizAnswer>

## 問4

猫がゴロゴロ音（purring）を出す仕組みとして正しいものはどれですか？

1. 胃の中で空気が振動して音が出る
2. しっぽの筋肉が振動して体全体に伝わる
3. 喉の筋肉が高速で収縮・弛緩を繰り返し、声帯付近で振動が生じる
4. 鼻腔内の特殊な膜が共鳴して音が出る


<QuizAnswer>

**答え: 3**

猫のゴロゴロ音は、喉頭（のど）の筋肉が1秒間に約25〜30回という高速で収縮と弛緩を繰り返すことで、声門が開閉し振動が生じる仕組みです。さらに2023年の研究では、声帯に埋め込まれた特殊な脂肪組織が低周波の振動を助けている可能性も報告されています。ゴロゴロ音は息を吸うときも吐くときも連続して出せるのが特徴です。

</QuizAnswer>

## 問5

猫が家具や柱にからだの側面をこすりつける行動の主な目的は何ですか？

1. 毛並みを整えるためのグルーミング行動
2. 頬や体にある臭腺からフェロモンをつけるマーキング行動
3. 筋肉をほぐすためのストレッチ行動
4. 寄生虫を落とすための衛生行動


<QuizAnswer>

**答え: 2**

猫の頬、あご、頭の上、しっぽの付け根などには臭腺（しゅうせん）があり、家具や壁にからだをこすりつけることで自分のフェロモンを付着させています。これはマーキング行動の一種で、自分の縄張りを示したり、なじみのあるにおいをつけて安心感を得るために行います。人間の足にすりすりするのも、同様の理由です。

</QuizAnswer>

## 問6

猫の一日の活動パターンについて、最も正しい説明はどれですか？

1. 猫は完全な夜行性の動物で、夜中に最も活発になる
2. 猫は完全な昼行性の動物で、日中に最も活発になる
3. 猫は薄明薄暮性の動物で、明け方と夕方に最も活発になる
4. 猫は活動時間帯が決まっておらず、常に同じ活動量を保つ


<QuizAnswer>

**答え: 3**

猫は「薄明薄暮性（はくめいはくぼせい）」の動物で、明け方と夕暮れ時に最も活発に活動します。これは猫の主な獲物であるネズミなどの小動物も同じ時間帯に活動するためです。「猫＝夜行性」と思われがちですが、完全な夜行性ではありません。ただし室内飼いの猫は、飼い主の生活リズムに合わせて活動パターンが変化することもあります。

</QuizAnswer>

## 問7

猫が自分のからだを舐めるグルーミング行動について、正しい説明はどれですか？

1. グルーミングは毛づくろいだけが目的で、他の機能はない
2. 猫は1日の約5%程度しかグルーミングに時間を使わない
3. グルーミングには体温調節の役割もあり、唾液の蒸発で体を冷やしている
4. グルーミングは生まれたときからできる本能的な行動である


<QuizAnswer>

**答え: 3**

猫のグルーミングには、毛並みを整える以外にも多くの役割があります。猫は肉球以外に汗腺がないため、体に唾液を塗ることで蒸発させ、体温を下げる効果があります。また、皮脂の分泌を促して毛を防水・保護したり、寄生虫やゴミを除去する衛生目的もあります。猫は1日の約30〜50%をグルーミングに費やすといわれています。

</QuizAnswer>

## 問8

猫同士が互いの体を舐め合う「アログルーミング（社会的グルーミング）」について、正しいものはどれですか？

1. 主に頭や首のあたりを舐め合うことが多い
2. 必ず同じ年齢の猫同士でしか行われない
3. 仲が悪い猫同士でも頻繁に見られる行動である
4. アログルーミングには社会的な意味はなく、単なる衛生行動にすぎない


<QuizAnswer>

**答え: 1**

アログルーミングでは、猫が自分では舐めにくい頭部や首周りを中心に舐め合うのが特徴です。この行動は信頼関係のある猫同士の絆を深め、グループ内で共通のにおいをつけるという社会的な役割を持っています。また、グループ内で優位な立場の猫が下位の猫を舐めることが多いという研究報告もあります。

</QuizAnswer>

## 問9

猫の遊び行動について、正しい説明はどれですか？

1. 猫の遊びは単なる暇つぶしで、本能とは無関係である
2. 猫はおもちゃよりも自分のしっぽで遊ぶことを最も好む
3. 猫の遊びは「見つめる→忍び寄る→飛びかかる→噛みつく」という狩りの動きを再現している
4. 十分な食事を与えていれば猫に遊びは必要ない


<QuizAnswer>

**答え: 3**

猫の遊びの多くは狩猟本能に基づいており、「獲物を見つめる→忍び寄って追いかける→飛びかかって捕まえる→とどめを刺す（噛みつく）」という一連の狩りの行動パターンを再現しています。室内飼いの猫にとって、遊びは狩猟本能を満たす大切な機会です。遊びが不足するとストレスや問題行動の原因になることがあるため、飼い主が定期的に遊んであげることが重要です。

</QuizAnswer>

## 問10

猫がゴロゴロと喉を鳴らすのは、嬉しいときだけですか？次のうち正しいものを選んでください。

1. ゴロゴロ音は嬉しいときにしか出さないため、必ずリラックスしている証拠である
2. ゴロゴロ音は怒っているときにだけ出す威嚇行動である
3. ゴロゴロ音はリラックス時だけでなく、体調が悪いときやストレスを感じたときにも出すことがある
4. ゴロゴロ音は生後1年以上にならないと出すことができない


<QuizAnswer>

**答え: 3**

ゴロゴロ音はリラックスして満足しているときに出すことが最も多いですが、それだけではありません。体調が悪いとき、痛みを感じているとき、不安やストレスを感じたときにも、自分を落ち着かせる「自己鎮静」の手段としてゴロゴロ音を出すことがあります。また、ゴロゴロ音の周波数（約25〜30Hz）には、骨密度の維持や組織の回復を促進する効果があるという研究もあります。

</QuizAnswer>
