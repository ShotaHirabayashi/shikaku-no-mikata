## 問1

現在のイエネコ（家猫）の直接の祖先とされている野生のネコ科動物はどれか。

1. ベンガルヤマネコ
2. リビアヤマネコ
3. マヌルネコ
4. ヨーロッパヤマネコ


<QuizAnswer>

**答え: 2**

2007年のミトコンドリアDNA解析により、イエネコはリビアヤマネコ（Felis lybica）と同じ系統に含まれることが確認されました。リビアヤマネコは中東や北アフリカに生息し、約1万年前に人間との共存を始めたと考えられています。

</QuizAnswer>

## 問2

古代エジプトで猫の姿をした女神として崇拝された神の名前はどれか。

1. イシス
2. セクメト
3. バステト
4. ハトホル


<QuizAnswer>

**答え: 3**

バステトは古代エジプト神話に登場する猫の女神で、「家を守る神」として崇拝されました。崇拝の中心地は下エジプトのブバスティスで、猫はバステトの聖なる獣とされ、死後はミイラにされて丁重に埋葬されました。

</QuizAnswer>

## 問3

日本に猫が本格的に伝来したとされる時代として最も適切なものはどれか。

1. 奈良時代前後（飛鳥～奈良時代）
2. 縄文時代
3. 江戸時代
4. 鎌倉時代


<QuizAnswer>

**答え: 1**

猫は中国大陸を経由して、飛鳥時代から奈良時代（6世紀末～8世紀）の頃に日本に伝わったとされています。渡来人や僧侶が経典をネズミの害から守るために猫を連れてきたと考えられています。日本で最初に「ねこ」という言葉が文献に登場するのは、705年の記述です。

</QuizAnswer>

## 問4

招き猫の発祥に関わる寺院として有名な、東京世田谷区にある寺はどれか。

1. 浅草寺
2. 増上寺
3. 泉岳寺
4. 豪徳寺


<QuizAnswer>

**答え: 4**

豪徳寺は招き猫発祥の地として最も有名な説のひとつです。彦根藩主の井伊直孝が鷹狩りの帰りに、寺の猫に手招きされて立ち寄ったところ雷雨を避けることができたという伝説が残っています。この故事から「福を招く猫」として信仰されるようになりました。

</QuizAnswer>

## 問5

スコティッシュフォールドの最大の特徴である「折れ耳」について、正しい記述はどれか。

1. すべての個体が折れ耳で生まれる
2. 折れ耳の個体は全体の約3割である
3. 折れ耳になるかどうかは生後1年で決まる
4. オスだけが折れ耳になる


<QuizAnswer>

**答え: 2**

スコティッシュフォールドは折れ耳が特徴ですが、実はすべての個体が折れ耳というわけではありません。立ち耳（スコティッシュストレート）が全体の約7割を占め、折れ耳の個体は約3割程度です。1961年にスコットランドで生まれた白猫「スージー」がこの品種の始祖とされています。

</QuizAnswer>

## 問6

三毛猫にオスがほとんどいない理由として正しいものはどれか。

1. オスは三毛の毛色遺伝子を持っていても発現しないため
2. 三毛のオスは生まれてもすぐに毛色が変化するため
3. 毛色を決めるO遺伝子がX染色体上にあり、オスはX染色体を1本しか持たないため
4. 三毛の毛色はミトコンドリアDNAで遺伝し、母親からしか受け継がれないため


<QuizAnswer>

**答え: 3**

猫の毛色のうち、オレンジ（茶色）を決めるO遺伝子はX染色体上に存在します。三毛猫になるにはOとoの両方が必要ですが、オス（XY）はX染色体を1本しか持たないため、通常はOかoのどちらか一方しか持てません。メス（XX）はX染色体を2本持つため、Ooの組み合わせが可能です。三毛猫のオスが生まれる確率は数千～数万分の1とされています。

</QuizAnswer>

## 問7

「ジェントルジャイアント（穏やかな巨人）」の愛称で知られる、純血種の中でも最大級の大きさを持つ猫の品種はどれか。

1. メインクーン
2. ノルウェージャンフォレストキャット
3. ラグドール
4. ブリティッシュショートヘア


<QuizAnswer>

**答え: 1**

メインクーンは「ジェントルジャイアント（穏やかな巨人）」の愛称で親しまれる大型猫で、純血種の中でも最大級です。アメリカのメイン州が原産とされ、おおらかで社交的な性格が特徴です。体重はオスで6〜8kg以上になることもあり、長い被毛とふさふさの尾を持ちます。

</QuizAnswer>

## 問8

「ボイスレスキャット」とも呼ばれ、あまり鳴かないことで知られる、美しいブルー（灰色）の被毛とグリーンの瞳が特徴的な猫の品種はどれか。

1. シャルトリュー
2. コラット
3. ブリティッシュショートヘア
4. ロシアンブルー


<QuizAnswer>

**答え: 4**

ロシアンブルーは短く密な銀灰色（ブルー）の被毛と美しいグリーンの瞳を持つ猫種です。他の猫と比べてあまり鳴かないことから「ボイスレスキャット」とも呼ばれています。上品で穏やかな性格ですが、飼い主には深い愛情を示すことで知られています。

</QuizAnswer>

## 問9

アビシニアンの被毛に見られる、1本の毛に複数の色の帯が入る独特の毛色パターンの名称はどれか。

1. ティッキング
2. ポイント
3. タビー
4. ソリッド


<QuizAnswer>

**答え: 1**

ティッキングとは、1本の毛に2〜3色の色帯が交互に入る毛色パターンのことです。アビシニアンはこのティッキングが最大の特徴で、光の加減で被毛が輝くように見えます。代表的な毛色には赤茶色の「ルディ」、青灰色の「ブルー」、赤色の「レッド」、淡いココア色の「フォーン」の4種類があります。

</QuizAnswer>

## 問10

アメリカンショートヘアの祖先猫がアメリカ大陸に渡ったとされる経緯として正しいものはどれか。

1. 古代のバイキングが北欧から連れてきた
2. イギリスからの移民船に乗って渡った
3. スペインの探検家が南米経由で持ち込んだ
4. アジアから太平洋を渡って伝わった


<QuizAnswer>

**答え: 2**

アメリカンショートヘアの祖先は、イギリス原産のブリティッシュショートヘアとされています。約400年前、イギリスからの移民がメイフラワー号などの船に猫を乗せてアメリカに渡りました。船上でネズミを捕る役目を担っていた猫たちがアメリカに定着し、やがて「アメリカンショートヘア」として品種登録されました。

</QuizAnswer>
