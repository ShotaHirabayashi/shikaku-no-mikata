## 問1

猫の毎日の主食として与えるべきキャットフードの種類はどれか。

1. 一般食
2. 総合栄養食
3. 間食
4. 療法食


<QuizAnswer>

**答え: 2**

総合栄養食とは、そのフードと水だけで猫の健康を維持できるよう栄養バランスが整った製品のことです。一般食（副食）はおかずやトッピングの位置づけであり、それだけでは必要な栄養素を補えません。療法食は特定の病気に対応した獣医師指導のもとで与えるフードです。

</QuizAnswer>

## 問2

猫が食べると赤血球が破壊され、溶血性貧血を引き起こす危険がある食べ物はどれか。

1. 鶏のささみ
2. かぼちゃ
3. 玉ねぎ
4. 白米


<QuizAnswer>

**答え: 3**

玉ねぎをはじめとするネギ類（長ネギ、ニンニク、ニラなど）には有機チオ硫酸化合物が含まれており、猫の赤血球を破壊して溶血性貧血を引き起こします。加熱しても毒性は消えないため、ネギ類が入った料理の汁なども与えてはいけません。

</QuizAnswer>

## 問3

猫にとって最も危険とされ、花や葉だけでなく花粉や花瓶の水に触れただけでも急性腎不全を引き起こす恐れがある植物はどれか。

1. アサガオ
2. ユリ
3. ヒマワリ
4. コスモス


<QuizAnswer>

**答え: 2**

ユリ科の植物は猫にとって最も危険な植物のひとつです。花、葉、茎、花粉、さらには花瓶の水にまで毒性があり、ごくわずかな摂取でも急性腎不全を引き起こし、死亡に至るケースもあります。猫のいる家庭ではユリ科の植物を置かないことが鉄則です。

</QuizAnswer>

## 問4

すべての猫に接種が推奨されるコアワクチン（3種混合ワクチン）に含まれない病気はどれか。

1. 猫ウイルス性鼻気管炎
2. 猫カリシウイルス感染症
3. 猫白血病ウイルス感染症
4. 猫汎白血球減少症


<QuizAnswer>

**答え: 3**

3種混合ワクチン（コアワクチン）は、猫ウイルス性鼻気管炎・猫カリシウイルス感染症・猫汎白血球減少症の3つを予防するものです。猫白血病ウイルス感染症はノンコアワクチンに分類され、外出する猫など感染リスクが高い場合に4種混合ワクチンとして追加接種します。

</QuizAnswer>

## 問5

猫砂の種類のうち、一般的に最も固まりやすく消臭力が高いとされるのはどれか。

1. 紙系
2. おから系
3. 鉱物系（ベントナイト）
4. 木系


<QuizAnswer>

**答え: 3**

鉱物系（ベントナイト）の猫砂は粒が細かく、尿をしっかりと固めることができ、消臭力にも優れています。自然の砂に近い感触のため猫の好みに合いやすいという利点もあります。ただし重量があること、粉塵が出やすいこと、燃えるゴミとして捨てられない自治体が多い点がデメリットです。

</QuizAnswer>

## 問6

ドライキャットフードに含まれる水分量の目安として正しいものはどれか。

1. 約10%以下
2. 約25〜35%
3. 約50%前後
4. 約75%以上


<QuizAnswer>

**答え: 1**

ドライフードの水分含有量は約10%以下（8〜12%程度）です。一方、ウェットフードは約75〜90%の水分を含みます。ドライフードは保存性が高く少量で必要なカロリーを摂取できますが、水分が少ないため、別途十分な飲み水を用意することが大切です。

</QuizAnswer>

## 問7

猫の避妊・去勢手術を行う推奨時期として最も適切なのはどれか。

1. 生後1〜2ヶ月
2. 生後6ヶ月前後
3. 生後1年半以降
4. 3歳以降


<QuizAnswer>

**答え: 2**

猫の避妊・去勢手術は、初めての発情期を迎える前の生後6ヶ月前後が推奨されています。早期に手術を行うことで、メス猫では乳腺腫瘍や子宮蓄膿症の予防、オス猫ではスプレー行動の防止や精巣疾患の予防といったメリットがあります。術後は基礎代謝が低下して太りやすくなるため、食事管理が必要です。

</QuizAnswer>

## 問8

猫の防災備蓄品として推奨されるフードや水の最低限の備蓄日数はどれか。

1. 1日分
2. 3日分
3. 5日分
4. 14日分


<QuizAnswer>

**答え: 3**

猫の防災備蓄品は最低5日分、理想的には7日分以上を用意することが推奨されています。普段食べ慣れたフードと水を備蓄し、「ローリングストック方式」（普段から少し多めに買い置きし、使った分を補充する方法）で管理するのが効果的です。また、環境省は災害時にペットとの「同行避難」を推奨しています。

</QuizAnswer>

## 問9

猫の爪切りで切ってよい部分として正しいのはどれか。

1. 爪のピンク色の部分
2. 爪の根元に近い部分
3. 爪の先端の白い（透明な）部分
4. 爪全体の半分まで


<QuizAnswer>

**答え: 3**

猫の爪切りでは、爪の先端にある白い（透明な）部分だけを切ります。ピンク色の部分には血管と神経が通っているため、そこを切ると出血し強い痛みを伴います。成猫の場合は2〜3週間に1回程度の頻度で爪切りを行うのが目安です。

</QuizAnswer>

## 問10

猫の下部尿路疾患（FLUTD）や尿路結石について正しい記述はどれか。

1. メス猫の方がオス猫より重症化しやすい
2. オス猫は尿道が細く長いため重症化しやすい
3. 尿路結石は子猫にだけ発症する病気である
4. 尿道閉塞が起きても1週間程度は様子を見てよい


<QuizAnswer>

**答え: 2**

オス猫はメス猫に比べて尿道が細く長いため、結石や結晶が詰まりやすく重症化しやすい傾向があります。尿道閉塞は24〜48時間で命に関わる緊急事態であり、排尿姿勢をとるのに尿が出ない場合はすぐに動物病院を受診する必要があります。血尿・頻尿・トイレ外での排尿などが主な初期症状です。

</QuizAnswer>
