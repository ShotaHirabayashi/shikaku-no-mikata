## 問1

小説『吾輩は猫である』の作者は誰か。

1. 森鷗外
2. 夏目漱石
3. 芥川龍之介
4. 太宰治


<QuizAnswer>

**答え: 2**

『吾輩は猫である』は夏目漱石が1905年（明治38年）に雑誌『ホトトギス』に発表した長編小説です。名前のない猫を語り手として、人間社会を風刺的に描いた作品で、日本文学を代表する猫の登場する作品として広く知られています。

</QuizAnswer>

## 問2

絵本『100万回生きたねこ』の作者は誰か。

1. いわさきちひろ
2. 五味太郎
3. 佐野洋子
4. 馬場のぼる


<QuizAnswer>

**答え: 3**

『100万回生きたねこ』は佐野洋子が1977年に講談社から出版した絵本です。何度も生まれ変わりを繰り返す猫が、やがて本当の愛を知る物語で、累計200万部以上を発行した大ベストセラーです。大人から子どもまで幅広く読み継がれています。

</QuizAnswer>

## 問3

スタジオジブリのアニメ映画『魔女の宅急便』に登場する、主人公キキの相棒の黒猫の名前は何か。

1. ジジ
2. ムーン
3. バロン
4. ルナ


<QuizAnswer>

**答え: 1**

『魔女の宅急便』（1989年公開）に登場する黒猫の名前はジジです。ジジはキキと会話ができるパートナーとして物語の重要な役割を担っています。キキの成長とともにジジの声が聞こえなくなるという演出も印象的な作品です。

</QuizAnswer>

## 問4

ことわざ「猫に小判」の意味として正しいものはどれか。

1. 猫はお金が好きだということ
2. 小さなものでも大切にすべきだということ
3. 見た目で判断してはいけないということ
4. 価値のわからない者に貴重なものを与えても無駄だということ


<QuizAnswer>

**答え: 4**

「猫に小判」は、どんなに貴重なものを与えても、その価値がわからない者にとっては何の役にも立たないという意味のことわざです。同様の意味を持つことわざに「豚に真珠」「馬の耳に念仏」などがあります。

</QuizAnswer>

## 問5

地域猫活動で行われる「TNR」の「T」は何の頭文字か。

1. Transfer（移動）
2. Treatment（治療）
3. Trap（捕獲）
4. Train（訓練）


<QuizAnswer>

**答え: 3**

TNRとは、Trap（捕獲）・Neuter（不妊去勢手術）・Return（元の場所に戻す）の頭文字をとった言葉です。飼い主のいない猫を捕獲して不妊去勢手術を行い、元いた場所に戻すことで、猫の繁殖を抑制し、殺処分を減らすための活動です。

</QuizAnswer>

## 問6

2022年6月に施行された改正動物愛護管理法により、犬猫のマイクロチップ装着が義務化された対象はどれか。

1. ブリーダーやペットショップなどの犬猫等販売業者
2. すべての犬猫の飼い主
3. 動物病院
4. 地方自治体の保健所


<QuizAnswer>

**答え: 1**

2022年6月1日施行の改正動物愛護管理法により、犬猫のマイクロチップ装着が義務化されたのは、ブリーダーやペットショップなどの犬猫等販売業者です。一般の飼い主については「努力義務」とされており、装着が推奨されていますが法的な強制ではありません。

</QuizAnswer>

## 問7

スタジオジブリのアニメ映画『猫の恩返し』に登場する猫の男爵「バロン」は、別のジブリ作品にも登場している。その作品はどれか。

1. となりのトトロ
2. 耳をすませば
3. 千と千尋の神隠し
4. もののけ姫


<QuizAnswer>

**答え: 2**

猫の男爵バロン（フンベルト・フォン・ジッキンゲン）は、1995年公開の『耳をすませば』に登場する猫の人形がもとになっています。『猫の恩返し』（2002年公開）は、『耳をすませば』の主人公・月島雫が書いた物語という位置づけのスピンオフ作品です。

</QuizAnswer>

## 問8

保護猫の譲渡において、正式に引き取る前に猫との相性や飼育環境を確認するために設けられる期間を何というか。

1. お見合い期間
2. 仮住まい期間
3. 観察期間
4. トライアル期間


<QuizAnswer>

**答え: 4**

保護猫の譲渡では、正式な引き渡しの前に「トライアル期間」が設けられるのが一般的です。通常1〜2週間程度で、猫が新しい環境や家族に馴染めるか、先住ペットとの相性はどうかなどを確認します。この期間を経て問題がなければ正式譲渡となり、誓約書・契約書を交わします。

</QuizAnswer>

## 問9

童話『長靴をはいた猫』を収録した『寓意のある昔話、またはコント集』の著者は誰か。

1. シャルル・ペロー
2. グリム兄弟
3. アンデルセン
4. イソップ


<QuizAnswer>

**答え: 1**

『長靴をはいた猫』は、フランスの詩人シャルル・ペローが1697年に出版した童話集に収録された作品です。知恵のある猫が貧しい飼い主を王女の婿にまで出世させる物語で、ヨーロッパに広く伝わる民話をもとにしています。グリム兄弟も類似の話を収録していますが、最も有名なのはペロー版です。

</QuizAnswer>

## 問10

TNR活動で不妊去勢手術済みの猫は、耳の先端をV字にカットされる。この猫の通称は何か。

1. しるし猫
2. ハート猫
3. 花猫
4. さくら猫


<QuizAnswer>

**答え: 4**

不妊去勢手術済みの猫は耳の先端をV字にカットされ、その形がさくらの花びらに似ていることから「さくら猫」と呼ばれます。この耳カットは、一度手術を受けた猫が再び捕獲されて不要な麻酔をかけられることを防ぐための大切な目印です。

</QuizAnswer>
