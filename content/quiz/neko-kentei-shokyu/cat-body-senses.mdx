## 問1

猫の目の奥にある「タペタム」とは、どのような役割を持つ組織ですか？

1. 涙を分泌して目を保護する組織
2. 光を反射して増幅させる反射板のような組織
3. 色を識別するための色素細胞の集まり
4. 紫外線から網膜を守るフィルターの役割を持つ組織


<QuizAnswer>

**答え: 2**

タペタムは猫の網膜の裏側にある組織で、目に入った光を鏡のように反射して増幅させる「反射板」の役割を果たしています。これにより、わずかな光でも人間の約6倍の明るさでものを見ることができます。夜に猫の目が光って見えるのも、このタペタムが光を反射しているためです。

</QuizAnswer>

## 問2

猫の聴覚について正しい説明はどれですか？

1. 猫の可聴域は人間とほぼ同じである
2. 猫の耳には約27個の筋肉があり、180度自在に動かせる
3. 猫は低音域に特に優れているが、高音域は人間より劣る
4. 猫の耳は左右同時にしか動かすことができない


<QuizAnswer>

**答え: 2**

猫の耳には約27個もの筋肉があり、左右の耳を独立して180度自在に動かすことができます。これにより効率よく周囲の音を集める「集音器」のような役割を果たしています。猫の可聴域は約20Hz〜60,000Hz以上と、人間よりもはるかに広く、特に高音域に優れています。

</QuizAnswer>

## 問3

猫が特定のにおいを嗅いだときに口を半開きにする反応を何といいますか？

1. パニック反応
2. フレーメン反応
3. タペタム反応
4. マーキング反応


<QuizAnswer>

**答え: 2**

フレーメン反応とは、猫が特定のにおい（主にフェロモン）を嗅いだときに、口を半開きにして上唇を引き上げるような独特の表情を見せる生理現象です。これは上あごにある「鋤鼻器（ヤコブソン器官）」というフェロモンを感知する特殊な器官に、より多くのにおい物質を取り込むために行われます。

</QuizAnswer>

## 問4

猫のフレーメン反応に関わる「ヤコブソン器官（鋤鼻器）」は体のどこにありますか？

1. 鼻の奥
2. 耳の内部
3. 上あご
4. 舌の裏側


<QuizAnswer>

**答え: 3**

ヤコブソン器官（鋤鼻器）は猫の上あごにある器官で、2つの小さな穴を通じてフェロモンなどの化学物質を感知します。猫がフレーメン反応で口を開けるのは、この上あごにある器官により多くのにおい物質を送り込むためです。なお、人間では胎児期にこの器官が退化してしまうため、フレーメン反応は起こりません。

</QuizAnswer>

## 問5

猫のヒゲ（触毛）の役割として正しくないものはどれですか？

1. 隙間を通れるかどうか体の幅を測るセンサー
2. 暗い場所でも平衡感覚を保つ助け
3. 食べ物の味を判別する味覚センサー
4. 空気の振動を感知して周囲の状況を把握する


<QuizAnswer>

**答え: 3**

猫のヒゲ（触毛）は根元に神経と血管が通っている高感度なセンサーで、空間認識（体が通れる幅の判断）、平衡感覚の維持、空気の振動の感知などの役割を担っています。しかし、味を判別する機能はありません。味覚は舌にある味蕾が担当しています。

</QuizAnswer>

## 問6

猫の味覚について正しい説明はどれですか？

1. 猫は人間と同じ5種類の味を感じることができる
2. 猫の味蕾の数は人間の約7,000個に対して約500個と少ない
3. 猫は甘味を最も強く感じることができる
4. 猫の味蕾の数は人間よりも多い


<QuizAnswer>

**答え: 2**

人間の舌には約7,000個の味蕾がありますが、猫の味蕾は約500個と非常に少なくなっています。また、猫は甘味受容体（T1R2）が機能していないため、甘味を感じることができません。猫が感じられる味は主に酸味・苦味・塩味の3種類で、肉食動物としてアミノ酸（旨味）にも敏感に反応します。

</QuizAnswer>

## 問7

猫の体で汗をかくことができる場所はどこですか？

1. 背中
2. お腹
3. 肉球
4. 耳の内側


<QuizAnswer>

**答え: 3**

猫の体の中で汗腺（エクリン腺）があるのは肉球だけです。肉球から汗をかくことで、滑り止めの機能を果たすほか、体温調節にも役立っています。また、緊張や恐怖を感じたときにも肉球に汗をかくことがあります。人間のように全身で汗をかいて体温調節することはできません。

</QuizAnswer>

## 問8

猫の爪の構造や仕組みについて正しい説明はどれですか？

1. 猫の爪は人間の爪と同じ平らな構造をしている
2. 猫の爪は常に出たままで引っ込めることはできない
3. 猫の爪は弾性靭帯の働きにより普段は格納されている
4. 猫の爪は一度伸びたら自然に剥がれ落ちることはない


<QuizAnswer>

**答え: 3**

猫の爪は「弾性靭帯」という靭帯の働きによって、普段は指の中に格納されています。筋肉の力で腱を引っ張ると、滑車の仕組みで爪が外に飛び出します。また、猫の爪はタマネギのように何層にも重なった構造をしており、新しい層が内側から成長することで古い外側の層が自然に剥がれ落ちます。

</QuizAnswer>

## 問9

猫の視覚について正しい説明はどれですか？

1. 猫の視力は人間と同じ程度である
2. 猫は赤・青・緑の3色すべてを認識できる
3. 猫は静止した物体よりも動く物体を捉える動体視力に優れている
4. 猫は暗い場所では人間よりも見えにくい


<QuizAnswer>

**答え: 3**

猫の視力は人間の10分の1程度（約0.1〜0.2）と決して良くありませんが、肉食動物として獲物を捕らえるために動体視力が非常に優れています。また、猫は赤色を認識することができず、認識できるのは主に緑と青の2色です。暗い場所ではタペタムの働きにより人間の約6倍の明るさで見ることができます。

</QuizAnswer>

## 問10

成猫の平熱（正常な体温）として正しい範囲はどれですか？

1. 35.0℃〜36.0℃
2. 36.5℃〜37.5℃
3. 38.0℃〜39.0℃
4. 40.0℃〜41.0℃


<QuizAnswer>

**答え: 3**

成猫の平熱は38.0℃〜39.0℃で、人間よりも約1℃高いのが特徴です。猫の体温は1日の中でも変動があり、一般的に早朝が最も低く、夕方にかけて徐々に高くなります。体温が39.5℃以上の場合は発熱、37.5℃以下の場合は低体温の可能性があるため、注意が必要です。

</QuizAnswer>
