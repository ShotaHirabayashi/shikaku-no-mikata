## 問1

SEOにおいて、検索エンジンの検索結果ページに広告枠を除いて表示される検索結果のことを何と呼ぶか。

1. リスティング検索
2. オーガニック検索（自然検索）
3. ダイレクト検索
4. リファラル検索


<QuizAnswer>

**答え: 2**

オーガニック検索（自然検索）とは、検索エンジンの検索結果ページにおいて、広告枠を除いた部分に表示される検索結果のことです。SEO（検索エンジン最適化）は、このオーガニック検索での表示順位を高めるための施策を指します。リスティング検索という用語は一般的ではなく、リスティング広告は有料の検索連動型広告を意味します。

</QuizAnswer>

## 問2

ユーザーの検索意図（検索クエリ）の分類において、「東京駅 行き方」や「Amazon ログイン」のように、特定のサイトや場所にたどり着きたいという意図を持つクエリは何と呼ばれるか。

1. Knowクエリ
2. Doクエリ
3. Buyクエリ
4. Goクエリ


<QuizAnswer>

**答え: 4**

Goクエリは、特定のウェブサイトや場所に「行きたい」という意図を持つ検索クエリです。「Amazon ログイン」のように特定サイトへのナビゲーションを目的としたものや、「東京駅 行き方」のように物理的な場所への移動を意図したものが含まれます。Knowクエリは「知りたい」、Doクエリは「やりたい」、Buyクエリは「買いたい」という意図をそれぞれ表します。

</QuizAnswer>

## 問3

リスティング広告（検索連動型広告）の課金方式として最も一般的なものはどれか。

1. インプレッション課金（CPM）
2. クリック課金（CPC）
3. 成果報酬課金（CPA）
4. 期間保証課金


<QuizAnswer>

**答え: 2**

リスティング広告（検索連動型広告）は、ユーザーが広告をクリックした際に費用が発生するクリック課金（CPC：Cost Per Click）方式が最も一般的です。広告が表示されただけでは費用が発生しないため、広告主にとって費用対効果を管理しやすい仕組みとなっています。CPM（Cost Per Mille）はディスプレイ広告で多く使われる課金方式です。

</QuizAnswer>

## 問4

ある広告が5,000回表示され、150回クリックされた場合、CTR（クリック率）はいくらか。

1. 0.3%
2. 3.0%
3. 5.0%
4. 30.0%


<QuizAnswer>

**答え: 2**

CTR（Click Through Rate：クリック率）は「クリック数 ÷ インプレッション数 × 100」で算出します。この場合、150 ÷ 5,000 × 100 = 3.0%となります。CTRは広告やコンテンツがユーザーの興味をどれだけ引いたかを測る基本的な指標であり、リスティング広告では一般的に2〜6%程度が平均とされています。

</QuizAnswer>

## 問5

ディスプレイ広告において、一度ウェブサイトを訪問したユーザーに対して再度広告を表示する手法を何と呼ぶか。

1. アフィリエイト広告
2. ネイティブ広告
3. リターゲティング（リマーケティング）
4. インフィード広告


<QuizAnswer>

**答え: 3**

リターゲティング（リマーケティング）は、過去に自社サイトを訪問したユーザーに対して、他のサイトやSNSを閲覧している際に再度広告を表示する手法です。すでに自社の商品やサービスに関心を示したユーザーに再アプローチするため、新規ユーザー向け広告と比較して2〜3倍高いコンバージョン率が期待できるとされています。Google広告では「リマーケティング」、Yahoo!広告では「リターゲティング」と呼ばれます。

</QuizAnswer>

## 問6

広告費が30万円、クリック数が600回であった場合、CPC（クリック単価）はいくらか。

1. 200円
2. 500円
3. 1,000円
4. 1,500円


<QuizAnswer>

**答え: 2**

CPC（Cost Per Click：クリック単価）は「広告費 ÷ クリック数」で算出します。この場合、300,000円 ÷ 600回 = 500円となります。CPCは広告の費用効率を評価する基本的な指標であり、キーワードの競合状況や業種によって大きく異なります。CPCを下げつつCTRを維持・向上させることが広告運用の基本的な目標の一つです。

</QuizAnswer>

## 問7

SNS広告の特徴として、最も適切なものはどれか。

1. 検索キーワードに連動して広告が表示される
2. 年齢・性別・興味関心など詳細なデモグラフィックターゲティングが可能である
3. オーガニック検索の順位を直接向上させる効果がある
4. クリック課金のみ対応しており、インプレッション課金は利用できない


<QuizAnswer>

**答え: 2**

SNS広告の大きな特徴は、SNSプラットフォームが保有するユーザーの登録情報や行動データを活用し、年齢・性別・地域・興味関心・職業などの詳細なデモグラフィックターゲティングが可能な点です。検索キーワード連動はリスティング広告の特徴であり、SNS広告はSEO（オーガニック検索順位）には直接影響しません。また、SNS広告ではCPC課金だけでなくCPM課金なども利用可能です。

</QuizAnswer>

## 問8

CPM（Cost Per Mille）の説明として正しいものはどれか。

1. 広告が1回クリックされるごとに発生する費用
2. 広告が1,000回表示されるごとに発生する費用
3. 広告経由で1件のコンバージョンが発生するごとにかかる費用
4. 広告が1万回表示されるごとに発生する費用


<QuizAnswer>

**答え: 2**

CPM（Cost Per Mille）は、広告が1,000回表示（1,000インプレッション）されるごとに発生する費用のことです。「Mille」はラテン語で「1,000」を意味します。ディスプレイ広告やSNS広告でよく使われる課金方式で、ブランド認知の向上を目的とした広告キャンペーンに適しています。CPC（クリック単価）やCPA（獲得単価）とは異なる指標です。

</QuizAnswer>

## 問9

検索クエリの分類において、「ワイヤレスイヤホン おすすめ」「ノートPC 比較」のように、商品の購入を検討しているユーザーが使用するクエリはどれに該当するか。

1. Knowクエリ
2. Goクエリ
3. Buyクエリ
4. Doクエリ


<QuizAnswer>

**答え: 3**

Buyクエリは「買いたい」という購買意図を持つ検索クエリで、「おすすめ」「比較」「ランキング」「価格」などのキーワードを伴うことが多いです。コンバージョン（購入・申込み）に最も近いクエリ分類であり、リスティング広告やECサイトのSEOにおいて特に重要視されます。Knowクエリは情報収集目的、Doクエリは行動目的、Goクエリはナビゲーション目的のクエリです。

</QuizAnswer>

## 問10

リスティング広告とディスプレイ広告を比較した場合、一般的に正しい記述はどれか。

1. ディスプレイ広告のほうがリスティング広告よりもCTRが高い
2. リスティング広告は画像や動画を使った視覚的な訴求に優れている
3. リスティング広告はすでにニーズが顕在化したユーザーへのアプローチに適している
4. ディスプレイ広告は検索キーワードに連動して表示される


<QuizAnswer>

**答え: 3**

リスティング広告（検索連動型広告）は、ユーザーが自ら検索キーワードを入力した結果に表示されるため、すでにニーズが顕在化したユーザーへのアプローチに適しています。一般的にリスティング広告のCTRは2〜6%程度であるのに対し、ディスプレイ広告は0.3〜1%程度と低い傾向にあります。画像や動画を活用した視覚的訴求はディスプレイ広告の特徴であり、ディスプレイ広告は潜在層への認知拡大に向いています。

</QuizAnswer>
