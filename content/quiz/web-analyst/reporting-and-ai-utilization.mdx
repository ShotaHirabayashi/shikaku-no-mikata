## 問1

ウェブサイトの滞在時間データのように、一部の極端に大きな値（外れ値）が含まれるデータの代表値として、最も適切な統計量はどれか。

1. 平均値（算術平均）
2. 中央値（メジアン）
3. 最頻値（モード）
4. 範囲（レンジ）


<QuizAnswer>

**答え: 2**

中央値はデータを大きさ順に並べたときの中央の値であり、外れ値の影響を受けにくい特徴があります。平均値は極端な値に引っ張られやすいため、たとえば9人が1分滞在し1人が100分滞在した場合、平均は約11分となり実態を反映しません。滞在時間や購入金額のように分布が偏りやすいデータでは、中央値がより適切な代表値となります。

</QuizAnswer>

## 問2

標準偏差に関する説明として、正しいものはどれか。

1. 標準偏差はデータの最大値と最小値の差を表す
2. 標準偏差が大きいほど、データは平均値の周りに集中している
3. 正規分布において、平均値±1標準偏差の範囲にデータの約68%が含まれる
4. 標準偏差は分散を2乗して求める


<QuizAnswer>

**答え: 3**

正規分布では、平均値±1標準偏差の範囲に約68%、±2標準偏差に約95%、±3標準偏差に約99.7%のデータが含まれます。選択肢1は「範囲（レンジ）」の説明です。選択肢2は逆で、標準偏差が小さいほどデータは平均値付近に集中します。選択肢4も逆で、標準偏差は分散の平方根（ルート）を取って求めます。

</QuizAnswer>

## 問3

月別の売上推移を示したい場合に、最も適切なグラフの種類はどれか。

1. 円グラフ
2. 散布図
3. 折れ線グラフ
4. レーダーチャート


<QuizAnswer>

**答え: 3**

時系列データ（時間の経過に伴う変化）を表現するには折れ線グラフが最も適切です。折れ線グラフは増減の傾向やトレンドを視覚的に把握しやすく、複数の系列を重ねて比較することもできます。円グラフは構成比を示すためのもので、散布図は2変数の相関関係を見るために使います。

</QuizAnswer>

## 問4

円グラフの使い方として、適切でないものはどれか。

1. 流入チャネルの構成比を表示する
2. カテゴリ数を5つ以下に絞って表示する
3. 3D表示にして視覚的なインパクトを高める
4. 全体に対する各要素の割合を示す


<QuizAnswer>

**答え: 3**

3D円グラフは奥行きによって面積の知覚が歪み、手前の要素が実際より大きく見えるなど、データを正確に伝えることができません。データの正確な表現を妨げるため、3D円グラフは避けるべきとされています。円グラフは全体を100%とした構成比を示す場合に有効ですが、カテゴリ数は5つ以下に抑えることが推奨されます。

</QuizAnswer>

## 問5

「アイスクリームの売上が増えると、水難事故も増える」というデータが得られた。この関係に関する解釈として最も適切なものはどれか。

1. アイスクリームの売上増加が水難事故の原因である
2. 水難事故の増加がアイスクリームの売上増加の原因である
3. 両者には相関関係があるが、因果関係があるとは言えない
4. 両者には相関関係も因果関係もない


<QuizAnswer>

**答え: 3**

これは「疑似相関」の典型例です。アイスクリームの売上と水難事故はともに「気温（夏）」という第三の変数（交絡因子）によって増加するため、見かけ上の相関関係はありますが、一方が他方の原因ではありません。相関関係があるからといって因果関係があるとは限らないことは、データ分析における最も重要な原則の一つです。

</QuizAnswer>

## 問6

ダッシュボード設計の原則として、適切でないものはどれか。

1. 最も重要なKPIを画面上部に配置する
2. できるだけ多くの指標を1画面に詰め込み網羅性を高める
3. 目標値や前期比などの比較基準を併記する
4. ユーザーが約5秒で要点を把握できるようにする


<QuizAnswer>

**答え: 2**

ダッシュボード設計では、1画面あたり5〜9個程度の可視化に絞ることが推奨されています。情報を詰め込みすぎると認知負荷が高まり、重要な指標が埋もれてしまいます。逆ピラミッド型（重要度順）のレイアウト、5秒ルール（すぐに要点が伝わる）、ベンチマーク（比較基準）の提示がダッシュボード設計の基本原則です。

</QuizAnswer>

## 問7

ウェブ解析レポートの構成として、最も適切なものはどれか。

1. 詳細データ → 分析結果 → 結論 → 改善提案の順に記載する
2. 結論・要点を最初に示し、その後に根拠となるデータと改善提案を記載する
3. グラフや図表のみで構成し、文章による説明は省略する
4. すべての指標を網羅的に記載し、読み手に解釈を委ねる


<QuizAnswer>

**答え: 2**

ウェブ解析レポートでは「結論ファースト」が基本原則です。まず要点（サマリー）を1ページに簡潔にまとめ、その後に根拠データや詳細分析、改善提案を記載します。意思決定者が短時間で要点を把握できることが重要であり、サマリーは「3分で1ページ」で説明できるレベルにまとめることが推奨されています。

</QuizAnswer>

## 問8

生成AIをマーケティングで活用する際の注意点として、最も重要なものはどれか。

1. AIが生成した内容はすべて正確なので、そのまま公開してよい
2. AIの出力は必ず人間がファクトチェックしてから公開する
3. 個人情報をプロンプトに含めるとより精度の高い出力が得られるので推奨される
4. AI生成コンテンツは著作権の問題が一切ないので自由に利用できる


<QuizAnswer>

**答え: 2**

生成AIには「ハルシネーション」と呼ばれる、もっともらしいが事実と異なる情報を生成するリスクがあります。そのため「Human-in-the-Loop（人間による確認）」が最も重要な安全策とされています。個人情報をプロンプトに入力することはプライバシーリスクがあり避けるべきです。また、AI生成コンテンツにも著作権上の課題があり、無条件に自由利用できるわけではありません。

</QuizAnswer>

## 問9

プロンプトエンジニアリングの手法で、AIに対して「ステップ・バイ・ステップで考えてください」と指示することで推論精度を高める手法の名称はどれか。

1. ゼロショットプロンプティング
2. フューショットプロンプティング
3. チェーン・オブ・ソート（CoT）プロンプティング
4. ロールプレイプロンプティング


<QuizAnswer>

**答え: 3**

チェーン・オブ・ソート（Chain of Thought）プロンプティングは、AIに段階的な思考過程を明示させることで、複雑な分析タスクの精度を向上させる手法です。ゼロショットは例示なしで指示する方法、フューショットは少数の例を示してパターンを学習させる方法、ロールプレイは特定の役割を与える方法です。

</QuizAnswer>

## 問10

2変数間の相関の強さを示す相関係数（r）について、正しい説明はどれか。

1. 相関係数は0から1の範囲の値をとり、1に近いほど相関が強い
2. 相関係数が0.8であれば、一方が他方の原因であると結論できる
3. 相関係数は-1から+1の範囲の値をとり、0に近いほど相関が弱い
4. 相関係数が負の値の場合、2つの変数に関係がないことを意味する


<QuizAnswer>

**答え: 3**

相関係数（r）は-1から+1の範囲をとり、+1に近いほど強い正の相関（一方が増えると他方も増える）、-1に近いほど強い負の相関（一方が増えると他方は減る）、0に近いほど相関が弱いことを示します。負の相関係数は「関係がない」のではなく「逆方向の関係がある」ことを意味します。また、相関が強くても因果関係の証明にはなりません。

</QuizAnswer>
