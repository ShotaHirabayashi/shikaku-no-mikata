## 問1

ウェブ解析の主な目的として、最も適切なものはどれか。

1. ウェブサイトのデザインを美しくすること
2. ウェブサイトのデータを収集・分析し、事業成果の改善につなげること
3. 競合サイトのアクセス数を正確に把握すること
4. サーバーの負荷を軽減するためにトラフィックを制御すること


<QuizAnswer>

**答え: 2**

ウェブ解析とは、ウェブサイトやアプリのデータを定量的に収集・分析し、ユーザー行動を理解することで、事業成果（コンバージョン、売上など）の改善につなげる活動です。単なるアクセス数の把握やデザイン改善ではなく、データに基づいた意思決定を通じてビジネスゴールの達成を支援することが本質的な目的です。

</QuizAnswer>

## 問2

GA4において、あるユーザーが1日にサイトを午前と午後の2回訪問し、合計5ページを閲覧した場合、セッション数・ユーザー数・ページビュー数の組み合わせとして正しいものはどれか。

1. セッション数: 1、ユーザー数: 1、PV数: 5
2. セッション数: 2、ユーザー数: 2、PV数: 5
3. セッション数: 2、ユーザー数: 1、PV数: 5
4. セッション数: 2、ユーザー数: 1、PV数: 2


<QuizAnswer>

**答え: 3**

セッションはユーザーがサイトを訪問してから離脱するまでの一連の行動を指すため、2回の訪問で2セッションとなります。ユーザー数は同一ユーザーであれば何度訪問しても「1」とカウントされます。PV数はページが表示された回数の合計なので5となります。

</QuizAnswer>

## 問3

GA4におけるセッションのタイムアウト時間として、デフォルトで設定されている値はどれか。

1. 10分
2. 20分
3. 30分
4. 60分


<QuizAnswer>

**答え: 3**

GA4では、ユーザーがサイト上で30分間操作を行わなかった場合にセッションがタイムアウトし、その後の操作は新しいセッションとしてカウントされます。この30分という値はデフォルト設定であり、GA4の管理画面から変更することも可能です。

</QuizAnswer>

## 問4

GA4における「エンゲージメント セッション」の条件として、正しくないものはどれか。

1. セッション中にコンバージョンイベント（キーイベント）が発生した
2. セッション中に2回以上のページビューがあった
3. サイトに滞在したアクティブ時間が10秒以上であった
4. セッション中にページを最後までスクロールした


<QuizAnswer>

**答え: 4**

GA4でエンゲージメント セッションと判定される条件は、(1)アクティブなエンゲージメント時間が10秒以上、(2)2回以上のページビュー（またはスクリーンビュー）、(3)コンバージョンイベント（キーイベント）が1回以上発生、のいずれかを満たすことです。「ページを最後までスクロールした」はエンゲージメントセッションの判定条件には含まれていません。

</QuizAnswer>

## 問5

GA4における直帰率の定義として正しいものはどれか。

1. 1ページだけ閲覧して離脱したセッションの割合
2. エンゲージメントが発生しなかったセッションの割合
3. サイト滞在時間が0秒だったセッションの割合
4. 最初のページで離脱したセッションの割合


<QuizAnswer>

**答え: 2**

GA4の直帰率は「エンゲージメントが発生しなかったセッションの割合」と定義されており、計算式は「1 − エンゲージメント率」です。旧UA（ユニバーサルアナリティクス）では「1ページだけ閲覧して離脱したセッションの割合」が直帰率でしたが、GA4では定義が大きく変更されています。

</QuizAnswer>

## 問6

GA4において、2024年3月以降「コンバージョン」の名称が変更された。変更後の名称として正しいものはどれか。

1. ゴールイベント
2. キーイベント
3. 目標達成イベント
4. アクションイベント


<QuizAnswer>

**答え: 2**

2024年3月に、GA4では従来の「コンバージョン」が「キーイベント」に名称変更されました。これはGoogle広告における「コンバージョン」との混同を避けるためです。GA4のキーイベントはすべての流入元を対象としたサイト上の重要なユーザー行動を指し、Google広告のコンバージョンは広告経由の成果のみを指すという違いがあります。

</QuizAnswer>

## 問7

Googleアナリティクスがユーザーを識別するために使用するCookieの種類として正しいものはどれか。

1. サードパーティCookie
2. セッションCookieのみ
3. ファーストパーティCookie
4. HTTPOnly Cookie


<QuizAnswer>

**答え: 3**

Googleアナリティクス（GA4）は、ファーストパーティCookieを使用してユーザーやセッションを識別します。ファーストパーティCookieとは、ユーザーが訪問しているウェブサイトのドメインから発行されるCookieのことです。GA4では「_ga」や「_ga_」などのCookieが使用され、デフォルトの有効期限は2年ですが、ブラウザの制限により実際には最大400日（約13か月）となっています。

</QuizAnswer>

## 問8

ウェブ解析においてデータを収集する際、ブラウザから解析サーバーへ情報を送信する小さなリクエストのことを何と呼ぶか。

1. クローラー
2. ビーコン
3. スパイダー
4. ボット


<QuizAnswer>

**答え: 2**

ビーコン（Webビーコン）とは、ウェブページに埋め込まれたタグ（JavaScriptコード等）によって、ユーザーのアクセス情報を解析サーバーに送信する仕組みのことです。GA4ではgtag.jsなどのトラッキングタグがページ読み込み時に実行され、ユーザーの行動データをビーコンとしてGoogleのサーバーに送信します。クローラーやスパイダーは検索エンジンがウェブページを巡回するプログラムを指す用語です。

</QuizAnswer>

## 問9

GA4における「離脱率」の説明として正しいものはどれか。

1. エンゲージメントが発生しなかったセッションの割合
2. サイト全体のうち、そのページでセッションが終了した割合
3. 検索結果からサイトに流入しなかった割合
4. 広告をクリックせずにページを閉じた割合


<QuizAnswer>

**答え: 2**

離脱率とは、特定のページにおいて、そのページがセッションの最後のページ（ユーザーがサイトを離れたページ）となった割合を示す指標です。すべてのページには一定の離脱率があり、直帰率とは異なる概念です。直帰率はGA4ではエンゲージメントが発生しなかったセッションの割合を指しますが、離脱率はあくまで各ページ単位で算出される指標です。

</QuizAnswer>

## 問10

ウェブ解析士が業務を行ううえで重視すべき倫理的な姿勢として、最も適切なものはどれか。

1. クライアントの要望に応じて、都合の良いデータのみを報告する
2. 収集したユーザーの個人データは、許可なく第三者に提供しない
3. 解析結果は専門用語を多用し、クライアントが自ら解釈できるようにする
4. アクセス数を増やすために、ボットによるトラフィックも成果として計上する


<QuizAnswer>

**答え: 2**

ウェブ解析士は、個人情報保護法やプライバシーポリシーに基づき、収集したユーザーデータを適切に管理し、許可なく第三者に提供しないことが求められます。データの恣意的な取捨選択やボットトラフィックの計上は倫理に反し、クライアントへの報告は専門用語に頼らず分かりやすく伝えることが重要です。ウェブ解析士はデータに基づく正確で公正な分析を通じて、事業改善に貢献する役割を担っています。

</QuizAnswer>
