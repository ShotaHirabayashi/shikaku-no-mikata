## 問1

チョコレートのテイスティングで、最初に行うべき手順はどれか。

1. 口に入れて舌の上で転がす
2. 外観（色・ツヤ・状態）を観察する
3. 鼻を近づけて香りを嗅ぐ
4. 折って割れる音を確認する


<QuizAnswer>

**答え: 2**

テイスティングは「外観→香り→食感（音）→味わい→余韻」の順で行うのが基本です。まず色合いやツヤ、表面の状態を目で確認し、品質や状態をチェックします。その後、香りを嗅ぎ、割って音や断面を確認し、口に含んで風味を評価します。

</QuizAnswer>

## 問2

チョコレートの保存に最も適した温度帯はどれか。

1. 0〜5℃
2. 15〜18℃
3. 25〜28℃
4. 30℃以上


<QuizAnswer>

**答え: 2**

チョコレートの適正保存温度は一般に15〜18℃、湿度50〜60%程度とされます。冷蔵庫（0〜5℃）では出し入れ時の結露でシュガーブルームを起こしやすく、高温ではココアバターが溶けてファットブルームの原因になります。直射日光や強い香りを避けて保管するのが基本です。

</QuizAnswer>

## 問3

チョコレート表面に白い斑点や粉っぽい膜が現れる「ファットブルーム」の主な原因はどれか。

1. 湿気がチョコレート表面で結露し、砂糖が溶けて再結晶化したため
2. ココアバターが温度変化で溶けて表面に染み出し、再結晶化したため
3. カカオポリフェノールが酸化したため
4. 乳成分が分離したため


<QuizAnswer>

**答え: 2**

ファットブルームは温度変化によりココアバターの結晶が不安定になり、表面に染み出して粗く再結晶化する現象です。一方、シュガーブルームは湿気で砂糖が溶け再結晶化したもので、白くザラついた質感になります。風味は劣化しますが食べても害はありません。

</QuizAnswer>

## 問4

「シングルオリジン」のチョコレートを最もよく説明しているのはどれか。

1. 単一の工場のみで生産されたチョコレート
2. 単一の国・地域・農園など限定された産地のカカオ豆を使用したチョコレート
3. 単一のショコラティエが手作りしたチョコレート
4. 単一の品種のカカオのみを使用したチョコレート


<QuizAnswer>

**答え: 2**

シングルオリジンとは、産地（国・地域・農園など）を限定したカカオ豆を使用したチョコレートを指します。産地特有のテロワール（風土）が風味に反映され、ブレンドにはない個性的な香味が楽しめるのが特徴です。ビーントゥバーの文脈でよく使われる概念です。

</QuizAnswer>

## 問5

「ビーントゥバー（Bean to Bar）」の意味として正しいものはどれか。

1. カカオ豆の選定からチョコレートバー（板チョコ）になるまで一貫して製造する手法
2. カカオ豆を一切使わず代用素材で作る製法
3. ボンボンショコラ専門の製造方式
4. 複数メーカーのチョコレートをブレンドする手法


<QuizAnswer>

**答え: 1**

ビーントゥバーは「豆（Bean）から板チョコ（Bar）まで」を一貫して自社で行う製造スタイルです。カカオ豆の選定・焙煎・摩砕・コンチング・テンパリング・成形までを管理することで、産地の個性を活かした個性派チョコレートが作れます。2000年代以降アメリカを中心に世界的ムーブメントとなりました。

</QuizAnswer>

## 問6

チョコレートとワインのペアリングの基本原則として、最も適切なものはどれか。

1. ワインの甘さよりチョコレートの甘さが強い方が望ましい
2. ワインの甘さがチョコレートの甘さと同等以上の方が調和しやすい
3. 必ず辛口の白ワインを合わせる
4. アルコール度数が低いほど合いやすい


<QuizAnswer>

**答え: 2**

一般にチョコレートよりワインの甘味が弱いと、ワインが酸っぱく感じられバランスが崩れます。そのためワインはチョコレートと同等以上の甘さを持つものを選ぶのが基本です。ダークチョコレートには甘口の赤（ポート、バニュルス等）、ミルクチョコレートには中甘口のワインなどがよく合うとされます。

</QuizAnswer>

## 問7

カカオやチョコレートの風味表現を体系的に整理した「フレーバーホイール」について、正しい説明はどれか。

1. カカオの収穫量を表すグラフ
2. 香り・味の特徴をカテゴリーごとに円形に配置した風味語彙の図
3. カカオ豆の発酵度合いを示す温度計
4. ショコラティエの技術ランキング表


<QuizAnswer>

**答え: 2**

フレーバーホイールは、フルーティ、フローラル、ナッツ、スパイス、アーシー（土っぽい）など、風味の特徴をカテゴリー別に円形配置した語彙ツールです。テイスティング時に感じた香味を共通言語で表現するのに役立ちます。カカオ専用のものも各機関から発表されています。

</QuizAnswer>

## 問8

ベルギーのショコラティエで、ピエール・マルコリーニやヴィタメールなどと並び称される、独自の前衛的なボンボンショコラで世界的に知られる人物はどれか。

1. ピエール・エルメ
2. ジャン＝ポール・エヴァン
3. ピエール・マルコリーニ
4. ジョエル・ロブション


<QuizAnswer>

**答え: 3**

ピエール・マルコリーニは1995年に世界菓子最高峰コンクール「クープ・デュ・モンド」で優勝したベルギーの著名ショコラティエです。自社でカカオ豆の選定・買付からビーントゥバーで製造を行うスタイルで知られ、繊細なボンボンショコラが評価されています。エルメ・エヴァンはフランスを代表するショコラティエ／パティシエです。

</QuizAnswer>

## 問9

コーヒーとチョコレートのペアリングについて、最も一般的に推奨される考え方はどれか。

1. 産地・焙煎度・酸味と苦味のバランスを考えて、風味が調和または対比するよう組み合わせる
2. 必ず深煎りコーヒーとホワイトチョコレートの組み合わせにする
3. インスタントコーヒーが最も適している
4. コーヒーの温度は冷たい方が必ずよい


<QuizAnswer>

**答え: 1**

コーヒーとチョコレートはどちらも産地や製法で香味が大きく変わるため、酸味・苦味・甘味・コクのバランスを意識して合わせます。フルーティな浅煎りコーヒーには酸味のあるシングルオリジンダーク、深煎りにはミルクチョコレートなど、調和または対比で組み合わせるのが基本です。

</QuizAnswer>

## 問10

チョコレートの香りを評価する際、口に含んで鼻から抜ける香りを指す用語はどれか。

1. アロマ（立ち香）
2. レトロネーザル（口中香・戻り香）
3. テクスチャー
4. アフターテイスト


<QuizAnswer>

**答え: 2**

香りは大きく、鼻から直接嗅ぐ「オルソネーザル（アロマ／立ち香）」と、口に含んだ際に鼻腔へ抜ける「レトロネーザル（口中香・戻り香）」に分けられます。チョコレートのテイスティングでは両方を評価することで、香りの広がりや余韻まで捉えることができます。アフターテイストは食後の余韻味を指します。

</QuizAnswer>
