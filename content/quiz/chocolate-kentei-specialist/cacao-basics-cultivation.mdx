## 問1

カカオの三大品種のうち、最も希少で「カカオの女王」とも呼ばれ、繊細で芳香豊かな風味を持つ品種はどれですか？

1. フォラステロ
2. クリオロ
3. トリニタリオ
4. ナシオナル


<QuizAnswer>

**答え: 2**

クリオロ種は世界のカカオ生産量のわずか数パーセント以下しか占めない希少品種で、繊細でフローラルな香りと複雑な風味を持ちます。病害虫に弱く栽培が難しいため、高級チョコレートの原料として珍重されます。「クリオロ」とはスペイン語で「土着の」という意味です。

</QuizAnswer>

## 問2

世界のカカオ生産量の大部分を占め、病害虫に強く生産性が高い品種はどれですか？

1. クリオロ
2. ナシオナル
3. フォラステロ
4. アリバ


<QuizAnswer>

**答え: 3**

フォラステロ種は世界のカカオ生産量の約80〜90%を占めるバルクカカオの主要品種です。病害虫や環境への耐性が強く、収量が多いため、西アフリカを中心に広く栽培されています。風味はクリオロに比べシンプルで力強い苦味と渋味が特徴です。

</QuizAnswer>

## 問3

クリオロとフォラステロの自然交配により生まれた、両者の長所を併せ持つカカオ品種はどれですか？

1. トリニタリオ
2. アメロナド
3. ナシオナル
4. クリオロ・モデルノ


<QuizAnswer>

**答え: 1**

トリニタリオ種は18世紀にカリブ海のトリニダード島でクリオロとフォラステロの自然交配により誕生しました。クリオロの豊かな風味とフォラステロの強靭さを併せ持ち、ファインカカオに分類されることが多い品種です。世界生産量の約10〜15%を占めます。

</QuizAnswer>

## 問4

カカオの木が栽培可能な「カカオベルト」とは、赤道を中心とした緯度何度の地域を指しますか？

1. 北緯・南緯それぞれ約10度以内
2. 北緯・南緯それぞれ約20度以内
3. 北緯・南緯それぞれ約30度以内
4. 北緯・南緯それぞれ約40度以内


<QuizAnswer>

**答え: 2**

カカオベルトは赤道を挟んだ北緯約20度から南緯約20度の地域を指します。年間平均気温27℃前後、年間降水量1,000〜2,500mm、高湿度といった高温多湿の熱帯気候がカカオ栽培に適しています。西アフリカ、中南米、東南アジアなどがこの地域に含まれます。

</QuizAnswer>

## 問5

現在、世界最大のカカオ生産国はどこですか？

1. ガーナ
2. インドネシア
3. エクアドル
4. コートジボワール


<QuizAnswer>

**答え: 4**

コートジボワール（アイボリーコースト）は世界最大のカカオ生産国で、世界全体の約40%以上を生産しています。ガーナがそれに次ぎ、両国で世界生産量の約60%を占めます。西アフリカ地域はフォラステロ種を中心としたバルクカカオの主産地です。

</QuizAnswer>

## 問6

カカオの花や実（カカオポッド）が、枝先ではなく幹や太い枝に直接実る性質を何といいますか？

1. 頂生
2. 幹生花（カウリフロリー）
3. 腋生
4. 房生


<QuizAnswer>

**答え: 2**

カカオの木は幹や太い枝に直接花を咲かせ、実を結ぶ「幹生花（カウリフロリー）」と呼ばれる性質を持ちます。これは熱帯雨林に多く見られる特徴です。1本のカカオの木には年間数千〜数万の花が咲きますが、結実するのはわずか数十個程度です。

</QuizAnswer>

## 問7

収穫したカカオ豆の発酵工程で、主に行われていることは何ですか？

1. カカオ豆の殻を取り除く
2. パルプ（果肉）の糖分を微生物が分解し、風味の前駆体を生成する
3. カカオ豆を細かく粉砕する
4. カカオ豆を高温でローストする


<QuizAnswer>

**答え: 2**

発酵工程ではカカオパルプ（果肉）に含まれる糖分を酵母や乳酸菌、酢酸菌が分解し、アルコールや酢酸を生成します。この過程でカカオ豆内部に化学変化が起き、チョコレート特有の風味の前駆体が形成されます。発酵期間は通常5〜7日間で、品質を左右する極めて重要な工程です。

</QuizAnswer>

## 問8

発酵後のカカオ豆を乾燥させる工程で、最終的に水分量を約何%まで下げる必要がありますか？

1. 約1〜3%
2. 約7〜8%
3. 約15〜20%
4. 約25〜30%


<QuizAnswer>

**答え: 2**

発酵後のカカオ豆は水分含有量が約60%もあり、これを天日乾燥や機械乾燥により約7〜8%まで下げる必要があります。乾燥が不十分だとカビが発生し品質が劣化するため、輸送・保管に適した水分量への調整が重要です。乾燥には通常1〜2週間ほどかかります。

</QuizAnswer>

## 問9

ICCO（国際ココア機関）が定める「ファインカカオ（フレーバーカカオ）」と「バルクカカオ」の違いについて、正しい説明はどれですか？

1. ファインカカオは生産量が多く、バルクカカオは希少である
2. ファインカカオは香味特性に優れた高品質カカオで、バルクカカオは大量生産される標準品種である
3. ファインカカオはアフリカ産限定で、バルクカカオは中南米産限定である
4. ファインカカオは加工用、バルクカカオは食用に区分されている


<QuizAnswer>

**答え: 2**

ファインカカオ（フレーバーカカオ）はクリオロやトリニタリオ、ナシオナルなど香味特性に優れた高品質なカカオで、世界生産量の約5〜10%程度です。一方バルクカカオは主にフォラステロ種で、大量生産される標準的なカカオを指します。ICCOが定期的に対象国・品種を見直しています。

</QuizAnswer>

## 問10

カカオ生産における「フェアトレード」の主な目的として、最も適切なものはどれですか？

1. カカオの遺伝子組み換え品種を普及させること
2. 生産者に公正な価格を保証し、持続可能な生活と労働環境を支えること
3. カカオを安価で大量に流通させること
4. カカオの輸出関税を引き上げること


<QuizAnswer>

**答え: 2**

フェアトレードは、カカオ生産者に最低保証価格やプレミアム（奨励金）を支払うことで、貧困や児童労働などの問題を解決し、持続可能な生産と生活を支える仕組みです。国際フェアトレード認証ラベルなどがあり、サステナビリティへの関心の高まりとともに普及が進んでいます。

</QuizAnswer>
