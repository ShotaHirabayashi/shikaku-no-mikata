## 問1

カカオ豆を加熱して香りを引き出し、外皮を剥がしやすくする工程を何と呼ぶか。

1. コンチング
2. ロースト（焙煎）
3. テンパリング
4. ウィノーイング


<QuizAnswer>

**答え: 2**

ローストはカカオ豆を加熱する工程で、チョコレート特有の香ばしい香りや風味を引き出す重要な役割を持ちます。温度や時間によって風味が大きく変化し、低温長時間や高温短時間など製品特性に合わせて調整されます。また、ロースト後は外皮が脆くなり、次のウィノーイング工程で除去しやすくなります。

</QuizAnswer>

## 問2

ロースト後のカカオ豆から外皮（ハスク）と胚芽を除去し、カカオニブを取り出す工程を何と呼ぶか。

1. ウィノーイング
2. リファイニング
3. コンチング
4. モールディング

<QuizAnswer>

**答え: 1**

ウィノーイング（winnowing）は、ローストしたカカオ豆を粗く砕いた後、風力などを使って軽い外皮（ハスク）を吹き飛ばし、可食部であるカカオニブを取り出す工程です。この工程によって得られるカカオニブが、以降の摩砕工程でカカオマスへと加工されていきます。

</QuizAnswer>

## 問3

カカオニブをすりつぶし、摩擦熱でココアバターを溶かして液状またはペースト状にしたものを何と呼ぶか。

1. ココアパウダー
2. クーベルチュール
3. カカオマス（チョコレートリカー）
4. ガナッシュ


<QuizAnswer>

**答え: 3**

カカオニブを摩砕すると、含まれるココアバター（約55%）が摩擦熱で溶け出し、液状のペーストになります。これをカカオマス、または英語でチョコレートリカーと呼びます。冷えて固まったものはビターチョコレートの主原料となり、また圧搾するとココアバターとココアケーキ（ココアパウダーの原料）に分離されます。

</QuizAnswer>

## 問4

チョコレート製造において、原料を細かく粉砕して粒子を約20ミクロン以下にし、舌触りを滑らかにする工程・装置を何と呼ぶか。

1. メランジャー
2. リファイナー（精錬機）
3. テンパリングマシン
4. ウィノワー


<QuizAnswer>

**答え: 2**

リファイナー（refiner）はロールミル等を使って、カカオマス・砂糖・粉乳などを細かく粉砕する機械です。粒子を約15〜20ミクロン以下まで細かくすることで、舌の上でザラつきを感じない滑らかな口当たりを生み出します。一般に粒径が20ミクロン以下になると、人間の舌は粒を感じにくくなるとされています。

</QuizAnswer>

## 問5

リファイニング後のチョコレート生地を長時間練り続け、揮発性の不快な酸味や水分を飛ばし、香りと風味をまろやかに整える工程を何と呼ぶか。

1. テンパリング
2. ウィノーイング
3. コンチング
4. モールディング


<QuizAnswer>

**答え: 3**

コンチング（conching）は、1879年にスイスのロドルフ・リンツが発明した工程で、長時間（数時間〜数十日）チョコレートを練り続けます。これにより不快な揮発性の酸や水分が飛び、ココアバターが粒子をコーティングして滑らかさが向上し、香りも調和します。コンチェ（コンチェル）と呼ばれる装置で行われます。

</QuizAnswer>

## 問6

チョコレートのテンパリング（調温）を行う最大の目的として、最も適切なものはどれか。

1. カカオの香りを揮発させるため
2. ココアバターを安定したV型結晶に整えるため
3. 砂糖を完全に溶かすため
4. 殺菌処理を行うため


<QuizAnswer>

**答え: 2**

テンパリングは、ココアバターの結晶を最も安定したV型（ベータ結晶）に揃えるための温度調整工程です。一般的には45〜50℃で溶かし、26〜27℃まで冷却してから30〜32℃に再加熱します。これにより、艶のある仕上がり、パキッとした食感（スナップ性）、口溶けの良さ、ファットブルーム防止が実現します。

</QuizAnswer>

## 問7

テンパリング後のチョコレートを型（モールド）に流し込み、冷却して固める工程の説明として正しいものはどれか。

1. 急速に0℃以下まで冷却するのが理想である
2. 振動を与えて気泡を抜き、10〜15℃前後で冷却する
3. 室温（25℃）で時間をかけてゆっくり固める
4. 湿度100%の環境で冷却するのが望ましい


<QuizAnswer>

**答え: 2**

成形・冷却工程では、テンパリング済みのチョコレートを型に流した後、振動を与えて気泡を抜き、表面を平らにします。その後10〜15℃程度の冷却トンネル等でゆっくり冷却することで、結晶が安定し艶のある仕上がりとなります。低温すぎる急冷や高湿度はファットブルームやシュガーブルームの原因となります。

</QuizAnswer>

## 問8

カカオマスを圧搾機にかけて分離される2つの製品の組み合わせとして正しいものはどれか。

1. カカオニブとカカオハスク
2. ココアバターとココアケーキ（ココアパウダー原料）
3. ホワイトチョコレートとミルクチョコレート
4. グラニュー糖と粉乳


<QuizAnswer>

**答え: 2**

カカオマスを油圧プレス等で圧搾すると、液状の脂肪分であるココアバターと、固形分のココアケーキに分離されます。ココアケーキを粉砕・篩い分けしたものがココアパウダーです。ココアバターはチョコレートやホワイトチョコレートの原料、化粧品にも使われ、ココアパウダーは飲料や製菓用に利用されます。

</QuizAnswer>

## 問9

ココアパウダー製造において、アルカリ処理（ダッチプロセス）を行う主な目的として正しいものはどれか。

1. カカオの酸味を和らげ、色を濃くして水に溶けやすくするため
2. 脂肪分を完全に除去するため
3. カフェイン量を増やすため
4. 賞味期限を短くするため


<QuizAnswer>

**答え: 1**

ダッチプロセス（アルカリ処理）は1828年にオランダのバンホーテンが発明した方法で、カカオに炭酸カリウムなどのアルカリを加えて処理します。これにより酸味が和らぎ、色が濃くなり、水への分散性が向上します。日本のJAS等でも、無調整のナチュラルココアと区別されており、製品特性に応じて使い分けられています。

</QuizAnswer>

## 問10

カカオ豆からチョコレートになるまでの一般的な製造工程の順序として正しいものはどれか。

1. ロースト → ウィノーイング → 摩砕 → リファイニング → コンチング → テンパリング → 成形・冷却
2. ウィノーイング → ロースト → コンチング → リファイニング → テンパリング → 摩砕 → 成形
3. リファイニング → ロースト → 摩砕 → ウィノーイング → テンパリング → コンチング → 成形
4. ロースト → コンチング → ウィノーイング → 摩砕 → リファイニング → 成形 → テンパリング


<QuizAnswer>

**答え: 1**

チョコレート製造の基本フローは、①ロースト（焙煎）→②ウィノーイング（外皮除去）→③摩砕（カカオマス化）→④砂糖・粉乳等の混合とリファイニング（微粒化）→⑤コンチング（練り上げ）→⑥テンパリング（結晶調整）→⑦成形・冷却の順で進みます。各工程が風味・食感・見た目を決定づける重要な役割を担っています。

</QuizAnswer>
