## 問1

カカオに含まれるポリフェノールの主成分として最も代表的なものはどれですか？

1. アントシアニン
2. カカオフラバノール（エピカテキン・カテキン類）
3. レスベラトロール
4. イソフラボン


<QuizAnswer>

**答え: 2**

カカオポリフェノールの主成分はフラバノール類で、特にエピカテキンとカテキンが代表的です。これらは強い抗酸化作用を持ち、血流改善や血圧低下などの健康効果が研究されています。アントシアニンはベリー類、レスベラトロールは赤ワイン、イソフラボンは大豆に多く含まれます。

</QuizAnswer>

## 問2

チョコレートに含まれ、苦味成分であると同時に穏やかな興奮作用を持つアルカロイドはどれですか？

1. ニコチン
2. モルヒネ
3. テオブロミン
4. キニーネ


<QuizAnswer>

**答え: 3**

テオブロミンはカカオ豆に特徴的に含まれるアルカロイドで、チョコレートの苦味の主成分です。カフェインに似た中枢神経刺激作用がありますが、より穏やかで持続的に作用します。利尿作用や血管拡張作用も知られていますが、犬猫など一部の動物には代謝できず中毒を起こす点に注意が必要です。

</QuizAnswer>

## 問3

テンパリングによって安定的に形成される、チョコレートに最適なココアバターの結晶型はどれですか？

1. I型
2. III型
3. V型
4. VI型


<QuizAnswer>

**答え: 3**

ココアバターには6種類の結晶型（I〜VI型）が存在し、テンパリングではV型結晶を安定的に作ることが目標です。V型は融点が約33〜34℃で口溶けがよく、つやとパリッとしたスナップ性を生みます。VI型は最も安定ですが融点が高く、長期保存中にV型から徐々に転移し、ファットブルームの原因となります。

</QuizAnswer>

## 問4

高カカオチョコレートに豊富に含まれ、整腸作用や血糖値の急上昇抑制に役立つ成分はどれですか？

1. 食物繊維
2. ビタミンC
3. コラーゲン
4. EPA


<QuizAnswer>

**答え: 1**

カカオには不溶性食物繊維（リグニンやセルロース等）が豊富に含まれ、高カカオチョコレートでは特に多くなります。食物繊維は便通改善や腸内環境の整備、糖の吸収を緩やかにする働きがあります。これによりチョコレートはGI値が比較的低く抑えられる食品の一つとされています。

</QuizAnswer>

## 問5

ココアバターを構成する主要な脂肪酸のうち、血中コレステロールを上げにくいとされる一価不飽和脂肪酸はどれですか？

1. パルミチン酸
2. ステアリン酸
3. オレイン酸
4. リノール酸


<QuizAnswer>

**答え: 3**

ココアバターは主にパルミチン酸・ステアリン酸・オレイン酸の3種で構成されます。このうちオレイン酸は一価不飽和脂肪酸で、オリーブオイルにも多く含まれLDLコレステロールを上げにくいとされます。またステアリン酸は飽和脂肪酸ですが、体内で代謝される際にコレステロールへの影響が比較的小さいことが知られています。

</QuizAnswer>

## 問6

カカオポリフェノールの健康効果として、近年の研究で報告されているものはどれですか？

1. 血管内皮機能の改善や血圧低下作用
2. 血糖値を急上昇させる作用
3. 体内で鉄分を破壊する作用
4. ビタミンDの合成促進作用


<QuizAnswer>

**答え: 1**

カカオフラバノールは血管内皮で一酸化窒素（NO）の産生を促し、血管を拡張させて血圧を下げたり血流を改善したりする効果が報告されています。また抗酸化作用による動脈硬化予防やLDL酸化抑制も研究されています。EUではカカオフラバノール摂取と血流維持に関する健康強調表示も認められています。

</QuizAnswer>

## 問7

ミルクチョコレートを食べる人が特に注意すべき食物アレルギーの原因として、最も代表的なものはどれですか？

1. 小麦
2. 乳（牛乳）
3. そば
4. 甲殻類


<QuizAnswer>

**答え: 2**

ミルクチョコレートには全粉乳や脱脂粉乳といった乳成分が含まれるため、乳アレルギーの人は摂取を避ける必要があります。日本の食品表示制度では「乳」は特定原材料（表示義務）の一つです。またナッツ類（ヘーゼルナッツ・アーモンド等）の混入や製造ライン共有によるコンタミネーションにも注意が必要です。

</QuizAnswer>

## 問8

チョコレートに含まれ、リラックス効果や幸福感に関わる神経伝達物質のような働きを示すと言われる成分はどれですか？

1. グルタミン酸
2. フェニルエチルアミン（PEA）
3. アスパラギン酸
4. グリシン


<QuizAnswer>

**答え: 2**

フェニルエチルアミン（PEA）はカカオに含まれるアミン類で、脳内で気分高揚や幸福感に関与すると言われ、しばしば「恋愛物質」とも呼ばれます。またカカオにはセロトニンの材料となるトリプトファンや、リラックスに関わるGABA様成分も含まれ、これらが「チョコレートを食べると幸せになる」という体感の科学的背景の一つとされています。

</QuizAnswer>

## 問9

GI（グリセミック・インデックス）値に関する説明として正しいものはどれですか？

1. 食品の脂質含有量を示す指標である
2. 食品が体内でビタミンに変換される割合を示す
3. 食後血糖値の上がりやすさを示す指標で、高カカオチョコレートは比較的低めになりやすい
4. 食品のカロリーそのものを表す数値である


<QuizAnswer>

**答え: 3**

GI値は食後血糖値の上昇度合いを示す指標で、ブドウ糖を100として相対評価されます。高カカオチョコレートは砂糖が少なく、脂質や食物繊維、ポリフェノールが豊富なため、糖の吸収が緩やかでGI値は比較的低い傾向にあります。ただしカロリーは決して低くないため、食べ過ぎには注意が必要です。

</QuizAnswer>

## 問10

チョコレートの美容・アンチエイジング効果に関する説明として、最も適切なものはどれですか？

1. カカオポリフェノールの抗酸化作用が、活性酸素による肌や血管の老化を抑える働きが期待されている
2. ホワイトチョコレートはカカオポリフェノールが最も豊富で美容効果が高い
3. チョコレートを食べるとビタミンAが体内で大量に生成される
4. カカオには紫外線を完全にブロックする成分が含まれている


<QuizAnswer>

**答え: 1**

カカオポリフェノール（フラバノール類）には強い抗酸化作用があり、活性酸素による細胞ダメージや肌の老化、血管の劣化を抑える働きが期待されています。一方、ホワイトチョコレートはカカオマスを含まないためポリフェノールはほとんど含まれません。なお、紫外線を「完全にブロック」するような成分は食品には存在せず、過大な期待は禁物です。

</QuizAnswer>
