## 問1

「チョコレート類の表示に関する公正競争規約」において、「チョコレート」と表示するために必要なカカオ分の最低含有量は何％以上と定められているか。

1. 21％以上
2. 35％以上
3. 50％以上
4. 70％以上


<QuizAnswer>

**答え: 2**

公正競争規約では、チョコレート生地のカカオ分は35％以上（うちココアバター18％以上）と規定されています。これに満たないものは「準チョコレート」として区別されます。なお、製品全体での「チョコレート」表示はカカオ分21％以上などの基準が適用されます。

</QuizAnswer>

## 問2

ホワイトチョコレートの主原料として正しい組み合わせはどれか。

1. カカオマス＋ココアバター＋砂糖
2. ココアバター＋砂糖＋乳成分
3. ココアパウダー＋砂糖＋乳成分
4. カカオマス＋砂糖＋乳成分


<QuizAnswer>

**答え: 2**

ホワイトチョコレートはカカオマス（茶色い固形分）を含まず、ココアバター・砂糖・乳成分を主原料とします。そのため白色をしており、カカオ由来の苦味やポリフェノールはほとんど含まれません。公正競争規約ではココアバター18％以上、乳固形分14％以上などが規定されています。

</QuizAnswer>

## 問3

カカオマスから分離して得られる、常温で固体・体温付近で溶ける性質をもつ植物性脂肪を何と呼ぶか。

1. ココアパウダー
2. ココアバター
3. 代用脂（CBE）
4. カカオニブ


<QuizAnswer>

**答え: 2**

ココアバターはカカオマスを圧搾して得られる淡黄色の植物性油脂で、約32〜35℃で融解する独特の性質を持ちます。この急峻な融解特性がチョコレートの口溶けの良さを生み出します。ココアパウダーは圧搾後の固形分を粉砕したものです。

</QuizAnswer>

## 問4

製菓用として広く用いられる「クーベルチュールチョコレート」の特徴として正しいものはどれか。

1. 乳固形分が極めて多く、コーティングに不向き
2. ココアバターを多く含み、流動性が高くコーティングや型抜きに適する
3. 砂糖を全く含まないビター専用の製品
4. 代用脂を主体とし、テンパリング不要


<QuizAnswer>

**答え: 2**

クーベルチュール（couverture）はフランス語で「覆うもの」を意味し、ココアバターを31％以上含み流動性が高いプロ用チョコレートを指します。コーティング・型抜き・ガナッシュなど幅広く使われ、テンパリングが必須です。国際規格ではカカオ分35％以上、無脂カカオ固形分2.5％以上などが定められています。

</QuizAnswer>

## 問5

ヘーゼルナッツやアーモンドのペーストを砂糖・カカオと練り合わせて作られる、滑らかな質感のチョコレート系製品を何と呼ぶか。

1. プラリネ
2. ガナッシュ
3. ジャンドゥーヤ
4. ボンボンショコラ


<QuizAnswer>

**答え: 3**

ジャンドゥーヤ（gianduja）はイタリア・ピエモンテ発祥で、ローストしたヘーゼルナッツのペーストにチョコレートと砂糖を加えた滑らかな製品です。ナッツ含有量は規定により決められており、独特のコクと香ばしさを持ちます。プラリネはナッツのキャラメリゼ系を指す場合が多いです。

</QuizAnswer>

## 問6

ガナッシュの基本的な構成として最も適切なものはどれか。

1. チョコレート＋生クリーム
2. チョコレート＋砂糖＋卵白
3. チョコレート＋ココアパウダー＋水
4. チョコレート＋バター＋小麦粉


<QuizAnswer>

**答え: 1**

ガナッシュはチョコレートと生クリーム（時にバターや洋酒を加える）を乳化させて作るクリーム状の生地で、ボンボンショコラのセンターやトリュフ、ケーキのフィリングに使われます。チョコレートと生クリームの比率により硬さや用途が変わります。

</QuizAnswer>

## 問7

「ボンボンショコラ」の説明として最も適切なものはどれか。

1. カカオ豆をそのまま砂糖でコーティングしたもの
2. ガナッシュやプラリネなどのセンターをチョコレートで包んだ一口サイズの菓子
3. 板チョコレートを薄く割った製品
4. チョコレートを使ったホットドリンク


<QuizAnswer>

**答え: 2**

ボンボンショコラ（bonbon de chocolat）は、ガナッシュ・プラリネ・キャラメル・リキュールなどのセンターをクーベルチュールで覆った一口サイズのチョコレート菓子です。ショコラティエの技術が最も問われる製品の一つで、形状や装飾も多彩です。

</QuizAnswer>

## 問8

ミルクチョコレートにおいて、生乳由来の風味やコク・色合いに寄与する原材料はどれか。

1. カカオマス
2. ココアバター
3. 乳固形分（全粉乳・脱脂粉乳など）
4. レシチン


<QuizAnswer>

**答え: 3**

ミルクチョコレートのまろやかな風味とミルキーな色合いは、全粉乳や脱脂粉乳などの乳固形分によってもたらされます。公正競争規約ではミルクチョコレートはカカオ分21％以上、乳固形分14％以上などが規定されています。レシチンは乳化剤として少量添加されます。

</QuizAnswer>

## 問9

チョコレートの「代用脂（CBE・CBR・CBSなど）」を使った製品の表示に関する説明として正しいものはどれか。

1. 代用脂を使用すれば必ず「チョコレート」と表示できる
2. 代用脂を一定量以上使用すると「準チョコレート」等として区別される場合がある
3. 代用脂はカカオの一部とみなされるため規制はない
4. 代用脂は風味改良目的のみで使われ、流通製品には存在しない


<QuizAnswer>

**答え: 2**

公正競争規約では、ココアバター以外の植物油脂（代用脂）を一定量以上使用した場合や規定のカカオ分に満たない場合、「準チョコレート」として区別されます。CBE（等価脂）・CBR（代替脂）・CBS（ラウリン系）等があり、コスト・耐熱性・加工適性のために使用されます。

</QuizAnswer>

## 問10

チョコレート類の公正競争規約におけるカカオ分の定義として正しいものはどれか。

1. カカオマスのみの量
2. ココアバターのみの量
3. カカオマスとココアバターの合計量（無脂カカオ固形分＋ココアバター）
4. ココアパウダーと砂糖の合計量


<QuizAnswer>

**答え: 3**

「カカオ分」とは、カカオ豆由来の成分すなわちカカオマス（無脂カカオ固形分）とココアバターを合わせた総量を指します。例えば「カカオ分70％」と表示されたダークチョコレートは、原料中の70％がカカオ由来成分であることを意味します。残りは主に砂糖や乳化剤などです。

</QuizAnswer>
