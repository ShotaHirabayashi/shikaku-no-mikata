## 問1

野菜の切り方のうち、「拍子木切り」の形状として正しいものはどれか。

1. 1cm角のサイコロ状に切る
2. 4〜5cm×1cm×1cmの棒状に切る
3. 野菜を鉛筆を削るように薄く削る
4. 丸い野菜を放射状に切る


<QuizAnswer>

**答え: 2**

拍子木切りは、野菜を4〜5cm×1cm×1cm程度の棒状に切る方法で、拍子木の形に似ていることからこの名前がつきました。選択肢1はさいの目切り、選択肢3はささがき、選択肢4はくし形切りの説明です。フライや煮物などに適した切り方です。

</QuizAnswer>

## 問2

緑色野菜を鮮やかな色のまま茹でるコツとして、**誤っている**ものはどれか。

1. たっぷりの湯で茹でる
2. 湯に塩を1〜2%加える
3. 鍋に蓋をして茹でる
4. 茹でたらすぐに冷水にとる


<QuizAnswer>

**答え: 3**

緑色野菜を茹でるときは蓋をしないのが正しい方法です。蓋をすると、野菜から出る有機酸が鍋の中にこもり、クロロフィル（葉緑素）が酸の影響でフェオフィチンに変化し、褐色に変色してしまいます。蓋をしないことで有機酸が蒸気とともに飛び、鮮やかな緑色を保つことができます。

</QuizAnswer>

## 問3

調味料の「さしすせそ」で、最初に入れるべきとされる調味料はどれか。

1. 塩
2. 酢
3. 砂糖
4. 醤油


<QuizAnswer>

**答え: 3**

「さしすせそ」の「さ」は砂糖を指し、最初に加えます。砂糖の分子（ショ糖、分子量342）は塩（分子量58.5）より大きく浸透しにくいため、先に入れて十分に染み込ませる必要があります。塩を先に入れると素材の細胞が脱水・収縮し、砂糖がさらに浸透しにくくなります。

</QuizAnswer>

## 問4

ほうれん草のアク抜きとして最も適切な方法はどれか。

1. 酢水に10分さらす
2. 米のとぎ汁で下茹でする
3. たっぷりの湯で茹でて冷水にとる
4. 重曹を入れた水に一晩浸ける


<QuizAnswer>

**答え: 3**

ほうれん草のアク（えぐみ）の主成分はシュウ酸です。シュウ酸は水溶性のため、たっぷりの湯で茹でることで湯に溶け出します。茹でた後は冷水にとることで、余熱による変色を防ぎ色鮮やかに仕上がります。選択肢2は大根、選択肢4はわらびなどの山菜に用いる方法です。

</QuizAnswer>

## 問5

脂溶性ビタミン（β-カロテンなど）を多く含むにんじんの調理法として、栄養の吸収率を最も高める方法はどれか。

1. 生のまますりおろす
2. たっぷりの水で茹でる
3. 油で炒める
4. 蒸し器で蒸す


<QuizAnswer>

**答え: 3**

β-カロテンは脂溶性のため、油と一緒に摂取すると吸収率が大幅に向上します。生食時の吸収率に比べ、油で調理した場合は3〜6倍に高まるとされています。茹でるや蒸すでも加熱による細胞壁の破壊で吸収率は上がりますが、油を使う炒め調理が最も効率的です。

</QuizAnswer>

## 問6

煮物を作る際に、野菜の角を薄く削り取る下処理を何というか。

1. かつらむき
2. 面取り
3. 乱切り
4. そぎ切り


<QuizAnswer>

**答え: 2**

面取り（めんとり）とは、煮物用の野菜の角を薄く削り取る下処理のことです。面取りをすることで、長時間煮ても煮崩れしにくくなり、見た目もきれいに仕上がります。かつらむきは大根などを薄く帯状にむく技法、乱切りは不規則な形に切る方法です。

</QuizAnswer>

## 問7

ごぼうの下処理として、変色を防ぎ白く仕上げるために最も適切な方法はどれか。

1. 塩水にさらす
2. 酢水にさらす
3. 砂糖水にさらす
4. 米のとぎ汁にさらす


<QuizAnswer>

**答え: 2**

ごぼうにはクロロゲン酸やタンニンなどのポリフェノールが含まれ、空気に触れると酸化して褐変します。酢水にさらすことで酸化反応を抑え、白く仕上げることができます。水だけでもアク抜きの効果はありますが、酢水の方がより白く仕上がります。

</QuizAnswer>

## 問8

加熱調理法のうち、水溶性ビタミンの損失が最も少ないのはどれか。

1. たっぷりの湯で茹でる
2. 蒸す
3. 水から煮る
4. 煮汁を多めにして煮る


<QuizAnswer>

**答え: 2**

蒸し調理は水蒸気で加熱するため、野菜が直接水に触れません。そのため、ビタミンCやビタミンB群などの水溶性ビタミンが水中に溶け出すことがなく、栄養素の損失が最も少ない加熱方法です。茹でるや煮る調理では、水溶性ビタミンが茹で汁や煮汁に流出してしまいます。

</QuizAnswer>

## 問9

調味料の「さしすせそ」の「せ」にあたる調味料はどれか。

1. 酢
2. 味噌
3. 醤油
4. 塩


<QuizAnswer>

**答え: 3**

「さしすせそ」の「せ」は醤油（しょうゆ）を指します。これは旧仮名遣いで醤油を「せうゆ」と書いたことに由来します。醤油は香りを活かすために調理の後半に加えるのが基本です。なお「そ」は味噌で、香りが飛びやすいため最後に加え、煮立たせないようにします。

</QuizAnswer>

## 問10

野菜を切るとき、繊維を断つ方向に切った場合の特徴として正しいものはどれか。

1. シャキシャキとした食感が残る
2. 柔らかく仕上がり、火が通りやすい
3. 煮崩れしにくくなる
4. 栄養素の流出が少なくなる


<QuizAnswer>

**答え: 2**

野菜の繊維を断つ方向に切ると、細胞壁が多く切断されるため柔らかい仕上がりになり、火も通りやすくなります。一方、繊維に沿って切るとシャキシャキとした食感が残ります。料理の目的に応じて繊維の方向を意識して切ることが、野菜の食感をコントロールする重要なポイントです。

</QuizAnswer>
