## 問1

厚生労働省が「健康日本21」で掲げている、成人の1日あたりの野菜摂取目標量はどれか。

1. 250g
2. 300g
3. 350g
4. 400g


<QuizAnswer>

**答え: 3**

厚生労働省は「健康日本21」において、生活習慣病の予防や健康の保持・増進を目的として、1日あたりの野菜摂取目標量を350gと定めています。しかし、実際の日本人の平均摂取量は約256gにとどまっており、目標に届いていないのが現状です。

</QuizAnswer>

## 問2

予防医学において、生活習慣の改善や健康教育などにより病気にかからないようにする取り組みを何というか。

1. 一次予防
2. 二次予防
3. 三次予防
4. 基礎予防


<QuizAnswer>

**答え: 1**

一次予防とは、生活習慣の改善、健康教育、予防接種などにより、病気の発症そのものを防ぐ取り組みのことです。二次予防は早期発見・早期治療、三次予防はリハビリテーションや再発防止を指します。生活習慣病対策では、この一次予防が最も重要とされています。

</QuizAnswer>

## 問3

「食事バランスガイド」のイラストに使われている形はどれか。

1. ピラミッド
2. コマ（回転するおもちゃ）
3. お皿
4. 三角形


<QuizAnswer>

**答え: 2**

食事バランスガイドは「コマ」の形をイラストに用いています。食事のバランスが悪くなるとコマが倒れてしまうことを表現しており、コマの回転は「運動」、コマの軸は「水・お茶」を意味しています。バランスのとれた食事と運動の両方が大切であるというメッセージが込められています。

</QuizAnswer>

## 問4

メタボリックシンドロームの診断において、日本の基準で必須条件となるのはどれか。

1. BMIが25以上であること
2. 血圧が140/90mmHg以上であること
3. 腹囲が基準値以上であること（内臓脂肪の蓄積）
4. 空腹時血糖値が126mg/dl以上であること


<QuizAnswer>

**答え: 3**

日本のメタボリックシンドロームの診断基準では、腹囲（ウエスト周囲径）が男性85cm以上・女性90cm以上であることが必須条件です。この必須条件に加えて、血圧・血糖・脂質の3項目のうち2つ以上が基準値から外れている場合にメタボリックシンドロームと診断されます。

</QuizAnswer>

## 問5

日本人の死因の上位を占める「がん」「心疾患」「脳血管疾患」をまとめて何と呼ぶか。

1. 三大成人病
2. 三大生活習慣病
3. 三大慢性疾患
4. 三大内臓疾患


<QuizAnswer>

**答え: 2**

がん（悪性新生物）、心疾患（心筋梗塞など）、脳血管疾患（脳卒中など）は、日本人の死因の上位を占めており、「三大生活習慣病」と呼ばれています。いずれも食習慣や運動習慣、喫煙、飲酒などの生活習慣が発症・進行に深く関わっている疾患です。

</QuizAnswer>

## 問6

「食事バランスガイド」で食事内容を5つの料理区分に分けているが、次のうち含まれないものはどれか。

1. 副菜
2. 主菜
3. 汁物
4. 果物


<QuizAnswer>

**答え: 3**

食事バランスガイドの5つの料理区分は「主食」「副菜」「主菜」「牛乳・乳製品」「果物」です。汁物は独立した区分としては設定されていません。汁物に含まれる野菜などは、使われている食材に応じて副菜などにカウントされます。

</QuizAnswer>

## 問7

健康日本21における1日の野菜摂取目標量350gのうち、緑黄色野菜の目標量はどれか。

1. 70g以上
2. 120g以上
3. 150g以上
4. 200g以上


<QuizAnswer>

**答え: 2**

1日の野菜摂取目標量350gの内訳は、緑黄色野菜が120g以上、その他の野菜（淡色野菜）が230g以上とされています。緑黄色野菜にはβ-カロテンをはじめとするビタミンやミネラルが豊富に含まれており、意識的に摂取することが推奨されています。

</QuizAnswer>

## 問8

予防医学において、健康診断やがん検診などで病気を早期に見つけ、早期に治療する段階を何というか。

1. 一次予防
2. 二次予防
3. 三次予防
4. 事後予防


<QuizAnswer>

**答え: 2**

二次予防とは、病気を早期に発見し、早期に治療を行うことで重症化を防ぐ取り組みです。定期的な健康診断やがん検診、特定健康診査（メタボ健診）などが代表例です。一次予防が「病気にならないようにする」段階であるのに対し、二次予防は「病気を早く見つけて対処する」段階にあたります。

</QuizAnswer>

## 問9

メタボリックシンドロームの日本の診断基準において、腹囲（ウエスト周囲径）の基準値の組み合わせとして正しいものはどれか。

1. 男性80cm以上・女性80cm以上
2. 男性90cm以上・女性80cm以上
3. 男性85cm以上・女性90cm以上
4. 男性90cm以上・女性90cm以上


<QuizAnswer>

**答え: 3**

日本のメタボリックシンドロームの診断基準では、腹囲が男性85cm以上・女性90cm以上とされています。これは内臓脂肪面積100cm²以上に相当します。国際的な基準とは男女の値が逆転している点が特徴的で、日本独自の基準として設定されています。

</QuizAnswer>

## 問10

生活習慣病の予防において、食生活を見直すために自分の食事内容を記録する方法は一般的に何と呼ばれるか。

1. 食事カウンセリング
2. フードファディズム
3. 食事記録（食事日記）
4. 栄養アセスメント


<QuizAnswer>

**答え: 3**

食事記録（食事日記）は、自分が食べたものを記録することで、食生活の偏りや問題点を客観的に把握し、改善に役立てる方法です。何をどれだけ食べているかを「見える化」することで、食事バランスの見直しにつながります。フードファディズムは食べものや栄養素の効果を過大に信じてしまう偏った考え方を指す別の概念です。

</QuizAnswer>
