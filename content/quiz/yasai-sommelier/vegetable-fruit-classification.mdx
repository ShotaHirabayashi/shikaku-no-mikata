## 問1

農林水産省の定義において、いちご・メロン・すいかはどのように分類されるか。

1. 果物（果樹）
2. 果実的野菜
3. 穀物
4. 特用作物


<QuizAnswer>

**答え: 2**

いちご・メロン・すいかは草本性の植物であるため、生産統計上は「野菜」に分類されます。しかし、流通や消費の場面では果物として扱われることが多いため、「果実的野菜」と呼ばれます。農林水産省の定義では、果物（果樹）は2年以上栽培する永年性の木本性植物から収穫される果実とされており、草本性のこれらは野菜に該当します。

</QuizAnswer>

## 問2

現在、日本で最も流通量が多い大根の品種はどれか。

1. 三浦大根
2. 聖護院大根
3. 青首大根（宮重系）
4. 練馬大根


<QuizAnswer>

**答え: 3**

青首大根（宮重系）は現在流通している大根の約90%を占める主力品種です。上部が緑色になるのが特徴で、辛みが穏やかなため、煮物・サラダ・おろしなど幅広い料理に使える万能型の大根です。三浦大根や練馬大根はかつて主流でしたが、現在は生産量が大幅に減少し、伝統野菜として扱われています。

</QuizAnswer>

## 問3

野菜ソムリエ協会が提唱する「氏育ちころたて」フレームワークにおいて、「氏（うじ）」が意味するものはどれか。

1. 栽培環境や栽培方法
2. 品種や来歴
3. 個々の青果物の状態
4. 適した調理法や食べ方


<QuizAnswer>

**答え: 2**

「氏育ちころたて」は野菜・果物の品質を多角的に評価するフレームワークです。「氏（うじ）」はその青果物の品種・来歴・ブランドなど、いわば素性を意味します。「育ち」は栽培環境や産地、「こ」は個体差、「ろ」は五感＋第六感による評価、「た」は食べ方・調理法、「て」は手当て・保存を表しています。

</QuizAnswer>

## 問4

りんごの品種のうち、日本で最も生産量が多く、全体の約50%を占める品種はどれか。

1. つがる
2. 王林
3. ジョナゴールド
4. ふじ


<QuizAnswer>

**答え: 4**

ふじは「国光」と「デリシャス」の交配によって生まれた品種で、日本のりんご生産量の約50%を占めます。甘みが強く酸味が少ない、蜜が入りやすい、貯蔵性が高いといった特徴があり、収穫期は11月頃です。世界的にも最も多く生産されているりんご品種の一つです。

</QuizAnswer>

## 問5

「野菜生産出荷安定法」に基づき、国民生活上特に重要な野菜として国が指定している「指定野菜」に含まれないものはどれか。

1. ブロッコリー
2. きゅうり
3. ごぼう
4. ほうれんそう


<QuizAnswer>

**答え: 3**

ごぼうは指定野菜ではなく「特定野菜」に分類されています。指定野菜は消費量が多く国民生活上重要な野菜として国が指定したもので、価格が著しく低落した場合に生産者補給金が交付される制度の対象です。2026年度からブロッコリーが新たに指定野菜に追加され、現在は15品目となっています。

</QuizAnswer>

## 問6

京野菜のうち、丸型で大型、肉質が緻密で田楽に最適とされるなすの品種はどれか。

1. 水なす
2. 米なす
3. 賀茂なす
4. 千両なす


<QuizAnswer>

**答え: 3**

賀茂なすは京都の伝統野菜で、丸型で300g以上にもなる大型のなすです。肉質が非常に緻密で油との相性が抜群であり、田楽が定番料理として知られています。水なすは大阪・泉州地域の品種で生食が可能、米なすはアメリカ品種由来でヘタが緑色なのが特徴です。

</QuizAnswer>

## 問7

果物の分類において、りんご・なし・びわなど、花托（花床）が発達して果実となるものを何類と呼ぶか。

1. 核果類
2. 漿果類
3. 殻果類
4. 仁果類


<QuizAnswer>

**答え: 4**

仁果類（じんかるい）は花托が発達して果肉となり、芯の部分に種子がある果実です。りんご、なし（日本梨・西洋梨）、びわ、かりんなどが該当します。核果類はもも・さくらんぼなど硬い核を持つもの、漿果類はぶどう・ブルーベリーなど果皮が柔らかく果汁が多いもの、殻果類はくり・くるみなど硬い殻に包まれたものです。

</QuizAnswer>

## 問8

大根の部位による味の違いについて、正しい説明はどれか。

1. 上部（葉に近い方）が最も辛い
2. 下部（先端）が最も甘い
3. 上部（葉に近い方）が甘く、下部（先端）が辛い
4. 部位による味の差はほとんどない


<QuizAnswer>

**答え: 3**

大根は部位によって味が異なります。上部（葉に近い方）は甘みがありサラダや生食に向いています。中部はバランスが良く煮物やおでんに適しています。下部（先端）は辛みが強く、大根おろしや薬味に使うのが適しています。この知識は料理に合わせた使い分けの基本として重要です。

</QuizAnswer>

## 問9

農林水産省の定義において、野菜と果物を区別する主な基準として正しいものはどれか。

1. 甘みがあるかどうか
2. 草本性か木本性か（草か木か）
3. 生食するか加熱するか
4. 種子があるかないか


<QuizAnswer>

**答え: 2**

農林水産省の定義では、野菜は「田畑に栽培される草本性の植物」であり、果物（果樹）は「2年以上栽培する永年性の木本性植物から収穫される果実」とされています。つまり、草本性（草）か木本性（木）かが最も基本的な区別基準です。この基準により、甘くても草本性のいちごやメロンは野菜に分類されます。

</QuizAnswer>

## 問10

りんごの品種のうち、酸味が強く製菓（アップルパイなど）に最も適しているとされる品種はどれか。

1. 王林
2. シナノスイート
3. 紅玉
4. ふじ


<QuizAnswer>

**答え: 3**

紅玉（こうぎょく）は酸味が強く、小玉で真っ赤な外見が特徴のりんご品種です。その酸味と香りの強さから、アップルパイやタルトなどの製菓用途に最適とされています。王林は酸味が少なく甘い黄緑色の品種、シナノスイートは甘みが強く食べやすい品種、ふじは甘みが強く酸味が少ない品種で、いずれも生食向きです。

</QuizAnswer>
