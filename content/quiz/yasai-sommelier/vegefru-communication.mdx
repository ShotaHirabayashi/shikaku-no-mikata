## 問1

野菜ソムリエの使命として、日本野菜ソムリエ協会が掲げている役割はどれか。

1. 野菜の品種改良を推進すること
2. 生産者と生活者の架け橋となること
3. 農産物の輸出を拡大すること
4. 飲食店の経営を支援すること


<QuizAnswer>

**答え: 2**

野菜ソムリエの使命は「生産者と生活者の架け橋」となることです。日本野菜ソムリエ協会は、野菜・果物の魅力や価値を社会に広めるスペシャリストを育成することを目的としており、生産側と消費側をつなぐコミュニケーターとしての役割が重視されています。

</QuizAnswer>

## 問2

ベジフルコミュニケーションの講座で主に養うことを目的としている力はどれか。

<QuizAnswer>

</QuizAnswer>

## 問3

コミュニケーションにおいて、言語情報・聴覚情報・視覚情報が矛盾した場合の影響度を示した「メラビアンの法則」で、最も影響が大きいとされるのはどれか。

1. 言語情報（話の内容）
2. 聴覚情報（声のトーンや話し方）
3. 触覚情報（握手や身体接触）
4. 視覚情報（表情や態度）


<QuizAnswer>

**答え: 4**

メラビアンの法則では、言語・聴覚・視覚の情報が矛盾している場合、視覚情報（表情や態度）が55%、聴覚情報（声のトーンや話し方）が38%、言語情報（話の内容）が7%の割合で印象に影響するとされています。ただし、これはすべてのコミュニケーション場面に当てはまるわけではなく、矛盾がある場合に限定された研究結果です。

</QuizAnswer>

## 問4

野菜ソムリエとして野菜・果物の魅力を伝える際、最も重要とされるコミュニケーションの姿勢はどれか。

1. 専門用語を多用して知識の豊富さを示す
2. 相手に応じて、受け取りやすいように情報を伝える
3. できるだけ多くの情報を一度に伝える
4. 自分の意見を強く主張して説得する


<QuizAnswer>

**答え: 2**

ベジフルコミュニケーションでは、「相手に応じて、適切な情報を相手が受け取りやすいように伝える」ことが基本姿勢とされています。専門知識をそのまま伝えるのではなく、相手の立場や知識レベルに合わせてわかりやすく伝えることが、野菜ソムリエに求められるコミュニケーション力です。

</QuizAnswer>

## 問5

メラビアンの法則における3つのV（3Vの法則）の正しい組み合わせはどれか。

1. Visual（視覚）・Verbal（言語）・Vocal（聴覚）
2. Visual（視覚）・Value（価値）・Voice（声）
3. Verbal（言語）・Virtual（仮想）・Vocal（聴覚）
4. Volume（音量）・Verbal（言語）・Visual（視覚）


<QuizAnswer>

**答え: 1**

メラビアンの法則は「3Vの法則」とも呼ばれ、Visual（視覚情報＝55%）、Vocal（聴覚情報＝38%）、Verbal（言語情報＝7%）の3つの頭文字に由来しています。コミュニケーションでは言葉の内容だけでなく、見た目や声のトーンも重要な要素であることを示しています。

</QuizAnswer>

## 問6

野菜・果物の魅力を人に伝えるプレゼンテーションにおいて、聞き手の理解を深めるために最も効果的な方法はどれか。

1. 資料を配布せず口頭のみで説明する
2. 自分が実際に食べて感じた体験や感動を交えて伝える
3. 統計データだけを並べて客観性を重視する
4. 聞き手の反応を見ずに一方的に話し続ける


<QuizAnswer>

**答え: 2**

野菜ソムリエのコミュニケーションでは、単なる知識の羅列ではなく、自分自身が実際に食べて感じた体験や感動を言葉にして伝えることが重視されています。体験に基づく表現は聞き手の共感を得やすく、野菜・果物の魅力をより効果的に伝えることができます。

</QuizAnswer>

## 問7

コミュニケーションにおける「傾聴」の姿勢として、適切でないものはどれか。

1. 相手の話にうなずきながら聴く
2. 相手の言葉を繰り返して確認する
3. 相手の話を遮って自分の意見を述べる
4. 相手の気持ちに共感を示す


<QuizAnswer>

**答え: 3**

傾聴とは、相手の話に真剣に耳を傾け、理解しようとする姿勢のことです。うなずきや相づち、オウム返し（相手の言葉の繰り返し）、共感の表現は傾聴の基本技法です。相手の話を遮って自分の意見を述べることは、傾聴の姿勢に反し、信頼関係の構築を妨げます。

</QuizAnswer>

## 問8

第一印象が形成されるまでの時間として、一般的に言われているのはどれか。

1. 約1時間
2. 約30分
3. 約3〜5秒
4. 約10分


<QuizAnswer>

**答え: 3**

第一印象は出会ってからわずか3〜5秒程度で形成されるといわれています。野菜ソムリエとして活動する際にも、最初の数秒間の表情・身だしなみ・あいさつが相手に与える印象を大きく左右します。そのため、第一印象を意識したコミュニケーションが重要です。

</QuizAnswer>

## 問10

プレゼンテーションで「PREP法」を用いる場合、正しい構成順序はどれか。

1. 例示（Example）→ 結論（Point）→ 理由（Reason）→ 再結論（Point）
2. 理由（Reason）→ 例示（Example）→ 結論（Point）→ 再結論（Point）
3. 結論（Point）→ 理由（Reason）→ 例示（Example）→ 再結論（Point）
4. 結論（Point）→ 例示（Example）→ 理由（Reason）→ 再結論（Point）


<QuizAnswer>

**答え: 3**

PREP法は、Point（結論）→ Reason（理由）→ Example（具体例）→ Point（再結論）の順に話を組み立てるプレゼンテーション技法です。最初に結論を述べることで聞き手が要点を把握しやすくなり、理由と具体例で説得力を高め、最後に結論を繰り返すことで印象に残るプレゼンテーションが可能になります。

</QuizAnswer>
