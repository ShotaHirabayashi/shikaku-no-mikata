## 問1

有機JAS認定を受けた農産物に関する記述として、正しいものはどれか。

1. 化学合成農薬を一切使用せず、天然由来の資材も使用できない
2. 播種・植付け前2年以上、化学合成農薬と化学肥料を使用していない圃場で生産される
3. 登録認定機関の審査は初回のみで、その後の検査は不要である
4. 認定を受けなくても、実態が有機栽培であれば「有機」と表示してよい


<QuizAnswer>

**答え: 2**

有機JAS規格では、播種・植付け前2年以上（多年生作物は収穫前3年以上）化学合成農薬・化学肥料を使用していない圃場で生産することが求められます。ただし、有機JAS別表に掲載されたボルドー液や硫黄剤など天然由来の資材は使用可能です。また、登録認定機関による年1回以上の検査が必要であり、認定なしに「有機」「オーガニック」と表示することはJAS法違反となります。

</QuizAnswer>

## 問2

2003年の改正により廃止された農産物の表示として、正しいものはどれか。

1. 「特別栽培農産物」
2. 「有機栽培農産物」
3. 「無農薬」「減農薬」
4. 「地域ブランド農産物」


<QuizAnswer>

**答え: 3**

2003年5月に改正・施行された「特別栽培農産物に係る表示ガイドライン」により、「無農薬」「減農薬」「無化学肥料」「減化学肥料」の表示は廃止されました。これらは消費者に誤解を与えやすいとして、「特別栽培農産物」に一本化されました。特別栽培農産物は、節減対象農薬の使用回数と化学肥料（窒素成分）の使用量がともに地域の慣行レベルの5割以下であることが基準です。

</QuizAnswer>

## 問3

食品表示法（2015年施行）は、それ以前の3つの法律の食品表示に関する規定を統合して制定された。統合された法律の組み合わせとして正しいものはどれか。

1. 食品安全基本法・JAS法・景品表示法
2. 食品衛生法・JAS法・健康増進法
3. 食品安全基本法・食品衛生法・景品表示法
4. 食品衛生法・健康増進法・景品表示法


<QuizAnswer>

**答え: 2**

食品表示法は2015年4月1日に施行され、食品衛生法（アレルゲン・添加物等の安全性に関する表示）、JAS法（原材料・原産地等の品質に関する表示）、健康増進法（栄養成分表示等）の3つの法律の食品表示に関する規定を一元化したものです。これにより、消費者にとってわかりやすい表示制度が整備されました。

</QuizAnswer>

## 問4

食品表示法に基づくアレルギー物質の表示義務に関する記述として、正しいものはどれか。

1. 特定原材料（義務表示）は5品目である
2. くるみは2025年4月から特定原材料（義務表示）に追加された
3. 特定原材料に準ずるもの（推奨表示）は10品目である
4. そばは推奨表示であり、表示義務はない


<QuizAnswer>

**答え: 2**

くるみは従来「特定原材料に準ずるもの（推奨表示）」でしたが、アレルギー症例の増加を受けて2023年3月に特定原材料に格上げされ、経過措置期間を経て2025年4月1日から完全義務化されました。現在、特定原材料（義務表示）は、えび・かに・くるみ・小麦・そば・卵・乳・落花生の8品目です。特定原材料に準ずるもの（推奨表示）は20品目となっています。

</QuizAnswer>

## 問5

2023年4月から施行された遺伝子組み換え食品の新しい表示制度に関する記述として、正しいものはどれか。

1. 意図せざる混入が5%以下であれば「遺伝子組換えでない」と表示できる
2. 分別生産流通管理を行い、遺伝子組み換えが不検出の場合のみ「遺伝子組換えでない」と表示できる
3. すべての農産物について遺伝子組み換え表示が義務付けられた
4. 「遺伝子組換え不分別」の表示義務は撤廃された


<QuizAnswer>

**答え: 2**

2023年4月施行の新制度では、「遺伝子組換えでない」と表示するための条件が厳格化されました。旧制度では意図せざる混入が5%以下であれば「でない」表示が可能でしたが、新制度ではIPハンドリング（分別生産流通管理）を実施し、かつ遺伝子組み換えが「不検出」の場合のみ表示が認められます。IPハンドリング済みで混入の可能性がある場合は「分別生産流通管理済み」等の任意表示が可能です。

</QuizAnswer>

## 問6

卸売市場における「仲卸業者」の主な役割として、最も適切なものはどれか。

1. 産地の出荷者から青果物を受託し、セリや相対取引で販売する
2. 目利きによる品質評価、品揃え、小分けなどの機能を担い、実需者に販売する
3. 市場の開設・運営を行い、取引のルールを管理する
4. 産地から市場への輸送を専門に行う


<QuizAnswer>

**答え: 2**

卸売市場には卸売業者（荷受）と仲卸業者という2つの主要な業者がいます。卸売業者は産地側の代理人として出荷者から受託しセリや相対取引で販売する役割を担います。一方、仲卸業者は実需者（小売店・飲食店等）側の代理人として、目利きによる品質評価、多様な品揃え、小分けなどの機能を果たし、買い手のニーズに合わせた販売を行います。

</QuizAnswer>

## 問7

生鮮食品の原産地表示に関する記述として、正しいものはどれか。

1. 国産品は「国産」とのみ表示すればよく、都道府県名の表示は不要である
2. 国産品は都道府県名、輸入品は原産国名を表示する
3. 輸入品は輸出港の名称を表示する
4. 生鮮食品には原産地表示の義務はない


<QuizAnswer>

**答え: 2**

食品表示基準に基づき、生鮮食品には原産地の表示が義務付けられています。国産品は都道府県名（市町村名やその他一般に知られた地名でも可）、輸入品は原産国名を表示します。なお、2022年4月からは全加工食品についても、重量割合上位1位の原材料に対する原料原産地表示が義務化されています。

</QuizAnswer>

## 問8

特別栽培農産物の基準に関する記述として、正しいものはどれか。

1. 化学合成農薬を一切使用していないことが条件である
2. 化学肥料の使用量が慣行レベルの3割以下であること
3. 節減対象農薬の使用回数と化学肥料（窒素成分）の使用量がともに慣行レベルの5割以下であること
4. 農林水産大臣による認定が必要である


<QuizAnswer>

**答え: 3**

特別栽培農産物は、節減対象農薬の使用回数が地域の慣行レベルの5割以下、かつ化学肥料（窒素成分）の使用量が地域の慣行レベルの5割以下の両方を満たす農産物です。有機JASのように完全不使用を求めるものではなく、また認定機関による認定制度でもありません。「慣行レベル」は各地域の都道府県が定めた基準に基づいています。

</QuizAnswer>

## 問9

2017年の改正JAS法（新JAS制度）に関する記述として、正しいものはどれか。

1. JAS規格の対象が産品のみからプロセスや試験方法等にも拡大された
2. すべての食品にJASマークの表示が義務化された
3. 有機JAS認定制度が廃止された
4. JAS規格の制定提案は行政機関のみが行えることになった


<QuizAnswer>

**答え: 1**

2017年のJAS法改正（新JAS制度）では、JAS規格の対象がそれまでの「モノ（産品）」だけでなく、「プロセス（取扱方法・生産方法）」「試験方法」「事業者」にまで拡大されました。また、民間事業者からの規格制定提案の仕組みが導入されました。JAS規格への適合は原則として任意制度ですが、有機農産物に「有機」「オーガニック」と表示する場合は有機JASマークが必須です。

</QuizAnswer>

## 問10

加工食品の原料原産地表示制度（2022年4月完全施行）に関する記述として、正しいものはどれか。

1. 原料原産地の表示が必要なのは22食品群に限定されている
2. すべての原材料について原産地表示が必要である
3. 重量割合上位1位の原材料について原産地表示が義務付けられている
4. 外食やテイクアウトにも原料原産地表示が義務付けられている


<QuizAnswer>

**答え: 3**

2022年4月に完全施行された新たな原料原産地表示制度では、対象が従来の22食品群＋4品目から全加工食品に拡大されました。ただし、表示が必要なのは重量割合上位1位の原材料についてです。表示方法は国別重量順表示が原則ですが、産地が頻繁に変わる場合は「又は表示」や「大括り表示」といった例外的表示方法も認められています。なお、外食・テイクアウト・インストア加工は表示義務の対象外です。

</QuizAnswer>
