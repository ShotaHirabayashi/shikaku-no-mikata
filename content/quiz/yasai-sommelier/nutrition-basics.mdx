## 問1

ビタミンは水への溶けやすさによって2つに分類されます。脂溶性ビタミンに該当するものとして正しい組み合わせはどれですか。

1. ビタミンA・ビタミンC・ビタミンE・ビタミンK
2. ビタミンA・ビタミンD・ビタミンE・ビタミンK
3. ビタミンB1・ビタミンD・ビタミンE・ビタミンK
4. ビタミンA・ビタミンD・ビタミンB12・ビタミンK


<QuizAnswer>

**答え: 2**

脂溶性ビタミンはビタミンA・D・E・Kの4種類です。脂溶性ビタミンは水に溶けず油脂に溶ける性質があり、主に脂肪組織や肝臓に貯蔵されます。摂りすぎると体内に蓄積されて過剰症を引き起こすことがあるため、適量の摂取が重要です。

</QuizAnswer>

## 問2

ビタミンCの特性として、正しいものはどれですか。

1. 脂溶性ビタミンであり、油と一緒に摂ると吸収率が高まる
2. 体内に蓄積されやすく、過剰摂取に注意が必要である
3. 水溶性ビタミンであり、コラーゲンの生成を助け、抗酸化作用を持つ
4. 血液凝固に不可欠な役割を果たす


<QuizAnswer>

**答え: 3**

ビタミンCは水溶性ビタミンに分類され、皮膚や粘膜の健康維持を助けるコラーゲンの生成に不可欠です。また、強い抗酸化作用を持ち、活性酸素から体を守る働きがあります。水溶性のため体内に蓄積されにくく、余分なものは尿中に排出されます。なお、血液凝固に関わるのはビタミンKです。

</QuizAnswer>

## 問3

五大栄養素のうち、体のエネルギー源となる「三大栄養素（エネルギー産生栄養素）」の正しい組み合わせはどれですか。

1. 炭水化物・脂質・ビタミン
2. たんぱく質・ミネラル・脂質
3. 炭水化物・脂質・たんぱく質
4. 炭水化物・ビタミン・ミネラル


<QuizAnswer>

**答え: 3**

五大栄養素は炭水化物・脂質・たんぱく質・ビタミン・ミネラルの5つです。このうちエネルギー源となるのは炭水化物・脂質・たんぱく質の三大栄養素です。ビタミンとミネラルはエネルギー源にはなりませんが、体の機能を調節し調子を整える重要な役割を果たします。

</QuizAnswer>

## 問4

食物繊維の種類について、正しい記述はどれですか。

1. 食物繊維はすべて水に溶ける性質を持っている
2. 水溶性食物繊維はゲル状になり、糖の吸収速度を遅くする作用がある
3. 不溶性食物繊維はコレステロールの吸収を阻害する作用がある
4. 食物繊維は人の消化酵素で完全に消化される


<QuizAnswer>

**答え: 2**

食物繊維は水溶性と不溶性の2種類に分けられます。水溶性食物繊維は水に溶けてゲル状になり、糖の消化吸収スピードを遅くして急激な血糖値の上昇を抑える作用があります。コレステロールの吸収を阻害するのも水溶性食物繊維の働きです。食物繊維は人の消化酵素では消化できない成分であることが大きな特徴です。

</QuizAnswer>

## 問5

フィトケミカル（ファイトケミカル）について、正しい記述はどれですか。

1. 動物性食品に多く含まれる化学物質の総称である
2. 植物が紫外線や害虫などから身を守るために作り出す化学物質である
3. ビタミンやミネラルと同じく、五大栄養素の一つである
4. 抗酸化作用はなく、主に味や食感に関わる成分である


<QuizAnswer>

**答え: 2**

フィトケミカルとは、野菜・果物・豆類・海藻などの植物性食品に含まれる色素・香り・アクなどの成分から発見された化学物質の総称です。植物が紫外線、有害物質、害虫などの外敵から自身を守るために作り出す物質であり、強い抗酸化作用を持つことが特徴です。五大栄養素には含まれませんが、「第7の栄養素」として注目されています。

</QuizAnswer>

## 問6

にんじんやかぼちゃなどの緑黄色野菜に多く含まれ、体内でビタミンAに変換される物質はどれですか。

1. リコピン
2. アントシアニン
3. β-カロテン
4. カテキン


<QuizAnswer>

**答え: 3**

β-カロテンはカロテノイドの一種で、にんじん・かぼちゃ・ほうれん草などの緑黄色野菜に多く含まれています。体内で必要な量だけビタミンAに変換されるため「プロビタミンA」とも呼ばれます。リコピンはトマトの赤い色素、アントシアニンはブルーベリーなどの紫色の色素、カテキンはお茶に含まれるポリフェノールで、いずれもビタミンAには変換されません。

</QuizAnswer>

## 問7

水溶性ビタミンの特性として、正しいものはどれですか。

1. 体内の脂肪組織に蓄積されるため、毎日摂取する必要はない
2. 過剰に摂取すると体内に蓄積され、過剰症を起こしやすい
3. 余分に摂取した分は尿中に排出されるため、こまめな摂取が望ましい
4. 油と一緒に調理すると吸収率が大幅に向上する


<QuizAnswer>

**答え: 3**

水溶性ビタミンは血液などの体液に溶け込んで働き、余分なものは尿として体外に排出されます。そのため体内に蓄積されにくく、過剰症のリスクは低い反面、不足しやすいため毎日こまめに摂取することが大切です。一方、脂肪組織に蓄積されるのは脂溶性ビタミン（A・D・E・K）の特性であり、油との調理で吸収率が上がるのも脂溶性ビタミンです。

</QuizAnswer>

## 問8

ミネラル（無機質）の一つであるカリウムの主な働きとして、正しいものはどれですか。

1. 骨や歯の主成分となる
2. 体内の余分なナトリウム（塩分）の排出を促す
3. 赤血球中のヘモグロビンの構成成分となる
4. 血液凝固に必要な因子を活性化する


<QuizAnswer>

**答え: 2**

カリウムは野菜や果物に豊富に含まれるミネラルで、体内の余分なナトリウム（塩分）を排出する作用があり、高血圧の予防に役立ちます。骨や歯の主成分となるのはカルシウム、ヘモグロビンの構成成分は鉄、血液凝固因子の活性化はカルシウムやビタミンKの働きです。

</QuizAnswer>

## 問9

ポリフェノールの説明として、正しいものはどれですか。

1. 動物性食品にのみ含まれるたんぱく質の一種である
2. 植物性食品に含まれ、強い抗酸化作用を持つフィトケミカルの一種である
3. 脂溶性ビタミンの一種で、ビタミンEと同じ働きをする
4. 人の消化酵素で消化できない食物繊維の一種である


<QuizAnswer>

**答え: 2**

ポリフェノールは植物性食品に広く含まれるフィトケミカルの代表的なグループで、強い抗酸化作用を持っています。ブルーベリーなどに含まれるアントシアニン、お茶に含まれるカテキン、大豆に含まれるイソフラボンなどが代表例です。ポリフェノールはビタミンでも食物繊維でもなく、植物が作り出す機能性成分です。

</QuizAnswer>

## 問10

ビタミンCと鉄の吸収に関する記述として、正しいものはどれですか。

1. ビタミンCは鉄の吸収を妨げるため、同時に摂取しないほうがよい
2. ビタミンCは非ヘム鉄を還元して吸収されやすい形に変え、鉄の吸収を促進する
3. 鉄はビタミンCを分解するため、同時摂取するとビタミンCの効果が失われる
4. 鉄の吸収にはビタミンDが必要であり、ビタミンCは関係がない


<QuizAnswer>

**答え: 2**

植物性食品に含まれる非ヘム鉄（三価鉄）はそのままでは吸収率が低いのですが、ビタミンCが三価鉄を二価鉄に還元することで吸収されやすい形に変わり、鉄の吸収率が高まります。これは栄養素の相互作用の代表的な例です。ほうれん草（鉄）とレモン（ビタミンC）の組み合わせなどが実践的に推奨されています。

</QuizAnswer>
