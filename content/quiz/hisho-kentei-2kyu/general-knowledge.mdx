## 問1

「M&A」の説明として適当なものを選びなさい。

1. 経営資源の選択と集中により、不採算部門を整理すること
2. 企業が市場調査に基づいて新商品を開発すること
3. 企業の合併と買収のこと
4. 企業間で技術提携の契約を結ぶこと


<QuizAnswer>

**答え: 3**

M&Aは「Mergers and Acquisitions」の略で、企業の合併（Mergers）と買収（Acquisitions）を意味します。企業の成長戦略や事業再編の手段として広く用いられており、秘書検定の一般知識では頻出の経営用語です。

</QuizAnswer>

## 問2

「コンプライアンス」の意味として適当なものを選びなさい。

1. 法令遵守
2. 企業統治
3. 情報公開
4. 危機管理


<QuizAnswer>

**答え: 1**

コンプライアンス（compliance）は「法令遵守」を意味し、企業が法律や社会規範を守って活動することを指します。なお、選択肢2の「企業統治」はコーポレートガバナンス、選択肢3の「情報公開」はディスクロージャー、選択肢4の「危機管理」はリスクマネジメントに該当します。

</QuizAnswer>

## 問3

固定費と変動費の合計が売上高と等しくなり、利益も損失もゼロになる売上高の地点を何というか。

1. 限界利益
2. 粗利益率
3. 経常利益
4. 損益分岐点


<QuizAnswer>

**答え: 4**

損益分岐点（BEP: Break-Even Point）とは、売上高と総費用（固定費＋変動費）が一致し、利益がちょうどゼロになる地点のことです。売上高が損益分岐点を上回れば黒字、下回れば赤字となります。経営分析の基本用語として秘書検定でもよく出題されます。

</QuizAnswer>

## 問4

株式会社における最高意思決定機関はどれか。

1. 取締役会
2. 株主総会
3. 監査役会
4. 代表取締役


<QuizAnswer>

**答え: 2**

株式会社の最高意思決定機関は株主総会です。株主総会では、取締役の選任・解任、定款の変更、決算の承認など、会社の重要事項が決議されます。取締役会は業務執行の意思決定機関であり、最高意思決定機関とは区別されます。

</QuizAnswer>

## 問5

マーケティングにおける「4P」に含まれないものはどれか。

1. Product（製品）
2. Person（人材）
3. Price（価格）
4. Promotion（販売促進）


<QuizAnswer>

**答え: 2**

マーケティングの4Pとは、Product（製品）、Price（価格）、Place（流通・販売経路）、Promotion（販売促進）の4つを指します。Person（人材）は4Pには含まれません。4Pはマーケティング戦略を立案する際の基本的なフレームワークとして広く知られています。

</QuizAnswer>

## 問6

「ROE」の説明として適当なものを選びなさい。

1. 総資産に対する利益の割合を示す指標
2. 売上高に対する営業利益の割合を示す指標
3. 自己資本に対する利益の割合を示す指標
4. 負債に対する自己資本の割合を示す指標


<QuizAnswer>

**答え: 3**

ROEは「Return On Equity」の略で、自己資本利益率のことです。株主が出資した自己資本を使って、企業がどれだけ効率よく利益を上げているかを示す指標です。なお、選択肢1の総資産に対する利益の割合はROA（総資産利益率）に該当します。

</QuizAnswer>

## 問7

「OEM」の説明として適当なものを選びなさい。

1. 他社ブランドの製品を製造すること
2. 企業の社会的責任を果たすこと
3. 電子商取引を行うこと
4. 製品の品質管理を徹底すること


<QuizAnswer>

**答え: 1**

OEMは「Original Equipment Manufacturer」の略で、他社のブランド名で販売される製品を製造することを指します。例えば、A社がB社ブランドの製品を製造・供給するような形態です。なお、選択肢2はCSR、選択肢3はEC（Electronic Commerce）に該当します。

</QuizAnswer>

## 問8

企業が建物や機械設備などの固定資産について、使用や時間の経過による価値の減少分を費用として計上することを何というか。

1. 引当金
2. 減価償却
3. 棚卸資産
4. 繰延資産


<QuizAnswer>

**答え: 2**

減価償却とは、建物・機械・車両などの固定資産の取得費用を、その資産の耐用年数にわたって分割し、毎年の費用として計上する会計処理のことです。固定資産は時間の経過や使用によって価値が減少するため、その減少分を各期の費用に配分します。

</QuizAnswer>

## 問9

事業者間の公正で自由な競争を促進し、消費者の利益を確保することを目的とした法律はどれか。

1. 独占禁止法
2. 消費者契約法
3. 不正競争防止法
4. 製造物責任法


<QuizAnswer>

**答え: 1**

独占禁止法（私的独占の禁止及び公正取引の確保に関する法律）は、事業者間の公正かつ自由な競争を促進し、消費者の利益を確保するとともに、国民経済の健全な発展を図ることを目的とした法律です。公正取引委員会がその運用を担っています。

</QuizAnswer>

## 問10

次のうち、「ニッチ市場」の説明として適当なものはどれか。

1. 市場全体の中で最も売上規模が大きい主要市場
2. 海外に新たに進出して開拓する市場
3. 複数の企業が共同で開発・運営する市場
4. 大企業が参入しない小規模で特定のニーズを持つすき間市場


<QuizAnswer>

**答え: 4**

ニッチ市場とは、大企業が手を出さないような小規模で専門的な「すき間市場」のことです。特定のニーズを持つ限られた顧客層を対象としており、大手企業との競争を避けつつ、専門性を活かしてビジネスを展開できる市場として注目されています。

</QuizAnswer>
