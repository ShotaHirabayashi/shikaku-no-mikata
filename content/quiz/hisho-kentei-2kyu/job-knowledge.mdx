## 問1

秘書の業務は「定型業務」と「非定型業務」に分けられる。次のうち、非定型業務に該当するものはどれか。

1. 上司宛ての郵便物の受け取りと仕分け
2. 毎朝の新聞記事のクリッピング
3. 予約のない来客への対応と取り次ぎ判断
4. 上司のスケジュール表の定期的な更新


<QuizAnswer>

**答え: 3**

非定型業務とは、あらかじめ手順が決まっておらず、その都度状況に応じた判断が求められる業務のことです。予約のない来客への対応は、相手の用件や緊急度を判断して上司に取り次ぐかどうかを決める必要があるため、非定型業務に分類されます。郵便物の仕分け、新聞クリッピング、スケジュール表の更新はいずれも手順が決まっている定型業務です。

</QuizAnswer>

## 問2

上司が出張する際、秘書が事前に準備すべきこととして、最も不適当なものはどれか。

1. 出張先での商談内容を秘書が事前にまとめて決定しておく
2. 交通機関の時刻や予約の手配を行う
3. 宿泊先の予約と確認を行う
4. 出張中のスケジュール表を作成して上司に渡す


<QuizAnswer>

**答え: 1**

出張先での商談内容を決定するのは上司自身の職務であり、秘書が勝手に決めるべきことではありません。秘書の出張手配業務としては、交通機関や宿泊先の予約・手配、出張スケジュール表の作成、必要な資料や書類の準備などが挙げられます。秘書はあくまで上司の業務を補佐する立場であり、上司の判断に関わる内容にまで踏み込まないことが重要です。

</QuizAnswer>

## 問3

上司が外出中に、約束のない来客があった。秘書の対応として最も適当なものはどれか。

1. 上司がいないので、すぐに帰ってもらうよう伝える
2. 上司の帰社予定時刻を伝え、そのまま待ってもらう
3. 来客の用件を聞かず、名刺だけ預かっておく
4. 来客の名前・用件を確認し、上司に連絡して指示を仰ぐ


<QuizAnswer>

**答え: 4**

上司不在時に約束のない来客があった場合、まず来客の名前や用件を確認し、上司に連絡を取って対応の指示を仰ぐのが基本です。すぐに帰ってもらうのは失礼にあたり、用件も聞かずに名刺だけ預かるのも不十分な対応です。また、勝手に待たせるのではなく、上司の判断を確認してから対応を決めることが求められます。

</QuizAnswer>

## 問4

上司への報告の仕方として、最も適当なものはどれか。

1. 経過を詳しく説明してから、最後に結論を述べる
2. まず結論を述べ、次に理由、必要に応じて経過を説明する
3. 自分の意見や推測を交えて、上司が判断しやすいようにする
4. 報告事項が複数ある場合は、発生した順番にすべて報告する


<QuizAnswer>

**答え: 2**

報告は「結論→理由→経過」の順序で行うのが原則です。上司は多忙であるため、まず結論を先に述べることで要点をすぐに把握してもらうことができます。経過から話し始めると要点がわかりにくくなります。また、報告では事実のみを正確に伝え、自分の意見や推測は求められない限り交えないことが基本です。

</QuizAnswer>

## 問5

伝言メモの書き方として、最も不適当なものはどれか。

1. 相手の名前・会社名・電話番号を正確に記載する
2. 用件の内容を簡潔にまとめて記載する
3. 伝言を受けた日時と受けた人の名前を記載する
4. 相手の話した内容を一字一句そのまま書き写す


<QuizAnswer>

**答え: 4**

伝言メモは、相手の話した内容を一字一句そのまま記録するのではなく、用件の要点を簡潔にまとめて記載するものです。伝言メモには、相手の名前・会社名・連絡先、用件の要点、受信日時、受けた人の名前を記載することが基本です。正確さは重要ですが、要点を整理してわかりやすく伝えることが求められます。

</QuizAnswer>

## 問6

上司の日程管理に関する記述として、最も適当なものはどれか。

1. 関係者に配布する予定表には、上司の私的な予定は記載しない
2. 予定の変更が生じた場合、関係者への連絡は翌日にまとめて行う
3. 上司の予定はすべて秘書の判断で組み入れてよい
4. 来客の予定は、当日の朝に上司に知らせれば十分である


<QuizAnswer>

**答え: 1**

関係者に配布する予定表には、業務上の予定のみを記載し、上司の私的な予定（通院や家族の行事など）は記載しないのが原則です。予定変更が生じた場合は速やかに関係者に連絡する必要があり、翌日まで待つのは不適切です。また、予定を組み入れる際は上司の確認を得ることが必要であり、来客予定も事前に十分な余裕をもって伝えるべきです。

</QuizAnswer>

## 問7

上司が出張中に、上司宛ての急ぎの用件で電話がかかってきた。秘書の対応として最も適当なものはどれか。

1. 上司の出張先の連絡先を相手に教え、直接連絡してもらう
2. 急ぎの用件であっても、上司の帰社まで待ってもらう
3. 秘書の判断で用件に対応し、事後に上司に報告する
4. 用件を聞き取り、出張先の上司に連絡して指示を仰ぐ


<QuizAnswer>

**答え: 4**

上司が出張中に急ぎの電話があった場合は、まず用件を正確に聞き取り、秘書から出張先の上司に連絡して指示を仰ぐのが適切な対応です。上司の出張先の連絡先を外部の人にそのまま教えるのは、上司のプライバシーや予定への配慮に欠けます。また、急ぎの用件を帰社まで待たせるのは不適切であり、秘書が独断で対応するのも職務の範囲を超えています。

</QuizAnswer>

## 問8

秘書が上司のスケジュールを調整する際の考え方として、最も適当なものはどれか。

1. 予定はできるだけ詰めて入れ、空き時間をなくすようにする
2. 重要な会議の前後には、準備や移動のための余裕時間を設ける
3. 先に入った予定を常に優先し、後からの予定変更には応じない
4. 上司の体調に関わらず、予定どおりにスケジュールを進める


<QuizAnswer>

**答え: 2**

スケジュール調整では、重要な会議や来客の前後に準備・移動・休憩のための余裕時間を確保することが大切です。予定を隙間なく詰め込むと、上司に負担がかかり、急な予定変更にも対応できなくなります。また、予定変更は状況に応じて柔軟に対応する必要があり、上司の体調への配慮も秘書の重要な職務です。

</QuizAnswer>

## 問9

上司から「明日の会議の資料を準備しておいてほしい」と指示を受けた。秘書の対応として最も適当なものはどれか。

1. 会議の内容・参加者・必要部数などを確認してから準備に取りかかる
2. 上司に細かいことを聞くのは失礼なので、自分の判断で資料を準備する
3. 資料の準備は秘書の仕事ではないので、担当部署に依頼する
4. 前回の会議と同じ資料を用意しておけば問題ない


<QuizAnswer>

**答え: 1**

上司から指示を受けた際は、不明点を確認してから作業に取りかかるのが基本です。会議の内容、参加者、必要な資料の種類や部数などを確認することで、的確な準備ができます。確認せずに自分の判断だけで進めると、上司の意図と異なる結果になる恐れがあります。指示の内容を正確に把握するために質問することは、失礼にはあたりません。

</QuizAnswer>

## 問10

上司の仕事に関する秘書の対応として、最も不適当なものはどれか。

1. 上司の業務内容や関係先を把握しておく
2. 上司が優先すべき仕事を理解し、それに合わせて補佐する
3. 複数の上司に仕える場合は、それぞれの業務内容や性格を理解しておく
4. 上司の仕事の進め方に問題があると感じたら、同僚に相談して改善を促す


<QuizAnswer>

**答え: 4**

上司の仕事の進め方について問題を感じた場合、同僚に相談するのは不適切です。上司に関することを第三者に話すことは職務上の信頼関係を損なう行為です。秘書は上司の業務内容や関係先をよく把握し、優先順位を理解した上で補佐にあたることが求められます。複数の上司に仕える場合は、それぞれの特性を理解して柔軟に対応することが大切です。

</QuizAnswer>
