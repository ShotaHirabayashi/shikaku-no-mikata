## 問1

秘書の役割として、最も適切なものはどれか。

1. 上司の仕事を自分の判断で代行し、効率を上げること
2. 上司よりも先に意思決定を行い、業務をリードすること
3. 上司が本来の業務に専念できるよう、補佐・支援すること
4. 上司の弱点を周囲に伝え、チームでカバーする体制を作ること


<QuizAnswer>

**答え: 3**

秘書の最も基本的な役割は、上司が本来の業務に専念できるように補佐・支援することです。上司の仕事を「代行」するのではなく、あくまでサポート役に徹することが求められます。意思決定は上司が行うものであり、秘書が先走って判断することは適切ではありません。

</QuizAnswer>

## 問2

秘書に求められる人柄として、不適当なものはどれか。

1. 誠実で責任感がある
2. 明朗で親しみやすい
3. 冷静で情緒が安定している
4. 自己主張が強く、自分の意見を押し通す


<QuizAnswer>

**答え: 4**

秘書には誠実さ、明朗さ、冷静さ、協調性などが求められます。自己主張が強く自分の意見を押し通す姿勢は、上司の補佐役としてふさわしくありません。秘書は上司の意向を尊重し、柔軟に対応できる素直さや謙虚さが重要です。

</QuizAnswer>

## 問3

秘書が上司から受けた指示に疑問を感じた場合の対応として、最も適切なものはどれか。

1. 自分の判断で指示内容を変更して実行する
2. その場で確認し、上司の意図を正確に把握してから取りかかる
3. 疑問があっても黙って指示どおりに実行する
4. 同僚に相談して、同僚の意見に従う


<QuizAnswer>

**答え: 2**

上司からの指示に疑問がある場合は、その場で確認し、上司の意図を正確に把握してから取りかかることが大切です。自分の判断で勝手に変更したり、疑問を抱えたまま実行したりすると、上司の意図と異なる結果になるおそれがあります。確認することは上司への信頼を損ねる行為ではなく、正確な仕事をするために必要なことです。

</QuizAnswer>

## 問4

秘書の機密保持に関する記述として、不適当なものはどれか。

1. 上司の個人的な情報を社内の人にも漏らさないようにする
2. 機密書類は保管から破棄まで責任を持って管理する
3. エレベーターや電車の中では仕事の話をしないように心がける
4. 親しい同僚には、まだ発表されていない人事異動の情報を教えてあげる


<QuizAnswer>

**答え: 4**

秘書は機密事項を知りうる立場にありますが、たとえ親しい同僚であっても、未発表の人事異動などの情報を漏らしてはなりません。機密保持は秘書にとって最も重要な資質の一つであり、社内外を問わず厳守する必要があります。選択肢1〜3はいずれも適切な機密保持の行動です。

</QuizAnswer>

## 問5

秘書の身だしなみとして、最も適切なものはどれか。

1. 流行のファッションを積極的に取り入れ、華やかな印象を与える
2. 個性を出すために、アクセサリーを多くつけるようにする
3. 清潔感があり、控えめで上品な服装を心がける
4. 上司より目立つ服装にして、来客に好印象を与える


<QuizAnswer>

**答え: 3**

秘書の身だしなみは、清潔感があり、控えめで上品であることが基本です。華美な服装や過度なアクセサリーは避け、職場にふさわしい落ち着いた装いが求められます。秘書は上司の補佐役であるため、上司より目立つことは適切ではなく、相手に不快感を与えない身だしなみを心がけることが大切です。

</QuizAnswer>

## 問6

秘書として職場の人間関係を円滑にするための心がけとして、不適当なものはどれか。

1. 社内のうわさ話に積極的に参加して、情報収集に努める
2. 他部署の人にも分け隔てなく、明るく丁寧に接する
3. 自分の失敗は素直に認め、謝罪する姿勢を持つ
4. 周囲の人の立場や気持ちを考えて行動する


<QuizAnswer>

**答え: 1**

秘書は職場の人間関係の潤滑油的な存在であることが求められますが、社内のうわさ話に積極的に参加することは適切ではありません。うわさ話に加わると機密漏洩のリスクが高まるだけでなく、信頼を失う原因にもなります。秘書には、誰に対しても公平で明るく接し、協調性を持って仕事に取り組む姿勢が求められます。

</QuizAnswer>

## 問7

上司が外出中に、上司あてに急ぎの案件で他部署の部長から電話があった。秘書の対応として、最も適切なものはどれか。

1. 上司が不在であることだけ伝え、戻り次第折り返すと言って電話を切る
2. 急ぎの案件であるため、上司に連絡を取り、指示を仰ぐ
3. 自分の判断で回答できる内容なので、上司に代わって対応する
4. 上司の外出先と携帯番号を相手に教え、直接連絡してもらう


<QuizAnswer>

**答え: 2**

急ぎの案件の場合、秘書は上司に連絡を取って指示を仰ぐことが適切です。秘書が自分の判断で勝手に回答することは越権行為にあたり、また上司の外出先や連絡先を無断で教えることも好ましくありません。上司の状況を確認した上で、上司の指示に従って対応することが秘書としての正しい判断です。

</QuizAnswer>

## 問8

秘書としての仕事の進め方について、最も適切なものはどれか。

1. 上司に言われたことだけをすればよく、先を読んで準備する必要はない
2. 複数の仕事を抱えた場合は、自分のやりやすい順に処理する
3. 仕事の結果だけでなく、途中経過も適宜上司に報告する
4. 効率を上げるために、上司への報告はまとめて一度に行う


<QuizAnswer>

**答え: 3**

秘書は仕事の結果だけでなく、途中経過も適宜タイミングよく上司に報告することが大切です。上司が状況を把握していないと、適切な判断や指示ができなくなります。また、報告をまとめて一度に行うのではなく、重要度や緊急度に応じてタイミングよく報告することが求められます。

</QuizAnswer>

## 問9

秘書に求められる「気配り」について、最も適切なものはどれか。

1. 上司のスケジュールや体調に注意を払い、さりげなくサポートする
2. 上司のプライベートな事情を詳しく聞き出し、周囲に共有する
3. 来客に対して、上司の仕事の進捗状況を詳しく説明する
4. 上司の欠点を指摘して改善を促す


<QuizAnswer>

**答え: 1**

秘書の気配りとは、上司のスケジュールや体調などに注意を払い、さりげなくサポートすることです。上司のプライベートに過度に踏み込んだり、その情報を周囲に共有したりすることは守秘義務に反します。また、上司の仕事の進捗を来客に詳しく説明することも適切ではなく、秘書は控えめに、かつ的確にサポートする姿勢が求められます。

</QuizAnswer>

## 問10

秘書の心構えとして適切なものを選べ。

1. 上司の指示がなくても、自分が正しいと思うことは独断で進める
2. 忙しいときは感情的になってもやむを得ない
3. 上司が間違ったことを言っても、その場では指摘せず後で二人のときに伝える
4. 自分の担当外の仕事は一切手伝わないようにする


<QuizAnswer>

**答え: 3**

上司が間違ったことを言った場合、人前で指摘すると上司の面目を損なうおそれがあるため、後で二人きりのときにさりげなく伝えることが適切です。秘書は常に冷静で情緒が安定していることが求められ、忙しくても感情的にならないよう心がける必要があります。また、協調性を持ち、必要に応じて担当外の仕事にも協力する姿勢が大切です。

</QuizAnswer>
