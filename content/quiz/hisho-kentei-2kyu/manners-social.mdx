## 問1

応接室における席次について、最も適切なものはどれか。

1. 出入り口に最も近い席が上座である
2. 窓側であれば出入り口に近くても上座となる
3. 出入り口から最も遠い席が上座である
4. 応接室では席次を気にする必要はない


<QuizAnswer>

**答え: 3**

応接室では、出入り口から最も遠い奥の席が上座、出入り口に最も近い席が下座となるのが基本です。これはお客様や目上の方を奥の落ち着いた席にご案内するためのマナーです。長椅子（ソファー）と一人掛けの椅子がある場合は、長椅子側が上座となります。

</QuizAnswer>

## 問2

タクシーに上司とお客様と3人で乗車する場合、上座はどこか。

1. 助手席
2. 運転席の後ろの席
3. 後部座席の中央
4. 助手席の後ろの席


<QuizAnswer>

**答え: 2**

タクシーでは、運転席の後ろが最上座、次に助手席の後ろ、後部座席の中央の順となり、助手席が最も下座です。タクシーの場合、運転席の後ろが最も安全な席とされていることが理由の一つです。秘書や目下の者は助手席に座り、料金の支払いや道案内を担当します。

</QuizAnswer>

## 問3

上司が自ら運転する自家用車に同乗する場合、上座はどこか。

1. 運転席の後ろの席
2. 後部座席の中央
3. 助手席の後ろの席
4. 助手席


<QuizAnswer>

**答え: 4**

上司や同行者が自ら運転する自家用車の場合は、助手席が上座となります。これはタクシーの席次とは異なるため注意が必要です。運転者のすぐ隣で会話がしやすく、運転者への敬意を示すことができるという理由から、助手席が最上座とされています。

</QuizAnswer>

## 問4

エレベーターの席次について、最も適切なものはどれか。

1. 操作盤の前が下座であり、出入り口から最も遠い奥が上座である
2. 操作盤の前が上座であり、出入り口から最も遠い奥が下座である
3. エレベーター内では席次は特に決まっていない
4. 出入り口の正面が上座であり、操作盤の位置は関係ない


<QuizAnswer>

**答え: 1**

エレベーターでは、操作盤の前が下座です。目下の者や案内役が操作盤の前に立ち、階数ボタンや「開」「閉」ボタンを操作します。上座は出入り口から最も遠い奥の位置で、入口から向かって左奥が最上座とされています。

</QuizAnswer>

## 問5

仏式の通夜に参列する際、不祝儀袋の表書きとして最も一般的に使えるものはどれか。

1. 御仏前
2. 御玉串料
3. 御霊前
4. 御花料


<QuizAnswer>

**答え: 3**

仏式の通夜・葬儀では「御霊前」が最も一般的な表書きです。「御仏前」は四十九日の法要以降に使うのが基本です（ただし浄土真宗では葬儀から「御仏前」を使います）。「御玉串料」は神式、「御花料」はキリスト教式の表書きです。宗派が不明な場合も「御霊前」が広く使えるとされています。

</QuizAnswer>

## 問6

お中元を贈る時期として、一般的に最も適切なものはどれか。

1. 5月上旬〜5月中旬
2. 7月上旬〜7月中旬
3. 9月上旬〜9月中旬
4. 11月上旬〜11月中旬


<QuizAnswer>

**答え: 2**

お中元は、日頃お世話になっている方への感謝を込めて夏に贈るもので、一般的に7月上旬〜7月中旬（関東地方の場合）が適切な時期です。なお、関西地方では7月中旬〜8月15日頃までとされるなど、地域によって多少の差があります。この時期を過ぎた場合は「暑中御見舞」「残暑御見舞」として贈ります。

</QuizAnswer>

## 問7

立食パーティーでのマナーとして、最も適切なものはどれか。

1. 料理は一度にたくさん皿に取り、何度も取りに行かないようにする
2. グラスと皿は両手に分けて持ち、両方を常に手にしておく
3. 名刺交換は席に着いてから行うのがマナーである
4. 皿は左手で持ち、右手を空けておくようにする


<QuizAnswer>

**答え: 4**

立食パーティーでは、皿やグラスは左手でまとめて持ち、右手を空けておくのがマナーです。右手を空けておくのは、握手や名刺交換にすぐ対応できるようにするためです。料理は少量ずつ取り、なくなったら新しい皿で取りに行くのが上品とされています。

</QuizAnswer>

## 問8

慶事の祝儀袋に関する説明として、最も適切なものはどれか。

1. 結婚祝いの水引は「結び切り」を使う
2. 結婚祝いの水引は「蝶結び」を使う
3. 出産祝いの水引は「結び切り」を使う
4. 結婚祝いには水引のない封筒を使う


<QuizAnswer>

**答え: 1**

結婚祝いの水引は「結び切り（真結び）」を使います。結び切りは一度結ぶとほどけないことから、「一度きりであってほしい」という意味が込められており、結婚祝いに適しています。一方、「蝶結び（花結び）」は何度でも結び直せることから、出産祝いや入学祝いなど、何度あっても喜ばしい慶事に用います。

</QuizAnswer>

## 問9

喪中の方へのお歳暮の対応として、最も適切なものはどれか。

1. 喪中の方にはお歳暮を一切贈ってはならない
2. 四十九日を過ぎていれば贈ってもよいが、のし紙や品物に配慮する
3. 喪中でも通常通り華やかな包装で贈ってよい
4. 喪中の場合はお歳暮の代わりに現金を贈る


<QuizAnswer>

**答え: 2**

お歳暮は日頃の感謝を伝える贈答であり、お祝いごととは異なるため、喪中でも贈ること自体はマナー違反ではありません。ただし、四十九日の忌明けを過ぎてから贈るのが望ましく、華やかな包装や生ものは避け、表書きを「御歳暮」ではなく「寒中御見舞」とするなどの配慮が必要です。

</QuizAnswer>

## 問10

弔事に参列する際の服装として、最も適切なものはどれか。

1. 通夜には華やかな色の服装で駆けつけるのがマナーである
2. 葬儀・告別式では明るいグレーのスーツを着用する
3. 通夜・葬儀ともに光沢のあるアクセサリーは避け、真珠の一連ネックレスは許容される
4. 葬儀では金色のバックルのベルトを着用してもよい


<QuizAnswer>

**答え: 3**

弔事では、光沢のある装飾品や派手なアクセサリーは避けるのがマナーです。ただし、真珠は「涙の象徴」とされ、一連のネックレスであれば着用が許容されます。二連のネックレスは「不幸が重なる」ことを連想させるため避けます。服装は黒の喪服が基本で、金具や光沢のある素材は控えるのが適切です。

</QuizAnswer>
