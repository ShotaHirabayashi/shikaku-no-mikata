## 問1

社外文書の構成として、「拝啓」などの頭語の後に続く部分の正しい順序はどれか。

1. 主文 → 前文 → 末文 → 結語
2. 末文 → 主文 → 前文 → 結語
3. 前文 → 主文 → 末文 → 結語
4. 主文 → 末文 → 前文 → 結語


<QuizAnswer>

**答え: 3**

社外文書は、頭語の後に「前文（時候の挨拶・安否の挨拶・感謝の挨拶）→ 主文（本題）→ 末文（結びの挨拶）→ 結語」の順で構成されます。前文で礼儀を尽くしてから本題に入り、末文で締めくくるのがビジネス文書の基本形式です。

</QuizAnswer>

## 問2

月別の売上高の推移を視覚的に表現したい場合、最も適切なグラフの種類はどれか。

1. 折れ線グラフ
2. 円グラフ
3. 帯グラフ
4. レーダーチャート


<QuizAnswer>

**答え: 1**

折れ線グラフは、時間の経過に伴うデータの変化（推移）を表現するのに最も適したグラフです。月別売上高のように時系列に沿った変化を可視化する場合に用いられます。円グラフは構成比、帯グラフは構成比の比較、レーダーチャートは複数項目のバランスを見るのに適しています。

</QuizAnswer>

## 問3

ファイリングの方法のうち、書類をフォルダに入れて引き出しに垂直に収納する方式を何というか。

1. ボックスファイリング
2. バーチカルファイリング
3. ラテラルファイリング
4. フラットファイリング


<QuizAnswer>

**答え: 2**

バーチカル（vertical＝垂直）ファイリングとは、書類を個別のフォルダに挟み、キャビネットの引き出しに垂直に立てて収納する方法です。書類を綴じる必要がないため、追加や差し替えが容易で、日常的に出し入れする書類の管理に適しています。

</QuizAnswer>

## 問4

社内文書の書き方として、適切なものはどれか。

1. 頭語と結語を必ず記載する
2. 時候の挨拶を冒頭に入れる
3. 「ございます」調の敬体で書く
4. 文末に「以上」と記載して締めくくる


<QuizAnswer>

**答え: 4**

社内文書は社外文書と異なり、頭語・結語や時候の挨拶は不要です。意思の伝達を優先するため、簡潔に書き、文末には「以上」と記載して締めくくるのが基本的な形式です。「ございます」調の丁寧な敬体は社外文書で使用するものであり、社内文書では「です・ます」調で簡潔に書きます。

</QuizAnswer>

## 問5

議事録に記載すべき必須項目として、適切でないものはどれか。

1. 会議名と開催日時
2. 出席者名
3. 出席者の役職の略歴
4. 決定事項


<QuizAnswer>

**答え: 3**

議事録の必須項目は、会議名、開催日時、開催場所、出席者名、議題、決定事項、今後の予定（ToDo）などです。出席者の役職の略歴は議事録に記載する必要はありません。議事録は会議で何が決まったかを正確に記録し、関係者に伝達するための文書です。

</QuizAnswer>

## 問6

郵便物の取り扱いに関する記述として、正しいものはどれか。

1. 速達は、届け先に届いたことを証明するサービスである
2. 書留は、通常より早く届けるためのサービスである
3. 内容証明は、届いた日時を証明するサービスである
4. 内容証明は、いつ・どのような内容の文書を・誰から誰あてに差し出したかを証明するサービスである


<QuizAnswer>

**答え: 4**

内容証明郵便は、いつ、いかなる内容の文書を、誰から誰あてに差し出したかということを日本郵便が証明する制度です。速達は通常より早く届けるサービス、書留は引き受けから配達までの送達過程を記録し万一の際に賠償するサービス、配達証明は届いたことを証明するサービスです。それぞれの違いを正確に理解しておくことが重要です。

</QuizAnswer>

## 問7

ある商品の市場における各社のシェア（占有率）を表すのに最も適切なグラフはどれか。

1. 円グラフ
2. 棒グラフ
3. 折れ線グラフ
4. 散布図


<QuizAnswer>

**答え: 1**

円グラフは、全体に対する各構成要素の割合（構成比）を示すのに最も適したグラフです。市場シェアのように「全体を100%として各社が何%を占めるか」を視覚的に表現する場合に用いられます。棒グラフは量の大小比較、折れ線グラフは時系列の推移、散布図は2つのデータの相関関係を示すのに適しています。

</QuizAnswer>

## 問8

オフィスの環境整備に関する記述として、適切なものはどれか。

1. 室温は年間を通じて一定の20℃に設定するのがよい
2. 事務室の照度は、一般的に300ルクス程度が適切とされている
3. コピー機やプリンターは使用頻度に関係なく各部署に1台ずつ配置する
4. 観葉植物はすべてのオフィスで禁止されている


<QuizAnswer>

**答え: 2**

事務室（一般的なオフィス）の照度は、JIS規格で300ルクス程度が推奨されています。室温は夏季と冬季で適温が異なり（夏季25〜28℃、冬季18〜22℃程度）、年間一定ではありません。OA機器は使用頻度や業務効率を考慮して配置し、観葉植物は空気清浄やリラックス効果から設置されることも一般的です。

</QuizAnswer>

## 問9

文書の保管・保存に関する記述として、適切なものはどれか。

1. 保管とは、使用頻度の低い文書を倉庫等に移して管理することである
2. 保存とは、日常的に使用する文書を手元に置いて管理することである
3. 文書の廃棄は、担当者の判断で自由に行ってよい
4. 保管は日常的に使う文書を手元に置くこと、保存は使用頻度の低い文書を別の場所で管理することである


<QuizAnswer>

**答え: 4**

文書管理において「保管」と「保存」は明確に区別されます。「保管」は日常的に使用する文書を手元（事務室内のキャビネット等）に置いて管理すること、「保存」は使用頻度が低くなった文書を倉庫等の別の場所に移して管理することを指します。文書の廃棄には所定の手続きや上司の承認が必要であり、担当者が自由に行うことはできません。

</QuizAnswer>

## 問10

社外文書における頭語と結語の組み合わせとして、正しいものはどれか。

1. 拝啓 ― 草々
2. 前略 ― 敬具
3. 前略 ― 草々
4. 謹啓 ― 拝答


<QuizAnswer>

**答え: 3**

頭語と結語には正しい組み合わせがあります。「前略」は前文を省略する際に使う頭語で、結語は「草々」です。「拝啓」の結語は「敬具」、「謹啓」の結語は「謹白」または「敬白」が正しい組み合わせです。秘書検定では頭語と結語の正しい対応関係がよく出題されるため、主要な組み合わせを覚えておくことが重要です。

</QuizAnswer>
