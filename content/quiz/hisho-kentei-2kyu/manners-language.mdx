## 問1

次のうち、尊敬語として正しい表現はどれか。

1. 「部長が申されたように、この方針で進めます」
2. 「お客様がおっしゃられました」
3. 「社長がご覧になりました」
4. 「先生が参られました」


<QuizAnswer>

**答え: 3**

「ご覧になる」は「見る」の正しい尊敬語です。選択肢1の「申す」は謙譲語であり、相手の動作に使うのは誤りです。選択肢2は「おっしゃる」＋「られる」で二重敬語にあたります。選択肢4の「参る」は謙譲語なので、相手の動作には使えません。

</QuizAnswer>

## 問2

「見る」の謙譲語として正しいものはどれか。

1. 拝見する
2. ご覧になる
3. 見られる
4. お見えになる


<QuizAnswer>

**答え: 1**

「拝見する」は「見る」の謙譲語で、自分が見る動作をへりくだって表現する言葉です。選択肢2の「ご覧になる」は「見る」の尊敬語であり、相手の動作に使います。選択肢4の「お見えになる」は「来る」の尊敬語であり、「見る」とは意味が異なります。

</QuizAnswer>

## 問3

電話応対で、相手の名前を確認するときの言い方として最も適切なものはどれか。

1. 「お名前をちょうだいできますか」
2. 「どちら様ですか」
3. 「お名前を申していただけますか」
4. 「恐れ入りますが、お名前をお伺いしてもよろしいでしょうか」


<QuizAnswer>

**答え: 4**

クッション言葉「恐れ入りますが」を添えた上で、「お伺いしてもよろしいでしょうか」と丁寧に尋ねるのが最も適切です。選択肢1の「お名前をちょうだいする」は、名前はもらうものではないため不適切な表現です。選択肢3の「申す」は謙譲語であり、相手の動作に使うのは誤りです。選択肢2は丁寧さに欠けます。

</QuizAnswer>

## 問4

来客に記名をお願いする際の言葉遣いとして、最も適切なものはどれか。

1. 「すみませんが、こちらにお名前をお書きください」
2. 「恐れ入りますが、こちらにご記入いただけますでしょうか」
3. 「こちらに名前を書いてもらえますか」
4. 「ここに記入してください」


<QuizAnswer>

**答え: 2**

ビジネスの場では、クッション言葉として「恐れ入りますが」を用い、「ご記入いただけますでしょうか」と依頼形で表現するのが最も丁寧で適切です。選択肢1の「すみませんが」はビジネスの接遇用語としてはややカジュアルです。選択肢3・4は敬語やクッション言葉が不足しており、来客への対応としては不適切です。

</QuizAnswer>

## 問5

次のうち、二重敬語にあたるものはどれか。

1. 「ご案内いたします」
2. 「お召し上がりになられますか」
3. 「お越しくださいますか」
4. 「お伝えいただけますか」


<QuizAnswer>

**答え: 2**

「お召し上がりになられますか」は、「召し上がる」（尊敬語）に「られる」（尊敬の助動詞）を重ねた二重敬語です。正しくは「召し上がりますか」または「お召し上がりになりますか」とします。選択肢1の「ご案内いたします」は「ご〜いたす」の形で正しい謙譲表現であり、二重敬語ではありません。

</QuizAnswer>

## 問6

ビジネス文書における頭語と結語の組み合わせとして、正しいものはどれか。

1. 謹啓 ― 敬具
2. 前略 ― 敬具
3. 拝啓 ― 草々
4. 拝啓 ― 敬具


<QuizAnswer>

**答え: 4**

「拝啓」と「敬具」は最も一般的な頭語・結語の組み合わせで、一般的な社外文書に用います。選択肢1の「謹啓」の結語は「謹白」または「敬白」が正しい組み合わせです。選択肢2の「前略」の結語は「草々」です。選択肢3は「拝啓」と「草々」を組み合わせており、これも誤りです。

</QuizAnswer>

## 問7

社外の人に電話で自社の部長（山田）について話すとき、最も適切な表現はどれか。

1. 「部長の山田はただいま外出しております」
2. 「山田部長はただいま外出しております」
3. 「山田部長様は外出されております」
4. 「山田さんは外出中でございます」


<QuizAnswer>

**答え: 1**

社外の人に対しては、自社の上司であっても敬称をつけず、「部長の山田」のように役職＋名前（呼び捨て）で表現するのが正しいビジネスマナーです。選択肢2の「山田部長」は社内では使えますが、社外の人に対しては身内に敬称をつけていることになり不適切です。選択肢3の「様」や選択肢4の「さん」も同様に社外向けには不適切です。

</QuizAnswer>

## 問8

来客に椅子をすすめるとき、最も適切な表現はどれか。

1. 「どうぞおかけください」
2. 「どうぞお座りください」
3. 「こちらに座ってください」
4. 「こちらにおかけになられてください」


<QuizAnswer>

**答え: 1**

来客に椅子をすすめる際は「おかけください」が最も適切な接遇表現です。選択肢2の「お座りください」は文法的には間違いではありませんが、ペットへのしつけを連想させるため、ビジネスの場では避けるべき表現とされています。選択肢3は敬語が不足しており、選択肢4は「おかけになる」＋「られる」で二重敬語にあたります。

</QuizAnswer>

## 問9

ビジネス文書で、会社宛てに送る場合の宛名の敬称として正しいものはどれか。

1. ○○株式会社様
2. ○○株式会社各位
3. ○○株式会社御中
4. ○○株式会社殿


<QuizAnswer>

**答え: 3**

会社や団体など組織宛てに文書を送る場合は「御中」を用います。選択肢1の「様」は個人宛ての敬称であり、会社名には使いません。選択肢2の「各位」は複数の個人に宛てる場合に使う敬称です。選択肢4の「殿」は主に公文書で用いられ、一般的なビジネス文書では使用を避けるのが通例です。

</QuizAnswer>

## 問10

次のうち、敬語の使い方として正しいものはどれか。

1. 「資料を拝見させていただきました」
2. 「こちらで承らせていただきます」
3. 「お客様がお見えになられました」
4. 「ご不明な点がございましたら、お申し付けください」


<QuizAnswer>

**答え: 4**

「ご不明な点がございましたら、お申し付けください」は正しい敬語表現です。選択肢1は「拝見する」（謙譲語）＋「させていただく」（謙譲語）で二重敬語です。正しくは「拝見しました」とします。選択肢2も「承る」（謙譲語）＋「させていただく」（謙譲語）の二重敬語です。選択肢3は「お見えになる」（尊敬語）＋「られる」（尊敬の助動詞）の二重敬語で、正しくは「お見えになりました」です。

</QuizAnswer>
