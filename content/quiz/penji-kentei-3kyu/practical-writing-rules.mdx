## 問1

硬筆書写技能検定3級の実技・第1問「速書き」で使用しなければならない筆記用具はどれか。

1. 万年筆
2. ボールペン
3. つけペン
4. サインペン


<QuizAnswer>

**答え: 2**

速書き（第1問）では、ボールペンの使用が指定されています。ボールペン以外の筆記用具で解答した場合は大幅に減点され、60点以下となってしまいます。試験当日は必ずボールペンを準備しましょう。

</QuizAnswer>

## 問2

硬筆書写技能検定3級の掲示文（第6問）で使用する用紙のサイズはどれか。

1. A3
2. A4
3. B4
4. B5


<QuizAnswer>

**答え: 3**

掲示文の問題ではB4サイズの用紙を縦に使用して横書きで書きます。試験では指定の解答用紙が配布されますが、練習の際もB4サイズの用紙を使って本番に近い形で練習することが大切です。

</QuizAnswer>

## 問3

掲示文を書く際、用紙の周囲にとるべき余白の目安として適切なものはどれか。

1. 約1cm
2. 約3cm
3. 約5cm
4. 余白は不要

<QuizAnswer>

**答え: 2**

掲示文では紙面いっぱいに書かず、周囲に約3cm程度の余白を取るのが原則です。適切な余白を設けることで見やすくなり、体裁の整った掲示文に仕上がります。

</QuizAnswer>

## 問4

はがきの宛名を縦書きで書く場合、番地などの数字の表記として正しいものはどれか。

1. アラビア数字（1, 2, 3…）で書く
2. 漢数字（一、二、三…）で書く
3. ローマ数字（Ⅰ, Ⅱ, Ⅲ…）で書く
4. 数字の種類は問われない


<QuizAnswer>

**答え: 2**

縦書きの場合は漢数字（一、二、三…）を使い、横書きの場合はアラビア数字（1, 2, 3…）を使うのが正しい書式です。縦横で数字の表記を統一することで、全体に調和のとれた体裁になります。

</QuizAnswer>

## 問5

硬筆書写技能検定3級の掲示文を書く際、鉛筆で事前に引くことが許可されているものはどれか。

1. 字の下書き
2. 割付けの線（レイアウト線）
3. 飾り枠の線
4. 何も引いてはならない


<QuizAnswer>

**答え: 2**

掲示文では鉛筆で割付けの線（レイアウトを決める補助線）を引くことが認められています。ただし、字の下書きは禁止されており、提出時には割付けの線を消しゴムできれいに消して提出する必要があります。

</QuizAnswer>

## 問6

はがきの宛名書きで、受取人の氏名の文字の大きさについて正しいものはどれか。

1. 住所と同じ大きさで書く
2. 住所よりも小さく書く
3. 住所よりも大きく書く
4. 差出人の名前と同じ大きさで書く


<QuizAnswer>

**答え: 3**

はがきの宛名書きでは、受取人の氏名は住所よりも大きな文字で書くのがマナーです。氏名を最も目立たせることで相手への敬意を表し、郵便配達の際にも読みやすくなります。

</QuizAnswer>

## 問7

硬筆書写技能検定3級の速書き（第1問）で、ひらがな・カタカナの文字の大きさについて正しいものはどれか。

1. 漢字と同じ大きさで書く
2. 漢字よりも大きめに書く
3. 漢字よりも少し小さめに書く
4. できるだけ小さく書く


<QuizAnswer>

**答え: 3**

速書きに限らず書写全般のルールとして、ひらがな・カタカナ・数字は漢字よりも少し小さめに書くのが基本です。漢字を大きく、仮名を小さくすることで、文章全体にメリハリが出て読みやすい体裁になります。

</QuizAnswer>

## 問8

掲示文のレイアウトにおいて、タイトル（題名）の書き方として最も適切なものはどれか。

1. 本文と同じ大きさで左寄せに書く
2. 本文より小さく右寄せに書く
3. 本文より大きく中央に配置して書く
4. 本文より大きく右寄せに書く


<QuizAnswer>

**答え: 3**

掲示文ではタイトルを最も大きな文字で書き、用紙の中央に収まるように配置するのが原則です。タイトルが目立つことで掲示の内容が一目でわかり、読み手に伝わりやすいレイアウトになります。

</QuizAnswer>

## 問9

はがきの宛名書きにおいて、差出人（自分）の住所と名前の書き方として正しいものはどれか。

1. 受取人の住所・名前と同じ大きさで書く
2. 受取人の住所・名前よりも大きく書く
3. 受取人の住所・名前よりも小さく書く
4. 差出人の情報は裏面に書くため表面には書かない


<QuizAnswer>

**答え: 3**

差出人の住所と名前は、受取人の住所と名前よりも小さく書くのが正しいマナーです。これは相手に対する敬意を表す意味があり、硬筆書写技能検定でもこの体裁が求められます。

</QuizAnswer>

## 問10

硬筆書写技能検定3級の速書きにおいて、各行の書き方として注意すべき点はどれか。

1. 行ごとに文字の大きさを変えて変化をつける
2. 各行の両端を揃え、字の大きさと字間をほぼ均等にする
3. 1行目だけ字下げし、残りは左端から書く
4. 行間をできるだけ狭くして多くの文字を詰める


<QuizAnswer>

**答え: 2**

速書きでは、各行の両端（書き出しと書き終わり）を揃え、字の大きさと字間をほぼ均等に保つことが重要です。限られた時間内でも体裁を意識し、全体が整って見えるように書くことが合格のポイントです。

</QuizAnswer>
