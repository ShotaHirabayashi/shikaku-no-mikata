## 問1

漢字の左側に位置する部分のことを何と呼ぶか。

1. 旁（つくり）
2. 冠（かんむり）
3. 偏（へん）
4. 脚（あし）


<QuizAnswer>

**答え: 3**

漢字の左側に位置する部分を「偏（へん）」と呼びます。例えば「林」の「きへん」、「海」の「さんずい」などが偏にあたります。偏は漢字の構成要素の中で最も種類が多く、意味を表す役割を担うことが多いです。

</QuizAnswer>

## 問2

漢字「問」「間」「開」などに共通する、外側から囲む部分「門」の名称はどれか。

1. くにがまえ
2. もんがまえ
3. かくしがまえ
4. はこがまえ


<QuizAnswer>

**答え: 2**

「門」の形で外側から囲む部首は「もんがまえ（門構え）」と呼ばれます。「問」「間」「開」「閉」などの漢字に使われています。このように漢字の外側を囲むように位置する部分を「構え（かまえ）」と総称します。

</QuizAnswer>

## 問3

漢字の上側から左下にかけて覆うように位置する部分のことを、七つの分類のうち何と呼ぶか。

1. 繞（にょう）
2. 冠（かんむり）
3. 垂（たれ）
4. 構え（かまえ）


<QuizAnswer>

**答え: 3**

漢字の上側から左下にかけて位置する部分を「垂（たれ）」と呼びます。「まだれ（广）」「がんだれ（厂）」「やまいだれ（疒）」「しかばね（尸）」などが垂に分類されます。上から垂れ下がるような形をしていることからこの名称がつけられています。

</QuizAnswer>

## 問4

漢字「記」「話」「読」などの左側にある部首の名称はどれか。

1. にんべん
2. いとへん
3. さんずい
4. ごんべん


<QuizAnswer>

**答え: 4**

「記」「話」「読」などの左側にある「言（訁）」の部分は「ごんべん（言偏）」と呼ばれます。言葉や話すことに関連する漢字に多く使われる偏です。「語」「説」「証」「訳」なども同じくごんべんを持つ漢字です。

</QuizAnswer>

## 問5

漢字「花」「草」「茶」の上部にある「艹」の部首名称はどれか。

1. たけかんむり
2. うかんむり
3. くさかんむり
4. あめかんむり


<QuizAnswer>

**答え: 3**

「艹」は「くさかんむり（草冠）」と呼ばれ、植物に関連する漢字に多く見られます。漢字の上側に位置する部分は「冠（かんむり）」と総称されます。他にも「うかんむり（宀）」「たけかんむり（⺮）」「あめかんむり（雨）」などがあります。

</QuizAnswer>

## 問6

漢字の右側に位置する部分のことを何と呼ぶか。

1. 偏（へん）
2. 旁（つくり）
3. 垂（たれ）
4. 繞（にょう）


<QuizAnswer>

**答え: 2**

漢字の右側に位置する部分を「旁（つくり）」と呼びます。例えば「都」の「おおざと」、「数」の「のぶん・ぼくづくり」などが旁にあたります。偏が意味を表すのに対し、旁は音（読み）を表すことが多いとされています。

</QuizAnswer>

## 問7

漢字「道」「近」「送」などに共通する、左側から下にかけて位置する「辶」の部首名称はどれか。

1. えんにょう
2. そうにょう
3. しんにょう（しんにゅう）
4. きにょう


<QuizAnswer>

**答え: 3**

「辶」は「しんにょう（しんにゅう）」と呼ばれ、道や移動に関連する漢字に多く使われます。漢字の左側から下にかけて位置する部分を「繞（にょう）」と総称します。他にも「えんにょう（廴）」「そうにょう（⻌の別形）」などが繞に分類されます。

</QuizAnswer>

## 問8

漢字「広」「店」「度」の上から左下にかけてある「广」の部首名称はどれか。

1. がんだれ
2. まだれ
3. しかばね
4. やまいだれ


<QuizAnswer>

**答え: 2**

「广」は「まだれ（麻垂れ）」と呼ばれ、建物や場所に関連する漢字に多く見られる垂（たれ）の一種です。「がんだれ（厂）」は「厚」「原」などに、「やまいだれ（疒）」は「病」「痛」などに、「しかばね（尸）」は「届」「届」などに使われます。

</QuizAnswer>

## 問9

漢字「想」「点」「熱」の下部にある「灬（れんが・れっか）」のように、漢字の下側に位置する部分の総称はどれか。

1. 冠（かんむり）
2. 偏（へん）
3. 旁（つくり）
4. 脚（あし）


<QuizAnswer>

**答え: 4**

漢字の下側に位置する部分を「脚（あし）」と呼びます。「灬」は「れんが（連火）」または「れっか（列火）」と呼ばれる脚の一種です。他にも「ひとあし（儿）」「こころ（心）」などが脚に分類されます。冠が上部にあるのに対し、脚は下部に位置します。

</QuizAnswer>

## 問10

漢字「国」「囲」「園」などに共通する「囗」の部首名称はどれか。

1. もんがまえ
2. はこがまえ
3. つつみがまえ
4. くにがまえ


<QuizAnswer>

**答え: 4**

「囗」は「くにがまえ（国構え）」と呼ばれ、「国」「囲」「園」「図」などの漢字に使われています。「もんがまえ（門）」は「門」の形で囲む部首で、「くにがまえ」とは形が異なります。構え（かまえ）は漢字の外側を囲むように位置する部分の総称で、他にも「ぎょうがまえ（行）」などがあります。

</QuizAnswer>
