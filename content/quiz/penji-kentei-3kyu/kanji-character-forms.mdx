## 問1

常用漢字表において、「國」という字体はどのように扱われるか。

1. 常用漢字の正しい字体である
2. 常用漢字の旧字体であり、現在の正しい字体は「国」である
3. 書写体として認められている
4. 常用漢字表に含まれない漢字である


<QuizAnswer>

**答え: 2**

「國」は「国」の旧字体です。1949年の「当用漢字字体表」で簡略化され、現在の常用漢字表では「国」が正しい字体として採用されています。硬筆書写技能検定では、旧字体は常用漢字の正しい字体とは認められないため、正誤判定では「誤り」と判断されます。

</QuizAnswer>

## 問2

硬筆書写技能検定3級の字体の正誤判定において、「正しい字体」とは何を基準にしているか。

1. 明朝体の印刷文字と完全に一致すること
2. 常用漢字表に示された字体（骨組み）に合致していること
3. 教育漢字として小学校で習う書き方であること
4. 書道の楷書の手本と同じ形であること


<QuizAnswer>

**答え: 2**

硬筆書写技能検定の字体の正誤判定は、常用漢字表に示された字体（漢字の骨組み）を基準としています。印刷文字（明朝体）と手書き文字では細部が異なることがありますが、字体の骨組みが正しければ許容されます。明朝体と完全一致する必要はなく、あくまで字体としての正しさが問われます。

</QuizAnswer>

## 問3

次のうち、旧字体と新字体（常用漢字の字体）の組み合わせとして正しいものはどれか。

1. 旧字体「醫」→ 新字体「医」
2. 旧字体「學」→ 新字体「字」
3. 旧字体「氣」→ 新字体「汽」
4. 旧字体「廣」→ 新字体「拡」


<QuizAnswer>

**答え: 1**

「醫」の新字体は「医」です。選択肢2の「學」の新字体は「学」であり「字」ではありません。選択肢3の「氣」の新字体は「気」であり「汽」ではありません。選択肢4の「廣」の新字体は「広」であり「拡」ではありません。旧字体と新字体の対応を正確に覚えることが大切です。

</QuizAnswer>

## 問4

常用漢字の字体において、印刷文字（明朝体）と手書き文字（筆写の楷書）の関係について、正しい説明はどれか。

1. 印刷文字と手書き文字は、細部まで完全に同じでなければならない
2. 印刷文字と手書き文字では、はねやとめなどの細部に違いがあっても字体が同じなら許容される
3. 手書き文字は印刷文字より画数が少なくてもよい
4. 印刷文字の形だけが正しく、手書き文字の形は認められない


<QuizAnswer>

**答え: 2**

常用漢字表では、印刷文字と手書き文字の間に見られる細部の違い（はね・とめ・はらいなど）は、字体の骨組みが同じであれば許容されるとしています。例えば「木」の縦画を印刷文字ではとめ、手書きではねにしても、どちらも正しいとされます。文化庁の「常用漢字表の字体・字形に関する指針」でもこの考え方が示されています。

</QuizAnswer>

## 問5

次のうち、旧字体と新字体の組み合わせとして **誤っている** ものはどれか。

1. 旧字体「圓」→ 新字体「円」
2. 旧字体「澤」→ 新字体「沢」
3. 旧字体「櫻」→ 新字体「桜」
4. 旧字体「鐵」→ 新字体「銀」


<QuizAnswer>

**答え: 4**

「鐵」の新字体は「鉄」であり、「銀」ではありません。「銀」は旧字体と新字体が同じ字です。選択肢1の「圓→円」、選択肢2の「澤→沢」、選択肢3の「櫻→桜」はいずれも正しい対応です。紛らわしい旧字体と新字体の対応は、検定でもよく問われるポイントです。

</QuizAnswer>

## 問6

硬筆書写技能検定3級の正誤判定問題で、ある漢字が「書写体」で書かれていた場合、その判定はどうなるか。

1. 書写体は手書き文字として正しいので「正しい」と判定する
2. 書写体は常用漢字の字体ではないので「誤り」と判定する
3. 書写体は旧字体と同じ扱いなので「正しい」と判定する
4. 書写体かどうかは判定に影響しない


<QuizAnswer>

**答え: 2**

書写体とは、古くから手書きで慣用的に使われてきた字体のことですが、常用漢字表に示された正しい字体とは異なります。硬筆書写技能検定の正誤判定では、常用漢字表の字体が基準となるため、書写体で書かれたものは「誤り」と判定されます。旧字体と同様に、書写体も常用漢字としては認められません。

</QuizAnswer>

## 問7

「令」という漢字について、印刷文字では下部が「マ」のような形、手書きでは「丶（点）と縦画」のように書くことがある。この違いについて正しい説明はどれか。

1. 印刷文字の形だけが正しく、手書きの形は誤りである
2. 手書きの形だけが正しく、印刷文字の形は誤りである
3. どちらも同じ字体であり、デザインの違い（書体差）として許容される
4. 二つは別の漢字であるため、使い分ける必要がある


<QuizAnswer>

**答え: 3**

「令」の下部の書き方は、印刷文字（明朝体）と手書き文字で異なることがありますが、これはデザインの違い（書体差）であり、どちらも正しい字体として認められています。文化庁の「常用漢字表の字体・字形に関する指針」（平成28年）でも、このような印刷文字と手書き文字の違いは字体の違いではないと明記されています。

</QuizAnswer>

## 問8

次のうち、常用漢字の字体として正しいものはどれか。

1. 「齋」（「斉」の旧字体）
2. 「斉」（常用漢字表の字体）
3. 「齊」（「斉」の異体字）
4. 「斎」（「斉」の別字）


<QuizAnswer>

**答え: 2**

「斉」が常用漢字表に示された正しい字体です。「齋」「齊」はいずれも旧字体・異体字であり、常用漢字の字体としては認められません。なお「斎」は「斉」とは別の漢字（「書斎」の「斎」）であり、こちらも常用漢字ですが別字です。字形が似ている漢字の区別は検定でも重要なポイントです。

</QuizAnswer>

## 問9

常用漢字の正誤判定において、「正しい字体を誤りと指摘してしまう」とどうなるか。

1. 特に問題はなく、減点されない
2. 減点の対象となる
3. その問題全体が無効になる
4. 正しい字体を誤りと指摘しても部分点がもらえる


<QuizAnswer>

**答え: 2**

硬筆書写技能検定の正誤判定問題では、正しい字体を誤りと指摘してしまうと減点の対象となります。つまり、誤っている字体を見つけるだけでなく、正しい字体を正しいと判断する力も求められます。むやみに多くの字を「誤り」と判断するのではなく、確実に誤りとわかるものだけを指摘することが重要です。

</QuizAnswer>

## 問10

次の旧字体と新字体の対応のうち、正しいものはいくつあるか。
　ア）「舊」→「旧」　イ）「變」→「変」　ウ）「辯」→「弁」　エ）「體」→「休」

1. 1つ
2. 2つ
3. 3つ
4. 4つ


<QuizAnswer>

**答え: 3**

正しい対応は、ア）「舊→旧」、イ）「變→変」、ウ）「辯→弁」の3つです。エ）の「體」の新字体は「体」であり、「休」ではありません。「体」と「休」は字形が似ていますが、「體」は「からだ」を意味し、「休」は「やすむ」を意味する全く別の漢字です。紛らわしい字形の区別は検定でよく出題されます。

</QuizAnswer>
