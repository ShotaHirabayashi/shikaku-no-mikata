## 問1

楷書の最も基本的な特徴として正しいものはどれか。

1. 点画を連続させて速く書く
2. 一画一画を離して丁寧に書く
3. 点画を自由に省略して書く
4. 曲線を多用してやわらかく書く


<QuizAnswer>

**答え: 2**

楷書の最大の特徴は、一画一画を続けずに筆を離して丁寧に書くことです。点画が明瞭で読み間違いが起こりにくく、正式な文書に使われる書体として長い歴史があります。連続や省略は行書・草書の特徴であり、楷書とは区別されます。

</QuizAnswer>

## 問2

行書の特徴として**誤っている**ものはどれか。

1. 点画が曲線的になる
2. 点画を連続して書くことがある
3. すべての点画を必ず省略して書く
4. 楷書と筆順が変わることがある


<QuizAnswer>

**答え: 3**

行書では点画を省略することが「ある」のであって、「すべての点画を必ず省略する」わけではありません。行書の主な特徴は、(1)点画が曲線的になる、(2)点画が連続する、(3)点画を省略することがある、(4)点画が変化する、(5)筆順が変わることがある、の5つです。

</QuizAnswer>

## 問3

行書における「筆脈（ひつみゃく）」の説明として最も適切なものはどれか。

1. 筆を紙に強く押しつけて書く技法のこと
2. 目には見えない点画のつながり・気脈のこと
3. 墨の濃淡をつけて書く表現技法のこと
4. 文字の大きさを変化させる構成法のこと


<QuizAnswer>

**答え: 2**

筆脈とは、字を書くときに実線では結ばれていないが気持ちの上でつながっている、目に見えない点画のつながりのことです。楷書にも筆脈はありますが、行書ではこの筆脈が実際の線として紙の上に現れるのが大きな特徴です。

</QuizAnswer>

## 問4

楷書における「三折法（さんせつほう）」とは何か。

1. 文字を三つの部分に分けてバランスをとる方法
2. 一画を始筆・送筆・終筆の三段階で書く方法
3. 三画ずつまとめて連続して書く方法
4. 横画・縦画・斜画の三種類を使い分ける方法


<QuizAnswer>

**答え: 2**

三折法（三過折とも呼ばれる）とは、楷書において一画ごとに「始筆（起筆）・送筆・終筆（収筆）」の三段階をしっかりそろえて書く基本技法です。これにより横線の打ち込み・中間の線・止めがはっきりした明瞭な文字になります。

</QuizAnswer>

## 問5

行書で「春」のように横画が連続する漢字を書く場合、どのような変化が起こりやすいか。

1. 横画をすべて省略して一本の縦画だけにする
2. 横画の間隔を広げて一画ずつ離して書く
3. 連続する横画をつなげて書く
4. 横画をすべて点に変えて書く


<QuizAnswer>

**答え: 3**

行書では、「春」のように横画が連続する漢字の場合、横の線をつなげて書くのが典型的な変化パターンです。これは行書の「点画の連続（続け書き）」という特徴の代表例であり、書く速度を上げつつも読みやすさを保つ工夫です。

</QuizAnswer>

## 問6

行書の表現の幅について、正しい説明はどれか。

1. 行書には厳格なルールがあり、一つの決まった形しか認められない
2. 直線的に書けば楷書に近く、曲線を増やせば草書に近くなる
3. 行書はすべて草書と同じ崩し方で書かなければならない
4. 行書は楷書とまったく同じ形で速く書くだけの書体である


<QuizAnswer>

**答え: 2**

行書は正書（公式書体）ではなかったため厳格なルールが存在せず、表現の幅が非常に広い書体です。直線的に書けば楷書に近いものになり、点画を連続させて曲線を増やせば草書に近いものになります。この自由度の高さが行書の大きな特徴です。

</QuizAnswer>

## 問7

行書で「然」の下部にある四つの点（れんが・れっか）を書く場合、どのような変化がよく見られるか。

1. 四つの点をそのまま一画ずつ離して書く
2. 四つの点を連続してつなげて書く
3. 四つの点をすべて省略して書かない
4. 四つの点を一本の横画に変える


<QuizAnswer>

**答え: 2**

行書では、「然」などの漢字にある四つの点（れんが）は、連続してつなげて書くのが一般的な変化パターンです。これは点画の連続（続け書き）の代表的な例で、完全に省略するのではなく、点と点をつないで一連の動きで書き表します。

</QuizAnswer>

## 問8

楷書と行書の使い分けについて、最も適切なものはどれか。

1. 行書は正式な公文書や証書に使うのが正しい
2. 楷書は手紙やメモなど速書きが必要な場面に向いている
3. 楷書は正式な文書に、行書は日常の手紙や実用的な場面に適している
4. 楷書と行書に使い分けの基準はなく、どちらを使っても同じである


<QuizAnswer>

**答え: 3**

楷書は一画一画が明瞭で読み間違いが起こりにくいため、正式な文書や公的な書類に適しています。一方、行書は速く書くことができ、かつ読みやすさも保たれるため、日常の手紙やメモなど実用的な場面に適しています。硬筆書写技能検定3級では、この書き分けの理解が求められます。

</QuizAnswer>

## 問9

楷書から行書への変化パターンとして、「木へん」に見られる典型的な変化はどれか。

1. 木へん全体を丸い点一つに省略する
2. 木へんの縦画を二本に増やす
3. 木へんの左払いの後の点（右払い）が省略される
4. 木へんの横画を縦画に変える


<QuizAnswer>

**答え: 3**

行書において、木へん（きへん）では2画目の左払いの後にある点（右払いに相当する部分）が省略されることが多いです。これは行書の「点画の省略」の代表的な例であり、書く速度を上げるための合理的な変化パターンとして知られています。

</QuizAnswer>

## 問10

行書で点画が連続する（続け書きになる）際に最も重要な意識はどれか。

1. できるだけ多くの画を省略して画数を減らすこと
2. 楷書の点画の流れ（筆脈）を意識してつなげること
3. すべての画を一筆で書ききること
4. 楷書よりも文字を大きく書くこと


<QuizAnswer>

**答え: 2**

行書で点画を連続させる際に重要なのは、楷書における点画の流れ（筆脈）を意識してつなげることです。楷書では目に見えなかった筆脈が、行書では実際の線として紙の上に現れます。やみくもに画を省略したりつなげたりするのではなく、元の楷書の構造を理解した上で自然な流れでつなげることが大切です。

</QuizAnswer>
