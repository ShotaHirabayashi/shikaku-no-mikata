## 問1

漢字「右」の正しい一画目はどれですか。

1. 横画（一）
2. 左払い（ノ）
3. 縦画（｜）
4. 口の最初の画


<QuizAnswer>

**答え: 2**

「右」の一画目は左払い（ノ）です。「左」の一画目が横画であるのに対し、「右」の一画目は左払いから始まります。「左」と「右」は一画目が逆になる代表的な組み合わせとして、筆順問題で最も頻出です。覚え方として、「右」は短い画（ノ）から、「左」は短い画（一）から書き始めると整理できます。

</QuizAnswer>

## 問2

漢字の筆順の基本原則として正しいものはどれですか。

1. 下から上へ、右から左へ書く
2. 上から下へ、左から右へ書く
3. 中心から外側へ書く
4. 縦画を先に、横画を後に書く


<QuizAnswer>

**答え: 2**

漢字の筆順の最も基本的な原則は「上から下へ、左から右へ」書くことです。この原則はほぼすべての漢字に共通して適用されます。例えば「三」は上の横画から順に下へ書き、「川」は左の縦画から順に右へ書きます。

</QuizAnswer>

## 問3

漢字「十」を書くとき、正しい筆順はどれですか。

1. 縦画を書いてから横画を書く
2. 横画を書いてから縦画を書く
3. どちらから書いてもよい
4. 左上の角から対角線に書く


<QuizAnswer>

**答え: 2**

「十」は横画を先に書いてから縦画を書きます。これは「横画と縦画が交差する場合は横画を先に書く」という筆順の原則に基づいています。この原則は「土」「士」「千」など多くの漢字にも適用される重要なルールです。

</QuizAnswer>

## 問4

漢字「必」の正しい一画目はどれですか。

1. 左上の点
2. 中央の点（心の最初の点の位置）
3. 左払い（ノ）
4. 右上の点


<QuizAnswer>

**答え: 2**

「必」の一画目は中央の点から書き始めます。「必」は筆順を間違えやすい漢字の代表例で、「心」に「ノ」を加えた形と誤解されがちですが、実際には中央の点→左払い→右上の点→左下の点→右払いの順で書きます。文部省「筆順指導の手びき」でも特に注意すべき漢字として挙げられています。

</QuizAnswer>

## 問5

漢字「田」の三画目として正しいのはどれですか。

1. 内部の横画
2. 内部の縦画
3. 外枠の右側の縦画
4. 外枠の下の横画


<QuizAnswer>

**答え: 2**

「田」の筆順は、①縦画（左側）②横画から折れて囲む画③内部の縦画④内部の横画⑤下の横画（閉じる画）です。三画目は内部の縦画であり、横画ではありません。囲む形の中に縦画と横画がある場合、縦画を先に書くのが正しい筆順です。

</QuizAnswer>

## 問6

左払いと右払いが交差する漢字（例：「人」「文」）の筆順として正しいものはどれですか。

1. 右払いを先に書く
2. 左払いを先に書く
3. 長い方の画を先に書く
4. 短い方の画を先に書く


<QuizAnswer>

**答え: 2**

左払いと右払いが交差する場合は、左払いを先に書くのが原則です。「人」「文」「父」「交」などがこのルールに従います。これは「左から右へ」という基本原則とも一致しており、筆順の重要な原則の一つです。

</QuizAnswer>

## 問7

漢字「左」の正しい一画目はどれですか。

1. 左払い（ノ）
2. 横画（一）
3. 縦画（｜）
4. 「工」の最初の画


<QuizAnswer>

**答え: 2**

「左」の一画目は横画（一）です。「右」の一画目が左払い（ノ）であるのに対し、「左」は横画から書き始めます。この違いは硬筆書写技能検定で非常によく出題されるポイントです。「左」は横画が長く払いが短い、「右」は払いが長く横画が短いという字形の特徴と関連づけて覚えると効果的です。

</QuizAnswer>

## 問8

囲む形（「口」「国」など）の漢字における筆順の原則として正しいものはどれですか。

1. 中の部分を先に書いてから外側を書く
2. 外側を先に書いてから中の部分を書き、最後に閉じる
3. 下の横画を先に書いてから囲む
4. 右側の縦画から書き始める


<QuizAnswer>

**答え: 2**

囲む形の漢字は「外側を先に書き、中の部分を書いてから、最後に下の横画で閉じる」のが原則です。例えば「国」では、まず外枠の「囗」の上と左右を書き、次に中の「玉」を書き、最後に下の横画で閉じます。この「先に囲んで、中を書き、最後に閉じる」という原則は多くの漢字に適用されます。

</QuizAnswer>

## 問9

漢字「飛」の一画目として正しいものはどれですか。

1. 左側の縦画
2. 上部の左払い
3. 上部の横画から折れてはねる画
4. 中央の縦画


<QuizAnswer>

**答え: 3**

「飛」の一画目は、上部のやや右上がりの横画を書き、そこから折れて曲げながら下ろし、上にはねる画です。「飛」は9画の漢字で、筆順を間違えやすい漢字として有名です。左側の部分から書くと誤解しがちですが、正しくは右上の横画＋折れはねから始まります。

</QuizAnswer>

## 問10

他の画を貫く縦画や横画がある漢字（例：「母」「世」）において、貫く画の筆順として正しいものはどれですか。

1. 貫く画を最初に書く
2. 貫く画を途中で書く
3. 貫く画を最後に書く
4. 貫く画は書かなくてよい


<QuizAnswer>

**答え: 3**

他の画を貫く画は最後に書くのが筆順の原則です。例えば「母」では、中を貫く横画（よこの点を含む部分）は後から書きます。「世」でも縦画を貫く長い横画は最後に書きます。これは、先に貫く画を書いてしまうと残りの画の位置が定めにくくなるためで、全体のバランスを整えやすくする合理的なルールです。

</QuizAnswer>
