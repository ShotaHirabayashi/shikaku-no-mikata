## 問1

企業が製品やサービスの原材料調達から廃棄・リサイクルまでのライフサイクル全体を通じて、環境への負荷を定量的に評価する手法を何というか。

1. カーボンフットプリント（CFP）
2. 環境影響評価（環境アセスメント）
3. ライフサイクルアセスメント（LCA）
4. マテリアルフローコスト会計（MFCA）


<QuizAnswer>

**答え: 3**

ライフサイクルアセスメント（LCA）は、製品やサービスの「ゆりかごから墓場まで」、すなわち資源採取・原料生産・製品製造・流通・使用・廃棄・リサイクルの全段階における環境負荷を定量的に評価する手法です。国際規格ISO14040およびISO14044で手順が定められています。カーボンフットプリントはLCAの手法を用いてCO2排出量に特化して算定するものであり、環境アセスメントは大規模開発事業に対する事前評価制度です。

</QuizAnswer>

## 問2

eco検定に合格した人は何と呼ばれるか。

1. エコリーダー
2. エコピープル
3. グリーンサポーター
4. エコマスター


<QuizAnswer>

**答え: 2**

eco検定（環境社会検定試験）の合格者は「エコピープル」と呼ばれます。エコピープルが2名以上集まって環境活動に取り組む場合は「エコユニット」として登録でき、東京商工会議所から活動支援を受けることができます。エコピープルはビジネスや日常生活の中で率先して環境配慮行動を実践する役割が期待されています。

</QuizAnswer>

## 問3

開発途上国の生産者や労働者に対して公正な価格で継続的に取引を行い、生活改善と自立を支援する貿易のしくみを何というか。

1. エシカル消費
2. グリーン購入
3. フェアトレード
4. ソーシャルビジネス


<QuizAnswer>

**答え: 3**

フェアトレード（公正貿易）は、開発途上国の原料や製品を適正な価格で継続的に取引することで、生産者や労働者の生活改善と自立を促す仕組みです。国際フェアトレード認証ラベルが付された製品は、経済・社会・環境に関する基準を満たしていることが保証されます。エシカル消費はフェアトレードを含むより広い概念で、人・社会・環境に配慮した消費行動全般を指します。

</QuizAnswer>

## 問4

企業が自社の環境保全への取り組みや環境負荷の状況をまとめ、社外のステークホルダーに公表する報告書として、最も適切なものはどれか。

1. 有価証券報告書
2. 環境報告書
3. 事業報告書
4. 内部統制報告書


<QuizAnswer>

**答え: 2**

環境報告書は、企業が環境保全に関する方針・目標・実績・環境負荷データなどをまとめて公表する文書です。環境省が「環境報告ガイドライン」を策定しており、企業の環境情報開示の指針となっています。近年では、環境情報に加えて社会・ガバナンス情報も統合した「統合報告書」や「サステナビリティレポート」を発行する企業も増えています。

</QuizAnswer>

## 問5

製品の原材料調達から廃棄までのライフサイクル全体で排出される温室効果ガスの量をCO2に換算して表示する仕組みを何というか。

1. カーボンオフセット
2. エコリーフ
3. カーボンフットプリント（CFP）
4. ウォーターフットプリント


<QuizAnswer>

**答え: 3**

カーボンフットプリント（CFP）は、製品やサービスのライフサイクル全体で排出される温室効果ガスの量をCO2換算で算定し、表示する仕組みです。LCA（ライフサイクルアセスメント）の手法を用いて算定されます。消費者が購入時に製品の環境負荷を比較・選択できるようにすることを目的としています。カーボンオフセットは排出した温室効果ガスを他の場所での削減・吸収で埋め合わせる仕組みです。

</QuizAnswer>

## 問6

人や社会、地域、環境に配慮した消費行動を総称して何というか。

1. グリーンコンシューマリズム
2. エシカル消費
3. ロハス消費
4. スローライフ消費


<QuizAnswer>

**答え: 2**

エシカル消費（倫理的消費）とは、人・社会・地域・環境に配慮した消費行動のことです。フェアトレード商品の購入、地産地消、障がい者支援につながる商品の選択、環境負荷の少ない製品の購入などが含まれます。消費者庁も推進しており、SDGsの目標12「つくる責任 つかう責任」とも深く関連しています。

</QuizAnswer>

## 問7

企業が利益追求だけでなく、環境・社会・ガバナンスに配慮しながら社会的責任を果たす経営の考え方を表す略称はどれか。

1. BCP
2. CSR
3. COP
4. SRI


<QuizAnswer>

**答え: 2**

CSR（Corporate Social Responsibility：企業の社会的責任）は、企業が利益を追求するだけでなく、環境保全・人権尊重・地域貢献・公正な事業慣行など、社会全体に対する責任を果たしながら経営を行う考え方です。近年ではCSRからさらに進んで、環境・社会課題の解決を事業機会として捉える「サステナビリティ経営」へと発展しています。BCPは事業継続計画、SRIは社会的責任投資の略です。

</QuizAnswer>

## 問8

NPOに関する次の記述のうち、最も適切なものはどれか。

1. NPOは営利を目的とした民間の組織である
2. NPOは政府機関の一部として環境政策を立案する
3. NPOは非営利で公益的な活動を行う民間組織であり、環境保全活動にも大きな役割を果たす
4. NPOは企業から独立した株式会社形態の組織である


<QuizAnswer>

**答え: 3**

NPO（Non-Profit Organization：非営利組織）は、営利を目的とせず、社会的な課題の解決に取り組む民間の組織です。環境分野では、自然保護活動、環境教育、リサイクル推進、政策提言など幅広い活動を行っています。1998年に施行されたNPO法（特定非営利活動促進法）により法人格の取得が容易になり、環境保全を含む多くの分野で市民活動の担い手として重要な役割を果たしています。

</QuizAnswer>

## 問9

地方自治体の環境施策に関する記述として、最も適切なものはどれか。

1. 地方自治体は国の法律に基づく施策のみを行い、独自の環境条例を制定することはできない
2. 地方自治体は環境基本条例を制定し、地域の特性に応じた独自の環境施策を展開できる
3. 地方自治体の環境施策は産業振興が最優先であり、環境保全は国の役割である
4. 地方自治体は住民への環境教育を行う義務はない


<QuizAnswer>

**答え: 2**

地方自治体は、国の環境基本法に基づく施策を実施するだけでなく、地域の自然環境や社会状況に応じた独自の環境基本条例を制定し、きめ細やかな環境施策を展開することができます。例えば、独自の排出基準の設定、環境教育の推進、地域の自然保護計画の策定などがあります。歴史的にも、公害問題では国に先駆けて自治体が条例で規制を行ったケースが多くあります。

</QuizAnswer>

## 問10

環境教育に関する記述として、最も適切なものはどれか。

1. 環境教育は学校教育のみで行われるものであり、社会教育の対象ではない
2. 環境教育等促進法（環境教育推進法）では、学校・家庭・地域・職場などあらゆる場での環境教育を推進している
3. 環境教育の目的は環境に関する専門知識の習得のみである
4. 環境教育は先進国のみで必要とされる取り組みである


<QuizAnswer>

**答え: 2**

「環境教育等による環境保全の取組の促進に関する法律」（環境教育等促進法）は、学校・家庭・地域・職場などあらゆる場で環境教育を推進することを定めています。環境教育は単なる知識の習得にとどまらず、環境に対する感性や問題解決能力を養い、主体的に環境保全活動に参加する態度を育成することを目的としています。体験型学習やESD（持続可能な開発のための教育）の考え方も重視されています。

</QuizAnswer>
