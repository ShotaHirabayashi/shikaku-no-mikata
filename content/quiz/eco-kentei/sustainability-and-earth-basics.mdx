## 問1

「持続可能な開発（Sustainable Development）」の概念を広く世界に普及させた、1987年に発表された報告書の名称は何か。

1. アジェンダ21
2. 成長の限界
3. 我ら共有の未来（Our Common Future）
4. 沈黙の春

<QuizAnswer>

**答え: 3**

「我ら共有の未来（Our Common Future）」は、1987年に「環境と開発に関する世界委員会」（通称：ブルントラント委員会）が発表した報告書です。この報告書の中で「将来の世代のニーズを満たす能力を損なうことなく、今日の世代のニーズを満たすような開発」という持続可能な開発の定義が示され、国際社会に広く浸透しました。なお「成長の限界」は1972年にローマクラブが発表した報告書、「沈黙の春」は1962年にレイチェル・カーソンが著した書籍です。

</QuizAnswer>

## 問2

SDGs（持続可能な開発目標）に関する記述として、正しいものはどれか。

1. 2000年の国連ミレニアムサミットで採択された
2. 先進国のみが達成すべき目標として設定されている
3. 17の目標と169のターゲットで構成されている
4. 達成期限は2040年に設定されている

<QuizAnswer>

**答え: 3**

SDGsは17の目標と169のターゲットで構成されています。SDGsは2015年の国連サミットで採択された「持続可能な開発のための2030アジェンダ」に含まれるもので、達成期限は2030年です。先進国だけでなく、発展途上国を含むすべての国が取り組むべきユニバーサル（普遍的）な目標である点が、前身のMDGs（ミレニアム開発目標）との大きな違いです。

</QuizAnswer>

## 問3

地球の大気に関する記述として、誤っているものはどれか。

1. 大気の主成分は窒素で、体積比で約78%を占める
2. 酸素は大気の体積比で約21%を占める
3. 二酸化炭素は大気の体積比で約4%を占める
4. 大気は対流圏・成層圏・中間圏・熱圏に分けられる

<QuizAnswer>

**答え: 3**

二酸化炭素の大気中の濃度は約0.04%（約420ppm）であり、約4%というのは誤りです。大気は主に窒素（約78%）と酸素（約21%）で構成されており、二酸化炭素はごく微量ですが、温室効果を持つため地球の気温に大きな影響を与えています。大気は高度によって対流圏・成層圏・中間圏・熱圏の4つの層に分けられます。

</QuizAnswer>

## 問4

1972年にスウェーデンのストックホルムで開催された国連人間環境会議のスローガンとして正しいものはどれか。

1. 宇宙船地球号
2. かけがえのない地球（Only One Earth）
3. 持続可能な開発
4. 我々の世界を変革する

<QuizAnswer>

**答え: 2**

1972年の国連人間環境会議（ストックホルム会議）は、環境問題を議題とした初めての大規模な国際会議であり、「かけがえのない地球（Only One Earth）」をスローガンとして掲げました。この会議では「人間環境宣言」が採択され、国連環境計画（UNEP）の設立が決定されました。「宇宙船地球号」は経済学者ボールディングの概念、「我々の世界を変革する」は2030アジェンダの正式名称の一部です。

</QuizAnswer>

## 問5

生態系における物質循環に関する記述として、正しいものはどれか。

1. 生態系の中で物質は一方向にのみ流れ、循環しない
2. 生産者（植物）は光合成により無機物から有機物を合成する
3. 分解者は有機物を合成して生態系に供給する役割を持つ
4. 物質循環において、エネルギーも物質と同様に循環する

<QuizAnswer>

**答え: 2**

生態系では、生産者（植物）が光合成によって二酸化炭素と水から有機物を合成し、それを消費者（動物）が利用し、分解者（菌類・細菌）が有機物を無機物に分解して自然に戻すという物質循環が行われています。エネルギーは太陽から一方向に流れ循環しませんが、物質（炭素・窒素・リンなど）は生態系の中を循環します。

</QuizAnswer>

## 問6

プラネタリーバウンダリー（地球の限界）の概念に関する記述として、正しいものはどれか。

1. 国連が1972年に提唱した概念である
2. 地球環境に関する9つの境界が定義されている
3. 2023年時点で、9つの境界のうち超過しているものはない
4. 経済成長の限界値を定量的に示した概念である

<QuizAnswer>

**答え: 2**

プラネタリーバウンダリーは、2009年にヨハン・ロックストロームらの国際的な科学者グループが提唱した概念で、人類が安全に活動できる範囲の境界として9つの項目（気候変動、生物圏の健全さ、土地利用変化、淡水利用、生物地球化学的循環、海洋酸性化、大気エアロゾルの負荷、成層圏オゾンの破壊、新規化学物質）を定義しています。2023年の研究では9つのうち6つで限界を超過していることが報告されました。

</QuizAnswer>

## 問7

世界の人口増加と環境負荷の関係に関する記述として、最も適切なものはどれか。

1. 世界人口は2024年時点で約50億人である
2. 人口増加は食料・水・エネルギー需要の増大を通じて環境負荷を高める
3. 先進国は人口が多いため、発展途上国より一人当たりの環境負荷が大きい
4. 人口増加と環境問題には直接的な関係はない

<QuizAnswer>

**答え: 2**

人口増加は食料生産のための農地拡大、水資源の消費増大、エネルギー需要の増加など、さまざまな形で環境負荷を高めます。世界人口は2022年に80億人を突破しており、約50億人は誤りです。一人当たりの環境負荷は先進国の方が大きい傾向にありますが、それは人口の多さではなく消費水準の高さが理由です。

</QuizAnswer>

## 問8

SDGsの目標1は「貧困をなくそう」であるが、目標6に該当するものはどれか。

1. ジェンダー平等を実現しよう
2. エネルギーをみんなに そしてクリーンに
3. 安全な水とトイレを世界中に
4. 働きがいも経済成長も

<QuizAnswer>

**答え: 3**

SDGsの目標6は「安全な水とトイレを世界中に」で、すべての人々に水と衛生へのアクセスと持続可能な管理を確保することを目指しています。目標5が「ジェンダー平等を実現しよう」、目標7が「エネルギーをみんなに そしてクリーンに」、目標8が「働きがいも経済成長も」です。SDGsの17目標の番号と内容の対応は、eco検定でも頻出の知識です。

</QuizAnswer>

## 問9

地球の水に関する記述として、正しいものはどれか。

1. 地球上の水の約97.5%は海水（塩水）である
2. 地球上の淡水の大部分は河川や湖沼に存在する
3. 水の循環において、蒸発量が最も多いのは陸地からである
4. 地球上で人間が利用しやすい淡水は全体の約30%である

<QuizAnswer>

**答え: 1**

地球上の水の約97.5%は海水（塩水）であり、淡水は約2.5%にすぎません。さらに、淡水の大部分は南極やグリーンランドの氷河・氷床として存在しており、河川や湖沼の水はごくわずかです。人間が利用しやすい淡水は地球全体の水の約0.01%程度とされています。また、水の蒸発量が最も多いのは陸地ではなく海洋からです。

</QuizAnswer>

## 問10

1992年にブラジルのリオデジャネイロで開催された国連環境開発会議（地球サミット）で合意された文書に含まれないものはどれか。

1. 環境と開発に関するリオ宣言
2. アジェンダ21
3. 森林原則声明
4. 人間環境宣言

<QuizAnswer>

**答え: 4**

「人間環境宣言」は1972年のストックホルム会議（国連人間環境会議）で採択されたものであり、1992年のリオサミットの成果文書ではありません。リオサミットでは「環境と開発に関するリオ宣言」「アジェンダ21」「森林原則声明」が合意され、さらに「気候変動枠組条約」と「生物多様性条約」への署名が開始されました。リオサミットには約180カ国が参加し、環境と開発の両立を目指す歴史的な会議となりました。

</QuizAnswer>
