## 問1

地球温暖化の主な原因とされる「温室効果ガス」のうち、人間活動によって最も多く排出され、地球温暖化への寄与が最も大きいものはどれか。

1. メタン（CH₄）
2. 一酸化二窒素（N₂O）
3. 二酸化炭素（CO₂）
4. フロン類（HFCs）


<QuizAnswer>

**答え: 3**

二酸化炭素（CO₂）は、化石燃料の燃焼や森林伐採などにより大量に排出され、温室効果ガス全体の排出量の約75%を占めています。メタンやN₂O、フロン類も温室効果ガスですが、排出量の規模では二酸化炭素が圧倒的に大きく、温暖化への寄与が最大です。

</QuizAnswer>

## 問2

2015年にCOP21で採択されたパリ協定の長期目標として、正しいものはどれか。

1. 世界の平均気温上昇を産業革命以前に比べて3℃以内に抑える
2. 世界の平均気温上昇を産業革命以前に比べて2℃より十分低く保ち、1.5℃に抑える努力を追求する
3. 先進国のみが2050年までにCO₂排出をゼロにする
4. 途上国の温室効果ガス排出量を2030年までに半減させる


<QuizAnswer>

**答え: 2**

パリ協定では、世界的な平均気温上昇を産業革命以前に比べて2℃より十分低く保つとともに、1.5℃に抑える努力を追求することが長期目標として定められました。この目標は先進国・途上国を問わずすべての締約国に適用される点が、京都議定書と大きく異なります。

</QuizAnswer>

## 問3

IPCC（気候変動に関する政府間パネル）第6次評価報告書において、地球温暖化に関する記述として正しいものはどれか。

1. 地球の平均気温は産業革命前から約0.5℃上昇した
2. 人間活動が温暖化の主な原因であることには疑う余地がない
3. 温暖化の進行は2000年以降停止している
4. 海洋の温暖化は確認されていない


<QuizAnswer>

**答え: 2**

IPCC第6次評価報告書（AR6）では、「人間の影響が大気、海洋及び陸域を温暖化させてきたことには疑う余地がない」と、過去の報告書よりさらに強い表現で断定しました。また、産業革命前からの気温上昇はすでに約1.1℃に達しており、2030年代には1.5℃に達する可能性が高いと指摘されています。

</QuizAnswer>

## 問4

日本政府が掲げる「2050年カーボンニュートラル」の意味として最も適切なものはどれか。

1. 2050年までに温室効果ガスの排出量をゼロにすること
2. 2050年までに温室効果ガスの排出量と吸収量を均衡させ、実質ゼロにすること
3. 2050年までに再生可能エネルギーの割合を100%にすること
4. 2050年までに化石燃料の輸入をゼロにすること


<QuizAnswer>

**答え: 2**

カーボンニュートラルとは、温室効果ガスの排出量から森林などによる吸収量を差し引いて、実質的にゼロにすることを意味します。排出そのものを完全にゼロにするのではなく、どうしても排出される分を吸収や除去で相殺する考え方です。日本は2020年10月に2050年カーボンニュートラルを宣言しました。

</QuizAnswer>

## 問5

再生可能エネルギーに該当しないものはどれか。

1. 地熱発電
2. バイオマス発電
3. LNG（液化天然ガス）火力発電
4. 風力発電


<QuizAnswer>

**答え: 3**

LNG（液化天然ガス）は化石燃料の一種であり、再生可能エネルギーには該当しません。地熱、バイオマス、風力はいずれも自然界に存在するエネルギーを利用した再生可能エネルギーです。LNG火力は石炭火力に比べてCO₂排出量が少ないものの、化石燃料であることに変わりありません。

</QuizAnswer>

## 問6

日本の温室効果ガス削減目標について、2025年2月に閣議決定された新たな中期目標の内容として正しいものはどれか。

1. 2030年度に2013年度比で26%削減
2. 2035年度に2013年度比で60%削減
3. 2035年度に1990年度比で50%削減
4. 2040年度に2013年度比で50%削減


<QuizAnswer>

**答え: 2**

日本政府は2025年2月に、パリ協定の1.5℃目標に整合する新たな目標として、2035年度に2013年度比で60%削減、2040年度に73%削減することを閣議決定し、国連気候変動枠組条約事務局に提出しました。これは従来の2030年度46%削減目標をさらに引き上げた野心的な目標です。

</QuizAnswer>

## 問7

温室効果ガスの排出に価格を付けることで排出削減を促す経済的手法を総称して何というか。

1. グリーンウォッシュ
2. カーボンプライシング
3. カーボンオフセット
4. エコロジカル・フットプリント


<QuizAnswer>

**答え: 2**

カーボンプライシングとは、炭素（CO₂）の排出に価格を付けることで、排出者に経済的インセンティブを与えて排出削減を促す政策手法の総称です。代表的なものに「炭素税」と「排出量取引制度（キャップ・アンド・トレード）」があります。日本では2026年度から排出量取引制度の本格稼働が予定されています。

</QuizAnswer>

## 問8

第7次エネルギー基本計画（2025年2月閣議決定）において、2040年度の電源構成に占める再生可能エネルギーの目標割合として示されたものはどれか。

1. 約20〜25%
2. 約30〜35%
3. 約40〜50%
4. 約60〜70%


<QuizAnswer>

**答え: 3**

2025年2月に閣議決定された第7次エネルギー基本計画では、2040年度の電源構成において再生可能エネルギーの割合を約40〜50%に引き上げることが目標として掲げられました。再生可能エネルギーは「主力電源化」を徹底するとされ、地域との共生と国民負担の抑制を図りながら最大限の導入を推進する方針です。

</QuizAnswer>

## 問9

原子力発電に関する記述として、最も適切なものはどれか。

1. 発電時にCO₂を大量に排出するため、脱炭素に貢献しない
2. 発電時にCO₂をほとんど排出しないが、放射性廃棄物の処理が課題である
3. 日本では2011年以降、すべての原子力発電所が永久に停止されている
4. ウラン燃料は再生可能エネルギーに分類される


<QuizAnswer>

**答え: 2**

原子力発電は発電時にCO₂をほとんど排出しないため、脱炭素の観点では利点がありますが、使用済み核燃料の処理・処分（高レベル放射性廃棄物の最終処分）が長年の課題となっています。日本では2011年の東京電力福島第一原発事故後に全基が停止しましたが、その後、新規制基準に適合した原発は順次再稼働しています。

</QuizAnswer>

## 問10

温室効果のメカニズムに関する説明として、正しいものはどれか。

1. 温室効果ガスは太陽からの可視光線を吸収して地球を温める
2. 温室効果ガスは地表から放射される赤外線を吸収し、再び地表に向けて放射することで気温を上昇させる
3. 温室効果は人間活動によって初めて生じた現象であり、自然界には存在しない
4. 温室効果ガスの濃度が高まると地球の気温は低下する


<QuizAnswer>

**答え: 2**

温室効果とは、地表から宇宙に向けて放射される赤外線（熱）の一部を温室効果ガスが吸収し、再び地表に向けて放射することで、地表付近の気温を高く保つ現象です。温室効果自体は自然界に元から存在し、これがなければ地球の平均気温は約マイナス19℃になるとされています。問題は、人間活動によって温室効果ガスの濃度が急速に上昇し、温暖化が加速していることです。

</QuizAnswer>
