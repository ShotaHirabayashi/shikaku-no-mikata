## 問1

日本の環境政策の根幹となる「環境基本法」が制定された年として正しいものはどれか。

1. 1967年
2. 1972年
3. 1993年
4. 2000年


<QuizAnswer>

**答え: 3**

環境基本法は1993年（平成5年）に制定されました。それ以前の公害対策基本法（1967年制定）と自然環境保全法を統合・発展させる形で、環境保全に関する基本理念と施策の基本方針を定めた法律です。環境基本法に基づき、政府は環境基本計画を策定する義務を負っています。

</QuizAnswer>

## 問2

2024年5月に閣議決定された「第六次環境基本計画」の計画期間として正しいものはどれか。

1. 2024年～2028年
2. 2024年～2030年
3. 2024年～2035年
4. 2024年～2050年


<QuizAnswer>

**答え: 2**

第六次環境基本計画の計画期間は2024年～2030年です。本計画は第一次環境基本計画の策定から30年の節目に策定され、「現在及び将来の国民一人一人のウェルビーイング／高い生活の質」の実現を環境政策の最上位目標として掲げていることが大きな特徴です。

</QuizAnswer>

## 問3

環境影響評価法（環境アセスメント法）に関する記述として、最も適切なものはどれか。

1. 環境アセスメントの実施主体は、国や地方自治体である
2. 対象事業は「第一種事業」と「第二種事業」に分類される
3. 環境アセスメントは事業完了後に環境への影響を評価する制度である
4. 環境影響評価法は1972年に制定された


<QuizAnswer>

**答え: 2**

環境影響評価法では、対象事業を規模が大きく必ず環境アセスメントを行う「第一種事業」と、実施するかどうかを個別に判断する「第二種事業」に分類しています。環境アセスメントの実施主体は事業者自身であり、事業の実施前に調査・予測・評価を行う制度です。環境影響評価法は1997年に制定されました。

</QuizAnswer>

## 問4

国連環境計画（UNEP）の本部が置かれている都市として正しいものはどれか。

1. ニューヨーク
2. ジュネーブ
3. ナイロビ
4. パリ


<QuizAnswer>

**答え: 3**

UNEPの本部はケニアのナイロビに置かれています。1972年の国連人間環境会議（ストックホルム会議）を受けて設立され、国連の主要な機関の本部を発展途上国に置いた最初の事例としても知られています。環境分野における国際協力の推進や、多くの国際環境条約の交渉を主導する役割を果たしています。

</QuizAnswer>

## 問5

1972年にスウェーデンのストックホルムで開催され、「かけがえのない地球（Only One Earth）」をスローガンとした国際会議の名称はどれか。

1. 地球サミット（国連環境開発会議）
2. 国連人間環境会議
3. 持続可能な開発に関する世界首脳会議
4. 国連環境特別総会


<QuizAnswer>

**答え: 2**

1972年にストックホルムで開催された「国連人間環境会議」は、環境問題を取り上げた最初の大規模な国際会議です。この会議で「人間環境宣言」が採択され、UNEPの設立が決定されました。なお、地球サミット（選択肢1）は1992年にリオデジャネイロで開催された会議です。

</QuizAnswer>

## 問6

グリーン購入法に関する記述として、最も適切なものはどれか。

1. 民間企業に環境配慮製品の購入を義務付ける法律である
2. 国等の機関に環境物品等の調達を推進する義務を課す法律である
3. 環境に有害な製品の製造を禁止する法律である
4. 消費者に環境ラベル付き商品の購入を義務付ける法律である


<QuizAnswer>

**答え: 2**

グリーン購入法（正式名称：国等による環境物品等の調達の推進等に関する法律）は、2000年に制定され2001年に施行されました。国や独立行政法人等に対して環境負荷の少ない物品・サービスの調達を推進する義務を課しています。民間企業や消費者への義務付けではなく、国等の率先的取り組みにより社会全体のグリーン購入を促進することを目的としています。

</QuizAnswer>

## 問7

ISO14001に関する記述として、誤っているものはどれか。

1. 環境マネジメントシステムに関する国際規格である
2. PDCAサイクルに基づく継続的改善を求めている
3. 認証を取得すると環境関連法規の遵守が免除される
4. 組織の規模や業種を問わず適用できる


<QuizAnswer>

**答え: 3**

ISO14001は環境マネジメントシステム（EMS）の国際規格であり、認証を取得しても環境関連法規の遵守が免除されることはありません。むしろ、法規制の遵守は ISO14001の基本的な要求事項の一つです。PDCAサイクル（計画→実行→点検→見直し）による継続的改善を求め、あらゆる規模・業種の組織が取得可能です。

</QuizAnswer>

## 問8

日本の「エコマーク」制度に関する記述として、正しいものはどれか。

1. 経済産業省が直接審査・認定を行っている
2. 商品のライフサイクル全体を考慮した認定基準が設けられている
3. 食品と医薬品のみを対象とした環境ラベルである
4. 2010年に創設された比較的新しい制度である


<QuizAnswer>

**答え: 2**

エコマークは、公益財団法人日本環境協会が第三者機関として審査・認定を行う環境ラベルで、商品の「ライフサイクル全体」（原料調達から製造、使用、廃棄・リサイクルまで）を考慮した認定基準が設けられています。1989年に創設され、文具や日用品、建材など幅広い商品カテゴリーが対象です。

</QuizAnswer>

## 問9

ESG投資の「ESG」が意味する3つの要素の組み合わせとして正しいものはどれか。

1. Economy（経済）、Society（社会）、Government（政府）
2. Environment（環境）、Safety（安全）、Governance（ガバナンス）
3. Environment（環境）、Social（社会）、Governance（ガバナンス）
4. Ecology（生態学）、Social（社会）、Green（グリーン）


<QuizAnswer>

**答え: 3**

ESGとは、Environment（環境）、Social（社会）、Governance（企業統治）の頭文字を取ったものです。ESG投資は、財務情報だけでなく、環境・社会・ガバナンスの非財務情報も考慮して投資判断を行う手法であり、2006年に国連が提唱した「責任投資原則（PRI）」をきっかけに世界的に広まりました。

</QuizAnswer>

## 問10

環境分野における資金調達手法「グリーンボンド」に関する記述として、最も適切なものはどれか。

1. 政府のみが発行できる特別な国債である
2. 環境改善効果のある事業（グリーンプロジェクト）に資金使途を限定した債券である
3. 環境税の一種として企業から徴収される課徴金である
4. 環境NGOへの寄付を目的とした金融商品である


<QuizAnswer>

**答え: 2**

グリーンボンドとは、再生可能エネルギーや省エネルギー、環境汚染防止など、環境改善効果のあるグリーンプロジェクトに資金使途を限定して発行される債券です。政府だけでなく企業や地方自治体なども発行でき、日本では2014年に日本政策投資銀行が国内初のグリーンボンドを発行しました。近年、発行額は急速に拡大しています。

</QuizAnswer>
