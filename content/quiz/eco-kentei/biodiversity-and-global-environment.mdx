## 問1

生物多様性条約では、生物多様性を3つのレベルで捉えている。次のうち、その3つのレベルの組み合わせとして正しいものはどれか。

1. 遺伝子の多様性・種の多様性・生態系の多様性
2. 遺伝子の多様性・個体の多様性・景観の多様性
3. 種の多様性・群集の多様性・生態系の多様性
4. 個体の多様性・種の多様性・地域の多様性


<QuizAnswer>

**答え: 1**

生物多様性条約では、生物多様性を「遺伝子の多様性」「種の多様性」「生態系の多様性」の3つのレベルで定義しています。遺伝子の多様性は同じ種でも個体ごとに遺伝的な違いがあること、種の多様性はさまざまな種が存在すること、生態系の多様性は森林・湿地・海洋など多様な環境が存在することを意味します。

</QuizAnswer>

## 問2

2010年に愛知県名古屋市で開催された生物多様性条約第10回締約国会議（COP10）で採択された、遺伝資源へのアクセスと利益配分（ABS）に関する国際的な枠組みは何か。

1. カルタヘナ議定書
2. ラムサール条約
3. 名古屋議定書
4. ワシントン条約


<QuizAnswer>

**答え: 3**

名古屋議定書は、2010年のCOP10で採択された、遺伝資源の取得の機会（アクセス）とその利用から生じる利益の公正かつ衡平な配分（ABS: Access and Benefit-Sharing）について定めた国際的な枠組みです。カルタヘナ議定書は遺伝子組換え生物の安全な取り扱いに関するもので、名古屋議定書とは異なります。

</QuizAnswer>

## 問3

オゾン層を破壊する物質の規制に関する国際的な取り決めとして、1987年に採択された議定書はどれか。

1. 京都議定書
2. モントリオール議定書
3. バーゼル条約
4. ストックホルム条約


<QuizAnswer>

**答え: 2**

モントリオール議定書は1987年に採択され、オゾン層を破壊するフロン（CFC）などの物質の生産・消費の段階的な削減を定めた国際的な枠組みです。この議定書は最も成功した環境条約の一つとされ、オゾン層破壊物質は1989年と比較して99%削減されました。京都議定書は温室効果ガスの削減に関するものです。

</QuizAnswer>

## 問4

海洋プラスチック汚染に関する説明として、最も適切なものはどれか。

1. マイクロプラスチックとは、直径1mm以下のプラスチック片のことである
2. 海洋プラスチックごみは海面に浮かぶもののみが問題とされている
3. マイクロプラスチックとは、直径5mm以下の微細なプラスチック片のことである
4. 海洋プラスチック汚染は沿岸域に限定された問題である


<QuizAnswer>

**答え: 3**

マイクロプラスチックとは、直径5mm以下の微細なプラスチック片のことを指します。紫外線や波の作用で細かく砕かれたプラスチックが海洋に蓄積し、有害化学物質を吸着する性質があるため、海洋生態系への深刻な影響が懸念されています。海洋プラスチックごみは海面だけでなく深海底にも沈み、沿岸域に限らず世界中の海洋に広がっています。

</QuizAnswer>

## 問5

酸性雨の一般的な定義として正しいものはどれか。

1. pH7.0以下の雨
2. pH6.5以下の雨
3. pH6.0以下の雨
4. pH5.6以下の雨


<QuizAnswer>

**答え: 4**

酸性雨は一般にpH5.6以下の雨と定義されます。通常の雨は大気中の二酸化炭素が溶け込むことでpH5.6程度の弱酸性を示しますが、硫黄酸化物（SOx）や窒素酸化物（NOx）が大気中で硫酸や硝酸に変化し雨に溶け込むことで、さらに酸性度が高くなります。酸性雨は森林の枯死や湖沼の酸性化、建造物の腐食など広範な被害をもたらします。

</QuizAnswer>

## 問6

2022年12月に開催された生物多様性条約COP15で採択された「昆明・モントリオール生物多様性枠組」における、2030年までの保全目標「30by30」の内容として正しいものはどれか。

1. 温室効果ガスを30%削減する
2. 陸と海の30%以上を保全する
3. 絶滅危惧種を30%減少させる
4. 外来種を30%削減する


<QuizAnswer>

**答え: 2**

「30by30（サーティ・バイ・サーティ）」目標とは、2030年までに地球上の陸域および海域の少なくとも30%を効果的に保全・管理するという目標です。これは2022年12月にカナダ・モントリオールで開催されたCOP15で採択された「昆明・モントリオール生物多様性枠組」の中核的な目標の一つであり、愛知目標の後継として位置づけられています。

</QuizAnswer>

## 問7

絶滅のおそれのある野生動植物の国際取引を規制する条約として正しいものはどれか。

1. ラムサール条約
2. 生物多様性条約
3. ワシントン条約（CITES）
4. ボン条約


<QuizAnswer>

**答え: 3**

ワシントン条約（CITES：絶滅のおそれのある野生動植物の種の国際取引に関する条約）は、1973年に採択され、絶滅のおそれのある野生動植物の国際的な商取引を規制しています。対象種は付属書Ⅰ（商業取引禁止）、付属書Ⅱ（取引許可制）、付属書Ⅲ（各国指定種）の3段階で規制されます。ラムサール条約は湿地の保全に関する条約です。

</QuizAnswer>

## 問8

日本における外来種問題に関する記述として、最も適切なものはどれか。

1. 外来種とは外国から持ち込まれた生物のみを指す
2. 特定外来生物に指定された生物は、飼育・栽培・運搬・輸入が原則禁止される
3. 外来種は在来種よりも生態系への適応力が低いため問題は軽微である
4. 特定外来生物法では植物は規制の対象外である


<QuizAnswer>

**答え: 2**

特定外来生物による生態系等に係る被害の防止に関する法律（外来生物法）では、生態系や人の生命・身体、農林水産業に被害を及ぼす外来生物を「特定外来生物」に指定し、飼育・栽培・保管・運搬・輸入・譲渡等を原則禁止しています。なお、外来種には国内の他の地域から持ち込まれた「国内外来種」も含まれ、植物も規制対象です。

</QuizAnswer>

## 問9

モントリオール議定書の2016年の改正（キガリ改正）で新たに規制対象に加えられた物質はどれか。

1. CFC（クロロフルオロカーボン）
2. HCFC（ハイドロクロロフルオロカーボン）
3. HFC（ハイドロフルオロカーボン）
4. 四塩化炭素


<QuizAnswer>

**答え: 3**

2016年のキガリ改正では、オゾン層は破壊しないものの強力な温室効果を持つHFC（ハイドロフルオロカーボン）が新たに規制対象に加えられました。HFCはCFCやHCFCの代替物質として普及しましたが、温室効果ガスとしての影響が大きいため、段階的な削減が義務付けられました。日本では2036年までに基準年比85%削減を目標としています。

</QuizAnswer>

## 問10

砂漠化に関する記述として、最も適切なものはどれか。

1. 砂漠化は自然現象のみが原因であり、人間活動とは無関係である
2. 砂漠化対処条約は、砂漠化の影響を受ける国の砂漠化防止を目的として1994年に採択された
3. 砂漠化の問題はアフリカ大陸のみに限定されている
4. 砂漠化対処条約はモントリオール議定書の一部として採択された


<QuizAnswer>

**答え: 2**

国連砂漠化対処条約（UNCCD）は1994年に採択され、深刻な干ばつや砂漠化に直面する国々（特にアフリカ）における砂漠化の防止を目的としています。砂漠化は過放牧、過耕作、森林の過剰伐採などの人間活動と、気候変動などの自然要因が複合的に作用して進行します。アフリカ以外にもアジアや南米など世界各地で問題となっています。

</QuizAnswer>
