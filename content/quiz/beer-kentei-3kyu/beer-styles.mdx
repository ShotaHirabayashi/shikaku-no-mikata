## 問1

ビールの大分類において、エールビールに共通する発酵方法はどれか。

1. 下面発酵
2. 上面発酵
3. 自然発酵
4. 混合発酵


<QuizAnswer>

**答え: 2**

エールビールは上面発酵酵母（エール酵母）を用いて比較的高い温度（15〜25℃程度）で発酵させるビールの総称です。発酵中に酵母が液面に浮き上がる特徴があり、フルーティーで豊かな香りと深い味わいが生まれます。一方、ラガービールは下面発酵酵母を用い、低温でじっくり発酵させます。

</QuizAnswer>

## 問2

世界で最も広く飲まれているビアスタイルである「ピルスナー」の発酵分類として正しいものはどれか。

1. エール（上面発酵）
2. 自然発酵
3. ラガー（下面発酵）
4. 混合発酵


<QuizAnswer>

**答え: 3**

ピルスナーはラガービール（下面発酵）に分類されるビアスタイルです。チェコのピルゼン（プルゼニュ）で誕生し、黄金色の美しい外観と爽快なホップの苦味、すっきりとしたのどごしが特徴です。日本の大手ビールメーカーが製造する主力商品の多くもピルスナースタイルに該当します。

</QuizAnswer>

## 問3

IPA（インディア・ペールエール）の味わいの特徴として最も適切なものはどれか。

1. 酸味が強く、フルーツのような風味がある
2. ホップ由来の強い苦味と柑橘系の香りが特徴的である
3. 焙煎麦芽によるコーヒーのような風味がある
4. バナナやクローブのようなフルーティーな香りがある


<QuizAnswer>

**答え: 2**

IPAはホップを大量に使用するビアスタイルで、ホップ由来の強めの苦味と、オレンジやグレープフルーツなどの柑橘類を思わせる華やかな香りが特徴です。もともとはイギリスからインドへビールを輸送する際、防腐効果のあるホップを多く加えたことが始まりとされています。

</QuizAnswer>

## 問4

スタウトとポーターの違いとして最も適切な説明はどれか。

1. スタウトはラガー、ポーターはエールに分類される
2. スタウトは小麦麦芽を主原料とし、ポーターは大麦麦芽を主原料とする
3. スタウトは麦芽化していない大麦をローストして使用する点が特徴である
4. ポーターはスタウトより色が薄く、ゴールド〜琥珀色である


<QuizAnswer>

**答え: 3**

スタウト（Stout）は「どっしりとした、強い」という意味で、ポーターから派生したビアスタイルです。ポーターとの大きな違いは、麦芽化せずに焙煎した大麦（ローストバーレイ）を使用する点にあります。アイルランドのギネス社がこのスタイルを確立し、コーヒーやチョコレートのような風味と濃厚なコクが特徴です。

</QuizAnswer>

## 問5

ドイツ発祥の白ビール「ヴァイツェン」の特徴として正しいものはどれか。

1. オレンジピールとコリアンダーで風味付けされる
2. ホップの苦味が非常に強い
3. バナナやクローブを思わせる香りがあり、苦味が少ない
4. 自然発酵による強い酸味がある


<QuizAnswer>

**答え: 3**

ヴァイツェンはドイツの伝統的な小麦ビールで、ヴァイツェン酵母が生み出すバナナやクローブのような独特のフルーティーな香りが最大の特徴です。苦味が少なく、泡立ちが豊かで、まろやかな口当たりから「飲むパン」と例えられることもあります。

</QuizAnswer>

## 問6

ベルジャンホワイト（ベルギーの白ビール）に風味付けとして使われる代表的なスパイスの組み合わせはどれか。

1. シナモンとナツメグ
2. オレンジピールとコリアンダーシード
3. ジュニパーベリーとローズマリー
4. カルダモンとクミン


<QuizAnswer>

**答え: 2**

ベルジャンホワイトは、大麦麦芽と麦芽化していない小麦を原料とするベルギーの上面発酵ビールです。麦汁の煮沸時にオレンジピール（オレンジの皮）とコリアンダーシードを加えることで、爽やかで華やかな風味に仕上げるのが最大の特徴です。同じ白ビールでもドイツのヴァイツェンとはスパイスを使う点で大きく異なります。

</QuizAnswer>

## 問7

ベルギーの伝統的なビアスタイル「ランビック」の最大の特徴はどれか。

1. 大量のホップによる強烈な苦味
2. 培養酵母を使わず、空気中の野生酵母で自然発酵させる
3. 高アルコール度数（15%以上）で長期熟成する
4. 燻製麦芽を使用したスモーキーな香り


<QuizAnswer>

**答え: 2**

ランビックはベルギーのブリュッセル南西部パヨッテンラント地域で醸造される伝統的なビールで、培養した酵母を一切使用せず、空気中に浮遊する野生酵母や微生物によって自然発酵させるのが最大の特徴です。「クールシップ」と呼ばれる浅い冷却槽で麦汁を一晩さらし、強い酸味と独特の風味が生まれます。

</QuizAnswer>

## 問8

オクトーバーフェストで伝統的に提供されるビアスタイル「メルツェン」の名前の由来として正しいものはどれか。

1. 醸造家メルツェン氏の名前に由来する
2. ドイツ語で「3月」を意味するMärzに由来する
3. ドイツ語で「祝祭」を意味するMerzenに由来する
4. 原料の麦芽品種メルツェンに由来する


<QuizAnswer>

**答え: 2**

メルツェン（Märzen）はドイツ語で「3月」を意味する「März」に由来します。かつてバイエルン州では夏季（4月〜9月）の醸造が禁止されていたため、禁止前の3月に仕込み、長期熟成させたビールがメルツェンです。まろやかでモルトのリッチな風味が特徴で、オクトーバーフェストビールとも呼ばれます。

</QuizAnswer>

## 問9

次のビアスタイルのうち、ラガー（下面発酵）に分類されるものはどれか。

1. ペールエール
2. スタウト
3. シュバルツ
4. ヴァイツェン


<QuizAnswer>

**答え: 3**

シュバルツはドイツ語で「黒」を意味するラガービール（下面発酵）です。見た目は真っ黒ですが、ラガー酵母による発酵のため、後味はすっきりと軽快な飲み口になります。一方、ペールエール・スタウト・ヴァイツェンはいずれもエール（上面発酵）に分類されるビアスタイルです。

</QuizAnswer>

## 問10

同じ「白ビール」に分類されるヴァイツェンとベルジャンホワイトの違いとして正しいものはどれか。

1. ヴァイツェンは下面発酵だが、ベルジャンホワイトは上面発酵である
2. ヴァイツェンは酵母由来の香りが特徴だが、ベルジャンホワイトはスパイスで風味付けする
3. ヴァイツェンはベルギー発祥だが、ベルジャンホワイトはドイツ発祥である
4. ヴァイツェンは大麦100%だが、ベルジャンホワイトは小麦を使用する


<QuizAnswer>

**答え: 2**

ヴァイツェンとベルジャンホワイトはどちらも小麦を使用する上面発酵の白ビールですが、風味の出し方が異なります。ヴァイツェンはヴァイツェン酵母が生み出すバナナやクローブのような香りが特徴で、スパイスは使用しません。一方、ベルジャンホワイトはオレンジピールやコリアンダーシードなどのスパイスを加えて爽やかな風味を生み出すのが特徴です。

</QuizAnswer>
