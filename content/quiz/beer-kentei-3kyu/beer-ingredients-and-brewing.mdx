## 問1

ビール醸造に主に使用される大麦の種類はどれか。

1. 四条大麦
2. 六条大麦
3. 二条大麦
4. 八条大麦


<QuizAnswer>

**答え: 3**

ビール醸造には主に二条大麦が使用されます。二条大麦は粒が大きくでんぷん含有量が多いこと、粒の大きさや形のばらつきが少ないこと、発芽力が旺盛で酵素の力が強いことから、ビール製造に最も適した品種とされています。六条大麦は主に麦茶や味噌などに使われます。

</QuizAnswer>

## 問2

製麦工程において、大麦を水に浸して吸水させる最初の工程を何というか。

1. 焙燥（ばいそう）
2. 浸麦（しんばく）
3. 糖化（とうか）
4. 煮沸（しゃふつ）


<QuizAnswer>

**答え: 2**

製麦工程の最初のステップは「浸麦（しんばく）」です。大麦を水に浸して十分な水分を吸収させ、発芽を促します。製麦は「浸麦→発芽→焙燥」の順で行われ、この工程を経て大麦は麦芽（モルト）になります。

</QuizAnswer>

## 問3

ホップの毬花（まりはな）の中にある、苦味成分や香り成分を含む黄色い粒状の物質を何というか。

1. タンニン
2. カラメル
3. ルプリン
4. デキストリン


<QuizAnswer>

**答え: 3**

ルプリンはホップの毬花の内部にある直径0.15〜0.20mm程度の黄色い顆粒状の物質です。この中にビールの苦味のもととなるα酸（アルファ酸）や、ビールに華やかな香りを与える精油成分が蓄えられています。ルプリンはビールの風味を決定づける非常に重要な成分です。

</QuizAnswer>

## 問4

ホップに含まれるα酸（アルファ酸）は、麦汁の煮沸によって異性化し、ビールに苦味を与える物質に変化する。この物質の名称はどれか。

1. イソアルファ酸
2. ポリフェノール
3. メラノイジン
4. フムロン


<QuizAnswer>

**答え: 1**

ホップに含まれるα酸（フムロンとも呼ばれる）は、煮沸の熱によって異性化（イソメリゼーション）し、「イソアルファ酸」に変化します。このイソアルファ酸がビールに爽快な苦味を与える主要成分です。なお、フムロン（選択肢4）はα酸の別名であり、煮沸前の状態を指します。

</QuizAnswer>

## 問5

ビール製造の仕込み工程において、麦芽中の酵素の働きによりでんぷんを糖に分解する工程を何というか。

1. 煮沸
2. ろ過
3. 糖化（マッシング）
4. 発酵


<QuizAnswer>

**答え: 3**

糖化（マッシング）は、粉砕した麦芽をお湯に混ぜ、60〜90分間かけて麦芽中のαアミラーゼやβアミラーゼなどの酵素の働きでデンプンを糖（主に麦芽糖）に分解する工程です。この工程で得られた液体が「麦汁（ばくじゅう）」と呼ばれ、ビールのもとになります。

</QuizAnswer>

## 問6

上面発酵酵母（エール酵母）の発酵温度として最も適切なものはどれか。

1. 0〜5℃
2. 15〜25℃
3. 35〜45℃
4. 50〜60℃


<QuizAnswer>

**答え: 2**

上面発酵酵母（エール酵母）は15〜25℃の比較的高い温度で発酵を行います。この温度帯ではフルーティーな香りのもとであるエステルなどが多く生成されます。一方、下面発酵酵母（ラガー酵母）は5〜10℃前後の低温で発酵し、すっきりとした味わいのビールを生み出します。

</QuizAnswer>

## 問7

ビール酵母がアルコール発酵で糖分を分解する際に生成される2つの主な物質の組み合わせとして正しいものはどれか。

1. アルコールと乳酸
2. アルコールと酢酸
3. アルコールと炭酸ガス
4. アルコールと窒素ガス


<QuizAnswer>

**答え: 3**

ビール酵母は麦汁中の糖分をアルコール（エタノール）と炭酸ガス（二酸化炭素）に分解します。これがアルコール発酵の基本的なメカニズムです。発酵で生じた炭酸ガスはビールの爽快な泡立ちのもとになり、アルコールとともにビールの基本的な特徴を形成します。

</QuizAnswer>

## 問8

ホップがビールに果たす役割として、正しくないものはどれか。

1. ビールに苦味と香りを与える
2. 麦汁中のでんぷんを糖に分解する
3. 雑菌の繁殖を抑えて腐敗を防ぐ
4. ビールの泡もちを良くする


<QuizAnswer>

**答え: 2**

でんぷんを糖に分解するのは麦芽に含まれる酵素（アミラーゼ）の働きであり、ホップの役割ではありません。ホップの主な役割は、ビールに独特の苦味と香りを与えること、抗菌作用によって雑菌の繁殖を抑えること、タンパク質を凝固・沈殿させてビールを澄んだものにすること、そして泡もちを良くすることです。

</QuizAnswer>

## 問9

麦芽を高温で焙燥し、色や風味を加えるために使用される麦芽の総称はどれか。

1. ベースモルト
2. スペシャルモルト
3. グリーンモルト
4. ドライモルト


<QuizAnswer>

**答え: 2**

スペシャルモルトは100℃以上の高温で焙燥して作られる麦芽で、ビールに色や独特の風味（カラメル、チョコレート、コーヒーなど）を加えるために使用されます。一方、ベースモルト（選択肢1）は低温で焙燥し、でんぷんと酵素の活性を残した麦芽で、糖化に不可欠な基本の麦芽です。

</QuizAnswer>

## 問10

ビール製造における「熟成」工程の説明として最も適切なものはどれか。

1. 麦芽を粉砕して湯と混ぜ、でんぷんを糖に変える工程
2. 麦汁にホップを加えて煮沸し、苦味と香りを付ける工程
3. 発酵後のビールを低温で貯蔵し、味をまろやかに整える工程
4. 完成したビールをろ過して酵母を取り除く工程


<QuizAnswer>

**答え: 3**

熟成（貯酒）は、主発酵を終えたビールを低温で約1か月間貯蔵する工程です。この間にビールの味わいがまろやかになり、香味が調和するとともに、炭酸ガスがビールに溶け込んで適切なガス圧に調整されます。選択肢1は糖化、選択肢2は煮沸の説明にあたります。

</QuizAnswer>
