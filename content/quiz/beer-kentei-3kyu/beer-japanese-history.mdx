## 問1

幕末の嘉永6年（1853年）頃、蘭書を参考に日本人として初めてビールを試醸したとされる蘭方医は誰か。

1. 緒方洪庵
2. 川本幸民
3. 中川清兵衛
4. 高野長英


<QuizAnswer>

**答え: 2**

川本幸民は蘭方医・化学者で、ペリー来航の頃にオランダの書物を参考にして江戸の私宅でビールを試醸したとされています。これが日本人によるビール醸造の最初の記録として知られています。

</QuizAnswer>

## 問2

1876年に北海道に設立され、翌年「冷製札幌ビール」を発売した醸造所の名称はどれか。

1. 麒麟麦酒醸造所
2. スプリングバレー・ブルワリー
3. 開拓使麦酒醸造所
4. 大阪麦酒醸造所


<QuizAnswer>

**答え: 3**

明治政府の北海道開拓使が設置した「開拓使麦酒醸造所」は、ドイツで醸造技術を学んだ中川清兵衛を迎えて1876年に完成しました。1877年に発売された「冷製札幌ビール」は、現在のサッポロビールの源流にあたります。

</QuizAnswer>

## 問3

1906年に札幌麦酒・日本麦酒・大阪麦酒の3社が合併して誕生し、当時の国内ビール市場シェア約7割を占めた会社はどれか。

1. 麒麟麦酒株式会社
2. 日本麦酒鑛泉株式会社
3. 帝国麦酒株式会社
4. 大日本麦酒株式会社


<QuizAnswer>

**答え: 4**

1906年3月、札幌麦酒・日本麦酒・大阪麦酒の3社が合同して大日本麦酒株式会社が設立されました。国内ビール市場の約7割を占める巨大企業となり、スエズ運河以東で最大のビール会社に成長しました。戦後の1949年に過度経済力集中排除法により朝日麦酒と日本麦酒（のちのサッポロビール）に分割されました。

</QuizAnswer>

## 問4

大日本麦酒の社長として同社をスエズ運河以東最大のビール会社に育て、「東洋のビール王」と呼ばれた人物は誰か。

1. 渋沢栄一
2. 馬越恭平
3. 鳥井信治郎
4. 大倉喜八郎


<QuizAnswer>

**答え: 2**

馬越恭平は大日本麦酒の社長に就任し、積極的な経営手腕で同社をスエズ運河以東最大のビール会社へと成長させました。その功績から「東洋のビール王」と称されています。また、日本初のビヤホールを開設するアイデアを出した人物としても知られています。

</QuizAnswer>

## 問5

1899年（明治32年）に東京・銀座に開業した、日本初のビヤホールの名称はどれか。

1. 銀座ライオン
2. 恵比寿ビヤホール
3. 麒麟ビヤホール
4. サッポロビヤホール


<QuizAnswer>

**答え: 2**

1899年8月、東京・銀座（現在の銀座8丁目付近）に「恵比寿ビヤホール（恵比寿ビール BEER HALL）」が開業しました。日本麦酒の恵比寿ビールの宣伝を目的に開設され、工場直送の生ビールが半リットル10銭で提供されました。これが日本初のビヤホールとされています。

</QuizAnswer>

## 問6

1994年の酒税法改正で、ビール製造免許に必要な最低製造数量が大幅に引き下げられた。改正後の最低製造数量として正しいものはどれか。

1. 年間6キロリットル
2. 年間60キロリットル
3. 年間200キロリットル
4. 年間600キロリットル


<QuizAnswer>

**答え: 2**

1994年の酒税法改正により、ビールの最低製造数量がそれまでの年間2,000キロリットルから年間60キロリットルへと大幅に引き下げられました。この規制緩和によって、大手メーカー以外の中小事業者もビール製造免許を取得できるようになり、全国各地で「地ビール」が誕生する契機となりました。

</QuizAnswer>

## 問7

2018年4月の酒税法改正で、「ビール」と定義される麦芽比率の基準が変更された。改正後の麦芽比率の基準として正しいものはどれか。

1. 麦芽比率25%以上
2. 麦芽比率50%以上
3. 麦芽比率67%以上
4. 麦芽比率75%以上


<QuizAnswer>

**答え: 2**

2018年4月の酒税法改正により、ビールの定義における麦芽比率の基準がそれまでの67%以上から50%以上に引き下げられました。また同時に、果実・香辛料などの副原料も麦芽重量の5%以内であれば使用が認められるようになり、ビールの多様化が進みました。

</QuizAnswer>

## 問8

酒税法上、「発泡酒」の税率区分において、麦芽比率に基づく区分として正しいものはどれか。

1. 50%以上 / 25%以上50%未満 / 25%未満の3区分
2. 67%以上 / 50%以上67%未満 / 50%未満の3区分
3. 50%以上 / 25%以上50%未満の2区分
4. 67%以上 / 25%以上67%未満 / 25%未満の3区分


<QuizAnswer>

**答え: 1**

発泡酒は酒税法上、麦芽比率によって「50%以上」「25%以上50%未満」「25%未満」の3区分に分けられ、麦芽比率が低いほど税率が低く設定されています。2018年の改正でビールの麦芽比率基準が50%以上となったため、発泡酒の最上位区分も50%以上に変更されました。

</QuizAnswer>

## 問9

1990年代半ばの規制緩和で生まれた「地ビールブーム」が一度衰退した後、2000年代半ばから品質を重視した新たなムーブメントとして再び注目を集めた際の呼び名はどれか。

1. プレミアムビール
2. マイクロブルワリー
3. クラフトビール
4. ホームブルー


<QuizAnswer>

**答え: 3**

1994年の規制緩和をきっかけに全国で約300もの地ビール醸造所が誕生しましたが、品質のばらつきなどから2003年頃にブームは一旦終息しました。その後、品質と個性を重視する醸造所が生き残り、2005年頃から「クラフトビール」という呼び名で再び注目を集めるようになりました。

</QuizAnswer>

## 問10

酒税法上、いわゆる「第三のビール（新ジャンル）」に該当する商品の特徴として正しいものはどれか。

1. 麦芽比率50%以上で、副原料に果実を使用したもの
2. 麦芽を使用せず、または発泡酒にスピリッツを加えたもの
3. 麦芽比率25%以上50%未満で、ホップを使用しないもの
4. 麦芽比率67%以上で、海外で製造・輸入されたもの


<QuizAnswer>

**答え: 2**

「第三のビール（新ジャンル）」とは、麦芽を使用せずエンドウたんぱくや大豆たんぱくなどを原料としたものや、発泡酒にスピリッツ（蒸留酒）を加えた「リキュール（発泡性）」に分類されるものを指します。ビールや発泡酒より低い税率が適用されていたため、低価格帯の商品として人気を集めました。

</QuizAnswer>
