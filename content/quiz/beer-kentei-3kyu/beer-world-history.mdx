## 問1

ビールの醸造に関する最古の文字記録を残したとされる古代文明はどれか。

1. 古代ギリシャ文明
2. 古代エジプト文明
3. シュメール文明
4. インダス文明


<QuizAnswer>

**答え: 3**

紀元前3000年頃、メソポタミアのシュメール人が楔形文字でビール醸造に関する記録を残しており、これが文献上最古のビールの記録とされています。シュメールではビールは最も身近な飲み物であり、「甕の中の水と穀物の粒」を表す楔形文字でビールが示されていました。

</QuizAnswer>

## 問2

古代エジプトでピラミッド建設に従事した労働者に対して、毎日支給されていたとされるビールの量はおよそどのくらいか。

1. 約1リットル
2. 約3リットル
3. 約5リットル
4. 約10リットル


<QuizAnswer>

**答え: 3**

古代エジプトでは、ピラミッド建設に動員された労働者に毎日約5リットルのビールが支給されていたとされています。ビールは水分やミネラルの補給、疲労回復に役立ち、「ビールがピラミッドを造った」と称されることもあります。

</QuizAnswer>

## 問3

中世ヨーロッパにおいて、修道院でビール醸造が盛んに行われた主な理由として最も適切なものはどれか。

1. 修道院の税収を確保するため
2. 飲用に適した安全な水の確保が困難だったため
3. 国王の命令でビール製造が義務づけられていたため
4. ワインの原料であるブドウが不足していたため


<QuizAnswer>

**答え: 2**

中世のヨーロッパでは飲用に適した清潔な水を確保するのが難しく、煮沸工程を経るビールやワインが安全な飲料として重宝されました。また、修道士の間では「ビールは液体のパン」と考えられ、断食中の栄養源としても醸造が奨励されました。

</QuizAnswer>

## 問4

1516年に「ビールは大麦、ホップ、水のみを原料とすべし」と定めた「ビール純粋令（ラインハイツゲボット）」を制定したのは誰か。

1. フリードリヒ大王
2. バイエルン公ヴィルヘルム4世
3. 神聖ローマ皇帝カール5世
4. プロイセン王フリードリヒ・ヴィルヘルム1世


<QuizAnswer>

**答え: 2**

ビール純粋令（Reinheitsgebot）は、1516年4月23日にバイエルン公ヴィルヘルム4世によって制定されました。この法律は粗悪なビールから消費者を守ること、パンの原料である小麦をビール醸造から守ること（食糧確保）、ビール価格の安定などを目的としていました。現在でもドイツの多くの醸造所がこの精神を指針としています。

</QuizAnswer>

## 問5

1842年に世界初の黄金色のピルスナービールを醸造した醸造家ヨーゼフ・グロールが活躍した都市はどこか。

1. ミュンヘン（ドイツ）
2. ウィーン（オーストリア）
3. プルゼニ（チェコ）
4. ブリュッセル（ベルギー）


<QuizAnswer>

**答え: 3**

1842年、バイエルン出身の醸造家ヨーゼフ・グロールは、チェコのプルゼニ（ピルゼン）で世界初の黄金色のラガービール「ピルスナー・ウルケル」を醸造しました。プルゼニ特有の軟水とザーツ産ホップ、モラビア産淡色モルトを使い、バイエルン式下面発酵で造られたこのビールが、現在世界で最も普及しているピルスナースタイルの原点です。

</QuizAnswer>

## 問6

19世紀後半、ラガービールの通年醸造を可能にしたアンモニア式冷凍機を発明し、1873年にミュンヘンのシュパーテン醸造所に第1号機を設置した人物は誰か。

1. ルイ・パスツール
2. カール・フォン・リンデ
3. ガブリエル・セドルマイヤー
4. アントン・ドレハー


<QuizAnswer>

**答え: 2**

カール・フォン・リンデが発明したアンモニア式冷凍機は、24時間で6トンの氷を製造でき、それまで冬季や洞窟貯蔵に依存していたラガービールの醸造を季節や場所に関係なく行えるようにしました。この冷蔵技術の発展により、ラガービールは世界中に急速に普及し、エールに代わってビールの主流となりました。

</QuizAnswer>

## 問7

トラピストビールの名称を使用するために必要な条件として正しいものはどれか。

1. ベルギー国内の修道院で醸造されていること
2. トラピスト会修道院の監督下で醸造され、国際トラピスト会修道士協会（ITA）の認証を受けていること
3. 中世から続く伝統的なレシピで醸造されていること
4. ローマ教皇庁の公認を受けた醸造所で造られていること


<QuizAnswer>

**答え: 2**

トラピストビールの呼称は法的に保護されており、1997年に設立された国際トラピスト会修道士協会（ITA）の基準を満たした醸造所のみが「Authentic Trappist Product」のロゴを使用できます。トラピストビールを生産する修道院はベルギーだけでなく、オランダ、オーストリア、イタリアなど世界各地に存在し、2021年時点で11箇所が認証されています。

</QuizAnswer>

## 問8

1979年にアメリカで自家醸造（ホームブリューイング）を合法化し、後のクラフトビール革命の土台を築いたとされる大統領は誰か。

1. リチャード・ニクソン
2. ロナルド・レーガン
3. ジョージ・H・W・ブッシュ
4. ジミー・カーター


<QuizAnswer>

**答え: 4**

1979年、ジミー・カーター大統領は禁酒法時代から禁止されていた自家醸造（ホームブリューイング）を合法化する法案に署名しました。これをきっかけにアメリカ各地でホームブルワーが誕生し、やがて商業醸造所を開業する者が続出。1978年に90軒に満たなかった醸造所数は、2019年には8,000を超えるまでに成長しました。

</QuizAnswer>

## 問9

現在、世界最大のビール会社であるアンハイザー・ブッシュ・インベブ（ABインベブ）の本社が置かれている国はどこか。

1. アメリカ
2. オランダ
3. ドイツ
4. ベルギー


<QuizAnswer>

**答え: 4**

アンハイザー・ブッシュ・インベブ（ABインベブ）はベルギーのルーヴェンに本社を置く世界最大のビール会社です。2008年にベルギーのインベブがアメリカのアンハイザー・ブッシュを買収して誕生し、2016年にはイギリスのSABミラーも吸収合併。500以上のブランドを展開し、世界のビール市場で約3割のシェアを占めています。

</QuizAnswer>

## 問10

1863年にオランダで創業し、現在世界第2位のシェアを持つビール会社はどれか。

1. ハイネケン
2. カールスバーグ
3. ギネス
4. バス


<QuizAnswer>

**答え: 1**

ハイネケンは1863年にヘラルド・A・ハイネケンによってオランダのアムステルダムで創業されました。現在では170か国以上で販売される世界的ブランドに成長し、ABインベブに次ぐ世界第2位のビール会社となっています。緑色のボトルと赤い星のロゴが特徴的なブランドとして広く知られています。

</QuizAnswer>
