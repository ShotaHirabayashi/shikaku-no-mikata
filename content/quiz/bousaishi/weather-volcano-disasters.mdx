## 問1

台風による高潮の「吸い上げ効果」について、気圧が1hPa低下すると海面はおよそ何cm上昇するか。

1. 約0.1cm
2. 約1cm
3. 約10cm
4. 約100cm


<QuizAnswer>

**答え: 2**

台風の中心付近は気圧が低いため、海面が持ち上げられます。この吸い上げ効果では、気圧が1hPa下がるごとに海面が約1cm上昇します。例えば、中心気圧950hPaの台風が平常1013hPaの海域に接近すると、約63cmの海面上昇が生じる計算になります。

</QuizAnswer>

## 問2

高潮における「吹き寄せ効果」による海面上昇は、風速とどのような関係にあるか。

1. 風速に比例する
2. 風速の2乗に比例する
3. 風速の3乗に比例する
4. 風速に関係なく一定である


<QuizAnswer>

**答え: 2**

吹き寄せ効果による海面上昇は風速の2乗に比例します。つまり、風速が2倍になると海面上昇は4倍になります。さらに、東京湾・伊勢湾・大阪湾のようなV字型の湾では、奥に向かって海面上昇が増幅されるため、特に大きな被害が発生しやすくなります。

</QuizAnswer>

## 問3

内水氾濫の説明として最も適切なものはどれか。

1. 河川の堤防が決壊して市街地に浸水する現象
2. 排水路や下水道の排水能力を超えた雨水が地上にあふれる現象
3. 津波が河川を遡上して沿岸部が浸水する現象
4. 地下水位の上昇により地下施設が浸水する現象


<QuizAnswer>

**答え: 2**

内水氾濫は、排水路や下水道の排水能力を超える降雨により、雨水が地上にあふれ出す現象です。アスファルト舗装の多い都市部で発生しやすい特徴があります。一方、選択肢1のように河川の堤防が越水・決壊して浸水する現象は「外水氾濫」と呼ばれ、区別されています。

</QuizAnswer>

## 問4

土砂災害のうち、時速20〜40kmと自動車並みの速度で渓流を流れ下る現象はどれか。

1. 地すべり
2. がけ崩れ
3. 土石流
4. 深層崩壊


<QuizAnswer>

**答え: 3**

土石流は、渓流の土砂・石・流木が水と一体となって時速20〜40kmもの速度で流れ下る現象です。地すべりは通常1日数mm程度とゆっくり移動し、がけ崩れは斜面が瞬間的に崩れ落ちる現象です。土石流は発生すると短時間で下流に到達するため、早めの避難が重要です。

</QuizAnswer>

## 問5

土砂災害の種類のうち、発生件数に比べて死者の割合が最も高いとされるものはどれか。

1. 土石流
2. がけ崩れ
3. 地すべり
4. 側方流動


<QuizAnswer>

**答え: 2**

がけ崩れ（斜面崩壊）は、斜面の表層部分が突然崩れ落ちる現象で、前兆が少なく瞬間的に発生するため、逃げ遅れが多くなります。年間約700件と発生件数も多く、発生件数あたりの死者の割合が最も高い土砂災害です。日頃から危険箇所を確認し、大雨時は早めの避難行動が求められます。

</QuizAnswer>

## 問6

火砕流の特徴として正しいものはどれか。

1. 溶岩がゆっくり斜面を流れ下り、温度は約300度Cである
2. 高温の火山ガスと火山灰等が一体となり、時速100km以上で流れ下ることがある
3. 火山砕屑物と雨水が混合して流れ下り、噴火後に降雨があると発生する
4. 火山灰が上空に噴き上げられ、風に乗って広範囲に降下する現象である


<QuizAnswer>

**答え: 2**

火砕流は、高温（600〜700度C）の火山ガスと火山灰・軽石などが混然一体となって高速で斜面を流れ下る現象です。速度は時速100kmを超え、最大で秒速100m（時速360km）に達することもあります。選択肢3はラハール（火山泥流）、選択肢4は降灰の説明です。火砕流は避難の時間的猶予がほぼなく、最も危険な火山現象の一つです。

</QuizAnswer>

## 問7

火山噴火警戒レベルにおいて、「入山規制」に該当するレベルはどれか。

1. レベル2
2. レベル3
3. レベル4
4. レベル5


<QuizAnswer>

**答え: 2**

火山噴火警戒レベル3は「入山規制」で、登山禁止や危険地域への立入規制が行われます。レベル2は「火口周辺規制」、レベル4は「高齢者等避難」で要配慮者の避難が求められ、レベル5は「避難」で危険な居住地域からの直ちに避難が必要です。なお、レベル1は「活火山であることに留意」であり、安全を意味するものではありません。

</QuizAnswer>

## 問8

線状降水帯の一般的な規模として最も適切なものはどれか。

1. 長さ5〜10km、幅1〜2km
2. 長さ50〜300km、幅20〜50km
3. 長さ500〜1000km、幅100〜200km
4. 長さ1000km以上、幅300km以上


<QuizAnswer>

**答え: 2**

線状降水帯は、次々と発生する発達した積乱雲が列をなし、数時間にわたってほぼ同じ場所を通過・停滞することで形成される降水域です。長さ50〜300km程度、幅20〜50km程度の線状の強い降水域が特徴で、同じ場所に長時間大雨が降り続くため、土砂災害や河川氾濫など甚大な被害をもたらします。

</QuizAnswer>

## 問9

ラハール（火山泥流）の説明として最も適切なものはどれか。

1. 火口から噴出した溶岩がゆっくり斜面を流れ下る現象
2. 火山砕屑物が水と混合して斜面を流れ下り、噴火後しばらくしてからも発生しうる現象
3. 火山ガスが地表付近に滞留し、周辺の生物に被害を及ぼす現象
4. マグマの圧力で山体が膨張し、山全体が崩壊する現象


<QuizAnswer>

**答え: 2**

ラハール（火山泥流）は、火山砕屑物が水と混合して山の斜面を流れ下る現象です。速度は時速数十kmから100kmを超えることもあります。重要な特徴として、噴火後に降雨があると発生することがあり、噴火から時間が経ってからも起こりうる点があります。1985年のコロンビア・ネバドデルルイス火山では、ラハールにより約23,000人が犠牲になりました。

</QuizAnswer>

## 問10

タイムライン（事前防災行動計画）の説明として最も適切なものはどれか。

1. 災害発生後の復旧・復興の手順を定めた計画
2. 災害の発生を前提に「いつ」「誰が」「何をするか」を時系列で整理した防災行動計画
3. 被災者への経済的支援の金額と手続きを定めた計画
4. 避難所の収容人数と運営体制を定めた計画


<QuizAnswer>

**答え: 2**

タイムラインは、災害の発生を前提に、起こり得る状況を想定し、「いつ」「誰が」「何をするか」を時系列で整理した事前防災行動計画です。主に台風や大雨など事前に予測可能な「進行型災害」を対象としています。広域タイムライン、市町村タイムライン、コミュニティタイムラインに加え、個人・家族単位で作成する「マイタイムライン」の普及も進められています。

</QuizAnswer>
