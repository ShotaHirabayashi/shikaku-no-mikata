## 問1

海溝型地震と内陸活断層型地震を比較した説明として、正しいものはどれか。

1. 内陸活断層型地震は海溝型地震よりも規模が大きく、マグニチュード9クラスになることが多い
2. 海溝型地震は震源が浅いため、揺れの範囲は震源直上の狭い範囲に限られる
3. 内陸活断層型地震は震源が浅く、震源直上では局所的に極めて強い揺れとなることがある
4. 海溝型地震では津波が発生することはなく、内陸活断層型地震で津波が発生する


<QuizAnswer>

**答え: 3**

内陸活断層型地震の震源の深さは約5〜20kmと比較的浅く、震源直上では極めて強い揺れが生じます。一方、海溝型地震はM8〜9クラスの大規模地震となりうるのが特徴で、広域に揺れが及び、大津波を引き起こす可能性があります。1995年の兵庫県南部地震（M7.3）は内陸活断層型地震の代表例です。

</QuizAnswer>

## 問2

津波の伝播速度に関する説明として、正しいものはどれか。

1. 津波の速度は水深に関係なく一定であり、常に時速約100kmで伝わる
2. 外洋の水深4,000m付近では、津波はジェット機並みの時速約700kmで伝わる
3. 津波は浅瀬に近づくほど速度が増し、波高は低くなる
4. 津波の伝播速度は波高に比例し、波が高いほど速く伝わる


<QuizAnswer>

**答え: 2**

津波の伝播速度は v=√(g×h)（g：重力加速度、h：水深）で求められ、水深が深いほど速くなります。外洋の水深4,000m付近では時速約700kmとジェット機並みの速度になります。浅瀬に近づくと速度は低下しますが、エネルギー保存の法則により波高が増大するため、沿岸部での被害が大きくなります。

</QuizAnswer>

## 問3

液状化現象が発生しやすい条件の組合せとして、最も適切なものはどれか。

1. 粘土質地盤・地下水位が深い・地盤が固い
2. 岩盤地盤・地下水位が浅い・地盤が緩い
3. 砂質地盤・地下水位が深い・地盤が固い
4. 砂質地盤・地下水位が浅い・地盤が緩い


<QuizAnswer>

**答え: 4**

液状化現象は、①砂質地盤（細粒〜中粒の砂で粒径が均一）、②地下水位が浅い（地表面から概ね10m以内）、③緩い地盤（N値が概ね20以下）の3条件が揃うと発生しやすくなります。埋立地、旧河道、干拓地などの地形で特にリスクが高く、震度5強以上かつ揺れの継続時間が長いほど発生しやすい傾向があります。

</QuizAnswer>

## 問4

建築基準法の耐震基準に関する説明として、正しいものはどれか。

1. 新耐震基準は1981年（昭和56年）6月1日に施行され、震度6強〜7程度でも倒壊・崩壊しないことを目標としている
2. 新耐震基準は1995年の阪神・淡路大震災を契機として施行された
3. 旧耐震基準の建物は震度6強〜7程度の揺れでも倒壊しないように設計されている
4. 建物が新耐震基準に該当するかどうかは、建物の竣工日（完成日）で判定する


<QuizAnswer>

**答え: 1**

新耐震基準は1978年の宮城県沖地震の被害を契機に建築基準法施行令が改正され、1981年6月1日に施行されました。旧耐震基準が「震度5程度で倒壊しない」ことを目標としていたのに対し、新耐震基準は「震度6強〜7程度でも倒壊・崩壊しない」二段階設計法を採用しています。なお、該当判定は竣工日ではなく「建築確認日」が基準となります。

</QuizAnswer>

## 問5

長周期地震動に関する説明として、正しいものはどれか。

1. 長周期地震動は周期が短いため、木造住宅に最も大きな被害を与える
2. 長周期地震動の階級は1〜10の10段階に分類されている
3. 長周期地震動は高層ビルを大きく長時間揺らし、室内の家具や設備に大きな被害をもたらすことがある
4. 長周期地震動は震源に近い場所でのみ発生し、遠方には伝わらない


<QuizAnswer>

**答え: 3**

長周期地震動は周期が1.6〜7.8秒程度の長い揺れで、高層ビルの固有周期と共振しやすいため、ビルを大きく長時間揺らします。室内では家具の転倒や移動、エレベーターの停止などの被害が生じます。長周期地震動の階級は1〜4の4段階で、2023年2月からは階級3以上が緊急地震速報（警報）の発表基準に追加されました。また、長周期地震動は遠方まで減衰しにくいのが特徴です。

</QuizAnswer>

## 問6

地震保険に関する説明として、正しいものはどれか。

1. 地震保険は火災保険とは別に単独で契約することができる
2. 地震保険の保険金額は火災保険の30〜50%の範囲内で設定し、建物は上限5,000万円、家財は上限1,000万円である
3. 地震による火災の損害は、火災保険だけで全額補償される
4. 地震保険の損害区分は「全損」と「半損」の2段階である


<QuizAnswer>

**答え: 2**

地震保険は火災保険に付帯して契約するもので、単独では契約できません。保険金額は火災保険の30〜50%の範囲内で設定し、建物は上限5,000万円、家財は上限1,000万円です。地震による火災は火災保険では補償されず、地震保険が必要です。損害区分は「全損（100%）」「大半損（60%）」「小半損（30%）」「一部損（5%）」の4段階に分類されています。

</QuizAnswer>

## 問7

地震の揺れを伝えるP波とS波に関する説明として、正しいものはどれか。

1. S波はP波より速く伝わるため、最初に到達する波である
2. P波は横波であり、S波は縦波である
3. P波もS波も固体・液体・気体のすべてを伝わることができる
4. P波は初期微動を、S波は主要動をもたらし、S波の揺れが建物被害の主な原因となる


<QuizAnswer>

**答え: 4**

P波（Primary wave）は縦波で速度が約5〜7km/sと速く最初に到達し、「カタカタ」とした初期微動をもたらします。S波（Secondary wave）は横波で速度が約3〜4km/sとP波より遅く、「ユサユサ」とした主要動をもたらし、建物被害の主な原因となります。なお、S波は固体中のみを伝わり、液体や気体の中は伝わることができません。

</QuizAnswer>

## 問8

南海トラフ地震臨時情報に関する説明として、正しいものはどれか。

1. 想定震源域内でマグニチュード8.0以上の地震が発生した場合、「巨大地震警戒」の臨時情報が発表され、事前避難対象地域では1週間の事前避難が求められる
2. 南海トラフ地震臨時情報は「調査中」と「調査終了」の2種類のみである
3. 南海トラフ地震臨時情報は、これまで一度も発表されたことがない
4. 「巨大地震注意」が発表された場合、対象地域の住民は全員直ちに避難しなければならない


<QuizAnswer>

**答え: 1**

南海トラフ地震臨時情報は「調査中」「巨大地震警戒」「巨大地震注意」「調査終了」の4種類があります。想定震源域内でMw8.0以上の地震（半割れケース）が発生した場合は「巨大地震警戒」が発表され、事前避難対象地域では1週間の事前避難が求められます。2024年8月の日向灘地震では「巨大地震注意」が初めて発表された実績があります。

</QuizAnswer>

## 問9

南海トラフ地震の被害想定（2025年3月公表）に関する説明として、正しいものはどれか。

1. 想定される死者数は最大約5万人で、そのほとんどが建物倒壊によるものである
2. 想定される死者数は最大約29万8,000人で、そのうち津波による死者が約21万5,000人を占める
3. 経済被害は約50兆円と想定されている
4. 南海トラフ地震は今後100年以内には発生しないと政府は評価している


<QuizAnswer>

**答え: 2**

2025年3月に公表された南海トラフ地震の最新被害想定では、死者数は最大約29万8,000人とされ、そのうち津波による死者が約21万5,000人と全体の約7割を占めます。経済被害は約292兆円と試算されています。津波による被害が極めて大きいことから、地震発生後はできるだけ早く高台や津波避難ビルへ避難する行動が重要です。

</QuizAnswer>

## 問10

緊急地震速報に関する説明として、正しいものはどれか。

1. 緊急地震速報は、S波を検知してから各地の揺れを予測して発表される
2. 緊急地震速報（警報）は、最大震度3以上が予想される場合に発表される
3. 緊急地震速報は、P波とS波の速度差を利用して、強い揺れ（S波）の到達前に発表するものである
4. 緊急地震速報は震源からの距離に関係なく、常に揺れの10分前に届く


<QuizAnswer>

**答え: 3**

緊急地震速報は、震源近くの地震計がまずP波（初期微動）を検知し、震源位置・マグニチュード・各地の予想震度を自動推定して、強い揺れをもたらすS波（主要動）が到達する前に発表する仕組みです。緊急地震速報（警報）は最大震度5弱以上が予想される場合に発表されます。ただし、直下型地震ではP波とS波の到達時間差が小さく、警報が揺れに間に合わないことがある点が限界として知られています。

</QuizAnswer>
