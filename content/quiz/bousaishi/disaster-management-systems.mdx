## 問1

地区防災計画について、正しい記述はどれか。

1. 都道府県知事が策定する広域的な防災計画である
2. 中央防災会議が作成する国の最上位計画である
3. 地区の居住者や事業者が自発的に作成する防災活動の計画である
4. 市町村長が単独で策定し、住民の関与は求められていない


<QuizAnswer>

**答え: 3**

地区防災計画は、平成25年（2013年）の災害対策基本法改正により創設された制度で、市町村内の一定の地区の居住者及び事業者が自発的に行う防災活動に関する計画です。行政主導の地域防災計画とは異なり、地区住民によるボトムアップ型の計画である点が特徴です。

</QuizAnswer>

## 問2

警戒レベルと避難情報の組み合わせとして正しいものはどれか。

1. 警戒レベル4は「避難指示」であり、危険な場所から全員避難が必要である
2. 警戒レベル3は「避難指示」であり、全住民が直ちに避難する段階である
3. 警戒レベル5は「避難指示」であり、危険な場所から全員避難が必要である
4. 警戒レベル4は「高齢者等避難」であり、高齢者のみが避難する段階である


<QuizAnswer>

**答え: 1**

令和3年（2021年）の避難情報ガイドライン改定により、警戒レベル4は「避難指示」に一本化され、対象地域の全員が危険な場所から避難する段階とされています。警戒レベル3は「高齢者等避難」、警戒レベル5は「緊急安全確保」（命の危険、直ちに安全確保）です。

</QuizAnswer>

## 問3

事業継続計画（BCP）について、最も適切な記述はどれか。

1. 策定後は内容を固定し、変更しないことが望ましい
2. 自然災害のみを対象としており、感染症は対象外である
3. 被害を完全に防ぐための防災計画と同じ意味である
4. 緊急事態においても中核事業の継続・早期復旧を目的とした計画である


<QuizAnswer>

**答え: 4**

BCPは、自然災害・大火災・感染症など様々な緊急事態に遭遇した場合に、事業資産の損害を最小限にとどめつつ、中核事業の継続または早期復旧を可能とするための計画です。策定後も経営環境やリスクの変化に応じて定期的に見直し・改善を行う継続的な取り組みが重要とされています。

</QuizAnswer>

## 問4

罹災証明書について、正しい記述はどれか。

1. 都道府県知事が発行する
2. 市町村長が被害の状況を調査した上で交付する
3. 申請がなくても被災者全員に自動的に交付される
4. 被害の程度にかかわらず一律の内容で発行される


<QuizAnswer>

**答え: 2**

罹災証明書は、災害対策基本法に基づき、被災者からの申請を受けて市町村長が交付するものです。市町村の職員が被害認定調査を行い、住家の被害程度を「全壊」「大規模半壊」「中規模半壊」「半壊」「準半壊」「準半壊に至らない（一部損壊）」に分類して証明します。

</QuizAnswer>

## 問5

被災者生活再建支援制度における基礎支援金の支給額として正しいものはどれか（複数人世帯の場合）。

1. 全壊世帯には100万円が支給される
2. 全壊世帯には300万円が支給される
3. 全壊世帯には50万円が支給される
4. 全壊世帯には200万円が支給される


<QuizAnswer>

**答え: 1**

被災者生活再建支援制度では、基礎支援金として全壊世帯に100万円、大規模半壊世帯に50万円が支給されます。これに加えて、住宅の再建方法に応じた加算支援金（建設・購入200万円、補修100万円、賃借50万円）が支給され、最大で合計300万円を受け取ることができます。

</QuizAnswer>

## 問6

ハザードマップについて、誤っている記述はどれか。

1. 洪水ハザードマップは河川の氾濫による浸水想定区域を示す
2. 土砂災害ハザードマップは土石流やがけ崩れの危険箇所を示す
3. 内水ハザードマップは河川の堤防決壊による氾濫を示す
4. 高潮ハザードマップは高潮による浸水想定区域を示す


<QuizAnswer>

**答え: 3**

内水ハザードマップは、大雨によって下水道などの排水施設の能力を超えた水が地表にあふれ出る「内水氾濫」の危険性を示したものです。河川の堤防決壊による氾濫は「外水氾濫」であり、これは洪水ハザードマップが対象とします。内水と外水の違いは防災上重要な知識です。

</QuizAnswer>

## 問7

災害対策基本法に基づく防災計画の階層構造として正しいものはどれか。

1. 防災基本計画（国）→ 防災業務計画（指定行政機関等）→ 地域防災計画（都道府県・市町村）
2. 地域防災計画（国）→ 防災基本計画（都道府県）→ 地区防災計画（市町村）
3. 防災業務計画（国）→ 地域防災計画（都道府県）→ 防災基本計画（市町村）
4. 地区防災計画（国）→ 防災基本計画（都道府県）→ 地域防災計画（市町村）


<QuizAnswer>

**答え: 1**

災害対策基本法に基づく防災計画は、中央防災会議が作成する「防災基本計画」を最上位に、指定行政機関や指定公共機関が作成する「防災業務計画」、都道府県や市町村の防災会議が作成する「地域防災計画」という階層構造になっています。さらに地区住民が作成する「地区防災計画」が加わります。

</QuizAnswer>

## 問8

警戒レベル3で求められる行動として最も適切なものはどれか。

1. 災害への心構えを高める
2. 高齢者等は危険な場所から避難する
3. 危険な場所から全員が避難する
4. 避難行動の確認をする


<QuizAnswer>

**答え: 2**

警戒レベル3は「高齢者等避難」に該当し、高齢者や障害のある方など避難に時間を要する方は危険な場所から避難を開始する段階です。警戒レベル1は「心構え」、警戒レベル2は「避難行動の確認」、警戒レベル4は「避難指示（全員避難）」、警戒レベル5は「緊急安全確保」にそれぞれ対応しています。

</QuizAnswer>

## 問9

危機管理における情報伝達の原則について、正しい記述はどれか。

1. 情報収集はボトムアップ（現場から上層部へ）、意思決定・指示はトップダウン（上層部から現場へ）が基本である
2. 情報収集も意思決定もすべてトップダウンで行うのが原則である
3. 緊急時であっても必ず全員の合議により意思決定を行う
4. 情報収集はトップダウン、意思決定はボトムアップが基本原則である


<QuizAnswer>

**答え: 1**

危機管理においては、現場の状況を迅速かつ正確に把握するために情報収集はボトムアップ（現場から上層部へ）で行い、それに基づく意思決定と指示はトップダウン（上層部から現場へ）で迅速に行うのが基本原則です。緊急時には迅速な判断が求められるため、トップが責任を持って決断し指示を出す体制が重要です。

</QuizAnswer>

## 問10

地域防災計画について、正しい記述はどれか。

1. 国の中央防災会議が全国一律の内容で作成する
2. 一度策定したら見直す必要はない
3. 都道府県や市町村がそれぞれの地域の実情に応じて作成する
4. 企業が自社の事業継続のために作成する計画である


<QuizAnswer>

**答え: 3**

地域防災計画は、災害対策基本法第40条・第42条に基づき、都道府県や市町村がそれぞれの防災会議に諮って作成する計画です。地域の自然条件や社会条件に応じた内容とし、防災のために処理すべき業務等を具体的に定めます。毎年検討を加え、必要に応じて修正することが法律で求められています。

</QuizAnswer>
