## 問1

災害時のトリアージにおいて、傷病者の治療優先順位を示すために用いられるトリアージタッグの色と区分の組み合わせとして、正しいものはどれか。

1. 赤色（最優先治療群）→ 黄色（待機的治療群）→ 緑色（保留群）→ 黒色（死亡・不処置群）
2. 黄色（最優先治療群）→ 赤色（待機的治療群）→ 緑色（保留群）→ 黒色（死亡・不処置群）
3. 赤色（最優先治療群）→ 緑色（待機的治療群）→ 黄色（保留群）→ 黒色（死亡・不処置群）
4. 赤色（最優先治療群）→ 黄色（待機的治療群）→ 黒色（保留群）→ 緑色（死亡・不処置群）


<QuizAnswer>

**答え: 1**

トリアージタッグは、赤（カテゴリーI：最優先治療群・重症）、黄（カテゴリーII：待機的治療群・中等症）、緑（カテゴリーIII：保留群・軽症）、黒（カテゴリー0：死亡または救命不可能）の4色で区分されます。災害現場では限られた医療資源を最大限に活用するため、この優先順位に従って治療が行われます。

</QuizAnswer>

## 問2

クラッシュシンドローム（挫滅症候群）に関する記述として、最も適切なものはどれか。

1. 長時間圧迫された四肢が解放された瞬間に、骨折による激しい痛みでショック状態となることをいう
2. 長時間圧迫された筋肉が解放されると、壊死した筋肉から流出したカリウムやミオグロビンが全身に回り、急性腎不全や心停止を引き起こすことがある
3. 圧迫部位に細菌が繁殖することで敗血症を起こし、全身状態が急速に悪化することをいう
4. 圧迫による血流障害で四肢の末端が壊疽を起こし、切断が必要となる状態をいう


<QuizAnswer>

**答え: 2**

クラッシュシンドロームは、建物の倒壊などで長時間（一般に2時間以上）筋肉が圧迫された後、救出により圧迫が解放されたときに発症します。壊死した筋肉組織から大量のカリウムやミオグロビンなどの有害物質が血中に放出され、急性腎不全や高カリウム血症による心停止を引き起こす危険があります。阪神・淡路大震災で広く知られるようになり、救出時には速やかな輸液などの医療処置が重要です。

</QuizAnswer>

## 問3

通電火災に関する記述として、最も適切なものはどれか。

1. 地震発生直後に電気配線がショートして発生する火災のことである
2. 停電から電力が復旧した際に、損傷した配線や転倒した電気器具から出火する火災のことである
3. 雷が電線に落ちて過電流が流れ、家庭内の電気器具が発火する現象のことである
4. 電力会社の変電所が被災し、異常な高圧電流が家庭に流れ込んで発生する火災のことである


<QuizAnswer>

**答え: 2**

通電火災とは、地震などで停電した後、電力が復旧した際に発生する火災のことです。地震で損傷した電気配線や、転倒したまま放置された電気ストーブ・アイロンなどに再通電することで出火します。阪神・淡路大震災では火災原因の多くが通電火災であったとされており、避難時にブレーカーを落とす（感震ブレーカーの設置も有効）ことが予防策として重要です。

</QuizAnswer>

## 問4

災害時の安否確認手段に関する記述として、誤っているものはどれか。

1. 災害用伝言ダイヤル「171」は、固定電話や携帯電話から音声メッセージを録音・再生できるサービスである
2. 災害用伝言板「web171」は、インターネットを通じてテキストによる安否情報を登録・確認できるサービスである
3. 災害用伝言ダイヤル「171」は、大規模災害発生時にNTTが運用を開始するもので、平常時には一切利用できない
4. 携帯電話各社も災害用伝言板サービスを提供しており、「web171」から各社の伝言板の情報を横断的に検索できる


<QuizAnswer>

**答え: 3**

災害用伝言ダイヤル「171」は大規模災害発生時にNTTが提供開始するサービスですが、平常時でも毎月1日・15日や防災週間（8月30日〜9月5日）、正月三が日などに体験利用が可能です。いざという時にスムーズに使えるよう、平常時から操作に慣れておくことが推奨されています。災害用伝言板「web171」からは各携帯電話事業者の伝言板も横断検索でき、より広範な安否確認が可能です。

</QuizAnswer>

## 問5

大規模災害後のライフライン復旧に関する記述として、一般的に正しいものはどれか。

1. 復旧の順序は一般的に、電力→通信→上水道→ガスの順で、ガスの復旧には最も時間がかかる傾向がある
2. 復旧の順序は一般的に、ガス→上水道→電力→通信の順で、通信の復旧に最も時間がかかる
3. すべてのライフラインはほぼ同時期に復旧するため、復旧順序に一般的な傾向はない
4. 復旧の順序は一般的に、上水道→ガス→通信→電力の順で、電力の復旧に最も時間がかかる


<QuizAnswer>

**答え: 1**

過去の大規模災害の経験から、ライフラインの復旧順序は一般的に電力が最も早く、次いで通信、上水道と続き、都市ガスの復旧には最も時間がかかる傾向があります。都市ガスは安全確認のため1軒ずつ開栓作業が必要であり、阪神・淡路大震災では復旧に約3か月を要しました。このため、カセットコンロなど代替手段の備蓄が重要です。

</QuizAnswer>

## 問6

エコノミークラス症候群（静脈血栓塞栓症）の予防法として、誤っているものはどれか。

1. こまめに水分を摂取する
2. 足首の曲げ伸ばしやふくらはぎのマッサージを定期的に行う
3. 安静にするため、なるべく同じ姿勢を保って動かないようにする
4. 弾性ストッキングを着用する


<QuizAnswer>

**答え: 3**

エコノミークラス症候群は、長時間同じ姿勢で座り続けることで下肢の深部静脈に血栓ができ、それが肺の血管を詰まらせる病気です。災害時の車中泊や狭い避難所生活で発症リスクが高まります。予防には、こまめな水分摂取、定期的な足の運動（足首の屈伸やふくらはぎのマッサージ）、弾性ストッキングの着用が有効であり、「安静にして動かない」ことはむしろ発症リスクを高めます。

</QuizAnswer>

## 問7

大量出血時の応急手当に関する記述として、最も適切なものはどれか。

1. 出血部位より心臓から遠い部分を強く縛る間接圧迫止血法を最初に行う
2. まず直接圧迫止血法を行い、清潔なガーゼや布を傷口に直接当てて強く圧迫する
3. 止血帯を常に第一選択として使用し、傷口の直接圧迫は行わない
4. 出血量が多い場合は、傷口を水で洗い流すことが最優先である


<QuizAnswer>

**答え: 2**

応急手当における止血法の基本は「直接圧迫止血法」です。清潔なガーゼやハンカチなどを傷口に直接当て、手で強く圧迫します。この方法はほとんどの出血に有効であり、最初に試みるべき止血法です。止血帯の使用は直接圧迫で止血できない場合の手段であり、不適切な使用は組織の壊死を招く危険があるため注意が必要です。感染防止のため、可能であればビニール手袋などを着用して処置します。

</QuizAnswer>

## 問8

避難所における感染症予防・衛生管理に関する記述として、誤っているものはどれか。

1. 手洗い・手指消毒を徹底し、トイレの衛生管理を適切に行うことが重要である
2. 発熱や下痢などの症状がある人は、別室に隔離するなどの対応が望ましい
3. 避難所では多くの人が密集するため感染症が拡大しやすく、換気も重要な対策である
4. 災害直後は非常事態であるため、食品の衛生管理は平常時ほど厳格でなくてよい


<QuizAnswer>

**答え: 4**

災害時の避難所では、多くの人が密集し衛生環境が悪化しやすいため、むしろ平常時以上に食品の衛生管理に注意が必要です。停電による冷蔵庫の停止、断水による手洗い不足などから食中毒が発生しやすくなります。手洗い・消毒の徹底、適切なトイレ管理、定期的な換気、有症状者の隔離対応など、総合的な衛生管理が感染症の集団発生を防ぐ鍵となります。

</QuizAnswer>

## 問9

災害時のトリアージで用いられるSTART法に関する記述として、正しいものはどれか。

1. 傷病者の氏名・年齢・既往歴を確認し、重症度を判定する方法である
2. 歩行の可否、呼吸、循環（脈拍）、意識レベルの順に評価し、短時間で優先度を判定する方法である
3. 医師のみが実施できる高度な医学的判定法であり、一般の救助者は使用できない
4. 搬送手段の確保状況に応じて治療優先度を変更する方法である


<QuizAnswer>

**答え: 2**

START法（Simple Triage And Rapid Treatment）は、災害現場で多数の傷病者を迅速に選別するためのトリアージ手法です。まず歩行可能かを確認し（歩ければ緑）、次に呼吸の有無、呼吸数、循環（橈骨動脈の脈拍や毛細血管再充満時間）、簡単な指示への反応（意識レベル）を順に評価します。医師でなくても実施可能で、1人あたり30秒〜1分程度で判定できるよう設計されています。

</QuizAnswer>

## 問10

クラッシュシンドロームが疑われる傷病者を救出する際の対応として、最も適切なものはどれか。

1. 一刻も早く圧迫物を取り除き、自力で歩かせて搬送する
2. 圧迫物を取り除く前から可能であれば輸液を開始し、救出後は速やかに医療機関へ搬送する
3. 圧迫されている部位をマッサージして血流を回復させてから圧迫物を取り除く
4. 圧迫物を取り除いた後、数時間その場で安静にさせて経過観察する


<QuizAnswer>

**答え: 2**

クラッシュシンドロームでは、圧迫が解除された瞬間に壊死した筋肉から有害物質が一気に血中に流入するため、可能であれば圧迫物の除去前から大量輸液を開始することが理想的です。救出後は腎不全や心停止のリスクがあるため、速やかに医療機関へ搬送する必要があります。圧迫部位のマッサージは有害物質の流出を促進するため禁忌であり、現場での長時間の経過観察は危険です。

</QuizAnswer>
