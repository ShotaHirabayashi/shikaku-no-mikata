## 問1

1923年（大正12年）に発生した関東大震災では、死者・行方不明者約10万5,000人のうち、最も多かった死因は何か。

1. 建物の倒壊による圧死
2. 火災による焼死
3. 津波による溺死
4. 土砂災害による生き埋め


<QuizAnswer>

**答え: 2**

関東大震災では、地震発生が昼食時であったことから同時多発的に大規模な火災が発生し、死者の約87％にあたる約9万1,800人が火災により命を落としました。特に「火災旋風」と呼ばれる炎の竜巻が発生し、被服廠（ひふくしょう）跡地では約3万8,000人が犠牲となるなど、甚大な被害をもたらしました。

</QuizAnswer>

## 問2

1959年（昭和34年）の伊勢湾台風による甚大な被害を契機として、日本の防災行政の基本となる法律が制定された。この法律はどれか。

1. 災害救助法
2. 水防法
3. 災害対策基本法
4. 大規模地震対策特別措置法


<QuizAnswer>

**答え: 3**

伊勢湾台風では死者・行方不明者5,098人という台風災害として明治以降最多の被害が生じました。この災害を契機に、それまで個別法の寄せ集めで対応していた防災体制を見直し、1961年（昭和36年）に災害対策基本法が制定されました。この法律により、予防から応急対策、復旧・復興まで一貫した防災体制が整備されました。

</QuizAnswer>

## 問3

1995年の阪神・淡路大震災における直接の死因として最も多かったものはどれか。

1. 火災による焼死
2. 津波による溺死
3. 窒息・圧死
4. 低体温症


<QuizAnswer>

**答え: 3**

阪神・淡路大震災の犠牲者6,434人のうち、死因の約77％が窒息・圧死でした。死者の9割以上が地震発生から15分以内に亡くなったと推定されており、老朽化した木造住宅の倒壊により就寝中に下敷きになったことが主な原因です。このことから、住宅の耐震化と家具の固定の重要性が強く認識されました。

</QuizAnswer>

## 問4

阪神・淡路大震災において、倒壊した建物から救出された人の大部分を助けたのは誰だったか。この経験から広まった「自助・共助・公助」の割合として語られる比率はどれか。

1. 自助1：共助2：公助7
2. 自助2：共助5：公助3
3. 自助3：共助3：公助4
4. 自助7：共助2：公助1


<QuizAnswer>

**答え: 4**

阪神・淡路大震災では、倒壊建物から救出された人の約97.5％が自力脱出または家族・近隣住民による救助であり、消防・警察・自衛隊による救出は約2％程度にとどまりました。この経験から「自助7：共助2：公助1」の比率が広く知られるようになり、大規模災害時には公的支援に頼りきらず、自分自身や地域の力で命を守ることの重要性が再認識されました。

</QuizAnswer>

## 問5

2011年の東日本大震災で、死因として最も多かったものはどれか。

1. 窒息・圧死
2. 溺死
3. 火災による焼死
4. 低体温症


<QuizAnswer>

**答え: 2**

東日本大震災の死者約18,000人のうち、約90.6％が溺死でした。これは巨大津波が主な死因であったことを示しています。また、犠牲者の約65％が60歳以上の高齢者であり、避難行動が困難であったことが被害を拡大させた要因の一つとされています。この教訓から、迅速な津波避難の重要性と高齢者等への避難支援の必要性が強調されました。

</QuizAnswer>

## 問6

東日本大震災後に注目された「震災関連死」の説明として正しいものはどれか。

1. 地震の揺れにより建物が倒壊して亡くなること
2. 津波に巻き込まれて溺死すること
3. 避難生活のストレスや持病の悪化など間接的な原因で亡くなること
4. 余震による二次的な建物崩壊で亡くなること


<QuizAnswer>

**答え: 3**

震災関連死とは、地震や津波の直接的な被害ではなく、避難所生活の長期化による体調悪化、ストレス、持病の悪化、医療機関の機能停止などの間接的な原因で死亡することを指します。東日本大震災では3,800人以上が震災関連死と認定されており、高齢者を中心に避難生活の過酷さが大きな課題となりました。この教訓から、避難所の環境改善や要配慮者への支援体制の強化が求められるようになりました。

</QuizAnswer>

## 問7

2016年の熊本地震に関する記述として正しいものはどれか。

1. 日本で初めて震度7が観測された地震である
2. 観測史上初めて同一地域で震度7が2回観測された
3. 前震よりも先に本震が発生した
4. 地震発生後の余震は1か月以内に収まった


<QuizAnswer>

**答え: 2**

2016年4月の熊本地震では、4月14日にマグニチュード6.5の前震、4月16日にマグニチュード7.3の本震が発生し、いずれも益城町で最大震度7を記録しました。同一地域でわずか28時間の間に震度7が2回観測されたのは観測史上初めてのことです。この経験から、最初の大きな揺れの後にさらに大きな地震が発生する可能性があることが広く認識されるようになりました。

</QuizAnswer>

## 問8

関東大震災で特に大きな被害をもたらした「火災旋風」の説明として正しいものはどれか。

1. 地震により電線がショートして発生する大規模な電気火災
2. 火災域やその周辺で発生する竜巻状の炎の渦
3. ガス管の破損によりガスが噴出して引火する現象
4. 建物の崩壊により暖房器具が倒れて連鎖的に延焼する現象


<QuizAnswer>

**答え: 2**

火災旋風とは、大規模な火災が発生した際に、上昇気流と周囲の風が相互に作用して竜巻状の炎の渦が発生する現象です。関東大震災では、旧陸軍被服廠跡地において火災旋風が発生し、避難していた約3万8,000人が犠牲になりました。火災旋風の内部は風速100m/秒以上に達することもあり、人や物を吹き飛ばし、広範囲に火の粉を飛散させて延焼を急速に拡大させます。

</QuizAnswer>

## 問9

平成30年7月豪雨（西日本豪雨）において、多くの住民が避難しなかった理由として調査で最も多かったものはどれか。

1. 避難指示が発令されなかったため
2. 避難所の場所がわからなかったため
3. 自分のところには被害が及ばないと思ったため
4. 交通手段がなく移動できなかったため


<QuizAnswer>

**答え: 3**

平成30年7月豪雨（西日本豪雨）では200人以上が犠牲となりましたが、避難しなかった理由として最も多かったのは「自分のところは大丈夫だと思った」という回答で、約47.6％を占めました。これは「正常性バイアス」と呼ばれる心理現象の典型例であり、危険が迫っていても「自分は大丈夫」と思い込んでしまう人間の心理的傾向です。この教訓から、避難行動を促す情報発信のあり方が見直されるきっかけとなりました。

</QuizAnswer>

## 問10

阪神・淡路大震災で被害が大きかった建物の特徴として正しいものはどれか。

1. 1981年（昭和56年）の新耐震基準以降に建てられた建物の倒壊が多かった
2. 1981年（昭和56年）の新耐震基準以前に建てられた建物の倒壊が目立った
3. 鉄筋コンクリート造の建物が木造建物より多く倒壊した
4. 建築年代による被害の差はほとんど見られなかった


<QuizAnswer>

**答え: 2**

阪神・淡路大震災では、1981年（昭和56年）に施行された新耐震基準以前に建てられた建物に倒壊が集中しました。一方、新耐震基準に適合した1982年以降の建物の倒壊率は10％未満にとどまり、耐震基準の有効性が実証されました。この教訓から、既存建物の耐震診断と耐震補強の重要性が強く認識され、住宅の耐震化促進が防災施策の柱の一つとなっています。

</QuizAnswer>
