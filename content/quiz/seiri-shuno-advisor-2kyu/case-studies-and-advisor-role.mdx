## 問1

整理収納アドバイザーがクライアントの自宅を訪問した際、最初に行うべきことはどれか。

1. すぐにモノを捨てる作業に取りかかる
2. アドバイザー自身の成功事例を詳しく紹介する
3. クライアントの暮らし方や困りごとについてヒアリングを行う
4. 収納用品の購入リストを作成する


<QuizAnswer>

**答え: 3**

整理収納アドバイザーの基本は、まずクライアントの暮らし方、家族構成、困りごとなどを丁寧にヒアリングすることです。片付かない原因や問題点を正確に把握することで、その人に合った最適な解決策を提案できます。いきなりモノを捨てたり収納用品を買ったりするのは、原因を把握しないまま対処することになり適切ではありません。

</QuizAnswer>

## 問2

整理収納アドバイザーの活動分野として、適切でないものはどれか。

1. 個人宅を訪問して整理収納のアドバイスを行う
2. セミナーや講座を開催して整理収納の知識を伝える
3. 不用品の買取・転売を主な業務として行う
4. 書籍の執筆やメディア出演を通じて情報発信する


<QuizAnswer>

**答え: 3**

整理収納アドバイザーの活動分野は、個人宅訪問でのアドバイス、セミナーや講座の開催、書籍執筆やメディア出演など多岐にわたります。不用品の買取・転売はリサイクル業者の業務であり、整理収納アドバイザーの本来の活動分野には含まれません。アドバイザーの役割は、モノとの関わり方を見直し、片付けやすい環境づくりを支援することです。

</QuizAnswer>

## 問3

整理収納のビフォーアフター事例を活用する目的として、最も適切なものはどれか。

1. クライアントに恥ずかしさを感じさせ、片付けの動機づけにする
2. アドバイザーの実力を誇示するためのツールとする
3. 具体的な変化をわかりやすく示し、整理収納の効果を実感してもらう
4. SNSでの集客のためだけに使用する


<QuizAnswer>

**答え: 3**

ビフォーアフター事例は、整理収納によってどのような変化が生まれるかを視覚的にわかりやすく伝えるためのものです。クライアントが仕上がりのイメージを持ちやすくなり、整理収納の効果を具体的に実感できます。恥ずかしさを与えたり、アドバイザーの自己アピールのために使ったりするものではありません。

</QuizAnswer>

## 問4

クライアントへの寄り添い方として、最も適切な対応はどれか。

1. 「なぜこんなに散らかしたのですか」と原因を追及する
2. アドバイザーの価値観に基づいてモノの要不要を判断する
3. クライアントの価値観やライフスタイルを尊重しながら一緒に考える
4. 効率重視で、クライアントの意見を聞かずにどんどん作業を進める


<QuizAnswer>

**答え: 3**

整理収納アドバイザーは、クライアントのライフスタイルや価値観を尊重し、寄り添いながら一緒に考えることが大切です。モノの要不要の判断はクライアント自身が行うべきであり、アドバイザーが自分の価値観を押し付けてはいけません。批判的な態度や一方的な作業進行は、クライアントとの信頼関係を損ないます。

</QuizAnswer>

## 問5

ヒアリングの基本技術として、最も重要なことはどれか。

1. 質問をできるだけ多くして、短時間で情報を集める
2. クライアントの話を傾聴し、生活上の困りごとや希望を丁寧に聞き取る
3. 事前にアンケートを取るだけで、当日のヒアリングは省略する
4. アドバイザーの経験から問題点を推測し、ヒアリングは最小限にする


<QuizAnswer>

**答え: 2**

ヒアリングで最も重要なのは、クライアントの話に真剣に耳を傾け（傾聴）、生活上の困りごとや理想の暮らしについて丁寧に聞き取ることです。整理収納の問題は人それぞれ異なるため、一方的に質問を浴びせたり、経験だけで推測したりせず、クライアント自身の言葉から本質的な課題を把握することが求められます。

</QuizAnswer>

## 問6

子どもの成長に合わせた整理収納の提案として、最も適切なものはどれか。

1. 子どもが小さいうちは整理を教える必要はない
2. 子どもの年齢や発達段階に応じて、自分で片付けられる仕組みを提案する
3. 子ども部屋は親がすべて管理し、子どもには触らせない
4. 成長しても収納方法は一度決めたら変えない方がよい


<QuizAnswer>

**答え: 2**

ライフステージ別の整理収納では、子どもの年齢や発達段階に応じた提案が重要です。小さい子どもでも自分で片付けやすい高さや仕組みを取り入れることで、整理の習慣を身につけることができます。子どもの成長に合わせて収納方法を見直していくことが、長期的に片付けやすい環境を維持するポイントです。

</QuizAnswer>

## 問7

整理収納アドバイザーが企業やオフィスで活動する場合の役割として、最も適切なものはどれか。

1. 社員の私物をすべて処分するよう指導する
2. 業務効率の向上を目指し、職場環境の整理収納を提案する
3. オフィスのインテリアデザインのみを担当する
4. 社員の勤務態度を評価して報告する


<QuizAnswer>

**答え: 2**

整理収納アドバイザーの活動分野は個人宅だけでなく、企業やオフィスにも広がっています。オフィスでは、書類やモノの整理収納を通じて業務効率の向上や生産性アップを目指した提案を行います。インテリアデザインや社員の評価はアドバイザーの役割ではなく、あくまで整理収納の観点から職場環境を改善することが目的です。

</QuizAnswer>

## 問8

高齢者のクライアントに整理収納のアドバイスを行う際、特に配慮すべきことはどれか。

1. 思い出の品も含め、モノをできるだけ多く処分するよう促す
2. 安全性を考慮し、動線上の障害物の除去や取り出しやすさを重視する
3. 高齢者には整理は難しいので、すべてアドバイザーが判断して進める
4. 若い世代と同じ基準でアドバイスする


<QuizAnswer>

**答え: 2**

高齢者への整理収納アドバイスでは、転倒防止のための動線確保や、身体的負担の少ない取り出しやすい収納など、安全性への配慮が特に重要です。思い出の品への愛着を尊重しながら、本人の意思を大切にして進めることが求められます。ライフステージに応じた対応が整理収納アドバイザーの重要なスキルです。

</QuizAnswer>

## 問9

整理収納アドバイザーがクライアントに提案・アドバイスを行う際の手法として、最も適切なものはどれか。

1. 他のクライアントの失敗事例を実名で紹介して注意を促す
2. 理論的な説明だけを行い、実際の作業はクライアントに任せる
3. クライアントの問題点を分析し、整理収納の理論に基づいた具体的な改善策を提案する
4. 流行の収納グッズを購入するよう勧めることを中心にアドバイスする


<QuizAnswer>

**答え: 3**

整理収納アドバイザーの提案・アドバイスは、クライアントが抱える問題点を正確に分析した上で、整理収納の理論に基づいた具体的な改善策を示すことが基本です。他のクライアントの情報を漏らすことは守秘義務に反し、収納グッズの購入ありきの提案は本質的な解決になりません。理論と実践を組み合わせた提案が効果的です。

</QuizAnswer>

## 問10

引っ越しというライフステージの変化に際して、整理収納アドバイザーが行う支援として最も適切なものはどれか。

1. 引っ越し業者の手配のみを代行する
2. 新居のインテリアコーディネートだけを行う
3. 引っ越しを機にモノの見直しを行い、新居での暮らしに合った整理収納プランを提案する
4. 旧居のモノをそのまますべて新居に持ち込むよう助言する


<QuizAnswer>

**答え: 3**

引っ越しはライフステージの大きな変化であり、モノを見直す絶好の機会です。整理収納アドバイザーは、新居の間取りやクライアントの新しい生活スタイルに合わせた整理収納プランを提案し、快適な新生活のスタートを支援します。単なる引っ越し作業の代行ではなく、整理収納の理論を応用した問題解決を行うことがアドバイザーの役割です。

</QuizAnswer>
