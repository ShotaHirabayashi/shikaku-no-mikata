## 問1

収納における「ゾーニング」とは、どのような考え方を指すか。

1. 収納スペースを色分けしてインテリアに統一感を持たせること
2. 家の中のそれぞれの場所に役割を持たせ、どこに何を置くかを決めること
3. 不要なモノを処分して収納スペースを空けること
4. 収納用品をブランドごとに分けて購入すること


<QuizAnswer>

**答え: 2**

ゾーニングとは、家の中の各スペースに役割を持たせて空間を分け、「どこに何を置くか」を決めることです。モノを使う場所の近くに収納の定位置をつくることで、必要なときにすぐ取り出せる仕組みになります。まず部屋単位でざっくりとゾーニングし、次に部屋の中のスペースごとに細かく決めていくのが効果的です。

</QuizAnswer>

## 問2

収納スペースを上段・中段・下段に分けて使う場合、最も適切な収納の仕方はどれか。

1. 上段に重いモノ、中段に使用頻度の低いモノ、下段に軽いモノを収納する
2. 上段・中段・下段すべてに均等にモノを詰め込む
3. 上段に使用頻度の低い軽いモノ、中段に使用頻度の高いモノ、下段に重いモノを収納する
4. 上段に日用品、中段に季節用品、下段に思い出の品を収納する


<QuizAnswer>

**答え: 3**

収納スペースの高さを活かすには、目線から腰の高さの「中段（ゴールデンゾーン）」に使用頻度の高いモノを置き、手が届きにくい「上段」には軽くて使用頻度の低いモノ、「下段」には重いモノを配置するのが基本です。この配置により、日常の出し入れがスムーズになり、安全面でも優れた収納になります。

</QuizAnswer>

## 問3

ラベリングの主な目的として、最も適切なものはどれか。

1. 収納ケースの見た目をおしゃれにするため
2. 誰にでも定位置がわかるようにし、自分も定位置を間違えないようにするため
3. 収納スペースの空き容量を管理するため
4. モノの購入日を記録して処分時期を判断するため


<QuizAnswer>

**答え: 2**

ラベリングの目的は大きく2つあります。1つ目は家族など誰にでもモノの定位置がわかるようにすること、2つ目は自分自身が定位置を間違えないようにすることです。ラベリングにより「使ったら元に戻す」が自然にでき、家族から「あれ、どこ？」と聞かれるストレスも減り、家の中が散らかりにくくなります。

</QuizAnswer>

## 問4

「見せる収納」のメリットとして、適切でないものはどれか。

1. 何がどこにあるか一目でわかり、取り出しやすい
2. ホコリがつきにくく掃除の手間が省ける
3. 家族や子どもがモノの位置を覚えやすい
4. 使い終わったら戻しやすい


<QuizAnswer>

**答え: 2**

「ホコリがつきにくく掃除の手間が省ける」のは「隠す収納」のメリットです。見せる収納はモノが外に出ているため、ホコリがつきやすくメンテナンスに手間がかかるというデメリットがあります。一方で、何がどこにあるか一目でわかる、取り出しやすく戻しやすい、家族がモノの位置を覚えやすいといった点が見せる収納の大きなメリットです。

</QuizAnswer>

## 問5

「隠す収納」を採用する際に特に気をつけるべきポイントはどれか。

1. 収納ケースはすべて透明なものを使わなければならない
2. 扉付きの家具は使用してはいけない
3. 目につかない分、モノが行方不明になったり不要品が増えたりしやすいため、ラベリングなどの工夫をする
4. 隠す収納にはモノを自由に入れてよいので、分類は不要である


<QuizAnswer>

**答え: 3**

隠す収納は生活感を抑えてスッキリした空間を演出できますが、目に見えない分、モノが行方不明になったり不要品が溜まりやすいというデメリットがあります。そのため、ボックスやカゴを使う場合はラベリングで中身を明示するなどの工夫が大切です。隠す収納でも「何がどこにあるか把握できる状態」を保つことが重要です。

</QuizAnswer>

## 問6

キッチンの収納で実践すべきテクニックとして、最も適切なものはどれか。

1. すべての調理器具を見せる収納にして壁面に吊るす
2. 調理台の近くによく使う調理器具を配置し、使用頻度に応じてゾーニングする
3. 食器は大きさに関係なくまとめて一つの引き出しに入れる
4. 調味料はすべて見えない場所に隠して生活感をなくす


<QuizAnswer>

**答え: 2**

キッチン収納の基本は、調理の動線に合わせたゾーニングです。よく使う調理器具はコンロや調理台の近くに配置し、使用頻度に応じて取り出しやすい位置を決めることが重要です。見せる収納と隠す収納を使い分け、よく使うモノは手に取りやすく、ストック類はまとめて収納するなど、使いやすさを優先した配置が効率的なキッチンをつくります。

</QuizAnswer>

## 問7

収納用品を選ぶ際のポイントとして、最も適切なものはどれか。

1. デザインが気に入ったものを先に購入し、それに合わせてモノを収納する
2. 収納したいモノとスペースのサイズを測ってから、それに合った収納用品を選ぶ
3. なるべく多くの種類の収納用品を揃えてバリエーションを持たせる
4. 最も高価な収納用品を選べば間違いない


<QuizAnswer>

**答え: 2**

収納用品を選ぶ際は、まず収納したいモノの量やサイズ、設置するスペースの寸法を正確に把握してから選ぶことが大切です。先に収納用品を買ってしまうと、サイズが合わずデッドスペースが生まれたり、かえって使いにくくなることがあります。また、統一感のある収納用品を選ぶことで見た目もスッキリし、管理もしやすくなります。

</QuizAnswer>

## 問8

家族が片づけやすい環境を整備するための工夫として、最も効果的なものはどれか。

1. 片づけのルールを細かく文書にして壁に貼る
2. 家族が自分でモノの場所を判断できるよう、ラベリングや見える収納を活用する
3. 家族には収納場所を教えず、すべて自分で管理する
4. 収納場所を頻繁に変えて、家族が柔軟に対応できるようにする


<QuizAnswer>

**答え: 2**

家族が片づけやすい環境をつくるには、誰でもモノの定位置がわかる仕組みが重要です。ラベリングで中身を明示したり、見える収納で直感的にモノの場所がわかるようにすることで、家族全員が「使ったら戻す」を自然に実践できます。特に子どもには、文字だけでなくイラストや写真を使ったラベルが効果的です。

</QuizAnswer>

## 問9

「見せる収納」と「隠す収納」の使い分けとして、最も適切な考え方はどれか。

1. どちらか一方に統一しなければ部屋のまとまりがなくなる
2. 見せる収納は子ども部屋のみで使うべきである
3. それぞれの収納に役割を持たせ、モノの種類や使用頻度に応じてハイブリッドに使い分ける
4. 隠す収納はリビングでは使わず、納戸や倉庫だけで使う


<QuizAnswer>

**答え: 3**

見せる収納と隠す収納は、それぞれにメリット・デメリットがあるため、「ハイブリッド化」が推奨されています。よく使うモノは見せる収納で取りやすく、書類や小物・掃除道具は隠してスッキリさせ、子どものおもちゃは見せる収納＋ラベルで管理するなど、それぞれに「役割」を持たせて使い分けることが効果的です。

</QuizAnswer>

## 問10

収納を維持管理しやすい仕組みにするためのポイントとして、最も適切でないものはどれか。

1. 収納の仕組みはできるだけシンプルにし、出し入れの手間を少なくする
2. 収納場所にラベリングをして、誰でも元に戻せるようにする
3. 収納スペースに余裕を持たせ、ぎゅうぎゅうに詰め込まない
4. 見た目の美しさを最優先し、出し入れに多少手間がかかっても仕方がないと考える


<QuizAnswer>

**答え: 4**

維持管理しやすい収納の仕組みづくりでは、見た目よりも「出し入れのしやすさ」を優先することが大切です。どんなに美しく収納しても、出し入れに手間がかかると元に戻すのが面倒になり、すぐに散らかってしまいます。シンプルな仕組み、ラベリング、適度な余裕を持たせることが、リバウンドしにくい収納の基本です。

</QuizAnswer>
