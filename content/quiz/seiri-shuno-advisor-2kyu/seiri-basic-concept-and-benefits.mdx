## 問1

整理収納アドバイザーの理論における「整理」の定義として、最も適切なものはどれですか。

1. 部屋をきれいに見せるためにモノを隠すこと
2. 必要・不必要なモノを区別し、不必要なモノを取り除くこと
3. モノを種類ごとに並べ替えること
4. 収納グッズを使ってモノをしまうこと


<QuizAnswer>

**答え: 2**

整理とは、モノを「必要」か「不必要」かに区別し、不必要なモノを取り除くことを指します。単に見た目をきれいにしたり、並べ替えたりすることではなく、モノの要・不要を判断するプロセスそのものが「整理」の本質です。この定義を正しく理解することが、整理収納アドバイザーの学びの出発点となります。

</QuizAnswer>

## 問2

「整理」「収納」「片づけ」の違いについて、正しい説明はどれですか。

1. 整理はモノをしまうこと、収納はモノを分けること、片づけはモノを捨てることである
2. 整理も収納も片づけも、すべて同じ意味で使い分ける必要はない
3. 整理はいる・いらないを区別すること、収納はいるモノをしまうこと、片づけは使ったモノを元に戻すことである
4. 整理はモノを元に戻すこと、収納はモノを分類すること、片づけはモノを捨てることである


<QuizAnswer>

**答え: 3**

整理収納アドバイザーの理論では、「整理」はモノの要・不要を区別すること、「収納」は必要と判断したモノを使いやすくしまうこと、「片づけ」は使い終わったモノを定位置に戻すことと明確に定義されています。この3つは異なる行為であり、「整理→収納→片づけ」という順序で行うことが重要です。

</QuizAnswer>

## 問3

整理収納を正しく行うことで得られる「時間的効果」として、最も適切なものはどれですか。

1. 収納グッズを買いに行く時間が増える
2. 片づけの方法を考える時間が楽しくなる
3. 探しモノがなくなり、時間を有効に使えるようになる
4. 家にいる時間が長くなる


<QuizAnswer>

**答え: 3**

整理収納の時間的効果とは、モノが整理され定位置が決まることで、探しモノに費やしていた時間がなくなることを意味します。ある調査では、人は一生のうち約150日分もの時間を探しモノに費やしているとも言われています。整理収納により、その時間を有意義な活動に充てることができるようになります。

</QuizAnswer>

## 問4

整理収納アドバイザーの理論において、「整理」と「整頓」の関係について正しいものはどれですか。

1. 整理と整頓はまったく同じ意味である
2. 整頓は不要なモノを捨てること、整理はモノを並べることである
3. 整理は要・不要を区別すること、整頓はモノを見た目よく整えることである
4. 整理は整頓の後に行うべきステップである


<QuizAnswer>

**答え: 3**

「整理」はモノの要・不要を判断して不要なモノを取り除くことであり、「整頓」はモノの配置を整え、見た目よく並べることを意味します。一見似た言葉ですが、本質的に異なる行為です。整頓だけでは不要なモノが残ったままになるため、まず整理を行ってから整頓することが大切です。

</QuizAnswer>

## 問5

整理収納の理論で、部屋が散らかる根本的な原因として最も適切なものはどれですか。

1. 収納スペースに対してモノが多すぎること
2. 部屋の面積が狭いこと
3. 掃除をする回数が少ないこと
4. 収納グッズが足りないこと


<QuizAnswer>

**答え: 1**

散らかる最大の原因は、収納スペースに対してモノの量が多すぎることです。部屋の広さや収納グッズの有無ではなく、持っているモノの総量が収容能力を超えていることが本質的な問題です。収納グッズを増やしても、モノの量を見直さなければ根本的な解決にはなりません。

</QuizAnswer>

## 問6

整理収納アドバイザーの理論における「アクティブ」なモノとは、どのようなモノを指しますか。

1. 新しく購入したばかりのモノ
2. 日常的に使っているモノ
3. いつか使うかもしれないモノ
4. コレクションとして飾っているモノ


<QuizAnswer>

**答え: 2**

「アクティブ」とは、現在日常的に使用しているモノのことです。対して「スタンバイ」は季節の衣類や行事用品など、今は使っていないが時期が来れば使うモノを指します。整理においては、モノをアクティブ・スタンバイ・スクラップなどに分類し、それぞれに適した管理方法をとることが重要です。

</QuizAnswer>

## 問7

整理のステップとして正しい順序はどれですか。

1. 減らす → 分ける → しまう
2. しまう → 分ける → 減らす
3. 分ける → しまう → 減らす
4. しまう → 減らす → 分ける


<QuizAnswer>

**答え: 1**

整理の基本ステップは「減らす→分ける→しまう（収める）」の順序です。まずモノの量を減らし、次に残したモノを用途や使用頻度などで分類し、最後に適切な場所に収めます。この順序を守らず、いきなりしまおうとすると、不必要なモノまで収納してしまい、根本的な解決になりません。

</QuizAnswer>

## 問8

整理収納を行うことで得られる「経済的効果」として、最も適切なものはどれですか。

1. 高価な収納家具を購入するようになる
2. 整理収納アドバイザーとして収入を得られるようになる
3. モノを大量にまとめ買いできるようになる
4. 同じモノを重複して買うことがなくなり、無駄な出費が減る


<QuizAnswer>

**答え: 4**

整理収納の経済的効果とは、何をどれだけ持っているかが把握できるようになることで、同じモノを二重に購入してしまう無駄がなくなることです。また、不要な衝動買いも減り、結果的に出費を抑えることができます。整理は「お金の節約」にも直結する重要な行為です。

</QuizAnswer>

## 問9

「スタンバイ」のモノに該当する例として、最も適切なものはどれですか。

1. 毎日使う食器
2. 壊れて使えなくなった家電
3. 季節外の衣類や季節家電
4. 毎朝使う歯ブラシ


<QuizAnswer>

**答え: 3**

「スタンバイ」とは、現在は使っていないが時期が来れば使うモノのことで、季節外の衣類（夏物・冬物）や季節家電（扇風機・ヒーターなど）が典型的な例です。毎日使う食器や歯ブラシは「アクティブ」、壊れて使えない家電は「スクラップ（不要なモノ）」に分類されます。

</QuizAnswer>

## 問10

整理収納の理論において、「整理」をせずに「収納」だけを行った場合に起こる問題として、最も適切なものはどれですか。

1. 収納スペースが余ってしまう
2. モノが少なくなりすぎて不便になる
3. 不必要なモノまでしまい込んでしまい、根本的に片づかない
4. 収納グッズが不要になる


<QuizAnswer>

**答え: 3**

整理（要・不要の区別）をせずに収納だけを行うと、不必要なモノまで丁寧にしまい込んでしまう結果になります。見た目は一時的にきれいになっても、モノの総量は変わっていないため、すぐにまた散らかってしまいます。「整理収納」という言葉の並び順こそが、まず整理を行い、その後に収納するという正しい手順を示しています。

</QuizAnswer>
