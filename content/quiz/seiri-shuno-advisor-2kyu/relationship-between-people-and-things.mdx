## 問1

モノが増える原因として、整理収納の観点から最も根本的な理由はどれか。

1. 収納スペースが足りないこと
2. モノの「入口」と「出口」のバランスが崩れていること
3. 家族の人数が多いこと
4. 片づけの時間が取れないこと


<QuizAnswer>

**答え: 2**

モノが増える根本的な原因は、家に入ってくるモノ（入口）の量に対して、出ていくモノ（出口）の量が少ないことにあります。収納スペースの不足や時間の問題は表面的な理由であり、モノの流れ全体を意識することが重要です。買う・もらうなどの「入口」と、捨てる・譲るなどの「出口」の両方を管理することで、モノの総量をコントロールできます。

</QuizAnswer>

## 問2

整理収納における「消費」「浪費」「投資」の分類で、「消費」に該当するものはどれか。

1. 衝動買いで購入したが一度も使わなかった雑貨
2. 将来のスキルアップのために購入した参考書
3. 日常生活に必要な食料品や日用品の購入
4. 値段の高いブランド品を見栄で購入すること


<QuizAnswer>

**答え: 3**

「消費」とは、日々の生活に欠かせないモノの購入や使用料のことで、食費・日用品・水道光熱費などが該当します。選択肢1は「浪費」、選択肢2は「投資」、選択肢4は「浪費」にあたります。自分の支出がどの分類に該当するかを意識することで、モノとの付き合い方を見直すことができます。

</QuizAnswer>

## 問3

モノを「使う」「使わない」に分ける際、判断に迷ったモノへの対処法として最も適切なものはどれか。

1. 迷ったらすべてすぐに捨てる
2. 迷ったモノはまとめて保管し、一定期間後に見直す
3. 迷ったモノはすべて残しておく
4. 家族に判断を任せる


<QuizAnswer>

**答え: 2**

迷ったモノは紙袋やボックスにまとめて保管し、見直し期間（例：半年〜1年）を設けるのが効果的です。期間が過ぎた後に確認すると、「この間まったく使わなかった」と実感でき、納得して手放す判断がしやすくなります。すべて即座に捨てたり、すべて残したりするのではなく、時間を味方につけた冷静な判断が大切です。

</QuizAnswer>

## 問4

同じモノでも、場所や状況によって「必要なモノ」にも「不要なモノ」にもなりうることを示す考え方として正しいものはどれか。

1. モノの価値はすべて購入価格で決まる
2. モノの価値は製造メーカーによって決まる
3. モノが必要かどうかは、人とそのモノとの関係性によって決まる
4. モノの必要性は見た目の新しさで決まる


<QuizAnswer>

**答え: 3**

モノの価値や必要性は、モノ自体の性質ではなく、人とそのモノとの関係性によって決まります。例えば、引き出しの中の短い鉛筆は「使えるモノ」ですが、道路に落ちていれば「ゴミ」として扱われます。同じモノでも、置かれた場所や持ち主との関わり方によって、その意味や価値が変わるのです。

</QuizAnswer>

## 問5

「もったいない」という心理がモノの整理を妨げる理由として、最も適切なものはどれか。

1. もったいないと感じることは常に正しい判断である
2. 「もったいない」が使わないモノを手放せなくする言い訳になりやすい
3. もったいないと思うモノは必ず将来使うことになる
4. もったいない精神があれば整理は不要になる


<QuizAnswer>

**答え: 2**

「もったいない」という気持ち自体は大切な感覚ですが、整理の場面では「高かったから」「まだ使えるから」と使わないモノを持ち続ける言い訳になりやすい側面があります。本当に「もったいない」のは、使わないモノにスペースや管理の手間を費やし続けることです。モノを大切にすることと、使わないモノを溜め込むことは別の問題として考える必要があります。

</QuizAnswer>

## 問6

モノの「出口」に該当する行動として、適切でないものはどれか。

1. 不要なモノをリサイクルショップに持ち込む
2. セールで安いからとまとめ買いをする
3. 使わなくなった服を友人に譲る
4. 壊れた家電を自治体のルールに従って処分する


<QuizAnswer>

**答え: 2**

セールでのまとめ買いはモノの「入口」にあたる行動であり、「出口」ではありません。モノの「出口」とは、捨てる・譲る・売る・リサイクルするなど、モノが家から出ていく行動のことです。モノの流れを管理するためには、入口を絞ると同時に、出口を意識的に設けることが大切です。

</QuizAnswer>

## 問7

モノとの関わり方における「浪費」に該当するものはどれか。

1. 毎日使う通勤用のカバンを購入する
2. 資格取得のためにテキストを購入する
3. 必要ないのにセールに惹かれて衝動買いした服
4. 季節ごとに必要な冷暖房器具を購入する


<QuizAnswer>

**答え: 3**

「浪費」とは、必要以上の贅沢や無駄な出費のことで、衝動買いで購入したものの使わないモノが典型例です。選択肢1と4は生活に必要な「消費」、選択肢2は将来のリターンを期待する「投資」にあたります。購入時に「本当に必要か」を立ち止まって考えることが、浪費を防ぐポイントです。

</QuizAnswer>

## 問8

モノを手放すことに迷いが生じる心理的原因として、最も一般的なものはどれか。

1. モノに思い出や感情的な価値を見出してしまうこと
2. モノの色が好みであること
3. モノが軽くて持ち運びやすいこと
4. モノが最新モデルであること


<QuizAnswer>

**答え: 1**

モノを手放せない最も一般的な心理的原因は、そのモノに思い出や感情的な価値（「あの人からもらった」「旅行の記念」など）を見出してしまうことです。実際にはモノそのものではなく、モノに紐づいた記憶や感情を大切にしたいという心理が働いています。モノと思い出を切り分けて考えることが、迷いを解消する第一歩になります。

</QuizAnswer>

## 問9

モノが家に入ってくる「入口」を管理する方法として、最も効果的なものはどれか。

1. 収納スペースを増やして受け入れ態勢を整える
2. 購入前に「今すぐ必要か」を一呼吸おいて考える習慣をつける
3. 家族にモノを買わないよう禁止する
4. すべての買い物をインターネット通販に切り替える


<QuizAnswer>

**答え: 2**

モノの「入口」を管理する最も効果的な方法は、購入前に「今すぐ必要か」と一呼吸おいて考える習慣をつけることです。「今だけ半額」「ポイントアップ中」などの誘惑に対しても、冷静に判断できるようになります。収納スペースを増やすことはモノの増加を助長し、買い物の禁止は現実的ではありません。

</QuizAnswer>

## 問10

整理収納の観点で、モノに対する「投資」の考え方として正しいものはどれか。

1. 投資とは高額な商品を購入することである
2. 投資とは使った金額以上の価値やリターンが期待できるモノへの支出である
3. 投資とは流行のモノをいち早く取り入れることである
4. 投資とは将来値上がりしそうなモノを買い集めることである


<QuizAnswer>

**答え: 2**

整理収納における「投資」とは、支出した金額以上の価値やリターンが期待できるモノへの支出のことです。自己成長につながる書籍の購入や、長く使える質の良い道具の購入などが該当します。高額であることや流行であることが投資の条件ではなく、自分の生活や将来に対してプラスの効果をもたらすかどうかが判断基準となります。

</QuizAnswer>
