## 問1

次の四字熟語の□に入る漢字として正しいものはどれか。

「一石二□」

1. 長
2. 鳥
3. 得
4. 手


<QuizAnswer>

**答え: 2**

「一石二鳥（いっせきにちょう）」は、一つの行動で二つの利益を得ることを意味する四字熟語です。もともとは英語のことわざ "Kill two birds with one stone" に由来し、一つの石を投げて二羽の鳥を捕まえるという意味から来ています。日常会話でも非常によく使われる頻出の四字熟語です。

</QuizAnswer>

## 問2

「以心伝心」の意味として最も適切なものはどれか。

1. 心を入れ替えて反省すること
2. 言葉を使わなくても互いの気持ちが通じ合うこと
3. 心配事が次から次へと伝わること
4. 他人の心を巧みに操ること


<QuizAnswer>

**答え: 2**

「以心伝心（いしんでんしん）」は、言葉や文字を使わなくても、心から心へ気持ちが通じ合うことを意味します。もとは禅宗の用語で、仏の教えが言葉によらず師の心から弟子の心へ直接伝わることを指しました。現在では、親しい間柄で言わなくても気持ちが分かり合える様子に広く使われます。

</QuizAnswer>

## 問3

次の四字熟語の□に入る漢字として正しいものはどれか。

「□機応変」

1. 臨
2. 林
3. 隣
4. 輪


<QuizAnswer>

**答え: 1**

「臨機応変（りんきおうへん）」は、その場の状況や変化に応じて、適切な手段や対応をとることを意味します。「臨機」はその場に臨むこと、「応変」は変化に応じることを表しています。仕事や日常生活でも頻繁に使われる重要な四字熟語です。

</QuizAnswer>

## 問4

「五里霧中」の意味として最も適切なものはどれか。

1. 五里も続く長い霧の中を旅すること
2. 物事の見通しが立たず、方針や判断に迷うこと
3. 遠くの景色が霧で美しく見えること
4. 五人の仲間と一緒に困難を乗り越えること


<QuizAnswer>

**答え: 2**

「五里霧中（ごりむちゅう）」は、物事の手がかりがつかめず、方針や見通しが全く立たない状態を意味します。中国の後漢時代、張楷（ちょうかい）という人物が五里にわたる霧を起こす術を使ったという故事に由来します。現在では「何をどうすればよいか分からない」という状況を表す際に広く用いられます。

</QuizAnswer>

## 問5

次の四字熟語の□に入る漢字として正しいものはどれか。

「試行□誤」

1. 再
2. 回
3. 錯
4. 過


<QuizAnswer>

**答え: 3**

「試行錯誤（しこうさくご）」は、いろいろな方法を繰り返し試みて、失敗を重ねながら解決策を見つけていくことを意味します。「試行」は試しに行うこと、「錯誤」は誤りや間違いを意味し、英語の "trial and error" に対応する表現です。学問や研究の場面だけでなく、日常的にも幅広く使われる四字熟語です。

</QuizAnswer>

## 問6

「自業自得」の意味として最も適切なものはどれか。

1. 自分の仕事は自分で片付けるべきだということ
2. 自分で事業を立ち上げて利益を得ること
3. 自分のした行いの報いを自分が受けること
4. 自分自身の能力だけで成功をつかむこと


<QuizAnswer>

**答え: 3**

「自業自得（じごうじとく）」は、自分が行った行為の報いを自分自身が受けることを意味する四字熟語です。もとは仏教用語で、自分の「業（ごう＝行い）」の結果を自分が「得る（受ける）」という因果応報の考えに基づいています。現在では主に悪い結果について「自分が招いたことだ」という意味で使われることが多いです。

</QuizAnswer>

## 問7

次の四字熟語の□に入る漢字として正しいものはどれか。

「□田引水」

1. 我
2. 水
3. 各
4. 毎


<QuizAnswer>

**答え: 1**

「我田引水（がでんいんすい）」は、自分の田んぼにだけ水を引き入れるということから、物事を自分に都合のよいようにすること・言うことを意味します。「我田」は自分の田、「引水」は水を引くという意味です。自分勝手な主張や行動を批判的に表現する場面でよく使われる四字熟語です。

</QuizAnswer>

## 問8

「四字熟語」とその意味の組み合わせとして正しいものはどれか。

1. 「一朝一夕」―― 朝から夕方まで一日中働くこと
2. 「半信半疑」―― 半分は信じ、半分は疑うこと
3. 「一長一短」―― 背が高い人と低い人がいること
4. 「大同小異」―― 大きく賛成し小さく反対すること


<QuizAnswer>

**答え: 2**

「半信半疑（はんしんはんぎ）」は、半分は信じながらも半分は疑っている、つまり本当かどうか決めかねている状態を正しく表しています。なお、「一朝一夕」はわずかな期間のこと、「一長一短」は良い点もあれば悪い点もあること、「大同小異」は細かい違いはあっても大体同じであることを意味します。

</QuizAnswer>

## 問9

次の四字熟語の□に入る漢字として正しいものはどれか。

「□転倒（ほんまつてんとう）」

1. 本末
2. 本未
3. 木末
4. 本抹


<QuizAnswer>

**答え: 1**

「本末転倒（ほんまつてんとう）」は、物事の根本的なことと些末なこととを取り違えることを意味します。「本」は物事の根本、「末」は枝葉末節のことを指し、それが「転倒」つまりひっくり返っている状態を表します。重要なことをおろそかにして、つまらないことにこだわる場面で使われる頻出の四字熟語です。

</QuizAnswer>

## 問10

「異口同音」の意味として最も適切なものはどれか。

1. 異なる言語で同じ歌を歌うこと
2. 口は違うことを言いながら心では同じことを思うこと
3. 多くの人が口をそろえて同じことを言うこと
4. 異なる楽器で同じ音を出すこと


<QuizAnswer>

**答え: 3**

「異口同音（いくどうおん）」は、多くの人がみな口をそろえて同じことを言うことを意味します。「異口」は異なる口（＝別々の人）、「同音」は同じ声・言葉という意味で、大勢の意見や発言が一致する様子を表します。会議や議論の場面で「全員が異口同音に賛成した」のように使われる、漢検3級頻出の四字熟語です。

</QuizAnswer>
