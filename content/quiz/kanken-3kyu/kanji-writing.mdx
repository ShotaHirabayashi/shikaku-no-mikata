## 問1

「会議の議事録をサクセイする」の「サクセイ」を漢字にしたとき、正しいものはどれか。

1. 作成
2. 作製
3. 策成
4. 作盛


<QuizAnswer>

**答え: 1**

「作成」は文書・計画・書類などを作り上げることを意味します。「作製」は物品や製品を作ることに使います。議事録は文書なので「作成」が正しい表記です。

</QuizAnswer>

## 問2

「彼の意見にイギを唱える」の「イギ」を漢字にしたとき、正しいものはどれか。

1. 意義
2. 威儀
3. 異議
4. 異義


<QuizAnswer>

**答え: 3**

「異議を唱える」は、反対意見や不服を申し立てるという慣用表現です。「意義」は物事の価値や意味を表し、「異義」は異なる意味を指す語で、ここでは文脈に合いません。

</QuizAnswer>

## 問3

「交通事故をボウシするために安全運転を心がける」の「ボウシ」を漢字にしたとき、正しいものはどれか。

1. 帽子
2. 防止
3. 妨止
4. 防枝


<QuizAnswer>

**答え: 2**

「防止」は、好ましくないことが起こらないように防ぎ止めることを意味します。「帽子」は頭にかぶるもので読みは同じですが意味が全く異なります。文脈から「事故を防ぐ」意味の「防止」が正解です。

</QuizAnswer>

## 問4

「新しい法律がシコウされた」の「シコウ」を漢字にしたとき、正しいものはどれか。

1. 思考
2. 志向
3. 試行
4. 施行


<QuizAnswer>

**答え: 4**

「施行」は法律や制度などを実際に実施・適用することを意味します。「思考」は考えること、「志向」は目指すこと、「試行」は試みに行うことで、法律に対しては「施行」を使います。

</QuizAnswer>

## 問5

「この問題のカイトウを用紙に書きなさい」の「カイトウ」を漢字にしたとき、正しいものはどれか。

1. 回答
2. 解答
3. 解凍
4. 回頭


<QuizAnswer>

**答え: 2**

「解答」は問題を解いて答えを出すことを意味し、試験やクイズなどで使います。「回答」は質問・要求・アンケートなどに対して返事をすることです。「問題」に対しては「解答」が適切です。

</QuizAnswer>

## 問6

「長年の努力がミノって、ついに優勝した」の「ミノって」を漢字で書いたとき、正しいものはどれか。

1. 実のって
2. 稔のって
3. 身のって
4. 蓑のって


<QuizAnswer>

**答え: 1**

「実る」は、果実がなることから転じて、努力などが成果を上げることを意味します。「努力が実る」は慣用的な表現です。「稔る」も「みのる」と読みますが、常用漢字ではなく漢検3級の範囲では「実る」が正解です。

</QuizAnswer>

## 問7

「国際社会でキンチョウが高まっている」の「キンチョウ」を漢字にしたとき、正しいものはどれか。

1. 均張
2. 緊張
3. 金張
4. 禁長


<QuizAnswer>

**答え: 2**

「緊張」は、心や体が張りつめた状態、または国際関係などが険悪な状態になることを意味します。「緊」は引き締まるという意味を持ち、「張」は張りつめるという意味で、この組み合わせが正しい表記です。

</QuizAnswer>

## 問8

「彼はキカイにめぐまれて成功した」の「キカイ」を漢字にしたとき、文脈上正しいものはどれか。

1. 機械
2. 器械
3. 機会
4. 奇怪


<QuizAnswer>

**答え: 3**

「機会」は、物事をするのにちょうどよい時機・チャンスを意味します。「めぐまれて成功した」という文脈から、チャンスに恵まれたという意味が適切です。「機械」「器械」は道具・装置のことで、この文脈には合いません。

</QuizAnswer>

## 問9

「規則をカイセイする必要がある」の「カイセイ」を漢字にしたとき、正しいものはどれか。

1. 快晴
2. 改正
3. 回生
4. 改姓


<QuizAnswer>

**答え: 2**

「改正」は、法律・規則・制度などの不適切な部分を改め正すことを意味します。「快晴」は天気、「回生」は生き返ること、「改姓」は姓を変えることで、規則に対して使うのは「改正」が正しい表記です。

</QuizAnswer>

## 問10

「この地域はノウギョウがさかんだ」の「ノウギョウ」を漢字にしたとき、正しいものはどれか。

1. 能業
2. 農漁
3. 納業
4. 農業


<QuizAnswer>

**答え: 4**

「農業」は、土地を利用して作物を栽培したり家畜を飼育したりする産業のことです。「農」は田畑を耕すことを意味し、「業」は仕事・なりわいを意味します。「さかんだ」という文脈からも、産業としての「農業」が正解です。

</QuizAnswer>
