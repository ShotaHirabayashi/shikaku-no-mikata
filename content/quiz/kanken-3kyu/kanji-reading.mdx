## 問1

「彼は会議で **詳細** な報告を行った。」の「詳細」の読みとして正しいものはどれか。

1. しょうさい
2. せいさい
3. しょうさ
4. せいさ


<QuizAnswer>

**答え: 1**

「詳細」は「しょうさい」と読みます。「詳」の音読みは「ショウ」、「細」の音読みは「サイ」です。「くわしくこまかいこと」という意味で、ビジネスや日常でも頻繁に使われる熟語です。

</QuizAnswer>

## 問2

「山の **麓** に小さな村がある。」の「麓」の読みとして正しいものはどれか。

1. いただき
2. ふち
3. ふもと
4. すそ


<QuizAnswer>

**答え: 3**

「麓」は「ふもと」と読む訓読みです。山のすそ野、山の下の方の部分を意味します。「いただき」は「頂」、「ふち」は「縁」、「すそ」は「裾」の読みであり、混同しないよう注意が必要です。

</QuizAnswer>

## 問3

「祖母は毎朝 **勤行** を欠かさない。」の「勤行」の読みとして正しいものはどれか。

1. きんこう
2. ごんぎょう
3. きんぎょう
4. ごんこう


<QuizAnswer>

**答え: 2**

「勤行」は仏教用語で「ごんぎょう」と読みます。仏前でお経を読むなどの仕事（つとめ）を意味します。「勤」には慣用音として「ゴン」の読みがあり、日常的な「キン」とは異なる読み方をする点がポイントです。

</QuizAnswer>

## 問4

「新しい法律が来月 **施行** される。」の「施行」の読みとして正しいものはどれか。

1. しこう
2. せぎょう
3. しぎょう
4. せこう


<QuizAnswer>

**答え: 1**

法律の「施行」は「しこう」と読みます。法令の効力を発生させることを意味します。建築業界では「せこう」と読む慣用もありますが、法律用語としての正式な読みは「しこう」です。漢検では「しこう」が正答として出題されます。

</QuizAnswer>

## 問5

「 **老舗** の和菓子屋で買い物をした。」の「老舗」の読みとして正しいものはどれか。

1. ろうほ
2. おいだな
3. しにせ
4. ろうてん


<QuizAnswer>

**答え: 3**

「老舗」は熟字訓で「しにせ」と読みます。代々続いている格式のある店を意味します。熟字訓とは、漢字一字ずつではなく熟語全体に訓読みが当てられたもので、「老」＝しに・「舗」＝せ、と分解できるわけではない点に注意しましょう。

</QuizAnswer>

## 問6

「彼の **怠慢** な態度が問題になった。」の「怠慢」の読みとして正しいものはどれか。

1. たいまん
2. だいまん
3. たいばん
4. だいばん


<QuizAnswer>

**答え: 1**

「怠慢」は「たいまん」と読みます。「怠」の音読みは「タイ」、「慢」の音読みは「マン」です。なまけておろそかにすることを意味します。「怠」を「ダイ」、「慢」を「バン」と読むのは誤りです。

</QuizAnswer>

## 問7

「 **境内** の桜が満開だ。」の「境内」の読みとして正しいものはどれか。

1. きょうだい
2. きょうない
3. けいない
4. けいだい


<QuizAnswer>

**答え: 4**

「境内」は「けいだい」と読みます。神社や寺院の敷地の中を意味する言葉です。「境」には「キョウ」と「ケイ」の二つの音読みがありますが、「境内」では「ケイ」を用います。「内」は連濁して「ダイ」と読む点も重要です。

</QuizAnswer>

## 問8

「友人に **土産** を買って帰った。」の「土産」の読みとして正しいものはどれか。

1. どさん
2. とさん
3. みやげ
4. つちさん


<QuizAnswer>

**答え: 3**

「土産」は熟字訓で「みやげ」と読みます。旅先などで買って人に贈る品物のことです。「どさん」という読みもありますが、その場合は「その土地の産物」という意味になります。日常会話で「おみやげ」として使う場合は「みやげ」が正しい読みです。

</QuizAnswer>

## 問9

「激しい雨で川が **氾濫** した。」の「氾濫」の読みとして正しいものはどれか。

1. はんらん
2. ぼんらん
3. はんなん
4. ひんらん


<QuizAnswer>

**答え: 1**

「氾濫」は「はんらん」と読みます。「氾」の音読みは「ハン」、「濫」の音読みは「ラン」です。川の水があふれ出ることを意味し、転じて物事が出回りすぎることにも使われます。「氾」を「ボン」や「ヒン」と読むのは誤りです。

</QuizAnswer>

## 問10

「彼女は **潔い** 性格で人気がある。」の「潔い」の読みとして正しいものはどれか。

1. うつくしい
2. いさぎよい
3. きよい
4. すがすがしい


<QuizAnswer>

**答え: 2**

「潔い」は「いさぎよい」と読みます。態度や行動に未練がなく、さっぱりしている様子を表します。「潔」の音読みは「ケツ」（潔白＝けっぱく）ですが、訓読みでは「いさぎよい」となります。「きよい」は「清い」の読みなので混同に注意しましょう。

</QuizAnswer>
