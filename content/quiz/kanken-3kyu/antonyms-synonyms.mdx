## 問1

「快適」の対義語として最も適切なものはどれか。

1. 不便
2. 劣悪
3. 不快
4. 困難


<QuizAnswer>

**答え: 3**

「快適」は心地よく気持ちがよいことを意味します。その反対は、気分が悪い・不愉快であることを表す「不快」です。「不便」は便利の対義語、「劣悪」は優良の対義語であり、意味の対応が異なります。

</QuizAnswer>

## 問2

「創造」の類義語として最も適切なものはどれか。

1. 想像
2. 独創
3. 製作
4. 破壊


<QuizAnswer>

**答え: 3**

「創造」は新しいものを初めてつくり出すことを意味します。「製作」もものをつくることを表すため、類義語として最も適切です。「想像」は心の中で思い描くことで意味が異なり、「独創」は独自の発想という意味で、つくる行為そのものとはずれがあります。

</QuizAnswer>

## 問3

「繁栄」の対義語として最も適切なものはどれか。

1. 衰退
2. 減少
3. 貧困
4. 消滅


<QuizAnswer>

**答え: 1**

「繁栄」は勢いが盛んで栄えることを意味します。その反対に、勢いが衰えて退くことを表す「衰退」が対義語です。「減少」は増加の対義語、「貧困」は裕福・富裕の対義語にあたり、「繁栄」との対応としては「衰退」が最も正確です。

</QuizAnswer>

## 問4

「温暖」の類義語として最も適切なものはどれか。

1. 高温
2. 温和
3. 炎熱
4. 快晴


<QuizAnswer>

**答え: 2**

「温暖」はあたたかく穏やかな気候を表す語です。「温和」もおだやかであたたかいという意味を持ち、類義語として適切です。「高温」や「炎熱」は暑さの程度が強すぎ、「快晴」は天気を表す語であり、意味の方向が異なります。

</QuizAnswer>

## 問5

「純粋」の対義語として最も適切なものはどれか。

1. 複雑
2. 不純
3. 汚染
4. 混乱


<QuizAnswer>

**答え: 2**

「純粋」はまじりけがなく清らかであることを意味します。その反対は、まじりけがあることを表す「不純」です。「複雑」は単純の対義語、「混乱」は秩序の対義語にあたり、「純粋」の対義語としては「不純」が正確です。

</QuizAnswer>

## 問6

「援助」の類義語として最も適切なものはどれか。

1. 救済
2. 支援
3. 保護
4. 依頼


<QuizAnswer>

**答え: 2**

「援助」は力を貸して助けることを意味します。「支援」も同様に、ささえ助けるという意味で、最も近い類義語です。「救済」は苦しみから救うこと、「保護」はかばい守ることで、やや意味の方向が異なります。「依頼」は頼むことであり、助ける側ではありません。

</QuizAnswer>

## 問7

「膨張」の対義語として最も適切なものはどれか。

1. 削減
2. 圧縮
3. 収縮
4. 縮小


<QuizAnswer>

**答え: 3**

「膨張」はふくらんで大きくなることを意味します。その反対は、縮んで小さくなることを表す「収縮」です。「膨張⇔収縮」は物理的にふくらむ・縮むという対の関係で、最も正確な対義語の組み合わせです。「縮小」は拡大の対義語、「圧縮」は外から力を加えて小さくすることで、やや意味合いが異なります。

</QuizAnswer>

## 問8

「歳入」の対義語として最も適切なものはどれか。

1. 支出
2. 歳出
3. 損失
4. 消費


<QuizAnswer>

**答え: 2**

「歳入」は国や地方公共団体の一年間の収入を意味します。その対義語は、一年間の支出を表す「歳出」です。「支出」は「収入」の対義語としては使えますが、「歳入」に対応する語としては「歳出」が正確な対義語となります。

</QuizAnswer>

## 問9

「慎重」の類義語として最も適切なものはどれか。

1. 丁寧
2. 用心
3. 臆病
4. 真剣


<QuizAnswer>

**答え: 2**

「慎重」は注意深く軽々しく行動しないことを意味します。「用心」も気をつけて注意することを表し、類義語として最も適切です。「丁寧」は礼儀正しさや細やかさを表し、「臆病」は必要以上に怖がることで、慎重とは意味が異なります。

</QuizAnswer>

## 問10

「供給」の対義語として最も適切なものはどれか。

1. 消費
2. 不足
3. 需要
4. 購入


<QuizAnswer>

**答え: 3**

「供給」は求めに応じて物資やサービスを提供することを意味します。その対義語は、物資やサービスを必要とし求めることを表す「需要」です。「供給と需要（需要と供給）」は経済の基本概念として対になる語です。「消費」は生産の対義語、「不足」は余剰の対義語にあたります。

</QuizAnswer>
