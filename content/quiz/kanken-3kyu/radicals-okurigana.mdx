## 問1

「届」の部首として正しいものはどれか。

1. 口（くち）
2. 田（た）
3. 尸（しかばね）
4. 由（よし）


<QuizAnswer>

**答え: 3**

「届」の部首は「尸（しかばね）」です。「届」は尸の中に「由」が入った構造ですが、漢字全体を囲む外側の部分が部首となります。「尸」を部首に持つ漢字には「届」「届」「屈」「展」「属」などがあります。

</QuizAnswer>

## 問2

「慎」の部首として正しいものはどれか。

1. 忄（りっしんべん）
2. 真（ま）
3. 目（め）
4. 心（こころ）


<QuizAnswer>

**答え: 1**

「慎」の部首は「忄（りっしんべん）」です。「りっしんべん」は「心」を偏の位置に置いたときの形で、感情や心の状態に関する漢字に多く使われます。「心（こころ）」と混同しやすいですが、左側の縦の点が「りっしんべん」であり、「慎」「悔」「惜」などが同じ部首です。

</QuizAnswer>

## 問3

「蒸」の部首として正しいものはどれか。

1. 水（みず）
2. 艹（くさかんむり）
3. 火（ひ）
4. 灬（れんが）


<QuizAnswer>

**答え: 2**

「蒸」の部首は「艹（くさかんむり）」です。「蒸」は下部に「灬（れんが）」もあるため迷いやすいですが、上部の「くさかんむり」が部首です。植物に関する漢字だけでなく、「蒸」「蔵」「薫」など幅広い漢字がくさかんむりを部首に持ちます。

</QuizAnswer>

## 問4

部首「冫」の名称として正しいものはどれか。

1. さんずい
2. うかんむり
3. こざとへん
4. にすい


<QuizAnswer>

**答え: 4**

「冫」の部首名は「にすい」です。点が二つ並んだ形で、冷たさや凍ることに関係する漢字に使われます。三つの点からなる「氵（さんずい）」と混同しやすいので注意が必要です。「冷」「凍」「冶」「凝」などが「にすい」を部首に持つ漢字です。

</QuizAnswer>

## 問5

「あぶない」を漢字と送り仮名で正しく書いたものはどれか。

1. 危い
2. 危ない
3. 危あない
4. 危ぶない


<QuizAnswer>

**答え: 2**

正しくは「危ない」です。送り仮名は「ない」の部分となります。「危い」と書いてしまう間違いが多いですが、活用語尾だけでなく「な」も送り仮名に含めるのがルールです。読み間違いを防ぐため、「あぶ」までを漢字に含め「ない」を送るのが正しい表記です。

</QuizAnswer>

## 問6

「いちじるしい」を漢字と送り仮名で正しく書いたものはどれか。

1. 著しい
2. 著るしい
3. 著い
4. 著じるしい


<QuizAnswer>

**答え: 1**

正しくは「著しい」です。送り仮名は「しい」の部分です。読み仮名が長い漢字は送り仮名を多くしたくなりますが、「著」に「しい」を送るのが正しい表記です。「著しい」は漢検3級でも頻出の送り仮名問題で、「著るしい」などと余分な仮名を送ってしまう誤りに注意しましょう。

</QuizAnswer>

## 問7

「いさぎよい」を漢字と送り仮名で正しく書いたものはどれか。

1. 潔よい
2. 潔ぎよい
3. 潔い
4. 潔きよい


<QuizAnswer>

**答え: 3**

正しくは「潔い」です。送り仮名は「い」だけです。「いさぎよ」までが漢字「潔」の読みに当たり、活用語尾の「い」のみを送ります。読みが長いため「潔よい」や「潔ぎよい」と余計に送り仮名を書いてしまいがちですが、送るのは「い」だけが正解です。

</QuizAnswer>

## 問8

次の文の中に誤字が一つある。誤字を含む語はどれか。

「彼は毎朝、新聞の **記治** を熱心に読んでいる。」

1. 毎朝
2. 新聞
3. 熱心
4. 記治


<QuizAnswer>

**答え: 4**

「記治」が誤りで、正しくは「記事」です。「事（じ）」と「治（じ）」は同じ読みを持つため混同しやすい漢字です。「記事」は新聞や雑誌に載せる文章のことで、「治」は政治・治療など別の意味の漢字です。同音異字の誤りに注意しましょう。

</QuizAnswer>

## 問9

次の文の中に誤字が一つある。誤字を含む語はどれか。

「環境問題について **真建** な議論が行われた。」

1. 環境
2. 真建
3. 議論
4. 問題


<QuizAnswer>

**答え: 2**

「真建」が誤りで、正しくは「真剣」です。「剣（けん）」と「建（けん）」は音読みが同じため書き間違えやすい漢字です。「真剣」は刀剣の「剣」を使い、「本気で取り組むさま」を意味します。「建」は建設・建築などに使う漢字で、意味が全く異なります。

</QuizAnswer>

## 問10

「衝」の部首として正しいものはどれか。

1. 重（おもい）
2. 十（じゅう）
3. 行（ぎょうがまえ）
4. 中（なか）


<QuizAnswer>

**答え: 3**

「衝」の部首は「行（ぎょうがまえ）」です。「行」が左右に分かれて「彳」と「亍」になり、その間に「重」が挟まれた構造です。「ぎょうがまえ」は道や行動に関係する部首で、「街」「術」「衛」なども同じ部首を持ちます。中央の「重」に惑わされず、外側の構えが部首であることを覚えましょう。

</QuizAnswer>
