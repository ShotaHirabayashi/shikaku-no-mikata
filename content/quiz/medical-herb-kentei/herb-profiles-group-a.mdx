## 問1

ウスベニアオイの学名として正しいものはどれか。

1. Matricaria chamomilla
2. Urtica dioica
3. Malva sylvestris
4. Sambucus nigra


<QuizAnswer>

**答え: 3**

ウスベニアオイの学名は Malva sylvestris で、アオイ科に属します。選択肢1はジャーマンカモミール、2はネトル、4はエルダーフラワーの学名です。

</QuizAnswer>

## 問2

エキナセアに期待される主な作用として最も適切なものはどれか。

1. 免疫賦活作用
2. 利胆作用
3. 抗うつ作用
4. 鎮痙作用


<QuizAnswer>

**答え: 1**

エキナセア（Echinacea purpurea）は免疫賦活作用で知られ、風邪やインフルエンザの初期に用いられます。体の免疫機能を高めることで、感染症への抵抗力を強化するとされています。

</QuizAnswer>

## 問3

エルダーフラワーの特徴に関する記述として正しいものはどれか。

1. キク科に属し、根を主に使用する
2. オトギリソウ科に属し、抗うつ作用がある
3. イラクサ科に属し、抗アレルギー作用がある
4. スイカズラ科に属し、発汗・利尿作用がある


<QuizAnswer>

**答え: 4**

エルダーフラワー（Sambucus nigra）はスイカズラ科に属し、花部を使用します。発汗作用と利尿作用があり、ヨーロッパでは「インフルエンザの特効薬」とも呼ばれ、風邪の初期症状に広く用いられてきました。

</QuizAnswer>

## 問4

ジャーマンカモミールの使用部位として正しいものはどれか。

1. 根
2. 花
3. 葉
4. 果実


<QuizAnswer>

**答え: 2**

ジャーマンカモミール（Matricaria chamomilla）は花部を使用します。キク科に属し、精油成分のカマズレンなどを含み、消炎作用や鎮静作用に優れています。「植物のお医者さん」とも呼ばれるハーブです。

</QuizAnswer>

## 問5

セントジョンズワートの学名として正しいものはどれか。

1. Taraxacum officinale
2. Echinacea purpurea
3. Hypericum perforatum
4. Malva sylvestris


<QuizAnswer>

**答え: 3**

セントジョンズワートの学名は Hypericum perforatum で、オトギリソウ科に属します。和名はセイヨウオトギリソウです。選択肢1はダンディライオン、2はエキナセア、4はウスベニアオイの学名です。

</QuizAnswer>

## 問6

ダンディライオンに期待される主な作用の組み合わせとして最も適切なものはどれか。

1. 利胆作用・強肝作用
2. 免疫賦活作用・抗ウイルス作用
3. 鎮静作用・抗うつ作用
4. 発汗作用・利尿作用


<QuizAnswer>

**答え: 1**

ダンディライオン（Taraxacum officinale）はキク科に属し、根と葉を使用します。胆汁の分泌を促す利胆作用と肝機能を高める強肝作用が主な特徴です。根を焙煎したものはタンポポコーヒーとしても親しまれています。

</QuizAnswer>

## 問7

ネトルに関する記述として正しいものはどれか。

1. キク科に属し、花部を使用する
2. アオイ科に属し、粘液質を豊富に含む
3. オトギリソウ科に属し、ヒペリシンを含む
4. イラクサ科に属し、抗アレルギー作用がある


<QuizAnswer>

**答え: 4**

ネトル（Urtica dioica）はイラクサ科に属し、葉部を使用します。フラボノイドのケルセチンなどを含み、抗アレルギー作用で知られています。花粉症の季節に用いられるハーブとして有名です。

</QuizAnswer>

## 問8

ウスベニアオイに豊富に含まれ、粘膜を保護する働きを持つ成分はどれか。

1. カマズレン
2. ヒペリシン
3. 粘液質
4. ケルセチン


<QuizAnswer>

**答え: 3**

ウスベニアオイは粘液質を豊富に含んでおり、のどや胃腸などの粘膜を保護する働きがあります。ハーブティーにするととろみのある液体になるのが特徴です。カマズレンはジャーマンカモミール、ヒペリシンはセントジョンズワートの成分です。

</QuizAnswer>

## 問9

エキナセアの科名と使用部位の組み合わせとして正しいものはどれか。

1. アオイ科 ― 花
2. スイカズラ科 ― 葉
3. イラクサ科 ― 根
4. キク科 ― 根および地上部


<QuizAnswer>

**答え: 4**

エキナセア（Echinacea purpurea）はキク科に属し、根および地上部（茎・葉・花）を使用します。北米先住民が古くから感染症の治療に用いてきたハーブで、現在でも免疫力を高める目的で広く利用されています。

</QuizAnswer>

## 問10

セントジョンズワートに含まれる特徴的な成分ヒペリシンと関連が深い作用はどれか。

1. 消炎・鎮静作用
2. 抗うつ作用
3. 利胆・強肝作用
4. 発汗・利尿作用


<QuizAnswer>

**答え: 2**

セントジョンズワートに含まれるヒペリシンやヒペルフォリンは、抗うつ作用に関与する主要成分です。軽度から中程度のうつ症状に用いられますが、多くの医薬品（経口避妊薬、免疫抑制剤など）と相互作用を起こすため、服用中の薬がある場合は注意が必要です。

</QuizAnswer>
