## 問1

フラボノイドの基本的な化学構造として正しいものはどれか。

1. ステロイド骨格を持つ脂溶性化合物である
2. C6-C3-C6の炭素骨格を基本構造とするポリフェノールの一種である
3. 窒素原子を含む塩基性化合物である
4. グルコースが直鎖状に結合した多糖類である


<QuizAnswer>

**答え: 2**

フラボノイドは2つのベンゼン環（C6）が3つの炭素鎖（C3）で結ばれたC6-C3-C6の基本骨格を持つポリフェノールの一種です。フラボン、フラボノール、イソフラボン、アントシアニジンなど多くのサブグループがあり、抗酸化作用や抗炎症作用など多彩な生理活性を示します。窒素を含むのはアルカロイド、多糖類は粘液質の特徴です。

</QuizAnswer>

## 問2

タンニンの代表的な生理作用である「収れん作用」の説明として最も適切なものはどれか。

1. 中枢神経を刺激して覚醒を促す作用
2. タンパク質と結合して組織を引き締め、粘膜を保護する作用
3. 平滑筋を弛緩させて痙攣を和らげる作用
4. 腸内細菌を増殖させて腸内環境を整える作用


<QuizAnswer>

**答え: 2**

収れん作用（アストリンジェント作用）とは、タンニンが組織のタンパク質と結合して変性させ、表面を引き締めることで粘膜を保護する働きのことです。カテキンなどのタンニンは、この作用により下痢の緩和や口腔内・消化管粘膜の保護に利用されます。お茶を飲んだときに感じる渋味も、タンニンの収れん作用によるものです。

</QuizAnswer>

## 問3

精油（エッセンシャルオイル）の特徴として誤っているものはどれか。

1. 揮発性を持ち、特有の芳香を有する
2. 主にテルペノイド系化合物で構成される
3. 水に溶けやすい親水性の成分である
4. 植物の花、葉、果皮、樹皮などから得られる


<QuizAnswer>

**答え: 3**

精油は水にほとんど溶けない脂溶性（親油性）の揮発性成分です。主にモノテルペン、セスキテルペンなどのテルペノイド系化合物で構成され、植物の様々な部位から抽出されます。この脂溶性の性質があるため、ハーブティーでは精油成分の抽出が限られ、蓋をして蒸気を逃がさない工夫が必要になります。

</QuizAnswer>

## 問4

アルカロイドに共通する化学的特徴はどれか。

1. 分子内に窒素原子を含む塩基性の化合物である
2. 糖と結合した配糖体として存在する
3. 脂肪酸が3つグリセロールに結合した構造を持つ
4. ビタミンC様の強い還元力を持つ


<QuizAnswer>

**答え: 1**

アルカロイドは分子内に窒素原子を含む塩基性の天然有機化合物の総称です。カフェイン、モルヒネ、アトロピンなどが代表例で、少量でも強い薬理作用を示すものが多いのが特徴です。「アルカロイド」の名称自体が「アルカリ（塩基）のようなもの」に由来しています。

</QuizAnswer>

## 問5

植物の粘液質（ミューシレージ）の主な働きとして最も適切なものはどれか。

1. 強い抗菌作用で病原菌を死滅させる
2. 粘膜表面を覆って保護し、炎症を和らげる
3. 交感神経を刺激して血圧を上昇させる
4. 肝臓でのコレステロール合成を阻害する


<QuizAnswer>

**答え: 2**

粘液質は水を含むとゲル状になる多糖類の一種で、消化管や気道の粘膜表面を覆うように保護膜を形成します。これにより刺激物質から粘膜を守り、炎症を鎮める作用（粘膜保護作用）を発揮します。ウスベニアオイなどに多く含まれ、のどの痛みや胃腸の炎症緩和に用いられます。

</QuizAnswer>

## 問6

苦味質（ビターズ）が体に及ぼす作用のメカニズムとして正しいものはどれか。

1. 舌の苦味受容体を刺激し、反射的に消化液の分泌を促進する
2. 腸管壁に直接作用して蠕動運動を抑制する
3. 血中のインスリン濃度を直接低下させる
4. 脳内のセロトニン受容体に結合して気分を高揚させる


<QuizAnswer>

**答え: 1**

苦味質は舌の苦味受容体（味蕾）を刺激することで、迷走神経を介した反射により胃液・胆汁・膵液などの消化液分泌を促進します。これを「苦味健胃作用」と呼び、食欲増進や消化促進に役立ちます。ダンディライオンの根などに含まれる苦味成分はこの作用を利用したものです。

</QuizAnswer>

## 問7

フェノール酸の代表例であるロスマリン酸やクロロゲン酸に共通して認められる主な作用はどれか。

1. 血液凝固を促進する作用
2. 抗酸化作用
3. 骨密度を増加させる作用
4. 血糖値を上昇させる作用


<QuizAnswer>

**答え: 2**

ロスマリン酸（ローズマリーやシソ科植物に含有）やクロロゲン酸（コーヒー豆などに含有）はいずれもフェノール酸に分類され、強い抗酸化作用を持つことが特徴です。フェノール酸はフェノール性ヒドロキシ基を持ち、活性酸素やフリーラジカルを消去することで細胞の酸化ダメージを防ぎます。さらに抗炎症作用も併せ持つことが知られています。

</QuizAnswer>

## 問8

植物化学（フィトケミカル）に関する説明として最も適切なものはどれか。

1. 植物が産生する化学成分のうち、三大栄養素（糖質・脂質・タンパク質）を指す
2. 植物が産生する二次代謝産物の総称で、自己防御や環境適応のために作られる
3. 人工的に合成された植物由来医薬品の総称である
4. 植物の光合成に直接関与する色素成分のみを指す


<QuizAnswer>

**答え: 2**

フィトケミカル（植物化学成分）とは、植物が紫外線や外敵から身を守るために産生する二次代謝産物の総称です。フラボノイド、テルペノイド、アルカロイド、フェノール酸などが含まれ、植物の生存に直接必要な一次代謝産物（糖質・アミノ酸など）とは区別されます。これらは人体にも抗酸化、抗炎症、免疫調節などの有益な作用をもたらします。

</QuizAnswer>

## 問9

メディカルハーブにおける「相乗効果（シナジー効果）」の説明として最も適切なものはどれか。

1. 同じハーブを大量に摂取するほど効果が比例して高まること
2. 複数の成分が協調的に働き、単独成分の効果の合計を上回る作用を生むこと
3. 有効成分を単離・精製することで副作用がなくなること
4. 1種類の成分が体内のあらゆる臓器に同時に作用すること


<QuizAnswer>

**答え: 2**

相乗効果（シナジー効果）とは、ハーブに含まれる複数の成分が互いに協調的に作用し合うことで、個々の成分を単独で用いた場合の効果の合計よりも大きな作用を生み出す現象です。これはメディカルハーブが単一成分の医薬品と異なる「多成分系」である大きな特徴であり、穏やかでありながら多面的な効果をもたらす理由の一つとされています。

</QuizAnswer>

## 問10

メディカルハーブが「多成分系」であることの利点として最も適切なものはどれか。

1. 単一成分の医薬品よりも常に強力な薬理効果を発揮する
2. 有効成分の特定が容易で、品質管理がしやすい
3. 複数の成分が多面的に作用し、副作用が緩和される傾向がある
4. 成分数が多いほど体内での分解速度が遅くなる


<QuizAnswer>

**答え: 3**

メディカルハーブには数十〜数百種類の成分が含まれる多成分系であり、これらの成分が体内で多面的に作用します。主要成分の効果を別の成分が補助・増強する一方で、副作用を緩和する方向に働く成分も含まれることがあります。この「穏やかで多面的な作用」が多成分系の利点であり、単一成分の強力な作用とは異なるメディカルハーブの特徴です。

</QuizAnswer>
