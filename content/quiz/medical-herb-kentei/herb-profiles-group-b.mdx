## 問1

ハイビスカスの学名として正しいものはどれか。

1. Hibiscus rosa-sinensis
2. Hibiscus sabdariffa
3. Hibiscus syriacus
4. Hibiscus mutabilis


<QuizAnswer>

**答え: 2**

メディカルハーブとして用いられるハイビスカスの学名は Hibiscus sabdariffa（ヒビスクス サブダリファ）です。アオイ科に属し、別名ローゼルとも呼ばれます。園芸品種のハイビスカス（H. rosa-sinensis）とは異なる種であることに注意しましょう。

</QuizAnswer>

## 問2

パッションフラワーは、その優れた鎮静作用から別名何と呼ばれているか。

1. 植物性の抗生物質
2. 飲むサラダ
3. 植物性の精神安定剤
4. 安産のハーブティー


<QuizAnswer>

**答え: 3**

パッションフラワー（Passiflora incarnata）は、穏やかで確実な鎮静作用を持つことから「植物性の精神安定剤（トランキライザー）」と呼ばれています。不安、緊張、不眠などに用いられ、トケイソウ科に属します。

</QuizAnswer>

## 問3

ペパーミントに含まれる主要な精油成分はどれか。

1. リナロール
2. カマズレン
3. l-メントール
4. ファルネソール


<QuizAnswer>

**答え: 3**

ペパーミント（Mentha x piperita）の精油の主成分はl-メントールです。このメントールが消化促進や鎮痙作用をもたらし、消化不良や腹部膨満感の緩和に役立ちます。シソ科に属し、使用部位は葉部です。

</QuizAnswer>

## 問4

メディカルハーブ検定の対象15種のうち、唯一カフェインを含むハーブはどれか。

1. ペパーミント
2. ローズヒップ
3. マテ
4. リンデン


<QuizAnswer>

**答え: 3**

マテ（Ilex paraguariensis）は、検定対象15種の中で唯一カフェインを含有するハーブです。モチノキ科に属し、カフェインやテオブロミンなどのアルカロイドによる覚醒作用のほか、利尿作用、脂肪分解作用があります。「飲むサラダ」とも呼ばれます。

</QuizAnswer>

## 問5

マルベリーに含まれるデオキシノジリマイシン（DNJ）の作用として正しいものはどれか。

1. 免疫細胞を活性化する
2. 腸管での糖の吸収を阻害する
3. 血管を拡張して血圧を下げる
4. 中枢神経を鎮静する


<QuizAnswer>

**答え: 2**

マルベリー（Morus alba）に含まれるDNJはα-グルコシダーゼ阻害作用を持ち、腸管での糖の吸収を抑制します。そのため食後の血糖値上昇を抑える効果があり、食前に飲むのが効果的です。クワ科に属し、使用部位は葉です。

</QuizAnswer>

## 問6

「安産のハーブティー」として知られ、子宮筋の調整作用があるハーブはどれか。

1. パッションフラワー
2. リンデン
3. ハイビスカス
4. ラズベリーリーフ


<QuizAnswer>

**答え: 4**

ラズベリーリーフ（Rubus idaeus）は「安産のハーブティー」として古くから知られ、子宮筋や骨盤周りの筋肉を調整する作用があります。バラ科に属し、主要成分のフラガリン（フラボノイド配糖体）が鎮痙作用に関与しています。月経痛やPMSにも用いられます。

</QuizAnswer>

## 問7

リンデンの使用部位として正しいものはどれか。

1. 根部
2. 葉部
3. 花部
4. 果実


<QuizAnswer>

**答え: 3**

リンデン（Tilia europaea）の使用部位は花部（花と苞）です。アオイ科（旧シナノキ科）に属し、発汗・利尿・鎮静作用があります。和名は西洋菩提樹で、風邪の初期症状や不安・不眠に用いられます。

</QuizAnswer>

## 問8

ローズヒップの使用部位として正しいものはどれか。

1. 花弁
2. 葉部
3. 偽果
4. 種子


<QuizAnswer>

**答え: 3**

ローズヒップ（Rosa canina）の使用部位は偽果（ぎか）です。「ビタミンCの爆弾」と呼ばれるほど豊富なビタミンCを含み、バラ科に属します。感染症予防や便秘にも適用され、ハイビスカスとのブレンドが定番です。

</QuizAnswer>

## 問9

ハイビスカスに豊富に含まれ、代謝促進や疲労回復に寄与する成分はどれか。

1. カフェイン
2. クエン酸
3. デオキシノジリマイシン
4. フラガリン


<QuizAnswer>

**答え: 2**

ハイビスカスにはクエン酸やリンゴ酸などの植物酸が豊富に含まれ、エネルギー代謝を促進し疲労回復を助けます。使用部位は萼（がく）で、鮮やかな赤色はアントシアニン色素（ヒビスシン）によるものです。

</QuizAnswer>

## 問10

パッションフラワーに含まれる主要なフラボノイド配糖体はどれか。

1. ルチン
2. ヒペロシド
3. ビテキシン
4. フラガリン


<QuizAnswer>

**答え: 3**

パッションフラワー（Passiflora incarnata）の主要成分にはフラボノイド配糖体のビテキシンがあり、鎮静作用に寄与しています。そのほかアルカロイド（ハルマン、ハルモール）やフラボノイドのアピゲニンも含まれます。使用部位は地上部全草です。

</QuizAnswer>
