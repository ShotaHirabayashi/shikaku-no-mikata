## 問1

胃もたれや消化不良の症状に対して、ブレンドティーとして組み合わせることが最も一般的なハーブの組み合わせはどれか。

1. エキナセア＋ローズヒップ
2. ジャーマンカモミール＋ペパーミント
3. ネトル＋エルダーフラワー
4. マテ＋ハイビスカス


<QuizAnswer>

**答え: 2**

ジャーマンカモミールの消炎・鎮痙作用とペパーミントのメントールによる胃筋の緊張緩和作用は、胃もたれや消化不良に対する代表的な組み合わせです。1:1の比率でブレンドし、食後に飲用するのが基本的な活用法です。

</QuizAnswer>

## 問2

花粉症対策として、花粉飛散の1〜2か月前から予防的に飲み始めるハーブブレンドに最も適した組み合わせはどれか。

1. パッションフラワー＋リンデン
2. マルベリー＋ダンディライオン
3. ネトル＋エルダーフラワー
4. ラズベリーリーフ＋ジャーマンカモミール


<QuizAnswer>

**答え: 3**

ネトルにはフラボノイドのクエルセチンによる抗ヒスタミン作用があり、エルダーフラワーには抗カタル作用（鼻粘膜の炎症を鎮める作用）があります。この2種のブレンドは花粉症の予防的ケアとして代表的であり、花粉飛散の1〜2か月前から継続的に飲用することが推奨されています。

</QuizAnswer>

## 問3

月経前症候群（PMS）に伴う下腹部の痛みやイライラに対し、子宮筋の調整作用を持つハーブとして最も適切なものはどれか。

1. ダンディライオン
2. ローズヒップ
3. ラズベリーリーフ
4. マテ


<QuizAnswer>

**答え: 3**

ラズベリーリーフにはフラガリンという成分が含まれており、子宮や骨盤周囲の筋群を調整する作用があります。「安産のハーブ」とも呼ばれ、PMSに伴う下腹部痛や生理痛の緩和に活用されます。月経の7〜10日前から飲用を開始するのが効果的です。

</QuizAnswer>

## 問4

肌荒れ対策として、粘液質による保湿効果を活かした外用の化粧水を作る場合、冷水抽出（水出し）が推奨されるハーブはどれか。

1. ペパーミント
2. エキナセア
3. ウスベニアオイ
4. セントジョンズワート


<QuizAnswer>

**答え: 3**

ウスベニアオイ（マロウ）は豊富な粘液質を含んでおり、冷水抽出（冷浸剤）にすることで粘液質が効率よく抽出されます。この冷浸液をそのまま化粧水代わりに使用でき、保湿効果に優れたスキンケアレシピとして知られています。

</QuizAnswer>

## 問5

肩こりや腰痛の緩和を目的として、浸出油（インフューズドオイル）を作り患部に塗布してマッサージする際に用いるハーブとして最も適切なものはどれか。

1. ハイビスカス
2. マルベリー
3. ネトル
4. セントジョンズワート


<QuizAnswer>

**答え: 4**

セントジョンズワートの浸出油（インフューズドオイル）には抗炎症・神経鎮静作用があり、神経痛や筋肉痛に効果的です。ドライハーブをオリーブ油などに2週間漬け込んで作ります。ただし、成分のヒペリシンに光毒性があるため、使用は夜間に行うことが重要です。

</QuizAnswer>

## 問6

ストレスによる不眠に対して、就寝前に飲む鎮静ブレンドとして最も適切な組み合わせはどれか。

1. マテ＋ハイビスカス＋ローズヒップ
2. エキナセア＋エルダーフラワー＋ローズヒップ
3. パッションフラワー＋ジャーマンカモミール＋リンデン
4. ネトル＋ダンディライオン＋ペパーミント


<QuizAnswer>

**答え: 3**

パッションフラワーは「植物性の精神安定剤」と呼ばれGABAの働きを促進し、ジャーマンカモミールは穏やかな鎮静・安眠作用があり、リンデンは甘い香りで高ぶった神経を鎮めます。この3種は不眠対策の代表的な鎮静ブレンドです。就寝30分〜1時間前にゆっくり飲むのが効果的です。

</QuizAnswer>

## 問7

風邪の引き始めに、発汗を促して体を温める目的で用いるブレンドとして最も適切な組み合わせはどれか。

1. マルベリー＋ダンディライオン＋マテ
2. エルダーフラワー＋リンデン＋ペパーミント
3. ラズベリーリーフ＋パッションフラワー＋ローズヒップ
4. ウスベニアオイ＋ネトル＋ハイビスカス


<QuizAnswer>

**答え: 2**

エルダーフラワーとリンデンはいずれも発汗促進作用を持ち、風邪の引き始めに体を温めて発汗させることで回復を助けます。ペパーミントの鼻づまり解消・発汗作用も加わり、風邪の初期症状に対する代表的なブレンドです。エルダーフラワーは「インフルエンザの特効薬」とも呼ばれています。

</QuizAnswer>

## 問8

ダイエット目的で糖質の吸収を抑えたい場合、食事の15〜30分前に飲むことが推奨されるハーブはどれか。

1. パッションフラワー
2. ジャーマンカモミール
3. エキナセア
4. マルベリー


<QuizAnswer>

**答え: 4**

マルベリー（桑の葉）にはDNJ（1-デオキシノジリマイシン）という成分が含まれており、小腸での糖の吸収を抑制する作用があります。この効果を最大限に活かすには、食事の15〜30分前に飲用することが重要です。食後に飲んでも効果が薄れるため、飲むタイミングがポイントとなります。

</QuizAnswer>

## 問9

美白・シミ対策を目的とした内服ブレンドとして、ビタミンCを豊富に含むハーブの組み合わせで最も適切なものはどれか。

1. セントジョンズワート＋パッションフラワー＋リンデン
2. ローズヒップ＋ハイビスカス＋ジャーマンカモミール
3. ネトル＋エルダーフラワー＋ペパーミント
4. ダンディライオン＋マテ＋マルベリー


<QuizAnswer>

**答え: 2**

ローズヒップはレモンの20〜40倍のビタミンCを含み、コラーゲン生成促進やメラニン抑制に効果的です。ハイビスカスのクエン酸・ビタミンCが肌のターンオーバーを促進し、ジャーマンカモミールに含まれるアズレンが消炎・美白効果を発揮します。この3種は美白・シミ対策の代表的な内服ブレンドです。

</QuizAnswer>

## 問10

鼻づまりやのどの症状がある風邪のとき、蒸気吸入（スチーム吸入）を行う際の注意点として正しいものはどれか。

1. ペパーミントを使う場合、メントールの刺激があるため目を閉じて行う
2. ハーブは冷水で抽出してから蒸気を発生させる
3. 蒸気吸入は30分以上継続して行うのが効果的である
4. 吸入中はタオルをかぶらず、蒸気を自然に吸い込むようにする


<QuizAnswer>

**答え: 1**

蒸気吸入では、洗面器にハーブを入れて熱湯を注ぎ、頭からタオルをかぶって蒸気を逃がさないようにして5〜10分間吸入します。特にペパーミントを使用する場合はメントールの刺激が強いため、必ず目を閉じて行うことが重要です。冷水ではなく熱湯を使い、タオルで覆って蒸気を効率よく吸入するのが正しい方法です。

</QuizAnswer>
