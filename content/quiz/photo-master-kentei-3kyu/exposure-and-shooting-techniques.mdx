## 問1

写真の明るさを決める「露出の三要素」として正しい組み合わせはどれか。

1. 絞り・シャッター速度・ホワイトバランス
2. 絞り・シャッター速度・ISO感度
3. シャッター速度・ISO感度・焦点距離
4. 絞り・ISO感度・画素数


<QuizAnswer>

**答え: 2**

露出の三要素とは、絞り（F値）・シャッター速度・ISO感度の3つを指します。絞りはレンズを通る光の量、シャッター速度はセンサーに光が当たる時間、ISO感度はセンサーの光に対する感度をそれぞれコントロールし、この3つの組み合わせによって写真の明るさ（露出）が決まります。

</QuizAnswer>

## 問2

絞り値（F値）をF4からF8に変更した場合、レンズを通過する光の量はどう変化するか。

1. 光の量は4倍になる
2. 光の量は2倍になる
3. 光の量は1/2になる
4. 光の量は1/4になる


<QuizAnswer>

**答え: 4**

F値は1段変わるごとに光の量が2倍または1/2になります。F4→F5.6で1/2、F5.6→F8でさらに1/2となるため、F4からF8への変更は2段分の絞り込みで、光の量は1/2×1/2＝1/4になります。F値が大きくなるほど絞りが絞られ、通過する光の量は少なくなります。

</QuizAnswer>

## 問3

シャッター速度を1/125秒から1/250秒に変更した場合、露出はどのように変化するか。

1. 1段分明るくなる
2. 2段分明るくなる
3. 1段分暗くなる
4. 変化しない


<QuizAnswer>

**答え: 3**

シャッター速度を1/125秒から1/250秒に速くすると、シャッターが開いている時間が半分になるため、センサーに届く光の量は半分（1/2）になります。光の量が半分になることは「1段分暗くなる」ことを意味します。シャッター速度を速くすると暗く、遅くすると明るくなるという関係を覚えておきましょう。

</QuizAnswer>

## 問4

雪景色を撮影したところ、雪が灰色っぽく暗い写真になってしまった。この場合の適切な対処法はどれか。

1. 露出補正をマイナス（−）にする
2. 露出補正をプラス（+）にする
3. ISO感度を下げる
4. シャッター速度を速くする


<QuizAnswer>

**答え: 2**

カメラの測光機能は画面全体を中間的な明るさ（18%グレー）に合わせようとするため、白い被写体（雪景色など）を撮影すると暗く写る傾向があります。これを補正するには露出補正をプラス（+）に設定して明るくします。逆に黒い被写体が明るく写ってしまう場合はマイナス（−）補正を行います。

</QuizAnswer>

## 問5

カメラの測光モードのうち、画面内の特定の狭い範囲だけを測光する方式はどれか。

1. 多分割測光（評価測光）
2. 中央部重点測光
3. スポット測光
4. 平均測光


<QuizAnswer>

**答え: 3**

スポット測光は、画面内のごく狭い範囲（通常は画面の2〜3%程度）だけを測光する方式です。逆光時やステージ撮影など、特定の被写体だけの明るさを正確に測りたい場面で有効です。多分割測光は画面全体を分割して総合的に判断し、中央部重点測光は画面中央付近を重視して測光します。

</QuizAnswer>

## 問6

写真の構図で、画面を縦横それぞれ三等分し、その交点上に被写体を配置する方法を何というか。

1. 日の丸構図
2. 対角線構図
3. 三分割構図
4. 二分割構図


<QuizAnswer>

**答え: 3**

三分割構図（三分割法）は、画面を縦横にそれぞれ三等分する線を引き、その線の交点に主被写体を配置する構図法です。被写体を中心からずらして配置することで画面に余白と奥行きが生まれ、バランスの取れた安定感のある写真に仕上がります。写真撮影の最も基本的な構図の一つとして広く用いられています。

</QuizAnswer>

## 問7

動きの速い被写体（走る子どもなど）を止まっているようにシャープに写すためには、どのような設定が最も効果的か。

1. 絞りを開放にする
2. ISO感度を最低にする
3. シャッター速度を速くする
4. 露出補正をプラスにする


<QuizAnswer>

**答え: 3**

動きの速い被写体をブレなくシャープに写し止めるためには、シャッター速度を速く設定することが最も効果的です。例えば走る人物なら1/500秒以上が目安となります。シャッター速度を速くすると光の量が減って暗くなるため、ISO感度を上げたり絞りを開けたりして露出を補う必要があります。これが「被写体ブレ」への基本的な対策です。

</QuizAnswer>

## 問8

ISO感度をISO 200からISO 800に上げた場合の説明として正しいものはどれか。

1. 感度が2段分上がり、同じ露出ならシャッター速度を2段遅くできる
2. 感度が2段分上がり、同じ露出ならシャッター速度を2段速くできる
3. 感度が4段分上がり、同じ露出ならシャッター速度を4段速くできる
4. 感度が1段分上がり、同じ露出ならシャッター速度を1段速くできる


<QuizAnswer>

**答え: 2**

ISO感度は数値が2倍になるごとに1段上がります。ISO 200→400で1段、ISO 400→800でもう1段なので、合計2段分感度が上がります。感度が2段分上がると同じ明るさでもセンサーが光を多く拾えるため、シャッター速度を2段分速くしても同じ露出が得られます。ただしISO感度を上げすぎるとノイズが増える点に注意が必要です。

</QuizAnswer>

## 問9

手ブレを防ぐために推奨される「シャッター速度の目安」として一般的に知られているルールはどれか。

1. シャッター速度を「焦点距離分の1秒」以上にする
2. シャッター速度を常に1/500秒以上にする
3. シャッター速度を「F値分の1秒」以上にする
4. シャッター速度を「ISO感度分の1秒」以上にする


<QuizAnswer>

**答え: 1**

手ブレを防ぐシャッター速度の目安として「焦点距離分の1秒」というルールが広く知られています。例えば焦点距離50mmのレンズなら1/50秒以上、200mmの望遠レンズなら1/200秒以上のシャッター速度が必要とされます。焦点距離が長くなるほどわずかなブレが拡大されるため、より速いシャッター速度が求められます。

</QuizAnswer>

## 問10

EV値（Exposure Value）について正しい説明はどれか。

1. EV値が1上がると光の量が3倍になる
2. EV値が1上がると光の量が2倍になる
3. EV値が1上がると光の量は1/2になる
4. EV値はカメラの画質を表す数値である


<QuizAnswer>

**答え: 3**

EV値（Exposure Value）は露出の明るさを数値化したもので、EV0はISO 100において絞りF1.0・シャッター速度1秒の組み合わせを基準としています。EV値が1段上がると被写体が2倍明るいことを意味し、適正露出を維持するにはカメラに入る光の量を1/2にする（絞りを1段絞る、またはシャッター速度を1段速くする）必要があります。つまりEV値が1上がると、必要な露光量は1/2になります。

</QuizAnswer>
