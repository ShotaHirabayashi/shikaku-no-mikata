## 問1

一眼レフカメラの内部には、レンズを通った光をファインダーへ導くための重要な部品がある。この部品は何か。

1. イメージセンサー
2. フォーカルプレーンシャッター
3. ミラー（反射鏡）
4. 液晶モニター


<QuizAnswer>

**答え: 3**

一眼レフカメラでは、レンズを通った光がカメラ内部のミラー（反射鏡）によって上方に反射され、ペンタプリズム（またはペンタミラー）を経由して光学ファインダーに導かれます。このミラーがあることが「一眼レフ（レフレックス）」の名前の由来です。ミラーレスカメラはこのミラーを省略した構造になっています。

</QuizAnswer>

## 問2

ミラーレス一眼カメラが一眼レフカメラよりも小型・軽量化しやすい主な理由として、最も適切なものはどれか。

1. イメージセンサーが小さいため
2. ミラーとペンタプリズムが不要なため
3. レンズマウントの口径が小さいため
4. バッテリーの容量が少ないため


<QuizAnswer>

**答え: 2**

ミラーレス一眼カメラは、一眼レフカメラに搭載されているミラー（反射鏡）とペンタプリズムを省略した構造です。これらの光学部品がなくなることでカメラボディを薄く・軽くでき、小型化が実現しやすくなっています。イメージセンサーのサイズ自体はフルサイズなど一眼レフと同等のものも多くあります。

</QuizAnswer>

## 問3

一眼レフカメラのファインダーに使われる「ペンタプリズム」の代わりに、軽量化のために用いられることがある部品はどれか。

1. ペンタミラー
2. ハーフミラー
3. ダイクロイックミラー
4. コンデンサーレンズ


<QuizAnswer>

**答え: 1**

ペンタプリズムはガラスの塊でできた五角形の光学部品で、正確な像をファインダーに導きますが重量があります。エントリークラスの一眼レフカメラでは、ガラスの代わりにミラーを組み合わせた「ペンタミラー」を採用することで軽量化を図っています。ペンタミラーはやや明るさや見え方で劣りますが、コストと重量の面で有利です。

</QuizAnswer>

## 問4

ミラーレス一眼カメラに搭載されている「EVF」とは何の略称か。

1. Electronic Variable Finder
2. Electronic View Finder
3. Enhanced Visual Focus
4. External Video Function


<QuizAnswer>

**答え: 2**

EVFは「Electronic View Finder（電子ビューファインダー）」の略称です。イメージセンサーが捉えた映像を小型液晶や有機ELディスプレイに表示する方式で、撮影前に露出やホワイトバランスの効果をリアルタイムで確認できるメリットがあります。一眼レフの光学ファインダー（OVF）とは異なり、ミラーを必要としません。

</QuizAnswer>

## 問5

一般的なデジタルカメラに使われる「フォーカルプレーンシャッター」の構造として正しいものはどれか。

1. レンズの中に組み込まれた絞り羽根で光を遮断する
2. 先幕と後幕の2枚の幕がイメージセンサー直前で走行する
3. 電子的に信号をオン・オフして露光を制御する
4. ミラーの上下動によって露光時間を制御する


<QuizAnswer>

**答え: 2**

フォーカルプレーンシャッターは、イメージセンサー（フィルム面）の直前に配置された先幕と後幕の2枚の幕で構成されるメカニカルシャッターです。先幕が走行して露光を開始し、設定されたシャッター速度に応じたタイミングで後幕が追いかけて露光を終了します。選択肢1はレンズシャッター、選択肢3は電子シャッターの説明です。

</QuizAnswer>

## 問6

デジタルカメラのイメージセンサーについて、「フルサイズ」と呼ばれるセンサーのサイズは、何を基準にした呼び方か。

1. 中判フィルムのサイズ
2. 35mmフィルム（ライカ判）のサイズ
3. APS-Cフィルムのサイズ
4. 映画用35mmフィルムのサイズ


<QuizAnswer>

**答え: 2**

「フルサイズ」とは、35mmフィルム（ライカ判）の撮像面積である約36mm×24mmとほぼ同じ大きさのイメージセンサーを指す呼称です。これより小さいセンサーサイズとしてAPS-C（約23.6mm×15.8mm）やマイクロフォーサーズ（約17.3mm×13mm）などがあり、フルサイズに対する「クロップ」サイズとして区別されています。

</QuizAnswer>

## 問7

カメラのオートフォーカス（AF）方式のうち、「位相差検出方式」の特徴として最も適切なものはどれか。

1. ピント合わせの速度は遅いが、精度が非常に高い
2. ピントのずれている方向と量を一度に検出でき、高速にピント合わせができる
3. 専用のAFセンサーが不要で、カメラの小型化に有利である
4. 暗い場所でのピント合わせに最も強い方式である


<QuizAnswer>

**答え: 2**

位相差検出方式は、レンズを通った光を2つに分けてセンサーに導き、2つの像のずれ（位相差）からピントのずれている方向と量を同時に算出します。これにより、一度の検出で合焦位置を特定できるため、高速なピント合わせが可能です。動きのある被写体の撮影に特に有利な方式として知られています。

</QuizAnswer>

## 問8

カメラの撮影モードで「Aモード（Avモード）」を選んだとき、撮影者が自分で設定する項目はどれか。

1. シャッタースピード
2. 絞り値（F値）
3. シャッタースピードと絞り値の両方
4. ISO感度のみ


<QuizAnswer>

**答え: 2**

Aモード（Aperture Priority＝絞り優先モード）では、撮影者が絞り値（F値）を設定し、それに対して適正露出となるシャッタースピードをカメラが自動的に決定します。背景のボケ量をコントロールしたい場合に便利なモードです。なお、シャッタースピードを自分で設定するのはSモード（シャッター優先モード）です。

</QuizAnswer>

## 問9

カメラの撮影モード「Mモード」の説明として正しいものはどれか。

1. カメラが絞りとシャッタースピードを自動で決める全自動モードである
2. 動画撮影専用のモードである
3. 絞り値とシャッタースピードの両方を撮影者が手動で設定するモードである
4. マクロ撮影に特化した近接撮影モードである


<QuizAnswer>

**答え: 3**

Mモード（Manual＝マニュアルモード）は、絞り値とシャッタースピードの両方を撮影者が自分で設定するモードです。露出を完全にコントロールできるため、意図的に明るくしたり暗くしたりする表現や、スタジオ撮影など一定の露出を保ちたい場面で活用されます。カメラ任せにしないため、露出の知識が求められるモードです。

</QuizAnswer>

## 問10

カメラやレンズを保管する際、湿度管理のために使用が推奨される保管用品として最も適切なものはどれか。

1. ビニール袋に入れて密封する
2. 防湿庫（ドライボックス）に保管する
3. 直射日光の当たる窓際に置く
4. 冷蔵庫に入れて保管する


<QuizAnswer>

**答え: 2**

カメラやレンズは湿気に弱く、高湿度の環境ではレンズにカビが発生する原因になります。防湿庫（ドライボックス）は内部の湿度を適切な範囲（40〜50%程度）に保つことができ、機材の保管に最適です。ビニール袋での密封は結露の原因になり、冷蔵庫は取り出し時の結露リスクが高いため、いずれもカメラ保管には不向きです。

</QuizAnswer>
