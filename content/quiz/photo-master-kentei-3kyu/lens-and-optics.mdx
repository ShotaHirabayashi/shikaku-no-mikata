## 問1

35mmフルサイズ換算で焦点距離50mm付近のレンズは、一般的に何と呼ばれるか。

1. 標準レンズ
2. 広角レンズ
3. 望遠レンズ
4. マクロレンズ


<QuizAnswer>

**答え: 1**

焦点距離50mm付近のレンズは、人間の視野に近い画角（約46度）を持つことから「標準レンズ」と呼ばれます。広角レンズは一般に35mm以下、望遠レンズは80mm以上の焦点距離を指します。標準レンズは自然な遠近感で撮影できるため、スナップや日常撮影に幅広く使われます。

</QuizAnswer>

## 問2

レンズの焦点距離を長くすると、画角はどのように変化するか。

1. 広くなる
2. 狭くなる
3. 変化しない
4. 焦点距離とは無関係に被写体距離で決まる


<QuizAnswer>

**答え: 2**

焦点距離が長くなるほど画角は狭くなり、被写体を大きく写すことができます。逆に焦点距離が短くなるほど画角は広くなり、広い範囲を写すことができます。これはレンズの光学的な性質によるもので、焦点距離と画角は反比例の関係にあります。

</QuizAnswer>

## 問3

広角レンズの特徴として正しいものはどれか。

1. 被写界深度が浅くなりやすい
2. 遠近感が圧縮される
3. 広い範囲を写すことができ、遠近感が強調される
4. 背景が大きくボケやすい


<QuizAnswer>

**答え: 3**

広角レンズは焦点距離が短く画角が広いため、広い範囲を写すことができます。また、近くのものは大きく、遠くのものは小さく写るパースペクティブ（遠近感）が強調されるのが特徴です。遠近感が圧縮されるのは望遠レンズの特徴であり、広角レンズは被写界深度が深くなりやすい傾向があります。

</QuizAnswer>

## 問4

F値（絞り値）の説明として正しいものはどれか。

1. F値が大きいほどレンズは明るい
2. F値はレンズの重量を示す値である
3. F値はレンズの全長を示す値である
4. F値が小さいほどレンズは明るい


<QuizAnswer>

**答え: 4**

F値はレンズの明るさを示す数値で、「焦点距離÷有効口径（レンズの有効な直径）」で求められます。F値が小さいほど多くの光を取り込めるため、レンズは明るくなります。例えばF1.4のレンズはF5.6のレンズよりも明るく、暗い場所での撮影に有利です。

</QuizAnswer>

## 問5

被写界深度を深く（ピントの合う範囲を広く）するための方法として正しいものはどれか。

1. 絞りを開ける（F値を小さくする）
2. 絞りを絞る（F値を大きくする）
3. 焦点距離の長いレンズに変える
4. 被写体にできるだけ近づく


<QuizAnswer>

**答え: 2**

被写界深度を深くするには、絞りを絞って（F値を大きくして）撮影します。F値を大きくすると光の通る範囲が狭まり、ピントの合う範囲が前後に広がります。逆に、焦点距離を長くしたり被写体に近づいたりすると被写界深度は浅くなるため、これらは正しい方法ではありません。

</QuizAnswer>

## 問6

ズームレンズと比較した場合の単焦点レンズの一般的な特徴として、最も適切なものはどれか。

1. 焦点距離を自由に変えられる
2. レンズ構成が複雑で大型になる
3. 開放F値が明るく、描写性能に優れる傾向がある
4. 1本であらゆる撮影シーンに対応できる


<QuizAnswer>

**答え: 3**

単焦点レンズは焦点距離が固定されている代わりに、レンズ構成がシンプルなため、開放F値が明るく（F1.4やF1.8など）、描写性能に優れる傾向があります。焦点距離を変えられるのはズームレンズの特徴です。単焦点レンズは小型軽量なものが多く、大きなボケを活かした撮影にも向いています。

</QuizAnswer>

## 問7

マクロレンズの最大の特徴として正しいものはどれか。

1. 超広角の画角で撮影できる
2. 手ブレ補正機能が必ず搭載されている
3. 被写体に非常に近づいて等倍程度の接写撮影ができる
4. 焦点距離を自由に変えられる


<QuizAnswer>

**答え: 3**

マクロレンズは最短撮影距離が非常に短く、被写体に近づいて大きく写すことに特化したレンズです。一般的に最大撮影倍率が等倍（1:1）またはそれに近い倍率での撮影が可能で、花や昆虫などの小さな被写体を画面いっぱいに撮影できます。なお、マクロレンズは通常の撮影距離でも使用でき、ポートレート撮影などにも活用されます。

</QuizAnswer>

## 問8

レンズの収差のうち、光の波長（色）によって屈折率が異なるために発生する収差を何と呼ぶか。

1. 球面収差
2. 歪曲収差
3. コマ収差
4. 色収差


<QuizAnswer>

**答え: 4**

色収差は、光の波長（色）ごとに屈折率が異なるために、各色の光が同じ位置に結像せず、像の周囲に色のにじみが生じる現象です。球面収差はレンズの中央と周辺で焦点位置がずれる現象、歪曲収差は像が樽型や糸巻き型に歪む現象、コマ収差は画面周辺で点像が彗星状に流れる現象を指します。色収差の補正には、屈折率や分散の異なるレンズを組み合わせる方法が用いられます。

</QuizAnswer>

## 問9

PLフィルター（偏光フィルター）の効果として正しいものはどれか。

1. 光量を均一に減少させ、シャッター速度を遅くできる
2. 紫外線をカットしてレンズを保護する
3. 水面やガラスの反射を除去し、青空や緑をより鮮やかに写せる
4. 画面周辺の光量落ちを補正する


<QuizAnswer>

**答え: 3**

PLフィルター（偏光フィルター）は、偏光を制御することで水面やガラス面などの不要な反射光を除去できるフィルターです。反射を取り除くことで被写体本来の色を引き出し、青空をより濃く、木の葉の緑をより鮮やかに写すことができます。光量を均一に減少させるのはNDフィルター、紫外線カットはUVフィルターの役割です。

</QuizAnswer>

## 問10

レンズの球面収差を効果的に補正するために用いられるレンズの種類はどれか。

1. フレネルレンズ
2. 非球面レンズ
3. 魚眼レンズ
4. リレーレンズ


<QuizAnswer>

**答え: 2**

非球面レンズは、レンズ表面が球面ではなく複雑なカーブを描いた形状をしており、球面収差を効果的に補正できます。通常の球面レンズではレンズの中央部と周辺部で光の集まる位置がずれますが、非球面レンズを使うことで周辺部の光も中央付近とほぼ同じ焦点に合わせることが可能になります。現代の高性能レンズには非球面レンズが広く採用されています。

</QuizAnswer>
