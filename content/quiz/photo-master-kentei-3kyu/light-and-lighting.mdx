## 問1

被写体の正面から光が当たる「順光」の特徴として、最も適切なものはどれか。

1. 被写体の輪郭が光で縁取られ、幻想的な雰囲気になる
2. 被写体の色が鮮やかに再現されるが、立体感が出にくい
3. 被写体の片側に強い影ができ、立体感が強調される
4. 被写体がシルエットになりやすい


<QuizAnswer>

**答え: 2**

順光は撮影者の背後から被写体の正面に光が当たる状態で、色の再現性が高く鮮やかな写真が撮れます。一方、影が被写体の背後に隠れるため、立体感や奥行きが出にくく平面的な印象になりやすいのが特徴です。

</QuizAnswer>

## 問2

写真撮影における「光の三要素」として正しい組み合わせはどれか。

1. 色温度・ホワイトバランス・露出
2. 明るさ・色・コントラスト
3. 強さ・方向・質
4. 順光・逆光・サイド光


<QuizAnswer>

**答え: 3**

光の三要素とは「強さ（光量）」「方向（光がどこから当たるか）」「質（硬い光か柔らかい光か）」の3つを指します。これらを理解しコントロールすることが、ライティングの基本となります。選択肢4は光の「方向」の種類であり、三要素そのものではありません。

</QuizAnswer>

## 問3

色温度の単位として正しいものはどれか。

1. ルクス（lx）
2. ルーメン（lm）
3. カンデラ（cd）
4. ケルビン（K）


<QuizAnswer>

**答え: 4**

色温度の単位はケルビン（K）です。数値が低いほど赤みを帯びた暖色系の光になり、数値が高いほど青みを帯びた寒色系の光になります。ルクスは照度、ルーメンは光束、カンデラは光度の単位であり、色温度とは異なります。

</QuizAnswer>

## 問4

曇天時の屋外で撮影すると写真が青みがかることがある。これを補正するためにホワイトバランスを「曇天」に設定すると、カメラはどのような補正を行うか。

1. 赤みを加えて補正する
2. 青みを加えて補正する
3. 緑みを加えて補正する
4. 色の彩度を下げて補正する


<QuizAnswer>

**答え: 1**

曇天時の光は色温度が約6000〜6500Kと高く、青みがかった光です。ホワイトバランスの「曇天」設定では、この青みを打ち消すために赤み（暖色）を加える補正を行い、見た目に近い自然な白を再現します。ホワイトバランスは光源の色かぶりと反対の色を加えることで補正する仕組みです。

</QuizAnswer>

## 問5

ストロボ（フラッシュ）の光量を表す指標として使われる「ガイドナンバー（GN）」の計算式として正しいものはどれか。（ISO100基準）

1. ガイドナンバー ＝ 撮影距離（m）＋ 絞り値（F値）
2. ガイドナンバー ＝ 撮影距離（m）－ 絞り値（F値）
3. ガイドナンバー ＝ 撮影距離（m）× 絞り値（F値）
4. ガイドナンバー ＝ 撮影距離（m）÷ 絞り値（F値）


<QuizAnswer>

**答え: 3**

ガイドナンバー（GN）は「撮影距離（m）× 絞り値（F値）」で求められます（ISO100・フル発光時）。例えばGN32のストロボでF4に設定した場合、光が届く距離は32÷4＝8mとなります。ガイドナンバーの数値が大きいほど、より遠くの被写体を照らすことができます。

</QuizAnswer>

## 問6

ストロボの光を天井や壁に向けて発光し、反射した間接光で被写体を照らす撮影技法を何というか。

1. スヌート撮影
2. バウンス撮影
3. スポット撮影
4. リフレクト撮影


<QuizAnswer>

**答え: 2**

バウンス撮影とは、ストロボの発光部を天井や壁に向けて発光し、反射（バウンス）させた間接光で被写体を照らす手法です。直接光に比べて光が拡散されるため、柔らかく自然な印象のライティングになり、硬い影も軽減されます。ポートレートや室内撮影で多用される基本テクニックです。

</QuizAnswer>

## 問7

次の光源を色温度が低い順に正しく並べたものはどれか。

1. ろうそく → 白熱電球 → 晴天の太陽光 → 曇天の空
2. 曇天の空 → 晴天の太陽光 → 白熱電球 → ろうそく
3. 白熱電球 → ろうそく → 曇天の空 → 晴天の太陽光
4. 晴天の太陽光 → 曇天の空 → ろうそく → 白熱電球


<QuizAnswer>

**答え: 1**

色温度はろうそくが約1500〜2000K、白熱電球が約2800〜3000K、晴天の太陽光が約5500K、曇天の空が約6500Kです。色温度が低いほど赤みを帯び、高いほど青みを帯びます。「赤い光ほど色温度が低い」という関係を覚えておくことがライティングの基本です。

</QuizAnswer>

## 問8

被写体の横から光が当たる「サイド光」の特徴として、最も適切なものはどれか。

1. 被写体の色が最も鮮やかに再現される
2. 被写体全体が均一に明るくなる
3. 被写体に明暗のコントラストが生まれ、立体感が強調される
4. 被写体の背後に光の輪（リムライト）ができる


<QuizAnswer>

**答え: 3**

サイド光は被写体の横から光が当たるため、光が当たる面と影になる面がはっきりと分かれ、明暗のコントラストが生まれます。これにより被写体の立体感や質感、力強さが強調されます。ポートレートや建築物の撮影で陰影を活かした表現に適しています。

</QuizAnswer>

## 問9

ストロボやライトの前に半透明の素材を置いて光を拡散させ、柔らかい光にする技法（またはその道具）を何というか。

1. レフ板
2. スヌート
3. ディフューズ（ディフューザー）
4. ハニカムグリッド


<QuizAnswer>

**答え: 3**

ディフューズとは光を拡散させて柔らかくする技法で、そのために使う半透明の道具をディフューザーと呼びます。ストロボに取り付けるタイプや、ライトの前に設置するソフトボックスなどがあります。レフ板は光を反射させる道具、スヌートやハニカムグリッドは光を狭い範囲に集中させるためのアクセサリーです。

</QuizAnswer>

## 問10

逆光で人物を撮影する際に起こりやすい問題と、その対処法の組み合わせとして最も適切なものはどれか。

1. 背景が暗くなる → シャッター速度を速くする
2. 人物の顔が暗くなる → レフ板やストロボで顔に光を補う
3. 写真全体が赤くなる → ホワイトバランスを電球モードにする
4. ピントが合いにくくなる → 望遠レンズに交換する


<QuizAnswer>

**答え: 2**

逆光では被写体の背後から光が当たるため、人物の顔（正面）が暗くなりやすいのが最大の問題点です。対処法としては、レフ板で光を反射させて顔に当てる、日中シンクロ（ストロボを焚いて補助光とする）などが有効です。逆光は髪の毛が輝くなど美しい表現にも使えるため、補助光を併用することで背景の美しさと顔の明るさを両立できます。

</QuizAnswer>
