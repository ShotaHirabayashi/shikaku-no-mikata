## 問1

デジタルカメラで撮影した画像の「画素数」について、正しい説明はどれか。

1. 画素数が多いほど、必ず画質が良くなる
2. 画素数とは、イメージセンサー上のフォトダイオードの総数のことである
3. 画素数は、画像の色の深さ（ビット数）を表す指標である
4. 画素数が多いほど、ファイルサイズは小さくなる


<QuizAnswer>

**答え: 2**

画素数とは、イメージセンサー上に配置されたフォトダイオード（受光素子）の総数を指します。例えば横6000×縦4000の画素が並んでいれば約2400万画素となります。画素数が多いほどファイルサイズは大きくなり、また画素数だけでなくセンサーサイズやレンズ性能も画質に影響するため、画素数が多い＝高画質とは限りません。

</QuizAnswer>

## 問2

JPEG形式の画像圧縮方式について、正しい説明はどれか。

1. 可逆圧縮方式であり、圧縮前のデータに完全に復元できる
2. 非可逆圧縮方式であり、圧縮によって一部のデータが失われる
3. 圧縮を行わないため、ファイルサイズが非常に大きい
4. テキストデータの圧縮にも広く使われている


<QuizAnswer>

**答え: 2**

JPEG形式は非可逆（不可逆）圧縮方式を採用しており、人間の目には知覚しにくい情報を間引くことでファイルサイズを小さくしています。そのため、一度圧縮したデータを完全に元の状態に戻すことはできません。保存を繰り返すたびに画質が劣化していくため、編集を繰り返す場合は注意が必要です。

</QuizAnswer>

## 問3

RAW形式で撮影するメリットとして、最も適切なものはどれか。

1. JPEGより常にファイルサイズが小さくなる
2. 撮影後にホワイトバランスや露出補正を高画質のまま調整できる
3. どのソフトウェアでもそのまま開いて閲覧できる
4. カメラ内で画像処理が完了しているため、後処理が不要である


<QuizAnswer>

**答え: 2**

RAW形式はセンサーが取得したデータをほぼそのまま記録するため、撮影後にホワイトバランス、露出、コントラストなどを画質の劣化を最小限に抑えながら調整できます。一方、ファイルサイズはJPEGより大きく、閲覧には専用のRAW現像ソフトが必要です。RAWは「デジタルネガ」とも呼ばれ、後処理の自由度が高い記録形式です。

</QuizAnswer>

## 問4

デジタルカメラの色空間設定で「sRGB」と「AdobeRGB」を比較したとき、正しい説明はどれか。

1. sRGBのほうがAdobeRGBより色域が広い
2. AdobeRGBは主にWeb表示用に標準化された色空間である
3. AdobeRGBはsRGBより色域が広く、特に緑から青にかけての高彩度領域が広い
4. RAW形式で記録する場合、カメラの色空間設定がそのまま記録データに反映される


<QuizAnswer>

**答え: 3**

AdobeRGBはAdobe社が策定した色空間で、sRGBよりも広い色域を持ち、特に緑から青にかけての高彩度領域で差が大きくなっています。sRGBはIEC（国際電気標準会議）が策定した標準規格で、Web表示やWindows環境の基準として広く普及しています。なお、RAW形式で記録する場合は色空間設定に関係なく、より広い色域で記録されます。

</QuizAnswer>

## 問5

デジタル画像における「解像度」について、正しい説明はどれか。

1. 解像度はdpi（dots per inch）やppi（pixels per inch）などの単位で表される
2. 解像度を上げると、必ずファイルの画素数が増える
3. 解像度は画像の色数を表す数値である
4. Web用画像の標準解像度は350dpiである


<QuizAnswer>

**答え: 1**

解像度とは単位長さあたりに含まれるドット数やピクセル数を表す指標で、dpi（dots per inch）やppi（pixels per inch）といった単位が使われます。同じ画素数の画像でも、出力サイズを変えれば解像度は変化します。一般的にWeb用画像は72ppi程度、印刷用画像は300〜350dpi程度が目安とされています。

</QuizAnswer>

## 問6

フォトレタッチにおける「トーンカーブ」の説明として、正しいものはどれか。

1. 画像の色相だけを変更するためのツールである
2. 横軸に入力（元の明るさ）、縦軸に出力（調整後の明るさ）をとり、カーブを操作して明暗やコントラストを調整する
3. 画像のピクセル数を増減させるためのツールである
4. 画像のファイル形式を変換するためのツールである


<QuizAnswer>

**答え: 2**

トーンカーブは、横軸に入力値（調整前の各ピクセルの明るさ）、縦軸に出力値（調整後の明るさ）をとったグラフ上でカーブを操作し、画像の明るさやコントラストを調整するツールです。カーブを上に持ち上げると画像が明るくなり、下に引き下げると暗くなります。S字カーブにするとコントラストが強まるなど、直感的な調整が可能です。

</QuizAnswer>

## 問7

デジタルカメラで使用される記録メディアのうち、SDカードの上位規格で大容量に対応しているものはどれか。

1. CFカード（CompactFlash）
2. SDXCカード
3. メモリースティック
4. xDピクチャーカード


<QuizAnswer>

**答え: 2**

SDカードには容量規格によってSD（最大2GB）、SDHC（最大32GB）、SDXC（最大2TB）の3種類があり、SDXCカードはSDカードの上位規格として大容量に対応しています。CFカードはSDカードとは別系統の規格であり、メモリースティックはソニー独自規格、xDピクチャーカードはオリンパスと富士フイルムの規格で、いずれもSDカードの上位規格ではありません。

</QuizAnswer>

## 問8

モニターキャリブレーションの目的として、最も適切なものはどれか。

1. モニターの消費電力を削減すること
2. モニターの表示解像度を上げること
3. モニターの色や明るさを正確な基準に合わせ、画像の色を正しく表示すること
4. モニターの表示速度を高速化すること


<QuizAnswer>

**答え: 3**

モニターキャリブレーションとは、モニターの色温度や明るさ、ガンマ値などを測定・調整し、正確な色再現ができるようにする作業です。モニターは使用を続けると白色の色温度や明るさが変化するため、定期的なキャリブレーション（目安として200時間ごと）が推奨されます。フォトレタッチやプリント出力において、画面表示と実際の仕上がりの色を一致させるために重要な工程です。

</QuizAnswer>

## 問9

インクジェットプリンターで写真をプリントする際の説明として、誤っているものはどれか。

1. 染料インクは光沢感のある仕上がりが得られやすい
2. 顔料インクは耐水性・耐光性に優れている
3. 一般的にCMYK（シアン・マゼンタ・イエロー・ブラック）の4色を基本としてインクを使用する
4. プリンターの解像度が低いほど、より精細な写真プリントが得られる


<QuizAnswer>

**答え: 4**

プリンターの解像度が高いほど、より精細で高画質なプリントが得られます。染料インクは用紙に浸透して発色するため光沢感のある仕上がりが得られ、顔料インクは用紙表面に定着するため耐水性・耐光性に優れています。インクジェットプリンターではCMYKの4色を基本に、機種によってはライトシアンやライトマゼンタなどを加えた多色インクを使用して、より滑らかな階調表現を実現しています。

</QuizAnswer>

## 問10

画像ファイル形式の「TIFF」について、正しい説明はどれか。

1. Web表示専用に開発されたファイル形式である
2. 非可逆圧縮のみに対応しており、保存のたびに画質が劣化する
3. 可逆圧縮または無圧縮で保存でき、画質の劣化がない
4. 動画の記録にも対応しているファイル形式である


<QuizAnswer>

**答え: 3**

TIFF（Tagged Image File Format）は、可逆圧縮（LZW圧縮など）または無圧縮で画像を保存できるファイル形式で、保存を繰り返しても画質が劣化しません。そのため、印刷用途やアーカイブなど高画質が求められる場面で広く使用されています。ファイルサイズはJPEGと比べて大きくなりますが、画質を優先する場合に適した形式です。

</QuizAnswer>
