## 問1

腸内に生息する多種多様な細菌の集合体を指す用語として、最も適切なものはどれか。

1. プロバイオティクス
2. プレバイオティクス
3. 腸内フローラ（腸内細菌叢）
4. シンバイオティクス


<QuizAnswer>

**答え: 3**

腸内フローラ（腸内細菌叢）とは、腸内に生息する数百種類、数十兆個にも及ぶ細菌の集合体のことです。これらの細菌がお花畑（フローラ）のように群生している様子からこの名前が付けられました。善玉菌・悪玉菌・日和見菌のバランスが健康状態に大きく影響します。

</QuizAnswer>

## 問2

プロバイオティクスの説明として、最も適切なものはどれか。

1. 善玉菌のエサとなるオリゴ糖や食物繊維のこと
2. 人体に有益な効果をもたらす生きた微生物のこと
3. 発酵食品に含まれる酵素の総称
4. 腸内で産生される短鎖脂肪酸のこと


<QuizAnswer>

**答え: 2**

プロバイオティクスとは、適切な量を摂取したときに宿主に有益な効果をもたらす生きた微生物のことです。代表的なものに乳酸菌やビフィズス菌があり、ヨーグルトや納豆などの発酵食品に含まれています。選択肢1の「善玉菌のエサとなる成分」はプレバイオティクスの説明です。

</QuizAnswer>

## 問3

プレバイオティクスに該当するものとして、最も適切なものはどれか。

1. ビフィズス菌
2. 乳酸菌
3. オリゴ糖
4. 納豆菌


<QuizAnswer>

**答え: 3**

プレバイオティクスとは、消化管上部で分解・吸収されずに大腸まで届き、善玉菌のエサとなって腸内環境を改善する食品成分のことです。オリゴ糖や食物繊維がその代表例です。ビフィズス菌・乳酸菌・納豆菌はいずれも微生物そのものであり、プロバイオティクスに分類されます。

</QuizAnswer>

## 問4

ヒトの免疫細胞の約70％が集中しているとされる臓器はどれか。

1. 肝臓
2. 腸
3. 脾臓
4. 骨髄


<QuizAnswer>

**答え: 2**

ヒトの免疫細胞の約70％は腸に集中しているとされています。そのため、発酵食品を摂取して腸内環境を整えることは、免疫力の向上に直接つながると考えられています。腸は「最大の免疫器官」とも呼ばれ、腸内フローラのバランスが免疫機能に大きく関わっています。

</QuizAnswer>

## 問5

納豆に含まれるビタミンKの量は、ゆでた大豆と比べておよそ何倍に増加するか。

1. 約5倍
2. 約20倍
3. 約50倍
4. 約120倍


<QuizAnswer>

**答え: 4**

納豆に含まれるビタミンKは、ゆでた大豆の約120倍にもなるとされています。これは発酵の過程で納豆菌がビタミンKを大量に産生するためです。このように発酵によって元の食材にはなかった栄養成分が新たに生み出されたり、大幅に増加したりすることが発酵食品の大きな特長です。

</QuizAnswer>

## 問6

プロバイオティクスとプレバイオティクスを組み合わせて摂取する考え方を何というか。

1. ポストバイオティクス
2. シンバイオティクス
3. バイオジェニックス
4. メタバイオティクス


<QuizAnswer>

**答え: 2**

シンバイオティクスとは、プロバイオティクス（有益な生きた微生物）とプレバイオティクス（善玉菌のエサとなる食品成分）を組み合わせて同時に摂取する方法のことです。善玉菌とそのエサを一緒に摂ることで、より効果的に腸内環境を改善できると注目されています。例えばヨーグルトにオリゴ糖を加えて食べるのが一例です。

</QuizAnswer>

## 問7

発酵食品が美容・美肌効果をもたらす理由として、最も適切なものはどれか。

1. 発酵により食品のカロリーが大幅に減少するため
2. 発酵過程で生成されるアミノ酸や酵素が新しい細胞の生成を促すため
3. 発酵食品には紫外線を直接ブロックする成分が含まれるため
4. 発酵により食品中の水分量が増加し肌を潤すため


<QuizAnswer>

**答え: 2**

発酵の過程でつくられるアミノ酸や酵素などが新しい細胞の生成を促すことで、美肌効果が期待できるとされています。また、発酵食品によって腸内環境が改善されると便秘が解消され、老廃物の排出がスムーズになることで肌荒れの改善にもつながります。

</QuizAnswer>

## 問8

発酵食品に含まれる成分と生活習慣病予防の関係について、正しいものはどれか。

1. 乳酸菌や大豆イソフラボンには悪玉コレステロールを減少させる働きがある
2. 発酵食品は血圧を上昇させる作用がある
3. 発酵食品を摂ると血糖値が急激に上がりやすくなる
4. 発酵食品に含まれるビタミンB群は血液循環を悪化させる


<QuizAnswer>

**答え: 1**

発酵食品に含まれる乳酸菌や大豆イソフラボンには、悪玉コレステロール（LDLコレステロール）を減少させ、動脈硬化を予防する働きがあるとされています。また、ビタミンB群は代謝を促進して血液循環を改善する効果があり、発酵食品は生活習慣病の予防に役立つと考えられています。

</QuizAnswer>

## 問9

発酵により納豆のビタミンB2は煮大豆と比較しておよそ何倍に増加するか。

1. 約2倍
2. 約10倍
3. 約50倍
4. 約100倍


<QuizAnswer>

**答え: 2**

納豆に含まれるビタミンB2は、煮大豆の約10倍に増加するとされています。ビタミンB2は皮膚や粘膜の健康維持に関わる栄養素で、エネルギー代謝にも重要な役割を果たします。このように、発酵によって元の食材の栄養価が飛躍的に高まることが発酵食品の大きな利点です。

</QuizAnswer>

## 問10

発酵食品の安全性と注意点について、正しいものはどれか。

1. 発酵食品は加工食品なので食物アレルギーの心配はない
2. 発酵食品は腐らないのでいつまでも食べられる
3. 発酵食品に含まれる生きた菌は加熱すると死滅するが、死菌にも腸内環境を整える効果が期待できる
4. 発酵食品は多く摂るほど健康効果が高まるため、摂取量に上限はない


<QuizAnswer>

**答え: 3**

発酵食品に含まれる乳酸菌などの生きた菌は加熱によって死滅しますが、死菌（死んだ菌体）であっても善玉菌のエサとなったり、免疫細胞を刺激したりすることで腸内環境を整える効果が期待できます。なお、発酵食品であっても原材料によるアレルギーの可能性はあり、塩分を多く含むものもあるため、適量を心がけることが大切です。

</QuizAnswer>
