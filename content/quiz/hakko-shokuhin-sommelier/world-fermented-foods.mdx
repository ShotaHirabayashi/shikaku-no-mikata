## 問1

ナチュラルチーズの分類において、熟成させずにそのまま食べるタイプのチーズはどれか。

1. 白カビタイプ
2. ウォッシュタイプ
3. フレッシュタイプ
4. セミハードタイプ


<QuizAnswer>

**答え: 3**

フレッシュタイプは、乳酸菌や酵素でミルクを凝固させた後、ホエイ（乳清）を分離させて熟成させずにすぐ食べられるチーズです。モッツァレラ、クリームチーズ、リコッタ、カッテージチーズなどが代表例で、さっぱりとした味わいが特徴です。

</QuizAnswer>

## 問2

世界三大ブルーチーズに含まれないものはどれか。

1. ロックフォール
2. カマンベール
3. ゴルゴンゾーラ
4. スティルトン


<QuizAnswer>

**答え: 2**

世界三大ブルーチーズは、フランスのロックフォール、イタリアのゴルゴンゾーラ、イギリスのスティルトンです。カマンベールはフランスを代表するチーズですが、白カビタイプに分類されます。ブルーチーズは内部に青カビを繁殖させて熟成させる、ピリッとした風味が特徴のチーズです。

</QuizAnswer>

## 問3

ドイツの伝統的な発酵食品「ザワークラウト」の主な原料はどれか。

1. キャベツ
2. 大根
3. きゅうり
4. 白菜


<QuizAnswer>

**答え: 1**

ザワークラウトは、千切りにしたキャベツを塩漬けにして乳酸菌発酵させたドイツの伝統的な発酵食品です。さっぱりとした酸味が特徴で、ソーセージなどの肉料理の付け合わせとして広く親しまれています。なお、白菜を使う代表的な発酵食品は韓国のキムチです。

</QuizAnswer>

## 問4

ワインの醸造における発酵方式として正しいものはどれか。

1. 並行複発酵
2. 単行複発酵
3. 二段階発酵
4. 単発酵


<QuizAnswer>

**答え: 4**

ワインはブドウの果汁に含まれる糖分を酵母が直接アルコールに変える「単発酵」で造られます。一方、ビールは麦芽のデンプンを糖化してから発酵させる「単行複発酵」、日本酒は糖化と発酵が同時に進む「並行複発酵」という方式をとります。

</QuizAnswer>

## 問5

インドネシアの伝統的な発酵食品「テンペ」について正しい説明はどれか。

1. 米にコウジカビを繁殖させて作る
2. 大豆にクモノスカビの一種を繁殖させて作る
3. 小麦を乳酸菌で発酵させて作る
4. そら豆を麹と塩水で発酵させて作る


<QuizAnswer>

**答え: 2**

テンペは、茹でた大豆にテンペ菌（クモノスカビの一種、リゾプス属）を繁殖させて発酵させたインドネシアの伝統食品です。納豆と同じく大豆の発酵食品ですが、粘り気や強い臭いがなくクセが少ないのが特徴で、「インドネシアの納豆」とも呼ばれます。

</QuizAnswer>

## 問6

タイの魚醤「ナンプラー」とベトナムの魚醤「ニョクマム」に共通して使われる主な原料はどれか。

1. イカ
2. エビ
3. カタクチイワシ
4. サケ


<QuizAnswer>

**答え: 3**

ナンプラー（タイ）とニョクマム（ベトナム）は、いずれもカタクチイワシを塩漬けにして長期間発酵させた魚醤です。材料や製法に大きな違いはありませんが、一般的にニョクマムの方が発酵度合いが低く、魚の香りがより強い傾向があります。魚醤は東南アジアを中心に各国で作られている発酵調味料です。

</QuizAnswer>

## 問7

パンが膨らむ仕組みとして正しい説明はどれか。

1. 酵母が糖を分解して炭酸ガスを発生させ、生地を膨張させる
2. 乳酸菌が乳酸を生成して生地内に気泡を作る
3. 小麦粉のグルテンが水と反応して自然に膨張する
4. 麹菌がデンプンを分解してガスを発生させる


<QuizAnswer>

**答え: 1**

パンの発酵では、酵母（イースト）が小麦粉の糖分を分解してアルコールと炭酸ガス（二酸化炭素）を生成します。この炭酸ガスがグルテンの網目構造に包まれて生地を膨らませます。また、発酵の過程で生じる香気成分がパン独特の風味を生み出しています。

</QuizAnswer>

## 問8

中国の発酵調味料「豆板醤」の主な原料となる豆はどれか。

1. 大豆
2. 小豆
3. ひよこ豆
4. そら豆


<QuizAnswer>

**答え: 4**

豆板醤は、そら豆に麹と塩水を加えて発酵させた中国・四川省発祥の発酵調味料です。唐辛子を加えた辛味のある味が特徴で、麻婆豆腐やエビチリなど中華料理に欠かせない調味料として広く使われています。大豆を原料とする中国の発酵調味料には甜麺醤などがあります。

</QuizAnswer>

## 問9

プロセスチーズの説明として正しいものはどれか。

1. 生乳を乳酸菌で発酵させただけの未熟成チーズである
2. ナチュラルチーズを加熱・溶融して成型したチーズである
3. 青カビを内部に繁殖させて熟成させたチーズである
4. 山羊乳を原料として作られたチーズの総称である


<QuizAnswer>

**答え: 2**

プロセスチーズは、1種類または数種類のナチュラルチーズを粉砕し、乳化剤を加えて75〜120℃で加熱・溶融した後、型に流し込んで冷やし固めたチーズです。加熱処理により発酵が止まるため、ナチュラルチーズに比べて保存性が高く、品質が安定しているのが特徴です。

</QuizAnswer>

## 問10

ブルガリアを発祥とし、世界中で食べられている代表的な発酵乳製品はどれか。

1. サワークリーム
2. ケフィア
3. ヨーグルト
4. パニール


<QuizAnswer>

**答え: 3**

ヨーグルトはブルガリアを発祥地とする代表的な発酵乳製品で、乳に乳酸菌を加えて発酵させて作られます。ブルガリアでは伝統的に常食されており、ノーベル賞学者メチニコフがブルガリア人の長寿とヨーグルトの関係を提唱したことで世界的に広まりました。ケフィアはコーカサス地方発祥、パニールはインドのフレッシュチーズです。

</QuizAnswer>
