## 問1

味噌は使用する麹の原料によって大きく分類されます。大豆に米麹を加えて作る味噌は何と呼ばれますか。

1. 麦味噌
2. 豆味噌
3. 米味噌
4. 調合味噌


<QuizAnswer>

**答え: 3**

米味噌は、大豆に米麹を加えて作る味噌で、日本全国で最も多く生産されています。麦味噌は麦麹、豆味噌は大豆のみを主原料とし、調合味噌はこれらを混合したものです。

</QuizAnswer>

## 問2

JAS規格で定められている醤油の種類として、正しくないものはどれですか。

1. 濃口醤油
2. 黒醤油
3. 再仕込醤油
4. 白醤油


<QuizAnswer>

**答え: 2**

JAS規格では醤油は「濃口醤油」「淡口醤油」「溜醤油」「再仕込醤油」「白醤油」の5種類に分類されます。「黒醤油」というJAS規格上の分類は存在しません。濃口醤油は日本の醤油生産量の8割以上を占めています。

</QuizAnswer>

## 問3

日本酒の醸造における「並行複発酵」の説明として正しいものはどれですか。

1. 麹によるデンプンの糖化と酵母によるアルコール発酵が同時に進行する
2. 乳酸発酵とアルコール発酵を別々のタンクで順番に行う
3. 酢酸発酵とアルコール発酵を同時に行う
4. 糖化を完了させてからアルコール発酵を開始する


<QuizAnswer>

**答え: 1**

並行複発酵とは、麹菌の酵素が米のデンプンを糖に分解する「糖化」と、酵母がその糖をアルコールに変える「アルコール発酵」が同じタンク内で同時に進行する醸造方法です。この技術は世界でも類を見ない高度な醸造法であり、醸造酒として高いアルコール度数を実現できる理由の一つです。

</QuizAnswer>

## 問4

豆味噌の主な産地として知られる地方はどこですか。

1. 東北地方
2. 九州地方
3. 関東地方
4. 中京地方（愛知・三重・岐阜）


<QuizAnswer>

**答え: 4**

豆味噌は大豆のみを主原料として作られる味噌で、中京地方（愛知県・三重県・岐阜県）が主な産地です。代表的なものに八丁味噌があります。一方、麦味噌は九州・四国・中国地方が主な産地として知られています。

</QuizAnswer>

## 問5

鰹節の製造工程で「カビ付け」を行う主な目的として最も適切なものはどれですか。

1. 鰹節に赤い色をつけるため
2. 節の表面の脂肪分を分解し、澄んだだしを取れるようにするため
3. 鰹節を柔らかくして削りやすくするため
4. 鰹節に甘味を加えるため


<QuizAnswer>

**答え: 2**

鰹節のカビ付けは、カビが節の表面の脂肪分を分解する働きを利用した工程です。これにより、動物性食品でありながら脂のない澄んだ透明な「だし」が取れるようになります。カビ付けは複数回繰り返され、最初に付いたカビを「1番カビ」と呼びます。

</QuizAnswer>

## 問6

本みりんの原料の組み合わせとして正しいものはどれですか。

1. もち米・米麹・本格焼酎
2. うるち米・麦麹・日本酒
3. もち米・麦麹・ワイン
4. うるち米・米麹・本格焼酎


<QuizAnswer>

**答え: 1**

本みりんは「もち米」「米麹」「本格焼酎（またはアルコール）」の3つを主原料として作られます。蒸したもち米に米麹と本格焼酎を混ぜ合わせて仕込み、糖化・熟成させた後に圧搾して作ります。アルコール度数は約13〜14%です。

</QuizAnswer>

## 問7

伝統的な納豆の製法として正しい説明はどれですか。

1. 生の大豆を塩水に漬けて常温で発酵させる
2. 煮た大豆に酵母菌を加えて発酵させる
3. 蒸した大豆に麹菌をまぶして発酵させる
4. 蒸した大豆を稲わらで包み、約40℃の環境で発酵させる


<QuizAnswer>

**答え: 4**

伝統的な納豆の製法は、蒸した大豆を煮沸消毒した稲わらで作った「わらづと」に入れ、約40℃前後の環境で発酵させるものです。稲わらに自然に付着している納豆菌が大豆に移行・増殖することで発酵が起こります。現在の工場生産では、純粋培養した納豆菌を直接噴霧する方法が主流です。

</QuizAnswer>

## 問8

再仕込醤油（さいしこみしょうゆ）の製法の特徴として正しいものはどれですか。

1. 通常の2倍の量の大豆を使用する
2. 発酵期間を通常の半分に短縮する
3. 食塩水の代わりに醤油を使って熟成させる
4. 小麦を一切使用せずに作る


<QuizAnswer>

**答え: 3**

再仕込醤油は、通常の醤油造りで食塩水を使うところを、すでに出来上がった醤油を使って仕込むことからこの名前がつきました。醤油で醤油を仕込むため、濃厚な味わいと深い色合いが特徴です。「甘露醤油」とも呼ばれ、刺身や寿司などに向いています。

</QuizAnswer>

## 問9

溜醤油（たまりしょうゆ）の原料の特徴として正しいものはどれですか。

1. 小麦が主原料で、大豆はほとんど使用しない
2. 大豆と小麦を同量使用する
3. 米麹を主原料とする
4. ほぼ大豆が主原料で、小麦はごく少量のみ使用する


<QuizAnswer>

**答え: 4**

溜醤油は、ほぼ大豆のみを主原料とし、小麦はごく少量しか使用しない醤油です。濃厚なうま味ととろみのある味わいが特徴で、主に中部地方で生産されています。なお、白醤油はこの逆で、小麦が主原料（小麦9：大豆1程度）となります。

</QuizAnswer>

## 問10

秋田県の伝統的な魚醤「しょっつる」の主な原料として使われる魚はどれですか。

1. サバ
2. ハタハタ
3. カツオ
4. サンマ


<QuizAnswer>

**答え: 2**

しょっつる（塩汁）は秋田県の伝統的な魚醤で、主にハタハタを原料として塩漬け・発酵させて作られます。日本三大魚醤の一つとされ、他に石川県能登地方の「いしる（魚汁）」、香川県の「いかなご醤油」があります。しょっつる鍋は秋田の郷土料理として有名です。

</QuizAnswer>
