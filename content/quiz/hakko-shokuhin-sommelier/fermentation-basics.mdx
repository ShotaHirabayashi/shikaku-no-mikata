## 問1

発酵の生化学的な定義として、最も適切なものはどれか。

1. 微生物が酸素を使って有機物を完全に分解すること
2. 食品を長期間保存するために塩漬けにすること
3. 微生物の働きにより有機物が分解され、人間にとって有益な物質が生成されること
4. 食品に含まれる水分が蒸発して成分が濃縮されること


<QuizAnswer>

**答え: 3**

発酵とは、微生物（細菌・酵母・カビなど）の働きによって有機物が分解され、人間にとって有益な物質が生み出される現象を指します。生化学的には「酸素のない状態で微生物により糖質が分解されてエネルギーが生み出されること」とも定義されますが、広義には微生物による有用な変化全般を「発酵」と呼びます。

</QuizAnswer>

## 問2

発酵と腐敗の違いについて、正しい説明はどれか。

1. 発酵は微生物の働きによるが、腐敗は化学反応のみで起こる
2. どちらも微生物の働きによる変化だが、人間にとって有益かどうかで区別される
3. 発酵は高温で起こり、腐敗は低温で起こる
4. 発酵は動物性食品にのみ起こり、腐敗は植物性食品にのみ起こる


<QuizAnswer>

**答え: 2**

発酵も腐敗も、微生物が有機物を分解するという点では同じメカニズムです。その結果が人間にとって有益で好ましいものであれば「発酵」、有害で不快なものであれば「腐敗」と呼ばれます。つまり、科学的なプロセスの違いではなく、人間にとっての有用性で区別されています。

</QuizAnswer>

## 問3

発酵が進むことで食品に起こる変化として、正しいものはどれか。

1. たんぱく質がそのまま蓄積され、硬くなる
2. 食品中の水分がすべて失われる
3. 糖分やビタミンが完全に消失する
4. でんぷんが糖分に、たんぱく質がアミノ酸に分解され、甘みや旨味が増す


<QuizAnswer>

**答え: 4**

発酵の過程では、微生物が産出する酵素の働きにより、でんぷんは糖分に、たんぱく質はアミノ酸に分解されます。これにより甘み成分や旨味成分が増加し、食品の味わいが豊かになります。また、発酵によって新たな香り成分が生まれたり、食感が変化したりすることも大きな特徴です。

</QuizAnswer>

## 問4

アルコール発酵の化学反応について、正しい説明はどれか。

1. グルコースがエタノールと二酸化炭素に分解される
2. グルコースが酢酸と水に分解される
3. グルコースが乳酸と二酸化炭素に分解される
4. グルコースが水とアンモニアに分解される


<QuizAnswer>

**答え: 1**

アルコール発酵では、1分子のグルコース（ブドウ糖）が、2分子のエタノール（アルコール）と2分子の二酸化炭素に分解されます。この反応は主に酵母の働きによって起こり、酒類の醸造やパンの膨らみの原理となっています。二酸化炭素が発生するのがアルコール発酵の特徴の一つです。

</QuizAnswer>

## 問5

乳酸発酵の特徴として、正しいものはどれか。

1. 反応に必ず酸素が必要である
2. 最終生成物としてアルコールが生じる
3. グルコースから乳酸が生成される
4. 高温（80℃以上）でのみ反応が進む


<QuizAnswer>

**答え: 3**

乳酸発酵では、グルコース（ブドウ糖）が分解されて乳酸が生成されます。ホモ乳酸発酵では1分子のグルコースから2分子の乳酸が生じます。乳酸発酵は酸素を必要としない嫌気的な反応であり、この過程で生じる乳酸の酸味が、発酵食品の風味に大きく寄与しています。

</QuizAnswer>

## 問6

酢酸発酵が、アルコール発酵や乳酸発酵と異なる大きな特徴はどれか。

1. 微生物を必要としない純粋な化学反応である
2. 酸素を必要とする好気的な反応である
3. 反応温度が0℃以下でなければ進まない
4. 糖を直接分解して酢酸を生成する


<QuizAnswer>

**答え: 2**

酢酸発酵は、アルコール発酵や乳酸発酵と異なり、酸素を用いる好気的な反応です。酢酸菌がエタノール（アルコール）を酸化して酢酸を生成します。このため酢酸発酵は厳密には「酸化反応」であり、酸素がない環境では進まないという点が他の発酵と大きく異なります。

</QuizAnswer>

## 問7

世界で最も古い発酵食品の一つとされているものはどれか。

1. チョコレート
2. こんにゃく
3. 豆腐
4. ヨーグルト（発酵乳）


<QuizAnswer>

**答え: 4**

世界最古の発酵食品は、牛乳から偶然にできたヨーグルト（発酵乳）とされており、紀元前5000年頃に中央アジアで誕生したと考えられています。中央アジアは腐敗菌が比較的少ない環境であったため、乳酸発酵が自然に起こりやすく、発酵乳が生まれやすい条件が整っていました。

</QuizAnswer>

## 問8

発酵が適切に進むための条件として、最も重要な要素の組み合わせはどれか。

1. 温度・湿度・時間
2. 光量・気圧・塩分濃度
3. 標高・風速・水質
4. 紫外線量・重力・磁場


<QuizAnswer>

**答え: 1**

発酵を適切に進めるためには、温度・湿度・時間の3つの条件が重要です。微生物にはそれぞれ活動に適した温度帯があり、適度な湿度（水分）は微生物の生育に不可欠です。また、発酵には一定の時間が必要であり、これらの条件を適切にコントロールすることで、望ましい発酵を実現できます。

</QuizAnswer>

## 問9

人類と発酵の関わりについて、正しい説明はどれか。

1. 発酵食品は19世紀に科学的に発明された比較的新しい技術である
2. 発酵食品は特定の地域でのみ発達し、世界的な広がりはなかった
3. 発酵食品は偶然の産物として生まれ、世界各地の食文化に根づいてきた
4. 発酵の原理は古代から完全に解明されていた


<QuizAnswer>

**答え: 3**

発酵食品は、人類が意図的に作り出したものではなく、自然環境の中で偶然に生まれたものが始まりとされています。その後、世界各地の気候や風土に応じて独自の発酵食品が発展し、各地域の食文化に深く根づいてきました。発酵のメカニズムが科学的に解明されたのは19世紀のパスツール以降ですが、経験的な利用はそれよりはるか昔から行われていました。

</QuizAnswer>

## 問10

発酵によって食品の保存性が高まる主な理由として、正しいものはどれか。

1. 発酵により食品の水分が完全に除去されるため
2. 発酵で生成される酸やアルコールが腐敗菌の繁殖を抑制するため
3. 発酵により食品が真空状態になるため
4. 発酵により食品の温度が常に低く保たれるため


<QuizAnswer>

**答え: 2**

発酵の過程で生成される乳酸・酢酸などの有機酸やアルコールは、食品のpHを下げたり、抗菌作用を発揮したりすることで、腐敗菌の繁殖を抑制します。これが発酵食品の保存性が高い主な理由です。人類は古くからこの性質を利用して、冷蔵技術がない時代にも食品を長期保存してきました。

</QuizAnswer>
