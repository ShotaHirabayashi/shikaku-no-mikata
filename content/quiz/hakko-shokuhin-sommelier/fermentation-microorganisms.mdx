## 問1

麹菌（コウジカビ）は、微生物の分類上どのグループに属するか。

1. 細菌（バクテリア）
2. 酵母
3. カビ（糸状菌）
4. ウイルス


<QuizAnswer>

**答え: 3**

麹菌はカビ（糸状菌）の一種で、学名を Aspergillus 属といいます。発酵に関わる微生物は大きく「カビ」「酵母」「細菌」の3種類に分類され、麹菌はカビに分類されます。日本の発酵食品に欠かせない存在であり、「国菌」とも呼ばれています。

</QuizAnswer>

## 問2

納豆菌の学名として正しいものはどれか。

1. Aspergillus oryzae
2. Lactobacillus bulgaricus
3. Saccharomyces cerevisiae
4. Bacillus subtilis var. natto


<QuizAnswer>

**答え: 4**

納豆菌は枯草菌（Bacillus subtilis）の一種であり、正式な学名は Bacillus subtilis var. natto です。選択肢1は麹菌（黄麹菌）、選択肢2は乳酸菌の一種（ブルガリア菌）、選択肢3は酵母（出芽酵母）の学名です。

</QuizAnswer>

## 問3

酢酸菌の増殖に関する説明として正しいものはどれか。

1. 酸素がなくても増殖できる嫌気性細菌である
2. 酸素を必要とする好気性細菌である
3. 酸素があると死滅する偏性嫌気性細菌である
4. 酸素の有無にかかわらず増殖する通性嫌気性細菌である


<QuizAnswer>

**答え: 2**

酢酸菌はアセトバクター属に代表される好気性細菌で、生育に酸素を必要とします。アルコールを酸化して酢酸を生成する反応には酸素が不可欠であり、酢の醸造では液面に酢酸菌が膜状に広がって増殖します。

</QuizAnswer>

## 問4

麹菌が生成する酵素のうち、でんぷんを糖に分解する酵素はどれか。

1. リパーゼ
2. プロテアーゼ
3. アミラーゼ
4. セルラーゼ


<QuizAnswer>

**答え: 3**

アミラーゼはでんぷんを糖に分解する酵素で、麹菌が豊富に生成します。プロテアーゼはたんぱく質をアミノ酸に分解する酵素、リパーゼは脂質を分解する酵素、セルラーゼはセルロース（植物繊維）を分解する酵素です。麹菌はこれら複数の酵素を同時に生成できることが大きな特徴です。

</QuizAnswer>

## 問5

納豆菌の大きな特徴である「芽胞（がほう）」について、正しい説明はどれか。

1. 芽胞は新しい個体を増やすための生殖細胞である
2. 芽胞は環境が悪化したときに形成される休眠状態で、100℃の煮沸にも耐える
3. 芽胞は酸素を取り込むための特殊な器官である
4. 芽胞は他の微生物を攻撃するために分泌する物質である


<QuizAnswer>

**答え: 2**

芽胞は納豆菌（枯草菌）が栄養分の枯渇など環境が悪化した際に形成する休眠状態の構造です。100℃の煮沸や紫外線、真空状態にも耐える驚異的な耐久性を持ちます。伝統的な納豆づくりで煮沸した稲わらを使うのは、雑菌を死滅させつつ納豆菌の芽胞だけを生き残らせるためです。

</QuizAnswer>

## 問6

乳酸菌の基本的な特徴として正しいものはどれか。

1. カビの一種であり、菌糸を伸ばして増殖する
2. 酵母の一種であり、出芽によって増殖する
3. 細菌の一種であり、糖を分解して乳酸を生成する
4. 原生動物の一種であり、運動性を持つ


<QuizAnswer>

**答え: 3**

乳酸菌は細菌（バクテリア）の一種で、糖を分解して主に乳酸を生成する微生物の総称です。単細胞の微生物であり、細胞分裂によって増殖します。ヨーグルトや漬物など多くの発酵食品の製造に関わっています。

</QuizAnswer>

## 問7

黄麹菌（Aspergillus oryzae）、白麹菌（Aspergillus kawachii）、黒麹菌（Aspergillus luchuensis）のうち、クエン酸を多く生成し、雑菌の繁殖を抑える特性を持つのはどれか。

1. 黄麹菌のみ
2. 白麹菌と黒麹菌
3. 黒麹菌のみ
4. 黄麹菌と白麹菌


<QuizAnswer>

**答え: 2**

白麹菌（A. kawachii）と黒麹菌（A. luchuensis）はクエン酸を多く生成する特徴があり、酸性環境をつくることで雑菌の繁殖を抑制します。焼酎や泡盛の製造で重要な役割を果たしています。一方、黄麹菌はクエン酸の生成量が少なく、主に糖化酵素（アミラーゼ）を多量に生産することが特徴です。

</QuizAnswer>

## 問8

酵母（イースト）による発酵で、糖から主に生成されるものの組み合わせとして正しいものはどれか。

1. 乳酸と水
2. 酢酸と水素
3. アルコール（エタノール）と二酸化炭素
4. アンモニアとメタン


<QuizAnswer>

**答え: 3**

酵母はサッカロマイセス属（Saccharomyces）に代表される単細胞の真菌で、糖を分解してアルコール（エタノール）と二酸化炭素を生成します。この反応がアルコール発酵であり、パンでは二酸化炭素が生地を膨らませ、酒類ではアルコールが生じます。

</QuizAnswer>

## 問9

麹菌が生成するプロテアーゼの働きとして正しいものはどれか。

1. でんぷんを糖に分解する
2. たんぱく質をアミノ酸に分解する
3. 脂質を脂肪酸とグリセリンに分解する
4. セルロースをブドウ糖に分解する


<QuizAnswer>

**答え: 2**

プロテアーゼはたんぱく質を分解してアミノ酸を生成する酵素です。このアミノ酸が発酵食品のうま味の元になります。麹菌はプロテアーゼのほかにも、でんぷんを分解するアミラーゼや脂質を分解するリパーゼなど、多種多様な酵素を生成することが特徴です。

</QuizAnswer>

## 問10

有用菌と有害菌に関する説明として、正しいものはどれか。

1. 有用菌と有害菌は完全に別の種類であり、同じ菌が両方になることはない
2. 麹菌の仲間には、有用な Aspergillus oryzae がある一方、毒素を産生する Aspergillus flavus もいる
3. 納豆菌は有用菌であるため、どのような条件でも安全に利用できる
4. カビはすべて有害であり、食品製造に利用されることはない


<QuizAnswer>

**答え: 2**

Aspergillus 属には、発酵食品に利用される有用な A. oryzae（黄麹菌）がいる一方で、アフラトキシンという強力な発がん性物質を産生する A. flavus も存在します。同じ属でも種によって有用菌にも有害菌にもなるため、発酵食品の製造では安全な菌株の管理が極めて重要です。

</QuizAnswer>
