## 問1

自家製味噌を仕込む際、茹でた大豆を麹と混ぜ合わせるタイミングとして最も適切なものはどれか。

1. 茹でたての熱い状態ですぐに混ぜる
2. 人肌程度（約35〜40℃）まで冷ましてから混ぜる
3. 完全に冷蔵庫で冷やしてから混ぜる
4. 大豆の温度は気にせずいつ混ぜてもよい


<QuizAnswer>

**答え: 2**

茹でた大豆が熱いまま麹と混ぜると、麹菌が熱で死滅してしまい、発酵がうまく進みません。人肌程度（約35〜40℃）まで冷ましてから混ぜることで、麹菌の働きを活かした良い味噌に仕上がります。冷やしすぎる必要はなく、人肌が適温の目安です。

</QuizAnswer>

## 問2

醤油麹を手づくりする際の基本的な材料の割合として正しいものはどれか。

1. 醤油1に対して麹3の割合
2. 醤油と麹を同量ずつ混ぜ合わせる
3. 醤油3に対して麹1の割合
4. 醤油に少量の麹を加える程度でよい


<QuizAnswer>

**答え: 2**

醤油麹の基本的な作り方は、醤油と麹を同量（1:1）で混ぜ合わせることです。仕込んだ翌日に麹が醤油を吸って水分が減るため、ひたひたになるまで醤油を足します。常温で約1週間（冬は10日以上）置き、とろみと麹の香りが出たら完成です。

</QuizAnswer>

## 問3

米麹から甘酒を手づくりする際、最も適切な発酵温度はどれか。

1. 30〜35℃
2. 40〜45℃
3. 55〜60℃
4. 70〜75℃


<QuizAnswer>

**答え: 3**

米麹甘酒の発酵に最適な温度は55〜60℃です。この温度帯で麹の酵素（アミラーゼ）が最も活発に働き、でんぷんを糖に分解して甘みが生まれます。温度が低すぎると甘さが出ず、高すぎると酵素が失活してしまうため、温度管理が甘酒づくりの最大のポイントです。

</QuizAnswer>

## 問4

ぬか床の日常管理において、かき混ぜる主な目的として最も適切なものはどれか。

1. ぬか床の塩分濃度を均一にするため
2. ぬか床に空気を送り込み、酸素を好む菌と嫌う菌のバランスを整えるため
3. 野菜の漬かりを早くするため
4. ぬか床の温度を下げるため


<QuizAnswer>

**答え: 2**

ぬか床をかき混ぜる最大の目的は、空気を送り込んで好気性菌と嫌気性菌のバランスを整えることです。かき混ぜが不足すると、酸素を嫌う菌が増えすぎて異臭の原因になります。容器の隅から上下を返すようにかき混ぜ、表面を平らにならして空気を適度に抜くのが正しい手入れ方法です。

</QuizAnswer>

## 問5

ぬか床の表面に白い膜が張った場合の対処法として、正しいものはどれか。

1. ぬか床全体が傷んでいるのですべて廃棄する
2. 白い膜は産膜酵母なので、薄い膜であればそのまま混ぜ込んでよい
3. 白い膜の部分に塩を大量にかけて殺菌する
4. 白い膜は必ず有害なカビなので、表面を厚く削り取る


<QuizAnswer>

**答え: 2**

ぬか床の表面にできる白い膜は「産膜酵母」という酵母菌の一種で、薄い膜であればそのままぬか床に混ぜ込んで問題ありません。ただし、黒・緑・オレンジなどの色がついている場合はカビですので、その部分は必ず取り除く必要があります。

</QuizAnswer>

## 問6

発酵食品の保存に関する注意点として、最も適切なものはどれか。

1. 発酵食品は腐らないので常温保存で問題ない
2. 開封後も賞味期限まではどの環境でも品質が保たれる
3. 夏場は特に腐りやすいため、冷蔵保存で温度管理を徹底する
4. 発酵食品は冷凍すると風味が向上するので冷凍が最適である


<QuizAnswer>

**答え: 3**

発酵食品は保存性が高いとされていますが、万能ではありません。特に夏場は温度が上がることで過発酵や雑菌の繁殖が起こりやすく、食中毒の原因になることもあります。開封後は冷蔵庫で適切な温度を保ち、衛生的に管理することが大切です。

</QuizAnswer>

## 問7

市販の発酵食品を選ぶ際、生きた菌を摂りたい場合に注意すべき表示のポイントとして最も適切なものはどれか。

1. 原材料に「砂糖」が含まれていないものを選ぶ
2. 加熱処理やアルコール添加がされていないものを選ぶ
3. 容量が大きいものほど発酵が進んでいるので大容量を選ぶ
4. 輸入品のほうが発酵期間が長いので輸入品を選ぶ


<QuizAnswer>

**答え: 2**

市販の発酵食品の中には、流通時の品質安定のために加熱殺菌処理やアルコール添加によって菌の働きを止めているものがあります。生きた菌を摂取したい場合は、「非加熱」「生」などの表示を確認し、加熱処理やアルコール添加がされていないものを選ぶことがポイントです。

</QuizAnswer>

## 問8

自家製味噌の仕込みで、容器に味噌を詰める際に最も注意すべきことはどれか。

1. 容器の半分程度まで詰め、上部に空間を十分に残す
2. 空気が入らないようにすき間なくしっかり詰め、表面を密閉する
3. ふんわりと軽く詰めて空気を多く含ませる
4. 金属製の容器に詰めて密封する


<QuizAnswer>

**答え: 2**

自家製味噌の仕込みでは、容器に空気が入らないようにすき間なくしっかり詰めることが非常に重要です。空気が残るとカビの原因になります。味噌を玉状にして容器に投げ入れるように詰め、表面をビニールシートで覆って空気との接触を防ぎ、風通しのよい冷暗所で保存します。

</QuizAnswer>

## 問9

発酵食品を料理に使う際、菌や酵素をできるだけ活かすための工夫として正しいものはどれか。

1. 味噌汁は味噌を最初から煮込んで長時間加熱するとよい
2. 塩麹は料理の仕上げにサッと和えるなど、加熱しすぎないようにする
3. 発酵食品は必ず100℃以上で加熱殺菌してから食べる
4. 発酵食品は電子レンジで温めると菌が活性化する


<QuizAnswer>

**答え: 2**

発酵食品に含まれる有用な菌や酵素は加熱に弱く、高温で長時間加熱すると死滅・失活してしまいます。塩麹やしょうゆ麹はできあがりの頃にサッと和える、味噌汁は火を止めてから味噌を溶き入れるなど、加熱を最小限にする工夫が効果的です。

</QuizAnswer>

## 問10

発酵食品をより効果的に摂取するための食べ合わせとして、最も適切なものはどれか。

1. 発酵食品は単品で大量に摂るほうが効果が高い
2. 発酵食品同士を組み合わせたり、食物繊維やオリゴ糖を含む食材と合わせて摂る
3. 発酵食品はカフェインと一緒に摂ると効果が倍増する
4. 発酵食品は空腹時に水だけで摂るのが最も効果的である


<QuizAnswer>

**答え: 2**

発酵食品は、複数の種類を組み合わせたり、食物繊維やオリゴ糖を含む食材と一緒に摂ることで効果が高まります。例えば、甘酒に含まれるオリゴ糖はヨーグルトの乳酸菌のエサとなり、納豆菌はキムチの乳酸菌の増殖を助けます。バランスよく組み合わせることで腸内環境の改善効果がより期待できます。

</QuizAnswer>
