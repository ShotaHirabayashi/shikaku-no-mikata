## 問1

食育セミナーを企画する際、参加者の関心を引くために最も重要とされるアプローチはどれか。

1. 専門用語を多用して権威性を示す
2. 食育に関心の低い層にも届くよう工夫する
3. 参加費を無料にして人数を最優先にする
4. 座学のみに絞り効率的に情報を伝える


<QuizAnswer>

**答え: 2**

農林水産省の「食育イベント開催の手引き」では、食育に関心の低い層へのアプローチが重要課題として挙げられています。キャッチーなタイトルの工夫や参加特典の設定、座学と体験を組み合わせたプログラム構成など、幅広い層が参加しやすい仕掛けづくりが効果的な食育セミナーのポイントです。

</QuizAnswer>

## 問2

料理教室を運営する際のアレルギー対応として、最も適切なものはどれか。

1. 当日に口頭でアレルギーの有無を確認する
2. アレルギー食材を一切使用しないメニューのみにする
3. 事前にアレルギー情報を収集し、代替食材を準備する
4. アレルギーのある参加者には参加を控えてもらう


<QuizAnswer>

**答え: 3**

料理教室では、参加者のアレルギー情報を事前に収集し、代替食材を準備しておくことが基本です。当日の口頭確認だけでは漏れが生じるリスクがあり、全メニューからアレルゲンを排除するのも現実的ではありません。万一に備えてアナフィラキシーへの対処法を把握し、賠償責任保険への加入も推奨されています。

</QuizAnswer>

## 問3

食生活改善推進員（ヘルスメイト）に関する記述として、正しいものはどれか。

1. 厚生労働省が直接任命する国家資格保持者である
2. 「私たちの健康は私たちの手で」を合言葉に活動している
3. 活動は栄養指導に限定されており、調理実習は行わない
4. 管理栄養士の資格がなければなることができない


<QuizAnswer>

**答え: 2**

食生活改善推進員（ヘルスメイト）は「私たちの健康は私たちの手で」を合言葉に、地域で食育活動を行うボランティアです。市町村が開催する養成講座を修了し、地域の協議会に入会することで活動を開始でき、特別な国家資格は不要です。全国で約9万6千名が活動しており、保育所や小学校での食育活動も行っています。

</QuizAnswer>

## 問4

第4次食育推進基本計画で新たに加わった重点事項はどれか。

1. 食の安全・安心の確保に関する食育の推進
2. 「新たな日常」やデジタル化に対応した食育の推進
3. 食料自給率の向上を目指す食育の推進
4. 和食文化の保護・継承に関する食育の推進


<QuizAnswer>

**答え: 2**

第4次食育推進基本計画（令和3〜7年度）では、コロナ禍を踏まえて「『新たな日常』やデジタル化に対応した食育の推進」が3つ目の重点事項として新規に追加されました。他の2つの重点事項は「生涯を通じた心身の健康を支える食育の推進」と「持続可能な食を支える食育の推進（SDGsの観点）」です。

</QuizAnswer>

## 問5

子ども食堂における食育活動として、農林水産省の調査で最も実施率が高いものはどれか。

1. 調理体験の場の提供
2. 食材の旬や栄養に関する話
3. 温かな団らんのある共食の場の提供
4. 食品ロス削減の学習プログラム


<QuizAnswer>

**答え: 3**

農林水産省の調査によると、子ども食堂で最も実施率が高い食育活動は「温かな団らんのある共食の場」の提供で86.5%です。次いで「配膳の手伝い」が56.2%、「食材の旬・栄養の話」が36.9%と続きます。子ども食堂は貧困対策から「みんなの居場所」へと発展しており、孤食の解消や地域コミュニティ形成の場としても重要な役割を担っています。

</QuizAnswer>

## 問6

「こ食」の一つである「個食」の意味として正しいものはどれか。

1. いつも同じものばかり食べること
2. 一人きりで食べること
3. 家族がそれぞれバラバラのものを食べること
4. 少量しか食べないこと


<QuizAnswer>

**答え: 3**

「個食」とは、家族が食卓を囲んでいても、それぞれがバラバラの別々のものを食べている状態を指します。一人で食べる「孤食」、同じものばかり食べる「固食」、少量しか食べない「小食」などと区別されます。共食の推進においては、これらの「こ食」の問題を理解し、家族で同じ食事を楽しむ機会を増やすことが重要とされています。

</QuizAnswer>

## 問7

農業体験を通じた食農教育について、JAグループが提唱する「食農教育」の特徴として最も適切なものはどれか。

1. 食育に加え、農業・地域・自然との関わりにも注目する教育である
2. 農業従事者の育成を主な目的とした職業教育である
3. 農産物の流通経路を学ぶ経済教育の一環である
4. 有機農業の普及を目的とした環境教育である


<QuizAnswer>

**答え: 1**

JAグループが提唱する「食農教育」は、食育に加えて農業・地域・自然との関わりにも注目する教育です。栽培から収穫までの農業体験を通じて、食べ物への関心を高め、旬の野菜や郷土料理の知識を深め、食への感謝の心を育むことが主な目的です。農業従事者の育成や特定の農法の普及を目的としたものではありません。

</QuizAnswer>

## 問8

農林水産省が設立した全国食育推進ネットワーク「みんなの食育」の主な目的として、最も適切なものはどれか。

1. 食育に関する国家資格の認定と管理
2. 食育活動に対する補助金の直接交付
3. 会員相互の情報交換・連携・協働した食育の実施
4. 食育に関する法律の制定と改正


<QuizAnswer>

**答え: 3**

「みんなの食育」は農林水産省が設立した全国食育推進ネットワークで、約570の団体・個人が加入しています。その主な目的は、会員同士の情報交換や連携を促進し、協働して食育活動を実施することです。医師・管理栄養士・農林漁業者・食品企業など多職種が参加し、それぞれの専門性を活かした横断的な連携が特徴です。

</QuizAnswer>

## 問9

子ども向けの料理教室を開催する際のポイントとして、最も適切なものはどれか。

1. 大人数で効率よく進行するため30名以上を一度に受け入れる
2. 安全のため子どもには見学のみとし、調理は講師が行う
3. 少人数制とし、全工程を子ども自身が体験できるようにする
4. 栄養学の講義を中心に据え、調理は補助的に行う


<QuizAnswer>

**答え: 3**

子ども向けの料理教室では、最大8名程度の少人数制で、全工程を子ども自身が体験できるようにすることがポイントです。年齢に応じた調理道具を準備し、2歳からでも「混ぜる・こねる・ちぎる」といった作業に参加できます。クイズ形式を取り入れるなど楽しみながら学べる工夫も効果的で、五感を使った体験が食への関心を高めます。

</QuizAnswer>

## 問10

食育活動における「共食（きょうしょく）」の効果として、調査データから明らかになっていることはどれか。

1. 共食の頻度と体重管理には強い相関がある
2. 共食をしている人は主食・主菜・副菜を揃える割合が高い
3. 共食は週1回で十分な効果が得られる
4. 共食の効果は子どもにのみ認められる


<QuizAnswer>

**答え: 2**

調査データによると、孤食をしていない人が主食・主菜・副菜を揃えて食べる割合は62.3%であるのに対し、週2日以上孤食をする人では42.4%と約20ポイントの差があります。共食には栄養バランスの改善だけでなく、食事マナーの伝達や食文化の継承といった効果もあり、子どもから高齢者まで幅広い年代にとって重要な食育の機会となっています。

</QuizAnswer>
