## 問1

厚生労働省の「授乳・離乳の支援ガイド」において、離乳食の開始時期の目安とされているのはどれか。

1. 生後3〜4か月頃
2. 生後5〜6か月頃
3. 生後7〜8か月頃
4. 生後9〜10か月頃


<QuizAnswer>

**答え: 2**

厚生労働省の「授乳・離乳の支援ガイド（2019年改定版）」では、離乳食の開始時期の目安を生後5〜6か月頃としています。この時期になると赤ちゃんの消化機能が発達し、首のすわりがしっかりして、食べ物に興味を示すなどのサインが見られるようになります。早すぎる開始は消化器官への負担となり、遅すぎると栄養不足や咀嚼機能の発達遅延につながる可能性があります。

</QuizAnswer>

## 問2

思春期の女子に特に不足しやすく、月経の開始に伴い必要量が増加するミネラルはどれか。

1. カルシウム
2. 亜鉛
3. 鉄
4. マグネシウム


<QuizAnswer>

**答え: 3**

思春期の女子は月経が始まることにより鉄の損失が増加するため、鉄欠乏性貧血になりやすくなります。学校給食のある日はほぼ必要量を充足しているのに対し、給食のない日は鉄の不足が顕著になるという調査結果もあります。思春期の女子は意識的に赤身の肉や魚、レバー、ほうれん草などの鉄を多く含む食品を摂取することが大切です。

</QuizAnswer>

## 問3

妊娠を計画している女性が、胎児の神経管閉鎖障害のリスク低減のために、通常の食事に加えてサプリメント等から摂取することが推奨されている栄養素はどれか。

1. ビタミンA
2. 葉酸
3. ビタミンD
4. カルシウム


<QuizAnswer>

**答え: 2**

厚生労働省は、妊娠を計画している女性および妊娠初期の女性に対し、通常の食事からの葉酸摂取（推奨量240μg）に加え、サプリメント等から1日400μgの葉酸を追加摂取することを推奨しています。葉酸は胎児の神経管閉鎖障害（二分脊椎や無脳症など）の発症リスクを低減させる効果があるとされています。

</QuizAnswer>

## 問4

高齢者のフレイル（虚弱）予防において、特に摂取量の確保が重要とされる栄養素はどれか。

1. 炭水化物
2. 脂質
3. たんぱく質
4. 食物繊維


<QuizAnswer>

**答え: 3**

フレイルとは加齢に伴い筋力や心身の活力が低下した状態を指し、その大きな要因の一つが低栄養です。特にたんぱく質の摂取不足は筋肉量・筋力の低下（サルコペニア）を引き起こし、フレイルの進行を加速させます。高齢者には体重1kgあたり1.0〜1.2gのたんぱく質摂取が推奨されており、肉・魚・卵・乳製品などから積極的に摂ることが大切です。

</QuizAnswer>

## 問5

老化の原因の一つとされる活性酸素を除去する働きを持つ物質を総称して何というか。

1. 酵素阻害物質
2. 抗酸化物質
3. 免疫促進物質
4. 代謝活性物質


<QuizAnswer>

**答え: 2**

体内で過剰に発生した活性酸素は、細胞や血管を酸化させ、老化やがん、生活習慣病の原因となります。この活性酸素を除去・無害化する働きを持つ物質を「抗酸化物質」と呼びます。代表的な抗酸化物質にはビタミンC、ビタミンE、ポリフェノール、カロテノイドなどがあり、野菜や果物を積極的に摂取することで補うことができます。

</QuizAnswer>

## 問6

腸内細菌叢（腸内フローラ）に関する記述として、正しいものはどれか。

1. ビフィズス菌は加齢とともに増加する
2. 乳児期の腸内にはビフィズス菌がほとんど存在しない
3. ビフィズス菌の割合は離乳期を境に減少し始める
4. 腸内細菌の種類は生涯を通じて一定である


<QuizAnswer>

**答え: 3**

ヒトの腸内には数百種類、約40兆個の細菌が生息しています。生まれたばかりの赤ちゃんの腸内ではビフィズス菌が約50%を占めますが、離乳期を境に減少し始め、成人では約10%、60代以降はさらに減少して約5%になるとされています。加齢に伴いビフィズス菌は減少し、大腸菌などの悪玉菌が増加する傾向があります。

</QuizAnswer>

## 問7

メタボリックシンドロームの診断基準において、必須条件とされるウエスト周囲径の基準値として正しいものはどれか。

1. 男性80cm以上、女性85cm以上
2. 男性85cm以上、女性90cm以上
3. 男性90cm以上、女性80cm以上
4. 男性90cm以上、女性85cm以上


<QuizAnswer>

**答え: 2**

日本のメタボリックシンドロームの診断基準では、内臓脂肪の蓄積を示すウエスト周囲径（おへその高さの腹囲）が男性85cm以上、女性90cm以上であることが必須条件です。これに加えて、高血圧・高血糖・脂質代謝異常の3項目のうち2つ以上に該当すると、メタボリックシンドロームと診断されます。

</QuizAnswer>

## 問8

乳幼児期の離乳食において、食物アレルギーへの対応として現在推奨されている考え方はどれか。

1. アレルギーの原因となりやすい食品は3歳まで一切与えない
2. アレルギー予防のため離乳食の開始時期をできるだけ遅らせる
3. 離乳食の進行に合わせて様々な食品を少量から試していく
4. アレルギーの家族歴がある場合は離乳食を1歳まで開始しない


<QuizAnswer>

**答え: 3**

以前はアレルゲンとなりやすい食品の摂取を遅らせることが推奨されていましたが、現在の「授乳・離乳の支援ガイド（2019年改定版）」では、離乳食の開始や特定食品の導入を遅らせてもアレルギー予防効果があるという科学的根拠はないとされています。離乳食の進行に合わせて、様々な食品を少量から試し、赤ちゃんの様子を観察しながら進めていくことが推奨されています。

</QuizAnswer>

## 問9

授乳期の母親の栄養管理に関する記述として、正しいものはどれか。

1. 授乳期は妊娠中よりもエネルギーの付加量が少なくてよい
2. 授乳期には通常より多くの水分摂取が必要である
3. 母乳の成分は母親の食事内容に一切影響されない
4. 授乳期のカフェイン摂取は母乳に移行しない


<QuizAnswer>

**答え: 2**

授乳期の母親は母乳の産生のために通常よりも多くの水分を必要とします。母乳の約88%は水分で構成されているため、十分な水分補給が重要です。また、母乳の成分は母親の食事内容によって変化することが知られており、カフェインやアルコールなども母乳に移行するため注意が必要です。授乳期のエネルギー付加量は+350kcal/日とされています。

</QuizAnswer>

## 問10

高齢者の食事において、誤嚥（ごえん）を予防するための工夫として、最も適切なものはどれか。

1. 水分はさらさらした状態のまま提供する
2. 食べ物にとろみをつけたり、やわらかく調理したりする
3. 食事中はなるべく横になった姿勢で食べる
4. できるだけ短時間で急いで食事を済ませる


<QuizAnswer>

**答え: 2**

高齢者は加齢に伴い嚥下（えんげ）機能が低下するため、誤嚥のリスクが高まります。誤嚥を予防するためには、食べ物をやわらかく調理する、とろみ剤を使ってとろみをつける、一口の量を少なくするなどの工夫が効果的です。さらさらした液体はかえって誤嚥しやすいため注意が必要で、食事中は上体を起こした姿勢を保ち、ゆっくり食べることが大切です。

</QuizAnswer>
