## 問1

食育基本法が制定されたのは何年か。

1. 平成12年（2000年）
2. 平成15年（2003年）
3. 平成17年（2005年）
4. 平成20年（2008年）


<QuizAnswer>

**答え: 3**

食育基本法は平成17年（2005年）6月17日に公布され、同年7月15日に施行されました。国民が健全な心身を培い、豊かな人間性を育むため、食育に関する施策を総合的かつ計画的に推進することを目的として制定された法律です。

</QuizAnswer>

## 問2

食育基本法の前文では、食育はどのように位置づけられているか。

1. 知育・徳育・体育の基礎となるもの
2. 徳育の一部として行われるもの
3. 体育と同等の教育分野であるもの
4. 知育を補完する特別活動であるもの


<QuizAnswer>

**答え: 1**

食育基本法の前文では、食育を「生きる上での基本であって、知育、徳育及び体育の基礎となるべきもの」と位置づけています。食に関する知識と食を選択する力を習得し、健全な食生活を実践できる人間を育てることが食育の根本的な考え方です。

</QuizAnswer>

## 問3

食育基本法に基づく食育推進に関する事務は、2015年の改正により、どの省庁が所管することになったか。

1. 内閣府
2. 農林水産省
3. 厚生労働省
4. 文部科学省


<QuizAnswer>

**答え: 2**

食育基本法の制定当初は内閣府が食育推進の事務を所管していましたが、平成27年（2015年）の改正により農林水産省に移管されました。これにより、食料の生産から消費までを一貫して担う農林水産省が食育推進の中心的な役割を果たすことになりました。

</QuizAnswer>

## 問4

箸のマナー違反（嫌い箸）のうち、どの料理を食べようか迷い、箸を持ったまま料理の上をあちこち動かす行為を何というか。

1. ねぶり箸
2. 渡し箸
3. 移り箸
4. 迷い箸


<QuizAnswer>

**答え: 4**

迷い箸とは、どの料理を食べようか決められず、箸を持ったまま料理の上をあちこちと動かすことをいいます。ねぶり箸は箸先をなめること、渡し箸は食器の上に箸を渡し置くこと、移り箸はいったん取りかけた料理をやめて別の料理に箸を移すことで、いずれも嫌い箸に該当します。

</QuizAnswer>

## 問5

日本の学校給食の起源とされる、明治22年（1889年）に貧困児童を対象に昼食を提供した学校がある都道府県はどこか。

1. 東京都
2. 山形県
3. 新潟県
4. 北海道


<QuizAnswer>

**答え: 2**

日本の学校給食は、明治22年（1889年）に山形県鶴岡町（現・鶴岡市）の私立忠愛小学校で、貧困家庭の児童を対象に無料で昼食を提供したのが始まりとされています。このことから、学校給食の発祥の地は山形県とされています。

</QuizAnswer>

## 問6

第4次食育推進基本計画（令和3年度～）の3つの重点事項に含まれないものはどれか。

1. 生涯を通じた心身の健康を支える食育の推進
2. 持続可能な食を支える食育の推進
3. 食品産業の国際競争力を高める食育の推進
4. 「新たな日常」やデジタル化に対応した食育の推進


<QuizAnswer>

**答え: 3**

第4次食育推進基本計画の3つの重点事項は、(1)生涯を通じた心身の健康を支える食育の推進、(2)持続可能な食を支える食育の推進、(3)「新たな日常」やデジタル化に対応した食育の推進です。食品産業の国際競争力に関する項目は含まれていません。

</QuizAnswer>

## 問7

食育基本法は全部で何条から構成されているか。

1. 33条
2. 25条
3. 40条
4. 50条


<QuizAnswer>

**答え: 1**

食育基本法は前文と全33条で構成されています。第1章「総則」（第1条～第15条）で基本理念や関係者の責務を定め、第2章で「食育推進基本計画等」、第3章で「基本的施策」、第4章で「食育推進会議等」について規定しています。

</QuizAnswer>

## 問8

学校給食法が制定されたのは何年か。

1. 昭和22年（1947年）
2. 昭和25年（1950年）
3. 昭和27年（1952年）
4. 昭和29年（1954年）


<QuizAnswer>

**答え: 4**

学校給食法は昭和29年（1954年）に制定されました。戦後の食糧難の時代にGHQの支援で再開された学校給食を、法的に位置づけたものです。平成20年（2008年）の改正では、学校給食を「食育」の推進に活用することが新たに盛り込まれ、「生きた教材」としての役割が明確化されました。

</QuizAnswer>

## 問9

食育基本法の前文において、食育を通じて子どもたちが身につけるべきとされているものはどれか。

1. 国際的な食文化の理解
2. 豊かな人間性と生きる力
3. 科学的な調理技術
4. 食品衛生の専門知識


<QuizAnswer>

**答え: 2**

食育基本法の前文では「子どもたちが豊かな人間性をはぐくみ、生きる力を身に付けていくためには、何よりも『食』が重要である」と述べられています。食育は単なる栄養知識の習得ではなく、様々な経験を通じて「食」に関する知識と食を選択する力を身につけ、健全な食生活を実践できる人間を育てることを目指しています。

</QuizAnswer>

## 問10

箸のマナー違反（嫌い箸）のうち、ご飯に箸を突き立てる行為を何というか。

1. 握り箸
2. 拝み箸
3. 刺し箸
4. 合わせ箸


<QuizAnswer>

**答え: 3**

刺し箸とは、ご飯や料理に箸を突き刺して立てる行為のことです。これは仏式の葬儀で故人に供える枕飯（まくらめし）の作法と同じであるため、日常の食事では非常に縁起が悪いとされ、強く忌み嫌われるマナー違反です。握り箸は箸を握るように持つこと、拝み箸は箸を両手で挟んで拝むようにすることを指します。

</QuizAnswer>
