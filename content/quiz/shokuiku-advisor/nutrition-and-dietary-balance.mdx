## 問1

三大栄養素（炭水化物・脂質・たんぱく質）のうち、1gあたりのエネルギー産生量が最も大きいのはどれか。

1. 炭水化物
2. 脂質
3. たんぱく質
4. 炭水化物とたんぱく質は同じで、脂質より大きい


<QuizAnswer>

**答え: 2**

三大栄養素のエネルギー産生量（アトウォーター係数）は、炭水化物が約4kcal/g、たんぱく質が約4kcal/g、脂質が約9kcal/gです。脂質は炭水化物やたんぱく質の2倍以上のエネルギーを産生するため、少量でも高エネルギーとなります。このため、脂質の過剰摂取は肥満の原因になりやすいとされています。

</QuizAnswer>

## 問2

ビタミンA・D・E・Kに共通する特徴として正しいものはどれか。

1. 水に溶けやすく、過剰に摂取しても尿から排出される
2. 脂質と一緒に摂取すると吸収率が高まる
3. 体内に蓄積されないため、毎日大量に摂取する必要がある
4. すべて補酵素として糖質代謝に関わっている


<QuizAnswer>

**答え: 2**

ビタミンA・D・E・Kは「脂溶性ビタミン」に分類され、油脂に溶けやすい性質を持ちます。そのため、脂質と一緒に摂取することで吸収率が高まります。脂溶性ビタミンは体内（主に肝臓や脂肪組織）に蓄積されやすいため、過剰摂取による健康障害のリスクがあり、摂りすぎに注意が必要です。覚え方として「DAKE（だけ）が脂溶性」という語呂合わせが知られています。

</QuizAnswer>

## 問3

「日本人の食事摂取基準」を策定している機関はどれか。

1. 農林水産省
2. 厚生労働省
3. 文部科学省
4. 消費者庁


<QuizAnswer>

**答え: 2**

「日本人の食事摂取基準」は厚生労働省が策定しており、5年ごとに改定されます。健康の保持・増進や生活習慣病の予防を目的に、エネルギーおよび各栄養素の摂取量の基準を示したものです。なお、農林水産省は食事バランスガイドの策定に関わり、文部科学省は日本食品標準成分表を策定しています。

</QuizAnswer>

## 問4

食事バランスガイドのイラスト（コマの形）において、最も多いサービング数（SV）が設定されている料理区分はどれか（基本形2,200kcal±200の場合）。

1. 主菜
2. 副菜
3. 主食
4. 牛乳・乳製品


<QuizAnswer>

**答え: 3**

食事バランスガイド（基本形2,200kcal±200）では、主食が5〜7SVと最も多く設定されています。続いて副菜が5〜6SV、主菜が3〜5SV、牛乳・乳製品が2SV、果物が2SVとなっています。コマの上部から主食→副菜→主菜→牛乳・乳製品→果物の順に配置されており、主食（ごはん・パン・麺類）を食事の中心に据えることが推奨されています。

</QuizAnswer>

## 問5

人体を構成するたんぱく質のもととなるアミノ酸は約20種類あるが、そのうち体内で合成できないため食事から摂取しなければならないアミノ酸を何というか。

1. 非必須アミノ酸
2. 分岐鎖アミノ酸
3. 必須アミノ酸
4. 遊離アミノ酸


<QuizAnswer>

**答え: 3**

体内で合成できない、あるいは十分な量を合成できないため、食事から摂取する必要があるアミノ酸を「必須アミノ酸（不可欠アミノ酸）」といいます。必須アミノ酸は全部で9種類（バリン、ロイシン、イソロイシン、スレオニン、メチオニン、フェニルアラニン、トリプトファン、リシン、ヒスチジン）あります。必須アミノ酸をバランスよく含む食品は「良質なたんぱく質」と呼ばれ、卵・肉・魚・大豆製品などが該当します。

</QuizAnswer>

## 問6

カルシウムの吸収を促進するビタミンとして最も適切なものはどれか。

1. ビタミンA
2. ビタミンB1
3. ビタミンC
4. ビタミンD


<QuizAnswer>

**答え: 4**

ビタミンDは小腸でのカルシウムの吸収を促進し、骨へのカルシウムの沈着を助ける働きがあります。ビタミンDは魚類やきのこ類に多く含まれるほか、紫外線を浴びることで皮膚でも合成されます。カルシウムの摂取と合わせてビタミンDを十分に摂ることが、骨粗しょう症の予防に重要です。

</QuizAnswer>

## 問7

食物繊維の働きに関する記述として、最も適切なものはどれか。

1. エネルギー源として三大栄養素と同程度のカロリーを持つ
2. 小腸で消化吸収され、血糖値を急激に上昇させる
3. 腸内環境を整え、便通を改善する効果がある
4. 摂取量が多いほど良いため、上限量は設定されていない


<QuizAnswer>

**答え: 3**

食物繊維は人の消化酵素では消化されにくい成分で、腸内環境の改善や便通の促進に大きな役割を果たします。また、食後の血糖値の急上昇を抑制したり、コレステロールの吸収を抑えたりする効果も知られています。日本人の食事摂取基準（2020年版）では、成人男性で21g以上/日、成人女性で18g以上/日を目標量としていますが、多くの日本人は目標量に達していないのが現状です。

</QuizAnswer>

## 問8

日本人の食事摂取基準（2020年版）において、成人男性の食塩相当量の目標量として正しいものはどれか。

1. 1日あたり10.0g未満
2. 1日あたり8.0g未満
3. 1日あたり7.5g未満
4. 1日あたり5.0g未満


<QuizAnswer>

**答え: 3**

日本人の食事摂取基準（2020年版）では、成人男性の食塩相当量の目標量は1日あたり7.5g未満、成人女性は6.5g未満と設定されています。WHOが推奨する5g未満と比べると高い数値ですが、日本人の食習慣を考慮して段階的に引き下げられてきました。なお、高血圧の予防や治療の観点からは、さらに少ない摂取量が望ましいとされています。

</QuizAnswer>

## 問9

炭水化物に含まれる「糖質」と「食物繊維」に関する説明として、正しいものはどれか。

1. 糖質と食物繊維はどちらも体内で消化吸収される
2. 炭水化物から食物繊維を除いたものが糖質である
3. 食物繊維は糖質の一種で、エネルギー源として重要である
4. 糖質は消化されないが、食物繊維は消化吸収される


<QuizAnswer>

**答え: 2**

炭水化物は「糖質」と「食物繊維」に大別されます。糖質は体内で消化吸収されてエネルギー源（約4kcal/g）となりますが、食物繊維はヒトの消化酵素では消化されにくい成分です。したがって「炭水化物＝糖質＋食物繊維」という関係が成り立ち、炭水化物から食物繊維を差し引いたものが糖質となります。食品表示でも、この関係を基に糖質量が算出されています。

</QuizAnswer>

## 問10

鉄の供給源として最も効率よく吸収される「ヘム鉄」を多く含む食品群はどれか。

1. ほうれん草・小松菜などの緑黄色野菜
2. レバー・赤身肉・かつおなどの動物性食品
3. 大豆・納豆・豆腐などの大豆製品
4. ひじき・わかめなどの海藻類


<QuizAnswer>

**答え: 2**

鉄には、動物性食品に多い「ヘム鉄」と、植物性食品に多い「非ヘム鉄」の2種類があります。ヘム鉄は吸収率が15〜25%程度と高く、非ヘム鉄の吸収率（2〜5%程度）と比べて効率よく体内に取り込まれます。レバー（特に豚レバー）、赤身肉、かつおなどの動物性食品はヘム鉄を豊富に含むため、鉄不足の予防に効果的です。非ヘム鉄はビタミンCと一緒に摂ると吸収率が向上します。

</QuizAnswer>
