## 問1

家庭外で調理・加工された弁当や惣菜などを購入し、自宅や職場に持ち帰って食べる食事形態を何というか。

1. 外食
2. 内食
3. 中食
4. 給食


<QuizAnswer>

**答え: 3**

中食（なかしょく）とは、弁当・惣菜・デリバリーなど、家庭外で調理された食品を購入して自宅や職場などで食べる食事形態のことです。外食は飲食店などで食事をすること、内食は家庭で食材から調理して食べることを指します。中食市場は共働き世帯や単身世帯の増加を背景に拡大を続けています。

</QuizAnswer>

## 問2

食品流通において、生産者と小売業者の間に入り、大量に商品を仕入れて必要な量を小売業者に販売する役割を担う業者を何というか。

1. 卸売業者
2. 仲買人
3. ブローカー
4. フランチャイザー


<QuizAnswer>

**答え: 1**

卸売業者は、生産者やメーカーから大量に商品を仕入れ、小売業者に必要な量を販売する「中間流通業」の役割を担っています。これにより、まとまった量で出荷したい生産者と、少量ずつ仕入れたい小売業者の間のギャップを埋め、流通全体のコスト削減や業務負担の軽減を図っています。

</QuizAnswer>

## 問3

幅広い品揃えの商品を部門別に管理し、対面販売を中心とした接客サービスを特徴とする大規模小売業の業態はどれか。

1. スーパーマーケット
2. ホームセンター
3. ドラッグストア
4. 百貨店（デパート）


<QuizAnswer>

**答え: 4**

百貨店（デパート）は、都市の繁華街やターミナル駅周辺に立地し、衣・食・住にわたる多種多様な商品を部門別に管理して販売する大規模小売店です。対面販売による丁寧な接客サービスが特徴で、高級ブランド品や贈答品の取り扱いにも強みを持っています。

</QuizAnswer>

## 問4

飲食店のメニュー管理において、売上高や売上構成比をもとにメニューをA・B・Cの3ランクに分類し、品目の見直しに活用する分析手法を何というか。

1. SWOT分析
2. ABC分析
3. マーケティングミックス
4. バリューチェーン分析


<QuizAnswer>

**答え: 2**

ABC分析とは、メニューごとの売上高や粗利益などのデータをもとに、貢献度の高い順にA（上位）・B（中位）・C（下位）とランク付けする手法です。この分析により、売れ筋メニューの強化や利益率の低いメニューの改廃を判断でき、食材の仕入れ・在庫管理の効率化や食品ロスの削減にもつながります。

</QuizAnswer>

## 問5

長時間営業（多くは24時間）、小規模店舗、住宅地やオフィス街など多様な場所に立地し、食品・日用品から公共料金の支払いまで幅広いサービスを提供する小売業態はどれか。

1. スーパーマーケット
2. 専門店
3. コンビニエンスストア
4. ディスカウントストア


<QuizAnswer>

**答え: 3**

コンビニエンスストアは、「便利さ（convenience）」を最大の特徴とする小売業態です。24時間営業や多様な立地に加え、食品・日用品の販売だけでなく、公共料金の支払い、ATM、宅配便の受付など生活インフラとしての機能も備えています。小規模ながら高回転で商品を入れ替える効率的な経営が特徴です。

</QuizAnswer>

## 問6

食品の流通経路において、青果・水産物・花きなどの生鮮食料品が集荷され、せりや入札によって価格が決定される施設を何というか。

1. 物流センター
2. ショッピングモール
3. 卸売市場
4. フードコート


<QuizAnswer>

**答え: 3**

卸売市場は、全国各地から生鮮食料品を集荷し、せりや入札などの方法で公正な価格を形成する流通の拠点です。卸売市場には、卸売業者（生産者から集荷して販売）と仲卸業者（卸売業者から購入し小売業者等に販売）がおり、日本の青果・水産物の流通において大きな割合を占めています。

</QuizAnswer>

## 問7

飲食店の経営において、料理の売価に対する食材費の割合を示す指標を何というか。

1. 利益率
2. 回転率
3. 歩留まり率
4. 原価率


<QuizAnswer>

**答え: 4**

原価率とは、料理の売価に占める食材費（原価）の割合のことで、「食材費÷売価×100」で算出します。一般的な飲食店の食材原価率は30％前後が目安とされています。原価率が高すぎると利益が出にくく、低すぎると顧客満足度が下がるため、メニュー全体のバランスを考えた管理が重要です。

</QuizAnswer>

## 問8

食品・衣料品・家庭用品など幅広い商品をワンフロアまたは複数フロアで総合的に取り扱い、セルフサービス方式を基本とする大型小売店のことを何というか。

1. GMS（総合スーパー）
2. カテゴリーキラー
3. 百貨店
4. アウトレットモール


<QuizAnswer>

**答え: 1**

GMS（General Merchandise Store：総合スーパー）は、食料品だけでなく衣料品・家庭用品・生活雑貨などを幅広く取り扱う大型小売店です。セルフサービス方式を基本とし、百貨店と比べて低価格での大量販売を特徴としています。イオンやイトーヨーカドーなどが代表的なGMSです。

</QuizAnswer>

## 問9

小売業者が独自に企画・開発し、自社の店舗で販売するブランド商品のことを何というか。

1. NB（ナショナルブランド）
2. OEM商品
3. PB（プライベートブランド）
4. SPA商品


<QuizAnswer>

**答え: 3**

PB（プライベートブランド）とは、小売業者や流通業者が独自に企画・開発し、自社店舗のみで販売するブランド商品のことです。メーカーが開発・販売するNB（ナショナルブランド）と比べて、中間マージンを削減できるため低価格で提供できるメリットがあります。コンビニやスーパーの多くがPB商品の開発に力を入れています。

</QuizAnswer>

## 問10

飲食店などで食事の場を提供され、その場で調理された料理を食べる食事形態を何というか。

1. 中食
2. 外食
3. 内食
4. 配食


<QuizAnswer>

**答え: 2**

外食とは、レストランや食堂などの飲食店で、食事の場の提供を受けて料理を食べる食事形態のことです。これに対し、家庭で食材を調理して食べるのが「内食（ないしょく・うちしょく）」、家庭外で調理された弁当や惣菜を購入して自宅等で食べるのが「中食（なかしょく）」です。この3つの分類は食マーケットを理解する上での基本となります。

</QuizAnswer>
