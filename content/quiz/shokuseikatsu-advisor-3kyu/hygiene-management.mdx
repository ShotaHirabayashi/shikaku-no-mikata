## 問1

食中毒予防の三原則として正しい組み合わせはどれか。

1. つけない・減らす・冷やす
2. 洗う・冷やす・加熱する
3. つけない・増やさない・やっつける
4. 分ける・冷やす・消毒する


<QuizAnswer>

**答え: 3**

食中毒予防の三原則は、細菌を食べ物に「つけない」（汚染防止）、食べ物に付着した細菌を「増やさない」（増殖抑制）、食べ物や調理器具に付着した細菌を「やっつける」（加熱殺菌）の3つです。この三原則は食品衛生管理の最も基本的な考え方であり、家庭でも飲食店でも共通して重要とされています。

</QuizAnswer>

## 問2

ノロウイルスによる食中毒が最も多く発生する時期はいつか。

1. 冬場（11月〜2月頃）
2. 夏場（6月〜8月頃）
3. 春先（3月〜5月頃）
4. 秋口（9月〜10月頃）


<QuizAnswer>

**答え: 1**

ノロウイルスによる食中毒は1年を通じて発生しますが、特に冬場（11月〜2月頃）に流行のピークを迎えます。一方、細菌性食中毒は気温と湿度が高くなる夏場に多く発生します。ノロウイルスは低温・乾燥した環境でも長期間生存できるため、冬季に感染が拡大しやすいのが特徴です。

</QuizAnswer>

## 問3

フグの毒の名称として正しいものはどれか。

1. ムスカリン
2. テトロドトキシン
3. アフラトキシン
4. ソラニン


<QuizAnswer>

**答え: 2**

フグに含まれる毒は「テトロドトキシン」という神経毒で、主に肝臓や卵巣に多く含まれます。テトロドトキシンは加熱しても分解されないため、調理による無毒化はできません。ムスカリンは毒キノコの毒素、アフラトキシンはカビ毒、ソラニンはジャガイモの芽に含まれる毒素です。

</QuizAnswer>

## 問4

HACCP（ハサップ）の正式名称として正しいものはどれか。

1. Health Analysis and Critical Control Plan
2. Hazard Assessment and Cooking Control Point
3. Hygiene Analysis and Contamination Control Process
4. Hazard Analysis and Critical Control Point


<QuizAnswer>

**答え: 4**

HACCPは「Hazard Analysis and Critical Control Point」の略で、日本語では「危害分析・重要管理点」と訳されます。食品の製造工程において、どの段階でどのような危害（ハザード）が発生しうるかを分析し、特に重要な管理点（CCP）を定めて継続的に監視・記録する衛生管理の手法です。日本では2021年6月から、原則としてすべての食品等事業者にHACCPに沿った衛生管理が義務化されています。

</QuizAnswer>

## 問5

食中毒菌が最も活発に増殖する温度帯（危険温度帯）として正しいものはどれか。

1. マイナス15℃〜0℃
2. 約10℃〜60℃
3. 約60℃〜75℃
4. 約75℃〜100℃


<QuizAnswer>

**答え: 2**

多くの食中毒菌は約10℃〜60℃の温度帯で増殖が可能であり、この範囲は「危険温度帯」と呼ばれます。特に30〜40℃付近で最も活発に増殖します。そのため、食品を安全に保存するには冷蔵庫で10℃以下に冷やすか、加熱して60℃以上を保つことが重要です。

</QuizAnswer>

## 問6

黄色ブドウ球菌による食中毒の特徴として正しいものはどれか。

1. 潜伏期間が1週間以上と長い
2. 主に冬場に多く発生する
3. 菌が作り出す毒素（エンテロトキシン）は加熱しても分解されにくい
4. 感染力が強く、人から人へ容易に感染する


<QuizAnswer>

**答え: 3**

黄色ブドウ球菌は食品中で増殖する際にエンテロトキシンという毒素を産生します。この毒素は100℃で30分加熱しても分解されないため、一度毒素が作られてしまうと加熱調理では食中毒を防げません。黄色ブドウ球菌は手指の傷口や化膿した部分に多く存在するため、調理時の手洗いや手袋の使用など「つけない」対策が特に重要です。

</QuizAnswer>

## 問7

ノロウイルスの消毒に最も効果的なものはどれか。

1. 次亜塩素酸ナトリウム
2. 消毒用エタノール（アルコール）
3. 逆性石鹸
4. 重曹水


<QuizAnswer>

**答え: 1**

ノロウイルスはアルコール（エタノール）消毒が効きにくいという特徴があります。ノロウイルスの消毒には、次亜塩素酸ナトリウム（家庭用塩素系漂白剤を希釈したもの）が最も効果的です。また、食品に対しては中心温度85〜90℃で90秒以上の加熱が有効な予防法とされています。

</QuizAnswer>

## 問8

鶏肉が主な原因食品となり、日本で発生件数が最も多い細菌性食中毒の原因菌はどれか。

1. 腸炎ビブリオ
2. ボツリヌス菌
3. ウエルシュ菌
4. カンピロバクター


<QuizAnswer>

**答え: 4**

カンピロバクターは、近年の日本における細菌性食中毒の発生件数で最も多い原因菌です。主に鶏肉の生食や加熱不足が原因で発生します。カンピロバクターは少量の菌でも感染が成立し、潜伏期間が2〜5日とやや長いのが特徴です。予防には鶏肉を中心部まで十分に加熱すること、生肉を扱った調理器具の洗浄・消毒が重要です。

</QuizAnswer>

## 問9

食中毒予防の三原則のうち「やっつける」に該当する対策として、一般的に食品の中心温度をどの程度まで加熱することが推奨されているか。

1. 50℃で3分以上
2. 65℃で30秒以上
3. 75℃で1分以上
4. 90℃で10分以上


<QuizAnswer>

**答え: 3**

ほとんどの食中毒菌は加熱により死滅するため、食品の中心部の温度が75℃で1分以上になるように加熱することが推奨されています。特に肉料理では、表面だけでなく中心部まで十分に火を通すことが重要です。なお、ノロウイルスの場合はより高い温度（85〜90℃で90秒以上）の加熱が必要とされています。

</QuizAnswer>

## 問10

家庭での食中毒予防として、調理後の食品の取り扱いで正しいものはどれか。

1. 調理後の食品は自然に冷めるまで常温で放置してから冷蔵庫に入れるのがよい
2. 調理後の食品はできるだけ早く食べるか、速やかに冷却して冷蔵保存する
3. 前日に作った煮物は、翌日そのまま食べても食中毒のリスクはない
4. 冷蔵庫に入れておけば細菌は完全に死滅するので安心である


<QuizAnswer>

**答え: 2**

調理後の食品は、細菌が増殖しやすい危険温度帯（約10〜60℃）にできるだけ長く置かないことが重要です。作った料理はなるべく早く食べるか、速やかに冷却して冷蔵庫で保存しましょう。常温での長時間放置は細菌増殖の原因になります。また、冷蔵庫は細菌の増殖を遅らせるだけで、完全に死滅させるわけではないため、過信は禁物です。

</QuizAnswer>
