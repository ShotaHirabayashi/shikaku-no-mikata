## 問1

食品表示基準において、生鮮食品は大きく3つに分類されます。「農産物」「水産物」と、もう1つは何ですか？

1. 乳製品
2. 林産物
3. 畜産物
4. 発酵食品


<QuizAnswer>

**答え: 3**

生鮮食品は「農産物」「水産物」「畜産物」の3つに分類されます。農産物は野菜・果物・穀物など、水産物は魚介類・海藻など、畜産物は食肉・卵・生乳などが該当します。林産物や乳製品は生鮮食品の分類区分には含まれません。

</QuizAnswer>

## 問2

次のうち、食品表示基準で「加工食品」に分類されるものはどれですか？

1. 合いびき肉（牛肉と豚肉を混合したもの）
2. 切り身のマグロ
3. 袋詰めのもやし
4. 枝付きのぶどう


<QuizAnswer>

**答え: 1**

合いびき肉は異なる種類の畜産物（牛肉と豚肉）を混合しているため、加工食品に分類されます。食品表示基準では、単に切断・選別・洗浄しただけのものは生鮮食品ですが、異なる種類の食品を混合したものは加工食品となります。切り身のマグロ、もやし、ぶどうはいずれも生鮮食品です。

</QuizAnswer>

## 問3

食品表示基準において、容器包装された加工食品にアレルギー表示が義務づけられている「特定原材料」は、2025年4月以降、何品目ですか？

1. 5品目
2. 7品目
3. 9品目
4. 8品目


<QuizAnswer>

**答え: 4**

2025年4月から「くるみ」が特定原材料に追加され、表示義務のある特定原材料は8品目になりました。8品目は、卵、乳、小麦、えび、かに、落花生（ピーナッツ）、そば、くるみです。これらは発症件数が多い、あるいは重篤な症状を引き起こしやすいことから、表示が義務づけられています。

</QuizAnswer>

## 問4

日本の食品添加物は大きく4種類に分類されます。次のうち、その4種類に含まれないものはどれですか？

1. 指定添加物
2. 合成添加物
3. 既存添加物
4. 天然香料


<QuizAnswer>

**答え: 2**

食品添加物は「指定添加物」「既存添加物」「天然香料」「一般飲食物添加物」の4つに分類されます。「合成添加物」という公式の分類区分はありません。指定添加物は厚生労働大臣が安全性を評価した上で指定したもの、既存添加物は長い食経験がある天然由来のものです。

</QuizAnswer>

## 問5

有機JASマークについて、正しい記述はどれですか？

1. 有機JASマークは農産物にのみ表示できる
2. 有機JASマークがなくても「有機」と表示して販売できる
3. 有機JASマークのデザインは太陽・雲・植物をイメージしている
4. 有機JASの認証は市区町村が行う


<QuizAnswer>

**答え: 3**

有機JASマークは太陽・雲・植物をイメージしたデザインです。有機JASマークは農産物だけでなく、有機加工食品、有機畜産物、有機飼料などにも表示できます。また、有機JASマークがない食品に「有機」「オーガニック」と表示して販売することは法律で禁止されています。認証は登録認証機関が行います。

</QuizAnswer>

## 問6

次のうち、遺伝子組換え食品の表示義務の対象作物に含まれないものはどれですか？

1. 小麦
2. 大豆
3. とうもろこし
4. ばれいしょ


<QuizAnswer>

**答え: 1**

遺伝子組換え表示の対象作物は、大豆、とうもろこし、ばれいしょ、なたね、綿実、アルファルファ、てん菜、パパイヤ、からしなの9作物です。小麦は対象作物に含まれていません。これらの対象作物およびそれを原料とする加工食品33品目に表示義務があります。

</QuizAnswer>

## 問7

加工食品の原材料表示のルールとして、正しいものはどれですか？

1. 原材料はあいうえお順に記載する
2. 添加物は原材料と区別せずに混合して記載する
3. 重量が最も少ない原材料から順に記載する
4. 原材料に占める重量の割合が多い順に記載する


<QuizAnswer>

**答え: 4**

加工食品の原材料は、使用した重量の割合が多い順に記載するルールになっています。つまり、先頭に書かれている原材料がその食品に最も多く含まれているということです。また、食品添加物は原材料と明確に区別して記載することが求められており、スラッシュ（／）で区切るか別欄に表示します。

</QuizAnswer>

## 問8

特定保健用食品（トクホ）について、正しい記述はどれですか？

1. 事業者の届出のみで表示が認められる
2. 国の審査を受けて消費者庁長官の許可が必要である
3. すべての健康食品に自動的に表示される
4. 農林水産省が許可を行う


<QuizAnswer>

**答え: 2**

特定保健用食品（トクホ）は、科学的根拠に基づいて健康への効果を表示することが消費者庁長官の許可により認められた食品です。個別の審査・許可が必要な点が、届出制の「機能性表示食品」との大きな違いです。トクホには、人のシルエットのマークが表示されています。

</QuizAnswer>

## 問9

食品添加物のうち、「保存料」の主な使用目的として正しいものはどれですか？

1. 食品中の微生物の増殖を抑え、保存性を高める
2. 食品に色をつけて見た目をよくする
3. 食品の味や香りを調える
4. 食品の栄養成分を強化する


<QuizAnswer>

**答え: 1**

保存料は、食品中の細菌やカビなどの微生物の増殖を抑制し、食品の保存性を高めることを目的とした食品添加物です。代表的なものにソルビン酸や安息香酸ナトリウムがあります。なお、選択肢2は「着色料」、選択肢3は「調味料・香料」、選択肢4は「強化剤」の役割に該当します。

</QuizAnswer>

## 問10

2025年4月から、アレルギー表示が義務化される「特定原材料」に新たに追加された品目はどれですか？

1. アーモンド
2. カシューナッツ
3. マカダミアナッツ
4. くるみ


<QuizAnswer>

**答え: 4**

2023年3月の食品表示基準改正により、くるみが「特定原材料に準ずるもの（推奨表示）」から「特定原材料（義務表示）」に格上げされ、2年間の経過措置期間を経て2025年4月1日から完全施行されました。アーモンドやカシューナッツ、マカダミアナッツは「特定原材料に準ずるもの」として推奨表示の対象にとどまっています。

</QuizAnswer>
