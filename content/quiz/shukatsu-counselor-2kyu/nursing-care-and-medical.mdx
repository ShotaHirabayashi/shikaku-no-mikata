## 問1

介護保険制度において、第1号被保険者の対象年齢として正しいものはどれか。

1. 40歳以上
2. 60歳以上
3. 65歳以上
4. 70歳以上


<QuizAnswer>

**答え: 3**

介護保険の被保険者は、65歳以上の「第1号被保険者」と、40歳から64歳までの医療保険加入者である「第2号被保険者」の2種類に分けられます。第1号被保険者は原因を問わず要介護認定を受ければサービスを利用できますが、第2号被保険者は加齢に伴う特定疾病が原因の場合のみ利用可能です。

</QuizAnswer>

## 問2

要介護認定の区分として正しい組み合わせはどれか。

1. 要支援1・2、要介護1〜3の合計5段階
2. 要支援1〜3、要介護1〜5の合計8段階
3. 要支援1・2、要介護1〜5の合計7段階
4. 要支援1・2、要介護1〜5の合計7段階に非該当を加えた8段階


<QuizAnswer>

**答え: 3**

要介護認定は「要支援1・2」と「要介護1〜5」の合計7段階に区分されます。要支援と認定された方には介護予防サービス（予防給付）が、要介護と認定された方には居宅サービスや施設サービス（介護給付）が提供されます。なお、いずれにも該当しない場合は「非該当（自立）」と判定されます。

</QuizAnswer>

## 問3

要介護認定の申請後、一次判定で行われるものとして正しいものはどれか。

1. 家庭裁判所の審判による判定
2. 認定調査員による心身の状況調査と主治医意見書に基づくコンピュータ判定
3. 本人の自己申告のみによる判定
4. かかりつけ医の診断書のみによる判定


<QuizAnswer>

**答え: 2**

要介護認定の一次判定は、市町村の認定調査員による心身の状況調査（認定調査）と主治医意見書に基づくコンピュータ判定で行われます。その後、保健・医療・福祉の学識経験者で構成される介護認定審査会による二次判定を経て、最終的な要介護度が決定されます。

</QuizAnswer>

## 問4

地域包括支援センターに配置される専門職の組み合わせとして正しいものはどれか。

1. 医師・看護師・介護福祉士
2. 保健師・社会福祉士・主任介護支援専門員（主任ケアマネジャー）
3. 薬剤師・理学療法士・社会福祉士
4. 医師・社会福祉士・介護福祉士


<QuizAnswer>

**答え: 2**

地域包括支援センターには、保健師・社会福祉士・主任介護支援専門員（主任ケアマネジャー）の3職種が配置されています。この3職種によるチームアプローチで、総合相談支援、介護予防ケアマネジメント、権利擁護、包括的・継続的ケアマネジメント支援の4つの業務を担っています。

</QuizAnswer>

## 問5

地域包括支援センターの業務に含まれないものはどれか。

1. 高齢者の権利擁護に関する相談対応
2. 介護予防ケアマネジメント
3. 要介護認定の最終判定
4. 総合相談支援


<QuizAnswer>

**答え: 3**

地域包括支援センターの主な業務は「総合相談支援」「介護予防ケアマネジメント」「権利擁護」「包括的・継続的ケアマネジメント支援」の4つです。要介護認定の最終判定は介護認定審査会が行うものであり、地域包括支援センターの業務ではありません。ただし、介護保険の申請窓口としての役割は担っています。

</QuizAnswer>

## 問6

認知症の種類のうち、最も患者数の割合が高いものはどれか。

1. 血管性認知症
2. レビー小体型認知症
3. 前頭側頭型認知症
4. アルツハイマー型認知症


<QuizAnswer>

**答え: 4**

認知症全体の約67.6%をアルツハイマー型認知症が占めており、最も割合が高い認知症です。次いで血管性認知症（約19.5%）、レビー小体型認知症（約4.3%）と続きます。アルツハイマー型認知症は、脳にアミロイドβというたんぱく質が蓄積し、神経細胞が破壊されて脳が萎縮することで発症します。

</QuizAnswer>

## 問7

リビングウィルに関する説明として正しいものはどれか。

1. 医師が患者の延命治療方針を決定する文書である
2. 日本では法律によりその効力が保証されている
3. 本人が元気なうちに終末期の医療について意思表示しておく文書である
4. 家族の同意があれば本人の署名がなくても有効である


<QuizAnswer>

**答え: 3**

リビングウィルとは、本人が判断能力のあるうちに、終末期における延命治療の希望や拒否などの医療に関する意思を書面に残しておくものです。日本ではリビングウィルの効力を保証する法律はありませんが、厚生労働省のガイドラインでは本人の意思を尊重するよう記されており、医療現場で重要な判断材料として扱われています。

</QuizAnswer>

## 問8

延命治療に含まれるものとして、適切でないものはどれか。

1. 人工呼吸器の装着
2. 経管栄養（胃ろう等）
3. 心肺蘇生措置
4. 緩和ケアによる痛みの軽減


<QuizAnswer>

**答え: 4**

延命治療とは、回復の見込みがない患者の生命を維持するために行われる医療行為で、人工呼吸器の装着、経管栄養（胃ろう等）、心肺蘇生措置、人工透析などが含まれます。一方、緩和ケアは患者の苦痛を和らげることを目的としたケアであり、延命を目的とする治療とは区別されます。

</QuizAnswer>

## 問9

成年後見制度における法定後見制度の3つの類型として正しいものはどれか。

1. 後見・保佐・補助
2. 後見・補佐・介助
3. 後見・代理・補助
4. 後見・保護・補助


<QuizAnswer>

**答え: 1**

法定後見制度は、本人の判断能力の程度に応じて「後見」「保佐」「補助」の3類型に分かれています。判断能力がほとんどない場合は「後見」、著しく不十分な場合は「保佐」、不十分な場合は「補助」が適用されます。いずれも家庭裁判所が成年後見人等を選任し、本人の権利を法律的に保護します。

</QuizAnswer>

## 問10

成年後見制度のうち、本人にまだ判断能力があるうちに将来に備えてあらかじめ契約を結んでおく制度はどれか。

1. 法定後見制度
2. 任意後見制度
3. 特別後見制度
4. 家族後見制度


<QuizAnswer>

**答え: 2**

成年後見制度は大きく「法定後見制度」と「任意後見制度」の2つに分けられます。任意後見制度は、本人に判断能力があるうちに、将来判断能力が不十分になった場合に備えて、あらかじめ任意後見人となる人やその権限の内容を自分で決めて契約しておく制度です。一方、法定後見制度はすでに判断能力が不十分になった方が対象で、家庭裁判所が後見人等を選任します。

</QuizAnswer>
