## 問1

「終活」という言葉が初めて使われたメディアはどれか。

1. NHKスペシャル
2. 週刊朝日
3. 読売新聞
4. サンデー毎日


<QuizAnswer>

**答え: 2**

「終活」という言葉は、2009年に『週刊朝日』の連載「現代終活事情」で初めて使われました。当時の副編集長・佐々木広人氏が生みの親とされています。就職活動の「就活」をもじって作られた造語です。

</QuizAnswer>

## 問2

「終活」という言葉が誕生したのは何年か。

1. 2007年
2. 2008年
3. 2009年
4. 2010年


<QuizAnswer>

**答え: 3**

「終活」という言葉は2009年に誕生しました。『週刊朝日』で2009年8月から12月にかけて「現代終活事情」という連載が行われたのが始まりです。翌2010年には新語・流行語大賞にノミネートされ、2012年にはトップテンに選出されています。

</QuizAnswer>

## 問3

「終活」という言葉は、何という言葉をもじって作られた造語か。

1. 婚活（婚姻活動）
2. 朝活（朝の活動）
3. 就活（就職活動）
4. 保活（保育園活動）


<QuizAnswer>

**答え: 3**

「終活」は就職活動を意味する「就活」をもじって作られた造語です。「人生の終わりのための活動」を略して「終活」と呼ぶようになりました。「就活」と同じ響きを持たせることで、前向きに取り組む活動というイメージが広まりました。

</QuizAnswer>

## 問4

エンディングノートに関する説明として正しいものはどれか。

1. 法的効力があり、遺言書と同等の効果を持つ
2. 公証役場で作成しなければ無効となる
3. 法的効力はないが、自分の希望や情報を自由に書き残せる
4. 15歳以上でなければ作成できない


<QuizAnswer>

**答え: 3**

エンディングノートには法的効力がありません。遺言書のように書き方や要件が法律で定められているものではなく、形式やルールは自由です。自分の思い、希望、家族へのメッセージ、各種情報などを気軽に書き残すことができるのが特徴です。

</QuizAnswer>

## 問5

終活カウンセラー2級の知識レベルとして正しいものはどれか。

1. 他者の終活を専門的にサポートできるレベル
2. 自分の終活ができるレベル
3. 終活セミナーの講師ができるレベル
4. 終活に関する法的助言ができるレベル


<QuizAnswer>

**答え: 2**

終活カウンセラー2級の知識レベルは「自分の終活ができる」程度とされています。2級では、自分でエンディングノートを書ける知識を身につけ、家族や友人に尋ねられた時に対応できるようになることを目指しています。他者を専門的にサポートするのは上級の役割です。

</QuizAnswer>

## 問6

終活が社会的に広まった背景として、最も適切なものはどれか。

1. インターネットの普及によりSNSでの情報発信が増えたため
2. 少子高齢化の進行や団塊世代の定年退職により老後への不安が高まったため
3. 海外の終活文化が日本に輸入されたため
4. 政府が終活を義務化する法律を制定したため


<QuizAnswer>

**答え: 2**

終活が広まった背景には、日本社会の少子高齢化の加速があります。団塊の世代が定年退職を迎え、年金の給付年齢の引き上げなどが議論される中で、老後を不安に思う人が増えたことが大きな要因です。終活は義務ではなく、自発的に取り組む活動です。

</QuizAnswer>

## 問7

エンディングノートに書く内容として、一般的に含まれないものはどれか。

1. 医療・介護に関する希望
2. 大切な人へのメッセージ
3. 裁判所への申立書
4. 各種アカウントのログイン情報


<QuizAnswer>

**答え: 3**

エンディングノートには、医療・介護の希望、葬儀やお墓の希望、大切な人へのメッセージ、財産の情報、各種アカウント情報など幅広い内容を自由に記載します。裁判所への申立書は法的書類であり、エンディングノートに書く内容ではありません。

</QuizAnswer>

## 問8

「終活」が新語・流行語大賞のトップテンに選出されたのは何年か。

1. 2009年
2. 2010年
3. 2011年
4. 2012年


<QuizAnswer>

**答え: 4**

「終活」は2010年に新語・流行語大賞にノミネートされ、2012年にトップテンに選出されました。2009年の言葉の誕生からわずか数年で社会的に広く認知されるようになり、多くの「終活本」も出版されるなど一大ムーブメントとなりました。

</QuizAnswer>

## 問9

終活において家族とのコミュニケーションが重要とされる理由として、最も適切なものはどれか。

1. 法律上、家族の同意がなければ終活を始められないため
2. 自分の希望を共有し、家族の理解と協力を得ることでトラブルを防げるため
3. エンディングノートは家族が代筆しなければならないため
4. 終活カウンセラー協会が家族との相談を義務づけているため


<QuizAnswer>

**答え: 2**

終活では自分の希望や整理の進捗を家族と共有し、意見を尊重しながら進めることが大切です。コミュニケーションを怠ると、相続や葬儀などをめぐって家族間のトラブルに発展する恐れがあります。終活は法的義務ではなく自発的な活動ですが、家族の理解と協力が円滑な終活の鍵となります。

</QuizAnswer>

## 問10

終活の定義として最も適切なものはどれか。

1. 亡くなった後の葬儀の手配だけを事前に行うこと
2. 財産のすべてを生前に処分すること
3. 人生の終わりを意識して、最期の準備や人生の総括を行う活動のこと
4. 高齢者だけが行う法的な義務のこと


<QuizAnswer>

**答え: 3**

終活とは「人生の終わりのための活動」の略称で、自らの死を意識して人生の最期を迎えるための様々な準備や、人生の総括を行うことを指します。葬儀だけでなく、医療・介護の希望、財産整理、家族へのメッセージなど幅広い領域を含み、年齢に関係なく誰でも自発的に始められる活動です。

</QuizAnswer>
