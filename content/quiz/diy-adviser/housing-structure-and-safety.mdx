## 問1

木造軸組工法（在来工法）において、柱と柱の間に斜めに取り付けて建物の耐震性を高める部材を何というか。

1. 間柱（まばしら）
2. 胴縁（どうぶち）
3. 筋交い（すじかい）
4. 母屋（もや）


<QuizAnswer>

**答え: 3**

筋交い（すじかい）は、柱や梁の間に斜めに交差させて取り付ける部材で、水平方向の力（地震や風圧）に抵抗して建物の耐震性を高める役割があります。木造軸組工法において構造上非常に重要な部材であり、建築基準法でも一定の設置が義務づけられています。

</QuizAnswer>

## 問2

住宅の基礎のうち、建物の底面全体をコンクリートで覆い、面で建物を支える方式を何というか。

1. 独立基礎
2. 布基礎
3. 杭基礎
4. べた基礎


<QuizAnswer>

**答え: 4**

べた基礎は、建物の底面全体を鉄筋コンクリートの盤（スラブ）で覆い、面で荷重を地盤に伝える基礎形式です。布基礎に比べて不同沈下（建物が不均一に沈むこと）に強く、湿気やシロアリの侵入を防ぎやすい利点があります。阪神淡路大震災以降、住宅ではべた基礎が主流となっています。

</QuizAnswer>

## 問3

RC造（鉄筋コンクリート造）の説明として、**誤っているもの**はどれか。

1. 鉄筋は引張力に強く、コンクリートは圧縮力に強い
2. 木造に比べて遮音性に優れている
3. 木造に比べて熱伝導率が低く、断熱性に優れている
4. 木造に比べて工期が長くなる傾向がある


<QuizAnswer>

**答え: 3**

RC造は鉄筋とコンクリートで構成されるため、コンクリートの熱伝導率が高く、木造に比べて断熱性は劣ります。夏は暑く冬は寒くなりやすい傾向があり、気密性が高いことと相まって結露やカビが発生しやすいというデメリットがあります。そのため、RC造住宅では断熱対策が特に重要です。

</QuizAnswer>

## 問4

木造住宅において、基礎の上にアンカーボルトで固定され、柱からの荷重を基礎に伝える水平部材を何というか。

1. 大引（おおびき）
2. 土台（どだい）
3. 根太（ねだ）
4. 桁（けた）


<QuizAnswer>

**答え: 2**

土台は基礎の上に直接アンカーボルトで固定される水平部材で、柱の荷重を基礎に伝える重要な役割を担います。地面に近く湿気やシロアリの被害を受けやすいため、ヒノキやヒバなどの耐久性が高い樹種を用いたり、防腐・防蟻処理を施すのが一般的です。

</QuizAnswer>

## 問5

住宅で結露が発生する主な原因として、最も適切なものはどれか。

1. 室内の気温が外気温より低いこと
2. 暖かく湿った空気が冷たい面に触れて露点温度以下になること
3. 換気を多くしすぎて外気が大量に入ること
4. 壁の断熱材が厚すぎること


<QuizAnswer>

**答え: 2**

結露は、暖かく水蒸気を多く含んだ空気が、窓ガラスや壁などの冷たい面に触れて露点温度以下に冷やされることで、空気中の水蒸気が水滴となって付着する現象です。結露を防ぐには、適切な換気で室内の湿度を下げること、断熱性を高めて室内側の表面温度を下げないことが効果的です。

</QuizAnswer>

## 問6

木造住宅の工法のうち、工場で製作した壁・床・屋根などのパネル（枠組み）を現場で組み立てる北米由来の工法はどれか。

1. 木造軸組工法（在来工法）
2. 丸太組構法（ログハウス）
3. 枠組壁工法（ツーバイフォー工法）
4. 鉄骨軸組工法


<QuizAnswer>

**答え: 3**

枠組壁工法（ツーバイフォー工法）は、2インチ×4インチの規格化された木材と合板で壁・床・屋根のパネルを構成し、面で建物を支える工法です。北米で発展した工法で、耐震性・気密性・断熱性に優れ、施工が比較的均一化しやすいという特徴があります。一方、在来工法は柱と梁の軸組で建物を支える日本の伝統的な工法です。

</QuizAnswer>

## 問7

住宅内での日常事故として最も多い事故の種類はどれか。

1. 転倒・転落
2. 感電
3. ガス中毒
4. 火傷（やけど）のみ


<QuizAnswer>

**答え: 1**

住宅内の事故で最も多いのは転倒・転落事故です。階段、浴室、敷居の段差など、住宅内には転倒リスクのある場所が多く存在します。特に高齢者の転倒事故は重大な怪我につながりやすいため、手すりの設置、段差の解消、滑りにくい床材の使用などの対策が重要です。

</QuizAnswer>

## 問8

住宅の屋根構造において、屋根の最も高い位置に水平に渡される部材を何というか。

1. 垂木（たるき）
2. 棟木（むなぎ）
3. 軒桁（のきげた）
4. 野地板（のじいた）


<QuizAnswer>

**答え: 2**

棟木（むなぎ）は屋根の最頂部に水平に架け渡される部材で、左右の垂木（たるき）の上端を受け止める役割を果たします。垂木は棟木から軒桁に向かって斜めに架ける部材で屋根面を構成し、野地板は垂木の上に張って屋根材の下地となる板です。

</QuizAnswer>

## 問9

住宅の換気方式のうち、給気を機械（換気扇等）で行い、排気は自然排気口から行う方式は第何種換気か。

1. 第1種換気
2. 第2種換気
3. 第3種換気
4. 第4種換気


<QuizAnswer>

**答え: 2**

換気方式は3種類に分類されます。第1種換気は給気・排気ともに機械で行う方式、第2種換気は給気を機械・排気を自然に行う方式、第3種換気は給気を自然・排気を機械で行う方式です。一般住宅では、コストが低く施工が容易な第3種換気が最も多く採用されています。第2種換気は室内が正圧になるためクリーンルームなどに適しています。

</QuizAnswer>

## 問10

木造住宅において、1階と2階を1本の柱で貫く柱を何というか。

1. 管柱（くだばしら）
2. 通し柱（とおしばしら）
3. 束柱（つかばしら）
4. 間柱（まばしら）


<QuizAnswer>

**答え: 2**

通し柱は、土台から軒桁まで1階と2階を途中で切断せずに1本で通す柱のことで、建物の四隅などの構造上重要な位置に配置されます。管柱は各階ごとに区切られた柱、間柱は柱と柱の間に立てて壁の下地を構成する細い柱、束柱は床下などで短く立てて上部を支える柱です。

</QuizAnswer>
