## 問1

水性塗料の特徴として、**正しいもの**はどれか。

1. 希釈にはペイントうすめ液を使用する
2. 乾燥が非常に速く、夏場なら10〜20分で表面が乾く
3. 臭いが少なく、使用後のハケや用具を水で洗うことができる
4. 酸化重合により強靭な塗膜を形成する

<QuizAnswer>

**答え: 3**

水性塗料は水で希釈でき、乾燥前であればハケや容器を水洗いできるのが大きな特徴です。シンナー臭がなく、引火性もないため、屋内作業にも適しています。選択肢1はペイントうすめ液で希釈する油性塗料、選択肢2は乾燥が速いラッカー塗料、選択肢4は油性塗料の硬化メカニズムの説明です。

</QuizAnswer>

## 問2

ラッカー塗料の希釈に使用する溶剤として、正しいものはどれか。

1. 水
2. ペイントうすめ液
3. ラッカーうすめ液
4. アルコール

<QuizAnswer>

**答え: 3**

塗料の希釈には、種類に応じた正しいうすめ液を使う必要があります。水性塗料は水、油性塗料はペイントうすめ液、ラッカー塗料はラッカーうすめ液で希釈します。間違ったうすめ液を使うと、塗料が固まったり分離したりして、正常に塗装できなくなります。

</QuizAnswer>

## 問3

シアノアクリレート系接着剤（瞬間接着剤）の特徴として、**誤っているもの**はどれか。

1. 空気中の水分（湿気）により硬化が始まる
2. 硬化時間が非常に短く、数秒〜数十秒で初期強度が発現する
3. 耐衝撃性・耐熱性に優れ、構造接着に最適である
4. 一液型であり、主剤と硬化剤を混合する必要がない

<QuizAnswer>

**答え: 3**

シアノアクリレート系接着剤（瞬間接着剤）は、空気中の水分により重合・硬化する一液型の接着剤で、数秒で接着できるのが最大の利点です。しかし、耐衝撃性と耐熱性が低いという弱点があり、大きな荷重がかかる構造接着には不向きです。構造接着にはエポキシ系接着剤が適しています。

</QuizAnswer>

## 問4

エポキシ系接着剤の説明として、正しいものはどれか。

1. 一液型で、空気中の湿気により硬化する
2. 主剤と硬化剤を混合して使用する二液型が一般的である
3. 硬化時間は数秒と非常に短い
4. 木材や紙の接着専用で、金属には使用できない

<QuizAnswer>

**答え: 2**

エポキシ系接着剤は、主剤（エポキシ樹脂）と硬化剤を混ぜ合わせることで化学反応が起こり硬化する二液型が一般的です。耐熱性・耐候性・耐薬品性に優れ、金属、ガラス、セラミック、木材など多種多様な材料を強力に接着できます。硬化には数十分〜数時間かかりますが、接着強度は非常に高くなります。

</QuizAnswer>

## 問5

シリコーン系シーリング材の特徴として、**正しいもの**はどれか。

1. 硬化後に上から塗装することができる
2. 耐水性・耐候性に優れ、浴室やキッチンなど水まわりに適する
3. 紫外線に弱いため、屋外使用には必ず上塗りが必要である
4. コンクリートのひび割れ補修に最も適している

<QuizAnswer>

**答え: 2**

シリコーン系シーリング材は耐水性・耐候性・耐熱性・撥水性に優れており、浴室やキッチンなどの水まわりの目地に最も広く使用されています。ただし、硬化後の表面に塗料がのらない（塗装不可）という大きな特徴があるため、塗装が必要な箇所には変成シリコーン系を使用します。

</QuizAnswer>

## 問6

塗装の下地処理で、塗料の密着性を高めるために塗装面を研磨紙で軽く荒らす処理のことを何と呼ぶか。

1. プライマー処理
2. マスキング
3. 足付け（目荒らし）
4. パテ埋め

<QuizAnswer>

**答え: 3**

塗装面をサンドペーパー（研磨紙）で軽く荒らして微細な傷をつけ、塗料の密着性を高める処理を「足付け」または「目荒らし」と呼びます。プライマー処理は下塗り剤の塗布、マスキングは塗装しない部分を保護テープで覆う作業、パテ埋めは凹みや穴を補修材で埋める作業のことです。

</QuizAnswer>

## 問7

変成シリコーン系シーリング材の特徴として、**誤っているもの**はどれか。

1. 硬化後に上から塗装することができる
2. 外壁の目地に広く使用される
3. シリコーン系と比べて耐久性がやや劣る
4. 水まわりには一切使用できない

<QuizAnswer>

**答え: 4**

変成シリコーン系シーリング材は、塗装が可能で外壁の目地に広く使用される汎用性の高いシーリング材です。水まわりにも使用可能ですが、シリコーン系と比べると耐久性がやや劣ります。「水まわりに一切使用できない」は誤りで、シリコーン系ほどではないものの水まわりにも対応できる万能タイプです。

</QuizAnswer>

## 問8

木工用接着剤（酢酸ビニル樹脂系エマルジョン接着剤）の特徴として、**正しいもの**はどれか。

1. 耐水性に優れ、屋外の木材接着に最適である
2. 水分が蒸発・吸収されることで硬化する
3. 二液を混合して使用する
4. 金属やガラスの接着にも適している

<QuizAnswer>

**答え: 2**

木工用接着剤（酢酸ビニル樹脂系エマルジョン接着剤）は、接着剤中の水分が木材に吸収されたり蒸発したりすることで樹脂が固まり接着する仕組みです。一液型で混合の手間がなく、安全性も高いため初心者にも扱いやすい接着剤です。ただし耐水性が低いため、屋外や水がかかる場所には不向きです。

</QuizAnswer>

## 問9

塗装作業における安全対策として、**最も不適切なもの**はどれか。

1. 油性塗料やラッカー塗料を使用する際は、十分な換気を行う
2. ラッカー塗料の使用時は、火気の近くでの作業を避ける
3. スプレー塗装を行う際は、防塵マスクを着用する
4. 油性塗料の乾燥を早めるため、密閉した室内でストーブを使用する

<QuizAnswer>

**答え: 4**

油性塗料やラッカー塗料には有機溶剤が含まれており、引火性があります。密閉した室内でストーブなどの火気を使用すると、引火や爆発の危険があるだけでなく、有機溶剤の蒸気を吸い込んで中毒を起こす恐れもあります。塗装作業中は十分な換気を確保し、火気厳禁を徹底することが安全の基本です。

</QuizAnswer>

## 問10

パテ材（充填剤）に関する説明として、**正しいもの**はどれか。

1. パテは乾燥・硬化後にサンドペーパーで研磨して平滑に仕上げることができる
2. パテは充填後すぐに上塗り塗装をしなければならない
3. エポキシパテは一液型で、チューブから出してそのまま使用する
4. パテは弾力性があるため、目地の動きに追従するシーリング材として使える

<QuizAnswer>

**答え: 1**

パテは壁面の穴や凹み、ひび割れなどを埋めて平滑にするための充填剤で、硬化後にサンドペーパーで研磨して表面を滑らかに仕上げることができます。エポキシパテは主剤と硬化剤を練り合わせて使う二液型が一般的です。パテは硬化後に硬くなるため、目地の動きに追従するシーリング材とは用途が異なります。

</QuizAnswer>
