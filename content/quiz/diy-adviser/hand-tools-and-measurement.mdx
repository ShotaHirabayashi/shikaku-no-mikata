## 問1

両刃のこぎりには、木目に沿って切る刃と木目に直角に切る刃がある。木目に対して直角に繊維を断ち切るように切断する刃はどれか。

1. 縦挽き刃
2. 横挽き刃
3. あさり刃
4. 導突刃


<QuizAnswer>

**答え: 2**

横挽き刃は木目に対して直角に繊維を断ち切るように切断するための刃です。一方、縦挽き刃は木目に沿って繊維を削るように切断します。横挽き刃は一つひとつの刃が切り出しナイフのような形状をしており、裏表で刃の向きが交互に並んでいるのが特徴です。

</QuizAnswer>

## 問2

のこぎりの刃を左右に交互に振り分けることを何というか。

1. 目立て
2. すくい
3. あさり
4. ひずみ取り


<QuizAnswer>

**答え: 3**

のこぎりの刃を左右に交互に振り分けることを「あさり」といいます。あさりをつけることで、切り溝（挽き溝）の幅が刃の厚さより広くなり、切断時の摩擦抵抗が減って切りやすくなります。また、おがくずの排出もスムーズになります。

</QuizAnswer>

## 問3

かんな（鉋）の裏金（裏刃）の主な役割として、最も適切なものはどれか。

1. 刃の切れ味を長持ちさせる
2. 削りくずを細かく砕く
3. 逆目（さかめ）を防止する
4. 台の反りを矯正する


<QuizAnswer>

**答え: 3**

裏金（裏刃・押さえ金）の主な役割は、木材を削った際に起こる逆目（さかめ）を防止することです。裏金は直接材料を削るものではなく、刃の直前で削りくずを折り曲げることで、繊維がめくれ上がる逆目ぼれを抑えます。二枚刃鉋は裏金によって切削角を大きくし、逆目を防ぐ構造になっています。

</QuizAnswer>

## 問4

かんな（鉋）の台の各部名称で、削る材料に接する面を何というか。

1. 上端（うわば）
2. 下端（したば）
3. 台頭（だいがしら）
4. 台尻（だいじり）


<QuizAnswer>

**答え: 2**

かんなの台で、削る材料に接する面を「下端（したば）」といいます。上端（うわば）は上面のことで、台頭は刃が出る方（進行方向の前側）、台尻は後ろ側を指します。下端の平面精度がかんなの仕上がりに大きく影響するため、「台直し」と呼ばれる調整が重要です。

</QuizAnswer>

## 問5

さしがね（曲尺）の裏面に刻まれている「丸目」の目盛りの用途として、正しいものはどれか。

1. 正方形の対角線の長さから一辺の長さを求める
2. 円の直径を測るとその円周の長さがわかる
3. 板材の体積を計算するために使う
4. ほぞ穴の深さを測定するために使う


<QuizAnswer>

**答え: 2**

丸目は円周率（約3.14）を掛けた目盛りになっており、円の直径を丸目で測ると、その読み取り値がそのまま円周の長さになります。一方、正方形の対角線から一辺の長さを求めるのは「角目」の役割です。角目は√2（約1.414）倍の目盛りになっています。

</QuizAnswer>

## 問6

のみ（鑿）を使ってほぞ穴を掘る際、のみの刃の裏（平らな面）を向ける方向として正しいものはどれか。

1. 常に手前（自分側）に向ける
2. 仕上げ面（残す側）に向ける
3. 削り取る側（不要な側）に向ける
4. のみの裏表はどちらを向けても同じである


<QuizAnswer>

**答え: 2**

のみの刃の裏（平らな面・裏押し面）は仕上げ面（残す側）に向けて使います。裏面が平らであるため、仕上げ面に当てることで正確で平滑な面を作ることができます。しのぎ面（斜めに研いである面）を仕上げ面に向けると、刃が材料に食い込んでしまい、きれいな仕上がりになりません。

</QuizAnswer>

## 問7

水平器（水準器）の気泡管で水平を確認する際、正しい判断方法はどれか。

1. 気泡が左端にあれば水平である
2. 気泡が標線の中央にあれば水平である
3. 気泡が見えなくなれば水平である
4. 液体の色が変われば水平である


<QuizAnswer>

**答え: 2**

気泡管水平器では、ガラス管内の気泡が標線（黒い基準線）の中央に位置するとき、測定面が水平であることを示します。気泡が中央より左にずれていれば左側が低く、右にずれていれば右側が低いことを意味します。水平器は安定した面にぴったり密着させて使うことが正確な測定の基本です。

</QuizAnswer>

## 問8

胴付きのこぎり（導突のこぎり）の特徴として、正しいものはどれか。

1. 刃が太く丸太の切断に適している
2. 背に背金が付いており、薄い刃で精密な切断ができる
3. 刃が曲線状で曲線切りに適している
4. 両面に刃があり縦挽き・横挽き兼用である


<QuizAnswer>

**答え: 2**

胴付きのこぎり（導突のこぎり）は、非常に薄い刃の背の部分に背金（金属の補強材）が付いている精密切断用ののこぎりです。細かい目の刃と背金による安定性により、ほぞの胴付き部分やダボの切断など、正確で細かい切断作業に適しています。背金があるため刃の深さ以上は切り込めないという制約があります。

</QuizAnswer>

## 問9

プラスドライバーのサイズ表記で、一般家庭のDIYで最もよく使われるサイズはどれか。

1. No.0
2. No.1
3. No.2
4. No.4


<QuizAnswer>

**答え: 3**

プラスドライバーのサイズはNo.0〜No.4まであり、数字が大きいほど先端が太くなります。No.2は最も汎用性が高く、一般家庭のDIYや家具の組み立て、家電のねじなど幅広い場面で使用されます。No.1は小さなねじ用、No.3は大きなねじ用で、No.2を基本として揃えるのがDIYの基本です。

</QuizAnswer>

## 問10

やすり（鑢）を使って金属を削る際の正しい使い方として、最も適切なものはどれか。

1. 往復させて素早く削る
2. 押す時に力を入れて削り、引く時は力を抜く
3. 引く時に力を入れて削り、押す時は力を抜く
4. やすりを斜め45度に傾けて円を描くように動かす


<QuizAnswer>

**答え: 2**

やすり（鑢）は「押しがけ」が基本で、押す時（前方へのストローク）に力を入れて切削し、引く時は力を抜いて戻します。やすりの目は押す方向に切れるように作られているため、引く時に力を入れると目詰まりの原因になり、やすりの寿命も短くなります。材料にやすりを密着させ、まっすぐ平行に押すことが正確な加工のコツです。

</QuizAnswer>
