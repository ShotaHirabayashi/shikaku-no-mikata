## 問1

電動ドリルドライバーとインパクトドライバーの違いについて、正しい記述はどれか。

1. インパクトドライバーは回転方向に打撃を加えてねじを締める工具である
2. ドリルドライバーはインパクトドライバーよりも締め付けトルクが大きい
3. インパクトドライバーにはクラッチ機能が標準装備されている
4. ドリルドライバーは主にコンクリートへの穴あけに使用する


<QuizAnswer>

**答え: 1**

インパクトドライバーは、モーターの回転に加えて回転方向に打撃（インパクト）を与えることで、強力なねじ締めを行う電動工具です。一方、ドリルドライバーはクラッチ機能により締め付けトルクを調整でき、繊細な作業に向いています。コンクリートへの穴あけには振動ドリルやハンマードリルが適しています。

</QuizAnswer>

## 問2

電動丸のこの使用時に発生する「キックバック」について、正しい記述はどれか。

1. キックバックとは、切断中に刃が前方に飛び出す現象である
2. キックバックは、回転する刃が材料に挟まれることで丸のこが後方に跳ね返る現象である
3. キックバックを防ぐには、刃の出し代をできるだけ大きくするとよい
4. キックバックは低速回転時にのみ発生する現象である


<QuizAnswer>

**答え: 2**

キックバックとは、回転するノコ刃が材料に挟まれたり噛んだりして摩擦抵抗が増大し、丸のこが突然後方に跳ね返る危険な現象です。防止するには、定規やガイドを使って真っ直ぐ切断すること、刃の出し代は材料の厚さ＋3mm程度に抑えること、材料をしっかり固定することが重要です。

</QuizAnswer>

## 問3

電動丸のこを安全に使用するための注意点として、**誤っている**ものはどれか。

1. 切断時は丸のこの真後ろに体を置かないようにする
2. 安全カバー（保護カバー）は円滑に動くことを確認してから使用する
3. 軍手を着用して作業すると繊維が巻き込まれる危険があるため、素手か革手袋で作業する
4. 作業効率を上げるため、安全カバーは固定して開いた状態にしておくとよい


<QuizAnswer>

**答え: 4**

安全カバー（保護カバー）は絶対に固定して開いた状態にしてはいけません。カバーは切断時に自動で開き、刃が材料から離れると自動で閉じる仕組みになっており、不使用時に刃を覆って安全を確保します。軍手は繊維が回転刃に巻き込まれる危険があるため、素手か手に密着する革手袋を使用します。

</QuizAnswer>

## 問4

ジグソーの特徴と使い方について、正しい記述はどれか。

1. ジグソーは刃を回転させて切断する電動工具である
2. ジグソーは直線切断専用の電動工具で、曲線切断はできない
3. ジグソーはブレードを上下に往復運動させて切断する電動工具である
4. ジグソーのブレードは木材専用で、金属やプラスチックの切断には対応していない


<QuizAnswer>

**答え: 3**

ジグソーは細長いブレード（刃）を上下に往復運動させて材料を切断する電動工具です。直線だけでなく曲線切断も得意としており、丸のこでは難しい曲線や切り抜き加工に適しています。またブレードを交換することで、木材だけでなくアルミや塩ビなどさまざまな素材の切断が可能です。

</QuizAnswer>

## 問5

電動サンダーの種類と用途について、正しい記述はどれか。

1. オービタルサンダーは、サンドペーパーを円形に回転させて研磨する
2. サンダーは主に木材の切断作業に使用する電動工具である
3. ランダムサンダーは、回転運動と偏心運動を組み合わせて研磨ムラを少なくする
4. サンダーで研磨する際は、木目に対して直角方向に動かすのが基本である


<QuizAnswer>

**答え: 3**

ランダムサンダーは、回転運動と偏心（ランダム）運動を組み合わせることで、同じ場所を繰り返し研磨することなく均一な仕上がりが得られます。オービタルサンダーは細かい楕円運動（振動）で研磨するタイプです。サンダーは切断ではなく研磨・表面仕上げ用の工具であり、研磨は木目に沿って行うのが基本です。

</QuizAnswer>

## 問6

トリマーとルーターの違いについて、正しい記述はどれか。

1. トリマーはルーターより大型でパワーが強く、深い溝切りに適している
2. ルーターはトリマーの約3倍のパワーがあり、固い材質や深い溝切りに適している
3. トリマーとルーターは全く同じ工具で、名称が異なるだけである
4. ルーターはDIY初心者が最初に購入すべき電動工具として推奨されている


<QuizAnswer>

**答え: 2**

ルーターはトリマーと同じく木材の切削加工を行う電動工具ですが、トリマーの約3倍のパワーがあり、大型で重量もあります。固い材質への加工や深い溝切り、大型家具の製作に適しています。DIY初心者には、扱いやすく小型軽量なトリマーから始めることが推奨されています。

</QuizAnswer>

## 問7

住宅の電気配線に関する記述として、正しいものはどれか。

1. DIYで自宅のコンセント増設や配線工事を行うのに資格は不要である
2. 第二種電気工事士の資格があれば、一般住宅の低圧（600V以下）の電気工事を行うことができる
3. 照明器具のスイッチ交換は、資格がなくても誰でも行うことができる
4. 電気工事士法に違反しても、行政指導のみで罰則はない


<QuizAnswer>

**答え: 2**

電気工事士法により、住宅の電気配線工事（コンセントの増設・移設、スイッチの交換など）を行うには電気工事士の資格が必要です。第二種電気工事士の資格を取得すれば、一般住宅や店舗などの低圧（600V以下）で受電する設備の工事を行うことができます。無資格で電気工事を行うと、懲役刑や罰金刑が科される可能性があります。

</QuizAnswer>

## 問8

給排水設備における「排水トラップ」の役割として、正しいものはどれか。

1. 排水の流速を速めて詰まりを防止する装置である
2. 排水管内に水を溜めて、下水からの悪臭やガスの逆流を防ぐ装置である
3. 排水管の水圧を高めて排水効率を上げる装置である
4. 大きなゴミを取り除くためのフィルターである


<QuizAnswer>

**答え: 2**

排水トラップは、排水管の途中に水（封水）を溜めておくことで、下水管からの悪臭や有害ガス、害虫が室内に侵入するのを防ぐ装置です。洗面台下のS字型やP字型の管がその代表例です。封水が蒸発して切れると（破封）、悪臭が逆流するため、長期間使用しない場合は定期的に水を流すことが大切です。

</QuizAnswer>

## 問9

家庭のガス設備に関する記述として、正しいものはどれか。

1. ガス配管の接続工事はDIYで自由に行うことができる
2. ガスコンロの交換は、ガス栓との接続部分も含めてすべて無資格で行える
3. ガス機器の設置・配管工事には専門の資格が必要であり、DIYでは行ってはならない
4. 都市ガスとプロパンガスは同じ器具で使用できるため、互換性がある


<QuizAnswer>

**答え: 3**

ガス配管の接続工事やガス機器の設置工事は、ガス事業法や液化石油ガス法により、有資格者（ガス可とう管接続工事監督者など）でなければ行うことができません。ガス漏れは爆発や一酸化炭素中毒など重大な事故につながるため、必ず専門業者に依頼する必要があります。また、都市ガスとプロパンガスでは供給圧力や熱量が異なるため、器具の互換性はありません。

</QuizAnswer>

## 問10

振動ドリルに関する記述として、正しいものはどれか。

1. 振動ドリルは回転方向に打撃を加えて穴をあける工具である
2. 振動ドリルは主に木材のねじ締め作業に使用する
3. 振動ドリルは軸方向（垂直方向）に振動を加えながら回転し、コンクリートやモルタルの穴あけに適している
4. 振動ドリルはインパクトドライバーよりも小型・軽量である


<QuizAnswer>

**答え: 3**

振動ドリルは、通常の回転に加えて軸方向（垂直方向）に振動を与えることで、コンクリートやモルタル、タイルなどの硬くてもろい材質への穴あけに適した電動工具です。回転方向に打撃を加えるのはインパクトドライバーの特徴です。振動ドリルはハイパワーなモーターを搭載しているため、インパクトドライバーと比べて大型で重量があります。

</QuizAnswer>
