## 問1

針葉樹の木材の一般的な特徴として、正しいものはどれか。

1. 重くて硬く、キズがつきにくい
2. 成長が遅く、緻密な組織を持つ
3. 軽くて柔らかく、加工しやすい
4. 美しい木目を持ち、主に家具に使われる


<QuizAnswer>

**答え: 3**

針葉樹はソフトウッドとも呼ばれ、軽くて柔らかく加工しやすいのが特徴です。まっすぐ育つため長い木材が取りやすく、柱や梁などの構造材として多く使われます。選択肢1・2・4は広葉樹（ハードウッド）の特徴です。

</QuizAnswer>

## 問2

木材の「繊維飽和点」における含水率は、乾量基準でおよそ何％か。

1. 約5％
2. 約15％
3. 約30％
4. 約50％


<QuizAnswer>

**答え: 3**

繊維飽和点とは、木材中の自由水がすべて抜け、細胞壁内の結合水のみで満たされている状態のことで、その含水率は乾量基準で約30％です。この点を下回ると木材の収縮・反り・割れが始まり、同時に強度が増加していきます。

</QuizAnswer>

## 問3

合板の製造方法として正しいものはどれか。

1. 木材を繊維状にほぐし、接着剤で固めて板にする
2. 木材の薄板（単板）を繊維方向が交互になるように奇数枚重ねて接着する
3. 木材のチップを加熱・圧縮して板状にする
4. 木材の小片をランダムに並べて接着・圧縮する

<QuizAnswer>

**答え: 2**

合板は原木をかつら剥きにした薄板（単板・ベニヤ）を、繊維方向が直交するように奇数枚重ねて接着した木質材料です。繊維方向を交互にすることで、反りやねじれに強く、方向による強度差が小さくなるという利点があります。選択肢1はMDF、選択肢3はパーティクルボード、選択肢4はOSBの説明です。

</QuizAnswer>

## 問4

MDF（中密度繊維板）の特徴として、誤っているものはどれか。

1. 木材を繊維状にほぐして接着剤で固めた板材である
2. 表面が滑らかで塗装に適している
3. 水や湿気に強く、水回りにそのまま使用できる
4. 木材の節や方向性がなく均質である


<QuizAnswer>

**答え: 3**

MDFは木材繊維を接着剤で固めた均質な板材で、表面が滑らかで塗装や加工に適しています。しかし、水や湿気には弱く、吸水すると膨張・変形しやすいため、水回りにそのまま使用するのは不適切です。防水処理を施すか、水回りには別の材料を選ぶ必要があります。

</QuizAnswer>

## 問5

日本の住宅で最も広く使用されている壁紙（クロス）の種類はどれか。

1. 紙クロス
2. 織物クロス
3. ビニールクロス
4. 珪藻土クロス


<QuizAnswer>

**答え: 3**

ビニールクロスは、塩化ビニール樹脂のシートに紙などを裏打ちした壁紙で、日本の住宅壁紙の約9割を占めるとされています。価格が安く、色やデザインが豊富で、耐久性や清掃性にも優れていることが普及の理由です。ただし、通気性は紙クロスや織物クロスに比べて劣ります。

</QuizAnswer>

## 問6

クッションフロアの説明として、正しいものはどれか。

1. 天然木の単板を表面に貼った木質系床材である
2. 塩化ビニル系のシート状床材で、発泡層によりクッション性がある
3. 陶磁器製のタイルを並べた床材である
4. カーペットの一種で、裏面にゴムを貼った床材である


<QuizAnswer>

**答え: 2**

クッションフロアは塩化ビニル系のシート状床材で、CFシートとも呼ばれます。表面のクリア層・プリント層、中間の発泡層、裏面の不織布という4層構造になっており、発泡層により柔らかい踏み心地が得られます。水に強いため、キッチンや洗面所など水回りに多く使用されます。

</QuizAnswer>

## 問7

パーティクルボードの説明として、正しいものはどれか。

1. 木材の薄板を交互に重ねて接着した板材である
2. 木材を繊維レベルまで細かくほぐして固めた板材である
3. 木材の小片やチップを接着剤とともに加熱・圧縮して作った板材である
4. 天然木の無垢材を幅はぎして作った板材である


<QuizAnswer>

**答え: 3**

パーティクルボードは、木材の小片（パーティクル）やチップを接着剤とともに加熱・圧縮して製造した板材です。無垢材に比べて安価で、反りや割れが少なく、大きなサイズも製造可能です。選択肢1は合板、選択肢2はMDF（中密度繊維板）の説明にあたります。

</QuizAnswer>

## 問8

タイルの材質のうち、最も吸水率が低く、外装や水回りに適しているものはどれか。

1. 陶器質タイル
2. せっ器質タイル
3. テラコッタタイル
4. 磁器質タイル


<QuizAnswer>

**答え: 4**

磁器質タイルは石英や長石などを約1,250℃の高温で焼成したもので、緻密で硬質な素地を持ち、吸水率は3.0％以下と最も低いタイルです。吸水性が低いため変形しにくく、凍害にも強いことから、外装や水回りの壁・床に広く用いられます。陶器質タイルは吸水率が高く、主に内装用です。

</QuizAnswer>

## 問9

木材を自然乾燥させた「気乾状態」の含水率は、乾量基準でおよそ何％か。

1. 約5％
2. 約15％
3. 約30％
4. 約45％


<QuizAnswer>

**答え: 2**

気乾状態とは、木材を自然の大気中で十分に乾燥させた状態のことで、含水率は乾量基準で約15％に安定します。この状態は木材が周囲の温湿度と平衡に達した状態であり、DIYや建築で使用する木材はこの程度の含水率に乾燥されていることが望ましいとされています。

</QuizAnswer>

## 問10

住宅用断熱材として広く使われているグラスウールの原料と特徴の組み合わせとして、正しいものはどれか。

1. 鉱石（玄武岩等）が原料で、耐火性に特に優れる
2. ガラス繊維が原料で、安価で防音性にも優れる
3. 発泡ポリスチレンが原料で、水に強く軽量である
4. セルロース繊維が原料で、調湿性に優れる


<QuizAnswer>

**答え: 2**

グラスウールはガラスを繊維状にした断熱材で、流通量が多く安価に入手できるため、住宅用断熱材として最も普及しています。耐火性やシロアリへの耐性があり、防音性にも優れています。選択肢1はロックウール、選択肢3は発泡スチロール系断熱材、選択肢4はセルロースファイバーの説明です。

</QuizAnswer>
