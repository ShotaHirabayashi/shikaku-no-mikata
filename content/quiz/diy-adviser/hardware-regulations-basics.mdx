## 問1

ねじの規格において、メートルねじのサイズを表す記号として正しいものはどれか。

1. W
2. M
3. UNC
4. G


<QuizAnswer>

**答え: 2**

メートルねじはミリメートル単位で設計された国際規格のねじで、サイズ表記の頭に「M」が付きます（例：M6は直径6mm）。「W」はウィットねじ、「UNC」はユニファイ並目ねじ、「G」は管用平行ねじの記号です。

</QuizAnswer>

## 問2

蝶番（丁番）に関する説明として、最も適切なものはどれか。

1. 蝶番は引き戸専用の金物である
2. 平蝶番は2枚の金属板を1本の回転軸で連結した最も基本的な蝶番である
3. 旗蝶番は扉の上部と下部に別々に取り付ける蝶番である
4. ピボットヒンジは軸部分の上下に旗のような取付板が付いた蝶番である


<QuizAnswer>

**答え: 2**

平蝶番は2枚の金属板（プレート）を1本の回転軸に連結させた最もオーソドックスな蝶番で、開き戸や箱の蓋などに広く使われます。旗蝶番は軸部分の上下に旗のような取付板が付いたタイプ、ピボットヒンジは扉の上部と下部に別々に取り付けるタイプです。蝶番は開き戸用の金物であり、引き戸には使いません。

</QuizAnswer>

## 問3

JISマークとJASマークに関する説明として、正しいものはどれか。

1. JISは日本農林規格の略称であり、木材製品に付けられる
2. JASは日本産業規格の略称であり、工業製品全般に付けられる
3. JISは工業製品に関する国家規格であり、JASは農林水産品や木質建材に関する規格である
4. JISマークもJASマークも、どちらも経済産業省のみが管轄している


<QuizAnswer>

**答え: 3**

JIS（Japanese Industrial Standards）は工業製品について定められた日本の国家規格で、コンクリートやガラス、ボード類などが対象です。JAS（Japanese Agricultural Standards）は農林水産品や木質建材（製材、合板、集成材等）に関する規格です。JISは経済産業省、JASは農林水産省がそれぞれ所管しています。

</QuizAnswer>

## 問4

ウィットねじに関する説明として、正しいものはどれか。

1. アメリカで開発され、ねじ山の角度は60°である
2. イギリスで考案され、ねじ山の角度は55°である
3. ドイツで開発され、ねじ山の角度は45°である
4. 日本独自の規格で、ねじ山の角度は60°である


<QuizAnswer>

**答え: 2**

ウィットねじはイギリスのジョセフ・ウィットウォースが考案したインチねじで、ねじ山の角度が55°であることが特徴です。サイズ表記には「W」が付きます。JIS規格としては1968年に廃止されましたが、水道管や建設関係では現在も使われています。なお、ユニファイねじやメートルねじのねじ山角度は60°です。

</QuizAnswer>

## 問5

電気工事士法に関する説明として、正しいものはどれか。

1. DIYアドバイザーの資格があれば屋内配線工事を行うことができる
2. 電気工事士の資格がなくても、自宅の屋内配線工事は自由に行える
3. 電気工事士の資格を持たない者は、屋内配線工事を行ってはならない
4. コンセントの増設は資格不要だが、スイッチの交換には電気工事士の資格が必要である


<QuizAnswer>

**答え: 3**

電気工事士法により、屋内配線工事やコンセント・スイッチの増設・交換などの電気工事は、電気工事士の資格を持つ者でなければ行うことができません。DIYアドバイザーの重要な役割の一つは、このような法律で有資格者以外の作業が禁じられている分野について、お客様に正しく説明し法令遵守を啓発することです。

</QuizAnswer>

## 問6

DIYアドバイザー資格制度の歴史について、正しいものはどれか。

1. 1990年に国土交通省の認定資格として創設された
2. 1983年に通商産業省（現・経済産業省）認定の資格として創設された
3. 2000年に厚生労働省の国家資格として創設された
4. 1975年に農林水産省の認定資格として創設された


<QuizAnswer>

**答え: 2**

DIYアドバイザー資格制度は1983年（昭和58年）に、当時の通商産業省（現・経済産業省）認定の資格として創設されました。試験の実施と資格の認定は、社団法人日本DIY協会（現・日本DIY・ホームセンター協会）が行っています。第1回試験は1984年に実施されました。

</QuizAnswer>

## 問7

釘の打ち方に関する説明として、最も適切なものはどれか。

1. 釘は常に木目に対して平行に打つのが正しい
2. 板の端に釘を打つ際は、下穴をあけると木材の割れを防止できる
3. 釘の長さは、打ち付ける板の厚さと同じ長さを選ぶのが基本である
4. 釘を斜めに打つことは強度低下の原因となるため、絶対に避けるべきである


<QuizAnswer>

**答え: 2**

板の端近くに釘を打つと木材が割れやすいため、あらかじめキリなどで下穴をあけておくと割れを防止できます。釘の長さは一般に打ち付ける板の厚さの2.5〜3倍が目安です。また、釘を斜めに打つ「斜め打ち」は、引き抜き強度を高める技法として意図的に使われることがあります。

</QuizAnswer>

## 問8

建築基準法に関する記述のうち、DIYとの関連で正しいものはどれか。

1. 自宅の内装をDIYで変更する場合、建築基準法の規制は一切適用されない
2. 10㎡以下の増築であれば、防火地域・準防火地域以外では建築確認申請が不要な場合がある
3. DIYで行う工事には消防法は適用されない
4. 建築基準法は新築のみに適用され、リフォームやDIYには適用されない


<QuizAnswer>

**答え: 2**

建築基準法では、防火地域・準防火地域以外において10㎡以下の増築・改築・移転の場合、建築確認申請が不要となる場合があります。ただし、DIYであっても建築基準法や消防法、都道府県の火災予防条例などの法規制は適用されます。内装制限など安全に関わる規定も遵守する必要があります。

</QuizAnswer>

## 問9

ボルトの強度区分に関する説明として、正しいものはどれか。

1. 強度区分は「4.8」のように小数点で区切った数値で表され、JIS規格で定められている
2. 強度区分の数値が小さいほど、引張強さが大きいことを示す
3. ボルトの強度区分は材質に関係なく、すべて同じである
4. 強度区分はボルトにのみ適用され、ナットには強度区分がない


<QuizAnswer>

**答え: 1**

ボルトの強度区分はJIS B 1051で規定されており、「4.8」「8.8」「10.9」のように小数点で区切った数値で表されます。小数点の前の数値は引張強さ（×100でMPa）、後の数値は降伏点比を示します。数値が大きいほど強度が高く、ナットにもJIS B 1052で強度区分が定められています。

</QuizAnswer>

## 問10

ホームセンターにおけるDIYアドバイザーの役割として、最も適切なものはどれか。

1. 顧客に代わって工事を請け負い、施工まで行う
2. 商品知識や作業方法の助言に加え、法規制で資格が必要な工事について正しく説明する
3. 販売促進のため、資格が必要な工事でも顧客に自分で行うよう勧める
4. 商品の販売のみに専念し、使い方の説明は行わない


<QuizAnswer>

**答え: 2**

DIYアドバイザーは、住まいの手入れや補修、改善などについて、商品知識や作業方法を助言する専門家です。電気工事やガス工事のように有資格者以外の作業が法律で禁じられている分野については、その旨を顧客に正しく説明し、法令遵守を啓発する重要な役割も担っています。工事の請負や無資格工事の推奨は適切ではありません。

</QuizAnswer>
