## 問1

ベルヌーイの定理に関する記述として正しいものはどれか。

1. 流体の速度が速いところでは圧力が高くなる
2. 流体の速度と圧力には関係がない
3. 流体の速度が速いところでは圧力が低くなる
4. 流体の速度が遅いところでは圧力が低くなる


<QuizAnswer>

**答え: 3**

ベルヌーイの定理は、連続した流体の中では流速が速いところほど圧力が低くなり、流速が遅いところほど圧力が高くなるという関係を示すものです。この原理は飛行機の翼やプロペラブレードが揚力・推力を発生させる基本的なメカニズムの説明に用いられます。

</QuizAnswer>

## 問2

固定翼機とマルチコプターの飛行特性の違いに関する記述として正しいものはどれか。

1. 固定翼機は前進速度がなければ揚力を得られないが、マルチコプターはホバリング（空中静止）が可能である
2. マルチコプターは固定翼機と同様に、前進しなければ揚力を得ることができない
3. 固定翼機はマルチコプターと同様にホバリングが可能である
4. マルチコプターは固定翼機に比べて、一般に長時間の飛行に適している


<QuizAnswer>

**答え: 1**

固定翼機は翼の前進運動による空気の流れで揚力を得るため、一定以上の速度を維持する必要があります。一方、マルチコプターはローターの回転によって揚力を発生させるため、前進速度がなくてもホバリング（空中静止）が可能です。ただし、マルチコプターはエネルギー消費が大きく、一般に固定翼機より飛行時間は短くなります。

</QuizAnswer>

## 問3

マルチコプターのプロペラ（ブレード）が推力を発生させる仕組みとして正しいものはどれか。

1. ブレードの回転によって生じる遠心力が推力となる
2. ブレードの表面摩擦が空気を押し出すことで推力が生まれる
3. ブレードの重量が落下することで反動が推力となる
4. ブレードの翼型断面が回転することで上下面に圧力差が生じ、推力が発生する


<QuizAnswer>

**答え: 4**

プロペラのブレードは翼型（エアフォイル）の断面形状を持っており、回転することで上面（低圧面）と下面（高圧面）に圧力差が生じます。この圧力差がブレードの回転面に垂直な方向への推力となります。原理的には固定翼機の翼が揚力を発生させるのと同じベルヌーイの定理に基づいています。

</QuizAnswer>

## 問4

マルチコプターが垂直に上昇するための操作として正しいものはどれか。

1. 前方のローターのみ回転数を上げる
2. すべてのローターの回転数を均等に上げる
3. 後方のローターのみ回転数を上げる
4. すべてのローターの回転数を均等に下げる


<QuizAnswer>

**答え: 2**

マルチコプターが姿勢を変えずに垂直上昇するには、すべてのローターの回転数を均等に上げて、総推力が機体の重力を上回る状態にする必要があります。特定のローターだけ回転数を変えると機体が傾いてしまい、垂直上昇ではなく水平方向の移動が生じます。逆に、均等に回転数を下げると下降します。

</QuizAnswer>

## 問5

マルチコプターが前進飛行する仕組みとして正しいものはどれか。

1. すべてのローターの回転数を均等に上げて推力を増やす
2. 前方のローターの回転数を上げ、後方を下げて機体を後方に傾ける
3. 後方のローターの回転数を上げ、前方を下げて機体を前方に傾ける
4. 左右のローターの回転数に差をつける


<QuizAnswer>

**答え: 3**

マルチコプターが前進するには、後方のローターの回転数を上げて前方のローターの回転数を下げることで、機体を前方（ノーズダウン方向）に傾けます。これにより推力の方向が斜め前方になり、推力の水平成分が前進する力となります。左右の回転数差をつけると左右方向の移動（ロール）になります。

</QuizAnswer>

## 問6

クアッドコプター（4枚ローター機）がプロペラの反トルクの影響を打ち消す方法として正しいものはどれか。

1. 隣り合うローターの回転方向を交互に逆向きにする
2. すべてのローターを同じ方向に回転させる
3. テールローターを装備してトルクを相殺する
4. ローターの回転数を常に変動させて打ち消す


<QuizAnswer>

**答え: 1**

クアッドコプターでは、4つのローターのうち対角に位置する2つを時計回り、残りの2つを反時計回りに回転させることで、各ローターが生じる反トルクを互いに打ち消し合います。これによりテールローターなしでも機体のヨー方向の安定を保てます。ヘリコプターがテールローターで反トルクを相殺するのとは異なるマルチコプター特有の方式です。

</QuizAnswer>

## 問7

対気速度40km/hで飛行しているドローンが、10km/hの向かい風を受けている場合、対地速度として正しいものはどれか。

1. 50km/h
2. 40km/h
3. 10km/h
4. 30km/h


<QuizAnswer>

**答え: 4**

対気速度は周囲の空気に対する速度であり、対地速度は地面に対する速度です。向かい風の場合、対地速度は「対気速度 − 風速」で求められるため、40km/h − 10km/h ＝ 30km/hとなります。逆に追い風の場合は対地速度が対気速度より大きくなります。風の影響を理解することは安全な飛行計画に不可欠です。

</QuizAnswer>

## 問8

地面効果（グランドエフェクト）に関する記述として正しいものはどれか。

1. 高度が高いほど地面効果は大きくなる
2. 地面近くを飛行するとき、揚力が増加する現象である
3. 地面近くを飛行するとき、揚力が減少する現象である
4. 風が強いときにのみ発生する現象である


<QuizAnswer>

**答え: 2**

地面効果（グランドエフェクト）とは、航空機が地面に近い高度で飛行する際に、ローターや翼が生み出す下向きの気流（ダウンウォッシュ）が地面に遮られ、翼端渦や誘導抗力が減少することで揚力が増加する現象です。一般にローター直径程度以下の高度で顕著に現れ、高度が上がるにつれて効果は減少します。離着陸時には地面効果の影響を考慮した操縦が必要です。

</QuizAnswer>

## 問9

回転翼機がホバリングから水平飛行に移行する際に生じる「転移揚力」の説明として正しいものはどれか。

1. 水平移動によりローターの効率が低下し、揚力が減少する現象
2. ホバリング中にのみ発生する揚力増加の現象
3. 水平移動によりローターディスクを通過する空気量が増え、ローター効率が上がり揚力が増加する現象
4. 急上昇時に一時的に揚力が失われる現象


<QuizAnswer>

**答え: 3**

転移揚力とは、ホバリング状態から水平方向に加速していくと、ローターディスクを通過する空気の量が増加し、誘導速度が減少してローターの効率が向上することで揚力が増加する現象です。転移揚力が得られると、ホバリング時より少ないパワーで機体を支えることが可能になります。これはヘリコプターやマルチコプターに共通する飛行特性です。

</QuizAnswer>

## 問10

オートローテーションの説明として正しいものはどれか。

1. エンジン停止時に、下方から流入する空気によってローターを回転させながら降下する飛行状態
2. エンジン出力でローターの回転数を自動的に上げる飛行方式
3. プロペラの回転数を自動制御で一定に保つ機能
4. 離陸時にローターを高速回転させてから上昇する技法


<QuizAnswer>

**答え: 1**

オートローテーションとは、エンジンが停止した際にメインローターがエンジンの駆動力ではなく、降下によって下方から上方へ流入する空気の力でローターを回転させ続ける飛行状態です。ヘリコプターではこの原理を利用して、エンジン故障時にも安全に降下・着陸することが可能です。マルチコプターでは構造上オートローテーションによる着陸は困難であり、フェールセーフ機能による対応が一般的です。

</QuizAnswer>
