## 問1

マルチコプターに搭載されるジャイロセンサーが検出するものとして正しいものはどれか。

1. 機体の位置情報
2. 機体の角速度（回転の速さ）
3. 気圧の変化による高度
4. バッテリーの電圧変動

<QuizAnswer>

**答え: 2**

ジャイロセンサーは機体の角速度（回転の速さ）を検出するセンサーです。機体がどの方向にどれだけの速さで回転しているかを測定し、姿勢制御に必要な情報をフライトコントローラーに提供します。位置情報はGNSS、高度は気圧センサー、電圧はバッテリー管理システムがそれぞれ担当しています。

</QuizAnswer>

## 問2

リポバッテリー（リチウムポリマーバッテリー）の1セルあたりの公称電圧として正しいものはどれか。

1. 1.5V
2. 3.0V
3. 3.7V
4. 4.2V

<QuizAnswer>

**答え: 3**

リポバッテリーの1セルあたりの公称電圧は3.7Vです。満充電時の電圧は約4.2V、放電終止電圧は約3.0Vとされています。1.5Vは一般的な乾電池の電圧であり、リポバッテリーとは異なります。セル電圧が3.0Vを下回ると過放電となり、バッテリーが膨張したり使用不能になる危険があります。

</QuizAnswer>

## 問3

送信機（プロポ）のモード2において、スロットル（上昇・下降）を操作するスティックはどれか。

1. 左スティックの上下
2. 右スティックの上下
3. 左スティックの左右
4. 右スティックの左右

<QuizAnswer>

**答え: 1**

モード2では、左スティックの上下でスロットル（上昇・下降）を、左スティックの左右でラダー（機首の左右回転）を操作します。右スティックの上下はエレベーター（前後移動）、右スティックの左右はエルロン（左右移動）です。モード2は世界的に最も普及している操作方式で、モード1ではスロットルとエレベーターの配置が左右逆になります。

</QuizAnswer>

## 問4

ドローンのフェールセーフ機能の説明として最も適切なものはどれか。

1. 飛行中の機体速度を自動的に最大にする機能
2. バッテリーの充電を自動で行う機能
3. 操縦者の技量を自動で判定する機能
4. 異常発生時に自動的に危険を回避する動作を行う機能

<QuizAnswer>

**答え: 4**

フェールセーフ機能とは、飛行中にドローンに異常が発生した場合に、自動的に危険を回避する動作を行う安全機能です。代表的な例として、送信機との電波が途切れた場合やバッテリー残量が低下した場合に、自動帰還（RTH：Return To Home）や自動着陸が作動します。安全な飛行を確保するために非常に重要な機能であり、飛行前に正しく設定しておく必要があります。

</QuizAnswer>

## 問5

GNSS（全球測位衛星システム）で3次元の位置（緯度・経度・高度）を特定するために最低限必要な衛星の数はいくつか。

1. 2つ
2. 4つ
3. 6つ
4. 8つ

<QuizAnswer>

**答え: 2**

GNSSで3次元の位置（緯度・経度・高度）を特定するためには、最低4つの衛星からの信号を受信する必要があります。3つの衛星で2次元の位置（緯度・経度）が算出でき、4つ目の衛星により高度の算出と受信機の時計誤差の補正が可能になります。受信できる衛星の数が多いほど測位精度は向上します。

</QuizAnswer>

## 問6

リポバッテリーを長期間使用せずに保管する場合、適切な充電量として正しいものはどれか。

1. 40〜60%程度
2. 0%（完全放電状態）
3. 100%（満充電状態）
4. 80〜90%程度

<QuizAnswer>

**答え: 1**

リポバッテリーを長期保管する場合は、充電量を40〜60%程度に調整して保管するのが適切です。完全放電状態（0%）での保管は過放電を引き起こしバッテリーを劣化させ、満充電状態（100%）での保管もセルの劣化を早めます。充電器のストレージモード機能を使えば、自動的に適切な充電量に調整できます。保管場所は直射日光を避け、22〜28℃程度の涼しい環境が理想です。

</QuizAnswer>

## 問7

6つのローター（回転翼）を持つヘキサコプターの構造上の利点として最も適切なものはどれか。

1. 機体が最も軽量で持ち運びやすい
2. バッテリーの消費量が最も少ない
3. 1つのローターが故障しても飛行を継続できる可能性がある
4. 操縦が最も簡単である

<QuizAnswer>

**答え: 3**

ヘキサコプターは6つのローターを持つため、1つのローターが故障しても残り5つのローターで揚力を確保し、飛行を継続できる可能性があるという冗長性（リダンダンシー）を持ちます。これは安全性の面で大きな利点です。ただし、ローター数が多い分モーターも増えるため、機体重量は増加し、バッテリー消費量もクアッドコプターより多くなります。

</QuizAnswer>

## 問8

TEM（Threat and Error Management：脅威とエラーの管理）における「脅威（Threat）」の説明として最も適切なものはどれか。

1. 操縦者が行った操作上のミスや判断の誤り
2. 機体のハードウェアの経年劣化
3. 飛行後に作成する報告書の記載事項
4. 操縦の複雑さを増大させる外的な要因や状況

<QuizAnswer>

**答え: 4**

TEMにおける「脅威（Threat）」とは、急な天候の変化、障害物の出現、他の航空機の接近など、操縦の複雑さを増大させる外的な要因や状況のことを指します。一方、操縦者の操作ミスや判断の誤りは「エラー（Error）」に分類されます。TEMは脅威を事前に予測し、エラーを管理することで、望ましくない状態（Undesired State）を未然に防ぐための安全管理の考え方です。

</QuizAnswer>

## 問9

マルチコプターの姿勢制御において、加速度センサーが果たす主な役割として正しいものはどれか。

1. GPSの信号を受信して位置を特定する
2. モーターの回転数を直接制御する
3. 重力方向を基準にして機体の傾きを検出する
4. 送信機からの電波強度を測定する

<QuizAnswer>

**答え: 3**

加速度センサーは重力加速度を基準として機体にかかる加速度を測定し、機体がどの方向にどれだけ傾いているかを検出します。ジャイロセンサーが角速度（回転の速さ）を検出するのに対し、加速度センサーは傾き（姿勢角）の情報を提供します。両センサーの情報を組み合わせることで、フライトコントローラーは正確な姿勢制御を実現しています。

</QuizAnswer>

## 問10

フライトプラン（飛行計画）を作成する際に確認すべき事項として、最も優先度が低いものはどれか。

1. 飛行空域の気象条件と風速
2. バッテリーの残量と想定される飛行可能時間
3. 機体の購入価格
4. 飛行経路上の障害物や周囲の環境

<QuizAnswer>

**答え: 3**

フライトプランを作成する際には、飛行空域の気象条件、バッテリー残量と飛行可能時間、飛行経路上の障害物の確認などが重要な確認事項です。機体の購入価格は飛行の安全性や計画に直接関係しないため、フライトプラン作成時の確認事項としては優先度が低いといえます。その他にも、離着陸地点の選定、緊急時の対応手順、飛行禁止空域の確認なども重要な事項です。

</QuizAnswer>
