## 問1

航空法において「無人航空機」として規制の対象となるのは、機体本体とバッテリーの合計重量がどれ以上のものか。

1. 25g以上
2. 50g以上
3. 100g以上
4. 200g以上


<QuizAnswer>

**答え: 3**

航空法第2条第22項により、機体本体の重量とバッテリーの重量の合計が100g以上のものが「無人航空機」と定義されます。2022年の法改正により、従来の200g以上から100g以上に引き下げられました。100g未満のものは「模型航空機」として扱われ、航空法上の無人航空機の規制は適用されません。

</QuizAnswer>

## 問2

航空法で定められた無人航空機の飛行禁止空域に該当しないものはどれか。

1. 空港等の周辺の空域
2. 河川敷の上空
3. 地表または水面から150m以上の高さの空域
4. 人口集中地区（DID地区）の上空


<QuizAnswer>

**答え: 2**

航空法で定められた飛行禁止空域は、（1）空港等の周辺の空域、（2）緊急用務空域、（3）地表または水面から150m以上の高さの空域、（4）人口集中地区（DID地区）の上空の4つです。河川敷の上空は航空法上の飛行禁止空域には指定されていません。ただし、自治体の条例等により飛行が制限される場合はあります。

</QuizAnswer>

## 問3

航空法において、無人航空機を飛行させる際に国土交通大臣の「承認」が必要となる飛行方法に該当しないものはどれか。

1. 日中に目視内で飛行させる
2. 夜間に飛行させる
3. 人又は物件から30m未満の距離で飛行させる
4. 催し場所の上空で飛行させる


<QuizAnswer>

**答え: 1**

日中に目視内で飛行させることは、航空法が定める通常の飛行方法であり、承認は不要です。承認が必要となるのは、（1）夜間飛行、（2）目視外飛行、（3）人又は物件から30m未満の距離での飛行、（4）催し場所の上空での飛行、（5）危険物の輸送、（6）物件投下の6つの飛行方法です。

</QuizAnswer>

## 問4

無人航空機の機体登録制度が施行されたのはいつか。

1. 2020年6月20日
2. 2021年6月20日
3. 2021年12月20日
4. 2022年6月20日


<QuizAnswer>

**答え: 4**

無人航空機の機体登録制度は2022年6月20日に施行されました。これにより、100g以上の無人航空機は国土交通省への登録が義務付けられ、登録されていない無人航空機を飛行させることはできなくなりました。登録記号を機体に表示し、原則としてリモートID機能を搭載する必要があります。

</QuizAnswer>

## 問5

無人航空機のリモートID機能の説明として、最も適切なものはどれか。

1. 操縦者の個人情報をインターネット経由で国土交通省に送信する機能
2. 機体の識別情報を電波で遠隔発信する機能
3. 機体の飛行映像をリアルタイムで地上に送信する機能
4. 操縦者のスマートフォンにGPS位置を表示する機能


<QuizAnswer>

**答え: 2**

リモートID機能とは、無人航空機の登録記号や製造番号などの識別情報を電波で遠隔発信するための仕組みです。自動車のナンバープレートのように、飛行中の無人航空機を識別できる「空のナンバープレート」としての役割を果たします。これにより、飛行中の機体が登録済みかどうかを地上から確認することが可能になります。

</QuizAnswer>

## 問6

日本国内でドローンに使用する無線設備に必要な「技適マーク」は、どの法律に基づく制度か。

1. 航空法
2. 個人情報保護法
3. 電波法
4. 小型無人機等飛行禁止法


<QuizAnswer>

**答え: 3**

技適マーク（技術基準適合証明等のマーク）は電波法に基づく制度です。日本国内で無線通信を行う機器は、電波法に定める技術基準に適合している必要があり、技適マークのない送信機やドローンを使用すると電波法違反となります。海外から個人輸入したドローンなどは技適マークがない場合があるため注意が必要です。

</QuizAnswer>

## 問7

小型無人機等飛行禁止法により、対象施設の敷地およびその周囲おおむね何メートル以内の上空での飛行が禁止されているか。

1. 300m
2. 500m
3. 1,000m
4. 1,500m


<QuizAnswer>

**答え: 1**

小型無人機等飛行禁止法では、国会議事堂、内閣総理大臣官邸、最高裁判所、皇居、対象防衛関係施設、対象原子力事業所などの重要施設の敷地およびその周囲おおむね300m以内の上空における小型無人機等の飛行が禁止されています。この法律は2015年の官邸ドローン事件をきっかけに制定されました。

</QuizAnswer>

## 問8

航空法に基づく無人航空機の飛行ルールに違反し、許可なく飛行禁止空域で飛行させた場合の罰則として正しいものはどれか。

1. 30万円以下の罰金
2. 6か月以下の懲役または30万円以下の罰金
3. 50万円以下の罰金のみ
4. 1年以下の懲役または50万円以下の罰金


<QuizAnswer>

**答え: 4**

航空法に違反して許可なく飛行禁止空域で無人航空機を飛行させた場合、1年以下の懲役または50万円以下の罰金が科されます。機体登録をせずに飛行させた場合も同様の罰則が適用されます。航空法の遵守は操縦者の重要な責任であり、違反した場合は刑事罰の対象となることを理解しておく必要があります。

</QuizAnswer>

## 問9

日本国内で一般的な民生用ドローンに使用されている2.4GHz帯の電波を利用する場合、無線の免許は必要か。

1. アマチュア無線技士4級以上の免許が必要
2. 技適マークのある機器であれば免許は不要
3. 第三級陸上特殊無線技士以上の免許が必要
4. 総務省への届出が必要


<QuizAnswer>

**答え: 2**

2.4GHz帯は一般的な民生用ドローンで広く使用されている周波数帯で、技適マークのある機器であれば無線免許は不要です。一方、5.7GHz帯や5.8GHz帯を使用する場合は、ホビー用途ではアマチュア無線技士4級以上、業務用途では第三級陸上特殊無線技士以上の免許が必要となります。

</QuizAnswer>

## 問10

航空法において、無人航空機を飛行させる場所や方法にかかわらず、すべての飛行で遵守しなければならない事項はどれか。

1. 飛行計画を国土交通大臣に提出すること
2. 第三者の上空を飛行しないこと
3. アルコール又は薬物の影響下で飛行させないこと
4. 補助者を必ず配置すること


<QuizAnswer>

**答え: 3**

航空法では、飛行空域や飛行方法にかかわらず、すべての飛行において遵守すべき事項が定められており、「アルコール又は薬物等の影響により正常な飛行ができないおそれがある間は飛行させないこと」はその一つです。その他にも、飛行前の機体点検や、航空機や他の無人航空機との衝突を予防するための注意義務などが遵守事項として定められています。

</QuizAnswer>
