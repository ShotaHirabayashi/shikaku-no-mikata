## 問1

対流圏における気温逓減率（高度が上がるにつれて気温が低下する割合）として、最も適切なものはどれか。

1. 100mにつき約0.35℃低下する
2. 100mにつき約0.50℃低下する
3. 100mにつき約0.65℃低下する
4. 100mにつき約1.00℃低下する


<QuizAnswer>

**答え: 3**

対流圏における平均的な気温逓減率は、高度が100m上がるごとに約0.65℃低下するとされています。これは国際標準大気（ISA）で定められた値であり、航空気象の基本知識として重要です。ただし、実際の気温逓減率は気象条件や地域によって変動することがあります。

</QuizAnswer>

## 問2

地球の大気は高度によっていくつかの層に分かれている。雲や雨などの天気現象が起こり、対流が活発に行われる最も地表に近い層の名称と、その上端のおおよその高さの組み合わせとして正しいものはどれか。

1. 成層圏・約50km
2. 対流圏・約11km
3. 中間圏・約80km
4. 対流圏・約50km


<QuizAnswer>

**答え: 2**

地表から約11km（緯度によって8〜16km程度の差がある）までの大気の層を「対流圏」といい、雲の発生、降水、風など私たちが日常経験する天気現象のほとんどがこの層で起こります。対流圏の上には成層圏があり、そこではオゾン層が紫外線を吸収しています。ドローンは対流圏の下層で飛行するため、この層の気象現象を理解することが重要です。

</QuizAnswer>

## 問3

ビューフォート風力階級（気象庁風力階級）は、風力をいくつの段階に分類しているか。

1. 10段階（風力0〜9）
2. 11段階（風力0〜10）
3. 12段階（風力0〜11）
4. 13段階（風力0〜12）


<QuizAnswer>

**答え: 4**

ビューフォート風力階級は、1805年にイギリス海軍のフランシス・ビューフォート（ボーフォート）提督が提唱したもので、風力0（平穏）から風力12（ハリケーン）までの13段階で風の強さを分類します。1964年に世界気象機関（WMO）の標準尺度として採用され、日本の気象庁風力階級もこれと同一の内容です。

</QuizAnswer>

## 問4

昼間、海岸付近で海から陸に向かって吹く風を何というか。

1. 海風
2. 陸風
3. 谷風
4. 山風


<QuizAnswer>

**答え: 1**

昼間は陸地が海よりも早く暖まるため、陸上の空気が上昇し、その補填として海から陸に向かって風が吹きます。これを「海風」といいます。逆に夜間は陸地が海より早く冷えるため、陸から海に向かって「陸風」が吹きます。この海風と陸風を合わせて「海陸風」と呼び、局地風の代表例です。

</QuizAnswer>

## 問5

十種雲形のうち、「入道雲」や「かみなり雲」とも呼ばれ、下層から上層（高さ1万メートル以上）にまで発達することがある雲はどれか。

1. 乱層雲
2. 高積雲
3. 積乱雲
4. 巻積雲


<QuizAnswer>

**答え: 3**

積乱雲は「入道雲」や「かみなり雲」とも呼ばれ、積雲がさらに発達して巨大な塔のように高く盛り上がった雲です。分類上は下層雲に属しますが、上部は1万2000m以上の上層にまで達することがあり、雷・突風・豪雨・降雹などの激しい気象現象を引き起こします。ドローン飛行にとって最も危険な雲の一つです。

</QuizAnswer>

## 問6

山岳地帯において、夜間に山の斜面に沿って冷たい空気が山頂から谷に向かって吹き下ろす風を何というか。

1. 谷風
2. 山風
3. 海風
4. フェーン


<QuizAnswer>

**答え: 2**

夜間は山の斜面が放射冷却により冷え、斜面に接した空気も冷やされて重くなり、山頂から谷に向かって吹き下ろします。これを「山風」といいます。逆に昼間は日射で斜面が暖められ、空気が谷から山に向かって這い上がる「谷風」が吹きます。山谷風は山岳地帯でのドローン飛行に影響を与える重要な局地風です。

</QuizAnswer>

## 問7

大気中のある区間で風速や風向が急激に変化する現象を何というか。

1. ダウンウォッシュ
2. タービュランス
3. サーマル
4. ウインドシア


<QuizAnswer>

**答え: 4**

ウインドシア（Wind Shear）とは、大気中のある2点間で風速や風向が急激に変化する現象のことです。ウインドシアは積乱雲の近くや前線付近、山岳の風下側などで発生しやすく、ドローンや航空機の飛行に大きな危険をもたらします。特に離着陸時にウインドシアに遭遇すると、急激な揚力変化により制御不能になるおそれがあります。

</QuizAnswer>

## 問8

高度と気圧の関係について、正しい記述はどれか。

1. 高度が高くなるほど気圧は高くなる
2. 高度が変わっても気圧は一定である
3. 高度が高くなるほど気圧は低くなる
4. 高度と気圧の間に一定の関係はない


<QuizAnswer>

**答え: 3**

気圧は、その地点より上にある大気の重さによって生じる圧力です。高度が上がるほど上に乗っている空気の量が減るため、気圧は低くなります。海面付近の標準気圧は約1013hPa（ヘクトパスカル）で、高度が上がるにつれて気圧は低下します。気圧の低下はドローンのプロペラ推力にも影響し、高高度では揚力が低下する要因となります。

</QuizAnswer>

## 問9

国土交通省の無人航空機飛行マニュアルにおいて、飛行を控えるべきとされている風速の基準として最も適切なものはどれか。

1. 風速3m/s以上
2. 風速5m/s以上
3. 風速8m/s以上
4. 風速10m/s以上


<QuizAnswer>

**答え: 2**

国土交通省航空局の無人航空機飛行マニュアルでは、「風速5m/s以上の状態では飛行させない」ことが明記されています。風速5m/sはビューフォート風力階級で風力3（軟風）に相当し、木の葉や小枝が絶えず動く程度の風です。ただし、機体の性能や操縦者の技量によっては、より厳しい基準を設けることが推奨されます。

</QuizAnswer>

## 問10

十種雲形において、「上層雲」に分類される雲の組み合わせとして正しいものはどれか。

1. 層雲・層積雲・積雲
2. 巻雲・巻層雲・巻積雲
3. 高積雲・高層雲・乱層雲
4. 積雲・積乱雲・乱層雲


<QuizAnswer>

**答え: 2**

十種雲形では、雲を発生する高度によって上層雲・中層雲・下層雲に分類します。上層雲（高度約5,000〜13,000m）には巻雲（けんうん）・巻層雲（けんそううん）・巻積雲（けんせきうん）の3種が含まれます。選択肢3は中層雲、選択肢1と4は下層雲の組み合わせです。上層雲はすべて名前に「巻」がつくことが覚えるポイントです。

</QuizAnswer>
