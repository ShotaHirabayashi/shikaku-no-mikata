## 問1

マルチコプターが空中の一定位置で静止飛行し続けることを何というか。

1. グライディング
2. クルージング
3. ホバリング
4. ストール


<QuizAnswer>

**答え: 3**

ホバリングとは、マルチコプターやヘリコプターなどの回転翼機が空中の一点に留まり、静止した状態を保つ飛行のことです。スロットル・エレベーター・エルロンなどを精緻に調整することで実現されます。ドローン操縦における最も基本的な技術の一つです。

</QuizAnswer>

## 問2

マルチコプターにおいて、機首を上下に傾ける回転運動を何というか。

1. ピッチ
2. ヨー
3. ロール
4. スロットル


<QuizAnswer>

**答え: 1**

ピッチとは、機体の左右方向を軸として機首を上下に傾ける回転運動のことです。マルチコプターではピッチ操作により機体が前後に傾き、前進・後退の移動が行われます。水平状態からの傾き角度をピッチ角と呼びます。

</QuizAnswer>

## 問3

機体を上から見たとき、機首の向きを水平面内で左右に回転させる動きを何というか。

1. ピッチ
2. ロール
3. エレベーター
4. ヨー


<QuizAnswer>

**答え: 4**

ヨーとは、機体の上下方向（垂直軸）を軸として、機首の向きを水平面内で左右に回転させる動きのことです。マルチコプターでは対角線上のローターの回転数を変化させることでヨー方向の回転を制御します。ヨー角は方位角とも呼ばれます。

</QuizAnswer>

## 問4

4つの回転翼（ローター）を持つマルチコプターの名称として正しいものはどれか。

1. トライコプター
2. クアッドコプター
3. ヘキサコプター
4. オクトコプター


<QuizAnswer>

**答え: 2**

クアッドコプターは4つのローターを持つマルチコプターで、「クアッド（quad）」は「4」を意味します。マルチコプターの中で最も主流のタイプであり、操作がしやすく安定した姿勢制御が可能です。トライ（3）、ヘキサ（6）、オクト（8）はそれぞれローター数を表す接頭語です。

</QuizAnswer>

## 問5

国際単位系（SI）において、速さを表す単位として正しいものはどれか。

1. km/h
2. kn（ノット）
3. m/s
4. mph


<QuizAnswer>

**答え: 3**

SI単位系では、速さは「m/s（メートル毎秒）」で表されます。km/hやノット、mphは日常的に使われることがありますが、いずれもSI単位ではありません。ドローン検定ではSI単位系の理解が求められるため、物理量を正しいSI単位で表すことが重要です。

</QuizAnswer>

## 問6

マルチコプターの操作において、機体を左右に傾ける操舵を何というか。

1. エルロン
2. ラダー
3. エレベーター
4. スロットル


<QuizAnswer>

**答え: 1**

エルロンは機体を左右に傾ける操舵のことで、ロール方向の制御に対応します。エレベーターは前後の傾き（ピッチ）、ラダーは機首の左右回転（ヨー）、スロットルは上昇・下降の制御をそれぞれ担当します。これらは固定翼機の操舵面の名称に由来しています。

</QuizAnswer>

## 問7

航空用語で「ストール」とはどのような状態を指すか。

1. 空中で静止している状態
2. 高速で水平飛行している状態
3. 着陸態勢に入った状態
4. 失速した状態


<QuizAnswer>

**答え: 4**

ストール（stall）とは「失速」を意味する航空用語です。翼やブレードの迎角が限界を超え、揚力が急激に低下して飛行を維持できなくなる状態を指します。空中で静止している状態はホバリングと呼ばれ、ストールとは異なります。

</QuizAnswer>

## 問8

6つの回転翼（ローター）を持つマルチコプターの名称として正しいものはどれか。

1. ペンタコプター
2. ヘキサコプター
3. オクトコプター
4. クアッドコプター


<QuizAnswer>

**答え: 2**

ヘキサコプターは6つのローターを持つマルチコプターで、「ヘキサ（hexa）」はギリシャ語で「6」を意味します。クアッドコプター（4ローター）に比べてより高い安定性と搭載能力を持ち、重い機材の搭載や風の強い環境での飛行に適しています。

</QuizAnswer>

## 問9

国際単位系（SI）の7つの基本単位のうち、質量の単位はどれか。

1. g（グラム）
2. N（ニュートン）
3. lb（ポンド）
4. kg（キログラム）


<QuizAnswer>

**答え: 4**

SI基本単位において、質量の単位はkg（キログラム）です。g（グラム）はSI接頭語を用いた派生的な表記であり、基本単位ではありません。N（ニュートン）は力の組立単位、lb（ポンド）はヤード・ポンド法の単位であり、いずれもSI基本単位には含まれません。

</QuizAnswer>

## 問10

機首（ノーズ）を常に円の中心方向に向けたまま円を描くように飛行する技法を何というか。

1. バンクターン
2. パイロンターン
3. ノーズインサークル
4. スパイラルフライト


<QuizAnswer>

**答え: 3**

ノーズインサークルとは、機首（ノーズ）を常に旋回の中心に向けた状態で円軌道を描く飛行技法です。被写体を中心に置いて周囲を旋回しながら撮影する際に用いられる空撮テクニックの一つです。エルロン・ラダー・エレベーターを同時に操作する必要があり、ドローン操縦の基本的な応用技術として知られています。

</QuizAnswer>
