## 問1

紫外線は波長の長さによって3種類に分類される。このうち、波長が最も長く、肌の真皮にまで到達して光老化の主な原因となる紫外線はどれか。

1. UVB
2. UVC
3. UVA
4. 赤外線


<QuizAnswer>

**答え: 3**

紫外線はUVA（波長320〜400nm）、UVB（波長280〜320nm）、UVC（波長100〜280nm）の3種類に分類されます。UVAは最も波長が長く、肌の真皮まで到達してコラーゲンやエラスチンを変性させ、シワ・たるみなどの光老化を引き起こします。UVBは主に表皮に影響し、UVCはオゾン層で吸収されるため地上にはほとんど届きません。

</QuizAnswer>

## 問2

紫外線B波（UVB）が肌に与える影響として、最も適切なものはどれか。

1. 真皮のコラーゲンを分解してシワを作る
2. 表皮に炎症を起こし、肌を赤くする（サンバーン）
3. メラニンを酸化させて肌を即座に黒くする
4. オゾン層に吸収されるため肌には届かない


<QuizAnswer>

**答え: 2**

UVBは波長が短く、主に表皮に強い影響を与えます。短時間でも肌が赤く炎症を起こすサンバーン（日焼けによる炎症反応）の原因となります。真皮まで到達してコラーゲンを変性させるのは主にUVAの作用であり、オゾン層で吸収されるのはUVCです。

</QuizAnswer>

## 問3

紫外線C波（UVC）に関する説明として、正しいものはどれか。

1. 窓ガラスを通過して室内にも届く
2. 3種類の紫外線の中で最も波長が長い
3. 肌の真皮まで到達しやすい
4. オゾン層によって吸収され、地表にはほとんど届かない


<QuizAnswer>

**答え: 4**

UVCは波長100〜280nmで、3種類の紫外線の中で最も波長が短くエネルギーが強いですが、大気圏上空のオゾン層によってほぼ完全に吸収されるため、通常は地表に届きません。窓ガラスを通過するのはUVAの特徴であり、波長が最も長いのもUVAです。

</QuizAnswer>

## 問4

日焼け止めに表示されるSPFについて、正しい説明はどれか。

1. UVAの防止効果を「＋」の数で表したもの
2. 紫外線をカットする成分の濃度を示す数値
3. UVBによるサンバーンをどれくらい長い時間防止できるかを示す数値
4. 日焼け止めの耐水性を示す指標


<QuizAnswer>

**答え: 3**

SPFは「Sun Protection Factor」の略で、主にUVBによるサンバーン（赤くなる日焼け）の防止効果を表す数値です。何も塗らない場合に比べて、UVBによる炎症をどれくらい長い時間防止できるかを示します。UVAの防止効果を「＋」の数で表すのはPAであり、SPFとは異なる指標です。

</QuizAnswer>

## 問5

日焼け止めに表示されるPAの「＋」の数は最大いくつまであるか。

1. 4つ（PA++++）
2. 3つ（PA+++）
3. 5つ（PA+++++）
4. 6つ（PA++++++）


<QuizAnswer>

**答え: 1**

PAは「Protection Grade of UVA」の略で、UVAの防止効果を表します。PA+（効果がある）、PA++（かなりある）、PA+++（非常にある）、PA++++（極めて高い）の4段階で表示されます。2013年に従来の3段階からPA++++が追加され、現在は4段階が最大です。

</QuizAnswer>

## 問6

肌の老化のうち、紫外線による「光老化」が占める割合として、一般的にいわれている数値はどれか。

1. 約50%
2. 約80%
3. 約30%
4. 約95%


<QuizAnswer>

**答え: 2**

肌の老化の約80%は紫外線による「光老化」が原因とされています。加齢による自然老化は約20%程度にすぎず、紫外線を長年浴び続けることで蓄積されたダメージがシミ・シワ・たるみなどの老化現象を引き起こします。そのため、日頃からの紫外線対策が肌老化の予防に非常に重要です。

</QuizAnswer>

## 問7

紫外線を浴びることで体内に発生し、細胞を攻撃して肌の老化を促進する物質はどれか。

1. コラーゲン
2. メラニン
3. ヒアルロン酸
4. 活性酸素


<QuizAnswer>

**答え: 4**

紫外線を浴びると体内で活性酸素が大量に発生します。活性酸素は細胞を酸化させてダメージを与え、コラーゲンやエラスチンの分解を促進し、シワ・たるみ・シミなどの老化現象を引き起こします。メラニンは紫外線から肌を守るために生成される色素であり、コラーゲンやヒアルロン酸は肌のうるおいや弾力を保つ成分です。

</QuizAnswer>

## 問8

女性ホルモンのエストロゲン（卵胞ホルモン）が減少すると、肌にどのような影響が現れるか。

1. 皮脂の過剰分泌によるニキビの増加
2. コラーゲンの生成が抑制され、肌が乾燥しやすくなる
3. メラニンの生成が完全に止まる
4. 表皮が厚くなり、肌がゴワつく


<QuizAnswer>

**答え: 2**

エストロゲンには肌のうるおいや弾力を保つ作用があり、コラーゲンの生成を促進する働きがあります。更年期などでエストロゲンが減少すると、コラーゲンの生成が抑制されて肌は乾燥傾向になり、シワやたるみといったエイジングサインが現れやすくなります。エストロゲンは20代でピークを迎え、30代後半から徐々に減少し始めます。

</QuizAnswer>

## 問9

肌の劣化を促進する内的要因として、喫煙が肌に与える影響の説明で正しいものはどれか。

1. 体内のビタミンCを大量に消費し、活性酸素を増加させる
2. 肌の水分量を増加させ、過剰な保湿状態を引き起こす
3. エストロゲンの分泌を促進して肌を過敏にする
4. 表皮のターンオーバーを早め、肌を薄くする


<QuizAnswer>

**答え: 1**

喫煙は体内のビタミンCを大量に消費します。ビタミンCはコラーゲンの生成やメラニンの抑制に重要な役割を果たす抗酸化ビタミンです。また、喫煙により活性酸素が増加し、血管が収縮して血行が悪くなるため、肌への栄養供給が低下します。その結果、シワやくすみ、肌荒れなどの老化が促進されます。

</QuizAnswer>

## 問10

UVAの特徴として、正しいものはどれか。

1. エネルギーが最も強く、短時間で肌に激しい炎症を起こす
2. オゾン層に吸収されるため地表には届かない
3. 曇りの日や窓ガラスを透過して室内にも届く
4. 表皮にのみ作用し、真皮には到達しない


<QuizAnswer>

**答え: 3**

UVAは波長が長いため、雲や窓ガラスを透過して室内にも到達します。そのため、曇りの日や室内にいるときでも紫外線対策が必要です。UVAはエネルギー自体はUVBより弱いものの、真皮にまで到達してコラーゲンやエラスチンにダメージを与え、長期的にシワやたるみの原因となります。短時間で肌に炎症を起こすのはUVBの特徴です。

</QuizAnswer>
