## 問1

ベースメイクの基本的な手順として、正しい順番はどれか。

1. 化粧下地 → ファンデーション → コンシーラー → フェイスパウダー
2. ファンデーション → 化粧下地 → フェイスパウダー → コンシーラー
3. コンシーラー → 化粧下地 → ファンデーション → フェイスパウダー
4. フェイスパウダー → コンシーラー → 化粧下地 → ファンデーション


<QuizAnswer>

**答え: 1**

ベースメイクの基本的な手順は、スキンケア・日やけ止めの後に「化粧下地 → ファンデーション → コンシーラー → フェイスパウダー」の順番で仕上げます。化粧下地でファンデーションの密着性を高め、ファンデーションで肌全体を整えた後、気になる部分をコンシーラーでカバーし、最後にフェイスパウダーで仕上げるのが基本です。なお、パウダーファンデーションを使う場合はコンシーラーをファンデーションの前に使用します。

</QuizAnswer>

## 問2

パウダーファンデーションの特徴として、最も適切なものはどれか。

1. 油分が多く、乾燥肌の人に最も適している
2. クリームのようなテクスチャーで、カバー力が最も高い
3. 水分量が多く、みずみずしい仕上がりになる
4. 油分が少なく顔料が多いため、脂性肌の人に向いている


<QuizAnswer>

**答え: 4**

パウダーファンデーションは油分が少なく顔料（粉）が多いのが特徴です。余分な皮脂を吸収してテカリを防ぎ、サラサラとした仕上がりになるため、脂性肌の人に適しています。一方、油分が多くカバー力が高いのはクリームファンデーション、みずみずしい仕上がりになるのはリキッドファンデーションの特徴です。

</QuizAnswer>

## 問3

丸顔の人がチークを入れるとき、顔をすっきり見せるための入れ方として最も適切なものはどれか。

1. 頬骨の高い位置に丸く入れる
2. 頬骨の高い位置からこめかみに向かって斜めに入れる
3. 頬の中央に横長に広く入れる
4. 頬骨の下に水平に入れる


<QuizAnswer>

**答え: 2**

丸顔の人は横幅が気になるため、チークは頬骨の高い位置からこめかみに向かって斜めに入れることで、顔に縦のラインを強調しシャープな印象を与えます。丸く入れると丸みを強調してしまうため避けたほうがよいでしょう。顔型に合わせたチークの入れ方を工夫することで、理想的な卵型に近づけることができます。

</QuizAnswer>

## 問4

眉の黄金比（ゴールデンバランス）において、眉尻の基本的な位置として正しいものはどれか。

1. 目尻の真上
2. 眉頭の真横の延長線上
3. 黒目の外側の真上
4. 小鼻と目尻を結んだ延長線上


<QuizAnswer>

**答え: 4**

眉の黄金比では、眉尻は「小鼻（口角）と目尻を斜めに結んだラインの延長線上」に位置するのが基本とされています。また、眉頭は小鼻のくぼみ（目頭）の真上、眉山は黒目の外側から目尻の間に位置するのが理想的です。この3点のバランスを意識することで、どんな顔型の人でも自然で美しい眉を描くことができます。

</QuizAnswer>

## 問5

パーソナルカラーの4シーズン分類において、「イエベ（イエローベース）」に分類されるシーズンの組み合わせとして正しいものはどれか。

1. 春（スプリング）と夏（サマー）
2. 夏（サマー）と冬（ウィンター）
3. 春（スプリング）と秋（オータム）
4. 秋（オータム）と冬（ウィンター）


<QuizAnswer>

**答え: 3**

パーソナルカラーの4シーズン分類では、イエローベース（イエベ）は春（スプリング）と秋（オータム）、ブルーベース（ブルベ）は夏（サマー）と冬（ウィンター）に分類されます。イエベは黄みを含んだ暖かみのある色が似合い、ブルベは青みを含んだ涼しげな色が似合うとされています。

</QuizAnswer>

## 問6

目の下の青クマをカバーするコンシーラーの色として、最も適切なものはどれか。

1. グリーン
2. パープル
3. オレンジ
4. ブルー


<QuizAnswer>

**答え: 3**

青クマは血行不良が原因で青みがかって見えるため、青の補色にあたるオレンジ色のコンシーラーで打ち消すのが効果的です。グリーンは赤みやニキビ跡のカバーに、パープルは黄ぐすみのカバーに適しています。コンシーラーの色選びでは、カバーしたい色の「補色」を使うことで自然に肌悩みを目立たなくすることができます。

</QuizAnswer>

## 問7

メイクアップにおいて「シェーディング」の役割として、最も適切なものはどれか。

1. 肌より暗い色をのせて影をつくり、顔の気になる部分を目立たなくする
2. 明るい色をのせて光を集め、顔に立体感を出す
3. 肌の色ムラを均一に整えて、ファンデーションのノリをよくする
4. まぶたに色をのせて、目元に奥行きを出す


<QuizAnswer>

**答え: 1**

シェーディングとは、実際の肌色よりも暗い色をのせることで影をつくり、顔の気になる部分（エラや額の広さなど）を目立たなくして小顔効果を生み出すメイクテクニックです。一方、明るい色で光を集めて立体感を出すのは「ハイライト」の役割です。シェーディングとハイライトを組み合わせることで、メリハリのある立体的な顔立ちに仕上げることができます。

</QuizAnswer>

## 問8

パーソナルカラーが「ブルベ冬（ウィンター）」の人に似合う色の特徴として、最も適切なものはどれか。

1. パステルカラーのような淡くソフトな色合い
2. テラコッタやカーキなど温かみのある深い色合い
3. コーラルピンクやアプリコットなど明るく華やかな色合い
4. ビビッドな原色やモノトーンなどコントラストの強い色合い


<QuizAnswer>

**答え: 4**

ブルベ冬（ウィンター）タイプの人は、宝石のようなクリアではっきりとした色合いが似合うのが特徴です。鮮やかなビビッドカラーや、黒・白・シルバーなどのモノトーン、コントラストの強い色の組み合わせが得意です。淡くソフトな色はブルベ夏、温かみのある深い色はイエベ秋、明るく華やかな暖色系はイエベ春の特徴です。

</QuizAnswer>

## 問9

面長の顔型の人がメイクで卵型に近づけるためのテクニックとして、最も適切なものはどれか。

1. 額の中央と顎先にハイライトを入れて縦のラインを強調する
2. チークを頬骨の高い位置から斜め下に向かって入れる
3. チークをやや横長に入れて横幅を強調し、額と顎にシェーディングを入れる
4. シェーディングをフェイスラインの左右に入れて横幅を抑える


<QuizAnswer>

**答え: 3**

面長の顔型は縦の長さが気になるため、チークをやや横長に入れることで横幅を強調し、額の生え際や顎先にシェーディングを入れて縦の長さを抑えるのが効果的です。縦のラインをさらに強調するハイライトや、斜め下に向かうチークは面長をより際立たせてしまいます。顔型別メイクの基本は、理想とされる「卵型」に近づけることを目標にします。

</QuizAnswer>

## 問10

眉の黄金比において、眉頭から眉山までの長さと、眉山から眉尻までの長さの理想的な比率はどれか。

1. 1：1
2. 2：1
3. 3：1
4. 1：2


<QuizAnswer>

**答え: 2**

眉の黄金比では、眉頭から眉山までの長さと、眉山から眉尻までの長さの理想的な比率は「2：1」とされています。また、眉の太さについては眉頭・眉山・眉尻がそれぞれ「3：2：1」の比率が理想的です。この黄金比を意識して眉を描くことで、バランスの取れた自然で美しい眉に仕上がります。

</QuizAnswer>
