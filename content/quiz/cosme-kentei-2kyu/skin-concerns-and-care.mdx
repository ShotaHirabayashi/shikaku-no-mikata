## 問1

ニキビの初期段階で、毛穴に皮脂や古い角質が詰まってできた状態を何と呼ぶか。

1. パプル（丘疹）
2. コメド（面皰）
3. 膿疱（のうほう）
4. 結節（けっせつ）


<QuizAnswer>

**答え: 2**

ニキビの初期段階では、毛穴の出口が角質で塞がれ、内部に皮脂が溜まった状態になります。この状態を「コメド（面皰）」と呼びます。コメドには、毛穴が閉じた白ニキビ（閉鎖面皰）と、毛穴が開いた黒ニキビ（開放面皰）の2種類があります。パプルや膿疱は炎症が進行した段階のニキビです。

</QuizAnswer>

## 問2

ニキビの原因菌であるアクネ菌（プロピオニバクテリウム・アクネス）の特徴として、正しいものはどれか。

1. 皮脂を栄養として増殖し、炎症を引き起こす
2. 外部環境から新たに感染する細菌である
3. 酸素が豊富な環境を好んで繁殖する
4. 真皮の深い層にのみ生息している


<QuizAnswer>

**答え: 1**

アクネ菌は皮膚の常在菌であり、外部から感染するものではありません。毛穴の中（皮脂腺付近）に生息し、酸素の少ない環境（嫌気性）を好みます。毛穴が詰まって皮脂が溜まると、それを栄養源として過剰に増殖し、炎症性物質を産生することでニキビの赤みや腫れを引き起こします。

</QuizAnswer>

## 問3

シミの一種である「肝斑（かんぱん）」の特徴として、正しいものはどれか。

1. 幼少期から現れ、遺伝的要因が大きい
2. 境界がはっきりした濃い褐色の円形シミである
3. 男女問わず同じ割合で発症する
4. 両頬に左右対称に現れ、30〜60代の女性に多い


<QuizAnswer>

**答え: 4**

肝斑は、主に30〜60代の女性に多く見られるシミで、両頬や額、口の周りなどに左右対称に現れるのが大きな特徴です。色は淡い褐色で、輪郭がぼんやりしています。女性ホルモンのバランスの乱れが主な原因と考えられており、妊娠中やピルの服用時に現れることもあります。幼少期から現れるのは「そばかす（雀卵斑）」の特徴です。

</QuizAnswer>

## 問4

ニキビ跡や傷跡、虫刺され跡などの炎症が治まった後にできる茶色いシミを何と呼ぶか。

1. 老人性色素斑
2. 脂漏性角化症
3. 炎症後色素沈着
4. 雀卵斑（そばかす）


<QuizAnswer>

**答え: 3**

炎症後色素沈着とは、ニキビや傷、やけど、虫刺されなどで皮膚に炎症が起こった後、その部位にメラニン色素が過剰に生成されて残る茶色いシミのことです。炎症の刺激によりメラノサイトが活性化することが原因です。時間の経過とともにターンオーバーによって徐々に薄くなることが多いですが、紫外線を浴びると悪化するため注意が必要です。

</QuizAnswer>

## 問5

目元や口元にできる細かい小ジワ「ちりめんジワ」の主な原因はどれか。

1. 表情筋の過度な使用
2. 肌の乾燥による水分不足
3. 真皮のコラーゲンの断裂
4. 皮下脂肪の萎縮


<QuizAnswer>

**答え: 2**

ちりめんジワは「乾燥ジワ」とも呼ばれ、肌表面の水分が不足することでできる表皮性の浅いシワです。角層の水分量が低下すると肌のキメが乱れ、細かいシワとして現れます。乾燥が原因であるため、十分な保湿ケアを行うことで改善が期待できます。一方、真皮のコラーゲンの断裂が原因の深いシワは「真皮ジワ」と呼ばれ、改善が難しくなります。

</QuizAnswer>

## 問6

加齢により肌のハリが失われ、毛穴が縦長の楕円形に広がって目立つようになった状態を何と呼ぶか。

1. たるみ毛穴
2. 開き毛穴
3. 詰まり毛穴
4. メラニン毛穴


<QuizAnswer>

**答え: 1**

たるみ毛穴は、加齢によって真皮のコラーゲンやエラスチンが減少し、肌のハリや弾力が低下することで毛穴が重力方向に引っ張られ、縦長の楕円形（しずく型）に広がった状態です。頬に多く見られます。開き毛穴は皮脂の過剰分泌が原因で丸く開いた状態、詰まり毛穴は角栓が毛穴に詰まった状態を指し、それぞれ原因とケア方法が異なります。

</QuizAnswer>

## 問7

肌が黄色っぽくくすむ「黄ぐすみ」の主な原因はどれか。

1. 血行不良による血色の低下
2. 古い角質の蓄積
3. メラニン色素の沈着
4. 糖化によるタンパク質の変性


<QuizAnswer>

**答え: 4**

黄ぐすみの主な原因は「糖化」です。糖化とは、体内の余分な糖がタンパク質と結合してAGEs（終末糖化産物）が生成される反応のことです。肌のコラーゲンなどのタンパク質が糖化すると黄褐色に変色し、肌全体が黄色っぽくくすんで見えます。血行不良が原因のくすみは青黒くなり、メラニン蓄積によるくすみは茶色っぽくなるなど、くすみの種類によって原因と見え方が異なります。

</QuizAnswer>

## 問8

目の下にできる「くま」の種類と原因の組み合わせとして、正しいものはどれか。

1. 青くま＝色素沈着、茶くま＝血行不良、黒くま＝たるみ
2. 青くま＝血行不良、茶くま＝色素沈着、黒くま＝たるみやくぼみ
3. 青くま＝たるみ、茶くま＝色素沈着、黒くま＝血行不良
4. 青くま＝血行不良、茶くま＝たるみ、黒くま＝色素沈着


<QuizAnswer>

**答え: 2**

くまは原因により3種類に分けられます。「青くま」は血行不良により目の下の薄い皮膚を通して毛細血管の血液が透けて青く見えるもので、睡眠不足や疲労時に目立ちます。「茶くま」はこすりすぎや紫外線などの刺激による色素沈着（メラニン）が原因です。「黒くま」は加齢によるたるみやくぼみが影を作ることで黒く見えるものです。

</QuizAnswer>

## 問9

毛穴に角栓が詰まってできる「詰まり毛穴」のお手入れ方法として、最も適切なものはどれか。

1. 指で強く押し出して角栓を除去する
2. 毎日スクラブ洗顔を行い角質を削り取る
3. 酵素洗顔やクレイパックで穏やかに角栓を除去する
4. 油分の多いクリームを厚く塗って毛穴を塞ぐ


<QuizAnswer>

**答え: 3**

詰まり毛穴のケアには、酵素洗顔料やクレイ（泥）パックなどを使って、肌に負担をかけずに穏やかに角栓を取り除くことが効果的です。指で無理に押し出すと肌を傷つけて炎症や色素沈着の原因になります。また、毎日のスクラブ洗顔は肌への刺激が強すぎるため逆効果です。適度な頻度で角質ケアを行い、保湿で肌を整えることが大切です。

</QuizAnswer>

## 問10

大人ニキビ（吹き出物）の特徴として、正しいものはどれか。

1. 主にTゾーン（額・鼻）にできやすい
2. フェイスライン（あご・口周り）にできやすく、同じ場所に繰り返しやすい
3. 思春期ニキビと同じく過剰な皮脂分泌だけが原因である
4. 10代のニキビより炎症が軽く、治りが早い


<QuizAnswer>

**答え: 2**

大人ニキビ（吹き出物）は、あごやフェイスライン、口周りなどのUゾーンにできやすいのが特徴で、同じ場所に繰り返しできやすい傾向があります。思春期ニキビが主にTゾーンの過剰な皮脂分泌が原因であるのに対し、大人ニキビはターンオーバーの乱れ、ホルモンバランスの変化、ストレス、乾燥など複合的な要因が関係しています。治りにくく跡が残りやすいため、適切なケアが重要です。

</QuizAnswer>
