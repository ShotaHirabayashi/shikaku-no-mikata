## 問1

睡眠中に分泌される成長ホルモンは、肌細胞の修復や再生を促進する重要なホルモンである。この成長ホルモンが最も多く分泌されるのはいつか。

1. 午前2時〜4時の時間帯
2. 入眠後最初の約3時間
3. 起床直前のレム睡眠時
4. 睡眠の後半にあたる明け方


<QuizAnswer>

**答え: 2**

成長ホルモンは入眠後最初の約3時間に集中的に分泌されます。特に入眠直後に訪れる深いノンレム睡眠のときに分泌量がピークとなります。かつて「22時〜2時がゴールデンタイム」と言われていましたが、現在では時間帯よりも入眠後の睡眠の質が重要とされています。

</QuizAnswer>

## 問2

美肌に重要な栄養素のうち、体内でコラーゲンの生成を助ける働きをもつビタミンとして、最も適切なものはどれか。

1. ビタミンD
2. ビタミンB1
3. ビタミンC
4. ビタミンK


<QuizAnswer>

**答え: 3**

ビタミンCはコラーゲンの生成に不可欠な栄養素です。タンパク質とビタミンCを一緒に摂取することでコラーゲンの合成が促進されます。また、ビタミンCには抗酸化作用もあり、メラニン色素の生成を抑える働きも期待できます。

</QuizAnswer>

## 問3

入浴時の湯温と自律神経の関係について、正しいものはどれか。

1. 38〜40℃のぬるめのお湯は副交感神経を優位にし、リラックス効果が得られる
2. 38〜40℃のぬるめのお湯は交感神経を刺激し、心身を活発にする
3. 42℃以上の熱めのお湯は副交感神経を優位にし、リラックス効果が得られる
4. 湯温による自律神経への影響はなく、入浴時間のみが重要である


<QuizAnswer>

**答え: 1**

38〜40℃程度のぬるめのお湯に浸かると副交感神経が優位になり、緊張がほぐれてリラックス効果が得られます。一方、42℃以上の熱いお湯では交感神経が刺激され、心身が活発な状態になります。リラックスや美肌のためにはぬるめのお湯にゆっくり浸かることが推奨されます。

</QuizAnswer>

## 問4

顔の筋肉のうち、目の周りを囲むようにつき、目の開閉に関わる筋肉の名称はどれか。

1. 前頭筋
2. 口輪筋
3. 咬筋
4. 眼輪筋


<QuizAnswer>

**答え: 4**

眼輪筋は目の周りを輪状に囲む筋肉で、まぶたの開閉を行います。前頭筋はおでこにあり眉を引き上げる筋肉、口輪筋は口の周りを囲み唇を閉じたりすぼめたりする筋肉、咬筋は上下の歯を噛み合わせる筋肉です。

</QuizAnswer>

## 問5

リンパの流れについて、正しいものはどれか。

1. リンパは心臓から末端に向かって流れる
2. 全身のリンパは最終的に鎖骨下の静脈に合流する
3. リンパは動脈と同じ経路を通って循環する
4. リンパの流れは双方向で、自由に行き来している


<QuizAnswer>

**答え: 2**

リンパの流れは一方通行で、身体の末端からリンパ節を経由しながら、最終的に鎖骨下にある静脈角で静脈に合流します。そのため、鎖骨リンパが滞ると全身のリンパの流れにも影響が出ます。リンパマッサージでは、この流れに沿って末端から鎖骨方向へ向かって行うことが基本です。

</QuizAnswer>

## 問6

自律神経と肌の関係について、正しいものはどれか。

1. 副交感神経が優位になると血管が収縮し、肌の代謝が低下する
2. 交感神経が優位になると血管が拡張し、肌の血流が増加する
3. 交感神経が優位になると血管が収縮し、肌がくすみやすくなる
4. 自律神経は内臓の働きのみを調整し、肌には影響しない


<QuizAnswer>

**答え: 3**

交感神経が優位になると肌の血管が収縮し、血行不良により肌の代謝が低下します。その結果、肌がくすみやすくなったり、バリア機能が低下したりします。逆に副交感神経が優位なリラックス状態では血流が豊富になり、肌の代謝が高まって健康的な肌状態が保たれます。

</QuizAnswer>

## 問7

有酸素運動が美肌に良いとされる主な理由として、最も適切なものはどれか。

1. 血行が促進され、肌に栄養や酸素が行き届きやすくなるため
2. 筋肉が増えることで顔の脂肪が直接燃焼されるため
3. 運動中の紫外線が肌のターンオーバーを促進するため
4. 激しい運動ほど成長ホルモンが増え、肌のハリが出るため


<QuizAnswer>

**答え: 1**

有酸素運動（ウォーキングやヨガなど）は血行を促進し、肌の隅々まで栄養や酸素が行き届きやすくなることで、肌の新陳代謝を高めます。また、適度な運動はストレス解消にもつながり、自律神経のバランスを整える効果も期待できます。過度な運動は逆に活性酸素を増やす可能性があるため、適度な運動が推奨されます。

</QuizAnswer>

## 問8

美肌に役立つビタミンとして、ビタミンA・ビタミンC・ビタミンEを合わせた総称は何か。

1. ビタミンBコンプレックス
2. マルチビタミン
3. ビタミンプレミックス
4. ビタミンACE（エース）


<QuizAnswer>

**答え: 4**

ビタミンA・ビタミンC・ビタミンEの3つを合わせて「ビタミンACE（エース）」と呼びます。ビタミンAは皮膚や粘膜を正常に保つ働き、ビタミンCはコラーゲン生成の促進や抗酸化作用、ビタミンEは抗酸化作用と血行促進の働きがあります。これらを一緒に摂ることで相乗効果が期待できます。

</QuizAnswer>

## 問9

顔のリンパマッサージを行う際の力加減として、最も適切なものはどれか。

1. 筋肉に届くよう、強めの力でしっかり押す
2. なでる程度の優しい力で行う
3. 痛みを感じるくらいの強い圧をかける
4. 骨に響くほどの力で深く押し込む


<QuizAnswer>

**答え: 2**

リンパは皮膚の表面に近いところを流れているため、なでる程度の優しい力で十分にリンパを流すことができます。力を入れすぎると逆にリンパ管を圧迫してしまい、効果が十分に発揮できなくなります。マッサージの方向は、顔の内側から外側へ、そして耳の前を通り首から鎖骨へ向かって流すのが基本です。

</QuizAnswer>

## 問10

肌の修復に関わる成長ホルモンが最も多く分泌される睡眠の種類はどれか。

1. レム睡眠
2. 入眠直前のうとうとした状態
3. 深いノンレム睡眠
4. 明け方の浅い睡眠


<QuizAnswer>

**答え: 3**

成長ホルモンは深いノンレム睡眠時に最も多く分泌されます。特に入眠後最初の約90分間に訪れる深いノンレム睡眠がピークとなります。そのため、寝つきを良くして最初の深い睡眠の質を高めることが、美肌のために重要です。就寝前のカフェイン摂取やスマートフォンの使用を避けるなどの工夫が推奨されます。

</QuizAnswer>
