## 問1

表皮を皮膚の外側から内側へ正しく並べたものはどれか。

1. 角層→有棘層→顆粒層→基底層
2. 顆粒層→角層→有棘層→基底層
3. 角層→顆粒層→有棘層→基底層
4. 角層→有棘層→基底層→顆粒層


<QuizAnswer>

**答え: 3**

表皮は外側から「角層」「顆粒層」「有棘層」「基底層」の4層で構成されています。基底層で生まれたケラチノサイト（角化細胞）は、有棘層・顆粒層を経て角層へと押し上げられ、最終的に垢として剥がれ落ちます。この4層の順番は美容皮膚科学の最も基本的な知識です。

</QuizAnswer>

## 問2

表皮の「ターンオーバー」について正しい説明はどれか。

1. 基底層で生まれた細胞が角層に達し、最終的に剥がれ落ちるまでの新陳代謝のこと
2. 真皮のコラーゲンが分解・再生を繰り返すこと
3. 皮脂腺から皮脂が分泌される周期のこと
4. 汗腺が活性化して発汗するサイクルのこと


<QuizAnswer>

**答え: 1**

ターンオーバーとは、表皮の基底層で生まれたケラチノサイトが有棘層・顆粒層を経て角層に達し、最終的に垢として剥がれ落ちるまでの表皮の新陳代謝の過程を指します。健康な肌では通常約28日間の周期で行われ、この周期が乱れると肌トラブルの原因になります。

</QuizAnswer>

## 問3

健康な肌における表皮のターンオーバーの周期は、通常どのくらいか。

1. 約14日間
2. 約28日間
3. 約42日間
4. 約56日間


<QuizAnswer>

**答え: 2**

健康な肌のターンオーバーは通常約28日間の周期で行われます。基底層で新しく生まれた細胞が角層に到達するまでに約14日間、角層にとどまり剥がれ落ちるまでに約14日間、合計約28日間とされています。加齢やストレスなどの影響でこの周期が長くなることがあります。

</QuizAnswer>

## 問4

真皮の構造について正しい記述はどれか。

1. 表皮よりも薄く、約0.1mmの厚さである
2. 主にケラチンというタンパク質で構成されている
3. 皮膚の最も外側に位置する層である
4. 約70%がコラーゲン線維で構成され、肌のハリを支えている


<QuizAnswer>

**答え: 4**

真皮は表皮の内側にあり、平均約2mmの厚さがあります。真皮の約70%はコラーゲン線維で構成されており、肌にハリと強度を与えています。その間をヒアルロン酸などの基質が水分を抱えながら満たし、エラスチン線維がコラーゲン線維同士をつなぎとめる立体構造になっています。

</QuizAnswer>

## 問5

角層に存在する天然保湿因子（NMF）のもとになるタンパク質はどれか。

1. フィラグリン
2. コラーゲン
3. エラスチン
4. メラニン


<QuizAnswer>

**答え: 1**

天然保湿因子（NMF）は、顆粒層で産生されるフィラグリンが角層へ移行する過程で分解されてできるアミノ酸などが主成分です。フィラグリンはプロフィラグリンから変換されたのち、最終的にアミノ酸に分解されてNMFとなります。NMFは角層内で水分を保持し、肌のうるおいを維持する重要な役割を果たしています。

</QuizAnswer>

## 問6

真皮においてコラーゲン・エラスチン・ヒアルロン酸などを産生する細胞はどれか。

1. ケラチノサイト（角化細胞）
2. メラノサイト（色素細胞）
3. ランゲルハンス細胞
4. 線維芽細胞


<QuizAnswer>

**答え: 4**

線維芽細胞は真皮に存在し、コラーゲンやエラスチン、ヒアルロン酸などの真皮の構成成分を産生する重要な細胞です。さらに、古くなったコラーゲンやエラスチンの分解処理も担っています。加齢とともに線維芽細胞の機能が低下すると、肌のハリや弾力が失われていきます。

</QuizAnswer>

## 問7

エクリン汗腺の特徴として正しいものはどれか。

1. 脇の下や外陰部など限られた部位にのみ存在する
2. 分泌される汗はタンパク質や脂質を多く含み白く濁っている
3. ほぼ全身に分布し、主に体温調節の役割を担っている
4. 思春期以降に発達し、体臭の原因となりやすい


<QuizAnswer>

**答え: 3**

エクリン汗腺はほぼ全身に分布しており、汗の蒸発によって体の熱を放散し、体温を一定に保つ重要な役割を果たしています。分泌される汗は無色透明でサラサラしています。一方、アポクリン汗腺は脇の下や外陰部など限られた部位に存在し、思春期以降に発達する汗腺です。

</QuizAnswer>

## 問8

角層のバリア機能の説明として最も適切なものはどれか。

1. メラニンを生成して紫外線を吸収する機能
2. 外部刺激の侵入を防ぎ、体内の水分の過剰な蒸散を防ぐ機能
3. コラーゲンを生成して肌に弾力を与える機能
4. 皮脂を分泌して肌表面をなめらかにする機能


<QuizAnswer>

**答え: 2**

角層のバリア機能とは、肌表面からほこりや細菌などの外部刺激が体内に侵入するのを防ぐとともに、体内の水分が体外に過剰に蒸散するのを防ぐ二重の防御機能のことです。角層の細胞間脂質（セラミドなど）やNMF（天然保湿因子）がこのバリア機能を支えています。

</QuizAnswer>

## 問9

エラスチンについて正しい記述はどれか。

1. 真皮の約70%を占める主要な構成成分である
2. 表皮の角層に存在し、水分を保持する成分である
3. 基底層で生成され、肌の色を決める色素である
4. 真皮に約2%存在し、ゴムのような伸縮性でコラーゲン線維同士をつなぎとめている


<QuizAnswer>

**答え: 4**

エラスチンは真皮に約2%存在する線維状タンパク質で、ゴムのような伸縮性を持っています。丈夫なコラーゲン線維同士をつなぎとめて強固な立体構造を構築し、肌の弾力性を維持する役割を果たしています。真皮の約70%を占めるのはコラーゲンであり、エラスチンと混同しないよう注意が必要です。

</QuizAnswer>

## 問10

肌タイプの分類で、皮脂量が多いが水分量が少なく、顔の部位によって肌状態が異なるタイプはどれか。

1. 普通肌
2. 脂性肌
3. 混合肌
4. 乾燥肌


<QuizAnswer>

**答え: 3**

混合肌は皮脂量が多く脂っぽい部分がある一方で、水分量が少なく乾燥している部分もある肌タイプです。Tゾーン（額・鼻）はべたつきやすく、頬や目元は乾燥しやすいなど、顔の部位によって肌状態が異なるのが特徴です。脂性肌は水分量・皮脂量ともに多い状態を指します。

</QuizAnswer>
