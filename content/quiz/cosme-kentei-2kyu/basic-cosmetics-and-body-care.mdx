## 問1

クレンジング料にはさまざまなタイプがありますが、一般的に最も洗浄力が強いとされるのはどれか。

1. クレンジングミルク
2. クレンジングクリーム
3. クレンジングオイル
4. クレンジングジェル（水性）


<QuizAnswer>

**答え: 3**

クレンジングオイルは油分と界面活性剤の配合量が最も多く、洗浄力が最も強いタイプです。ウォータープルーフのメイクや濃いメイクもしっかり落とせる一方、乾燥肌や敏感肌の方には刺激が強くなることもあります。一般的な洗浄力の順は、オイル＞バーム＞リキッド＞ジェル＞クリーム＞ミルクとされています。

</QuizAnswer>

## 問2

洗顔時に使用するぬるま湯の温度として最も適切なものはどれか。

1. 20〜25℃
2. 30〜35℃
3. 38〜40℃
4. 42〜45℃


<QuizAnswer>

**答え: 2**

洗顔に適した温度は30〜35℃程度のぬるま湯です。皮脂が溶け始める温度が約32℃とされており、この温度帯であれば余分な皮脂を落としつつ、肌に必要なうるおいを守ることができます。熱すぎるお湯は必要な皮脂まで奪い、乾燥の原因になります。

</QuizAnswer>

## 問3

洗顔料を泡立てて顔を洗う際、最初に泡をのせる部位として正しいのはどれか。

1. Tゾーン（額・鼻）
2. 目元・口元
3. Uゾーン（頬・あご）
4. 耳の周り


<QuizAnswer>

**答え: 1**

洗顔はまず皮脂分泌の多いTゾーン（額と鼻）から泡をのせて洗い始めます。次に面積の広いUゾーン（頬からあご）へ進み、最後に皮膚が薄くデリケートな目元・口元はさっとなでる程度にします。皮脂量の多い部位から洗うことで、肌への負担を最小限に抑えられます。

</QuizAnswer>

## 問4

基礎化粧品をつける正しい順番はどれか。

1. 乳液 → 化粧水 → クリーム → 美容液
2. 美容液 → 化粧水 → 乳液 → クリーム
3. 化粧水 → 乳液 → 美容液 → クリーム
4. 化粧水 → 美容液 → 乳液 → クリーム


<QuizAnswer>

**答え: 4**

基礎化粧品は水溶性成分の配合比率が高い（テクスチャーが軽い）ものから順につけるのが基本です。化粧水で水分を補い、美容液で目的に合った美容成分を与え、乳液で油分の膜をつくり、最後にクリームでしっかりと蓋をします。先に油分の多いものを塗ると、後から水溶性の成分が浸透しにくくなります。

</QuizAnswer>

## 問5

乳液の主な役割として最も適切なものはどれか。

1. 肌に美白成分を届ける
2. 肌表面に油分の膜をつくり、水分の蒸発を防ぐ
3. 古い角質を取り除く
4. 毛穴の奥の汚れを溶かし出す


<QuizAnswer>

**答え: 2**

乳液は水分と油分をバランスよく含んだエマルジョン（乳化物）で、肌の表面に薄い油分の膜をつくることで、化粧水で補った水分の蒸発を防ぐ役割があります。化粧水だけでは水分が蒸発しやすいため、乳液で蓋をすることがスキンケアの基本です。

</QuizAnswer>

## 問6

シートマスク（シートパック）を使用するタイミングとして、一般的に正しいのはどれか。

1. 洗顔直後、化粧水の前に使う
2. 乳液の後、クリームの前に使う
3. 化粧水の後、乳液の前に使う
4. すべてのスキンケアの最後に使う


<QuizAnswer>

**答え: 3**

シートマスクは一般的に化粧水で肌を整えた後、乳液やクリームの前に使用します。化粧水で肌の土台を整えてからシートマスクの美容成分を浸透させ、その後に乳液やクリームで蓋をするのが基本の流れです。ただし製品によって推奨順序が異なる場合もあるので、使用前に確認しましょう。

</QuizAnswer>

## 問7

ヘアケア製品のうち、髪の内部に栄養を補給しダメージを補修する目的のものはどれか。

1. トリートメント
2. リンス
3. シャンプー
4. コンディショナー


<QuizAnswer>

**答え: 1**

トリートメントは髪の内部に栄養成分を浸透させ、ダメージを補修することを主な目的としています。一方、リンスやコンディショナーは主に髪の表面をコーティングして保護・なめらかにする役割です。トリートメントにはシャンプー後に数分置いてから洗い流す「インバストリートメント」と、洗い流さない「アウトバストリートメント」があります。

</QuizAnswer>

## 問8

爪やすり（ネイルファイル）の正しい使い方として適切なものはどれか。

1. やすりを爪に対して90度（垂直）に当てて削る
2. 素早く往復させてノコギリのように削る
3. 爪が湿っている入浴直後に削る
4. 一定方向に動かして削る


<QuizAnswer>

**答え: 4**

爪やすりは一定方向に動かして削るのが正しい使い方です。往復させてノコギリのように削ると爪の層がずれてしまい、二枚爪や爪が割れる原因になります。爪に対して45度くらいの角度でやすりを当て、一方向に動かしましょう。また、爪が湿った状態では割れやすいため、乾いた状態で行うのが基本です。

</QuizAnswer>

## 問9

クレンジングミルクの特徴として正しいものはどれか。

1. 固形状で肌の温度で溶けてオイル状になる
2. 水を基材としており界面活性剤が多めに含まれる
3. 洗浄力がマイルドで肌への負担が少ない
4. ウォータープルーフのメイクも簡単に落とせる


<QuizAnswer>

**答え: 3**

クレンジングミルクは乳液状のやさしい洗浄力が特徴で、クレンジング料の中では最もマイルドなタイプの一つです。必要な皮脂を残しながらメイクや汚れを落とせるため、乾燥肌や敏感肌の方に適しています。ただし洗浄力が穏やかな分、濃いメイクやウォータープルーフの化粧品を落とすには不向きです。

</QuizAnswer>

## 問10

シャンプー後にトリートメントとコンディショナーを併用する場合、正しい使用順序はどれか。

1. コンディショナー → トリートメント
2. トリートメント → コンディショナー
3. どちらを先に使っても効果は同じ
4. 同時に混ぜて使う


<QuizAnswer>

**答え: 2**

トリートメントとコンディショナーを併用する場合は、シャンプー後にまずトリートメントで髪の内部を補修し、その後にコンディショナーで髪の表面をコーティングして保護するのが正しい順序です。先にコンディショナーで表面を覆ってしまうと、トリートメントの栄養成分が髪の内部に浸透しにくくなります。

</QuizAnswer>
