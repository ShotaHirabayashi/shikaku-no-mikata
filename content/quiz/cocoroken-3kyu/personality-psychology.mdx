## 問1

クレッチマーの類型論において、「肥満型（ピクニック型）」の体型と結びつけられた気質はどれか。

1. 分裂気質
2. 粘着気質
3. 循環気質
4. 統合気質


<QuizAnswer>

**答え: 3**

クレッチマーは体型と気質の関連を研究し、肥満型（ピクニック型）には循環気質（社交的で気分の波がある）、細長型（レプトゾーム型）には分裂気質、闘士型（アスレティック型）には粘着気質を対応させました。循環気質は躁うつ病（双極性障害）との関連が指摘されています。

</QuizAnswer>

## 問2

ユングの類型論における2つの態度（外向・内向）と組み合わされる4つの心理機能に含まれないものはどれか。

1. 思考
2. 直観
3. 意志
4. 感情


<QuizAnswer>

**答え: 3**

ユングは心理機能として「思考」「感情」「感覚」「直観」の4つを挙げました。「意志」はユングの4機能には含まれません。これらの機能と外向・内向の2つの態度を組み合わせることで、8つの性格類型が導かれます。

</QuizAnswer>

## 問3

オルポートの特性論において、その人の人生全体を支配するほど強力だが、ごくまれにしか見られない特性は何と呼ばれるか。

1. 中心特性
2. 二次的特性
3. 表面特性
4. 枢機特性（主要特性）


<QuizAnswer>

**答え: 4**

オルポートは個人の特性を3段階に分類しました。枢機特性（cardinal trait）は人生全体を支配するほど強力で、非常にまれです。中心特性（central trait）は1人につき5〜10個程度あり、その人らしさを表す特性です。二次的特性は特定の状況でのみ現れる特性を指します。

</QuizAnswer>

## 問4

キャッテルが因子分析を用いて抽出した、行動の背後にある基本的な特性を何と呼ぶか。

1. 共通特性
2. 根源特性
3. 表面特性
4. 類型特性


<QuizAnswer>

**答え: 2**

キャッテルは観察可能な行動の相関から見出される「表面特性」と、その背後にある基本的な「根源特性（source trait）」を区別しました。因子分析によって16の根源特性を抽出し、これをもとに16PF（16パーソナリティ因子質問紙）を開発しました。

</QuizAnswer>

## 問5

ビッグファイブ（主要5因子モデル）の5つの因子に含まれないものはどれか。

1. 誠実性
2. 協調性
3. 自律性
4. 神経症的傾向


<QuizAnswer>

**答え: 3**

ビッグファイブの5因子は、開放性（Openness）・誠実性（Conscientiousness）・外向性（Extraversion）・協調性（Agreeableness）・神経症的傾向（Neuroticism）です。頭文字をとって「OCEAN」と覚えることもあります。「自律性」はビッグファイブの因子には含まれません。

</QuizAnswer>

## 問6

フロイトの防衛機制のうち、受け入れがたい欲求や衝動を社会的に望ましい活動に置き換えることを何というか。

1. 抑圧
2. 合理化
3. 昇華
4. 反動形成


<QuizAnswer>

**答え: 3**

昇華（sublimation）とは、社会的に受け入れがたい欲求や衝動を、芸術・スポーツなど社会的に価値ある活動に向け変えることです。抑圧は不快な記憶や欲求を意識から締め出すこと、合理化はもっともらしい理由づけで自分を正当化すること、反動形成は本心とは逆の行動をとることを指します。

</QuizAnswer>

## 問7

バンデューラが提唱した「自己効力感」の4つの情報源に含まれないものはどれか。

1. 成功体験（遂行行動の達成）
2. 代理体験（モデリング）
3. 条件づけ体験
4. 言語的説得


<QuizAnswer>

**答え: 3**

バンデューラによれば、自己効力感の4つの情報源は、①成功体験（遂行行動の達成）、②代理体験（他者の成功を観察する）、③言語的説得（励ましや説得）、④生理的・情動的状態（身体の反応や気分）です。「条件づけ体験」は自己効力感の情報源には含まれません。

</QuizAnswer>

## 問8

フェスティンガーの認知的不協和理論において、自分の中に矛盾する2つの認知がある場合に生じる不快な状態を低減する方法として適切でないものはどれか。

1. 一方の認知を変化させる
2. 新しい認知を追加して矛盾を緩和する
3. 認知の重要性を低く見積もる
4. 矛盾する認知の数を意図的に増やす


<QuizAnswer>

**答え: 4**

認知的不協和の低減方法は、①一方の認知を変える、②矛盾を和らげる新たな認知を加える、③認知の重要性を下げる、の3つが代表的です。矛盾する認知を増やすことは不協和をさらに強めてしまうため、低減方法にはなりません。フェスティンガーの有名な実験では、低報酬群のほうが態度変容が大きいことが示されました。

</QuizAnswer>

## 問9

ロジャーズの理論において、自分が「こうありたい」と望む自己像を何というか。

1. 現実自己
2. 理想自己
3. 公的自己
4. 社会的自己


<QuizAnswer>

**答え: 2**

ロジャーズは自己概念を、現在の自分についての認識である「現実自己（real self）」と、こうありたいと願う「理想自己（ideal self）」に区別しました。この2つの間のズレ（不一致）が大きいほど心理的な不適応が生じやすいとされています。

</QuizAnswer>

## 問10

フロイトの防衛機制のうち、本心とは正反対の態度や行動をとることで不安を抑えようとするメカニズムはどれか。

1. 投影
2. 退行
3. 否認
4. 反動形成


<QuizAnswer>

**答え: 4**

反動形成（reaction formation）とは、受け入れがたい感情や欲求に対し、それとは正反対の態度や行動をとることで不安から身を守る防衛機制です。投影は自分の感情を他者に帰属させること、退行は発達上の早い段階の行動に戻ること、否認は現実の出来事を認めないことを指します。

</QuizAnswer>
