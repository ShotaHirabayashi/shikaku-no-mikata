## 問1

心理検査において「同じ検査を同じ対象者に一定期間をおいて2回実施し、その結果の相関を確認する方法」は、何を確認するための手法か。

1. 内容的妥当性
2. 構成概念妥当性
3. 再検査信頼性
4. 基準関連妥当性


<QuizAnswer>

**答え: 3**

同一の検査を同一の対象者に時間をおいて再度実施し、2回の得点の相関係数を求める方法は「再検査信頼性」と呼ばれます。これは検査結果の時間的安定性を評価するものです。信頼性とは測定結果の一貫性・安定性のことであり、妥当性（測定したいものを正しく測定しているか）とは異なる概念です。

</QuizAnswer>

## 問2

ウェクスラー式知能検査の特徴として正しいものはどれか。

1. 精神年齢を生活年齢で割ってIQを算出する
2. 検査結果は単一の総合IQのみで示される
3. 下位検査ごとの得点から認知的な強みと弱みを分析できる
4. 集団式検査であり、一度に多数の受検者に実施できる


<QuizAnswer>

**答え: 3**

ウェクスラー式知能検査は、言語理解・知覚推理・ワーキングメモリー・処理速度などの下位指標ごとに得点を算出でき、個人内のプロフィール分析（ディスクレパンシー分析）が可能です。精神年齢÷生活年齢で算出するのはビネー式の比率IQであり、ウェクスラー式では偏差IQ（平均100、標準偏差15）を用います。また、ウェクスラー式は個別式検査です。

</QuizAnswer>

## 問3

ロールシャッハ・テストの説明として正しいものはどれか。

1. 不完全な文章を完成させることでパーソナリティを把握する
2. あいまいな場面の絵を見せて物語を作らせる検査である
3. 左右対称のインクブロットを見せて「何に見えるか」を尋ねる検査である
4. 実のなる木を1本描かせて心理状態を分析する検査である


<QuizAnswer>

**答え: 3**

ロールシャッハ・テストは、スイスの精神科医ヘルマン・ロールシャッハが1921年に開発した投影法の心理検査です。左右対称のインクブロット（インクのしみ）10枚を1枚ずつ提示し、「何に見えるか」を自由に回答させます。選択肢1は文章完成法（SCT）、選択肢2はTAT（主題統覚検査）、選択肢4はバウムテストの説明です。

</QuizAnswer>

## 問4

内田クレペリン精神検査の実施方法として正しいものはどれか。

1. 絵カードを見せて自由に連想させる
2. 1桁の足し算を連続して行い、作業量の変化を分析する
3. 「はい」「いいえ」で回答する質問紙に答えさせる
4. 積み木を使って指定された模様を再現させる


<QuizAnswer>

**答え: 2**

内田クレペリン精神検査は、内田勇三郎がクレペリンの連続加算法を発展させた作業検査法です。1桁の足し算を前半15分・休憩5分・後半15分で連続実施し、1分ごとの作業量を折れ線グラフ（作業曲線）にして定型曲線と比較することで、性格特徴や適性を判定します。言語や文化の影響を受けにくく、意図的な回答の歪曲が困難という特徴があります。

</QuizAnswer>

## 問5

心理検査の「妥当性」の意味として最も適切なものはどれか。

1. 検査を繰り返し実施しても同じ結果が得られること
2. 測定しようとしているものを正しく測定していること
3. 検査の実施手続きが統一されていること
4. 検査結果が正規分布に従うこと


<QuizAnswer>

**答え: 2**

妥当性（validity）とは、その検査が「測定しようとしているものを実際に正しく測定しているかどうか」を意味します。選択肢1は「信頼性」の説明、選択肢3は「標準化」の説明です。妥当性には内容的妥当性、基準関連妥当性、構成概念妥当性などの種類があります。

</QuizAnswer>

## 問6

質問紙法による性格検査の短所として最も適切なものはどれか。

1. 実施に高度な専門的訓練が必要である
2. 集団での一斉実施ができない
3. 社会的望ましさによる回答の歪曲が生じやすい
4. 検査結果の客観的な採点が困難である


<QuizAnswer>

**答え: 3**

質問紙法は客観的な採点が可能で集団施行も容易という長所がある一方、回答者が社会的に望ましい方向に回答を歪める「社会的望ましさバイアス」が生じやすいという短所があります。実施に高度な訓練が必要なのは投影法の特徴であり、質問紙法はむしろ簡便に実施・採点できることが利点です。

</QuizAnswer>

## 問7

偏差値の説明として正しいものはどれか。

1. 平均を0、標準偏差を1として算出する
2. 平均を100、標準偏差を15として算出する
3. 平均を50、標準偏差を10として算出する
4. 平均を50、標準偏差を15として算出する


<QuizAnswer>

**答え: 3**

偏差値は、集団における個人の相対的位置を示す得点で、平均が50、標準偏差が10になるよう変換した値です。計算式は「偏差値＝10×（個人得点−平均）÷標準偏差＋50」です。なお、選択肢2の「平均100、標準偏差15」はウェクスラー式知能検査の偏差IQの基準です。

</QuizAnswer>

## 問8

心理検査を実施する際の倫理的配慮として、検査の目的や方法を事前に説明し、被検査者の自発的な同意を得ることを何というか。

1. ラポール
2. インフォームドコンセント
3. カウンセリング
4. アセスメント


<QuizAnswer>

**答え: 2**

インフォームドコンセント（informed consent）とは、検査の目的・方法・結果の利用法などを事前に十分に説明し、被検査者の自発的な同意を得ることです。心理検査における重要な倫理的原則の一つであり、被検査者の権利を守るために欠かせない手続きです。ラポールは信頼関係のことで、アセスメントは心理的評価全般を指す用語です。

</QuizAnswer>

## 問9

TAT（主題統覚検査）を開発した研究者は誰か。

1. ロールシャッハ
2. コッホ
3. マレー
4. ビネー


<QuizAnswer>

**答え: 3**

TAT（Thematic Apperception Test：主題統覚検査）は、マレー（H.A. Murray）とモーガンによって1935年にハーバード大学で開発された投影法の心理検査です。あいまいな場面が描かれた絵を見せて物語を作らせ、その内容から深層心理を分析します。ロールシャッハはインクブロットテスト、コッホはバウムテスト、ビネーは知能検査の開発者です。

</QuizAnswer>

## 問10

心理検査の「標準化」の説明として最も適切なものはどれか。

1. 検査結果を偏差値に変換すること
2. 検査の実施手続き・採点方法・解釈基準を統一し、基準となるデータを作成すること
3. 検査項目を専門家が確認して妥当性を保証すること
4. 検査結果を被検査者にわかりやすく伝えること


<QuizAnswer>

**答え: 2**

標準化とは、検査の実施手続き・採点方法・結果の解釈基準を統一することです。大規模な代表的サンプル（標準化サンプル）に検査を実施して基準値（ノルム）を作成し、個人の得点を集団の中で位置づけられるようにします。これにより、誰が・いつ・どこで実施しても同じ条件で比較可能な結果が得られるようになります。

</QuizAnswer>
