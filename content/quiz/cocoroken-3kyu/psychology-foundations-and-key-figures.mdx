## 問1

1879年にドイツのライプツィヒ大学で世界初の心理学実験室を開設し、心理学を哲学から独立した実験科学として確立した人物は誰か。

1. ジェームズ
2. フロイト
3. ヴント
4. ワトソン


<QuizAnswer>

**答え: 3**

ヴィルヘルム・ヴントは1879年にライプツィヒ大学に世界初の心理学実験室を開設し、「実験心理学の父」と呼ばれています。この出来事は心理学が独立した学問として成立した起点とされており、意識を要素に分解して分析する内観法を用いた研究を行いました。

</QuizAnswer>

## 問2

「心理学の対象は意識ではなく、外部から観察可能な行動である」と主張し、行動主義心理学を創始した人物は誰か。

1. ワトソン
2. スキナー
3. パヴロフ
4. ソーンダイク


<QuizAnswer>

**答え: 1**

ジョン・B・ワトソンは1913年に「行動主義者から見た心理学」を発表し、心理学の研究対象を内面的な意識ではなく、客観的に観察・測定できる行動に限定すべきだと主張しました。これにより心理学の研究方法が大きく転換し、行動主義心理学が確立されました。

</QuizAnswer>

## 問3

マズローやロジャーズが中心となって提唱した人間性心理学は、心理学の歴史の中でどのように位置づけられているか。

1. 第一の勢力
2. 第三の勢力
3. 第四の勢力
4. 第二の勢力


<QuizAnswer>

**答え: 2**

人間性心理学は「第三の勢力（Third Force）」と呼ばれます。第一の勢力である行動主義が人間を機械的・受動的に扱いすぎ、第二の勢力である精神分析が病理や無意識に偏りすぎているとして、人間の成長や自己実現など肯定的側面に焦点を当てた立場として登場しました。

</QuizAnswer>

## 問4

パヴロフが犬を用いた実験で発見した古典的条件づけの説明として、最も適切なものはどれか。

1. 自発的な行動の結果として報酬を与えることで行動頻度が増加する
2. 観察によって他者の行動を模倣する学習が成立する
3. 試行錯誤を繰り返すことで問題解決の効率が上がる
4. 中性刺激と無条件刺激を繰り返し対提示することで、中性刺激だけで反応が生じるようになる


<QuizAnswer>

**答え: 4**

古典的条件づけ（レスポンデント条件づけ）は、もともと反応を引き起こさない中性刺激（ベルの音など）を、反射的反応を引き起こす無条件刺激（餌など）と繰り返し対提示することで、中性刺激だけで反応（唾液分泌）が生じるようになる学習です。選択肢1はオペラント条件づけ、選択肢2は観察学習の説明です。

</QuizAnswer>

## 問5

スキナーが提唱したオペラント条件づけにおいて、行動の後に好ましい結果を与えることでその行動の頻度が増加することを何というか。

1. 消去
2. 般化
3. 正の強化
4. 負の罰


<QuizAnswer>

**答え: 3**

正の強化とは、ある行動の直後に好ましい刺激（強化子）を与えることで、その行動が繰り返される頻度が増加する現象です。例えば、レバーを押したら餌が出るスキナー箱の実験がこれにあたります。「正」は刺激を加えること、「強化」は行動頻度が増えることを意味します。

</QuizAnswer>

## 問6

「全体は部分の単なる総和以上のものである」という考え方を基本原理とする心理学の学派はどれか。

1. 行動主義心理学
2. ゲシュタルト心理学
3. 構成主義心理学
4. 機能主義心理学


<QuizAnswer>

**答え: 2**

ゲシュタルト心理学は、ヴェルトハイマー、ケーラー、コフカらによって創始された学派で、「全体は部分の単なる寄せ集め以上のまとまりを持つ」という考え方を基本原理としています。例えば、メロディは個々の音符の集まり以上の「まとまり」として知覚されるように、知覚の全体的な構造や体制化を重視しました。

</QuizAnswer>

## 問7

次のうち、「応用心理学」に分類されるものはどれか。

1. 臨床心理学
2. 知覚心理学
3. 認知心理学
4. 学習心理学


<QuizAnswer>

**答え: 1**

臨床心理学は、基礎心理学の知見を心理的問題の理解や支援に応用する分野であり、応用心理学に分類されます。一方、知覚心理学・認知心理学・学習心理学は、人間の心の一般的な法則やメカニズムを解明することを目的とした基礎心理学に分類されます。

</QuizAnswer>

## 問8

心理学の研究方法のうち、独立変数を操作して従属変数への影響を測定し、因果関係を最も厳密に特定できる方法はどれか。

1. 観察法
2. 調査法
3. 面接法
4. 実験法


<QuizAnswer>

**答え: 4**

実験法は、研究者が独立変数（原因と想定される要因）を意図的に操作し、従属変数（結果として測定する変数）への影響を調べる方法です。実験群と統制群（対照群）を設けて比較することで、因果関係を最も厳密に特定できます。ただし、人為的な実験場面であるため、日常場面への一般化（生態学的妥当性）には注意が必要です。

</QuizAnswer>

## 問9

認知心理学の基本的な考え方として、人間の心の働きをある装置の情報処理過程になぞらえるアプローチがとられる。そのモデルとして最も適切なものはどれか。

1. 蒸気機関モデル
2. コンピュータモデル
3. 電話交換機モデル
4. 時計仕掛けモデル


<QuizAnswer>

**答え: 2**

認知心理学は1950〜60年代の「認知革命」を経て発展し、人間の心の働きをコンピュータの情報処理過程になぞらえて理解する「情報処理アプローチ」を基本としています。入力（感覚）→処理（知覚・注意・記憶・思考）→出力（行動）というプロセスでとらえ、注意・記憶・問題解決などを研究対象とします。

</QuizAnswer>

## 問10

心理学における「健康と病気の連続性モデル」の考え方として、最も適切なものはどれか。

1. 健康な人は病気になることはない
2. 健康と病気は明確に区別でき、どちらかに分類される
3. 健康と病気は連続体（グラデーション）上にあり、明確な境界線はない
4. 病気になった人は健康な状態に戻ることはできない


<QuizAnswer>

**答え: 3**

連続性モデルでは、健康と病気は「あるかないか」の二者択一ではなく、完全な健康から完全な病気までの連続体（スペクトラム）上にあると考えます。人はこの連続体の上を状態に応じて移動するものであり、心理的な「正常」と「異常」についても明確な境界線を引くことは難しく、程度の問題としてとらえるのが現代心理学の基本的な立場です。

</QuizAnswer>
