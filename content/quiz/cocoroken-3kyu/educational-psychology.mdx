## 問1

学習者が活動そのものへの興味や楽しさから自発的に取り組む動機づけを何というか。

1. 外発的動機づけ
2. 達成動機づけ
3. 内発的動機づけ
4. 社会的動機づけ


<QuizAnswer>

**答え: 3**

内発的動機づけとは、報酬や罰といった外部からの要因ではなく、活動そのものが持つ面白さや興味から生じる動機づけのことです。例えば、パズルが楽しくて解き続けるような場合がこれにあたります。一方、外発的動機づけは、ご褒美や成績など外部の要因によって行動が促される場合を指します。

</QuizAnswer>

## 問2

ブルーム（Bloom）が体系化した教育評価のうち、学習の途中段階でつまずきを発見し、指導改善に役立てるために行う評価はどれか。

1. 診断的評価
2. 総括的評価
3. 相対的評価
4. 形成的評価


<QuizAnswer>

**答え: 4**

形成的評価とは、学習の進行中に行われる評価で、学習者のつまずきや理解度を把握して指導を改善するために実施されます。小テストやワークシートの確認などが代表的な方法です。なお、診断的評価は学習開始前に行うレディネスの把握、総括的評価は学習終了後に行う到達度の総合判定を指します。

</QuizAnswer>

## 問3

森田洋司が提唱した「いじめの四層構造モデル」において、いじめを見て見ぬふりをする層を何というか。

1. 観衆
2. 傍観者
3. 加害者
4. 仲裁者


<QuizAnswer>

**答え: 2**

森田洋司のいじめ四層構造モデルでは、「被害者」「加害者」「観衆」「傍観者」の4つの層でいじめの集団構造を説明します。「観衆」はいじめをはやし立てる存在であるのに対し、「傍観者」は見て見ぬふりをする存在です。傍観者層が暗黙のうちにいじめを助長するため、傍観者を仲裁者に変える働きかけが重要とされています。

</QuizAnswer>

## 問4

モレノ（Moreno）が考案した、学級内の対人関係を客観的に測定する手法はどれか。

1. ソシオメトリックテスト
2. バウムテスト
3. TAT（主題統覚検査）
4. ロールシャッハテスト


<QuizAnswer>

**答え: 1**

ソシオメトリックテストは、モレノが提唱した集団内の人間関係を測定する方法です。「一緒に活動したい人」「したくない人」などの質問によって成員間の選択・排斥関係を把握し、その結果を図示したものをソシオグラムと呼びます。これにより、人気児（スター）、孤立児、排斥児などの分類が可能になります。

</QuizAnswer>

## 問5

教師がある児童に対して「この子は伸びる」と高い期待を持つことで、実際にその児童の成績が向上する現象を何というか。

1. ハロー効果
2. ゴーレム効果
3. ピグマリオン効果
4. プラセボ効果


<QuizAnswer>

**答え: 3**

ピグマリオン効果（ローゼンタール効果）とは、ローゼンタールとジェイコブソンが1968年の実験で示した現象で、教師の高い期待が児童の成績向上をもたらすことを指します。教師が無意識に声かけを増やしたり温かい態度をとることが媒介要因とされています。逆に、低い期待が成績を低下させる現象はゴーレム効果と呼ばれます。

</QuizAnswer>

## 問6

内発的に動機づけられていた行動に対して外的報酬を与えることで、かえって内発的動機づけが低下する現象を何というか。

1. エンハンシング効果
2. アンダーマイニング効果
3. 学習性無力感
4. 社会的促進


<QuizAnswer>

**答え: 2**

アンダーマイニング効果とは、デシ（Deci）が1971年のソーマキューブパズル実験で示した現象で、もともと楽しんで取り組んでいた活動に金銭などの物質的報酬を与えると、報酬がなくなった後に取り組み意欲が低下するというものです。一方、言語的な賞賛（「よくできたね」など）は内発的動機づけを高めることがあり、これをエンハンシング効果と呼びます。

</QuizAnswer>

## 問7

エビングハウス（Ebbinghaus）の忘却曲線に関する説明として、最も適切なものはどれか。

1. 忘却は学習後一定の速度で進行する
2. 忘却は学習直後に急速に進み、その後は緩やかになる
3. 忘却は学習直後には起こらず、一定時間後に急速に進む
4. 忘却は記憶の種類に関わらず同じ速度で起こる


<QuizAnswer>

**答え: 2**

エビングハウスは1885年に無意味綴りを用いた記憶実験を行い、忘却の経過を「忘却曲線」として示しました。この研究により、記憶は学習直後に急速に失われ（20分後に約42%忘却）、その後は忘却の速度が緩やかになることが明らかになりました。このパターンは負の加速曲線と呼ばれます。

</QuizAnswer>

## 問8

学習の効率を高める方法として、一定の間隔をあけて繰り返し学習する方法を何というか。

1. 集中学習
2. 分散学習
3. 発見学習
4. プログラム学習


<QuizAnswer>

**答え: 2**

分散学習とは、学習と学習の間に適度な休憩や間隔をおいて繰り返し学習する方法です。一度に長時間まとめて学習する集中学習（いわゆる一夜漬け）と比較して、分散学習の方が長期的な記憶の定着に優れていることが多くの研究で示されています。この現象は分散効果（spacing effect）と呼ばれます。

</QuizAnswer>

## 問9

教育評価において、あらかじめ設定された到達基準に照らして学習者を評価する方法を何というか。

1. 相対評価
2. 個人内評価
3. 絶対評価（目標準拠評価）
4. 集団準拠評価


<QuizAnswer>

**答え: 3**

絶対評価（目標準拠評価）とは、「80点以上がA」のようにあらかじめ設定された到達目標や基準に基づいて学習者を評価する方法です。学習者個人の到達度を明確に把握できる利点があります。一方、相対評価（集団準拠評価）は集団内での順位や位置に基づいて評価する方法で、「上位10%がA」のような形式をとります。

</QuizAnswer>

## 問10

レヴィン（Lewin）が分類したリーダーシップの3類型のうち、集団の意思決定に成員の参加を促し、最も望ましいとされる型はどれか。

1. 放任型
2. 専制型
3. 権威型
4. 民主型


<QuizAnswer>

**答え: 4**

レヴィンは、リーダーシップを「専制型」「民主型」「放任型」の3つに分類しました。民主型は集団の意思決定にメンバーの参加を促し、話し合いを通じて方針を決めるスタイルで、メンバーの満足度や集団の持続的な生産性が最も高いとされています。専制型はリーダーがすべてを決定するため短期的には生産性が高いものの不満が蓄積しやすく、放任型はリーダーが関与しないため最も効果が低いとされます。

</QuizAnswer>
