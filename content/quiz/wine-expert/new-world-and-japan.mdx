## 問1

ニュージーランド最大のワイン産地であり、ソーヴィニヨン・ブランの銘醸地として世界的に知られる、南島北東部に位置する産地はどこか。

1. ホークス・ベイ
2. マールボロ
3. セントラル・オタゴ
4. ワイララパ


<QuizAnswer>

**答え: 2**

マールボロはニュージーランド南島の北東部に位置し、同国のワイン生産量の約80％以上を占める最大の産地です。昼夜の寒暖差が大きく水はけの良い土壌が特徴で、青々としたハーブ香と溌剌とした酸味を持つソーヴィニヨン・ブランの世界的銘醸地として知られています。

</QuizAnswer>

## 問2

チリを代表するブドウ品種カルメネールは、長年ある品種と混同されていた。その品種はどれか。

1. カベルネ・ソーヴィニヨン
2. カベルネ・フラン
3. メルロー
4. シラー


<QuizAnswer>

**答え: 3**

カルメネールはボルドー原産の黒ブドウ品種で、19世紀にチリに持ち込まれた後、約150年にわたりメルローと混同されていました。1994年にフランスの研究者によって別品種であることが確認され、1998年にチリ政府が正式に独立品種として認定しました。

</QuizAnswer>

## 問3

アメリカのワイン法において、ラベルにAVA（政府認定栽培地域）名を表示する場合、そのAVA内で栽培されたブドウを最低何％以上使用しなければならないか。

1. 75％
2. 85％
3. 90％
4. 95％


<QuizAnswer>

**答え: 2**

アメリカではAVA名をラベルに表示する場合、当該AVA内で栽培されたブドウを85％以上使用する必要があります。ただし州名表示の場合は75％以上（オレゴン州は95％以上）、畑名表示の場合は95％以上と、表示の種類によって基準が異なります。

</QuizAnswer>

## 問4

オーストラリアのワインにおけるGI（地理的表示）制度が導入された年はいつか。

1. 1983年
2. 1988年
3. 1990年
4. 1993年


<QuizAnswer>

**答え: 4**

オーストラリアのGI（Geographical Indication）制度は1993年に導入されました。この制度のもとでは、品種名・産地名・ヴィンテージいずれもラベルに表示する場合は85％以上の使用が求められます。フランスのAOCのような醸造法や品種の規定はなく、産地表示に特化した制度である点が特徴です。

</QuizAnswer>

## 問5

日本で最初にワインの地理的表示（GI）の指定を受けた産地はどこか。

1. 北海道
2. 山梨
3. 長野
4. 山形


<QuizAnswer>

**答え: 2**

2013年7月に国税庁長官の指定を受けた「GI山梨」が、日本初のワインの地理的表示です。GI山梨では、使用できるブドウ品種が甲州やマスカット・ベーリーAなど42品種に限定されており、山梨県内で醸造・貯蔵・容器詰めされたものであることが求められます。

</QuizAnswer>

## 問6

アルゼンチンのワイン生産量の約8割を占める最大の産地であり、マルベックの銘醸地として名高い州はどこか。

1. メンドーサ
2. サン・フアン
3. サルタ
4. パタゴニア


<QuizAnswer>

**答え: 1**

メンドーサ州はアンデス山脈の東麓に位置し、アルゼンチンのワイン生産量の約80％を占める最大の産地です。標高850〜1,500メートルの高地にブドウ畑が広がり、昼夜の大きな寒暖差がマルベックから凝縮感のある高品質な赤ワインを生み出しています。世界10大ワイン首都のひとつにも選ばれています。

</QuizAnswer>

## 問7

南アフリカ固有のブドウ品種ピノタージュは、ピノ・ノワールとある品種の交配によって生まれた。その品種はどれか。

1. グルナッシュ
2. ムールヴェードル
3. サンソー（エルミタージュ）
4. カリニャン


<QuizAnswer>

**答え: 3**

ピノタージュは1925年に南アフリカのステレンボッシュ大学のアブラハム・アイザック・ペロード教授が、ピノ・ノワールとサンソー（南アフリカではエルミタージュと呼ばれていた）を交配して作出した品種です。南アフリカを代表する固有品種として、ステレンボッシュなどで広く栽培されています。

</QuizAnswer>

## 問8

アメリカ・オレゴン州では、ラベルにブドウ品種名を表示する場合、連邦法より厳しい独自基準を設けている。オレゴン州で品種名を表示するために必要な最低使用比率はいくらか。

1. 75％
2. 85％
3. 95％
4. 90％


<QuizAnswer>

**答え: 4**

オレゴン州は全米で最も厳しい独自のラベル表示規定を持っています。連邦法では品種表示に75％以上の使用が求められますが、オレゴン州では90％以上の使用が必要です。また州名表示についても連邦法の75％ではなく95％以上という厳格な基準を設けており、ピノ・ノワールの銘醸地としての品質を守っています。

</QuizAnswer>

## 問9

日本の固有品種「甲州」について正しい記述はどれか。

1. 赤ワイン用の品種であり、山形県が最大の産地である
2. 日本で最も生産量の多い赤ワイン用品種である
3. 川上善兵衛によって交配された品種である
4. 日本のワイン用ブドウとして最も生産量が多い白ブドウ品種であり、山梨県が全国生産量の約90％を占める


<QuizAnswer>

**答え: 4**

甲州は日本を代表する白ワイン用ブドウ品種で、ワイン用ブドウとしては日本で最も生産量が多い品種です。そのほとんどが山梨県で栽培されており、全国生産量の約90％を山梨県が占めています。なお、川上善兵衛が交配したのはマスカット・ベーリーAであり、甲州は古くから日本に自生していた品種です。

</QuizAnswer>

## 問10

南アフリカのワイン法における原産地呼称制度の名称として正しいものはどれか。

1. WO（Wine of Origin）
2. DO（Denominación de Origen）
3. GI（Geographical Indication）
4. AVA（American Viticultural Area）


<QuizAnswer>

**答え: 1**

南アフリカではWO（Wine of Origin＝ワイン・オブ・オリジン）制度が1973年に整備されました。この制度はワインの原産地を保証するもので、地域（Region）、地区（District）、小地区（Ward）などの階層で産地が管理されています。DOはスペイン、GIはオーストラリア、AVAはアメリカの制度です。

</QuizAnswer>
