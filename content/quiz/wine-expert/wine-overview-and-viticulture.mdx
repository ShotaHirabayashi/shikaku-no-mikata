## 問1

ワインの分類において、「フォーティファイドワイン（酒精強化ワイン）」の説明として正しいものはどれか。

1. 果汁に香草やスパイスを加えて風味をつけたワイン
2. 醸造過程でブランデー等のスピリッツを添加しアルコール度数を高めたワイン
3. 瓶内二次発酵により炭酸ガスを含むワイン
4. ブドウ果汁を濃縮して糖度を高めてから発酵させたワイン


<QuizAnswer>

**答え: 2**

フォーティファイドワイン（酒精強化ワイン）は、醸造過程でブランデーなどのスピリッツを添加してアルコール度数を高めたワインです。代表的なものにシェリー、ポートワイン、マデイラなどがあります。選択肢1はフレーバードワイン、選択肢3はスパークリングワインの説明です。

</QuizAnswer>

## 問2

アルコール発酵の化学反応として正しいものはどれか。

1. ブドウ糖が酵母の作用でエチルアルコールと二酸化炭素に分解される
2. ブドウ糖が乳酸菌の作用で乳酸とエチルアルコールに分解される
3. リンゴ酸が酵母の作用でエチルアルコールと酢酸に分解される
4. ブドウ糖が酢酸菌の作用でエチルアルコールと酸素に分解される


<QuizAnswer>

**答え: 1**

アルコール発酵とは、ブドウ果汁中のブドウ糖（グルコース）と果糖（フルクトース）が酵母（イースト）の働きによりエチルアルコールと二酸化炭素（CO₂）に分解される反応です。化学式は「C₆H₁₂O₆ → 2C₂H₅OH + 2CO₂」で表されます。この反応はルイ・パスツールによって科学的に解明されました。

</QuizAnswer>

## 問3

マロラクティック発酵（MLF）について正しい説明はどれか。

1. ブドウ糖が乳酸菌によって乳酸に変換される反応
2. リンゴ酸が乳酸菌によって乳酸と二酸化炭素に変換される反応
3. 酒石酸が酵母によってリンゴ酸に変換される反応
4. クエン酸が酢酸菌によって酢酸に変換される反応


<QuizAnswer>

**答え: 2**

マロラクティック発酵（MLF）は、ワイン中のリンゴ酸（マリックアシッド）が乳酸菌の作用によって乳酸（ラクティックアシッド）と二酸化炭素に変換される反応です。これによりワインの酸味がまろやかになり、味わいに複雑さが加わります。赤ワインではほぼ必ず行われますが、フレッシュな酸味を残したい白ワインでは意図的に抑制されることもあります。

</QuizAnswer>

## 問4

ブドウの仕立て方のうち、「ギュイヨ仕立て」の特徴として正しいものはどれか。

1. 支柱を使わず、樹を自立させる株仕立ての一種である
2. 長梢剪定を基本とし、結果母枝を1本または2本残す仕立て方である
3. 主枝を水平に伸ばし、そこから短い結果母枝を上向きに出す仕立て方である
4. 樹をT字型に広げ、棚の上にブドウを這わせる仕立て方である


<QuizAnswer>

**答え: 2**

ギュイヨ仕立て（Guyot）は長梢剪定を基本とする代表的な仕立て方で、結果母枝（バゲット）を1本残すギュイヨ・サンプル（片枝）と2本残すギュイヨ・ドゥーブル（双枝）があります。選択肢3はコルドン仕立て（短梢剪定）の説明です。ギュイヨ仕立てはブルゴーニュをはじめ世界中で広く採用されています。

</QuizAnswer>

## 問5

ブドウ栽培における「フィロキセラ」について正しい説明はどれか。

1. カビの一種で、果実に灰色の胞子を形成しブドウを腐敗させる病害である
2. 北アメリカ原産のアブラムシの一種で、ブドウの根に寄生して樹を枯死させる害虫である
3. 高温多湿の環境で発生しやすい細菌性の病害で、葉に斑点が生じる
4. 冬季の低温によりブドウ樹の幹が裂ける生理障害である


<QuizAnswer>

**答え: 2**

フィロキセラ（ブドウネアブラムシ）は北アメリカ原産の害虫で、19世紀後半にヨーロッパに持ち込まれ、ヨーロッパ系品種（ヴィティス・ヴィニフェラ）のブドウ畑を壊滅的に荒廃させました。根に寄生するため殺虫剤での駆除が困難であり、現在ではアメリカ系品種の台木にヨーロッパ系品種を接ぎ木する方法が主な対策として世界中で行われています。

</QuizAnswer>

## 問6

赤ワインの醸造における「マセラシオン（醸し）」の主な目的はどれか。

1. ワインの酸度を下げてまろやかな味わいにすること
2. 果皮から色素（アントシアニン）やタンニンを抽出すること
3. 酵母の活動を活性化させてアルコール度数を高めること
4. ワイン中の不純物を沈殿させて清澄化すること


<QuizAnswer>

**答え: 2**

マセラシオン（醸し）は、果皮や種子を果汁と接触させる工程で、赤ワイン醸造において不可欠なプロセスです。主な目的は果皮からアントシアニン（赤色色素）やタンニン、香味成分を抽出することです。低温で行うコールド・マセラシオンは発酵前に5〜15℃で数日間行い、穏やかに色素とアロマを抽出する手法として知られています。

</QuizAnswer>

## 問7

ブドウ栽培における「ベト病（ミルデュー）」について正しいものはどれか。

1. 乾燥した気候で発生しやすく、果粒の表面に白い粉状の胞子が現れる
2. 高温多湿の条件で発生しやすく、葉の裏面に白いカビが生じる
3. 冷涼な気候でのみ発生し、主に幹部を侵す
4. 害虫の一種で、ブドウの葉を食害する


<QuizAnswer>

**答え: 2**

ベト病（ミルデュー／Mildiou）は高温多湿の条件で発生しやすいカビ（糸状菌）による病害で、葉の表面に油染み状の斑点が現れ、裏面に白いカビが生じます。19世紀にアメリカから持ち込まれた病害で、ボルドー液（硫酸銅と石灰の混合液）が有効な防除剤として発見されました。選択肢1は「うどんこ病（オイディウム）」の特徴に近い説明です。

</QuizAnswer>

## 問8

ワイン用ブドウ栽培における「テロワール」の構成要素として、一般的に含まれないものはどれか。

1. 土壌の性質と地質構造
2. 気候（気温・降水量・日照時間）
3. 醸造家の技術と設備
4. 地形（標高・斜面の向き・水はけ）


<QuizAnswer>

**答え: 3**

テロワール（Terroir）とは、ブドウ畑を取り巻く自然環境の総体を指すフランス語の概念です。土壌、気候、地形、水文条件、生物多様性など、ブドウの生育に影響を与える自然的要因が含まれます。醸造家の技術や設備は人的要因であり、テロワールの構成要素には通常含まれません。テロワールはワインの個性や品質を決定づける重要な概念として広く認識されています。

</QuizAnswer>

## 問9

「ビオディナミ（バイオダイナミクス）農法」について正しい説明はどれか。

1. 化学合成農薬のみを制限し、化学肥料の使用は認めている農法
2. ルドルフ・シュタイナーの思想に基づき、天体の運行に合わせた農作業や特殊な調剤を用いる農法
3. 遺伝子組み換え技術を積極的に活用して病虫害に強い品種を栽培する農法
4. 温室やビニールハウスを利用して年間を通じてブドウを栽培する農法


<QuizAnswer>

**答え: 2**

ビオディナミ（バイオダイナミクス）農法は、オーストリアの哲学者ルドルフ・シュタイナーが1924年に提唱した農業思想に基づく栽培法です。化学合成農薬や化学肥料を使用せず、月や惑星の運行に合わせた農暦（播種カレンダー）に従って農作業を行い、牛の角に詰めた牛糞（プレパラシオン500）などの特殊な調剤を用います。認証団体としてはデメテール（Demeter）が代表的です。

</QuizAnswer>

## 問10

ブドウの栽培適地とされる気候帯について、一般的にワイン用ブドウの栽培に適した生育期間の平均気温の範囲はどれか。

1. 5〜10℃
2. 12〜22℃
3. 25〜35℃
4. 30〜40℃


<QuizAnswer>

**答え: 2**

ワイン用ブドウ（ヴィティス・ヴィニフェラ）の栽培に適した生育期間の平均気温は概ね12〜22℃とされています。12℃未満では積算温度が不足してブドウが十分に成熟できず、22℃を超えると高温により果実の酸度低下や品質劣化が起こりやすくなります。このため、北緯30〜50度、南緯20〜40度の範囲がワインベルトと呼ばれるブドウ栽培適地帯となっています。

</QuizAnswer>
