## 問1

1855年のパリ万国博覧会を機に制定されたメドック格付けにおいて、第1級（プルミエ・クリュ）に格付けされたシャトーの数はいくつか。

1. 3
2. 4
3. 5
4. 6


<QuizAnswer>

**答え: 3**

1855年メドック格付けの第1級は、シャトー・ラフィット・ロスチャイルド、シャトー・ラトゥール、シャトー・マルゴー、シャトー・オー・ブリオン、シャトー・ムートン・ロスチャイルドの5シャトーです。ムートン・ロスチャイルドは当初第2級でしたが、1973年に第1級に昇格しました。なお、オー・ブリオンは例外的にグラーヴ地区から選ばれています。

</QuizAnswer>

## 問2

ブルゴーニュ地方のAOCの階層構造を上位から正しく並べたものはどれか。

1. グラン・クリュ → 村名 → プルミエ・クリュ → 地方名
2. プルミエ・クリュ → グラン・クリュ → 村名 → 地方名
3. グラン・クリュ → プルミエ・クリュ → 村名 → 地方名
4. 村名 → グラン・クリュ → プルミエ・クリュ → 地方名


<QuizAnswer>

**答え: 3**

ブルゴーニュの畑の階層構造は、上位から「グラン・クリュ（特級畑）」「プルミエ・クリュ（1級畑）」「村名（コミュナル）」「地方名（レジョナル）」の4段階です。ボルドーがシャトー（生産者）を格付けするのに対し、ブルゴーニュでは畑（クリマ）が格付けされている点が大きな違いです。

</QuizAnswer>

## 問3

シャンパーニュ地方において、ノンヴィンテージのシャンパーニュに求められる最低熟成期間（瓶詰め後、澱とともに）はどれか。

1. 9ヶ月以上
2. 12ヶ月以上
3. 15ヶ月以上
4. 24ヶ月以上


<QuizAnswer>

**答え: 3**

シャンパーニュのノンヴィンテージ（NV）は瓶詰め後15ヶ月以上の熟成が義務付けられています。一方、ミレジメ（ヴィンテージ）シャンパーニュは36ヶ月（3年）以上の熟成が求められます。この長い熟成期間がシャンパーニュの複雑な風味を生み出す要因の一つです。

</QuizAnswer>

## 問4

ボルドー右岸のポムロル地区に関する記述として正しいものはどれか。

1. メドック格付けと同じ1855年に格付けが制定された
2. サンテミリオンと同様に約10年ごとに格付けが見直される
3. 公式の格付け制度が存在しない
4. 1級から3級までの3段階の格付けがある


<QuizAnswer>

**答え: 3**

ポムロル地区はボルドー右岸に位置し、シャトー・ペトリュスなど世界的に著名なシャトーがあるにもかかわらず、公式の格付け制度が存在しません。ボルドー右岸で公的格付けを持つのはサンテミリオンのみで、サンテミリオンの格付けは定期的に見直されます。

</QuizAnswer>

## 問5

アルザス地方のグラン・クリュに認められている主要4品種に含まれないものはどれか。

1. リースリング
2. ゲヴュルツトラミネール
3. シルヴァネール
4. ピノ・グリ


<QuizAnswer>

**答え: 3**

アルザスのグラン・クリュに認められている主要4品種は、リースリング、ゲヴュルツトラミネール、ピノ・グリ、ミュスカの4つです。シルヴァネールはアルザスで栽培されている品種ですが、グラン・クリュの主要認定品種には含まれません（例外的にゾッツェンベルクのみシルヴァネールが認められています）。

</QuizAnswer>

## 問6

北部ローヌ地方の赤ワインに使用が認められている主要品種はどれか。

1. グルナッシュ
2. ムールヴェードル
3. シラー
4. カリニャン


<QuizAnswer>

**答え: 3**

北部ローヌ（セプテントリオナル）の赤ワインは、コート・ロティやエルミタージュなどの銘醸地を含み、シラー種の単一品種で造られるのが特徴です。一方、グルナッシュ、ムールヴェードル、カリニャンなどは南部ローヌ（メリディオナル）で主にブレンドに使用される品種です。

</QuizAnswer>

## 問7

ソーテルヌの格付けにおいて、唯一「プルミエ・クリュ・シュペリュール（特別第1級）」に分類されているシャトーはどれか。

1. シャトー・ギロー
2. シャトー・ディケム
3. シャトー・クリマン
4. シャトー・リューセック


<QuizAnswer>

**答え: 2**

シャトー・ディケム（Château d'Yquem）は、1855年のソーテルヌ格付けにおいて唯一「プルミエ・クリュ・シュペリュール（Premier Cru Supérieur／特別第1級）」に分類されました。世界最高峰の貴腐ワインとして知られ、他のシャトーとは別格の扱いとなっています。ギロー、クリマン、リューセックは第1級に格付けされています。

</QuizAnswer>

## 問8

ロワール渓谷のサンセール地区とプイィ・フュメ地区で主に使用される白ブドウ品種はどれか。

1. シュナン・ブラン
2. ミュスカデ
3. ソーヴィニヨン・ブラン
4. シャルドネ


<QuizAnswer>

**答え: 3**

サンセールとプイィ・フュメはロワール渓谷上流の中央フランス地区に位置し、ソーヴィニヨン・ブランから造られる辛口白ワインで世界的に有名です。シュナン・ブランはロワール中流域（トゥーレーヌ、アンジュー）、ミュスカデ（ムロン・ド・ブルゴーニュ）は下流域のペイ・ナンテ地区で主に栽培されています。

</QuizAnswer>

## 問9

シャンパーニュにおいて「ブラン・ド・ブラン」とはどのようなワインを指すか。

1. 黒ブドウのみから造られた白いシャンパーニュ
2. 白ブドウのみから造られたシャンパーニュ
3. 白ブドウと黒ブドウを同量ブレンドしたシャンパーニュ
4. ロゼのシャンパーニュの別名


<QuizAnswer>

**答え: 2**

「ブラン・ド・ブラン（Blanc de Blancs）」は「白の白」という意味で、白ブドウ（主にシャルドネ）のみから造られたシャンパーニュを指します。一方、黒ブドウ（ピノ・ノワール、ピノ・ムニエ）のみから造られたものは「ブラン・ド・ノワール（Blanc de Noirs）＝黒の白」と呼ばれます。

</QuizAnswer>

## 問10

フランスのワイン法において、AOC（AOP）の下位に位置し、より広い地域名で産地を表示できるカテゴリーはどれか。

1. ヴァン・ド・ターブル
2. IGP（Indication Géographique Protégée）
3. ヴァン・ド・フランス
4. クリュ・ブルジョワ


<QuizAnswer>

**答え: 2**

IGP（地理的表示保護）は、AOC（AOP）の下位に位置するカテゴリーで、かつてのヴァン・ド・ペイ（Vin de Pays）に相当します。AOCより規定が緩やかで、使用品種や製法の自由度が高く、より広い地域名で産地表示ができます。ヴァン・ド・フランスはさらに下位の地理的表示のないカテゴリー（旧ヴァン・ド・ターブル）、クリュ・ブルジョワはメドックの格付け制度の一つです。

</QuizAnswer>
