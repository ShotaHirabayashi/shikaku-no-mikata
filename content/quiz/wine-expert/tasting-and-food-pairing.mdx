## 問1

ワインのテイスティングにおいて、グラスに注いだワインの液面の縁（表面がグラスに触れている層）を何と呼ぶか。

1. レッグ
2. ディスク
3. ティアーズ
4. メニスカス


<QuizAnswer>

**答え: 2**

ワインの液面のグラスに触れている縁の部分を「ディスク」と呼びます。ディスクの厚みはワインの粘性と比例し、厚いほどアルコール度数や残糖量が高い傾向があります。レッグ（ティアーズ）はグラスの内壁を伝い落ちるワインの涙状の筋を指す別の外観用語です。

</QuizAnswer>

## 問2

ワインの香りの分類において、「第2アロマ」の由来として正しいものはどれか。

1. ブドウ品種そのものに由来する香り
2. 発酵過程で生成される香り
3. 樽熟成に由来する香り
4. 瓶内熟成による変化で生まれる香り


<QuizAnswer>

**答え: 2**

第2アロマは発酵過程で生成される香りです。酵母が糖をアルコールに変換する際にエステル類が生まれ、バナナやキャンディなどの香りが生じます。第1アロマはブドウ品種由来、第3アロマ（ブーケ）は樽熟成や瓶内熟成に由来する香りです。

</QuizAnswer>

## 問3

赤ワインのテイスティングにおいて、若いワインほど色調はどのような傾向を示すか。

1. オレンジがかった色調になる
2. レンガ色を帯びる
3. 紫がかった色調になる
4. 琥珀色に近づく


<QuizAnswer>

**答え: 3**

若い赤ワインは紫がかった色調を示します。熟成が進むにつれて、アントシアニン（色素成分）が変化し、ガーネット、レンガ色、オレンジがかった色調へと変化していきます。外観の色調を観察することで、ワインのおおよその熟成度合いを推測できます。

</QuizAnswer>

## 問4

ワインと料理のペアリングにおける基本原則として、最も適切なものはどれか。

1. 赤ワインには必ず魚料理を合わせる
2. ワインと料理の産地を合わせる
3. デザートには辛口の白ワインを合わせる
4. 料理よりも常にワインの味わいが強いものを選ぶ


<QuizAnswer>

**答え: 2**

ワインと料理のペアリングの基本原則の一つに「産地を合わせる」があります。同じ産地の料理とワインは、長い歴史の中で共に発展してきたため、自然に調和する組み合わせとなります。他にも「色を合わせる」「味わいの重さを合わせる」などの原則があります。

</QuizAnswer>

## 問5

ワインの香りの分類で、「第3アロマ」の別名として正しいものはどれか。

1. アロマティック
2. ヴァラエタルアロマ
3. ブーケ
4. プリムール


<QuizAnswer>

**答え: 3**

第3アロマは「ブーケ」とも呼ばれ、樽熟成や瓶内熟成の過程で生まれる複雑な香りを指します。ヴァニラ、ロースト、なめし革、腐葉土などの香りが代表的です。「アロマ」がブドウや発酵由来の若い香りを指すのに対し、「ブーケ」は熟成による香りを意味します。

</QuizAnswer>

## 問6

フルボディの赤ワイン（カベルネ・ソーヴィニヨン主体など）の一般的な提供適温として、最も適切なものはどれか。

1. 4〜6℃
2. 8〜10℃
3. 12〜14℃
4. 16〜18℃


<QuizAnswer>

**答え: 4**

フルボディの赤ワインは16〜18℃が適温とされています。温度が高めになることで渋み（タンニン）がまろやかに感じられ、複雑な香りも開きやすくなります。逆に冷やしすぎると渋みが強調され、香りも閉じてしまいます。

</QuizAnswer>

## 問7

ワインのテイスティングで、グラスを回した後にグラス内壁をゆっくり伝い落ちるワインの筋を何と呼ぶか。

1. エッジ
2. ローブ
3. レッグ（涙）
4. ディスク


<QuizAnswer>

**答え: 3**

グラスの内壁を伝い落ちるワインの筋は「レッグ」または「涙（ティアーズ）」と呼ばれます。これはワインの粘性を示す指標で、レッグがゆっくり落ちるほどアルコール度数や残糖量が高い傾向があります。マランゴニ効果（表面張力の差）によって生じる現象です。

</QuizAnswer>

## 問8

ワインの長期保存に最も適した温度帯はどれか。

1. 0〜5℃
2. 13〜15℃
3. 20〜22℃
4. 25〜28℃


<QuizAnswer>

**答え: 2**

ワインの長期保存には13〜15℃が最適とされています。この温度帯ではワインがゆっくりと適切に熟成します。温度が高すぎるとワインの劣化が早まり、低すぎると熟成が進みにくくなります。また、温度変化が少ないことも重要な保存条件です。

</QuizAnswer>

## 問9

テイスティングにおける「味わい」の評価項目で、口に含んだ瞬間の第一印象を表す用語はどれか。

1. フィニッシュ
2. アタック
3. ミッドパレット
4. アフターテイスト


<QuizAnswer>

**答え: 2**

ワインを口に含んだ瞬間の第一印象を「アタック」と呼びます。アタックでは味わいの強さや印象（軽い・やや強い・強いなど）を評価します。その後の口中での味わいの展開を「ミッドパレット」、飲み込んだ後に残る風味を「フィニッシュ（余韻）」と呼び、それぞれ別の評価ポイントとなります。

</QuizAnswer>

## 問10

ワインと料理のペアリングにおいて、「マリアージュ」という用語の本来の意味はどれか。

1. 調理法
2. 結婚
3. 収穫
4. 熟成


<QuizAnswer>

**答え: 2**

「マリアージュ」はフランス語で「結婚」を意味します。ワインと料理が互いに引き立て合い、調和して一体となる理想的な組み合わせを表現する言葉です。単なる「組み合わせ」を意味する「ペアリング」よりも、相互に高め合う特別な相性の良さを強調したニュアンスがあります。

</QuizAnswer>
