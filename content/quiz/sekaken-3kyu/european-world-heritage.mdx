## 問1

モン・サン・ミシェルが浮かぶフランス西海岸の湾の名前は何か。

1. ビスケー湾
2. セーヌ湾
3. サン・マロ湾
4. リヨン湾


<QuizAnswer>

**答え: 3**

モン・サン・ミシェルはフランス西海岸のサン・マロ湾に浮かぶ小島の上に建つ修道院です。サン・マロ湾はヨーロッパでも潮の干満の差が最も激しい場所の一つとして知られており、その差は15メートル以上にもなります。この潮の満ち引きによって島が陸続きになったり海に囲まれたりする神秘的な景観が特徴です。

</QuizAnswer>

## 問2

ヴェルサイユ宮殿の建造を命じたフランスの国王は誰か。

1. ルイ13世
2. ルイ14世
3. ルイ16世
4. ナポレオン1世


<QuizAnswer>

**答え: 2**

ヴェルサイユ宮殿はフランス絶対王政の象徴であり、「太陽王」と呼ばれたルイ14世が自らの宮殿として建造を命じました。バロック様式の壮麗な建築と、約40年の歳月をかけて完成された広大な庭園が見事に調和しています。1979年に世界遺産に登録されました。

</QuizAnswer>

## 問3

ギリシャの世界遺産「アテネのアクロポリス」の丘の上に建つ、最も有名な神殿の名前は何か。

1. パルテノン神殿
2. ポセイドン神殿
3. アポロン神殿
4. ゼウス神殿


<QuizAnswer>

**答え: 1**

アクロポリスの丘の上に建つパルテノン神殿は、紀元前5世紀に女神アテナに捧げるために建設された古代ギリシャ建築の最高傑作です。ドーリア式の柱が並ぶ壮大な神殿で、古代ギリシャ文明の繁栄を今に伝えています。アテネのアクロポリスは1987年に世界遺産に登録されました。

</QuizAnswer>

## 問4

ローマのコロッセオについて、正しい記述はどれか。

1. 中世に教会として建設された
2. 最大約1万人を収容できた
3. 古代エジプトの影響を受けた建築である
4. 最大約5万人を収容できる円形闘技場であった


<QuizAnswer>

**答え: 4**

コロッセオは西暦80年に完成した古代ローマの円形闘技場で、最大約5万人もの観客を収容することができました。周囲527メートル、高さ48メートルという世界最大級の規模を誇り、剣闘士の戦いなどが行われていました。「ローマ歴史地区、教皇領とサン・パオロ・フオーリ・レ・ムーラ大聖堂」の一部として1980年に世界遺産に登録されています。

</QuizAnswer>

## 問5

フィレンツェ歴史地区において、15〜16世紀にルネサンス芸術家を保護・支援した有力一族はどれか。

1. ハプスブルク家
2. ブルボン家
3. メディチ家
4. ボルジア家


<QuizAnswer>

**答え: 3**

フィレンツェは15〜16世紀にかけてメディチ家の庇護のもとで経済的・文化的に大きく繁栄し、ルネサンスの中心地となりました。レオナルド・ダ・ヴィンチやミケランジェロなどの天才芸術家を輩出し、ヨーロッパ全体の芸術・文化に多大な影響を与えました。フィレンツェ歴史地区は1982年に世界遺産に登録されています。

</QuizAnswer>

## 問6

イスラム建築の最高傑作とされるアルハンブラ宮殿がある都市はどこか。

1. マドリード（スペイン）
2. グラナダ（スペイン）
3. イスタンブール（トルコ）
4. カイロ（エジプト）


<QuizAnswer>

**答え: 2**

アルハンブラ宮殿はスペイン南部のグラナダにあり、13世紀にナスル朝（グラナダ王国）によって建設が始められました。アラベスク文様の繊細な装飾や、パティオ（中庭）を中心とした美しい構造が特徴で、「イスラム建築の華」とも呼ばれています。1984年に世界遺産に登録されました。

</QuizAnswer>

## 問7

イギリスの世界遺産であるウェストミンスター宮殿の時計塔は、2012年に正式名称が変更された。その現在の名称はどれか。

1. ビクトリア・タワー
2. クロック・タワー
3. ビッグベン・タワー
4. エリザベス・タワー


<QuizAnswer>

**答え: 4**

ウェストミンスター宮殿の時計塔は、2012年にエリザベス2世の在位60周年（ダイヤモンド・ジュビリー）を記念して、それまでの「クロック・タワー」から「エリザベス・タワー」に改称されました。なお「ビッグベン」は塔の中にある重さ約13.7トンの大鐘の名前であり、時計塔そのものの名称ではありません。ウェストミンスター宮殿は現在イギリスの国会議事堂として使用されています。

</QuizAnswer>

## 問8

世界遺産「アウシュヴィッツ・ビルケナウ ナチス・ドイツの強制絶滅収容所」がある国はどこか。

1. ポーランド
2. ドイツ
3. オーストリア
4. チェコ


<QuizAnswer>

**答え: 1**

アウシュヴィッツ強制収容所はポーランド南部のオシフィエンチム市にあり、第二次世界大戦中にナチス・ドイツが設置した最大規模の強制収容所です。1979年に世界遺産に登録され、ホロコーストの悲劇を二度と繰り返さないための記憶と教訓を伝える「負の遺産」として知られています。

</QuizAnswer>

## 問9

モン・サン・ミシェルは、その壮大な姿から古くから何と呼ばれてきたか。

1. 「海の王冠」
2. 「西洋の驚異」
3. 「大西洋の宝石」
4. 「フランスの奇跡」


<QuizAnswer>

**答え: 2**

モン・サン・ミシェルは、海上にそびえる修道院の神秘的かつ壮大な姿から、古くから「西洋の驚異（La Merveille）」と呼ばれてきました。また「海上のピラミッド」という別名でも知られています。1979年に「モン・サン・ミシェルとその湾」として世界遺産に登録されました。

</QuizAnswer>

## 問10

イタリアの世界遺産「フィレンツェ歴史地区」は、街全体に芸術作品や歴史的建造物が残ることから何と呼ばれているか。

1. 花の都
2. 永遠の都
3. 屋根のない博物館
4. 芸術の首都


<QuizAnswer>

**答え: 3**

フィレンツェ歴史地区は、13世紀から16世紀にかけて建てられた歴史的建造物や彫刻などの芸術作品が街中に残されていることから、「屋根のない博物館」と呼ばれています。なお「花の都」はフィレンツェの語源（花を意味するイタリア語）に由来する別称であり、「永遠の都」はローマを指す言葉です。

</QuizAnswer>
