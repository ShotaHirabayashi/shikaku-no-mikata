## 問1

屋久島と白神山地が世界自然遺産に登録された年として正しいものはどれか。

1. 1990年
2. 1996年
3. 1993年
4. 2000年


<QuizAnswer>

**答え: 3**

屋久島と白神山地は、1993年（平成5年）12月に日本で初めて世界自然遺産に登録されました。この2件は同時に登録されており、日本における世界自然遺産の第1号・第2号にあたります。なお、同年には文化遺産として法隆寺と姫路城も登録されています。

</QuizAnswer>

## 問2

「富岡製糸場と絹産業遺産群」が所在する都道府県はどこか。

1. 長野県
2. 群馬県
3. 栃木県
4. 埼玉県


<QuizAnswer>

**答え: 2**

富岡製糸場と絹産業遺産群は群馬県に所在し、2014年に世界文化遺産に登録されました。構成資産は富岡製糸場（富岡市）、田島弥平旧宅（伊勢崎市）、高山社跡（藤岡市）、荒船風穴（下仁田町）の4件で、すべて群馬県内にあります。

</QuizAnswer>

## 問3

「明治日本の産業革命遺産」の正式名称に含まれる産業分野の組み合わせとして正しいものはどれか。

1. 製鉄・製鋼、造船、石炭産業
2. 製鉄・製鋼、紡績、石炭産業
3. 製糸、造船、石油産業
4. 製鉄・製鋼、造船、鉄道産業


<QuizAnswer>

**答え: 1**

正式名称は「明治日本の産業革命遺産 製鉄・製鋼、造船、石炭産業」です。2015年に世界文化遺産に登録され、19世紀後半から20世紀初頭にかけて日本が急速に産業化を遂げた過程を証言する遺産群で、8エリア23資産で構成されています。

</QuizAnswer>

## 問4

原爆ドーム（広島平和記念碑）の説明として正しいものはどれか。

1. 1945年の空襲で被害を受けた軍事施設の跡である
2. 2000年に世界文化遺産に登録された
3. 登録時に全ての加盟国が賛成した
4. 戦争の悲惨さを伝える「負の遺産」の一つとされている


<QuizAnswer>

**答え: 4**

原爆ドームは1996年に世界文化遺産に登録され、核兵器の惨禍を伝える「負の遺産」の代表例とされています。もともとは広島県物産陳列館として建てられた建物で、1945年8月6日の原子爆弾投下により被爆しました。登録時にはアメリカと中国が反対の立場を示したことでも知られています。

</QuizAnswer>

## 問5

世界自然遺産「知床」の大きな特徴として最も適切なものはどれか。

1. 亜熱帯から亜高山帯までの植生の垂直分布が見られる
2. 流氷が育む豊かな海洋生態系と陸域生態系の相互関係がある
3. 大陸と一度も地続きになったことがない海洋島である
4. 世界最大級のブナ原生林が広がっている


<QuizAnswer>

**答え: 2**

知床は2005年に北海道で唯一の世界自然遺産に登録されました。最大の特徴は、冬に流れ着く流氷が豊富な栄養分を海にもたらし、プランクトンからサケ、ヒグマ、シマフクロウへと続く食物連鎖を支えている点です。海洋生態系と陸域生態系が密接に結びついた希少な環境が評価されました。

</QuizAnswer>

## 問6

世界自然遺産「小笠原諸島」について正しい説明はどれか。

1. 北海道の北東に位置する島々である
2. 1993年に日本初の世界自然遺産として登録された
3. 一度も大陸と地続きになったことがない海洋島で、固有種が多い
4. 流氷の影響で独特の生態系が形成されている


<QuizAnswer>

**答え: 3**

小笠原諸島は2011年に世界自然遺産に登録されました。約4800万年前から形成された海洋島で、一度も大陸と地続きになったことがないため、島にたどり着いた生物が独自の進化を遂げ、多くの固有種が生まれました。特に陸産貝類（カタツムリの仲間）は106種中100種が固有種で、固有種率94％という驚異的な数字を示しています。

</QuizAnswer>

## 問7

「奄美大島、徳之島、沖縄島北部及び西表島」が世界自然遺産に登録された年はいつか。

1. 2011年
2. 2015年
3. 2018年
4. 2021年


<QuizAnswer>

**答え: 4**

「奄美大島、徳之島、沖縄島北部及び西表島」は2021年7月に世界自然遺産に登録されました。日本で5番目の世界自然遺産であり、アマミノクロウサギやヤンバルクイナ、イリオモテヤマネコなど多くの固有種・希少種の生息地として、その生物多様性が高く評価されています。

</QuizAnswer>

## 問8

「明治日本の産業革命遺産」の構成資産に含まれるものはどれか。

1. 端島炭坑（軍艦島）
2. 富岡製糸場
3. 足尾銅山
4. 別子銅山


<QuizAnswer>

**答え: 1**

端島炭坑（通称：軍艦島）は長崎県に位置し、「明治日本の産業革命遺産」の23構成資産の一つです。1890年から本格的な石炭採掘が行われ、良質な石炭は主に官営八幡製鉄所へ供給されました。富岡製糸場は別の世界遺産「富岡製糸場と絹産業遺産群」として登録されています。

</QuizAnswer>

## 問9

富岡製糸場が開業した年と、導入した技術の国の組み合わせとして正しいものはどれか。

1. 1868年（明治元年）・イギリス
2. 1872年（明治5年）・フランス
3. 1882年（明治15年）・ドイツ
4. 1890年（明治23年）・アメリカ


<QuizAnswer>

**答え: 2**

富岡製糸場は1872年（明治5年）に明治政府の殖産興業政策のもとで開業しました。フランス人技師ポール・ブリュナの指導により、フランスの製糸技術を導入した日本初の本格的な機械製糸工場です。この技術移転により日本の絹産業は飛躍的に発展し、1920年代には世界一の生糸輸出国となりました。

</QuizAnswer>

## 問10

世界自然遺産「白神山地」が所在する都道府県の組み合わせとして正しいものはどれか。

1. 北海道と岩手県
2. 岩手県と宮城県
3. 青森県と秋田県
4. 秋田県と山形県


<QuizAnswer>

**答え: 3**

白神山地は青森県と秋田県にまたがる山地帯で、1993年に屋久島とともに日本初の世界自然遺産に登録されました。人の影響をほとんど受けていない原生的なブナ天然林が世界最大級の規模で分布していることが評価されています。ブナ林の多様な生態系には、クマゲラやニホンカモシカなどの希少な動物も生息しています。

</QuizAnswer>
