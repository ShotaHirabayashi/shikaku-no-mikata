## 問1

1978年に世界遺産リストに登録され、世界自然遺産の第1号とされる遺産はどれか。

1. イエローストーン国立公園
2. ガラパゴス諸島
3. グレート・バリア・リーフ
4. セレンゲティ国立公園


<QuizAnswer>

**答え: 2**

ガラパゴス諸島（エクアドル）は1978年に世界自然遺産として登録され、世界遺産リストのID番号も1番です。チャールズ・ダーウィンの進化論に大きな影響を与えた島々で、ゾウガメやウミイグアナなど数多くの固有種が生息しています。一度も大陸と陸続きになったことがない海洋島であり、独自の進化を遂げた生物の宝庫です。

</QuizAnswer>

## 問2

オーストラリア北東岸に広がる、全長約2,600kmにおよぶ世界最大のサンゴ礁地帯として知られる自然遺産はどれか。

1. ニンガルー・コースト
2. ロード・ハウ諸島
3. シャーク湾
4. グレート・バリア・リーフ


<QuizAnswer>

**答え: 4**

グレート・バリア・リーフは1981年に自然遺産として登録された、オーストラリア北東岸に広がる世界最大のサンゴ礁地帯です。2,900以上の暗礁群と約900の島からなり、400種以上のサンゴや1,500種以上の魚類が生息しています。宇宙空間からも確認できるほど広大で、生物が作り出した単一の構造物としても世界最大です。

</QuizAnswer>

## 問3

アメリカ・アリゾナ州にあるグランド・キャニオン国立公園の峡谷を、数百万年かけて浸食によって形成した川はどれか。

1. コロラド川
2. ミシシッピ川
3. ミズーリ川
4. リオ・グランデ川


<QuizAnswer>

**答え: 1**

グランド・キャニオンは、コロラド川がコロラド高原を数百万年かけて浸食して形成した全長約450km、深さ約1.6kmの大峡谷です。1979年に世界自然遺産に登録されました。峡谷の岩肌には約20億年前からの地層が露出しており、地球の歴史を直接観察できる貴重な場所として高く評価されています。

</QuizAnswer>

## 問4

トルコの「ギョレメ国立公園とカッパドキアの岩石遺跡群」は、世界遺産のどの分類に該当するか。

1. 文化遺産
2. 自然遺産
3. 複合遺産
4. 無形文化遺産


<QuizAnswer>

**答え: 3**

カッパドキアは、凝灰岩が風雨の浸食によって形成されたキノコ岩などの自然景観と、ビザンツ時代の洞窟教会や岩窟住居群などの文化的価値の両方が評価され、1985年に複合遺産として登録されました。複合遺産は文化遺産と自然遺産の両方の登録基準を満たす必要があり、世界遺産の中でもっとも数が少ない分類です。

</QuizAnswer>

## 問5

2025年の第47回世界遺産委員会終了時点での世界遺産の総登録件数として正しいものはどれか。

1. 987件
2. 1,248件
3. 1,523件
4. 1,073件


<QuizAnswer>

**答え: 2**

2025年の第47回世界遺産委員会終了時点で、世界遺産の総数は1,248件です。内訳は文化遺産972件、自然遺産235件、複合遺産41件で、文化遺産が全体の約8割を占めています。世界遺産条約の締約国は196か国、世界遺産を保有する国は170か国に上ります。

</QuizAnswer>

## 問6

2024年に開催された第46回世界遺産委員会の開催都市はどこか。

1. パリ
2. 北京
3. カイロ
4. ニューデリー


<QuizAnswer>

**答え: 4**

第46回世界遺産委員会は2024年7月21日〜31日にインドのニューデリーで開催されました。この会議では日本の「佐渡島の金山」を含む24件（文化遺産19件、自然遺産4件、複合遺産1件）の新規登録が決定されました。なお、翌2025年の第47回はフランスのパリ（ユネスコ本部）で開催されています。

</QuizAnswer>

## 問7

2025年時点で、世界遺産のうち複合遺産の登録件数として正しいものはどれか。

1. 41件
2. 96件
3. 135件
4. 235件


<QuizAnswer>

**答え: 1**

2025年時点で複合遺産は41件であり、全1,248件の世界遺産の中でもっとも少ない分類です。複合遺産は文化遺産と自然遺産の両方の登録基準を満たす必要があるため登録のハードルが高く、代表例にはペルーのマチュ・ピチュ、トルコのカッパドキア、オーストラリアのウルル＝カタ・ジュタ国立公園などがあります。なお選択肢4の235件は自然遺産の件数です。

</QuizAnswer>

## 問8

グレート・バリア・リーフが直面している深刻な環境問題として、海水温の上昇によりサンゴが大量死する現象は何と呼ばれるか。

1. 赤潮
2. 砂漠化
3. 白化現象
4. 富栄養化


<QuizAnswer>

**答え: 3**

グレート・バリア・リーフでは、地球温暖化による海水温の上昇がサンゴの白化現象を引き起こし、深刻な問題となっています。サンゴと共生する褐虫藻が高水温により離脱し、サンゴが白くなって死滅する現象です。2021年にはユネスコが危機遺産への登録を提案しましたが、オーストラリア政府の働きかけにより見送られました。

</QuizAnswer>

## 問9

2025年の第47回世界遺産委員会が開催された都市はどこか。

1. ソフィア
2. ニューデリー
3. パリ
4. 釜山


<QuizAnswer>

**答え: 3**

第47回世界遺産委員会は2025年7月にフランスのパリ（ユネスコ本部）で開催されました。当初はブルガリアのソフィアでの開催が予定されていましたが、パリに変更されています。この会議では26件の新規登録が決定し、世界遺産の総数は1,248件となりました。なお、2026年の第48回は韓国の釜山で開催予定です。

</QuizAnswer>

## 問10

オーストラリア中央部にある「ウルル＝カタ・ジュタ国立公園」の説明として正しいものはどれか。

1. 世界最大の熱帯雨林が広がる自然遺産である
2. 先住民アボリジニの聖地である巨大な一枚岩を含む複合遺産である
3. タスマニア島にある原生林の自然遺産である
4. 世界最大のサンゴ礁を有する自然遺産である


<QuizAnswer>

**答え: 2**

ウルル＝カタ・ジュタ国立公園は、巨大な一枚岩ウルル（エアーズロック）とカタ・ジュタ（オルガ岩群）を含むオーストラリア中央部の砂漠地帯にある遺産です。先住民アボリジニにとっての聖地としての文化的価値と、独特な地形がもつ自然的価値の両方が評価され、複合遺産として登録されています。カッパドキアやマチュ・ピチュと並ぶ代表的な複合遺産の一つです。

</QuizAnswer>
