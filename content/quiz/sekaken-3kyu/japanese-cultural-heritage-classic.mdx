## 問1

法隆寺地域の仏教建造物と姫路城が、日本で初めて世界文化遺産に登録された年はいつか。

1. 1990年
2. 1992年
3. 1993年
4. 1994年


<QuizAnswer>

**答え: 3**

法隆寺地域の仏教建造物と姫路城は、1993年に日本で最初の世界文化遺産として登録されました。これは日本が1992年に世界遺産条約を批准した翌年のことです。同年には自然遺産として屋久島と白神山地も登録され、合計4件が日本初の世界遺産となりました。

</QuizAnswer>

## 問2

姫路城は、その美しい白壁の外観から何という別名で呼ばれているか。

1. 白鷺城
2. 烏城
3. 金鯱城
4. 青葉城


<QuizAnswer>

**答え: 1**

姫路城は白漆喰で塗られた美しい白壁が特徴で、その優雅な姿が羽を広げた白鷺に見えることから「白鷺城（しらさぎじょう）」と呼ばれています。江戸時代以前に建設された天守が現存する「現存12天守」の一つであり、大天守など8棟が国宝に指定されています。

</QuizAnswer>

## 問3

「古都京都の文化財」を構成する社寺・城の数はいくつか。

1. 14件
2. 15件
3. 16件
4. 17件


<QuizAnswer>

**答え: 4**

「古都京都の文化財」は、鹿苑寺（金閣寺）、慈照寺（銀閣寺）、清水寺、平等院、二条城など17件の社寺・城で構成されています。これらは京都市・宇治市・大津市の2府県3市にまたがって所在しており、1994年に世界文化遺産に登録されました。

</QuizAnswer>

## 問4

厳島神社を現在の規模の社殿に造営した人物は誰か。

1. 源頼朝
2. 足利義満
3. 平清盛
4. 徳川家康


<QuizAnswer>

**答え: 3**

厳島神社は593年の創建と伝えられていますが、1168年に平清盛が寝殿造の様式を取り入れて現在の規模に造営しました。瀬戸内海の海上に建ち並ぶ社殿群は、宮島（厳島）そのものをご神体とする信仰に基づいた独特の構成です。1996年に世界文化遺産に登録されました。

</QuizAnswer>

## 問5

「古都奈良の文化財」の構成資産のうち、聖武天皇の発願により建立され「奈良の大仏」で知られる寺院はどれか。

1. 東大寺
2. 興福寺
3. 薬師寺
4. 唐招提寺


<QuizAnswer>

**答え: 1**

東大寺は、聖武天皇が国家の安泰を願って建立した寺院で、本尊の盧舎那仏（奈良の大仏）で広く知られています。「古都奈良の文化財」は1998年に登録され、東大寺のほか興福寺、春日大社、薬師寺、唐招提寺、元興寺、春日山原始林、平城宮跡の計8資産で構成されています。

</QuizAnswer>

## 問6

白川郷・五箇山の合掌造り集落で、茅葺き屋根の葺き替えなどの際に行われる住民同士の相互扶助の仕組みを何と呼ぶか。

1. 講（こう）
2. 結（ゆい）
3. 座（ざ）
4. 組（くみ）


<QuizAnswer>

**答え: 2**

「結（ゆい）」とは、合掌造りの茅葺き屋根の葺き替えなど大がかりな作業を、集落の住民が互いに助け合って行う相互扶助の仕組みです。白川郷・五箇山は豪雪地帯であり、厳しい自然環境の中で人々が協力して暮らしてきた伝統が高く評価され、1995年に世界文化遺産に登録されました。

</QuizAnswer>

## 問7

次のうち、「古都京都の文化財」の構成資産に含まれないものはどれか。

1. 清水寺
2. 二条城
3. 平等院
4. 東大寺


<QuizAnswer>

**答え: 4**

東大寺は「古都奈良の文化財」の構成資産であり、「古都京都の文化財」には含まれません。「古都京都の文化財」は清水寺、二条城、平等院のほか、鹿苑寺（金閣寺）、龍安寺、延暦寺など17件の社寺・城で構成されています。両遺産の構成資産の違いは検定でも頻出のポイントです。

</QuizAnswer>

## 問8

「紀伊山地の霊場と参詣道」を構成する3つの霊場は、吉野・大峰、熊野三山と、もう1つは何か。

1. 高野山
2. 比叡山
3. 高尾山
4. 出羽三山


<QuizAnswer>

**答え: 1**

「紀伊山地の霊場と参詣道」は、吉野・大峰、熊野三山、高野山の3つの霊場と、それらを結ぶ参詣道（熊野参詣道、大峯奥駈道、高野参詣道）で構成されています。高野山は816年に空海（弘法大師）が開いた真言密教の聖地です。2004年に世界文化遺産に登録されました。

</QuizAnswer>

## 問9

法隆寺の回廊の柱に見られる、柱の中央部がふくらんだ形状の建築技法を何と呼ぶか。

1. アーチ
2. エンタシス
3. フレスコ
4. モザイク


<QuizAnswer>

**答え: 2**

エンタシスとは、柱の中央部をわずかにふくらませる建築技法で、法隆寺の回廊の円柱などに見られます。この技法は古代ギリシャのパルテノン神殿の柱にも見られることで知られ、東西の建築文化の交流を示す重要な特徴として注目されています。

</QuizAnswer>

## 問10

「古都京都の文化財」は、京都府の京都市・宇治市のほかに、もう1つどの市に所在する構成資産を含んでいるか。

1. 奈良市
2. 大阪市
3. 大津市
4. 神戸市


<QuizAnswer>

**答え: 3**

「古都京都の文化財」は京都府の京都市・宇治市に加えて、滋賀県大津市に所在する延暦寺（比叡山）を含んでいます。そのため正式名称は「古都京都の文化財（京都市、宇治市、大津市）」となっており、2府県3市にまたがる世界遺産です。この点は検定でもよく問われるポイントです。

</QuizAnswer>
