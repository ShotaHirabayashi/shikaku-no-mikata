## 問1

万里の長城がユネスコの世界遺産に登録された年はいつか。

1. 1979年
2. 1983年
3. 1987年
4. 1994年


<QuizAnswer>

**答え: 3**

万里の長城は1987年に世界遺産（文化遺産）に登録されました。北方民族の侵入を防ぐ目的で紀元前7世紀頃から各国が築いた城壁を、秦の始皇帝が連結したのが始まりとされています。現存する長城の多くは明代に大規模な補強工事が行われたものです。

</QuizAnswer>

## 問2

カンボジアにあるアンコール遺跡群の中心的建造物「アンコール・ワット」を建設した王は誰か。

1. スーリヤヴァルマン2世
2. ジャヤーヴァルマン7世
3. チャンドラグプタ
4. パチャクティ


<QuizAnswer>

**答え: 1**

アンコール・ワットは12世紀前半にクメール王朝のスーリヤヴァルマン2世（在位1113〜1145年頃）によって建設されました。一方、選択肢2のジャヤーヴァルマン7世はアンコール・トムの大部分を築いた王です。アンコール遺跡群は1992年に世界遺産に登録されました。

</QuizAnswer>

## 問3

インドの世界遺産「タージ・マハル」について、正しい説明はどれか。

1. ヒンドゥー教の神殿として建設された
2. イギリス植民地時代に総督府として建てられた
3. デリーのムガル帝国初代皇帝の墓である
4. ムガル帝国第5代皇帝が亡き妃のために建てた霊廟である


<QuizAnswer>

**答え: 4**

タージ・マハルは、ムガル帝国第5代皇帝シャー・ジャハーンが、1631年に亡くなった愛妃ムムターズ・マハルのために建設した白大理石の霊廟です。1631年から約17年の歳月をかけて完成し、インド・イスラーム建築の最高傑作とされています。1983年に世界遺産に登録されました。

</QuizAnswer>

## 問4

マチュ・ピチュの歴史保護区は、世界遺産の分類としてどれに該当するか。

1. 文化遺産
2. 複合遺産
3. 自然遺産
4. 無形文化遺産


<QuizAnswer>

**答え: 2**

マチュ・ピチュの歴史保護区は1983年に複合遺産として世界遺産に登録されました。複合遺産とは、文化遺産と自然遺産の両方の登録基準を満たす遺産のことです。マチュ・ピチュはインカ帝国の歴史的遺跡としての文化的価値と、絶滅危惧種を含む貴重な生態系を有する自然的価値の両方が認められています。

</QuizAnswer>

## 問5

ニューヨークにある「自由の女神像」は、どの国からアメリカに贈られたものか。

1. イギリス
2. オランダ
3. フランス
4. ドイツ


<QuizAnswer>

**答え: 3**

自由の女神像は、アメリカ合衆国の独立100周年を記念して、フランスから贈られたものです。フランスの彫刻家フレデリック・バルトルディが設計し、エッフェル塔で知られるギュスターヴ・エッフェルが内部の鉄骨構造を担当しました。1886年に完成し、1984年に世界遺産に登録されています。

</QuizAnswer>

## 問6

エジプトの世界遺産「メンフィスとその墓地遺跡—ギザからダハシュールまでのピラミッド地帯」が世界遺産に登録された年はいつか。

1. 1979年
2. 1984年
3. 1988年
4. 1992年


<QuizAnswer>

**答え: 1**

メンフィスとその墓地遺跡は1979年に世界遺産（文化遺産）に登録されました。これは世界遺産条約に基づく最初の世界遺産登録が行われた年であり、最も早く登録された遺産の一つです。ギザの3大ピラミッドをはじめ、約80基のピラミッドがギザからダハシュールにかけて南北約30kmにわたって残されています。

</QuizAnswer>

## 問7

パキスタンの世界遺産「モヘンジョダロの遺跡群」について、誤っているものはどれか。

1. インダス文明の都市遺跡である
2. 碁盤目状の道路や下水道など高度な都市計画が見られる
3. 紀元前2500年頃から繁栄した
4. 「王の丘」という意味の名前をもつ


<QuizAnswer>

**答え: 4**

モヘンジョダロは現地のシンド語で「死の丘」を意味しており、「王の丘」ではありません。紀元前2500年頃から紀元前1800年頃にかけて繁栄したインダス文明最大級の都市遺跡で、碁盤目状に整備された道路や、浴場・井戸・水洗トイレ・下水道などの高度な都市インフラが発見されています。1980年に世界遺産に登録されました。

</QuizAnswer>

## 問8

ペルーの世界遺産「ナスカとパルパの地上絵」の地上絵が長期間消えずに残っている主な理由はどれか。

1. 絵が硬い岩盤に深く刻まれているため
2. 極めて乾燥した気候で浸食がほとんどないため
3. 地元住民が定期的に修復を続けてきたため
4. 特殊な染料で着色されているため


<QuizAnswer>

**答え: 2**

ナスカ周辺は年間降水量がわずか数mmという極めて乾燥した地域であり、雨による浸食がほとんど起きないことが地上絵が消えない最大の理由です。さらに、石灰質の土壌が昼夜の寒暖差によって酸化・硬化するため風化にも強くなっています。地上絵は紀元前2世紀〜後7世紀頃にナスカ文化の人々によって描かれたと考えられています。

</QuizAnswer>

## 問9

マチュ・ピチュの遺跡を1911年に「発見」したとされるアメリカの探検家は誰か。

1. ハワード・カーター
2. コロンブス
3. ハイラム・ビンガム
4. シュリーマン


<QuizAnswer>

**答え: 3**

マチュ・ピチュは、1911年にアメリカの探検家・歴史学者ハイラム・ビンガムによって学術的に「発見」されました。選択肢1のハワード・カーターはエジプトのツタンカーメン王墓を発見した考古学者、選択肢4のシュリーマンはトロイア遺跡を発掘したドイツの考古学者です。マチュ・ピチュは15世紀にインカ帝国の皇帝パチャクティの命で建設されたと考えられています。

</QuizAnswer>

## 問10

アンコール遺跡群は1992年に世界遺産に登録されると同時に危機遺産にも登録されたが、危機遺産リストから解除されたのはいつか。

1. 2004年
2. 2000年
3. 2010年
4. 1998年


<QuizAnswer>

**答え: 1**

アンコール遺跡群は1992年の世界遺産登録と同時に、内戦による荒廃や盗掘の被害から危機遺産リストにも登録されました。その後、国際的な保全活動が進められた結果、2004年に危機遺産リストから解除されました。日本を含む多くの国が修復・保全プロジェクトに貢献しています。

</QuizAnswer>
