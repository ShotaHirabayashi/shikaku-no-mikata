## 問1

コンピュータの五大装置に含まれないものはどれか。

1. 演算装置
2. 記憶装置
3. 通信装置
4. 制御装置


<QuizAnswer>

**答え: 3**

コンピュータの五大装置は「入力装置・出力装置・記憶装置・演算装置・制御装置」の5つです。通信装置は五大装置には含まれません。演算装置と制御装置を合わせたものがCPU（中央処理装置）と呼ばれます。

</QuizAnswer>

## 問2

10進数の「13」を2進数で表したものはどれか。

1. 1011
2. 1101
3. 1110
4. 1010


<QuizAnswer>

**答え: 2**

10進数の13を2で繰り返し割ると、13÷2=6余り1、6÷2=3余り0、3÷2=1余り1、1÷2=0余り1となり、余りを下から順に並べると「1101」になります。確認すると、1×8+1×4+0×2+1×1=13となり正しいことがわかります。

</QuizAnswer>

## 問3

CPUの性能指標として使われる「クロック周波数」の単位はどれか。

1. バイト（B）
2. ヘルツ（Hz）
3. ビット毎秒（bps）
4. ドット毎インチ（dpi）


<QuizAnswer>

**答え: 2**

クロック周波数はCPUが1秒間に発生させるクロック信号の回数を表し、単位はヘルツ（Hz）です。一般的にGHz（ギガヘルツ）で表記され、値が大きいほど処理速度が速い傾向にあります。bpsはデータ通信速度、dpiは解像度の単位です。

</QuizAnswer>

## 問4

主記憶装置（メインメモリ）に使われる半導体メモリで、電源を切ると内容が消える特性をもつものはどれか。

1. ROM
2. RAM
3. SSD
4. HDD


<QuizAnswer>

**答え: 2**

RAM（Random Access Memory）は揮発性メモリであり、電源を切るとデータが消えます。主記憶装置にはこのRAMが使われています。ROM（Read Only Memory）は不揮発性で電源を切ってもデータが保持されます。SSDやHDDは補助記憶装置に分類されます。

</QuizAnswer>

## 問5

OSの機能の一つであるタスク管理において、複数のプログラムを短い時間間隔で切り替えながら同時に実行しているように見せる方式を何というか。

1. マルチタスク
2. マルチブート
3. マルチキャスト
4. マルチプレクサ


<QuizAnswer>

**答え: 1**

マルチタスクとは、CPUの処理時間を短い単位で区切り、複数のプログラムに順番に割り当てることで、あたかも同時に実行しているように見せる方式です。マルチブートは複数のOSを選択起動する機能、マルチキャストは特定の複数の相手にデータを送信する通信方式です。

</QuizAnswer>

## 問6

フローチャートにおいて、条件による分岐を表す図形はどれか。

1. 長方形
2. 平行四辺形
3. ひし形
4. 角丸長方形


<QuizAnswer>

**答え: 3**

フローチャートでは、ひし形（菱形）は条件判断（分岐）を表します。長方形は処理、平行四辺形はデータの入出力、角丸長方形（端子記号）は開始や終了を表します。これらはJIS規格で定められた標準的な記号です。

</QuizAnswer>

## 問7

補助記憶装置に関する説明として、最も適切なものはどれか。

1. SSDはHDDに比べて衝撃に弱い
2. SSDは磁気ディスクを使ってデータを記録する
3. HDDはフラッシュメモリを使ってデータを記録する
4. SSDはHDDに比べてデータの読み書きが高速である


<QuizAnswer>

**答え: 4**

SSD（Solid State Drive）はフラッシュメモリを使用しており、機械的な駆動部分がないため、磁気ディスクを回転させて読み書きするHDD（Hard Disk Drive）に比べてデータの読み書きが高速です。また、SSDは駆動部品がないため衝撃にも強いという特長があります。

</QuizAnswer>

## 問8

システムの信頼性設計において、システムの一部に障害が発生しても全体としての機能を維持し続けるように設計する考え方を何というか。

1. フールプルーフ
2. フェールセーフ
3. フォールトトレラント
4. フェールオーバー


<QuizAnswer>

**答え: 3**

フォールトトレラントとは、システムの一部に障害が発生しても、冗長構成などにより全体として正常に動作し続けられるように設計する考え方です。フェールセーフは障害時に安全な状態に移行する設計、フールプルーフは人間が誤操作しても危険が生じないようにする設計、フェールオーバーは障害時に待機系に自動切替する仕組みです。

</QuizAnswer>

## 問9

論理演算において、二つの入力がともに「1」のときだけ出力が「1」になる演算はどれか。

1. AND演算
2. OR演算
3. NOT演算
4. XOR演算


<QuizAnswer>

**答え: 1**

AND演算（論理積）は、すべての入力が「1」のときのみ出力が「1」になる演算です。OR演算（論理和）はいずれかの入力が「1」であれば出力が「1」になり、NOT演算（否定）は入力を反転させます。XOR演算（排他的論理和）は二つの入力が異なるときに「1」を出力します。

</QuizAnswer>

## 問10

OSが提供するファイル管理の仕組みにおいて、ファイルを階層的に整理するための入れ物を何というか。

1. レジスタ
2. ディレクトリ（フォルダ）
3. セクタ
4. バッファ


<QuizAnswer>

**答え: 2**

ディレクトリ（フォルダ）は、OSがファイルを階層的に整理・管理するための入れ物です。ディレクトリの中にさらにディレクトリを作ることで、ツリー構造（木構造）でファイルを管理できます。レジスタはCPU内部の高速な記憶領域、セクタはディスクの最小記録単位、バッファはデータを一時的に保持する領域です。

</QuizAnswer>
