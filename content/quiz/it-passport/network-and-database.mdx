## 問1

インターネット上で、ドメイン名とIPアドレスの対応関係を管理し、名前解決を行うシステムはどれか。

1. DHCP
2. DNS
3. SMTP
4. FTP


<QuizAnswer>

**答え: 2**

DNS（Domain Name System）は、人間が覚えやすいドメイン名（例: example.com）を、コンピュータが通信に使うIPアドレス（例: 192.168.1.1）に変換する仕組みです。DHCPはIPアドレスの自動割り当て、SMTPはメール送信、FTPはファイル転送に使われるプロトコルです。

</QuizAnswer>

## 問2

TCP/IPの4階層モデルにおいて、HTTPやSMTPなどのプロトコルが属する層はどれか。

1. ネットワークインタフェース層
2. インターネット層
3. トランスポート層
4. アプリケーション層


<QuizAnswer>

**答え: 4**

HTTPやSMTP、FTP、DNSなどのプロトコルはアプリケーション層に属します。アプリケーション層は、ユーザーが直接利用するサービスを提供する最上位の層です。トランスポート層にはTCPやUDP、インターネット層にはIPが属します。

</QuizAnswer>

## 問3

関係データベースにおいて、表の「行」に相当する用語として最も適切なものはどれか。

1. カラム
2. レコード
3. フィールド
4. スキーマ


<QuizAnswer>

**答え: 2**

関係データベースでは、表の行を「レコード（タプル）」、列を「カラム（フィールド、属性）」と呼びます。1つのレコードが1件分のデータを表します。スキーマはデータベースの構造定義のことです。

</QuizAnswer>

## 問4

無線LANの規格であるIEEE 802.11に関連するブランド名として広く使われているものはどれか。

1. Bluetooth
2. Wi-Fi
3. NFC
4. Zigbee


<QuizAnswer>

**答え: 2**

Wi-Fiは、IEEE 802.11シリーズの無線LAN規格に基づく製品の相互接続性を認証するブランド名です。Bluetoothは近距離の機器間通信、NFCは非接触型の近距離通信技術、Zigbeeはセンサーネットワーク向けの低消費電力通信規格です。

</QuizAnswer>

## 問5

SQLにおいて、テーブルから条件に合致するデータを抽出するために使用する命令はどれか。

1. INSERT
2. UPDATE
3. SELECT
4. DELETE


<QuizAnswer>

**答え: 3**

SELECT文は、テーブルからデータを検索・抽出するためのSQL命令です。WHERE句と組み合わせて条件を指定できます。INSERTはデータの挿入、UPDATEはデータの更新、DELETEはデータの削除に用いられます。

</QuizAnswer>

## 問6

IPアドレスのうち、組織内のネットワークで自由に使用できるように予約されたアドレスはどれか。

1. グローバルIPアドレス
2. プライベートIPアドレス
3. ループバックアドレス
4. ブロードキャストアドレス


<QuizAnswer>

**答え: 2**

プライベートIPアドレスは、組織内のLANなどで自由に使用できるIPアドレスで、インターネット上では直接使用できません。代表的な範囲として 192.168.0.0〜192.168.255.255 などがあります。インターネットに接続するにはNAT等でグローバルIPアドレスに変換する必要があります。

</QuizAnswer>

## 問7

電子メールの送信に使われるプロトコルはどれか。

1. POP3
2. IMAP
3. SMTP
4. HTTP


<QuizAnswer>

**答え: 3**

SMTP（Simple Mail Transfer Protocol）は、電子メールを送信・転送するためのプロトコルです。POP3やIMAPはメールの受信（取得）に使われるプロトコルです。HTTPはWebページの閲覧に使用されるプロトコルであり、メール送信には使いません。

</QuizAnswer>

## 問8

関係データベースの正規化に関する説明として、最も適切なものはどれか。

1. データを暗号化してセキュリティを高めること
2. データの冗長性を排除し、データの整合性を保ちやすくすること
3. データベースのバックアップを自動化すること
4. データベースの処理速度を最大化するためにインデックスを設定すること


<QuizAnswer>

**答え: 2**

正規化とは、関係データベースにおいてデータの重複（冗長性）を排除し、データの矛盾や不整合が起こりにくい構造にテーブルを分割・整理することです。第1正規形、第2正規形、第3正規形などの段階があります。正規化により更新時の異常を防ぎ、データの整合性を維持しやすくなります。

</QuizAnswer>

## 問9

LANの接続形態（トポロジ）の一つで、中央にハブやスイッチを配置し、各端末を放射状に接続する形態はどれか。

1. バス型
2. リング型
3. スター型
4. メッシュ型


<QuizAnswer>

**答え: 3**

スター型は、中央の集線装置（ハブやスイッチ）に各端末を個別に接続する形態です。現在の有線LANで最も一般的な構成です。バス型は1本の回線に全端末を接続する形態、リング型は環状に接続する形態、メッシュ型は各端末同士を相互に接続する形態です。

</QuizAnswer>

## 問10

Webページを閲覧するとき、ブラウザとWebサーバ間の通信を暗号化するプロトコルはどれか。

1. HTTP
2. HTTPS
3. FTP
4. TELNET


<QuizAnswer>

**答え: 2**

HTTPS（HTTP over TLS/SSL）は、HTTPの通信をTLS（Transport Layer Security）で暗号化したプロトコルです。URLが「https://」で始まるWebサイトでは、通信内容が暗号化されるため、盗聴や改ざんのリスクを低減できます。HTTPは暗号化なしの通信であり、FTPはファイル転送、TELNETは遠隔ログインのプロトコルです。

</QuizAnswer>
