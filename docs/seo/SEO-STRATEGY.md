# シカクのミカタ SEO戦略プラン

**策定日:** 2026年3月16日
**対象サイト:** https://shikaku-no-mikata.com
**事業領域:** マイナー資格・通信講座の徹底比較メディア

---

## 1. 現状分析

### サイト概要
- **記事数:** 44本（通常記事）+ 79本（クイズコンテンツ）
- **技術基盤:** Next.js 14 (App Router) + MDX + Tailwind CSS
- **カテゴリ:** 6カテゴリ（ライフスタイル、クリエイティブ、IT・Web、医療・健康、ビジネス、資格比較）
- **分析ツール:** Google Analytics 4（G-FRHH5NDL94）導入済み

### カテゴリ別記事分布
| カテゴリ | 記事数 | 比率 |
|---------|-------|------|
| ライフスタイル | 25 | 57% |
| クリエイティブ | 5 | 11% |
| 医療・健康 | 4 | 9% |
| IT・Web | 4 | 9% |
| 資格比較 | 3 | 7% |
| ビジネス | 3 | 7% |

### 現在の強み
1. **マイナー資格への深い特化** — 温泉ソムリエ、ペットシッター士、発酵食品ソムリエ等、大手が深くカバーしない領域
2. **資格クラスター構造の部分的実現** — 整理収納アドバイザー（4記事）、薬膳コーディネーター（3記事）等
3. **ネガティブキーワードへの対応** —「意味ない」系記事が3本あり、競合が避ける領域に踏み込み済み
4. **クイズコンテンツ** — 79問のクイズでユーザーエンゲージメント向上
5. **技術的SEO基盤** — SSG、JSON-LD構造化データ、OGP/Twitter Card、サイトマップ整備済み

### 現在の課題
1. **記事数不足** — 安定トラフィックには100-200本が目安
2. **クラスター未完成** — 多くの資格で記事セットが不完全
3. **カテゴリ偏り** — ライフスタイルに57%集中、IT・Web/ビジネスが手薄
4. **ピラーコンテンツ不足** — カテゴリ横断のまとめ記事が少ない
5. **E-E-A-T弱い** — 著者情報、体験談、独自データが不足
6. **内部リンク構造** — クラスター間の導線が未最適化
7. **カテゴリ未設定記事** — 2記事がカテゴリ未割り当て

## 2. 競合分析サマリー

### 主要競合
| 競合 | タイプ | 強み | 弱み |
|-----|-------|------|------|
| BrushUP学び | プラットフォーム | 講座数6万超、圧倒的網羅性 | マイナー資格の深い解説なし |
| ユーキャン | 通信講座 | ブランド力、SEO投資 | 自社講座以外の情報薄い |
| 日本の資格・検定 | メディア | 独自調査データ、権威性 | マイナー資格カバー弱い |
| 資格の難易度 | 個人サイト | 網羅性（全資格A-E評価） | 記事の深さに欠ける |
| 資格の取り方 | 個人サイト | 実用的切り口 | 差別化要因が薄い |

- **本音レビュー（意味ない系）** — 大手は構造的に書けない
- **資格クラスター戦略** — 1資格を深掘りする専門性

## 7. 内部リンク戦略

### リンク構造設計
```
ピラーコンテンツ（まとめ記事）
  ├── クラスター: 資格A
  │   ├── 概要記事 ←→ 難易度記事
  │   ├── 概要記事 ←→ 独学記事
  │   ├── 概要記事 ←→ 通信講座記事
  │   └── すべての記事 → ピラーコンテンツ
  ├── クラスター: 資格B
  │   └── （同上）
  └── 関連資格クラスター間のクロスリンク
      例: 薬膳コーディネーター ←→ 食生活アドバイザー
```

### 実装方針
1. 各クラスター内の記事は相互リンクで接続
2. すべてのクラスター記事からピラーコンテンツへリンク
3. 関連カテゴリのクラスター間でクロスリンク
4. サイドバーに「同じ資格の記事」セクション（既に実装済み — 強化）
5. 記事末尾に「関連資格」セクションの追加

## 9. リスク要因と対策

| リスク | 影響度 | 対策 |
|-------|-------|------|
| AI生成コンテンツの評価厳格化（Google 2026年方針） | 高 | 一次情報（体験談・独自調査）の充実、人間らしい文体 |
| 大手メディアのマイナー資格参入 | 中 | クラスター戦略で専門性の壁を構築、速度で勝負 |
| コアアップデートによる順位変動 | 中 | E-E-A-T投資、多角的なトラフィック獲得（SNS等） |
| 記事品質の維持困難（量産時） | 中 | テンプレート化 + 品質チェックリスト運用 |
| 試験制度変更への対応遅れ | 低 | 定期的な公式サイトチェック体制 |
