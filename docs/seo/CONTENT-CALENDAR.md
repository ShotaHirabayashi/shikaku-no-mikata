## コンテンツ品質チェックリスト

各記事公開前に確認:

### 基本品質
- [ ] タイトルにターゲットKWが含まれている
- [ ] meta description が120文字以内で魅力的
- [ ] H2/H3の構造が論理的
- [ ] 文字数 3,000字以上
- [ ] 公式サイトへの出典リンクがある
- [ ] 最終更新日が正しい

### SEO
- [ ] frontmatter の category / tags が適切
- [ ] 同クラスター内の他記事へのリンクがある
- [ ] 関連ピラーコンテンツへのリンクがある
- [ ] 画像にalt属性がある
- [ ] 表やリストを適切に使用している

### E-E-A-T
- [ ] 一次情報（公式データ・体験談）が含まれている
- [ ] 著者の専門性が示されている
- [ ] 情報が最新である（年度を確認）
- [ ] 「〜とは」形式の定義文がある（AI検索対策）
