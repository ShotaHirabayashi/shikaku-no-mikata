## 技術実装の優先順位（エンジニアタスク）

### 最優先（Phase 1で実装）
```
1. FAQPage JSON-LD スキーマの実装
   - 資格概要記事・難易度記事に「よくある質問」セクション追加
   - app/[slug]/page.tsx に FAQPage 構造化データ生成ロジック追加

2. BreadcrumbList JSON-LD の追加
   - 現在のパンくずUIに対応するJSON-LDを出力

3. 著者プロフィールセクション
   - frontmatter に author フィールド追加
   - 記事ページに著者情報表示コンポーネント追加

4. aboutページのリニューアル
   - 運営者情報、保有資格、運営方針の充実
```

### 高優先（Phase 2で実装）
```
5. HowTo JSON-LD スキーマ
6. Review/AggregateRating スキーマ
7. llms.txt の設置
8. 内部リンク自動提案ロジック
```

### 中優先（Phase 3以降）
```
9. ItemList スキーマ
10. Person スキーマ
11. IndexNow
12. サイト内検索改善
```
